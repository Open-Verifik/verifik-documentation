# 🇨🇦 퀘벡 — 운전면허

이 API는 **문서 번호**로 캐나다 **퀘벡**에서 발급된 운전면허를 공식 기록과 대조합니다. 등급, 상태, 발급 관련 정보, 확인 번호 등이 반환될 수 있습니다. **KYC**, **모빌리티 온보딩**, **컴플라이언스**에 사용하세요.

**`Authorization` 헤더가 필요합니다.**

## API 레퍼런스

### 엔드포인트

```
GET https://api.verifik.co/v2/ca/quebec/driver-license
```

면허증에 표기된 **문서 번호**로 퀘벡 운전면허를 검증합니다.

### 헤더(Headers)

| 이름           | 값                  |
| -------------- | ------------------- |
| Content-Type   | `application/json`  |
| Authorization  | `Bearer <token>`    |

### 매개변수(Parameters)

| 이름 | 형식 | 필수 | 설명 | 예시 |
| ---- | ---- | ---- | ---- | ---- |
| `documentNumber` | string | **예** | 조회할 면허 번호. 증명서와 동일하게 입력합니다. | `L12345678901` |
