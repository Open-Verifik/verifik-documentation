# 🇨🇱 칠레 — 운전면허

본 서비스는 운전면허 번호를 쿼리 매개변수로 사용하여 칠레 운전면허에 대한 상세 정보를 조회할 수 있도록 합니다. RUT(고유 납세자 번호), 주소, 면허 등급, 통제 일자 등 운전면허와 관련된 핵심 데이터를 구조화된 형식으로 제공합니다.

### 엔드포인트

```
GET https://api.verifik.co/v2/cl/driver-license
```

칠레 운전면허를 **면허 번호**(쿼리 `documentNumber`)로 조회합니다. 등기소 가용성에 따라 일반적으로 **RUT**, **면허 등급**, **제한 사항**, **통제 일자**, **자치구**, **주소** 필드가 반환됩니다. 운전 자격이 중요한 **차량**, **인력**, **KYC** 점검에 적합합니다.

### 헤더(Headers)

| 이름           | 값                  |
| -------------- | ------------------- |
| Accept         | `application/json`  |
| Authorization  | `Bearer <token>`    |

### 매개변수(Parameters)

| 이름             | 형식    | 필수 | 설명                                                     | 예시        |
| ---------------- | ------- | ---- | -------------------------------------------------------- | ----------- |
| `documentNumber` | string  | **예** | 조회할 면허 번호. 공백이나 점 없이 입력합니다.           | `12345678`  |

## 주요 기능

- **RUT 검증**: 칠레 납세자 번호의 진위를 확인합니다.
- **면허 등급 정보**: 면허 등급(A, B, C 등)에 대한 상세 정보를 제공합니다.
- **통제 일자**: 면허 통제 일자와 마지막 통제 일자를 제공합니다.
- **운전자 정보**: 운전자의 이름, 주소, 자치구를 조회할 수 있습니다.
- **면허 제한 사항**: 운전 제한 사항에 대한 정보를 제공합니다.
- **절차 정보**: 면허 발급 절차(재발급, 갱신 등) 상세 정보를 제공합니다.
- **다양한 프로그래밍 언어 지원**: JavaScript, Python, PHP, Swift 예제를 제공합니다.
- **실시간 데이터**: 최신의 운전면허 정보를 조회할 수 있습니다.
- **종합 오류 처리**: 다양한 시나리오에 대한 상세한 오류 응답을 제공합니다.
- **구조화된 응답**: 서명 검증을 포함한 잘 정리된 JSON 응답을 제공합니다.
- **칠레 등기소 연동**: 공식 칠레 운전면허 데이터베이스에 직접 접근합니다.
