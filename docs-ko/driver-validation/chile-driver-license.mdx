# 칠레 운전면허

**면허 번호**로 칠레 운전면허 상세를 조회합니다. **RUT**, 주소, **등급**, **통제일**, 시군구, **제한**, 절차(재발급 등) 등이 레지스트리에 따라 반환됩니다.

**플릿**, **인력**, **모빌리티**, 운전 자격 확인이 필요한 KYC에 적합합니다.

### Endpoint

```
https://api.verifik.co/v2/cl/driver-license
```

**`documentNumber`**(공백 없음)로 GET 요청합니다.

### Headers

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parameters

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `documentNumber` | string | Yes | 면허 번호 |

### Request

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/cl/driver-license", {
  params: { documentNumber: "12345678" },
  headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});
console.log(data);
```

  </TabItem>

## 기능

- **RUT·운전자**: 성명, 주소, 시군구.
- **등급·제한**: A, B, C 등 및 운전 제한.
- **통제일**: 정기·최종 통제.
- **절차**: 재발급 등(소스 반환 시).
- **REST JSON** 및 오류 처리.
