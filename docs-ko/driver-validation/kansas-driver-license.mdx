# 🇺🇸 캔자스주(미국) — 운전면허

**캔자스주**에서 발급된 운전면허의 **상태, 제한, 승인(endorsement), 지정(designation)** 등을 API로 검증합니다. 유효한 면허 번호와 필수 본인 확인 파라미터를 전달하면, 면허 상태·만료일·제한·승인·지정 등의 세부 정보를 응답으로 받을 수 있습니다.

**운전 자격 확인** 및 **규정 준수** 워크플로에 활용할 수 있습니다.

### 엔드포인트

```
GET https://api.verifik.co/v2/usa/kansas/driver-license
```

**`documentNumber`**(면허 번호), **`dateOfBirth`**(생년월일, **DD/MM/YYYY**), **`firstName`**, **`lastName`**을 쿼리 매개변수로 전달합니다. 날짜 형식은 API 요구사항을 따르세요.

### 헤더(Headers)

| 이름          | 값                 |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### 매개변수(Parameters)

| 이름 | 형식 | 필수 | 설명 | 예시 |
| ---- | ---- | ---- | ---- | ---- |
| `documentNumber` | string | 예 | 캔자스주 운전면허 번호. | `K12345678` |
| `dateOfBirth` | string | 예 | 생년월일 (**DD/MM/YYYY**). | `29/01/1974` |
| `firstName` | string | 예 | 이름. | `MATEO` |
| `lastName` | string | 예 | 성. | `VERIFIK` |
