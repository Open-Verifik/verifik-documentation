# 🇵🇪 페루 — 운전면허

이 서비스는 **DNI**(*Documento Nacional de Identidad*)로 페루 운전면허에 대한 상세 정보를 조회합니다. 응답에는 유효 면허, 절차 이력, 규정 과정, 제한, 제재 등이 포함되며 **차량 관리**, **인력** 및 **KYC** 검증에 유용합니다.

### Endpoint

```
GET https://api.verifik.co/v2/pe/driver-license
```

`documentType=DNI`와 8자리 `documentNumber`로 조회합니다. **성명**, **면허 등급**(A, AIV, B), **licenses**, **procedures**, **regulatoryCourses**, **sanctions**, **tickets** 등이 반환됩니다(등록부 기준).

### 헤더

| Name          | Value              |
| ------------- | ------------------ |
| Accept        | `application/json` |
| Authorization | `Bearer <token>`   |

### 매개변수

| 이름 | 유형 | 필수 | 설명 | 예시 |
| ---- | ---- | ---- | ---- | ---- |
| `documentType` | string | **예** | 문서 유형. 허용 값: `DNI`. | `DNI` |
| `documentNumber` | string | **예** | 조회할 DNI(공백·점 없음). | `44141274` |

### 참고

- `documentNumber`는 공식 등록부에 운전면허 기록이 있는 DNI와 일치해야 합니다.

## 주요 기능

- **DNI 조회**: `documentType=DNI` 및 문서 번호.
- **면허 등급**: A, AIV, B 및 임시 허가 지표.
- **유효 면허**: 번호, 등급, 상태, 제한, 발급 센터, 만료일.
- **절차 이력**: 갱신(Revalidación) 및 상태(예: Vigente, VENCIDA).
- **규정 과정**: 등록·이수 정보(해당 시).
- **제재·벌금**: 등록부에 따라 선택적 배열.
- **구조화된 응답**: Verifik 서명 및 인증 메타데이터가 포함된 JSON.
