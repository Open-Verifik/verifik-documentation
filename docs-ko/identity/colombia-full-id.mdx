### 엔드포인트

```
GET https://api.verifik.co/v2/co/cedula/extra
```

**`documentType`**=`CC`, **`documentNumber`**, 발급일 **`date`**(`DD/MM/YYYY`). 동일 필드로 **`POST`** JSON 도 지원합니다.

### 헤더(Headers)

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 매개변수(Parameters)

| name | type | required | description |
| --- | --- | --- | --- |
| `documentType` | string | yes | **`CC`** 만. |
| `documentNumber` | string | yes | 공백·구두점 없는 번호. |
| `date` | string | yes | 문서 **발급일** **`DD/MM/YYYY`**. |

### 요청(Request)

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/cedula/extra", {
	params: { documentType: "CC", documentNumber: "123456789", date: "10/10/2020" },
	headers: {
		Accept: "application/json",
		Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
	},
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os, requests

url = "https://api.verifik.co/v2/co/cedula/extra"
headers = {"Accept": "application/json", "Authorization": f"Bearer {os.getenv('VERIFIK_TOKEN')}"}
params = {"documentType": "CC", "documentNumber": "123456789", "date": "10/10/2020"}
r = requests.get(url, headers=headers, params=params)
print(r.json())
```

### 참고(Notes)

- **`date`** 는 **발급일**입니다 (생년월일 아님).
- 민감 개인정보가 포함될 수 있습니다.
