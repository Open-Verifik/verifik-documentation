# 🇨🇱 칠레 — 납세자(RUT) 조회

본 엔드포인트는 칠레 **RUT**에 대한 **세무 지향** 신원 데이터가 필요할 때 사용합니다(민정 등기소 cedula 흐름이 아님). **RUN/민정** 조회에 최적화된 **[칠레 신원 검증](/identity/chile)** (`v2/cl/cedula`)을 보완합니다.

## API 레퍼런스

### 엔드포인트

```
https://api.verifik.co/v2/cl/taxpayer
```

응답은 **세무 등록부** 사용에 부합하는 이름 필드와 식별자를 반환합니다. **KYB**, **공급업체 온보딩**, **RUT**가 **등록 납세자** 데이터와 일치해야 하는 **세금계산서** 검증에 적합합니다.

### 헤더(Headers)

| 이름           | 값                  |
| -------------- | ------------------- |
| Accept         | `application/json`  |
| Authorization  | `Bearer <token>`    |

### 매개변수(Parameters)

| 이름             | 형식    | 필수 | 설명                                                            |
| ---------------- | ------- | ---- | --------------------------------------------------------------- |
| `documentType`   | string  | 예   | `RUT`이어야 합니다.                                             |
| `documentNumber` | string  | 예   | 납세자 식별자(숫자만; 형식은 서비스가 정규화).                  |

### 요청(Request)

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/cl/taxpayer", {
  params: { documentType: "RUT", documentNumber: "263007905" },
  headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});
console.log(data);
```
