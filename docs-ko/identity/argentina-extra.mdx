# 아르헨티나 신원 검증(확장)

Verifik 신원 확인 API는 공식 정부 데이터로 아르헨티나 국민을 인증합니다. 이 엔드포인트는 표준 cédula 응답을 넘어 **생년월일**, **성별**, 구조화된 이름 토큰, **생존 상태**(`isAlive`)를 반환합니다.

## 이 API는 무엇을 검증합니까?

당사 API는 공식 기록과 직접 연결되어 다음을 검증합니다.

-   **성명 및 DNI 번호**: *Documento Nacional de Identidad*(DNI), 문서 유형 `DNIAR` 지원.
-   **신원 일치**: 문서 번호가 등록된 사람과 일치하는지 확인.
-   **추가 데이터**: **dateOfBirth**, **gender**, **firstName**, **lastName**, **fullName**, **arrayName**(이름 파트), **isAlive** 반환.

이를 검증하면 상대방이 실존하며 유효한 문서를 보유하고 있음을 확신할 수 있어 신원 도용 및 사기 위험을 크게 낮출 수 있습니다.

## API 레퍼런스

### 엔드포인트

```
https://api.verifik.co/v2/ar/cedula/extra
```

### 헤더

| 이름          | 값                 |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### 매개변수

| 이름             | 형식   | 필수 | 설명                                                                 |
| ---------------- | ------ | -------- | --------------------------------------------------------------------------- |
| `documentType`   | string | 예      | 문서 유형. 허용 값: `DNIAR`.                                      |
| `documentNumber` | string | 예      | 조회할 DNI 번호(공백 없음).                                        |

### 참고

- `documentType`는 `DNIAR`여야 합니다.
- 가능한 경우 `gender`는 한 글자 코드(예: `F` 또는 `M`)로 반환됩니다.
- `isAlive`는 조회 시 등록부가 해당인을 생존으로 보고하는지 나타냅니다.

-   **핀테크 및 은행**: 생년월일 및 생존 상태 확인으로 온보딩 강화.
-   **전자상거래 및 배송**: 배송 기사 또는 고가치 계정 보유자를 더 강한 신원 신호로 검증.
-   **HR 및 채용**: 공식 기록과 대조하여 후보자 신원 세부 정보 확인.
-   **보험 및 헬스케어**: 검증된 생년월일로 자격 및 연령 관련 규칙 지원.

## 공식 출처 및 신뢰성

아르헨티나 공식 정부 출처와 직접 연결하여 검증되고 최신 정보를 제공합니다.
모든 조회는 엄격한 보안 및 규제 기준에 따라 처리됩니다.

## 주요 이점

-   **더 풍부한 신원 신호**: 이름 및 문서 번호 외 생년월일, 성별, 생존 상태.
-   **자동화된 컴플라이언스**: 사용자 경험을 해치지 않고 KYC를 간소화.
-   **즉시 결과**: 수 초 내 처리, 실시간 온보딩에 적합.
-   **신뢰할 수 있는 데이터**: 공식 기록에서 직접 취득.
-   **쉬운 통합**: REST API 또는 호환 SDK로 연결.

## 컴플라이언스 및 보안

데이터 안전을 최우선으로 합니다. Verifik은 HTTPS/TLS 1.3과 엄격한 개인정보 기준을 사용합니다.
서비스는 24/7 가용성을 모니터링하며 역할 기반 액세스 제어를 제공합니다.
