### 엔드포인트

```
GET https://api.verifik.co/v2/co/situacion-militar
```

**쿼리**로 **`documentType=CC`** 와 **`documentNumber`** 를 보냅니다. **CC**(시민증)만 허용됩니다.

### 헤더(Headers)

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 매개변수(Parameters)

| name | type | required | description |
| --- | --- | --- | --- |
| `documentType` | string | yes | 반드시 `CC`. CE·PPT 등은 이 엔드포인트에서 거절됩니다. |
| `documentNumber` | string | yes | 조회할 시민증 번호. |

### 요청(Request)

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/situacion-militar", {
	params: { documentType: "CC", documentNumber: "1121329661" },
	headers: {
		Accept: "application/json",
		Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
	},
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os, requests

url = "https://api.verifik.co/v2/co/situacion-militar"
headers = {"Accept": "application/json", "Authorization": f"Bearer {os.getenv('VERIFIK_TOKEN')}"}
params = {"documentType": "CC", "documentNumber": "1121329661"}
r = requests.get(url, headers=headers, params=params)
print(r.json())
```

### 참고(Notes)

- **`documentType`** 은 **`CC`** 만 유효합니다. UI에 다른 신분 유형이 보여도 **`CC`** 가 아니면 **409** 가 납니다.
