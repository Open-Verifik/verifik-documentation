# 코스타리카 국적 신원 확인(CCCR)

공식 데이터로 **CCCR** 신분을 확인합니다.

## API 참조

### 엔드포인트

```
https://api.verifik.co/v2/cr/cedula
```

**`documentType`**을 `CCCR`로, **`documentNumber`**를 공백 없이 전달합니다.

### 헤더

| 이름 | 값 |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 매개변수

| 이름 | 유형 | 필수 | 설명 |
| --- | --- | --- | --- |
| `documentType` | string | 예 | 허용값: `CCCR`. |
| `documentNumber` | string | 예 | 신분증 번호(공백 없음). |

### 요청

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/cr/cedula", {
  params: { documentType: "CCCR", documentNumber: "123456789" },
  headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});
console.log(data);
```
