# 🇵🇪 페루 — DNI(확장 정보)

Verifik의 신원 확인 API는 페루 시민의 DNI(*Documento Nacional de Identidad*) 번호로 확장된 신원 정보를 조회할 수 있게 해 줍니다. KYC(Know Your Customer) 절차를 자동화하고, 사기를 예방하며, 규제 요구사항을 손쉽게 충족할 수 있도록 설계되었습니다.

성명, 시민 상태, 출생일, 문서 정보, 주소, 사진을 포함한 종합 개인 정보를 빠르고 안전하게 자동으로 조회해야 하는 비즈니스를 위해 만들어졌습니다.

**이 정보에 안전하게 접근하려면 Authorization 헤더가 필요합니다.**

## 이 API는 무엇을 검증합니까?

당사의 API는 페루의 공식 기록과 직접 연결되어 다음을 검증합니다.

-   **성명 및 DNI 번호**: *Documento Nacional de Identidad*(DNI) 문서를 지원합니다.
-   **신원 일치 여부**: 제공된 이름이 DNI 번호와 일치하는지 확인합니다.
-   **확장 개인 정보**: 시민 상태, 출생일, 성별, 주소 정보를 반환합니다.
-   **문서 정보**: 문서 종류, 번호, 발급일, 만료일, 검증 자릿수를 검증합니다.
-   **지리 정보**: 위치 식별을 위한 ubigeoReniec 코드를 제공합니다.
-   **사진 데이터**: 시민의 사진을 base64 형식으로 제공합니다.

이러한 항목을 검증함으로써 거래 상대가 실존하며 유효한 문서를 보유하고 있음을 확신할 수 있고, 신원 도용 및 사기 위험을 크게 낮출 수 있습니다.

### 엔드포인트

```
GET https://api.verifik.co/v3/pe/cedula/extra
```

### 헤더(Headers)

| 이름           | 값                  |
| -------------- | ------------------- |
| Content-Type   | `application/json`  |
| Authorization  | `Bearer <token>`    |

### 매개변수(Parameters)

| 이름 | 형식 | 필수 | 설명 | 예시 |
| ---- | ---- | ---- | ---- | ---- |
| `documentType` | string | **예** | 문서 종류. 허용 값: DNI. | `DNI` |
| `documentNumber` | string | **예** | 조회 대상자의 문서 번호. | `123456789` |

## 일반적인 활용 사례

-   **고급 KYC 절차**: 종합적인 신원 확인을 통해 컴플라이언스와 사기 탐지를 강화합니다.
-   **금융 서비스**: 은행 및 금융 서비스 신청을 위한 상세 신원 데이터를 검증합니다.
-   **고용 검증**: 채용 절차에서 종합 개인 정보를 확인합니다.
-   **컴플라이언스 및 실사**: 확장 신원 정보가 요구되는 페루 규정을 준수합니다.
-   **사기 탐지**: 사진과 인구 통계 정보를 포함한 확장 데이터를 활용해 고급 사기 예방을 수행합니다.

## 공식 출처와 신뢰성

페루 정부의 공식 기관(RENIEC)과 직접 연결되어 검증되고 최신 상태의 정보를 제공합니다.
모든 조회는 보안 및 규제 표준을 엄격히 준수하여 처리됩니다.

## 주요 이점

-   **자동화된 컴플라이언스**: 사용자 경험을 해치지 않으면서 KYC 점검을 자동화하고 사기를 방지합니다.
-   **확장된 데이터 접근**: 사진, 주소, 시민 상태, 인구 통계 정보를 포함한 종합 신원 정보를 받을 수 있습니다.
-   **즉각적인 결과**: 수 초 내에 검증을 처리하므로 실시간 신원 확인에 적합합니다.
-   **신뢰할 수 있는 데이터**: 공식 RENIEC 기록을 출처로 하는 데이터를 사용합니다.
-   **간편한 통합**: REST API 또는 호환 SDK를 통해 손쉽게 연결할 수 있습니다.

## 컴플라이언스와 보안

Verifik은 데이터 안전을 최우선으로 합니다. 고급 암호화(HTTPS / TLS 1.3)와 엄격한 개인정보 관리 표준을 사용해 기밀성을 보장합니다.
서비스는 24/7 가용성 모니터링을 적용하고, 팀 접근 제어를 위해 역할 기반 접근 제어를 제공합니다.

## Verifik 소개

Verifik은 라틴아메리카 전역에서 신원 확인, 컴플라이언스, 사기 예방을 제공하는 선도적인 플랫폼입니다.
저희 API는 KYC, KYB, AML, 생체 인증 절차를 자동화하며, 콜롬비아, 멕시코, 페루, 칠레, 우루과이, 아르헨티나, 브라질 및 그 외 지역의 공식 데이터 소스와 비즈니스를 연결합니다.
