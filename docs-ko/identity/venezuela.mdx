# 베네수엘라 국적 신원 확인(CCVE)

Verifik API는 **CCVE**(*Cédula de Identidad*) 기반으로 **베네수엘라 국민**을 인증합니다. KYC를 간소화하고 사기를 줄이며 규정 준수를 돕습니다.

## 무엇을 검증하나요?

-   **이름 및 문서 번호**: 국적 신분증 **CCVE**.
-   **일치 여부**: 반환 이름이 등록 정보와 일치합니다.
-   **선택 필드**: 생년월일, 성별 등 데이터 제공 시 포함될 수 있습니다.

## API 참조

### 엔드포인트

```
https://api.verifik.co/v2/ve/cedula
```

이 엔드포인트는 **베네수엘라 국민**의 **CCVE** 신분을 조회합니다. **`documentNumber`**에 신분증 번호를 전달하면 서비스가 **CCVE**로 처리하고 구조화된 이름 필드를 반환합니다. **외국인 거주자**는 [`GET /v2/ve/foreigner-id`](/verifik-ko/identity/venezuela-foreigner-id)(**CEVE**)를 사용하세요.

### 헤더

| 이름 | 값 |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 매개변수

| 이름 | 유형 | 필수 | 설명 |
| --- | --- | --- | --- |
| `documentNumber` | string | 예 | 국적 신분증(CCVE) 번호. 쿼리에 `documentType`은 필요 없으며 이 엔드포인트에서는 **CCVE**가 적용됩니다. |
