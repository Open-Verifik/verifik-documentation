# 🇻🇪 베네수엘라 — 외국인 신분증(CEVE)

**CEVE**(*Cédula de Identidad de Extranjero*)를 사용하는 **외국인** 신원을 확인합니다.

## 무엇을 검증하나요?

-   **CEVE 신원**: 문서 번호와 등록된 이름의 일치.
-   **문서 유형**: 응답에 `documentType: "CEVE"`가 포함됩니다.

## API 참조

### 엔드포인트

```
https://api.verifik.co/v2/ve/foreigner-id
```

**외국인** **CEVE** 전용입니다. **`documentNumber`**만 전달하면 서비스가 **CEVE**로 처리합니다. **국민(CCVE)**은 [`GET /v2/ve/cedula`](/verifik-ko/identity/venezuela)를 사용하세요.

### 헤더

| 이름 | 값 |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 매개변수

| 이름 | 유형 | 필수 | 설명 |
| --- | --- | --- | --- |
| `documentNumber` | string | 예 | CEVE 번호. 쿼리에 `documentType` 불필요—이 엔드포인트는 **CEVE**입니다. |
