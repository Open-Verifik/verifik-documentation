### 엔드포인트

```
GET https://api.verifik.co/v2/co/foreigner-id/ppt
```

**`documentNumber`** 와 발급일 **`expeditionDate`**(`DD/MM/YYYY`)로 조회합니다. 동일 필드로 **`POST`** JSON 본문도 지원합니다.

### 헤더(Headers)

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 매개변수(Parameters)

| name | type | required | description |
| --- | --- | --- | --- |
| `documentNumber` | string | yes | 공백·구두점 없는 PPT 번호. |
| `expeditionDate` | string | yes | 발급일 **`DD/MM/YYYY`**. |

### 요청(Request)

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/foreigner-id/ppt", {
	params: { documentNumber: "123456789", expeditionDate: "07/03/2022" },
	headers: {
		Accept: "application/json",
		Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
	},
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os, requests

url = "https://api.verifik.co/v2/co/foreigner-id/ppt"
headers = {"Accept": "application/json", "Authorization": f"Bearer {os.getenv('VERIFIK_TOKEN')}"}
params = {"documentNumber": "123456789", "expeditionDate": "07/03/2022"}
r = requests.get(url, headers=headers, params=params)
print(r.json())
```

### 참고(Notes)

- 날짜 형식은 **`DD/MM/YYYY`** 입니다.
- **`GET`** / **`POST`** 모두 동일 처리입니다.
- 원천 장애 시 **404** / **500** 이 날 수 있습니다.
