### 엔드포인트

```
GET https://api.verifik.co/v2/co/foreigner-id/ppt
```

**`documentNumber`** 와 발급일 **`expeditionDate`**(`DD/MM/YYYY`)로 조회합니다. 동일 필드로 **`POST`** JSON 본문도 지원합니다.

### 헤더(Headers)

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 서류 요건

**대상:** 콜롬비아 **Migración Colombia**에서 **이민 상태**(예: VIGENTE, 만료일)가 필요한 *Permiso de Protección Temporal* (**PPT**) 보유 **베네수엘라 국민**.

| 필드 | 안내 |
| --- | --- |
| **`documentNumber`** | PPT 번호, **숫자만**. 콜롬비아에서 허가 번호는 보통 **최대 7자리**. 일부 시스템(SENA, 급여)은 **15자리**로 패딩 — 기록에 있는 숫자 값을 전송. Migración 예: `2081381`. |
| **`expeditionDate`** | 발급일 **`DD/MM/YYYY`**(예: `07/03/2022`). 필수. |

**별도 제공:** `documentType=PPT`로 [`/v2/co/cedula`](/verifik-ko/identity-validation/colombia/colombian-citizen)에서 이름형 조회(5–10자리 검증, 발급일 불필요).

전체 비교: [신분증 유형 가이드](/verifik-ko/identity-validation/colombia/colombia-identity-documents-guide).

### 매개변수(Parameters)

| name | type | required | description |
| --- | --- | --- | --- |
| `documentNumber` | string | yes | PPT 번호, **숫자만**. 일반적으로 **최대 7자리**(일부 시스템은 15자리 패딩). 예: `2081381`. |
| `expeditionDate` | string | yes | 발급일 **`DD/MM/YYYY`**(예: `07/03/2022`). 문서와 일치해야 합니다. |

### 요청(Request)

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/foreigner-id/ppt", {
	params: { documentNumber: "123456789", expeditionDate: "07/03/2022" },
	headers: {
		Accept: "application/json",
		Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
	},
});
console.log(data);
```

import os, requests

### 참고(Notes)

- 날짜 형식은 **`DD/MM/YYYY`** 입니다.
- **`GET`** / **`POST`** 모두 동일 처리입니다.
- 원천 장애 시 **404** / **500** 이 날 수 있습니다.
