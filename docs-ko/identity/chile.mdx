# 칠레 국적 신원 확인(RUN)

Verifik 신원 API는 **Rol Único Nacional(RUN)**으로 칠레 국민을 인증합니다. **KYC** 절차를 단순화하고 사기를 줄이며 규정 준수를 돕습니다.

사용자·직원·고객의 실제 신원을 빠르고 안전하게 확인해야 하는 기업을 위해 설계했습니다.

## 이 API가 검증하는 것

공식 기록과 연결하여 다음을 확인합니다.

- **성명 및 RUN**: *Rol Único Nacional* 지원.
- **일치 여부**: 제공한 이름이 RUN 번호와 대응하는지.

이를 통해 사칭 위험을 낮추고 가입 신뢰도를 높입니다.

## API 참조

### Endpoint

```
https://api.verifik.co/v2/cl/cedula
```

칠레 민정 기준 신원 확인: GET으로 **`documentType`** `RUN`, **`documentNumber`**(공백 없음)를 전달하면 응답에 **KYC**·온보딩·사기 점검에 쓸 수 있는 구조화된 이름 필드가 포함됩니다.

### Headers

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parameters

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `documentType` | string | Yes | 허용값: `RUN`. |
| `documentNumber` | string | Yes | 공백·구분자 없는 RUN. |

### Request

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/cl/cedula", {
  params: { documentType: "RUN", documentNumber: "212957739" },
  headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});
console.log(data);
```

## 활용 사례

- **핀테크·은행**: 계좌 개설·대출 시 신원 확인.
- **이커머스·배송**: 활성화 전 사용자·배송원 인증.
- **채용**: 서류 검증.
- **보험·의료**: 가입·혜택 전 본인 확인.

## 공식 출처

칠레 정부(민정 등) 공식 출처와 연결하여 검증된 정보를 제공합니다.

## 이점

- **자동 컴플라이언스**: 사용자 마찰을 최소화한 KYC.
- **빠른 응답**: 실시간 온보딩에 적합.
- **신뢰 데이터**: 공식 등록부 기반.

## 보안

HTTPS/TLS, 개인정보 보호 및 가용성 모니터링.

## Verifik 소개

Verifik는 라틴아메리카 지역의 신원·KYB·사기 방지 API를 제공합니다.
