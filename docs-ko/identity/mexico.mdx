# 멕시코 신원 확인

Verifik API는 **CURP**(*Clave Única de Registro de Población*)로 멕시코 시민 인증을 지원합니다. KYC를 간소화하고 사기를 줄이며 규정 준수를 돕습니다.

## 무엇을 검증하나요?

공식 기록과 연결하여 다음을 확인합니다.

-   **성명 및 CURP**
-   **개인 정보**: 생년월일, 성별, 국적
-   **일치 여부**: 제공한 이름이 CURP와 일치하는지

## API 참조

### 엔드포인트

```
https://api.verifik.co/v2/mx/curp
```

이 엔드포인트는 공식 기록을 기준으로 멕시코 국민의 **CURP**(Clave Única de Registro de Población)를 검증합니다. GET 요청에 `documentType`과 `documentNumber`를 전달하면, CURP가 인식될 때 성공 응답으로 신원 기록이 확인되며 검증된 이름과 인구통계 정보가 반환됩니다.

### 헤더

| 이름 | 값 |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 매개변수

| 이름 | 유형 | 필수 | 설명 |
| --- | --- | --- | --- |
| `documentType` | string | 예 | 허용 값: `CURP`. |
| `documentNumber` | string | 예 | 검증할 CURP. |

### 요청

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/mx/curp", {
  params: { documentNumber: "ABCD890513ABCDEF09", documentType: "CURP" },
  headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os, requests

url = "https://api.verifik.co/v2/mx/curp"
headers = {"Accept": "application/json", "Authorization": f"Bearer {os.getenv('VERIFIK_TOKEN')}"}
params = {"documentNumber": "ABCD890513ABCDEF09", "documentType": "CURP"}
r = requests.get(url, headers=headers, params=params)
print(r.json())
```

## 활용 사례

-   **핀테크·은행**: 계좌 개설, 대출
-   **이커머스·배송**: 사용자·배송 기사 인증
-   **채용**: 지원자 서류 검증
-   **보험·의료**: 가입·혜택 전 본인 확인

## 출처와 신뢰성

멕시코 공식 출처(예: RENAPO)와 연동, 보안 기준 준수.

## 이점

-   **자동 컴플라이언스**: 저마찰 KYC
-   **즉시 결과**: 실시간 온보딩
-   **신뢰 데이터**: 공식 기록
-   **쉬운 연동**: REST API·SDK

## 규정 준수·보안

HTTPS/TLS 1.3, 개인정보 보호, 가용성 모니터링.
