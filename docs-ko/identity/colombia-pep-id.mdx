### 엔드포인트

```
GET https://api.verifik.co/v2/co/foreigner-id/pep
```

**`documentNumber`** 와 발급일 **`expeditionDate`**(`DD/MM/YYYY`)로 조회합니다. 동일 필드로 **`POST`** JSON 본문도 지원합니다.

### 헤더(Headers)

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 서류 요건

**대상:** *Permiso Especial de Permanencia* (**PEP**) **이민 허가**를 보유한 콜롬비아 거주 **베네수엘라 국민** — AML «정치적 주요 인물» 스크리닝이 아닙니다([콜롬비아 PEP(AML)](/verifik-ko/identity-validation/colombia/colombian-politically-exposed-persons) 참고).

| 필드 | 안내 |
| --- | --- |
| **`documentNumber`** | PEP 번호, **숫자만**. 콜롬비아에서 **항상 15자리**. 공백·구두점 없이 15자리 전송. 예: `123456789012345`. |
| **`expeditionDate`** | 발급일 **`DD/MM/YYYY`**(예: `10/10/2024`). 필수. |

전체 비교: [신분증 유형 가이드](/verifik-ko/identity-validation/colombia/colombia-identity-documents-guide).

### 매개변수(Parameters)

| name | type | required | description |
| --- | --- | --- | --- |
| `documentNumber` | string | yes | PEP 번호, **숫자만**. **15자리**(콜롬비아 고정 길이). 예: `123456789012345`. |
| `expeditionDate` | string | yes | 발급일 **`DD/MM/YYYY`**(예: `10/10/2024`). 문서와 일치해야 합니다. |

### 요청(Request)

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/foreigner-id/pep", {
	params: { documentNumber: "123456789", expeditionDate: "10/10/2024" },
	headers: {
		Accept: "application/json",
		Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
	},
});
console.log(data);
```

### 참고(Notes)

- 날짜 형식은 **`DD/MM/YYYY`** 입니다.
- **`GET`** / **`POST`** 모두 동일 처리입니다.
- 원천 장애 시 **404** / **500** 이 날 수 있습니다.
