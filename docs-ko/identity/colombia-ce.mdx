### 엔드포인트

```
GET https://api.verifik.co/v2/co/foreigner-id/ce
```

**`documentNumber`** 와 발급일 **`expeditionDate`**(`DD/MM/YYYY`)로 조회합니다. 동일 필드로 **`POST`** JSON 본문도 지원합니다.

### 헤더(Headers)

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 서류 요건

**대상:** 콜롬비아에 합법 거주하는 외국인으로 *Cédula de Extranjería* (**CE**)를 보유한 경우입니다. 콜롬비아 국민(**`CC`**, `/v2/co/cedula`)이나 베네수엘라 **PPT**·이민 **PEP**에는 해당하지 않습니다.

| 필드 | 안내 |
| --- | --- |
| **`documentNumber`** | CE 번호, **숫자만**(공백·구두점 없음). 콜롬비아에서 CE는 보통 **6 또는 7자리**(가변 길이). 예: `712046`, `1156043`. |
| **`expeditionDate`** | 발급일 **`DD/MM/YYYY`** 형식(예: `15/06/2020`). 필수 — 잘못된 형식은 **409**가 자주 반환됩니다. |

전체 비교: [신분증 유형 가이드](/verifik-ko/identity-validation/colombia/colombia-identity-documents-guide).

### 매개변수(Parameters)

| name | type | required | description |
| --- | --- | --- | --- |
| `documentNumber` | string | yes | CE 번호, **숫자만**. 콜롬비아에서 일반적으로 **6–7자리**. 예: `712046`, `1156043`. |
| `expeditionDate` | string | yes | 발급일 **`DD/MM/YYYY`**(예: `15/06/2020`). 문서와 일치해야 합니다. |

### 요청(Request)

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/foreigner-id/ce", {
	params: { documentNumber: "987654321", expeditionDate: "15/06/2020" },
	headers: {
		Accept: "application/json",
		Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
	},
});
```

### 참고(Notes)

- 날짜 형식은 **`DD/MM/YYYY`** 입니다.
- **`GET`** / **`POST`** 모두 동일 처리입니다.
- 원천 장애 시 **404** / **500** 이 날 수 있습니다.
