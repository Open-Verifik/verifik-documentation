# 🇵🇪 페루 — 외국인 신분증(CE)

페루 신원 검증 서비스는 외국인 신분증 번호로 페루에 거주하는 개인의 신원을 확인할 수 있도록 합니다. 응답에는 대상자의 전체 성명과 함께 이민 상태, 국적, 거주 만료일, 외국인 신분증 상세 정보 등이 포함되며, 응답의 진위를 인증하는 서명도 함께 제공됩니다.

본 서비스는 페루 외국인 거주자를 검증하고 이민 상태와 문서 유효성을 확인할 때 유용합니다.

## 엔드포인트

```
GET https://api.verifik.co/v2/pe/foreigner-id/ce
```

### 헤더(Headers)

| 이름           | 값                  |
| -------------- | ------------------- |
| Content-Type   | `application/json`  |
| Authorization  | `Bearer <token>`    |

### 매개변수(Parameters)

| 이름             | 형식    | 필수 | 설명                                | 예시           |
| ---------------- | ------- | ---- | ----------------------------------- | -------------- |
| `documentNumber` | string  | **예** | 조회할 대상자의 문서 번호.        | `005015372`    |
| `dateOfBirth`    | string  | **예** | 문서 명의자의 출생일.             | `16/08/1993`   |

## 주요 기능

- **외국인 신분증 검증**: 페루의 외국인 신분증을 검증합니다.
- **이민 상태 점검**: 현재 이민 상태(예: HUMANITARIA)를 확인할 수 있습니다.
- **국적 정보**: 외국인 거주자의 국적 정보를 제공합니다.
- **문서 유효성**: 외국인 신분증 만료일과 직전 발급일을 확인합니다.
- **거주 상태**: 거주 만료일을 검증합니다.
- **개인 정보**: 전체 성명, 이름, 성, 출생일을 제공합니다.
- **문서 번호 검증**: 외국인 문서 번호를 검증합니다.
- **날짜 형식 검증**: 출생일의 올바른 형식(DD/MM/YYYY)을 보장합니다.
- **다양한 프로그래밍 언어 지원**: JavaScript, Python, PHP, Swift 예제를 제공합니다.
- **실시간 데이터**: 최신의 외국인 정보를 조회할 수 있습니다.
- **종합 오류 처리**: 다양한 시나리오에 대한 상세한 오류 응답을 제공합니다.
- **페루 이민청 연동**: 공식 페루 이민 데이터베이스에 직접 접근합니다.
- **외국인 문서 관리**: 외국인 신분증의 전체 라이프사이클을 추적합니다.
- **거주 컴플라이언스**: 거주 상태와 만료일을 모니터링합니다.
