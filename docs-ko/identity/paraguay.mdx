# 파라과이 신원 검증

Verifik 신원 검증 API는 **파라과이 정부 공식 데이터**를 사용해 시민을 인증합니다. **KYC**를 간소화하고 사기를 줄이며 규정 준수를 지원합니다.

사용자·직원·고객의 실제 신원을 **빠르고 안전하며 자동으로** 확인해야 하는 기업을 위해 설계되었습니다.

## 이 API가 검증하는 항목

공식 기록에 직접 연결하여 다음을 확인합니다.

-   **성명 및 CIC 번호**: *Cédula de Identidad Civil* (CIC) 지원.
-   **일치 여부**: 제공된 이름이 CIC 번호와 일치하는지 확인.

이를 통해 사칭 위험을 낮추고 디지털 온보딩 신뢰도를 높일 수 있습니다.

## API 참고

### Endpoint

```
https://api.verifik.co/v2/py/cic
```

### Headers

| Name          | Value              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parameters

| Name           | Type    | Required | Description                                    |
| -------------- | ------- | -------- | ---------------------------------------------- |
| `documentNumber` | string | Yes      | CIC 문서 번호. |

### Request

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/py/cic", {
  params: { documentNumber: "1234567" },
  headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});
console.log(data);
```

  </TabItem>

## 일반 사용 사례

-   **핀테크·은행**: 계좌 개설·대출 시 즉시 신원 확인.
-   **이커머스·배송**: 활성화 전 사용자·배송 기사 인증.
-   **채용**: 지원자 신분 증명 검증.
-   **보험·의료**: 가입·혜택 전 신원 확인.

## 공식 출처 및 신뢰성

**파라과이 정부 공식 출처**에 직접 연결하여 검증된 최신 정보를 제공합니다.

## 주요 이점

-   **자동 컴플라이언스**: 사용자 마찰 없이 KYC 강화.
-   **즉시 결과**: 실시간 디지털 온보딩에 적합.
-   **신뢰 데이터**: 공식 기록 기반.
-   **쉬운 연동**: REST API 및 SDK.

## 컴플라이언스 및 보안

HTTPS/TLS 1.3 암호화, 엄격한 개인정보 기준. **24/7** 가용 모니터링 및 역할 기반 접근 제어.

## Verifik 소개

Verifik는 라틴아메리카에서 신원 검증·컴플라이언스·사기 방지를 제공하는 선도 플랫폼입니다.
KYC·KYB·AML 및 바이오메트릭 검증 API로 **파라과이** 및 인근 국가의 공식 데이터 소스와 연결합니다.
