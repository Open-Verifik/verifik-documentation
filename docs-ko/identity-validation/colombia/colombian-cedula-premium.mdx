### 엔드포인트

```
GET https://api.verifik.co/v2/co/cedula/premium
```

`GET https://verifik.app/v2/co/cedula/premium` — 동일. **`POST`**(동일 필드) 지원.

**CC 전용** — `documentType`·발급일(`date`) 없이 **`documentNumber`**만 전달합니다(발급일은 내부에서 처리된 뒤 **cédula extra**와 같은 확장 흐름).

### 헤더

| 이름 | 값 |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 매개변수

| 이름 | 형식 | 필수 | 설명 |
| --- | --- | --- | --- |
| `documentNumber` | string | 예 | 숫자만(정규화 후). **5~10자**. |

### 요청

<Tabs>
  <TabItem value="get" label="GET">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/cedula/premium", {
	params: { documentNumber: "1234567890" },
	headers: {
		Accept: "application/json",
		Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
	},
});
console.log(data);
```

console.log(data);
