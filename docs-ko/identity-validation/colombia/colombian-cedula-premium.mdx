# 콜롬비아 — 국민 신분증 프리미엄 (CC)

**목적:** 콜롬비아 **CC(국민신분증)** 번호를 **공식 출처**에 대조하여 **정형화된 신원 레코드**를 반환합니다(KYC·AML·규정 절차용). **등록상 성명·생년월일·발급일시·발급지** 및, 출처가 제공하는 경우 **성별·생존 여부**를 포함하며, **서명 기반 인증** 정보가 응답에 포함됩니다. **제출은 번호만** — **발급일은 서버에서 자동으로 확정**되며(요청에 문서 유형·발급일 **미포함**), 상세 수준은 **cédula extra**와 **동일**합니다. **기본** `/v2/co/cedula` 엔드포인트보다 **크레딧**이 **높습니다**(연쇄 해상도 단계).

## API 참고

### 엔드포인트

```
GET https://api.verifik.co/v2/co/cedula/premium
```

**응답:** **`data`**(제출한 번호에 대해 출처가 반환한 신원 속성), **`signature`**(페이로드에 대한 인증 메타데이터), **`id`**(요청 식별자)를 반환합니다. **HTTP 404**는 본 엔드포인트 흐름에서 해당 번호에 대한 **완전한 신원 레코드를 구성할 수 없음**을 뜻합니다. **HTTP 409**는 상위 연동 전 **입력 유효성 검사**에 실패(예: `documentNumber` 길이)했음을 뜻합니다. 요청당 **크레딧**은 **기본** `/v2/co/cedula`보다 **높습니다**(연쇄 해상도). **CC**만 지원, 일반 **5~10**자리 숫자.

### 헤더

| 이름 | 값 |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 매개변수

| 이름 | 형식 | 필수 | 설명 |
| --- | --- | --- | --- |
| `documentNumber` | string | 예 | 콜롬비아 **CC**(국민신분증) 번호. **비숫자는 정규화로 제거**되며, **5~10**자 **숫자**로 유효성이 검사됩니다. `documentType`·발급일은 **불필요**(프리미엄). |

### 직접 프리미엄 vs 동적 가격

이 페이지는 **명시적 프리미엄 엔드포인트**(`/v2/co/cedula/premium`)를 설명합니다. 직접 호출하면 항상 **프리미엄 요금**이 적용됩니다.

**동적 가격**은 표준 [**기본 신분증 엔드포인트**](/verifik-ko/identity-validation/colombia/colombian-citizen#dynamic-pricing)(`/v2/co/cedula`)를 호출할 때 동적 쿼리가 자동으로 확장 검증 경로로 에스컬레이션되는 경우에 적용됩니다. 에스컬레이션이 **HTTP 200**을 반환하면 이 경로와 동일한 **프리미엄 등급**이 청구되며 표준 등급이 아닙니다. [SLA — 동적 가격(청구)](/verifik-ko/legal/service-level-agreement#viiia-동적-가격-청구)를 참조하세요.
