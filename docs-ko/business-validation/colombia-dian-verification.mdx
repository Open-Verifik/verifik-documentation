# 🇨🇴 콜롬비아 — 세금 정보 확인 (DIAN)

콜롬비아 국세청 **DIAN**(*Dirección de Impuestos y Aduanas Nacionales*)에 등록된 **납세자(NIT)** 정보를 조회해 등록 상태·세무 관련 정보를 확인합니다. 상세 응답 예시는 영문 문서 [Colombia DIAN Verification](https://docs.verifik.co/business-validation/colombia-dian-verification)를 참고하세요.

## API 참고

### 엔드포인트

```
GET https://api.verifik.co/v2/co/company/dian
```

동일 경로: `GET https://verifik.app/v2/co/company/dian`

### 헤더

| 이름 | 값 |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 매개변수

| 이름 | 형식 | 필수 | 설명 |
| --- | --- | --- | --- |
| `documentType` | string | 예 | 반드시 **`NIT`**. |
| `documentNumber` | string | 예 | NIT. 서버에서 **공백 제거**; 검증 자릿수 포함·미포함 가능. API 검증상 **최소 5자**. |

### 참고

- 요청은 **`GET` 쿼리**만 사용합니다(본문 없음). JSON 응답에는 **`Accept`** 헤더를 사용하세요.

### 요청

  <TabItem value="javascript" label="JavaScript">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/company/dian", {

	headers: {
		Accept: "application/json",
		Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
	},
});
console.log(data);
```
