# 🇨🇴 콜롬비아 — RUES 전체 기업 검증 (v3)

콜롬비아 **RUES**(*Registro Único Empresarial y Social*)에서 **NIT**와 **등록 구분(category)**으로 **전체(풀)** 기업 정보를 조회합니다. 요약 수준의 응답이 필요하면 **`GET /v3/co/rues`**를 사용하세요(영문: [Colombia RUES (v3)](https://docs.verifik.co/business-validation/colombia-rues-v3)).

## API 참고

### 엔드포인트

```
GET https://api.verifik.co/v3/co/rues-complete
```

동일 경로: `GET https://verifik.app/v3/co/rues-complete`

### 헤더

| 이름 | 값 |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 매개변수

| 이름 | 형식 | 필수 | 설명 |
| --- | --- | --- | --- |
| `documentType` | string | 예 | 반드시 **`NIT`**. |
| `documentNumber` | string | 예 | NIT(서버에서 비영숫자 문자 제거). |
| `category` | string | 아니오 | **`RM`**, **`PROP`**, **`RUNEOL`**, **`RNT`**, **`ESAL`**, **`ESOL`**, **`JUEGOS`**, **`EXTRANJERAS`**. 생략 시 **`RM`**. |

### 참고

- 이 경로는 위 매개변수만 허용합니다. **`municipality`**, **`chamber`**, **`page`**는 지원하지 않습니다.
- 상세한 응답 예시와 필드 설명은 영문 문서 [RUES Complete Business Validation (v3)](https://docs.verifik.co/business-validation/rues-complete-v3)를 참고하세요.
