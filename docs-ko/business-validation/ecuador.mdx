# 에콰도르 기업 검증

Verifik의 기업 검증 API는 정부 공식 데이터를 활용해 에콰도르 기업의 진위를 확인할 수 있도록 합니다. KYB(Know Your Business) 절차를 자동화하고 사기를 방지하며 규제 요구사항을 충족할 수 있도록 설계되었습니다.

기업, 파트너, 공급업체의 적법성을 빠르고 안전하게 자동으로 확인해야 하는 비즈니스를 위해 만들어졌습니다.

## 이 API는 무엇을 검증합니까?

당사의 API는 공식 기록과 직접 연결되어 다음을 검증합니다.

-   **상호 및 RUCEC**: *Registro Único de Contribuyentes del Ecuador*(RUCEC)를 지원합니다.
-   **기업 정보**: 상호, 상태, 납세자 유형, 체제, 주요 경제 활동, 회계·원천징수 여부, 납세자 일자, 법정 대리인을 반환합니다.
-   **회사 일치 여부**: 입력된 기업 정보가 RUCEC 번호와 일치하는지 확인합니다.

이러한 항목을 검증함으로써 사기 위험을 낮추고 컴플라이언스 워크플로를 지원할 수 있습니다.

## API 레퍼런스

### 엔드포인트

```
GET https://api.verifik.co/v3/ec/company
```

**`documentType`** `RUCEC`와 **`documentNumber`**(13자리, 구분자 없음)로 에콰도르 법인을 조회합니다. 경제 활동, 체제, 법정 대리인 등 등기 중심 필드를 반환하며 **KYB**, **공급업체 심사**, **컴플라이언스**에 활용할 수 있습니다.

### 헤더(Headers)

| 이름           | 값                  |
| -------------- | ------------------- |
| Content-Type   | `application/json`  |
| Authorization  | `Bearer <token>`    |

### 매개변수(Parameters)

| 이름             | 형식   | 필수 | 설명                                         | 예시            |
| ---------------- | ------ | ---- | -------------------------------------------- | --------------- |
| `documentType`   | string | 예   | 문서 종류. 허용 값: `RUCEC`.                 | `RUCEC`         |
| `documentNumber` | string | 예   | 공백·마침표 없는 13자리 RUCEC.               | `1793197955001` |

### Notes

- `documentNumber`는 구분자 없이 정확히 13자리여야 합니다.
- 샌드박스 데모 RUC: `1790008959001` (ACERO COMERCIAL ECUATORIANO S.A.).
- 라이브 예시: `1793197955001`, `1191795453001`.
- 레거시 `GET /v2/ec/company`는 더 간소한 응답 형태로 계속 제공됩니다.

## 일반적인 사용 사례

-   **B2B 온보딩**: 파트너 또는 공급업체 등록 시 기업을 즉시 검증합니다.
-   **금융 서비스**: 계좌 개설 또는 거래 처리 전에 기업을 인증합니다.
-   **컴플라이언스 및 실사**: KYC/KYB 워크플로의 일부로 기업 정보를 검증합니다.
-   **이커머스 및 마켓플레이스**: 플랫폼에서 판매를 허용하기 전에 기업의 적법성을 확인합니다.

## 공식 출처 및 신뢰성

에콰도르 정부 공식 출처와 직접 연결되어 검증되고 최신인 정보를 제공합니다.
모든 조회는 보안 및 규제 기준을 엄격히 준수하여 처리됩니다.

## 주요 이점

-   **자동화된 컴플라이언스**: 파트너에게 마찰을 더하지 않으면서 사기를 방지하도록 KYB 검사를 간소화합니다.
-   **즉시 결과**: 실시간 기업 온보딩에 적합하도록 수 초 내에 검증을 처리합니다.
-   **신뢰할 수 있는 데이터**: 공식 기록에서 직접 가져온 데이터를 활용합니다.
-   **쉬운 통합**: REST API로 손쉽게 연결하거나 호환 SDK를 사용합니다.

## 컴플라이언스 및 보안

데이터 안전을 최우선으로 합니다. Verifik는 고급 암호화(HTTPS/TLS 1.3)와 엄격한 개인정보 관리 기준을 사용해 기밀성을 보장합니다.
서비스는 가용성을 위해 24/7 모니터링되며, 역할 기반 접근 제어로 팀 접근을 안전하게 유지합니다.

## Verifik 소개

Verifik는 라틴아메리카 전역의 신원 검증, 컴플라이언스, 사기 방지를 위한 선도 플랫폼입니다.
당사 API는 KYC, KYB, AML, 생체 인증 프로세스를 자동화하며, 콜롬비아, 멕시코, 페루, 칠레, 에콰도르, 아르헨티나, 브라질 등 공식 데이터 소스와 비즈니스를 연결합니다.
