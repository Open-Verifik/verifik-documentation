# 파나마 기업 검증

**RUC**(*Registro Único de Contribuyente*)와 검증숫자(**`dv`**)로 등록 법인을 조회합니다. **KYB**·공급망 실사에 적합합니다.

## 이 API로 확인할 수 있는 것

- **RUC + dv**: 납세자 번호와 검증 숫자.
- **등기 정보**: 상호, 조직 형태, 상태, 대표자 등(데이터 제공 시).

## API 참조

### Endpoint

```
https://api.verifik.co/v2/pa/company
```

파나마 **RUC** 기반 기업 조회. **`documentType`**(`RUC`), **`documentNumber`**, **`dv`**를 전달합니다. 한 번의 HTTPS 호출로 **KYB**와 규정 준수 절차를 지원합니다.

### 헤더

| 이름 | 값 |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 매개변수

| 이름 | 유형 | 필수 | 설명 |
| --- | --- | --- | --- |
| `documentType` | string | 예 | `RUC` 고정. |
| `documentNumber` | string | 예 | RUC 번호(하이픈 포함 형식). |
| `dv` | string | 예 | 검증 숫자. |

### 요청

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/pa/company", {

  headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});
```
