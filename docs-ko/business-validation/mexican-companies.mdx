# 멕시코 기업 검증

Verifik API는 멕시코 정부 공식 데이터로 기업을 인증합니다. KYB를 간소화하고 사기를 줄이며 규정 준수를 돕습니다.

## 검증 내용

-   **상호 및 FME**(*Folio Mercantil Electrónico*)
-   **등록 정보**: 상호, 시, 등기
-   **일치 여부**: 정보가 FME와 일치하는지

## API 참조

### 엔드포인트

```
https://api.verifik.co/v2/mx/company
```

이 엔드포인트는 **FME**(Folio Mercantil Electrónico) 식별자로 멕시코 법인을 조회합니다. GET에 `documentType=FME`와 `documentNumber`를 사용하여 정부 상업등기 출처의 법적 상호명·도시 등 공식 등록 정보를 가져옵니다.

### 헤더

| 이름          | 값              |
| ------------- | ---------------- |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### 매개변수

| 이름 | 유형 | 필수 | 설명 |
| --- | --- | --- | --- |
| `documentType` | string | 예 | 허용 값: `FME`. |
| `documentNumber` | string | 예 | FME 식별자(예: `N-2021007300`). |

### 요청

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/mx/company", {
  params: { documentType: "FME", documentNumber: "N-2021007300" },
  headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});
console.log(data);
```
