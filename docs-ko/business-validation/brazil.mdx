# 브라질 기업 검증

Verifik 기업 검증 API는 공식 정부 데이터로 브라질 기업을 인증합니다. KYB 프로세스, 사기 방지, 규제 준수를 지원하도록 설계되었습니다.

## 이 API는 무엇을 검증하나요?

공식 기록과 연결하여 다음을 검증합니다.

- **상호 및 CNPJ 번호**: *Cadastro Nacional da Pessoa Jurídica*(CNPJ)를 지원합니다.
- **기업 정보**: 상호, 법적 성격, 개업일, 상태, 주소 등을 반환합니다.
- **경제 활동**: 주·부업종 코드와 설명을 제공합니다.
- **일치 여부**: 기업 정보가 CNPJ 번호와 일치하는지 확인합니다.

## API 레퍼런스

### 엔드포인트

```
https://api.verifik.co/v2/br/company
```

브라질 법인을 공식 **CNPJ** 등기와 대조합니다. `documentType=CNPJ`와 구분 기호 없는 14자리 `documentNumber`를 전달합니다. 일반적으로 등기 상태, 주소, 법적 성격, 주·부 경제 활동 등이 포함됩니다. **KYB**, **공급업체 실사**, **컴플라이언스**에 활용하세요.

### 헤더(Headers)

| 이름           | 값                  |
| -------------- | ------------------- |
| Accept         | `application/json`  |
| Authorization  | `Bearer <token>`    |

### 매개변수(Parameters)

| 이름 | 형식 | 필수 | 설명 |
| ---- | ---- | ---- | ---- |
| `documentType` | string | **예** | 문서 유형. 허용 값: `CNPJ`. |
| `documentNumber` | string | **예** | 구분 기호 없는 14자리 CNPJ. |
