# 🇨🇴 콜롬비아 — DIAN 전자 청구 (법적 청구자)

콜롬비아 국세청 **DIAN**에 등록된 **전자 청구(전자세금계산서, facturación electrónica)** 및 **법적 청구자(legal invoicer)** 정보를 **NIT**로 조회합니다. 일반 납세자 상태·의무는 **`GET /v2/co/company/dian`**([세금 정보 확인](https://docs.verifik.co/business-validation/colombia-dian-verification))를 사용하세요. 상세 응답 예시는 영문 [Colombia Legal Invoicer Verification](https://docs.verifik.co/business-validation/colombia-legal-invoicer-verification)를 참고하세요.

## API 참고

### 엔드포인트

```
GET https://api.verifik.co/v2/co/company/dian/invoicer
```

동일 경로: `GET https://verifik.app/v2/co/company/dian/invoicer`

### 헤더

| 이름 | 값 |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 매개변수

| 이름 | 형식 | 필수 | 설명 |
| --- | --- | --- | --- |
| `documentType` | string | 예 | 반드시 **`NIT`**. |
| `documentNumber` | string | 예 | NIT. 서버에서 **공백 제거**; 검증 자릿수 포함·미포함 가능. API 검증상 **최소 5자**. |

### 참고

- **`GET` 쿼리만** 사용합니다(본문 없음). upstream에서 청구자 **이메일**이 없으면 **404**가 날 수 있습니다.
- JSON 응답에는 **`Accept`** 헤더를 사용하세요.
