# 칠레 기업 검증(RUT)

Verifik 기업 검증 API는 공식 정부 데이터로 칠레 법인을 인증합니다. **KYB**를 간소화하고 사기 위험을 줄이며 규정 준수를 돕습니다.

파트너·공급사·거래처의 적법성을 빠르게 확인해야 하는 팀을 위해 설계했습니다.

**문서 유형·번호**로 상호, 업종·세부 활동, 허용 서비스 범위 등을 조회할 수 있습니다.

### Endpoint

```
https://api.verifik.co/v2/cl/company
```

**RUT**로 법인을 조회합니다. 상호, 경제활동, 공식 등기 정보를 **KYB**·실사에 활용하세요.

### Headers

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parameters

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `documentType` | string | Yes | `RUT` |
| `documentNumber` | string | Yes | 법인 RUT(통합 문서 형식) |

### Request

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/cl/company", {
  params: { documentType: "RUT", documentNumber: "212957739" },
  headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os, requests

url = "https://api.verifik.co/v2/cl/company"
headers = {"Accept": "application/json", "Authorization": f"Bearer {os.getenv('VERIFIK_TOKEN')}"}
params = {"documentType": "RUT", "documentNumber": "212957739"}
```

## 검증 범위
