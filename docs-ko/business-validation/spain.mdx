# 스페인 기업 검증

공식 출처로 스페인 기업을 인증합니다. KYB 프로세스를 지원합니다.

파트너·공급사·거래 상대 검증에 적합합니다.

## 이 API로 확인할 수 있는 것

공식 레코드를 조회합니다:

-   **기업 식별자**: CIF/NIF 및 관련 정보.
-   **기업 정보**: 상호, 상태(제공 시).
-   **일치 여부**: 식별자와 정보가 일치하는지.

## API 참조

### Endpoint

```
https://api.verifik.co/v2/es/company
```

`documentNumber`(기업 식별자)로 스페인 기업 등록 정보를 조회합니다.

### 헤더

| 이름 | 값 |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### 매개변수

| 이름 | 유형 | 필수 | 설명 |
| --- | --- | --- | --- |
| `documentNumber` | string | 예 | 기업 식별자(CIF/NIF). |
