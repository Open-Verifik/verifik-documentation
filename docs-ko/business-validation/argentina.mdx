# 아르헨티나 기업 검증

Verifik의 기업 검증 API는 공식 정부 데이터를 사용해 아르헨티나 기업을 인증할 수 있도록 합니다. KYB(Know Your Business) 절차를 자동화하고, 사기를 예방하며, 규제 요구사항을 손쉽게 충족할 수 있도록 설계되었습니다.

기업, 파트너, 벤더의 합법성을 빠르고 안전하게 자동으로 확인해야 하는 비즈니스를 위해 만들어졌습니다.

## 이 API는 무엇을 검증합니까?

당사 API는 공식 기록과 직접 연결되어 다음을 검증합니다.

-   **회사명 및 CUIT 번호**: *Código Único de Identificación Tributaria*(CUIT)를 지원합니다.
-   **기업 정보**: 회사명, 법적 형태, 계약일, 경제 활동 등을 반환합니다.
-   **경제 활동**: 주요 및 부차적 활동 코드와 설명을 제공합니다.
-   **기업 일치**: 입력된 기업 정보가 CUIT 번호와 일치하는지 확인합니다.

이를 검증하면 거래 상대 기업이 적법하게 등록되어 있음을 확인할 수 있어 사기 위험을 크게 줄이고 규정 준수를 보장할 수 있습니다.

## API 레퍼런스

### 엔드포인트

```
GET https://api.verifik.co/v2/ar/company
```

**`documentType`** `CUIT`와 **`documentNumber`**로 아르헨티나 법인을 조회합니다. **KYB**, **벤더 심사**, **컴플라이언스**를 지원하는 등록 필드를 반환합니다.

### 헤더

| 이름          | 값                 |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### 매개변수

| 이름             | 형식   | 필수 | 설명                                       |
| ---------------- | ------ | ---- | ------------------------------------------ |
| `documentType`   | string | 예 | 문서 유형. 허용 값: `CUIT`.            |
| `documentNumber` | string | 예 | 하이픈 없는 11자리 CUIT.               |

## 일반적인 사용 사례

-   **B2B 온보딩**: 파트너 또는 벤더 등록 시 기업을 즉시 검증합니다.
-   **금융 서비스**: 계좌 개설 또는 거래 처리 전에 기업을 인증합니다.
-   **컴플라이언스 및 실사**: KYC/KYB 워크플로의 일부로 기업 정보를 검증합니다.
-   **전자상거래 및 마켓플레이스**: 판매 허용 전에 기업의 합법성을 확인합니다.

## 공식 출처 및 신뢰성

아르헨티나 공식 정부 출처와 직접 연결하여 검증되고 최신 정보를 제공합니다.
모든 조회는 엄격한 보안 및 규제 기준에 따라 처리됩니다.

## 주요 이점

-   **자동화된 컴플라이언스**: 파트너 경험을 해치지 않고 KYB 검사를 간소화하여 사기를 예방합니다.
-   **즉시 결과**: 수 초 내에 검증을 처리하여 실시간 기업 온보딩에 적합합니다.
-   **신뢰할 수 있는 데이터**: 공식 기록에서 직접 가져온 데이터를 사용합니다.
-   **쉬운 통합**: REST API 또는 호환 SDK를 통해 쉽게 연결할 수 있습니다.

## 컴플라이언스 및 보안

데이터 안전을 최우선으로 합니다. Verifik은 고급 암호화(HTTPS/TLS 1.3)와 엄격한 개인정보 관리 기준을 사용하여 기밀성을 보장합니다.
서비스는 24/7 가용성을 모니터링하며, 팀 접근을 안전하게 유지하는 역할 기반 액세스 제어를 제공합니다.

## Verifik 소개

Verifik은 라틴 아메리카 전역에서 신원 확인, 컴플라이언스, 사기 방지를 위한 선도 플랫폼입니다.
당사 API는 KYC, KYB, AML 및 생체 인증 프로세스를 자동화하여 기업을 콜롬비아, 멕시코, 페루, 칠레, 우루과이, 아르헨티나 등의 공식 데이터 소스와 연결합니다.
