# 🇨🇴 콜롬비아 — SENA 증명서

본 엔드포인트는 콜롬비아 국민의 **SENA**(Servicio Nacional de Aprendizaje) **교육 및 인증 기록**을 반환합니다. SENA 공식 디지털 증명서 포털에서 가져옵니다. 각 행에는 프로그램명, 인증 유형, 발급일 및 가능한 경우 디지털 증명서 다운로드 URL이 포함됩니다.

**채용 심사**, **기술 검증**, **교육 컴플라이언스** 확인에 사용할 수 있습니다.

## 반환 데이터

- **`data`** 아래 **증명서 행 배열**(**`record`**, **`title`**, **`type`**, **`program`**, **`certificationDate`**, **`certificationSignature`**, **`certificateDownload`**)
- Verifik **서명**이 포함된 응답

## API 레퍼런스

### 엔드포인트

```
GET https://api.verifik.co/v2/co/sena/certificados
```

**GET**만 지원. **쿼리 매개변수**만 사용.

쿼리: **`documentType`**, **`documentNumber`**.

### 헤더(Headers)

| 이름 | 값 |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 매개변수(Parameters)

| name | type | required | description |
| --- | --- | --- | --- |
| `documentType` | string | yes | **`CC`**, **`TI`**, **`CE`**, **`PA`**, **`RC`**, **`PEP`** 중 하나. |
| `documentNumber` | string | yes | 공백·구두점 없이 입력하는 신분증 번호. |

### 요청(Request)

<Tabs>
  <TabItem value="javascript" label="JavaScript">

import axios from "axios";

## 기능

- **신분증 유형**: **CC**, **TI**, **CE**, **PA**, **RC**, **PEP**
- **GET** 쿼리 매개변수
- SENA 기록이 여러 개인 경우 **여러 행** 반환

## 사용 사례

- **기술 역량** 및 **직업 교육** 검증
- **고용** 실사 및 **하도급업체** 온보딩
- SENA 자격이 필요한 직무의 **교육 컴플라이언스**

### 참고

- SENA가 **증명서 없음**을 보고하면 **404**(*「No se ha encontrado certificados」*).
- **샌드박스**: `documentNumber` **`10000001`**–**`10000010`** 고정 샘플, **`90040401`** → **404**.
- 미들웨어는 현재 **CC**와 **NIT**만 검증. 카탈로그의 다른 유형은 SENA 포털 지원에 따름.
- 포털 **가용성** 및 **CAPTCHA**는 환경 의존. **5xx** 가능.
- 콜롬비아 법 및 정책에 따라 **개인·교육** 데이터로 처리하세요.
