# 투표

## 콜롬비아 시민 투표 정보 조회

### Endpoint

```
https://api.verifik.co/v2/co/registraduria/votacion
```

투표 정보 서비스를 통해 « Registraduría Nacional del Estado Civil »에서 콜롬비아 시민의 투표 세부 정보를 조회할 수 있습니다. 시민의 문서 번호를 제공하면 주소, 주(departamento), 시(municipality), 투표 테이블, 투표소 등의 정보에 접근할 수 있습니다.

이 서비스는 유권자 등록을 확인하고 투표 과정을 지원하는 데 유용합니다.

### Headers

| 이름          | 값                 |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Query Parameters

| 이름           | 타입   | 필수? | 설명                                              | 예시         |
| -------------- | ------ | ----- | ------------------------------------------------- | ------------ |
| documentNumber | String | 예    | 조회할 사람의 문서 번호(공백·점 없음).          | `123456789`  |

### Request

```javascript
import axios from 'axios';

const options = {
  method: 'GET',
  url: 'https://api.verifik.co/v2/co/registraduria/votacion',
  params: {documentNumber: '123456789'},
  headers: {
    Accept: 'application/json',
    Authorization: 'Bearer <your_token>'
  }
};

try {
  const { data } = await axios.request(options);
  console.log(data);
} catch (error) {
  console.error(error);
}
```

### Response

```json
{
  "success": true,
  "data": {
    "documentNumber": "123456789",
    "firstName": "María",
    "lastName": "González",
    "fullName": "María González",
    "address": "Calle 123 #45-67",
    "department": "Cundinamarca",
    "municipality": "Bogotá D.C.",
    "pollingTable": "123",
    "votingStation": "Colegio San José",
    "votingStationAddress": "Carrera 7 #32-16",
    "status": "active"
  }
}
```

### Error Responses

```json
{
  "success": false,
  "error": "Voter not found",
  "code": "VOTER_NOT_FOUND"
}
```

## 사용 사례

- **유권자 등록 확인**: 등록 상태 확인
- **선거일 지원**: 유권자가 투표소를 찾도록 도움
- **인구통계 분석**: 유권자 분포 분석
- **선거 모니터링**: 참여 및 등록 추적
- **시민 참여**: 민주적 과정과 유권자 교육 지원
