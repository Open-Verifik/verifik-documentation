# 인도 — EPIC (Voter ID) 투표 및 투표소 정보

이 엔드포인트는 유효한 **10자 EPIC**(*Elector's Photo Identity Card*)를 기준으로 유권자가 **어디에 배정되어 투표하는지** — 의회구, 지구, 섹션(part), 건물명 및 관련 위치 필드 — 를 반환합니다. 물류·지원·컴플라이언스를 위해 **투표소 맥락**이 필요할 때 [이름 조회 API](/verifik-ko/legal/epic-eci)를 보완합니다.

## 이 API는 무엇을 검증하나요?

API는 주어진 EPIC에 대한 선거 **위치 및 배정** 데이터를 반환합니다. 예:

- **선거구 및 섹션**: 의회구(AC) 식별자, 섹션 번호·이름, 해당 시 일련번호.
- **지리**: 출처가 노출하는 경우 지구, 주, 의회 연관 정보.
- **투표소**: EPIC에 대해 반환된 건물명, 주소, 방/부스 세부 정보.
- **EPIC 일치**: `documentNumber`에 완전한 유효 **10자** EPIC를 사용해야 합니다.

## API 레퍼런스

### Endpoint

```
https://api.verifik.co/v2/in/epic/votacion
```

`documentNumber`에 EPIC를 넣어 **GET** 요청을 보냅니다. 응답은 **투표 위치** 데이터에 맞춰져 있으며(`GET /v2/in/epic`의 이름 전용 프로필이 아님).

### Headers

| 이름          | 값                 |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parameters

| 이름             | 타입   | 필수     | 설명                                                                          | 예시        |
| ---------------- | ------ | -------- | ----------------------------------------------------------------------------- | ----------- |
| `documentNumber` | string | **예**   | EPIC / Voter ID. **정확히 10자**(영숫자, 발급 형식)여야 합니다.            | `IHM2796746` |

### Notes

- `documentNumber`는 **정확히 10자**여야 합니다. 잘못된 길이/형식은 `409`를 반환할 수 있습니다.
- 필드 존재 여부는 주와 ECI 데이터 가용성에 따라 달라질 수 있으며, 선택 문자열에 `null`이 나타날 수 있습니다.
- **유권자 이름과 친척 이름만** 필요하면 [**GET /v2/in/epic**](/verifik-ko/legal/epic-eci)를 사용하세요.

## 일반적인 사용 사례

- **유권자 안내 데스크**: EPIC에 맞는 투표소, AC, 건물로 시민을 안내.
- **물류 및 현장 운영**: 파견 또는 대면 지원 전 배정 사전 확인.
- 조회 시점에 EPIC가 매핑되는 선거 지리를 보여줘야 하는 **규제·감사 워크플로**.

## 공식 출처와 신뢰성

Verifik는 ECI 지향 선거 데이터를 정규화하여 단일 API 계약을 제공합니다. 결과는 조회 시점의 공식 등록부에 따라 달라집니다. 유효하지 않거나 현재 출처에서 반환되지 않는 EPIC에 대해 항상 `404`를 처리하세요.

## 주요 이점

- **종단 TLS**와 성공 시 Verifik `signature`가 있는 일관된 JSON 형태.
- **이름 조회 보완**: 이름과 위치가 모두 필요할 때 `/v2/in/epic`와 함께 KYC 스타일 EPIC 확인에 사용.

## 컴플라이언스 및 보안

`Authorization` 토큰을 보호하고 공개 분석에 원시 EPIC 값을 **절대 기록하지 마세요**. 선거 데이터 보존은 정책과 현지 법률 범위 내에서 유지하세요. Verifik는 전송 보안과 플랫폼 수준 모니터링을 적용합니다.
