### 엔드포인트

```
GET https://api.verifik.co/v2/co/rama/proceso
```

**`processNumber`** 를 **경로 마지막 세그먼트**로 붙입니다. 보통 [사법 절차](/verifik-ko/legal/colombian-legal-processes) 목록의 **`llaveProceso`** 입니다.

### 헤더(Headers)

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 매개변수(Parameters)

| name | type | required | description |
| --- | --- | --- | --- |
| `processNumber` | string | yes | 경로 파라미터. 최소 7자, 공백 제거. |

### 요청(Request)

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const processNumber = "18738473";

const { data } = await axios.get(
	`https://api.verifik.co/v2/co/rama/proceso/${processNumber}`,
	{
		headers: {
			Accept: "application/json",
			Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
		},
	},
);
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os, requests

process_number = "18738473"
url = f"https://api.verifik.co/v2/co/rama/proceso/{process_number}"
headers = {
    "Accept": "application/json",
    "Authorization": f"Bearer {os.getenv('VERIFIK_TOKEN')}",
}
r = requests.get(url, headers=headers)
print(r.json())
```

### 참고(Notes)

- **`processNumber`** 는 **경로**에 붙습니다(API Explorer UI는 `processNumber` 필드로 표시될 수 있음).
- **409** 는 누락·형식 오류(7자 미만 등)일 때 흔합니다.
