# FBI 배경 조회

## API 레퍼런스

### 엔드포인트

```
https://api.verifik.co/v2/fbi
```

`documentType`과 `documentNumber`, 또는 `fullName`을 사용해 FBI가 공개한 기록을 조회합니다. 일치하는 항목이 있을 경우 응답에는 신원 필드와 대상자에 대한 공개 FBI URL이 포함될 수 있습니다.

**참고:** 날짜(출생일 또는 발급일)는 `dd/mm/yyyy` 형식이어야 합니다.

### 헤더(Headers)

| 이름           | 값                  |
| -------------- | ------------------- |
| Content-Type   | `application/json`  |
| Authorization  | `Bearer <token>`    |

### 매개변수(Parameters)

:::info
지원되는 **문서 종류**의 전체 목록과 각 종류별 필수 매개변수는 [Document Types](/background-check/document-types) 가이드를 참고해 주세요.
:::

| 이름             | 형식    | 필수 | 설명                                                                                                       |
| ---------------- | ------- | ---- | ---------------------------------------------------------------------------------------------------------- |
| `documentType`   | string  | 아니오 | 조회할 문서 종류.                                                                                          |
| `documentNumber` | string  | 아니오 | 조회할 문서 번호. 공백이나 점 없이 입력합니다.                                                              |
| `fullName`       | string  | 아니오 | documentType과 documentNumber 대신 개인 또는 법인의 이름을 직접 전달할 수 있습니다.                          |

### 활용 사례

- **신원 검증:** FBI가 공개한 정보를 기준으로 문서를 검증합니다.
- **컴플라이언스:** 민감한 직위 또는 거래에 대한 검증을 지원합니다.
- **사기 예방:** 관계 형성 전에 신원을 확인합니다.

<TabItem value="javascript" label="JavaScript">

```javascript
import axios from "axios";

const options = {
  method: "GET",
  url: "https://api.verifik.co/v2/fbi",
  params: { documentType: "CC", documentNumber: "80251972" },
  headers: {
    Accept: "application/json",
    Authorization: "jwt <tu_token>",
  },
};

try {
  const { data } = await axios.request(options);
  console.log(data);
} catch (error) {
  console.error(error);
}
```
