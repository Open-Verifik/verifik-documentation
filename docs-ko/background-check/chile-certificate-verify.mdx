# Registro Civil 증명서 검증 (칠레)

## API 참조

### 엔드포인트

```
GET https://api.verifik.co/v2/cl/registro-civil/certificado
```

**folio**와 **검증 코드**(`Código Verificador`)를 사용하여 **Registro Civil (SRCEI)** 증명서를 확인합니다.

### 헤더 (Headers)

| 이름           | 값                 |
| -------------- | ------------------ |
| Accept         | `application/json` |
| Authorization  | `Bearer <token>`   |

### 매개변수

| 이름                | 형식   | 필수   | 설명                                           |
| ------------------- | ------ | ------ | ---------------------------------------------- |
| `folio`             | string | **예** | 증명서 folio(10–15자). 예: `50070080804`.       |
| `verificationCode`  | string | **예** | 검증 코드. 예: `2aacbb9a636a`.                  |

### 요청 (Request)

<Tabs>
<TabItem value="javascript" label="JavaScript">

```javascript
import axios from 'axios';

const { data } = await axios.get('https://api.verifik.co/v2/cl/registro-civil/certificado', {
  params: { folio: '50070080804', verificationCode: '2aacbb9a636a' },
  headers: { Accept: 'application/json', Authorization: 'Bearer <토큰>' },
});
console.log(data);
```

</TabItem>
<TabItem value="python" label="Python">

```python
import http.client

conn = http.client.HTTPSConnection("api.verifik.co")
headers = {"Accept": "application/json", "Authorization": "Bearer <토큰>"}
conn.request("GET", "/v2/cl/registro-civil/certificado?folio=50070080804&verificationCode=2aacbb9a636a", headers=headers)
print(conn.getresponse().read().decode("utf-8"))
```

</TabItem>
</Tabs>

### 참고

- `folio`: **10~15**자.
- **샌드박스 모드**: `folio=50070080804`, `verificationCode=2aacbb9a636a`; `folio=9004040100001` → **404**.
