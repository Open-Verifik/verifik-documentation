# 콜롬비아 — INPEC 수감 조회

## API 레퍼런스

### 엔드포인트

```
https://api.verifik.co/v2/co/inpec
```

INPEC 수감 조회 서비스는 콜롬비아 국립 교정·교도소 연구소(INPEC)가 관리하는 시설에 수감 중인 개인의 현재 신분을 확인할 수 있도록 합니다. 본 API 엔드포인트는 INPEC가 운영하는 모든 구금 시설에서 대상자가 현재 자유를 박탈당한 상태인지 검증하며, 법적 신분, 입소 형태, 수감 시설 정보, 교정 시스템 내 고유 식별 번호 등 상세 정보를 제공합니다.

이 서비스는 사법기관, 법 집행기관, 국가 통제 기관 및 권한 있는 조직이 법적, 보안 또는 실사 목적으로 콜롬비아 교정 시스템 내 개인의 현재 상태를 확인해야 하는 경우에 적합합니다.

### 헤더(Headers)

| 이름           | 값                  |
| -------------- | ------------------- |
| Content-Type   | `application/json`  |
| Authorization  | `Bearer <token>`    |

### 매개변수(Parameters)

:::info
지원되는 **문서 종류**의 전체 목록과 각 종류별 필수 매개변수는 [Document Types](/background-check/document-types) 가이드를 참고해 주세요.
:::

| 이름             | 형식    | 필수 | 설명                                                                       |
| ---------------- | ------- | ---- | -------------------------------------------------------------------------- |
| `documentType`   | string  | 예   | 문서 종류. 허용 값: CC, CE.                                                |
| `documentNumber` | string  | 예   | 조회 대상자의 문서 번호. 공백이나 점 없이 입력합니다.                      |
| `firstSurname`   | string  | 예   | 조회 대상자의 첫 번째 성(姓).                                              |

### 응답 필드

#### Data 객체

| 필드             | 형식    | 설명                                                          |
| ---------------- | ------- | ------------------------------------------------------------- |
| `documentNumber` | string  | 조회 대상자의 문서 번호                                       |
| `documentType`   | string  | 문서 종류 (CC 또는 CE)                                        |
| `firstSurname`   | string  | 조회 대상자의 첫 번째 성                                      |
| `legend`         | string  | 기록 발견 여부를 나타내는 메시지                              |
| `records`        | array   | 수감 기록 배열 (기록이 없으면 빈 배열)                        |

#### Records 객체 (기록 발견 시)

| 필드               | 형식    | 설명                                                                              |
| ------------------ | ------- | --------------------------------------------------------------------------------- |
| `admissionStatus`  | string  | 입소 상태 (예: 시설 내 수감 시 "INTRAMURAL")                                      |
| `gender`           | string  | 수감자의 성별 (MASCULINO/FEMENINO)                                                |
| `identification`   | string  | 식별 번호                                                                         |
| `legalStatus`      | string  | 법적 신분 (예: "CONDENADO" — 기결수, "SINDICADO" — 피의자)                        |
| `name`             | string  | 수감자의 전체 성명                                                                |
| `prison`           | string  | 수감되어 있는 교도소 또는 교정 복합 시설의 이름                                    |
| `uniqueNumber`     | string  | INPEC 시스템 내 고유 식별 번호                                                    |

### 주요 기능

- **실시간 수감 상태 조회**: INPEC 시설 내 현재 수감 상태를 실시간으로 확인할 수 있습니다.
- **종합 정보 제공**: 법적 신분, 입소 형태, 성별, 수감 시설 정보 등을 반환합니다.
- **법적 컴플라이언스**: 사법 및 보안 목적의 INPEC 공식 데이터를 제공합니다.
- **구조화된 응답**: 통합이 용이한 형식으로 데이터를 정리해 제공합니다.
- **다양한 프로그래밍 언어 지원**: JavaScript, Python, PHP, Swift 예제를 제공합니다.
- **오류 처리**: 다양한 시나리오에 대한 상세한 오류 응답을 제공합니다.

### 활용 사례

- **배경 조회**: 개인의 현재 수감 기록 보유 여부를 확인합니다.
- **법적 실사**: 사법 절차에 필요한 법적 신분을 확인합니다.
- **보안 심사**: 채용 또는 보안 등급 부여 시 정보 검증을 수행합니다.
- **가족 확인**: 친족이 구금 상태를 확인할 수 있도록 지원합니다.
- **컴플라이언스 검증**: 민감 직위에 대한 규제 준수를 보장합니다.
