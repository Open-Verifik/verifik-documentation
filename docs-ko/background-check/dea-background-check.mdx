# DEA 배경 조회

## API 레퍼런스

### 엔드포인트

```
https://api.verifik.co/v2/dea
```

`documentType`과 `documentNumber`, 또는 `fullName`을 사용해 미국 마약단속국(DEA) 감시 명단 대상으로 개인 또는 법인을 조회합니다. API는 일치 여부를 반환하며, 해당하는 경우 공식 DEA 기록의 URL도 함께 제공합니다.

**참고:** 날짜(출생일 또는 발급일)는 `dd/mm/yyyy` 형식이어야 합니다.

### 헤더(Headers)

| 이름           | 값                  |
| -------------- | ------------------- |
| Content-Type   | `application/json`  |
| Authorization  | `Bearer <token>`    |

### 매개변수(Parameters)

:::info
지원되는 **문서 종류**의 전체 목록과 각 종류별 필수 매개변수는 [Document Types](/background-check/document-types) 가이드를 참고해 주세요.
:::

| 이름             | 형식    | 필수 | 설명                                                                                                       |
| ---------------- | ------- | ---- | ---------------------------------------------------------------------------------------------------------- |
| `documentType`   | string  | 아니오 | 조회할 문서 종류.                                                                                          |
| `documentNumber` | string  | 아니오 | 조회할 문서 번호. 공백이나 점 없이 입력합니다.                                                              |
| `fullName`       | string  | 아니오 | documentType과 documentNumber 대신 개인 또는 법인의 이름을 직접 전달할 수 있습니다.                          |

### 활용 사례

- **규제 컴플라이언스:** 미국 연방 규정상 DEA 관련 위험 신호를 점검합니다.
- **사기 예방:** 통제 약물 단속 기록과 연관된 위험을 식별합니다.
- **KYC 및 AML:** 의료, 제약, 물류, 금융 분야의 컴플라이언스 프로그램을 지원합니다.
