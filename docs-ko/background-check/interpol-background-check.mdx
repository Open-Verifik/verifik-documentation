# 인터폴 배경 조회

## API 레퍼런스

### 엔드포인트

```
https://api.verifik.co/v2/interpol
```

본 엔드포인트는 자연인 또는 법인에 대한 인터폴 연계 형사 배경 정보를 `documentType`과 `documentNumber`, 또는 `fullName`만으로 조회합니다. 정상 응답에는 신원 필드가 포함되며, 기록이 존재할 경우 매치 상세 정보가 함께 반환됩니다. 본 서비스는 안전하고 권한 있는 법 집행 및 보안 용도로 사용되도록 설계되었습니다.

**참고:** 날짜(출생일 또는 발급일)는 `dd/mm/yyyy` 형식이어야 합니다.

### 헤더(Headers)

| 이름           | 값                  |
| -------------- | ------------------- |
| Content-Type   | `application/json`  |
| Authorization  | `Bearer <token>`    |

### 매개변수(Parameters)

:::info
지원되는 **문서 종류**의 전체 목록과 각 종류별 필수 매개변수는 [Document Types](/background-check/document-types) 가이드를 참고해 주세요.
:::

| 이름             | 형식    | 필수 | 설명                                                                                                       |
| ---------------- | ------- | ---- | ---------------------------------------------------------------------------------------------------------- |
| `documentType`   | string  | 아니오 | 조회할 문서 종류                                                                                           |
| `documentNumber` | string  | 아니오 | 조회할 문서 번호. 공백이나 점 없이 입력합니다.                                                              |
| `fullName`       | string  | 아니오 | documentType과 documentNumber 대신 개인 또는 법인의 이름을 직접 전달할 수 있습니다.                          |

### 활용 사례

- **형사 배경 검증:** 채용, 보안 등급 부여 또는 법적 컴플라이언스(국제 기록 포함)에 활용합니다.
- **법 집행:** 권한 있는 기관이 인터폴 데이터를 기반으로 검증을 수행합니다.

import Foundation

let headers = [

    print(httpResponse)
  }
})

dataTask.resume()
