# 콘트랄로리아 증명서

## API 레퍼런스

### 엔드포인트

```
https://api.verifik.co/v2/co/contraloria/certificado
```

본 서비스는 문서 종류와 번호를 입력하여 콜롬비아 총감사원(Contraloría General de Colombia)이 발급한 품행 증명서를 검증합니다. 응답에는 조회 일자와 함께 base64로 인코딩된 PDF 형식의 증명서가 포함됩니다. 콜롬비아 기업 또는 개인의 정상 신분 상태를 신속히 확인하는 데 적합한 서비스입니다.

### 헤더(Headers)

| 이름           | 값                  |
| -------------- | ------------------- |
| Accept         | `application/json`  |
| Authorization  | `Bearer <token>`    |

### 매개변수(Parameters)

:::info
지원되는 **문서 종류**의 전체 목록과 각 종류별 필수 매개변수는 [Document Types](/background-check/document-types) 가이드를 참고해 주세요.
:::

| 이름             | 형식    | 필수 | 설명                                                                       |
| ---------------- | ------- | ---- | -------------------------------------------------------------------------- |
| `documentType`   | string  | 예   | 문서 종류. 허용 값: CC, CE, PA, PEP.                                       |
| `documentNumber` | string  | 예   | 조회 대상자의 문서 번호. 공백이나 점 없이 입력합니다.                      |
