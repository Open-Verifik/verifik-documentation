# 콘트랄로리아 증명서

## API 참조

### Endpoint

```
https://api.verifik.co/v2/co/contraloria/certificado
```

이 서비스는 콜롬비아 총감사원(Contraloría General de Colombia)이 발급한 품행 증명서를 서류 유형과 번호를 입력해 검증하며, 조회 일자와 base64로 인코딩된 PDF 형식의 증명서를 반환합니다. 콜롬비아 개인 또는 기업의 상태를 빠르게 확인하기에 적합합니다.

### 헤더

| 이름          | 값                 |
| ------------- | ------------------ |
| Accept        | `application/json` |
| Authorization | `Bearer <token>`   |

### 매개변수

:::info
지원되는 **문서 유형** 및 필수 매개변수 전체 목록은 [문서 유형](/background-check/document-types) 가이드를 참조하세요.
:::

| 이름             | 유형   | 필수 | 설명 |
| ---------------- | ------ | ---- | ---- |
| `documentType`   | string | 예   | CC, CE, PA 또는 PEP. |
| `documentNumber` | string | 예   | 공백·마침표 없이. |
