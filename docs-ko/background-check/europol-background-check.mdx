# 유로폴 배경 조회

## API 레퍼런스

### 엔드포인트

```
https://api.verifik.co/v2/europol
```

`documentType`과 `documentNumber`, 또는 `fullName`만으로 유로폴 형사 기록 데이터를 실시간으로 조회합니다. 응답은 일치 여부를 나타내며, 일치 시 유로폴 직접 참조 URL이 포함될 수 있습니다.

**참고:** 날짜(출생일 또는 발급일)는 `dd/mm/yyyy` 형식이어야 합니다.

### 헤더(Headers)

| 이름           | 값                  |
| -------------- | ------------------- |
| Content-Type   | `application/json`  |
| Authorization  | `Bearer <token>`    |

### 매개변수(Parameters)

:::info
지원되는 **문서 종류**의 전체 목록과 각 종류별 필수 매개변수는 [Document Types](/background-check/document-types) 가이드를 참고해 주세요.
:::

| 이름             | 형식    | 필수 | 설명                                                                                                       |
| ---------------- | ------- | ---- | ---------------------------------------------------------------------------------------------------------- |
| `documentType`   | string  | 아니오 | 조회할 문서 종류.                                                                                          |
| `documentNumber` | string  | 아니오 | 조회할 문서 번호. 공백이나 점 없이 입력합니다.                                                              |
| `fullName`       | string  | 아니오 | documentType과 documentNumber 대신 개인 또는 법인의 이름을 직접 전달할 수 있습니다.                          |

### 활용 사례

- **사기 예방:** 민감한 거래 이전에 유로폴 형사 데이터베이스에 등재된 인물을 식별합니다.
- **신원 검증:** 제공된 개인정보의 정합성을 확인합니다.
- **컴플라이언스:** 은행, 금융, 부동산 분야의 KYC 및 AML 점검을 지원합니다.

import Foundation

let headers = [

let request = NSMutableURLRequest(url: NSURL(string: "https://api.verifik.co/v2/europol?fullName=Mateo%20Verifik")! as URL,

  if (error != nil) {

  } else {

    print(httpResponse)
  }
})

dataTask.resume()
