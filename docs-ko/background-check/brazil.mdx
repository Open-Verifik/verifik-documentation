# 배경 조회 (CPF)

## API 레퍼런스

### 엔드포인트

```
https://api.verifik.co/v2/br/background-check
```

**브라질 개인 배경 조회**를 **CPF**와 **출생일**로 수행합니다. 정상 응답에는 신원 관련 필드가 포함되며, 데이터 제공자가 제공할 경우 **base64로 인코딩된 PDF** 보고서가 함께 반환됩니다. **채용 검증**, **실사**, **컴플라이언스** 목적에 적합합니다. PDF와 메타데이터의 처리 및 보존 정책은 자체 법적 기준에 따라 관리하시기 바랍니다.

### 헤더(Headers)

| 이름           | 값                  |
| -------------- | ------------------- |
| Accept         | `application/json`  |
| Authorization  | `Bearer <token>`    |

### 매개변수(Parameters)

:::info
지원되는 **문서 종류**의 전체 목록과 각 종류별 필수 매개변수는 [Document Types](/background-check/document-types) 가이드를 참고해 주세요.
:::

| 이름             | 형식    | 필수 | 설명                                                              |
| ---------------- | ------- | ---- | ----------------------------------------------------------------- |
| `documentType`   | string  | 예   | 문서 종류. 허용 값: CPF.                                          |
| `documentNumber` | string  | 예   | 조회 대상자의 문서 번호.                                          |
| `dateOfBirth`    | string  | 예   | 조회 대상자의 출생일. 형식: dd/mm/yyyy.                           |

### 주요 기능

-   **CPF 검증**: 브라질 CPF 문서의 진위를 확인합니다.
-   **종합 배경 데이터**: 전체 이름, 문서 상세 정보, 인증 번호, PDF 보고서를 반환합니다.
-   **구조화된 응답**: 통합이 용이하도록 정리된 데이터 형식을 제공합니다.
-   **다양한 프로그래밍 언어 지원**: JavaScript, Python, PHP, Swift 예제를 제공합니다.
-   **오류 처리**: 다양한 시나리오에 대한 상세한 오류 응답을 제공합니다.
