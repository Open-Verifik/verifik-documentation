# 범죄 경력 증명서 (아르헨티나)

## API 참조

### 엔드포인트

```
GET https://api.verifik.co/v2/ar/dnrec/antecedentes
```

공식 PDF에 인쇄된 **신청 코드**(`Solicitud`)와 **보안 코드**(`Cod. Segur.`)를 사용하여 아르헨티나 **DNREC** 범죄 경력 증명서를 확인합니다.

### 헤더 (Headers)

| 이름           | 값                 |
| -------------- | ------------------ |
| Accept         | `application/json` |
| Authorization  | `Bearer <token>`   |

### 매개변수

| 이름            | 형식   | 필수   | 설명                                              |
| --------------- | ------ | ------ | ------------------------------------------------- |
| `requestCode`   | string | **예** | 신청 코드(`Solicitud`). 예: `02118499487`.         |
| `securityCode`  | string | **예** | 보안 코드(`Cod. Segur.`). 예: `371488F861`.      |

### 요청 (Request)

<Tabs>
<TabItem value="javascript" label="JavaScript">

```javascript
import axios from 'axios';

const options = {
  method: 'GET',
  url: 'https://api.verifik.co/v2/ar/dnrec/antecedentes',
  params: { requestCode: '02118499487', securityCode: '371488F861' },
  headers: { Accept: 'application/json', Authorization: 'Bearer <토큰>' },
};

const { data } = await axios.request(options);
console.log(data);
```

</TabItem>
<TabItem value="python" label="Python">

```python
import http.client

conn = http.client.HTTPSConnection("api.verifik.co")
headers = {"Accept": "application/json", "Authorization": "Bearer <토큰>"}
conn.request("GET", "/v2/ar/dnrec/antecedentes?requestCode=02118499487&securityCode=371488F861", headers=headers)
print(conn.getresponse().read().decode("utf-8"))
```

### 참고

- 두 매개변수 모두 필수이며, 누락 시 **409** `MissingParameter`가 반환됩니다.
- **샌드박스 모드**: `requestCode=02118499487`, `securityCode=371488F861`; `requestCode=90040401` → **404**.
