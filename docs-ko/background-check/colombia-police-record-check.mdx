# 경찰 범죄 경력 조회

## API 참조

### Endpoint

```
https://api.verifik.co/v2/co/policia/consultar
```

신분증 번호로 형사 기록 및 사법 당국 미결 사항을 확인합니다(헌법 제248조). 이름·성과 법적 상태 정보가 포함됩니다.

### 헤더

| 이름          | 값                 |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### 매개변수

:::info
지원되는 **문서 유형** 및 필수 매개변수 전체 목록은 [문서 유형](/background-check/document-types) 가이드를 참조하세요.
:::

| 이름             | 유형   | 필수 | 설명 |
| ---------------- | ------ | ---- | ---- |
| `documentType`   | string | 예   | CC 또는 CE. |
| `documentNumber` | string | 예   | 공백·마침표 없이. |
