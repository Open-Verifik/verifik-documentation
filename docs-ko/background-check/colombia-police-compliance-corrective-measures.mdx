# 시정 조치 준수 (RNMC)

## API 참조

### Endpoint

```
https://api.verifik.co/v2/co/policia/rnmc
```

시정 조치 미이행 여부를 확인합니다. 문서 유형·번호와 확인 기준 날짜를 전달하세요.

### 헤더

| 이름          | 값                 |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### 매개변수

:::info
지원되는 **문서 유형** 및 필수 매개변수 전체 목록은 [문서 유형](/background-check/document-types) 가이드를 참조하세요.
:::

| 이름             | 유형   | 필수 | 설명 |
| ---------------- | ------ | ---- | ---- |
| `documentType`   | string | 예   | CC 또는 CE. |
| `documentNumber` | string | 예   | 공백·마침표 없이. |
| `date`           | string | 예   | 문서 발급일 (DD/MM/YYYY). |
