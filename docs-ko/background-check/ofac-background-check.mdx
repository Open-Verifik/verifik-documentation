# OFAC 배경 조회

## API 레퍼런스

### 엔드포인트

```
https://api.verifik.co/v2/ofac
```

`documentType`과 `documentNumber`, 또는 `fullName`을 사용해 미국 재무부 해외자산통제실(OFAC)의 특별 지정 국적자(SDN) 및 차단 대상 명단에 개인 또는 법인이 포함되는지 확인합니다. 결과는 제재 컴플라이언스, 무역 검증, KYC/AML 워크플로우에 활용할 수 있습니다.

**참고:** 날짜(출생일 또는 발급일)는 `dd/mm/yyyy` 형식이어야 합니다.

### 헤더(Headers)

| 이름           | 값                  |
| -------------- | ------------------- |
| Content-Type   | `application/json`  |
| Authorization  | `Bearer <token>`    |

### 매개변수(Parameters)

:::info
지원되는 **문서 종류**의 전체 목록과 각 종류별 필수 매개변수는 [Document Types](/background-check/document-types) 가이드를 참고해 주세요.
:::

| 이름             | 형식    | 필수 | 설명                                                                                                       |
| ---------------- | ------- | ---- | ---------------------------------------------------------------------------------------------------------- |
| `documentType`   | string  | 아니오 | 조회할 문서 종류.                                                                                          |
| `documentNumber` | string  | 아니오 | 조회할 문서 번호. 공백이나 점 없이 입력합니다.                                                              |
| `fullName`       | string  | 아니오 | documentType과 documentNumber 대신 개인 또는 법인의 이름을 직접 전달할 수 있습니다.                          |

### 활용 사례

- **제재 컴플라이언스:** OFAC 집행 대상인 차단 거래 상대를 회피합니다.
- **위험 완화:** 미국 및 국제 무역 규정 위반 시 부과되는 제재를 줄입니다.
- **KYC 및 AML:** 은행, 보험, 부동산 분야의 고객 검증을 강화합니다.
