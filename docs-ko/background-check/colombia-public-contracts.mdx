# 공공 계약

## API 참조

### Endpoint

```
https://api.verifik.co/v2/co/contracts
```

문서 유형과 번호로 국가와 체결한 공공 계약 목록을 가져옵니다. 계약자, 금액, 기관, 기간 등이 포함됩니다.

투명성과 공공 계약 모니터링을 지원합니다.

### 헤더

| 이름          | 값                 |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### 매개변수

:::info
지원되는 **문서 유형** 및 필수 매개변수 전체 목록은 [문서 유형](/background-check/document-types) 가이드를 참조하세요.
:::

| 이름             | 유형   | 필수 | 설명 |
| ---------------- | ------ | ---- | ---- |
| `documentType`   | string | 예   | CC 또는 NIT. |
| `documentNumber` | string | 예   | 요청할 문서 번호. |
