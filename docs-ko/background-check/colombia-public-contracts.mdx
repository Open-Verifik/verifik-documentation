# 공공 계약

## API 레퍼런스

### 엔드포인트

```
https://api.verifik.co/v2/co/contracts
```

공공 계약 서비스는 문서 번호와 종류를 입력하여 개인 또는 법인이 체결한 공공 계약 목록을 조회할 수 있도록 합니다. 응답에는 계약자 정보, 계약 금액, 관련 기관, 계약 기간 등 각 계약의 주요 정보가 포함됩니다.

본 서비스는 콜롬비아 정부와의 공공 계약을 투명하게 모니터링할 수 있도록 지원합니다.

### 헤더(Headers)

| 이름           | 값                  |
| -------------- | ------------------- |
| Content-Type   | `application/json`  |
| Authorization  | `Bearer <token>`    |

### 매개변수(Parameters)

:::info
지원되는 **문서 종류**의 전체 목록과 각 종류별 필수 매개변수는 [Document Types](/background-check/document-types) 가이드를 참고해 주세요.
:::

| 이름             | 형식    | 필수 | 설명                                                                       |
| ---------------- | ------- | ---- | -------------------------------------------------------------------------- |
| `documentType`   | string  | 예   | 문서 종류. 허용 값: CC, NIT.                                               |
| `documentNumber` | string  | 예   | 조회할 문서 번호.                                                          |
