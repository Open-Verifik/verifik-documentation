# UN(ONU) 제재 명단 조회

## API 레퍼런스

### 엔드포인트

```
https://api.verifik.co/v2/onu
```

`documentType`과 `documentNumber`, 또는 `fullName`을 사용해 개인 또는 법인이 유엔 제재 명단에 포함되는지 조회합니다. 결과는 거래 상대 검증 시 글로벌 컴플라이언스, KYC, AML 워크플로우에 활용할 수 있습니다.

**참고:** 날짜(출생일 또는 발급일)는 `dd/mm/yyyy` 형식이어야 합니다.

### 헤더(Headers)

| 이름           | 값                  |
| -------------- | ------------------- |
| Content-Type   | `application/json`  |
| Authorization  | `Bearer <token>`    |

### 매개변수(Parameters)

:::info
지원되는 **문서 종류**의 전체 목록과 각 종류별 필수 매개변수는 [Document Types](/background-check/document-types) 가이드를 참고해 주세요.
:::

| 이름             | 형식    | 필수 | 설명                                                                                                       |
| ---------------- | ------- | ---- | ---------------------------------------------------------------------------------------------------------- |
| `documentType`   | string  | 아니오 | 조회할 문서 종류                                                                                           |
| `documentNumber` | string  | 아니오 | 조회할 문서 번호. 공백이나 점 없이 입력합니다.                                                              |
| `fullName`       | string  | 아니오 | documentType과 documentNumber 대신 개인 또는 법인의 이름을 직접 전달할 수 있습니다.                          |

### 활용 사례

- **국제 컴플라이언스:** 온보딩 또는 거래 전에 개인 및 법인을 UN 제재 명단과 대조해 검증합니다.
- **위험 평가:** 제재 대상자와의 거래로 인한 재무 및 평판 리스크를 줄입니다.
- **규제 프로그램:** 은행, 금융, 보험 분야의 KYC/AML 요구사항을 지원합니다.
