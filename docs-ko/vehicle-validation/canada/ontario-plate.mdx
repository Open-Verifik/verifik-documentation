# 🇨🇦 캐나다 — 온타리오 번호판 만료

**번호판 번호**와 **차량 허가 번호**(온타리오 등록 문서 / 그린 카드)로 온타리오 번호판 스티커 만료일을 확인합니다.

일반적인 응답 필드: `expired`, `plateNumber`, `permitNumber`, `expiryDate`, `success`. `expired`는 `expiryDate`가 현재 시각보다 이전이면 `true`입니다.

온타리오 정부 등록부에서 번호판 스티커 유효성을 빠르게 확인할 때 사용하세요. 차량 온보딩, 컴플라이언스, 차량 검증에 적합합니다.

## API 참조

### Endpoint

```
https://api.verifik.co/v2/ca/ontario/plate
```

**`plate`** 및 **`permitNumber`** 쿼리 파라미터와 함께 GET 요청을 보냅니다.

### 헤더

| 이름 | 값 |
| ---- | -- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 매개변수

| 이름 | 유형 | 필수 | 설명 | 예시 |
| ---- | ---- | ---- | ---- | ---- |
| `plate` | string | **예** | 온타리오 승용 번호판 **ABCD123** 형식(문자 4자 + 숫자 3자, 공백 없음). | `BKTR456` |
| `permitNumber` | string | **예** | 온타리오 등록 문서의 허가 번호 (8자). | `N9166001` |

### 요청

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/ca/ontario/plate", {
  params: { plate: "BKTR456", permitNumber: "N9166001" },
  headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});
console.log(data);
```
