# 스페인

## API 레퍼런스

### 엔드포인트

```
https://api.verifik.co/v2/es/vehicle
```

스페인 차량 번호판에 대한 차량 정보를 반환합니다. 일치하는 항목이 있을 경우 제조사, 모델, 연식, 차량 종류 등의 필드가 포함됩니다.

### 헤더(Headers)

| 이름           | 값                  |
| -------------- | ------------------- |
| Content-Type   | `application/json`  |
| Authorization  | `Bearer <token>`    |

### 매개변수(Parameters)

| 이름     | 형식    | 필수 | 설명                                  |
| -------- | ------- | ---- | ------------------------------------- |
| `plate`  | string  | 예   | 조회할 번호판(공백 없이).             |

### 요청(Request)

<Tabs>
  <TabItem value="javascript" label="JavaScript">

```javascript
import axios from "axios";

const options = {
  method: "GET",
  url: "https://api.verifik.co/v2/es/vehicle",
  params: { plate: "5684CCD" },
  headers: {
    Accept: "application/json",
    Authorization: "Bearer <your_token>",
  },
};

try {
  const { data } = await axios.request(options);
  console.log(data);
} catch (error) {
  console.error(error);
}
```

## 활용 사례

- **보험 및 인수 심사**: 번호판으로 차량 속성을 확인합니다.
- **마켓플레이스 및 물류**: 운영 차량 또는 등록 차량을 검증합니다.
- **컴플라이언스**: 스페인에 등록된 차량과 관련된 점검을 지원합니다.
