# 페루 — 차량 보험

## 엔드포인트

```
https://api.verifik.co/v2/pe/vehiculo/soat
```

본 서비스는 페루 차량의 보험 상태에 대한 정보를 제공합니다. 응답에는 보험사명, 보험 시작·종료일, 차량 번호판, 보험증권 번호, 사용 유형, 차량 등급, 보험 상태 등 상세 정보가 포함됩니다.

## 헤더(Headers)

| 이름           | 값                  |
| -------------- | ------------------- |
| Content-Type   | `application/json`  |
| Authorization  | `Bearer <token>`    |

## 매개변수(Parameters)

| 이름     | 형식    | 필수 | 설명                                            |
| -------- | ------- | ---- | ----------------------------------------------- |
| `plate`  | string  | 예   | 조회할 번호판. 공백이나 점 없이 입력합니다.     |

## 활용 사례

- **SOAT 검증**: 의무 교통 사고 보험의 현재 상태를 검증합니다.
- **보험 서비스**: 활성 보험증권과 만료일을 확인합니다.
- **교통 통제**: 의무 보험 가입 여부 준수를 검증합니다.
- **운수 기업**: 운영 차량 보험을 검증합니다.
- **컴플라이언스 검증**: 보험증권 유효성을 확인합니다.

## 주요 기능

- ✅ 차량 번호판으로 조회
- ✅ 종합 SOAT 보험증권 정보
- ✅ 보험사명
- ✅ 보험 시작·종료일
- ✅ 보험 상태(VIGENTE/INACTIVO)
- ✅ 고유 보험증권 코드
- ✅ SBS 보험사 코드
- ✅ 경찰 통제 일자
- ✅ Verifik.co로 인증된 응답
