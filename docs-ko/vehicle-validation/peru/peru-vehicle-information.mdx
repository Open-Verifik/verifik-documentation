# 페루 — 차량 정보

페루 차량 정보 서비스는 번호판(**`plate`**)에 대한 등록 차량 상세를 반환합니다: 제조사, 모델, 연식, 엔진·차대 식별자, 상태, 가능한 경우 소유자, 선택적 기술 사양서 필드.

### Endpoint

```
GET https://api.verifik.co/v3/pe/vehiculo/placa
```

**`plate`**(공백·점 없음)로 페루 차량을 조회합니다. 응답에는 가능한 경우 `brand`, `model`, `year`, `engineSerial`, `chasisSerial`, `vin`, `serial`, `color`, `status`, `owners`, `technicalSheet`(`engine`, `performance`, `dimensions`)가 포함될 수 있습니다. **검증**, **보험**, **차량 관리**, **컴플라이언스** 워크플로에 사용하세요.

### Headers

| Name          | Value              |
| ------------- | ------------------ |
| Accept | `application/json` |
| Authorization | `Bearer <token>`   |

### Parameters

| Name    | Type   | Required | Description | Example |
| ------- | ------ | -------- | ----------- | ------- |
| `plate` | string | **Yes**  | 조회할 번호판(공백·점 없음). | `ABC123` |

### Request

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from 'axios';

const { data } = await axios.get('https://api.verifik.co/v3/pe/vehiculo/placa', {
  params: { plate: 'ABC123' },
  headers: {
    Accept: 'application/json',
    Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
  },
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os
import requests

url = "https://api.verifik.co/v3/pe/vehiculo/placa"
headers = {"Accept": "application/json", "Authorization": f"Bearer {os.getenv('VERIFIK_TOKEN')}"}
params = {"plate": "ABC123"}
response = requests.get(url, headers=headers, params=params)
print(response.json())
```

### Notes

- 일치하는 기술 데이터가 없을 때 `technicalSheet`는 `null`이거나 부분적으로만 채워질 수 있습니다.
- `ABC10001`–`ABC10010`과 같은 샌드박스 번호판은 안정적인 데모 프로필을 반환합니다.
