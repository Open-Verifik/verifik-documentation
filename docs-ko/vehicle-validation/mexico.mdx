# 멕시코

### 엔드포인트

```
https://api.verifik.co/v2/mx/vehiculo/placa
```

차량 정보 서비스는 멕시코에서 번호판을 기준으로 제조사, 모델, 연식, VIN 등 차량 상세 정보를 제공합니다. GET 요청의 `plate` 쿼리 매개변수에 번호판을 넣으면, 일치할 경우 등록 관련 필드가 응답에 포함됩니다.

### 헤더

| 이름          | 값              |
| ------------- | ---------------- |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### 매개변수

| 이름  | 유형   | 필수 | 설명              | 예시     |
| ----- | ------ | ---- | ----------------- | -------- |
| plate | String | 예   | 조회할 차량 번호판. | `ABC1234` |

### 요청

```javascript
import axios from 'axios';

const options = {
  method: 'GET',
  url: 'https://api.verifik.co/v2/mx/vehiculo/placa',
  params: {plate: 'ABC1234'},
  headers: {
    Accept: 'application/json',
    Authorization: 'Bearer <your_token>'
  }
};

try {
  const { data } = await axios.request(options);
  console.log(data);
} catch (error) {
  console.error(error);
}
```

### 응답

```json
{
  "success": true,
  "data": {
    "plate": "ABC1234",
    "make": "Nissan",
    "model": "Sentra",
    "year": "2021",
    "engine": "2.0L",
    "transmission": "CVT",
    "fuelType": "Gasoline",
    "vin": "1N4AL3AP8JC123456",
    "registrationDate": "10/08/2021",
    "status": "active",
    "owner": "Carlos Rodríguez",
    "state": "Jalisco",

```

### 오류 응답

```json
{
  "success": false,
  "error": "Vehicle not found",
  "code": "VEHICLE_NOT_FOUND"
}
```

## 활용 사례

- **보험**: 보험 심사용 차량 정보 확인
- **딜러**: 구매 전 차량 정보 검증
- **플릿**: 법인 차량 관리
- **법 집행**: 등록·소유 확인
- **실사**: 거래 전 정보 검증
