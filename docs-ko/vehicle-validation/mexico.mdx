# 멕시코

### 엔드포인트

```
https://api.verifik.co/v2/mx/vehiculo/placa
```

차량 정보 서비스는 번호판을 기반으로 멕시코 내 차량의 상세 정보를 제공합니다. 본 서비스는 차량의 제조사, 모델, 연식, VIN 및 기타 관련 상세 정보를 반환합니다. 번호판을 GET 요청의 `plate` 쿼리 매개변수로 전달하면, 일치하는 항목이 있을 경우 등록 관련 필드가 응답에 포함됩니다.

### 헤더(Headers)

| 이름           | 값                  |
| -------------- | ------------------- |
| Content-Type   | `application/json`  |
| Authorization  | `Bearer <token>`    |

### 매개변수(Parameters)

| 이름   | 형식   | 필수 | 설명                          | 예시      |
| ------ | ------ | ---- | ----------------------------- | --------- |
| plate  | String | 예   | 조회할 차량 번호판.           | `ABC1234` |

### 요청(Request)

```javascript
import axios from 'axios';

const options = {
  method: 'GET',
  url: 'https://api.verifik.co/v2/mx/vehiculo/placa',
  params: {plate: 'ABC1234'},
  headers: {
    Accept: 'application/json',
    Authorization: 'Bearer <your_token>'
  }
};

try {
  const { data } = await axios.request(options);
  console.log(data);
} catch (error) {
  console.error(error);
}
```

### 응답(Response)

```json
{
  "success": true,
  "data": {
    "plate": "ABC1234",
    "make": "Nissan",
    "model": "Sentra",
    "year": "2021",
    "engine": "2.0L",
    "transmission": "CVT",
    "fuelType": "Gasoline",
    "vin": "1N4AL3AP8JC123456",
    "registrationDate": "10/08/2021",
    "status": "active",
    "owner": "Carlos Rodríguez",

```

### 오류 응답

```json
{
  "success": false,
  "error": "Vehicle not found",
  "code": "VEHICLE_NOT_FOUND"
}
```

## 활용 사례

- **보험 검증**: 보험 목적의 차량 상세 정보를 검증합니다.
- **자동차 딜러**: 매매 전 차량 정보를 검증합니다.
- **차량 운영 관리**: 사내 차량 정보를 추적합니다.
- **법 집행**: 차량 등록 및 소유 관계를 검증합니다.
- **실사**: 거래 목적의 차량 정보를 검증합니다.
