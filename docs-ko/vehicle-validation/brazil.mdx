# 🇧🇷 브라질 — 차량 정보

## 차량 정보 조회

### 엔드포인트

```
GET https://api.verifik.co/v2/br/vehicle
```

브라질 **번호판**이 주어지면 **Verifik**는 단일 요청으로 구조화된 차량 데이터를 반환하여 실물 자산을 검증·정규화된 속성에 연결할 수 있습니다. **차량 운영·모빌리티 온보딩**, **보험 견적·사고 처리**, **담보 대출·담보 심사**, 번호판이 주 식별자인 **마켓플레이스·딜러 점검** 등을 대상으로 합니다.

Verifik API는 해당 번호판에 대한 **정규화된 차량 레코드**를 반환합니다. **번호판**, **제조사**, **모델**, **모델 연식** 및 **제조 연식**, **차대(VIN)**, **엔진**, **색상**, **차체 유형**, **연료**, **변속기**, **도어 수**, **제조사 또는 공장**, 차량에 적용되는 경우 **FIPE 코드**, 해당 등록에 **불이행 또는 제한 지표**가 있으면 그 지표 등을 포함합니다. 계속 진행하기 전에 실제 차량이 공식 정보와 일치하는지 확인하는 데 사용합니다.

### 헤더(Headers)

| 이름           | 값                  |
| -------------- | ------------------- |
| Accept         | `application/json`  |
| Authorization  | `Bearer <token>`    |

### 매개변수(Parameters)

| 이름     | 형식    | 필수 | 설명 |
| -------- | ------- | ---- | ---- |
| `plate`  | string  | **예** | 조회할 번호판(영문·숫자; 대소문자는 정규화됨). 예: `PZY7H82`. |

### 응답 필드(Response fields)

| 필드 | 형식 | 설명 |
| ---- | ---- | ---- |
| `bodyType` | string | 등록 기관에서 반환한 차량 유형 라벨입니다. |
| `brand` | string | 차량 제조사/브랜드입니다. |
| `chassis` | string | 등록 기관에서 반환한 차대번호 또는 VIN입니다. |
| `color` | string | 등록된 차량 색상입니다. |
| `country` | string | 차량 기록의 국가입니다. |
| `denatranWarning` | string | `Sem Restrição` 같은 Denatran 또는 등록 상태 알림입니다. |
| `doors` | string | 등록 소스에서 반환한 승차 정원입니다. |
| `engine` | string | 등록상의 배기량 또는 엔진 참조값입니다. |
| `factory` | string | 원산지 또는 제조 지역입니다. |
| `fipeCodes` | string[] | 사용 가능한 경우 차량과 관련된 FIPE 참조 코드입니다. |
| `fuelType` | string | 등록된 연료 유형입니다. |
| `irregularitiesCount` | number | 발견된 활성 제한 또는 이상 항목 수입니다. |
| `irregularityCode` | string | 활성 제한 라벨을 하나의 문자열로 합친 값이며, 제한이 없으면 비어 있습니다. |
| `manufacturer` | string | 등록 기관에서 반환한 제조사명입니다. |
| `model` | string | 등록된 모델 설명입니다. |
| `modelYear` | string | 모델 연도입니다. |
| `plate` | string | 조회 결과로 반환된 정규화된 번호판입니다. |
| `transmission` | string | 사용 가능한 경우 변속기 값이며, 등록 소스 기록은 빈 문자열을 반환할 수 있습니다. |
| `vehicle` | string | 등록상의 차량 카테고리입니다. |
| `version` | string | 사용 가능한 경우 가장 적합한 FIPE 모델/버전 매칭입니다. |
| `yearOfManufacture` | string | 제조 연도입니다. |

### 참고

- 쿼리 문자열에서 **필수**인 것은 **`plate`** 뿐입니다.
- 등록 소스 기록은 **`transmission`** 값을 빈 문자열로 반환할 수 있습니다.
- **`doors`**는 등록 소스의 승차 정원을 의미합니다.
- FIPE 매칭이 없으면 **`fipeCodes`**는 빈 배열일 수 있습니다.
- 기본 소스에서 데이터를 찾지 못해 fallback 제공자를 사용하는 경우 필드 범위가 조금 달라질 수 있습니다.
- 일부 계정에서는 **번호판별 확장 기술 시트** 제품이 활성화됩니다. 해당 기능이 켜져 있으면 AppFeature 카탈로그에 안내된 URL을 사용하세요.
