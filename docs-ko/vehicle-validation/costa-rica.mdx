# 🇨🇷 코스타리카 — 차량 정보

본 서비스는 **번호판**을 사용해 코스타리카에 등록된 차량의 상세 정보를 반환합니다. 일반적으로 현재 소유자, 차량 사양(배기량, 재정 평가액, 중량, 정원, 출력, 구동 방식), **위반** 이력, 출처가 노출하는 경우 **이전 소유자** 정보가 포함됩니다.

차량 운영 점검, 보험, 컴플라이언스 워크플로우에 활용할 수 있습니다.

## API 레퍼런스

### 엔드포인트

```
https://api.verifik.co/v2/cr/vehicle
```

**`plate`** 쿼리 매개변수(공백이나 구두점 없음)와 함께 GET 요청을 전송합니다. 정상 응답에는 구조화된 차량 데이터, 소유자 필드, 그리고 가용 시 이력 배열이 포함됩니다.

### 헤더(Headers)

| 이름           | 값                  |
| -------------- | ------------------- |
| Accept         | `application/json`  |
| Authorization  | `Bearer <token>`    |

### 매개변수(Parameters)

| 이름     | 형식    | 필수 | 설명                                              | 예시      |
| -------- | ------- | ---- | ------------------------------------------------- | --------- |
| `plate`  | string  | **예** | 조회할 번호판. 공백이나 점 없이 입력합니다.    | `ABC123`  |

## 주요 기능

- **차량 정보 조회**: 코스타리카 차량 등기소에서 종합 차량 상세 정보를 조회합니다.
- **번호판 검증**: 코스타리카에 등록된 차량 번호판을 검증합니다.
- **현재 소유자 정보**: 현재 차량 소유자 상세 정보를 조회합니다.
- **차량 사양**: 배기량, 재정 평가액, 중량, 정원, 출력, 구동 방식을 제공합니다.
- **위반 이력**: 교통 위반 및 침해의 전체 이력을 조회합니다.
- **소유 이력**: 모든 이전 차량 소유자의 시간순 목록을 확인합니다.
- **사법 기관**: 위반을 처리한 사법 기관 정보를 조회합니다.
- **통고 정보**: 위반에 대한 통고 번호와 요약 번호를 조회합니다.
- **다양한 프로그래밍 언어 지원**: JavaScript, Python, PHP, Swift 예제를 제공합니다.
- **실시간 데이터**: 최신의 차량 정보를 조회할 수 있습니다.
- **종합 오류 처리**: 다양한 시나리오에 대한 상세한 오류 응답을 제공합니다.
- **코스타리카 등기소 연동**: 공식 코스타리카 차량 데이터베이스에 직접 접근합니다.
