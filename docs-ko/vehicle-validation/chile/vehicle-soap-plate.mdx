# 🇨🇱 칠레 — 차량 SOAP (번호판)

**번호판만**으로 칠레 **SOAP**(*Seguro Obligatorio de Accidentes Personales*)를 조회합니다. Boostr로 SOAP를 조회하고, 증권 번호가 있으면 AACH로 보강하며, SOAP가 없으면 Carvuk에서 차량 속성을 반환합니다.

소스가 응답할 때 **중첩** 페이로드:

- **`plate`** — 칠레 번호판(Patente), **4–6**자 영숫자
- **`message`** — SOAP 발견 시 빈 값; 활성 SOAP 없이 차량만 있으면 `No_active_SOAP_certificate_found_for_this_plate`
- **`soap`** — 발견 시: **`status`**, **`policyNumber`**, **`insuranceCompany`**, **`validFrom`**, **`validTo`**, **`folioNumber`**, **`ownerName`**, **`ownerRut`**, **`premium`**
- **`vehicle`** — **`brand`**, **`model`**, **`year`**, **`type`**, **`engineNumber`**
- Verifik **서명** 응답

번호판만 알려진 **보험 컴플라이언스**, **플릿 온보딩**, **클레임**에 사용합니다.

### 엔드포인트

```
GET v2/cl/vehicle-soap-plate
```

**번호판**(`plate`)만으로 조회합니다. **`soap`**의 **status**, **insuranceCompany**, **ownerName**, **validFrom**, **validTo** 및 **`vehicle`**의 **brand**, **model**, **year** 등이 포함됩니다.

### 헤더

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 매개변수

| Name | Type | Required | Description | Example |
| --- | --- | --- | --- | --- |
| `plate` | string | **Yes** | 칠레 번호판(Patente), 4–6자 영숫자. | `HTRT86` |

### 참고

- **활성 SOAP 없음**은 **HTTP 200**과 빈 **`soap`**, **`message: No_active_SOAP_certificate_found_for_this_plate`** — 404가 아닙니다.
- **Live 번호판:** `CLTK27`, `HTRT86`, `RRWX51`, `THHV64`, `CSXZ52`, `CSXZ57`는 활성 SOAP; `CSXZ27`는 SOAP 없이 차량만.
- **샌드박스:** **`BBCC12`**–**`BBCC21`**, **`HTRT86`**는 안정적 중첩 프로필; **`ERR404`** → **404**.
- 번호판+증권 조회는 [차량 SOAP (번호판+증권)](/verifik-ko/vehicle-validation/chile/vehicle-soap) 참조.
- 레지스트리 **가용성**은 환경에 따라 다르며 **409** 타임아웃 가능.
