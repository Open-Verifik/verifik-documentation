# 🇨🇱 칠레 - 차량 기술 검사

칠레 차량의 **기술 검사 / 점검** 데이터가 필요할 때 이 endpoint를 사용하세요. 번호판으로 차량 정보, 검사소, 증명서 번호, 만료일, 검사 상태, 그리고 가능한 경우 기술 사양서 필드를 조회합니다.

관련 칠레 차량 endpoint:

- **차량 정보 (`v2/cl/vehicle`)**: 제조사, 모델, 소유자 RUT, 엔진, 차대, 벌금, 도난 여부 등 일반 등록 데이터.
- **차량 기술 검사 (`v3/cl/vehicle`)**: 기술 검사 상태, 이력, 가능한 경우 기술 사양서 데이터.
- **차량 SOAP (`v2/cl/vehicle-soap`)**: 번호판과 증권 번호로 의무 신체 상해 보험 증명서 조회.

### Endpoint

```text
GET https://api.verifik.co/v3/cl/vehicle
```

번호판에 대한 칠레 기술 검사 데이터를 반환합니다. 차량 속성, 검사 이력, 가능한 경우 선택적 `technicalSheet`(`engine`, `performance`, `dimensions`)를 포함합니다. 차량 준수, 보험, 청구, 온보딩 워크플로에 사용하세요.

### Headers

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parameters

| Name | Type | Required | Description | Example |
| --- | --- | --- | --- | --- |
| `plate` | string | **Yes** | v3에서 지원하는 칠레 번호판 형식입니다. 예: 네 글자와 두 숫자, 또는 두 글자와 네 숫자. | `XH6640` |

### Request

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v3/cl/vehicle", {
	params: { plate: "XH6640" },
	headers: {
		Accept: "application/json",
		Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
	},
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os, requests

url = "https://api.verifik.co/v3/cl/vehicle"

r = requests.get(url, headers=headers, params=params)
print(r.json())
```

### Notes

- 일치하는 기술 데이터가 없을 때 `technicalSheet`는 `null`이거나 부분적으로만 채워질 수 있습니다.
- `XH6640`, `FHDJ31`, `DCCH18`은 안정적인 기술 검사 프로필을 반환합니다.
- `BB985`는 v3 번호판 형식과 맞지 않아 `409 Invalid_plate_format`을 반환합니다.
