# 🇨🇱 칠레 - 차량 기술 검사

이 endpoint는 칠레 차량의 **기술 검사 / 검사 이력** 을 번호판으로 조회합니다. 일반 차량 등록 정보는 `v2/cl/vehicle`, SOAP 의무 보험 증명서는 `v2/cl/vehicle-soap`을 사용하세요.

### Endpoint

```text
GET https://api.verifik.co/v3/cl/vehicle
```

### 파라미터

| 이름 | 타입 | 필수 | 설명 | 예시 |
| --- | --- | --- | --- | --- |
| `plate` | string | **예** | v3에서 지원하는 칠레 번호판 형식입니다. 네 글자와 두 숫자 또는 두 글자와 네 숫자입니다. | `XH6640` |

### 요청 예시

```javascript
const { data } = await axios.get("https://api.verifik.co/v3/cl/vehicle", {
	params: { plate: "XH6640" },
	headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});
```

### 오류

- `404 NotFound`: 레코드를 찾을 수 없습니다.
- `409 Invalid_plate_format`: 번호판 형식이 올바르지 않습니다.
- `409 timeout_data_source`: 외부 데이터 소스가 응답하지 않았습니다.

### Notes

`XH6640`, `FHDJ31`, `DCCH18`은 고정 프로필을 반환합니다. `BB985`는 `409 Invalid_plate_format`을 반환합니다.
