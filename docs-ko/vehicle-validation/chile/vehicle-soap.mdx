# 🇨🇱 칠레 — 차량 SOAP

칠레 등록 차량의 **SOAP**(*Seguro Obligatorio de Accidentes Personales*, 의무 개인상해 보험) 조회는 차량의 필수 인명사고 보험을 검증합니다. **번호판**과 **보험증권 번호**로 AACH 레지스트리에서 보험사, 소유자, 유효기간 및 차량 속성을 조회합니다.

## 차량 SOAP 조회

### 엔드포인트

```
GET https://api.verifik.co/v2/cl/vehicle-soap
```

**번호판**(`plate`)과 **보험증권 번호**(`policyNumber`; 서비스는 AACH 레지스트리에서 마지막 네 자리를 사용)로 조회합니다. 일반 필드: **status**, **insuranceCompany**, **ownerName**, **ownerRut**, **validFrom**, **validTo**, **brand**, **model**, **manufactureYear**, **engineNumber**, **premium** — **보험 컴플라이언스**, **플릿 온보딩**, **클레임** 워크플로에 사용합니다.

### 헤더

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 매개변수

| Name | Type | Required | Description | Example |
| --- | --- | --- | --- | --- |
| `plate` | string | **Yes** | 칠레 번호판(Patente), 4–6자 영숫자. | `HTRT86` |
| `policyNumber` | string | **Yes** | 보험증권 번호; AACH SOAP 조회에 마지막 네 자리 사용. | `94596506` |

### 참고

- **샌드박스 모드:** **`BBCC12`**–**`BBCC21`** 번호판은 고정 SOAP 프로필 반환(`BBCC15` = **VENCIDO**); **`HTRT86`** + **`policyNumber=94596506`** 데모 프로필; **`ERR404`** → **404**; 매핑되지 않은 번호판은 임의 **200**.
- **샌드박스 예:** `BBCC12`/`94590001`(세단, MAPFRE), `BBCC18`/`94590007`(픽업, CHUBB), `HTRT86`/`94596506`(TOYOTA COROLLA 데모).
- 칠레 **일반 차량** 조회는 [🇨🇱 칠레 — 차량](/verifik-ko/vehicle-validation/chile) 참조.
- 레지스트리 **가용성**은 환경에 따라 다르며, 장애 시 **409** 타임아웃이 발생할 수 있습니다.
