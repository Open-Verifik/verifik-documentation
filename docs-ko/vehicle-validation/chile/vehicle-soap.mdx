# 🇨🇱 칠레 — 차량 SOAP

칠레 등록 차량의 **SOAP**(의무 개인상해 보험) 증명서를 **번호판**과 **보험증권 번호**(AACH 레지스트리)로 조회합니다.

## 차량 SOAP 조회

### 엔드포인트

```
GET https://api.verifik.co/v2/cl/vehicle-soap
```

### 헤더

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 매개변수

| name | type | required | description |
| --- | --- | --- | --- |
| `plate` | string | yes | 칠레 번호판(4–6자). |
| `policyNumber` | string | yes | 보험증권 번호(마지막 4자리 사용). |

### 요청

<Tabs>
  <TabItem value="javascript" label="JavaScript">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/cl/vehicle-soap", {
	params: { plate: "HTRT86", policyNumber: "94596506" },
	headers: {
		Accept: "application/json",
		Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
	},
});
console.log(data);
```

</Tabs>
