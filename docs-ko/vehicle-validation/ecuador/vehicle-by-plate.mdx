# 🇪🇨 에콰도르 — 차량 검증

이 서비스는 번호판(**`plate`**)으로 에콰도르에 등록된 차량 정보를 반환합니다: 제조사, 모델, 연식, 차종, VIN, 엔진 번호, 검사 메타데이터, 가능한 경우 기술 사양서 필드.

### Endpoint

```
GET https://api.verifik.co/v3/ec/vehiculo/placa
```

**`plate`**(공백·점 없음)로 에콰도르 차량을 조회합니다. 응답에는 가능한 경우 `brand`, `model`, `year`, `vin`, `engineNumber`, `vehicleClass`, `vehicleType`, `color`, `serviceType`, `revision`, `technicalSheet`(`engine`, `performance`, `dimensions`)가 포함될 수 있습니다. **검증**, **보험**, **차량 관리**, **컴플라이언스** 워크플로에 사용하세요.

### Headers

| Name          | Value              |
| ------------- | ------------------ |
| Accept | `application/json` |
| Authorization | `Bearer <token>`   |

### Parameters

| Name | Type | Required | Description | Example |
| ---- | ---- | -------- | ----------- | ------- |
| `plate` | string | **Yes** | 조회할 번호판(공백·점 없음). | `PBB3239` |

### Request

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from 'axios';

const { data } = await axios.get('https://api.verifik.co/v3/ec/vehiculo/placa', {
  params: { plate: 'PBB3239' },
  headers: {
    Accept: 'application/json',
    Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
  },
});
console.log(data);
```

```python
import os

print(response.json())
```

### Notes

- 일치하는 기술 데이터가 없을 때 `technicalSheet`는 `null`이거나 부분적으로만 채워질 수 있습니다.
