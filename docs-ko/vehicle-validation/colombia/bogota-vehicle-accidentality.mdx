### 엔드포인트

```
GET https://api.verifik.co/v2/co/bogota/vehicle/accidentality
```

보고타에 등록된 차량의 사고 이력을 조회합니다. 쿼리 **`plate`** 에 번호판을 넣습니다. 응답의 `accident` 배열에 일자, 서식 번호(`form`), 심각도(`seriousness`)가 포함됩니다.

### 헤더(Headers)

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 매개변수(Parameters)

| name | type | required | description |
| --- | --- | --- | --- |
| `plate` | string | yes | 조회할 번호판. 5–6자. 공백·마침표 없음. |

### 요청(Request)

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/bogota/vehicle/accidentality", {
  params: { plate: "ABC123" },
  headers: { 
    Accept: "application/json", 
    Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` 
  },
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os, requests

url = "https://api.verifik.co/v2/co/bogota/vehicle/accidentality"
headers = {"Accept": "application/json", "Authorization": f"Bearer {os.getenv('VERIFIK_TOKEN')}"}
params = {"plate": "ABC123"}
r = requests.get(url, headers=headers, params=params)
print(r.json())
```

### 참고(Notes)

- **SOLO DAÑOS** 는 재산 피해만 있음(부상 없음)을 뜻합니다.
- **`form`** 은 공식 사고 보고서 참조 번호입니다.
- **`date`** 는 스페인어 형식(요일 포함)일 수 있습니다.
