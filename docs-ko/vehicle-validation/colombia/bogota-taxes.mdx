### 엔드포인트

```
GET https://api.verifik.co/v2/co/bogota/vehicle/tax
```

보고타에 등록된 차량의 세금 의무 정보를 조회합니다. 소유자 **`documentType`**, **`documentNumber`** 와 차량 **`plate`** 를 **쿼리 문자열**로 보냅니다. 응답에는 차량 세부 정보와 과세 연도별 의무 상태가 포함됩니다.

### 헤더(Headers)

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 매개변수(Parameters)

| name | type | required | description |
| --- | --- | --- | --- |
| `plate` | string | yes | 조회할 번호판. 5–6자. 공백·마침표 없음. |
| `documentNumber` | string | yes | 조회할 신분/사업자 번호. 공백·마침표 없음. |
| `documentType` | string | yes | 증명 유형. 허용: `CC`, `NIT`. |

### 요청(Request)

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/bogota/vehicle/tax", {
  params: { 
    plate: "ABC123", 
    documentType: "CC", 
    documentNumber: "123456789" 
  },
  headers: { 
    Accept: "application/json", 
    Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` 
  },
});
console.log(data);
```

  <TabItem value="python" label="Python">

```python
import os, requests

url = "https://api.verifik.co/v2/co/bogota/vehicle/tax"

r = requests.get(url, headers=headers, params=params)
print(r.json())
```

### 참고(Notes)

- **`CC`** 는 콜롬비아 시민 신분증, **`NIT`** 은 납세자(사업자) 식별번호입니다.
- **`obligationStatus`** 가 `Moroso` 이면 미납 세금이 있습니다.
