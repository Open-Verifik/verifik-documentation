### 엔드포인트

```
GET https://api.verifik.co/v2/co/runt/vehiculo
```

**번호판**과 명의자 **`documentType`**·**`documentNumber`** 로 **RUNT** 에 등록된 차량의 **일반 등록 정보·기본 특성**을 조회합니다. **`/v2/co/runt/vehicle-by-plate`** 의 전체 이력 보고서보다 가벼운 용도입니다. 필요 시 **`audit`** 쿼리(문자열 불리언)를 지원합니다.

### 헤더(Headers)

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 매개변수(Parameters)

| name | type | required | description |
| --- | --- | --- | --- |
| `documentType` | string | yes | 명의자 문서 유형. **`CC`**, **`CE`**, **`PA`**, **`NIT`** 중 하나. |
| `documentNumber` | string | yes | 문서 번호(공백·구두점 없음). |
| `plate` | string | yes | 조회할 번호판(API 검증 **5–9**자). 서버에서 대문자로 정규화됩니다. |

### 요청(Request)

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/runt/vehiculo", {
	params: {
		documentType: "CC",
		documentNumber: "123456789",
		plate: "ABC123",
	},
	headers: {
		Accept: "application/json",
		Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
	},
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os, requests

url = "https://api.verifik.co/v2/co/runt/vehiculo"
headers = {"Accept": "application/json", "Authorization": f"Bearer {os.getenv('VERIFIK_TOKEN')}"}
params = {"documentType": "CC", "documentNumber": "123456789", "plate": "ABC123"}
r = requests.get(url, headers=headers, params=params)
print(r.json())
```
