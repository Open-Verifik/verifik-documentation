### 엔드포인트

```
https://api.verifik.co/v2/co/simit/resoluciones
```

SIMIT 결의 서비스는 콜롬비아 내 특정 개인 또는 기업에 대한 교통 위반 결의의 상세 정보를 조회할 수 있도록 합니다. 응답에는 위반자의 이름, 결의 상태, 결의 일자, 미납 금액 등의 데이터가 포함됩니다.

### 헤더(Headers)

| 이름           | 값                  |
| -------------- | ------------------- |
| Accept         | `application/json`  |
| Authorization  | `Bearer <token>`    |

### 매개변수(Parameters)

| 이름             | 형식    | 필수 | 설명                                                          |
| ---------------- | ------- | ---- | ------------------------------------------------------------- |
| `documentType`   | string  | 예   | 문서 종류. 허용 값: `CC`, `PA`, `CE`, `TI`, `RC`.             |
| `documentNumber` | string  | 예   | 조회할 문서 번호. 공백이나 점 없이 입력합니다.                |

### 요청(Request)

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/simit/resoluciones", {
  params: { 
    documentType: "CC", 
    documentNumber: "123456789" 
  },
  headers: { 
    Accept: "application/json", 
    Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` 
  },
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os, requests

url = "https://api.verifik.co/v2/co/simit/resoluciones"
```

### 비고

- SIMIT 결의는 교통 위반과 관련된 공식 결정을 의미합니다.
- "Pendiente de pago" 상태는 해당 결의가 미납 상태임을 나타냅니다.
- 결의 번호는 특정 행정 결정을 추적하는 데 사용됩니다.
- 금액은 콜롬비아 페소(COP)로 제공됩니다.
