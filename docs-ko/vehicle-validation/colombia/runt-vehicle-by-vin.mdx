### 엔드포인트

```
GET https://api.verifik.co/v2/co/runt/vehicle-by-vin
```

**`vin`** 을 **쿼리**로 보내 콜롬비아 **RUNT** 에 등록된 차량의 **전체 이력·기술 보고서**를 조회합니다. 번호판이 없거나 현장 점검 시 차대 번호로 본차 여부를 확인할 때 유용합니다.

### 헤더(Headers)

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 매개변수(Parameters)

| name | type | required | description |
| --- | --- | --- | --- |
| `vin` | string | yes | 조회할 17자리 VIN(공백 없음, RUNT 등록 형식). |

### 요청(Request)

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/runt/vehicle-by-vin", {
	params: { vin: "JTEBH9FJ7DK091425" },
	headers: {
		Accept: "application/json",
		Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
	},
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os, requests

url = "https://api.verifik.co/v2/co/runt/vehicle-by-vin"
headers = {"Accept": "application/json", "Authorization": f"Bearer {os.getenv('VERIFIK_TOKEN')}"}
params = {"vin": "JTEBH9FJ7DK091425"}
r = requests.get(url, headers=headers, params=params)
print(r.json())
```

### 참고(Notes)

- **VIN** 은 차량당 부여된 **17자리** 식별 번호입니다.
- 응답에는 일반 정보·SOAT·검사 이력 등 **RUNT** 에서 제공되는 필드가 포함될 수 있습니다(데이터에 따라 다름).
- **`ACTIVO`** 는 등록이 유효한 상태의 예시입니다.
