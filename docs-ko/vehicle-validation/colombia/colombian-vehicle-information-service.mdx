### 엔드포인트

```
GET https://api.verifik.co/v2/co/runt/vehicle-by-plate
```

소유자 **`documentType`**·**`documentNumber`** 와 **`plate`** 를 **쿼리**로 보내 **RUNT** 의 **전체 이력·기술 보고서**를 조회합니다(SOAT, RC, 검사·등록 신청 등, 제공 시).

### 헤더(Headers)

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 매개변수(Parameters)

| name | type | required | description |
| --- | --- | --- | --- |
| `documentType` | string | yes | 소유자 신분증 유형. `CC`, `CE`, `PA`, `NIT` 중 하나. |
| `documentNumber` | string | yes | 소유자 문서 번호(공백·마침표 없음). |
| `plate` | string | yes | 조회할 번호판(보통 5–9자). |

### 요청(Request)

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/runt/vehicle-by-plate", {
	params: {
		documentType: "CC",
		documentNumber: "123456789",
		plate: "ABC123",
	},
	headers: {
		Accept: "application/json",
		Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
	},
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os, requests

url = "https://api.verifik.co/v2/co/runt/vehicle-by-plate"
headers = {"Accept": "application/json", "Authorization": f"Bearer {os.getenv('VERIFIK_TOKEN')}"}
params = {"documentType": "CC", "documentNumber": "123456789", "plate": "ABC123"}
r = requests.get(url, headers=headers, params=params)
print(r.json())
```

### 참고(Notes)

- RUNT는 기술 사양·법적 상태를 포함한 넓은 범위의 정보를 반환합니다.
- **SOAT** 는 콜롬비아 의무 교통사고 보험입니다.
- **`VIGENTE`** 는 보험이 유효한 경우의 예시입니다.
- **`ACTIVO`** 는 등록이 유효한 상태의 예시입니다.
