### 엔드포인트

```
https://api.verifik.co/v2/co/simit/comparendos
```

교통 위반 통고 서비스는 콜롬비아의 특정 차량에 연결된 교통 위반 통고에 대한 상세 정보를 제공합니다. 차량의 번호판을 입력하면 통고 상태, 일자, 위치, 위반 종류 등 각 통고에 대한 종합 정보를 조회할 수 있습니다.

### 헤더(Headers)

| 이름           | 값                  |
| -------------- | ------------------- |
| Accept         | `application/json`  |
| Authorization  | `Bearer <token>`    |

### 매개변수(Parameters)

| 이름             | 형식    | 필수 | 설명                                                          |
| ---------------- | ------- | ---- | ------------------------------------------------------------- |
| `documentType`   | string  | 예   | 문서 종류. 허용 값: `CC`, `PA`, `CE`, `TI`, `RC`.             |
| `documentNumber` | string  | 예   | 조회할 문서 번호. 공백이나 점 없이 입력합니다.                |

### 요청(Request)

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/simit/comparendos", {
  params: { 
    documentType: "CC", 
    documentNumber: "123456789" 
  },
  headers: { 
    Accept: "application/json", 
    Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` 
  },
});
```

### 비고

- SIMIT은 콜롬비아의 통합 교통 위반 시스템입니다.
- "Comparendo"는 교통 위반 통고를 의미하는 스페인어 용어입니다.
- 과태료는 콜롬비아 페소(COP)로 제공됩니다.
- 상태 "Pendiente"는 통고가 미납 상태임을 의미합니다.
- 사진 단속 위반은 `fotodeteccion: true` 로 표시됩니다.
