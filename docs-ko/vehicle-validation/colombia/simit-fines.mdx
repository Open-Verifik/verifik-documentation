### 엔드포인트

```
https://api.verifik.co/v2/co/simit/comparendos
```

교통 단속 서비스는 콜롬비아에서 특정 차량과 연결된 단속 정보를 자세히 제공합니다. 차량 번호판을 입력하면 각 단속의 상태, 날짜, 장소, 위반 유형을 확인할 수 있습니다.

### 헤더

| 이름 | 값 |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 매개변수

| 이름 | 유형 | 필수 | 설명 |
| --- | --- | --- | --- |
| `documentType` | string | 예 | 문서 유형. 유효 값: `CC`, `PA`, `CE`, `TI`, `RC`. |
| `documentNumber` | string | 예 | 조회할 문서 번호(공백·마침표 없음). |

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/simit/comparendos", {
  params: { 
    documentType: "CC", 
    documentNumber: "123456789" 
  },
  headers: { 
    Accept: "application/json", 
    Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` 
  },
});
console.log(data);
```

### 참고

- SIMIT는 콜롬비아의 통합 교통 위반 관리 시스템입니다.
- « Comparendo »는 스페인어로 교통 단속을 뜻합니다.
- 금액 단위는 콜롬비아 페소(COP)입니다.
- 상태 « Pendiente »는 단속 요금이 미납임을 나타냅니다.
- 사진 단속은 `fotodeteccion: true`로 표시됩니다.
