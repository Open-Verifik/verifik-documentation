### 엔드포인트

```
GET https://api.verifik.co/v2/co/simit/suspensiones
```

**SIMIT** 에서 운전면허 **정지 또는 취소** 등 신분과 연관된 제재 정보를 조회합니다(데이터 제공 시 관련 결의·일자 포함). **`documentType`** 과 **`documentNumber`** 로 명의자를 지정합니다.

### 헤더(Headers)

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 매개변수(Parameters)

| name | type | required | description |
| --- | --- | --- | --- |
| `documentType` | string | yes | 명의자 문서 유형. 보통 `CC`, `PA`, `CE`, `TI`, `RC`, `NIT` 등. |
| `documentNumber` | string | yes | 조회할 문서 번호(공백·구두점 없음). 길이 초과 시 `409` 가 날 수 있습니다. |

### 요청(Request)

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/simit/suspensiones", {
	params: { documentType: "CC", documentNumber: "123456789" },
	headers: {
		Accept: "application/json",
		Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
	},
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os, requests

url = "https://api.verifik.co/v2/co/simit/suspensiones"
headers = {"Accept": "application/json", "Authorization": f"Bearer {os.getenv('VERIFIK_TOKEN')}"}
params = {"documentType": "CC", "documentNumber": "123456789"}
r = requests.get(url, headers=headers, params=params)
print(r.json())
```

### 참고(Notes)

- **SIMIT** 은 콜롬비아 전국 교통 위반·과태료 통합 정보 체계입니다.
- `data` 필드명과 배열 구조는 **상위 SIMIT** 응답을 따르며, 예시는 대표적인 형태입니다.
- **`suspensiones`** 가 비어 있으면 해당 조회에서 정지 내역이 없었을 수 있습니다(비즈니스 규칙과 함께 해석).
