### 엔드포인트

```
https://api.verifik.co/v2/co/medellin/vehicle/fines
```

**콜롬비아 메델린** 번호판에 대한 **지역 교통 벌금 및 comparendo**를 반환합니다. 위반 내역, 금액, 위반자 정보(가능한 경우)를 포함합니다. **시정 준수**, **차량 관리**, 메델린 mobility 기록에 한정된 **실사**에 사용하세요.

주요 필드: **`plate`**, **`informacionComparendo`**, **`valorTotalAdeudado`** 및 관련 배열.

### 헤더

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 매개변수

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `plate` | string | Yes | 번호판(5–8자). 공백이나 점 없이. |

### 참고

- **Sandbox:** 번호판 **`ABC10001`**–**`ABC10010`**은 고정 fixture; **`ERR404`** → **404**.
- **메델린만** 해당; 전국 SIMIT는 [SIMIT - 벌금](/vehicle-validation/colombia/simit-fines) 참조.
- 시청 포털 **가용성**은 변동될 수 있으며, 장애 시 **5xx** 오류가 발생할 수 있습니다.
