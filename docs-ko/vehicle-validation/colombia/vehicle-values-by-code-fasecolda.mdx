### 엔드포인트

```
GET https://api.verifik.co/v2/co/fasecolda/values-by-code
```

Fasecolda **`codeFasecolda`**(카탈로그 코드)를 **쿼리**로 보내 차량 속성과 상업 가격 정보를 조회합니다.

### 헤더(Headers)

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 매개변수(Parameters)

| name | type | required | description |
| --- | --- | --- | --- |
| `codeFasecolda` | string | yes | Fasecolda 차량 구성 코드. |

### 요청(Request)

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/fasecolda/values-by-code", {
  params: { codeFasecolda: "08032036" },
  headers: {
    Accept: "application/json",
    Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
  },
});
console.log(data);
```

  <TabItem value="python" label="Python">

```python
import os, requests

url = "https://api.verifik.co/v2/co/fasecolda/values-by-code"

print(r.json())
```

### 참고(Notes)

- Fasecolda 코드는 모델·트림 조합을 가리키는 식별자입니다.
- 번호판 조회(values-by-plate)와 동일한 수준의 상세 정보가 반환될 수 있습니다.
- 금액은 콜롬비아 페소(COP)입니다.
- **`bcpp`** 는 기준 상업 기본 가격입니다.
- **`estado: "USADO"`** 는 중고 가치 구분 예시입니다.
