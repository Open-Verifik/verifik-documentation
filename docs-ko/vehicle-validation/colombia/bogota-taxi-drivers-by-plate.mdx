### 엔드포인트

```
GET https://api.verifik.co/v2/co/bogota/taxi-drivers/plate
```

보고타 등록 택시의 운전자 정보와 운행카드 내용을 반환합니다. 차량 **번호판(plate)** 은 URL 마지막 **경로 세그먼트**로 전달합니다（예: `…/plate/WMN482`）. 쿼리 문자열이 아닙니다. 5–6자이며 공백이나 마침표를 넣지 않습니다.

### 헤더(Headers)

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 매개변수(Parameters)

| name | type | required | description |
| --- | --- | --- | --- |
| `plate` | string | yes | `/plate/` 다음 경로 세그먼트. 5–6자. 공백·마침표 없음. |

### 요청(Request)

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/bogota/taxi-drivers/plate/WMN482", {
  headers: {
    Accept: "application/json",
    Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
  },
});
console.log(data);
```
