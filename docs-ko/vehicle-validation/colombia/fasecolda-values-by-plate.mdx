### 엔드포인트

```
GET https://api.verifik.co/v2/co/fasecolda/values-by-plate
```

콜롬비아 등록 차량의 등급, 제조사, 모델, 연식, 추정 가치, 적재량, 연료 유형 등을 조회합니다. **번호판**은 **쿼리** **`plate`** 로 전달합니다(공백·마침표 없음).

### 헤더(Headers)

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 매개변수(Parameters)

| name | type | required | description |
| --- | --- | --- | --- |
| `plate` | string | yes | 조회할 번호판(공백·마침표 없음). |

### 요청(Request)

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/fasecolda/values-by-plate", {
  params: { plate: "ABC123" },
  headers: {
    Accept: "application/json",
    Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
  },
});
console.log(data);
```

### 참고(Notes)

- Fasecolda는 콜롬비아 공식 차량 가치 산정 체계입니다.
- **`valueModel`** 은 연식별 과거 가격 배열입니다.
- 금액은 콜롬비아 페소(COP)입니다.
- **`bcpp`** 는 기준 상업 기본 가격입니다.
- **`estado: "USADO"`** 는 중고 가치 구분 예시입니다.
