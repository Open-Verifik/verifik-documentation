# 🇪🇨 에콰도르 — 차량 벌금

번호판에 연결된 에콰도르 차량의 등록 정보(번호판, 등급, 모델 등), 현재 상태, 서비스 유형, 등록 만료일, **미납 벌금** 등을 조회합니다.

교통법규 준수 및 차량 관리에 활용할 수 있습니다.

### 엔드포인트

```
GET https://api.verifik.co/v2/ec/vehiculo/placa/multas
```

**`plate`**(번호판)로 조회합니다. 등록 정보와 **미납 벌금**을 반환하여 **컴플라이언스**, **차량 관리**, **리스크** 확인에 활용할 수 있습니다.

### 헤더

| 이름           | 값                  |
| -------------- | ------------------- |
| Content-Type   | `application/json`  |
| Authorization  | `Bearer <token>`    |

### 매개변수

| 이름    | 형식   | 필수   | 설명                                      | 예        |
| ------- | ------ | ------ | ----------------------------------------- | --------- |
| `plate` | string | **예** | 조회할 번호판(공백·마침표 없음).         | `ABC1234` |

## 기능

- **벌금 정보**: 번호판에 연결된 벌금 목록 및 상세
- **등록 정보**: 등급, 모델, 상태, 만료일 등
- **차량 사양**: 배기량, 제조국, 연식 등(출처가 반환하는 범위)
- **컴플라이언스**: 교통법규 기반 모니터링
- **다국어 샘플**: JavaScript, Python, PHP, Swift
