# 🇨🇱 칠레 — 차량 정보

칠레 차량 검증 서비스는 차량 정보에 대한 종합적인 검증 기능을 제공합니다. 본 서비스를 통해 칠레에 등록된 차량의 상세 정보를 조회할 수 있습니다.

관련 칠레 차량 endpoint:

- **차량 정보 (`v2/cl/vehicle`)**: 일반 차량 등록 정보.
- **차량 기술 검사 (`v3/cl/vehicle`)**: 기술 검사 상태와 이력. [칠레 - 차량 기술 검사](/verifik-ko/vehicle-validation/chile/vehicle-technical-review)를 참조하세요.
- **차량 SOAP (`v2/cl/vehicle-soap`)**: 의무 보험 증명서. [칠레 - 차량 SOAP](/verifik-ko/vehicle-validation/chile/vehicle-soap)를 참조하세요.

## 차량 정보 조회

### 엔드포인트

```
https://api.verifik.co/v2/cl/vehicle
```

**번호판**(`plate`, 공백이나 점 없음)으로 조회합니다. 일반적으로 반환되는 필드에는 **제조사**, **모델**, **연식**, **VIN/차대 번호**, **엔진**, **색상**, **차량 종류**, **과태료 요약**, 그리고 등기소가 노출하는 경우 **소유자 RUT** 가 포함됩니다. **차량 운영 온보딩**, **담보 점검**, **보험금 청구** 워크플로우에 활용할 수 있습니다.

### 헤더(Headers)

| 이름           | 값                  |
| -------------- | ------------------- |
| Accept         | `application/json`  |
| Authorization  | `Bearer <token>`    |

### 매개변수(Parameters)

| 이름     | 형식    | 필수 | 설명                                                | 예시      |
| -------- | ------- | ---- | --------------------------------------------------- | --------- |
| `plate`  | string  | **예** | 현재 형식 `BB-CC-12` (`BBCC12`로 전송, 4자리 문자 + 2자리 숫자, 공백/점 없음). | `BBCC12` |

## 주요 기능

### 차량 정보 조회
- **차량 등록**: 칠레 내 차량 등록을 검증합니다.
- **기술 사양**: 차대 번호, 엔진 번호, 제조사, 모델 정보를 제공합니다.
- **차량 상세 정보**: 연식, 색상, 종류 정보를 조회할 수 있습니다.
- **소유자 정보**: 차량 소유자의 RUT 및 상세 정보를 조회합니다.
- **과태료 상태**: 차량과 연관된 미납 과태료 여부를 확인합니다.
- **도난 상태**: 차량에 도난 신고가 있는지 확인합니다.
- **대중교통 분류**: 대중교통 분류를 점검합니다.

### 일반 기능
- **다양한 프로그래밍 언어 지원**: JavaScript, Python, PHP, Swift 예제를 제공합니다.
- **실시간 데이터**: 최신 정보를 조회할 수 있습니다.
- **종합 오류 처리**: 다양한 시나리오에 대한 상세한 오류 응답을 제공합니다.
- **구조화된 응답**: 서명 검증을 포함한 잘 정리된 JSON 응답을 제공합니다.
- **칠레 등기소 연동**: 공식 칠레 차량 데이터베이스에 직접 접근합니다.
