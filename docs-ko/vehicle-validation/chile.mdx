# 칠레 차량 정보

칠레에 등록된 차량에 대한 상세 정보를 조회합니다. **플릿**, **보험**, **담보**, 운영 컴플라이언스에 활용하세요.

## 번호판 조회

### Endpoint

```
https://api.verifik.co/v2/cl/vehicle
```

**`plate`**(공백·점 없음)로 조회합니다. 일반적으로 **제조사**, **모델**, **연식**, **차대/VIN**, **엔진**, **색**, **차종**, **과태료 요약**, 레지스트리가 노출하면 **소유자 RUT** 등이 포함됩니다.

### Headers

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parameters

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `plate` | string | Yes | 번호판(공백·점 없음) |

### Request

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/cl/vehicle", {
  params: { plate: "BB985" },
  headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os, requests

url = "https://api.verifik.co/v2/cl/vehicle"
headers = {"Accept": "application/json", "Authorization": f"Bearer {os.getenv('VERIFIK_TOKEN')}"}
params = {"plate": "BB985"}
r = requests.get(url, headers=headers, params=params)
print(r.json())
```

## 기능

### 차량 정보
- **등록**: 칠레 등록 여부 확인.
- **제원**: 차대·엔진·제조사·모델.
- **상세**: 연식·색·차종.
- **소유자**: RUT(가능 시).
- **과태료**: 부과 현황.
- **도난**: 소스 제공 시.
- **대중교통**: 해당 시 분류.

### 일반
- **Node.js·Python** 예제.
- **실시간** 레지스트리 데이터.
- **구조화된 오류** 응답.
- **`signature`** 포함 JSON.
