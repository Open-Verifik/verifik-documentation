# 🇧🇴 볼리비아 — 차량 정보

본 서비스는 번호판으로 볼리비아에 등록된 차량의 상세 정보를 조회합니다. 보험 증권 번호, 브랜드, 등급(예: 업무용 오토바이), 등록지, 서비스 유형(예: 자가용), 모델 등을 반환합니다.

### 엔드포인트

```
GET https://api.verifik.co/v2/bo/vehicle
```

번호판(**`plate`**)으로 볼리비아 차량을 조회합니다. 차량 상세·등록 정보 등을 반환하여 **차량 검증** 및 **컴플라이언스**에 활용할 수 있습니다.

### 헤더

| 이름           | 값                  |
| -------------- | ------------------- |
| Content-Type   | `application/json`  |
| Authorization  | `Bearer <token>`    |

### 매개변수

| 이름    | 형식   | 필수   | 설명                    | 예        |
| ------- | ------ | ------ | ----------------------- | --------- |
| `plate` | string | **예** | 조회할 차량 번호판.     | `ABC1234` |

## 기능

- **차량 정보 조회**: 볼리비아 차량 등록부에서 상세 정보를 가져옵니다.
- **번호판 검증**: 국내 등록 번호판을 조회합니다.
- **보험 정보**: 차량 보험 증권 번호를 조회합니다.
- **차량 분류**: 등급(예: 업무용 오토바이) 등 상세 정보를 제공합니다.
- **등록지**: 등록 관할 정보에 접근합니다.
- **서비스 유형**: 자가용·상업용 등 구분을 확인합니다.
- **브랜드·모델**: 제조사 및 모델 정보를 조회합니다.
- **다국어 샘플**: JavaScript, Python, PHP, Swift 예제 제공
- **실시간 데이터**: 최신 차량 정보에 접근합니다.
- **오류 처리**: 다양한 시나리오에 대한 오류 응답을 반환합니다.
