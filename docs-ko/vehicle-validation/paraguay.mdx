# 🇵🇾 파라과이 — 차량 정보

본 서비스는 차량 번호판을 사용해 파라과이 내 차량의 상세 정보를 제공합니다. 응답에는 제조사, 차대, 종류, 연식, 차축 수, 소유자 이름, 서비스 종류, 차량의 현재 상태, 번호판 등 차량 상세 정보가 포함됩니다.

이러한 정보는 차량 검증, 소유권 점검, 컴플라이언스 모니터링 등 다양한 목적에 활용할 수 있습니다.

## 엔드포인트

```
GET https://api.verifik.co/v2/py/vehicle
```

### 헤더(Headers)

| 이름           | 값                  |
| -------------- | ------------------- |
| Content-Type   | `application/json`  |
| Authorization  | `Bearer <token>`    |

### 매개변수(Parameters)

| 이름     | 형식    | 필수 | 설명                  | 예시      |
| -------- | ------- | ---- | --------------------- | --------- |
| `plate`  | string  | **예** | 조회할 번호판.    | `ABC123`  |

## 주요 기능

- **차량 정보 조회**: 파라과이 차량 등기소에서 종합 차량 상세 정보를 조회합니다.
- **번호판 검증**: 파라과이에 등록된 차량 번호판을 검증합니다.
- **차량 사양**: 다음을 포함한 상세 차량 사양에 접근할 수 있습니다.
  - 제조사 및 브랜드 정보
  - 차량 식별을 위한 차대 번호
  - 차량 종류 및 분류
  - 제조 연식 및 모델 상세
  - 차량 구성에 따른 차축 수
- **소유 관계 정보**: 현재 차량 소유자 상세 정보를 조회합니다.
- **서비스 종류 분류**: 차량의 서비스 종류 정보(예: CARGA NACIONAL)를 조회합니다.
- **차량 상태 모니터링**: 차량의 현재 상태와 신분을 확인합니다.
- **컴플라이언스 검증**: 차량 컴플라이언스와 등록 상태를 검증합니다.
- **다양한 프로그래밍 언어 지원**: JavaScript, Python, PHP, Swift 예제를 제공합니다.
- **실시간 데이터**: 최신의 차량 정보를 조회할 수 있습니다.
- **종합 오류 처리**: 다양한 시나리오에 대한 상세한 오류 응답을 제공합니다.
- **파라과이 등기소 연동**: 공식 파라과이 차량 데이터베이스에 직접 접근합니다.
- **차량 종류 분류**: 트럭, 탱크 차량, 상용 차량 등 다양한 차량 종류를 지원합니다.
- **소유 관계 검증**: 법적 및 상업적 목적의 차량 소유 관계를 검증합니다.
