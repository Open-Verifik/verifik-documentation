# 🇵🇾 파라과이 — 차량 정보

**번호판**으로 파라과이에 등록된 차량의 상세 정보를 조회합니다. 브랜드, 차대, 유형, 연식, 차축 수, 소유자, 서비스 유형, 차량 상태, 번호판 등이 포함될 수 있습니다.

차량 확인, 소유권 점검, 컴플라이언스 모니터링에 활용할 수 있습니다.

## Endpoint

```
GET https://api.verifik.co/v2/py/vehicle
```

### Headers

| Name          | Value              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parameters

| Name | Type | Required | Description | Example |
| ---- | ---- | -------- | ----------- | ------- |
| `plate` | string | **Yes** | 조회할 번호판. | `ABC123` |

## 기능

- **등록소 조회**: 파라과이 차량 등록 정보
- **번호판 검증**: 국내 등록 번호판
- **사양**: 브랜드·차대·유형·연식·차축
- **소유자**: 명의 정보
- **서비스·상태**: 서비스 유형(예: CARGA NACIONAL) 및 차량 상황
- **언어 예제**: JavaScript, Python, PHP, Swift
- **실시간 데이터**: 공식 등록 기준 최신 정보
