# Antecedentes na Polícia Nacional da Colômbia

## Referência da API

### Endpoint

```
https://api.verifik.co/v2/co/policia/consultar
```

O serviço de consulta de antecedentes na Polícia Nacional da Colômbia permite verificar o histórico criminal de uma pessoa fornecendo o número do documento de identidade. Ele consulta a existência de assuntos jurídicos pendentes com as autoridades judiciais colombianas, conforme o artigo 248 da Constituição Política da Colômbia. A resposta inclui o nome completo, primeiro nome, sobrenome e detalhes adicionais sobre o status legal. Este serviço é ideal para empresas e organizações que realizam verificações de antecedentes para fins de contratação ou diligência prévia.

### Cabeçalhos

| Nome          | Valor              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parâmetros

:::info
Para a lista completa de **tipos de documento** suportados e seus parâmetros obrigatórios específicos, consulte o guia [Tipos de documento](/background-check/document-types).
:::

| Nome             | Tipo   | Obrigatório | Descrição                                                                 |
| ---------------- | ------ | ----------- | ------------------------------------------------------------------------- |
| `documentType`   | string | Sim         | Tipo de documento. Parâmetros válidos: CC, CE.                            |
| `documentNumber` | string | Sim         | Número do documento da pessoa a ser consultada, sem espaços ou pontos.    |
