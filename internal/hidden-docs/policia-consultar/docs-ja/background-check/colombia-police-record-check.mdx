# コロンビア — 警察記録照会

## API リファレンス

### エンドポイント

```
https://api.verifik.co/v2/co/policia/consultar
```

コロンビア警察記録照会サービスでは、ID 文書番号を指定することで、コロンビアにおける個人の犯罪歴を照会できます。コロンビア憲法第 248 条に従い、コロンビア司法当局における係属中の法的事項の有無を確認します。レスポンスには対象者の氏名、名、姓、その他法的状態に関する詳細が含まれます。採用やデューデリジェンス目的でバックグラウンド調査を行う企業・組織に最適です。

### ヘッダー

| 名前          | 値                 |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### パラメーター

:::info
対応する **文書タイプ** とそれぞれの必須パラメーターの一覧については、[文書タイプ](/background-check/document-types) ガイドをご参照ください。
:::

| 名前             | 型     | 必須 | 説明                                                                       |
| ---------------- | ------ | ---- | -------------------------------------------------------------------------- |
| `documentType`   | string | はい | 文書タイプ。許可される値: CC, CE。                                         |
| `documentNumber` | string | はい | 照会対象者の文書番号。スペースやドットを含めずに入力してください。         |
