# 🇨🇦 Canada — Expiration plaque Ontario

Vérifiez l'expiration de la vignette de plaque Ontario à l'aide du **numéro de plaque** et du **numéro de permis du véhicule** (carte verte / document d'immatriculation délivré par l'Ontario).

Champs de réponse typiques : `expired`, `plateNumber`, `permitNumber`, `expiryDate` et `success`. Le drapeau `expired` vaut `true` lorsque `expiryDate` est antérieure à l'heure actuelle.

Utilisez ce service pour confirmer rapidement la validité de la vignette de plaque auprès du registre gouvernemental de l'Ontario. Idéal pour l'onboarding flotte, la conformité et les contrôles de véhicules.

## Référence API

### Endpoint

```
https://api.verifik.co/v2/ca/ontario/plate
```

Envoyez une requête GET avec les paramètres de requête **`plate`** et **`permitNumber`**.

### En-têtes

| Nom            | Valeur              |
| -------------- | ------------------- |
| Accept         | `application/json`  |
| Authorization  | `Bearer <token>`    |

### Paramètres

| Nom | Type | Requis | Description | Exemple |
| --- | ---- | ------ | ----------- | ------- |
| `plate` | string | **Oui** | Plaque passager Ontario au format **ABCD123** (4 lettres + 3 chiffres, sans espaces). | `BKTR456` |
| `permitNumber` | string | **Oui** | Numéro de permis figurant sur le document d'immatriculation Ontario (8 caractères). | `N9166001` |

### Requête

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/ca/ontario/plate", {
  params: { plate: "BKTR456", permitNumber: "N9166001" },
  headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});
console.log(data);
```
