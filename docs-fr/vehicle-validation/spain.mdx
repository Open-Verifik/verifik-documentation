# Espagne

## Référence API

### Endpoint

```
https://api.verifik.co/v2/es/vehicle
```

Retourne les informations de véhicule pour une plaque espagnole (marque, modèle, année, type, etc.) lorsque le véhicule est trouvé.

### En-têtes

| Nom          | Valeur              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Paramètres

| Nom | Type | Obligatoire | Description |
| --- | --- | --- | --- |
| `plate` | string | Oui | Plaque à interroger (sans espaces). |

### Requête

<Tabs>
  <TabItem value="javascript" label="JavaScript">

```javascript
import axios from "axios";

const options = {
  method: "GET",
  url: "https://api.verifik.co/v2/es/vehicle",
  params: { plate: "5684CCD" },
  headers: {
    Accept: "application/json",
    Authorization: "Bearer <your_token>",
  },
};

try {
  const { data } = await axios.request(options);
  console.log(data);
} catch (error) {
  console.error(error);
}
```

## Cas d'usage

- **Assurance** : confirmer les attributs du véhicule
- **Marketplaces & logistique** : valider un véhicule annoncé
- **Conformité** : appuyer les contrôles liés aux véhicules immatriculés en Espagne
