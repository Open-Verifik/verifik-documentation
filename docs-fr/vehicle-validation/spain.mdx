# Espagne

## Référence de l'API

### Endpoint

```
https://api.verifik.co/v2/es/vehicle
```

Retourne les informations d'un véhicule pour une plaque d'immatriculation espagnole, incluant des champs comme la marque, le modèle, l'année et le type de véhicule lorsqu'une correspondance existe.

### En-têtes (Headers)

| Nom            | Valeur              |
| -------------- | ------------------- |
| Content-Type   | `application/json`  |
| Authorization  | `Bearer <token>`    |

### Paramètres

| Nom     | Type   | Requis  | Description                                    |
| ------- | ------ | ------- | ---------------------------------------------- |
| `plate` | string | **Oui** | Plaque d'immatriculation à interroger (sans espaces). |

### Requête (Request)

<Tabs>
  <TabItem value="javascript" label="JavaScript">

```javascript
import axios from "axios";

const options = {
  method: "GET",
  url: "https://api.verifik.co/v2/es/vehicle",
  params: { plate: "5684CCD" },
  headers: {
    Accept: "application/json",
    Authorization: "Bearer <votre_token>",
  },
};

try {
  const { data } = await axios.request(options);
  console.log(data);
} catch (error) {
  console.error(error);
}
```

## Cas d'usage

- **Assurance et souscription** : confirmez les attributs d'un véhicule à partir de sa plaque.
- **Places de marché et logistique** : validez les véhicules de votre flotte ou listés sur votre plateforme.
- **Conformité** : appuyez les contrôles liés aux véhicules immatriculés en Espagne.
