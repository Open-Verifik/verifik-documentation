### Endpoint

```
https://api.verifik.co/v2/co/simit/comparendos
```

Le service de contraventions routières fournit des informations détaillées sur les contraventions liées à un véhicule donné en Colombie. En saisissant le numéro de plaque du véhicule, vous pouvez consulter pour chaque contravention le statut, la date, le lieu et le type d'infraction.

### En-têtes

| Nom | Valeur |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Paramètres

| Nom | Type | Obligatoire | Description |
| --- | --- | --- | --- |
| `documentType` | string | Oui | Type de document. Valeurs valides : `CC`, `PA`, `CE`, `TI`, `RC`. |
| `documentNumber` | string | Oui | Numéro de document à consulter, sans espaces ni points. |

  headers: {

### Remarques

- SIMIT est le système intégré colombien des infractions routières.
- « Comparendo » est le terme espagnol pour une contravention routière.
- Les montants sont exprimés en pesos colombiens (COP).
- Le statut « Pendiente » indique que le paiement de la contravention est en attente.
- Les infractions détectées par photo sont signalées par `fotodeteccion: true`.
