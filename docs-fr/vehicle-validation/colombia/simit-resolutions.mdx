### Endpoint

```
https://api.verifik.co/v2/co/simit/resoluciones
```

Le service des résolutions SIMIT donne accès aux informations détaillées sur les résolutions d'infractions routières pour une personne ou une entreprise en Colombie : nom de l'infracteur, statut de la résolution, date et montant dû.

### En-têtes

| Nom | Valeur |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Paramètres

| Nom | Type | Obligatoire | Description |
| --- | --- | --- | --- |
| `documentType` | string | Oui | Type de document. Valeurs valides : `CC`, `PA`, `CE`, `TI`, `RC`. |
| `documentNumber` | string | Oui | Numéro de document à consulter, sans espaces ni points. |

### Remarques

- Les résolutions SIMIT sont des décisions administratives officielles sur les infractions routières.
- Le statut « Pendiente de pago » indique que le paiement de la résolution est en attente.
- Les numéros de résolution permettent de suivre chaque décision administrative.
- Les montants sont exprimés en pesos colombiens (COP).
