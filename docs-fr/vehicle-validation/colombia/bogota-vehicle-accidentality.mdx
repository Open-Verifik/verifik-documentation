### Endpoint

```
GET https://api.verifik.co/v2/co/bogota/vehicle/accidentality
```

Retourne l’historique d’accidents d’un véhicule immatriculé à Bogotá. Passez la **plaque** en paramètre **`plate`**. La réponse contient un tableau `accident` (dates, numéro de formulaire `form`, gravité `seriousness`).

### En-têtes (Headers)

| Nom | Valeur |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Paramètres

| Nom | Type | Requis | Description |
| --- | --- | --- | --- |
| `plate` | string | **Oui** | Plaque d’immatriculation, sans espaces ni points. |

### Remarques

- **SOLO DAÑOS** signifie dommages matériels uniquement (pas de blessés).
- Le champ **`form`** est la référence du rapport d’accident officiel.
- Les dates peuvent être renvoyées en espagnol (jour de la semaine inclus).
