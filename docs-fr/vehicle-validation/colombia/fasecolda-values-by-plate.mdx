### Endpoint

```
GET https://api.verifik.co/v2/co/fasecolda/values-by-plate
```

Récupérez les détails des véhicules enregistrés en Colombie (classe, marque, modèle, année, valeur estimée, charge, carburant, etc.). Passez la **plaque** en paramètre **`plate`** dans la **query** (sans espaces ni points).

### En-têtes

| Nom | Valeur |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Paramètres

| Nom | Type | Obligatoire | Description |
| --- | --- | --- | --- |
| `plate` | string | Oui | Plaque sans espaces ni points. |

### Notes

- Fasecolda est le système officiel de valorisation des véhicules en Colombie.
- Le tableau **`valueModel`** contient des valeurs historiques par année modèle.
- Montants en pesos colombiens (COP).
- **`bcpp`** : prix commercial de base.
- **`estado: "USADO"`** : exemple pour un véhicule d'occasion.
