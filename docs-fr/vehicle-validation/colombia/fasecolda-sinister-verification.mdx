### Endpoint

```
GET https://api.verifik.co/v2/co/fasecolda/sinister
```

Interroge l’historique d’accidents et de sinistres d’assurance (**sinister**) auprès de Fasecolda. Transmettez la **plaque** colombienne en paramètre **`plate`**.

### En-têtes (Headers)

| Nom | Valeur |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Paramètres

| Nom | Type | Requis | Description |
| --- | --- | --- | --- |
| `plate` | string | **Oui** | Plaque du véhicule, sans espaces ni points. |

### Requête (Request)

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/fasecolda/sinister", {
  params: { plate: "ABC123" },
  headers: {
    Accept: "application/json",
    Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
  },
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">


### Remarques

- **Sinister** renvoie ici aux sinistres / dossiers d’assurance au sens colombien.
- **Pérdida Menor Cuantía** correspond à une perte de faible montant.
- Chaque ligne peut contenir un identifiant, la date d’accident et le type de protection.
