### Endpoint

```
https://api.verifik.co/v2/co/simit/acuerdos
```

Le service des accords de paiement SIMIT permet de récupérer efficacement les accords de paiement liés aux infractions routières dans le système SIMIT en Colombie. Avec les paramètres appropriés, vous obtenez des informations sur les accords, l'infracteur, les numéros de résolution et les montants dus.

### En-têtes

| Nom | Valeur |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Paramètres

| Nom | Type | Obligatoire | Description |
| --- | --- | --- | --- |
| `documentType` | string | Oui | Type de document. Valeurs valides : `CC`, `PA`, `CE`, `TI`, `RC`. |
| `documentNumber` | string | Oui | Numéro de document à consulter, sans espaces ni points. |

### Remarques

- Les accords SIMIT permettent de payer les amendes routières en plusieurs fois.
- Le statut « Acuerdo de pago » indique un accord de paiement actif.
- `permitePago: "N"` signifie que le paiement n'est pas autorisé pour le moment.
- Les montants sont exprimés en pesos colombiens (COP).
- Les numéros de résolution identifient chaque accord.
