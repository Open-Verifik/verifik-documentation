### Endpoint

```
https://api.verifik.co/v2/co/simit/acuerdos
```

Le service Accords SIMIT propose une méthode efficace pour récupérer les accords de paiement liés aux infractions routières dans le système SIMIT colombien. En interrogeant le service avec les détails pertinents, vous obtenez des informations complètes sur les accords de paiement, dont les coordonnées du contrevenant, les numéros de résolution et les montants dus.

### En-têtes (Headers)

| Nom | Valeur |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Paramètres

| Nom | Type | Requis | Description |
| --- | --- | --- | --- |
| `documentType` | string | **Oui** | Type de document. Valeurs autorisées : `CC`, `PA`, `CE`, `TI`, `RC`. |
| `documentNumber` | string | **Oui** | Numéro de document à interroger, sans espaces ni points. |

### Notes

- Les accords SIMIT permettent aux contrevenants de régler leurs amendes routières en plusieurs versements.
- Le statut « Acuerdo de pago » indique un accord de paiement actif.
- `permitePago: "N"` signifie que le paiement n'est pas autorisé actuellement.
- Les montants sont exprimés en pesos colombiens (COP).
- Les numéros de résolution servent à suivre les accords.
