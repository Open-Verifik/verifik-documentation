### Point d'accès

```
https://api.verifik.co/v2/co/bogota/vehicle/fines
```

Ce service permet de consulter les amendes associées à un véhicule à Bogotá (Colombie) par numéro de plaque. La réponse inclut le type d'amende, le statut, le numéro, les dates et les détails de paiement.

### En-têtes

| Nom | Valeur |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Paramètres

| Nom | Type | Requis | Description |
| --- | --- | --- | --- |
| `plate` | string | Oui | Plate to consult, without spaces or points. |

### Requête

<Tabs>

  headers: { 
    Accept: "application/json", 
    Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`

### Notes

- Le statut « vigente » indique que l'amende est active et doit être payée.
- Le champ `total` inclut le montant initial plus les intérêts accumulés.
- Utilisez le champ `number` pour référencer une amende lors du paiement ou d'un recours.
