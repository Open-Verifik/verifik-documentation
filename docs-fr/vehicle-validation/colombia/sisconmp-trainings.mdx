### Endpoint

```
GET https://api.verifik.co/v2/co/sisconmp/trainings
```

Vérifiez la **formation et l’enregistrement** des opérateurs dans le **SISCONMP** (système colombien de contrôle et de suivi du personnel). Passez **`documentType`** et **`documentNumber`** en **paramètres de requête** (organisme, intitulé du cours, dates, statut du permis).

### En-têtes

| Nom | Valeur |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Paramètres

| Nom | Type | Obligatoire | Description |
| --- | --- | --- | --- |
| `documentType` | string | Oui | Type de document : `CC`, `CE` ou `PA`. |
| `documentNumber` | string | Oui | Numéro du participant sans espaces ni points. |

### Requête

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/sisconmp/trainings", {
  params: {
    documentType: "CC",
    documentNumber: "123456789",
  },
  headers: {
    Accept: "application/json",
    Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
  },
});
console.log(data);
```

### Notes

- Le SISCONMP encadre le personnel qui transporte des marchandises dangereuses.
- **CURSO BASICO** est un exemple de formation de base.
- Les attestations ont une durée de validité et peuvent devoir être renouvelées.
- **`inactive: "No"`** : exemple d’enregistrement actif.
