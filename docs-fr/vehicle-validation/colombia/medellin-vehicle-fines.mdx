### Endpoint

```
https://api.verifik.co/v2/co/medellin/vehicle/fines
```

Ce service renvoie les **amendes et comparendos locaux** pour une plaque à **Medellín, Colombie**, incluant les infractions, montants et infracteurs lorsque disponibles. Utilisez-le pour la **conformité municipale**, la **gestion de flotte** et la **due diligence** limitée aux registres de mobilité de Medellín.

Champs typiques : **`plate`**, **`informacionComparendo`**, **`valorTotalAdeudado`** et tableaux associés.

### En-têtes

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Paramètres

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `plate` | string | Yes | Plaque (5–8 caractères), sans espaces ni points. |

### Remarques

- **Sandbox :** plaques **`ABC10001`**–**`ABC10010`** renvoient des fixtures ; **`ERR404`** → **404**.
- Couvre **Medellín uniquement** ; pour SIMIT national voir [SIMIT - Amendes](/vehicle-validation/colombia/simit-fines).
- La **disponibilité** du portail municipal peut varier ; des pannes peuvent renvoyer des erreurs **5xx**.
