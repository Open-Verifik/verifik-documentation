### Endpoint

```
GET https://api.verifik.co/v2/co/simit/suspensiones
```

Interroge le **SIMIT** pour connaître les **suspensions ou annulations** du permis de conduire (et données associées lorsque la source les fournit). Identifie le titulaire avec **`documentType`** et **`documentNumber`**.

### En-têtes

| Nom | Valeur |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Paramètres

| Nom | Type | Obligatoire | Description |
| --- | --- | --- | --- |
| `documentType` | string | Oui | Type de pièce du titulaire : `CC`, `PA`, `CE`, `TI`, `RC`, `NIT`, etc. |
| `documentNumber` | string | Oui | Numéro sans espaces ni ponctuation. Une longueur excessive peut renvoyer `409`. |

### Requête

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/simit/suspensiones", {
	params: { documentType: "CC", documentNumber: "123456789" },
	headers: {
		Accept: "application/json",
		Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
	},
});
console.log(data);
```

  </TabItem>

```python
import os, requests

r = requests.get(url, headers=headers, params=params)
```

### Notes

- Le **SIMIT** est le registre national colombien des infractions et sanctions routières.
- La structure exacte de **`data`** dépend du téléchargement **SIMIT** (l’exemple est indicatif).
