### Point d'accès

```
https://api.verifik.co/v2/co/simit/consultar
```

Ce service permet d'accéder aux contraventions et amendes routières associées à un numéro de document en Colombie. The response includes comprehensive details about each traffic ticket, such as the type of violation, the amount of the fine, and the current status of the ticket.

### En-têtes

| Nom | Valeur |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Paramètres

| Nom | Type | Requis | Description |
| --- | --- | --- | --- |
| `documentType` | string | Oui | Document type. Valid parameters: `CC`, `PA`, `CE`, `TI`, `RC`, `NIT`. |
| `documentNumber` | string | Oui | Numéro de document à consulter, sans espaces ni points. |

### Notes

- Service complet SIMIT : amendes, cours et accords de paiement pour une personne.
- "Multas" contains traffic violation details with infraction codes and descriptions.
- "Cursos" shows traffic education courses taken by the person.
- "AcuerdosPago" displays payment agreements for outstanding fines.
- Amounts are provided in Colombian Pesos (COP).
