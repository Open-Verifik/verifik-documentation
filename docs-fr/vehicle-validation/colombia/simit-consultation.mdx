# SIMIT - Consultation

Le service de consultation SIMIT donne accès à des informations complètes sur les infractions routières pour les véhicules et les conducteurs en Colombie.

### Endpoint

```
https://api.verifik.co/v2/co/simit/consultar
```

### En-têtes

| Nom | Valeur |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Paramètres

| Nom | Type | Obligatoire | Description |
| --- | --- | --- | --- |
| `documentType` | string | Oui | Type de document. Paramètres valides : `CC`, `PA`, `CE`, `TI`, `RC`, `NIT`. |
| `documentNumber` | string | Oui | Numéro de document à consulter, sans espaces ni points. |

### Remarques

- Ce service complet fournit toutes les données SIMIT pour une personne, y compris les amendes, les cours et les accords de paiement.
- « Multas » contient les détails des infractions routières avec codes et descriptions.
- « Cursos » indique les cours de sensibilisation routière suivis par la personne.
- « AcuerdosPago » affiche les accords de paiement pour les amendes impayées.
- Les montants sont exprimés en pesos colombiens (COP).
