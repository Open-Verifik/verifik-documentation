### Endpoint

```
GET https://api.verifik.co/v2/co/bogota/vehicle/tax
```

Donne les obligations fiscales d’un véhicule immatriculé à Bogotá. Indiquez le type et le numéro du document du propriétaire (`documentType`, `documentNumber`) et la **plaque** (`plate`) en **paramètres de requête**. La réponse inclut le détail du véhicule et le statut des obligations par exercice.

### En-têtes (Headers)

| Nom | Valeur |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Paramètres

| Nom | Type | Requis | Description |
| --- | --- | --- | --- |
| `plate` | string | **Oui** | Plaque d’immatriculation, sans espaces ni points. |
| `documentNumber` | string | **Oui** | Numéro de document, sans espaces ni points. |
| `documentType` | string | **Oui** | Type de document autorisé : `CC` ou `NIT`. |

### Requête (Request)

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/bogota/vehicle/tax", {
  params: { 
    plate: "ABC123", 
    documentType: "CC", 
    documentNumber: "123456789" 
  },
  headers: { 
    Accept: "application/json", 
    Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` 
  },
});
console.log(data);
```

### Remarques

- **`CC`** = carte d’identité colombienne ; **`NIT`** = numéro d’identification fiscale (entreprise ou particulier assujetti).
- Si **`obligationStatus`** vaut **Moroso**, des impôts restent dus.
