### Endpoint

```
GET https://api.verifik.co/v2/co/fasecolda/values-by-code
```

Récupérez les attributs et valeurs commerciales du véhicule Fasecolda avec le paramètre **`codeFasecolda`** (code catalogue).

### En-têtes

| Nom | Valeur |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Paramètres

| Nom | Type | Obligatoire | Description |
| --- | --- | --- | --- |
| `codeFasecolda` | string | Oui | Code Fasecolda de la configuration du véhicule. |

### Requête

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/fasecolda/values-by-code", {
  params: { codeFasecolda: "08032036" },
  headers: {
    Accept: "application/json",
    Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
  },
});
console.log(data);
```

### Notes

- Le code Fasecolda identifie modèle et finition/équipement.
- Le niveau de détail est comparable au service par plaque (`values-by-plate`).
- Montants en pesos colombiens (COP).
- **`bcpp`** : prix commercial de base.
- **`estado: "USADO"`** : exemple de valeur pour un véhicule d'occasion.
