### Endpoint

```
https://api.verifik.co/v2/co/simit/consultar/placa
```

Ce service permet d'obtenir des informations détaillées sur les amendes et contraventions routières associées à un numéro de plaque d'immatriculation en Colombie. La réponse inclut pour chaque contravention le type d'infraction, le montant de l'amende et le statut actuel.

### En-têtes

| Nom | Valeur |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Paramètres

| Nom | Type | Obligatoire | Description |
| --- | --- | --- | --- |
| `plate` | string | Oui | Plaque à consulter, sans espaces ni points. |

### Remarques

- Ce service complet fournit toutes les données SIMIT pour un véhicule, y compris les amendes, les cours et les accords de paiement.
- « Multas » contient les détails des infractions routières liées à la plaque du véhicule.
- « Cursos » indique les cours de sensibilisation routière liés aux infractions du véhicule.
- « AcuerdosPago » affiche les accords de paiement pour les amendes impayées.
- Les montants sont exprimés en pesos colombiens (COP).
