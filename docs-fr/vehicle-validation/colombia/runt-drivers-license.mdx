### Endpoint

```
GET https://api.verifik.co/v2/co/runt/conductor
```

Interroge le **RUNT** pour le **permis de conduire** avec **`documentType`** et **`documentNumber`**. La réponse peut inclure statut, catégories, certificats, demandes et indication de sanctions.

### En-têtes

| Nom | Valeur |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Paramètres

| Nom | Type | Obligatoire | Description |
| --- | --- | --- | --- |
| `documentType` | string | Oui | Type de pièce. Valeurs acceptées par l’API : **`CC`**, **`CE`**, **`PA`**, **`PPT`**. |
| `documentNumber` | string | Oui | Numéro sans espaces ni ponctuation. |

### Requête

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/runt/conductor", {
	params: { documentType: "CC", documentNumber: "123456789" },
	headers: {
		Accept: "application/json",
		Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
	},
});
console.log(data);
```

- La structure dépend du retour **RUNT**.
