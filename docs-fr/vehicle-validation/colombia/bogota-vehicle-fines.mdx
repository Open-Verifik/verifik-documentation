### Endpoint

```
GET https://api.verifik.co/v2/co/bogota/vehicle/fines
```

Permet d’interroger les contraventions liées à un véhicule à Bogotá, Colombie, à partir de la **plaque** (`plate`). La réponse inclut le type, le statut, le numéro, les dates et les informations de paiement.

### En-têtes (Headers)

| Nom | Valeur |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Paramètres

| Nom | Type | Requis | Description |
| --- | --- | --- | --- |
| `plate` | string | **Oui** | Plaque à interroger, sans espaces ni points. |

### Requête (Request)

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/bogota/vehicle/fines", {

  headers: { 
    Accept: "application/json", 
    Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` 
  },
});
```

### Remarques

- Le statut **vigente** signifie que l’amende est en cours et doit être payée.
- Le champ **`total`** peut inclure le montant initial et les intérêts accumulés.
- Utilisez le champ **`number`** pour référencer une contravention précise (paiement ou recours).
