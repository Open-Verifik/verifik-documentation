### Endpoint

```
GET https://api.verifik.co/v2/co/vehiculo/pico-y-placa
```

Informations **temps réel** sur le **Pico y Placa** pour les véhicules immatriculés à **Bogotá**. La restriction repose sur le **dernier chiffre** de la plaque. Des **exceptions** (handicap, etc.) peuvent figurer dans la réponse quand elles s’appliquent à la **`plate`**.

### En-têtes

| Nom | Valeur |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Paramètres

| Nom | Type | Obligatoire | Description |
| --- | --- | --- | --- |
| `plate` | string | Oui | Plaque sans espaces (format bogotanais). |

### Notes

- Uniquement véhicules immatriculés à **Bogotá**.
- La règle s’appuie sur le **dernier chiffre** de la plaque.
- **Dérogations** (handicap, urgence, etc.) peuvent apparaître comme exceptions.
- **`ACTIVO`** : exemple d’exception en cours de validité.
