# 🇨🇷 Costa Rica — Informations véhicule

Ce service retourne des informations détaillées sur un véhicule immatriculé au Costa Rica à partir de sa **plaque d'immatriculation**. Les données typiques incluent le propriétaire actuel, les caractéristiques du véhicule (cylindrée, valeur fiscale, poids, capacité, puissance, traction), l'historique des **infractions** et celui des **propriétaires précédents** lorsqu'ils sont exposés par la source.

Utilisez-le pour les vérifications de flotte, l'assurance et les flux de conformité.

## Référence de l'API

### Endpoint

```
https://api.verifik.co/v2/cr/vehicle
```

Envoyez une requête GET avec le paramètre **`plate`** (sans espaces ni ponctuation). Une réponse réussie inclut les données structurées du véhicule, les champs propriétaire et les tableaux d'historique lorsqu'ils sont disponibles.

### En-têtes (Headers)

| Nom            | Valeur              |
| -------------- | ------------------- |
| Accept         | `application/json`  |
| Authorization  | `Bearer <token>`    |

### Paramètres

| Nom     | Type   | Requis  | Description                                         | Exemple   |
| ------- | ------ | ------- | --------------------------------------------------- | --------- |
| `plate` | string | **Oui** | Plaque à interroger, sans espaces ni points.        | `ABC123`  |

## Fonctionnalités

- **Récupération d'informations véhicule** : obtenez les détails complets depuis le registre automobile costaricien.
- **Validation de plaque** : vérifiez les plaques d'immatriculation au Costa Rica.
- **Propriétaire actuel** : récupérez les coordonnées du propriétaire actuel.
- **Caractéristiques du véhicule** : cylindrée, valeur fiscale, poids, capacité, puissance et traction.
- **Historique des infractions** : accédez à l'historique complet des infractions routières.
- **Historique des propriétaires** : consultez la liste chronologique de tous les propriétaires précédents.
- **Autorité judiciaire** : informations sur l'autorité judiciaire en charge des infractions.
- **Informations de contravention** : numéros de contravention et de dossier pour les infractions.
- **Plusieurs langages de programmation** : exemples en JavaScript, Python, PHP et Swift.
- **Données en temps réel** : informations à jour sur le véhicule.
- **Gestion d'erreurs complète** : réponses d'erreur détaillées pour chaque scénario.
- **Intégration au registre costaricien** : accès direct à la base officielle des véhicules au Costa Rica.
