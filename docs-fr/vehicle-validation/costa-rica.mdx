# Véhicule au Costa Rica (plaque)

Retourne propriétaire actuel, caractéristiques techniques, infractions et historique de titulaires lorsque la source les fournit.

## Référence API

### Endpoint

```
https://api.verifik.co/v2/cr/vehicle
```

Envoyez le paramètre **`plate`** (plaque sans espaces ni ponctuation).

### En-têtes

| Nom | Valeur |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Paramètres

| Nom | Type | Obligatoire | Description | Exemple |
| --- | --- | --- | --- | --- |
| `plate` | string | Oui | Plaque à consulter. | `ABC123` |

### Requête

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/cr/vehicle", {
  params: { plate: "ABC123" },
  headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});
console.log(data);
```
