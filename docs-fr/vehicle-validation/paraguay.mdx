# 🇵🇾 Paraguay — Informations véhicule

Ce service renvoie des informations détaillées sur un véhicule au **Paraguay** à partir de son **numéro de plaque**. La réponse inclut notamment la marque, le châssis, le type, l'année, le nombre d'essieux, le propriétaire, le type de service, le statut du véhicule et la plaque.

Utile pour la vérification de véhicules, les contrôles de propriété et le suivi de conformité.

## Endpoint

```
GET https://api.verifik.co/v2/py/vehicle
```

### Headers

| Name          | Value              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parameters

| Name | Type | Required | Description | Example |
| ---- | ---- | -------- | ----------- | ------- |
| `plate` | string | **Yes** | Plaque à consulter. | `ABC123` |

## Fonctionnalités

- **Consultation registre** : détails véhicule auprès du registre paraguayen
- **Validation de plaque** : plaques immatriculées au Paraguay
- **Spécifications** : marque, châssis, type, année, essieux
- **Propriétaire** : informations sur le titulaire
- **Service et statut** : type de service (ex. CARGA NACIONAL) et situation du véhicule
- **Langages** : exemples JavaScript, Python, PHP et Swift
- **Données à jour** : informations récentes du registre officiel
