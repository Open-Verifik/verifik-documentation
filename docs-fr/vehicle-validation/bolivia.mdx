# 🇧🇴 Bolivie — Informations véhicule

Le service d'informations véhicule en Bolivie permet de récupérer des détails précis sur un véhicule immatriculé dans le pays. En fournissant le numéro de plaque, le service renvoie des informations telles que le numéro de police, la marque, la classe (p. ex. moto de travail), le lieu d'immatriculation, le type de service (p. ex. particulier) et le modèle.

### Point d'accès

```
GET https://api.verifik.co/v2/bo/vehicle
```

Recherchez un véhicule par plaque (**`plate`**). Renvoie les détails d'immatriculation et du véhicule pour la **vérification** et la **conformité**.

### En-têtes

| Nom | Valeur |
| --- | --- |
| Content-Type | `application/json` |
| Authorization | `Bearer <token>` |

### Paramètres

| Nom | Type | Requis | Description | Exemple |
| --- | --- | --- | --- | --- |
| `plate` | string | **Oui** | Plaque d'immatriculation à consulter. | `ABC1234` |

## Fonctionnalités

- **Récupération d'informations véhicule** : obtenez des détails complets depuis le registre bolivien
- **Validation de plaque** : vérifiez les plaques immatriculées en Bolivie
- **Informations d'assurance** : récupérez les numéros de police d'assurance
- **Classification du véhicule** : classe détaillée (p. ex. moto de travail)
- **Lieu d'immatriculation** : accédez au lieu d'enregistrement
- **Type de service** : particulier, commercial ou autre
- **Marque et modèle** : fabricant et modèle du véhicule
- **Langages multiples** : JavaScript, Python, PHP et Swift
- **Données en temps réel** : informations actuelles et à jour
- **Gestion d'erreurs complète** : réponses d'erreur détaillées
