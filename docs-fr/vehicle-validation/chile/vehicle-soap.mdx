# 🇨🇱 Chili — SOAP véhicule

La consultation **SOAP** véhicule (*Seguro Obligatorio de Accidentes Personales*) valide l'assurance obligatoire contre les accidents corporels pour un véhicule immatriculé au Chili. Interrogez par **plaque** et **numéro de police** pour obtenir l'assureur, le titulaire, les dates de validité et les attributs du véhicule dans le registre AACH.

## Recherche SOAP véhicule

### Endpoint

```
GET https://api.verifik.co/v2/cl/vehicle-soap
```

Interrogez par **plaque** (`plate`) et **numéro de police** (`policyNumber` ; le service utilise les quatre derniers chiffres dans le registre AACH). Les champs typiques incluent **status**, **insuranceCompany**, **ownerName**, **ownerRut**, **validFrom**, **validTo**, **brand**, **model**, **manufactureYear**, **engineNumber** et **premium** — utile pour la **conformité assurance**, l'**onboarding de flottes** et les **sinistres**.

### En-têtes

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Paramètres

| Name | Type | Required | Description | Example |
| --- | --- | --- | --- | --- |
| `plate` | string | **Yes** | Plaque chilienne (Patente), 4–6 caractères alphanumériques. | `HTRT86` |
| `policyNumber` | string | **Yes** | Numéro de police ; les quatre derniers chiffres sont utilisés pour la consultation SOAP AACH. | `94596506` |

### Remarques

- **Mode sandbox :** les plaques **`BBCC12`**–**`BBCC21`** renvoient des profils SOAP fixes (`BBCC15` = **VENCIDO**) ; **`HTRT86`** + **`policyNumber=94596506`** renvoie le profil démo ; **`ERR404`** renvoie **404** ; les plaques non mappées renvoient un **200** aléatoire.
- **Exemples sandbox :** `BBCC12`/`94590001` (berline, MAPFRE), `BBCC18`/`94590007` (pick-up, CHUBB), `HTRT86`/`94596506` (démo TOYOTA COROLLA).
- Pour la consultation **véhicule particulier** au Chili, voir [🇨🇱 Chili — Véhicule](/verifik-fr/vehicle-validation/chile).
- La **disponibilité** du registre dépend de l'environnement ; les pannes peuvent renvoyer des erreurs **409** (timeout).
