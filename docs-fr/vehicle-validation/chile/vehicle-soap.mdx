# 🇨🇱 Chili — SOAP véhicule

Consultez le certificat **SOAP** (*Seguro Obligatorio de Accidentes Personales*) d'un véhicule au Chili par **plaque** et **numéro de police** (registre AACH).

## Recherche SOAP véhicule

### Endpoint

```
GET https://api.verifik.co/v2/cl/vehicle-soap
```

### En-têtes

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Paramètres

| name | type | required | description |
| --- | --- | --- | --- |
| `plate` | string | yes | Plaque chilienne (4–6 caractères). |
| `policyNumber` | string | yes | Numéro de police (4 derniers chiffres utilisés). |

### Requête

<Tabs>
  <TabItem value="javascript" label="JavaScript">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/cl/vehicle-soap", {
	params: { plate: "HTRT86", policyNumber: "94596506" },
	headers: {
		Accept: "application/json",
		Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
	},
});
console.log(data);
```

  </TabItem>
</Tabs>
