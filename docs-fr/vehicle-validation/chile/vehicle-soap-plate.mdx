# 🇨🇱 Chili — SOAP véhicule (par plaque)

Consultez le certificat **SOAP** (*Seguro Obligatorio de Accidentes Personales*) avec **la plaque uniquement**. Réponse imbriquée **`soap`** / **`vehicle`**, message vide si SOAP trouvé, ou `No_active_SOAP_certificate_found_for_this_plate` si le véhicule existe sans SOAP actif.

### Endpoint

```
GET v2/cl/vehicle-soap-plate
```

### En-têtes

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Paramètres

| name | type | required | description |
| --- | --- | --- | --- |
| `plate` | string | yes | Plaque chilienne (4–6 caractères). |
| `force` | string | no | `true` pour ignorer le cache. |

### Notes

- **Pas de SOAP actif** = **HTTP 200** avec **`soap` vide**, pas un 404.
- Plaques live : `CLTK27`, `HTRT86`, `RRWX51`, `THHV64`, `CSXZ52`, `CSXZ57` (SOAP actif) ; `CSXZ27` (sans SOAP).
- Voir aussi [SOAP véhicule (plaque + police)](/verifik-fr/vehicle-validation/chile/vehicle-soap).
