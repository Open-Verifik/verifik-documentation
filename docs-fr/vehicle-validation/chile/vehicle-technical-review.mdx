# 🇨🇱 Chili - Contrôle technique véhicule

Utilisez cet endpoint lorsque vous avez besoin des données de **contrôle technique / inspection** d’un véhicule chilien par plaque : détails du véhicule, centre d’inspection, numéro de certificat, date d’expiration, statut de révision et champs de fiche technique lorsqu’ils sont disponibles.

Endpoints véhicule Chili associés :

- **Informations véhicule (`v2/cl/vehicle`)** : données générales du registre telles que marque, modèle, RUT du titulaire, moteur, châssis, amendes et indicateurs de vol.
- **Contrôle technique véhicule (`v3/cl/vehicle`)** : statut d’inspection, historique et données de fiche technique lorsqu’elles sont disponibles.
- **SOAP véhicule (`v2/cl/vehicle-soap`)** : certificat d’assurance obligatoire accidents corporels par plaque et numéro de police.

### Endpoint

```text
GET https://api.verifik.co/v3/cl/vehicle
```

Renvoie les données de contrôle technique chiliennes pour une plaque, y compris les attributs du véhicule, l’historique d’inspection et un `technicalSheet` optionnel (`engine`, `performance`, `dimensions`) lorsqu’il est disponible. Utile pour la conformité de flottes, l’assurance, les sinistres ou l’onboarding véhicule.

### Headers

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parameters

| Name | Type | Required | Description | Example |
| --- | --- | --- | --- | --- |
| `plate` | string | **Yes** | Plaque chilienne au format pris en charge par v3, par exemple quatre lettres et deux chiffres, ou deux lettres et quatre chiffres. | `XH6640` |

### Request

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v3/cl/vehicle", {
	params: { plate: "XH6640" },
	headers: {
		Accept: "application/json",
		Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
	},
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os, requests

url = "https://api.verifik.co/v3/cl/vehicle"

r = requests.get(url, headers=headers, params=params)
print(r.json())
```

### Notes

- `technicalSheet` peut être `null` ou partiellement renseigné lorsque les données techniques correspondantes ne sont pas disponibles.
- `XH6640`, `FHDJ31` et `DCCH18` renvoient des profils stables de contrôle technique.
- `BB985` renvoie `409 Invalid_plate_format` car le format de plaque v3 n’est pas respecté.
