# 🇨🇱 Chili - Contrôle technique véhicule

Utilisez cet endpoint pour obtenir le **contrôle technique / historique d’inspection** d’un véhicule chilien par plaque. Il complète `v2/cl/vehicle` (informations générales du registre) et `v2/cl/vehicle-soap` (assurance obligatoire SOAP).

### Endpoint

```text
GET https://api.verifik.co/v3/cl/vehicle
```

### Paramètres

| Nom | Type | Requis | Description | Exemple |
| --- | --- | --- | --- | --- |
| `plate` | string | **Oui** | Plaque chilienne compatible v3 : quatre lettres et deux chiffres, ou deux lettres et quatre chiffres. | `XH6640` |

### Requête

```javascript
const { data } = await axios.get("https://api.verifik.co/v3/cl/vehicle", {
	params: { plate: "XH6640" },
	headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});
```

### Réponse 200

```json
{
	"data": {
		"plate": "XH6640",
		"vehicle": {
			"plate": "XH6640",
			"type": "CAMIONETA (CAP. DE CARGA MENOR A 1.750 KGS.) [B] (A2)",
			"mark": "PEUGEOT",
			"model": "PARTNER TOLE 1.9",
			"year": "2004",
			"engineNumber": "10DXDM0003203",
			"chasisNumber": "60548210",
			"vin": null,
			"sealType": "SELLO VERDE"
		},
		"history": [
			{
				"date": "2026-01-12",
				"plantCode": "PRT-014",
				"plantName": "REVISIONES METROPOLITANA",
				"certificateNumber": "RT-XH6640-2026",
				"expirationDate": "2027-01-12",
				"status": "APROBADA"
			}
		]
	}
}
```

### Erreurs

- `404 NotFound`: aucun enregistrement trouvé.
- `409 Invalid_plate_format`: format de plaque non accepté.
- `409 timeout_data_source`: source externe indisponible ou délai dépassé.

### Notes

`XH6640`, `FHDJ31` et `DCCH18` renvoient des profils stables. `BB985` renvoie `409 Invalid_plate_format`.
