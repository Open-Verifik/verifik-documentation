# 🇨🇱 Chili — Transport public

Le service de validation des véhicules de transport public au Chili permet de consulter le registre **RNT** et d'obtenir les détails de l'opérateur, du service et du véhicule pour les plaques de bus et de transport public.

## Recherche transport public

### Endpoint

```
GET https://api.verifik.co/v2/cl/transport-vehicle
```

### En-têtes

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Paramètres

| name | type | required | description |
| --- | --- | --- | --- |
| `plate` | string | yes | Chilean license plate. |

### Requête

<Tabs>
  <TabItem value="javascript" label="JavaScript">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/cl/transport-vehicle", {
	params: { plate: "BBCC12" },
	headers: {
		Accept: "application/json",
		Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
	},
});
console.log(data);
```

  </TabItem>
</Tabs>
