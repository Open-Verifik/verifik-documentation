# Pérou — Informations véhicule

Le service Informations véhicule du Pérou renvoie les détails enregistrés pour une plaque (**`plate`**) : marque, modèle, année, identifiants moteur et châssis, statut, propriétaires lorsqu’ils sont disponibles, et champs optionnels de fiche technique.

### Endpoint

```
GET https://api.verifik.co/v3/pe/vehiculo/placa
```

Interrogez un véhicule péruvien par **`plate`** (sans espaces ni points). La réponse peut inclure `brand`, `model`, `year`, `engineSerial`, `chasisSerial`, `vin`, `serial`, `color`, `status`, `owners` et `technicalSheet` (`engine`, `performance`, `dimensions`) lorsqu’ils sont disponibles. Utile pour la **vérification**, l’**assurance**, les **flottes** et la **conformité**.

### Headers

| Name          | Value              |
| ------------- | ------------------ |
| Accept | `application/json` |
| Authorization | `Bearer <token>`   |

### Parameters

| Name    | Type   | Required | Description | Example |
| ------- | ------ | -------- | ----------- | ------- |
| `plate` | string | **Yes**  | Numéro de plaque à consulter, sans espaces ni points. | `ABC123` |

### Request

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from 'axios';

const { data } = await axios.get('https://api.verifik.co/v3/pe/vehiculo/placa', {
  params: { plate: 'ABC123' },
  headers: {
    Accept: 'application/json',
    Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
  },
});
console.log(data);
```

  <TabItem value="python" label="Python">

```python
import os
import requests

url = "https://api.verifik.co/v3/pe/vehiculo/placa"

response = requests.get(url, headers=headers, params=params)
print(response.json())
```

### Notes

- `technicalSheet` peut être `null` ou partiellement renseigné lorsque les données techniques correspondantes ne sont pas disponibles.
- Les plaques sandbox telles que `ABC10001`–`ABC10010` renvoient des profils de démo stables.
