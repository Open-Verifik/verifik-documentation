# Buenos Aires — Infractions routières

### Point d'accès

```
GET https://api.verifik.co/v2/ar/buenos-aires/traffic-infractions
```

Utilisez ce point d'accès pour vérifier si une plaque présente des infractions routières à Buenos Aires. La réponse inclut la plaque, le total des enregistrements, les numéros de dossier et d'acte, le statut, les montants, l'autorité émettrice, les dates et les détails d'infraction lorsque des enregistrements sont trouvés.

### En-têtes

| Name          | Value              |
| ------------- | ------------------ |
| Accept        | `application/json` |
| Authorization | `Bearer <token>`   |

### Paramètres

| Nom | Type | Requis | Description |
| ---- | ---- | -------- | ----------- |
| `plate` | string | **Yes** | Plaque à interroger, sans espaces ni ponctuation. Exemple : `ABC10001`. |

### Champs de réponse

| Champ | Type | Description |
| ----- | ---- | ----------- |
| `plate` | string | Plaque normalisée. |
| `totalRecords` | string | Nombre d'enregistrements renvoyés par la source. |
| `infractions` | object[] | Enregistrements d'infractions associés à la plaque. |
| `infractions[].caseNumber` | string \| null | Numéro de dossier. |
| `infractions[].actNumber` | string \| null | Numéro d'acte. |
| `infractions[].status` | string \| null | Statut public du dossier. |
| `infractions[].isExpired` | boolean \| null | Indique si l'enregistrement est expiré. |
| `infractions[].totalAmount` | string \| null | Montant total renvoyé par la source. |
| `infractions[].details` | object[] | Articles et descriptions de l'infraction. |

### Remarques

- Ce point d'accès est limité aux infractions routières de Buenos Aires.
- `infractions` peut être un tableau vide lorsque la plaque n'a pas d'infractions renvoyées.
