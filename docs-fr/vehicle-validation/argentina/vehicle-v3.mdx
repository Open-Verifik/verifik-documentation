# 🇦🇷 Argentine — Véhicule v3

### Point d'accès

```
GET https://api.verifik.co/v3/ar/vehicle
```

Utilisez ce point d'accès pour récupérer les détails d'un véhicule argentin à partir de la plaque. La réponse inclut des identifiants normalisés tels que le numéro de moteur, le châssis, la marque, le modèle, l'année, la couleur, le type d'unité, le statut de police active et les champs de fiche technique lorsqu'ils sont disponibles.

### En-têtes

| Name          | Value              |
| ------------- | ------------------ |
| Accept        | `application/json` |
| Authorization | `Bearer <token>`   |

### Paramètres

| Nom | Type | Requis | Description |
| ---- | ---- | -------- | ----------- |
| `plate` | string | **Yes** | Plaque d'immatriculation à interroger, sans espaces ni ponctuation. Exemple : `ABC10001`. |

### Requête

<Tabs>

```javascript
import axios from 'axios';
```

### Champs de réponse

| Champ | Type | Description |
| ----- | ---- | ----------- |
| `plate` | string | Plaque normalisée. |
| `engineNumber` | string \| null | Numéro de moteur renvoyé par la source. |
| `chassis` | string \| null | Identifiant de châssis renvoyé par la source. |
| `year` | string \| null | Année du véhicule. |
| `brand` | string \| null | Marque du véhicule. |
| `modelGroup` | string \| null | Famille ou groupe de modèle. |
| `model` | string \| null | Libellé modèle/version. |
| `color` | string \| null | Couleur enregistrée. |
| `type` | string \| null | Type d'unité du véhicule. |
| `isActivePolicyFound` | boolean | Indique si une police active a été trouvée. |
| `technicalSheet` | object \| null | Données de fiche technique lorsqu'elles sont disponibles. |

### Remarques

- `technicalSheet` peut être `null` ou partiellement renseigné lorsque les données techniques correspondantes ne sont pas disponibles.
- Le service normalise la casse de la plaque avant d'interroger la source.
