# 🇦🇷 Argentine — Véhicule v3

### Endpoint

```
GET https://api.verifik.co/v3/ar/vehicle
```

Utilisez cet endpoint pour récupérer les détails d'un véhicule argentin à partir de la plaque. La réponse inclut des identifiants normalisés tels que le numéro de moteur, le châssis, la marque, le modèle, l'année, la couleur, le type d'unité, le statut de police active et les champs de fiche technique lorsqu'ils sont disponibles.

### Headers

| Name          | Value              |
| ------------- | ------------------ |
| Accept        | `application/json` |
| Authorization | `Bearer <token>`   |

### Parameters

| Name | Type | Required | Description |
| ---- | ---- | -------- | ----------- |
| `plate` | string | **Yes** | Format Mercosur `AA-123-BB` (envoyez `AA123BB`, sans espaces ni ponctuation). |

### Request

<Tabs>
  <TabItem value="javascript" label="JavaScript">

```javascript
import axios from 'axios';

const { data } = await axios.get('https://api.verifik.co/v3/ar/vehicle', {

  headers: {
    Accept: 'application/json',
    Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
  },
});

console.log(data);
```

### Response fields

| Field | Type | Description |
| ----- | ---- | ----------- |
| `plate` | string | Plaque normalisée. |
| `engineNumber` | string \| null | Numéro de moteur lorsqu'il est disponible. |
| `chassis` | string \| null | Identifiant de châssis lorsqu'il est disponible. |
| `year` | string \| null | Année du véhicule. |
| `brand` | string \| null | Marque du véhicule. |
| `modelGroup` | string \| null | Famille ou groupe de modèle. |
| `model` | string \| null | Libellé modèle/version. |
| `color` | string \| null | Couleur enregistrée. |
| `type` | string \| null | Type d'unité du véhicule. |
| `isActivePolicyFound` | boolean | Indique si une police active a été trouvée. |
| `technicalSheet` | object \| null | Données de fiche technique lorsqu'elles sont disponibles. |

### Notes

- `technicalSheet` peut être `null` ou partiellement renseigné lorsque les données techniques correspondantes ne sont pas disponibles.
- Le service normalise la casse de la plaque avant la consultation.
