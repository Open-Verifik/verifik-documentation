# Buenos Aires — Contrôle technique

### Point d'accès

```
GET https://api.verifik.co/v2/ar/buenos-aires/technical-inspection
```

Utilisez ce point d'accès pour consulter l'historique de contrôle technique à Buenos Aires par plaque. La réponse renvoie des enregistrements avec identifiants de vérification, dates de vérification et d'expiration, type de résultat, numéro de vignette, commentaires, marque et modèle du véhicule, et nom du centre de contrôle.

### En-têtes

| Name          | Value              |
| ------------- | ------------------ |
| Accept        | `application/json` |
| Authorization | `Bearer <token>`   |

### Paramètres

| Nom | Type | Requis | Description |
| ---- | ---- | -------- | ----------- |
| `plate` | string | **Yes** | Format Mercosur `AA-123-BB` (envoyez `AA123BB`, sans espaces ni ponctuation). |

### Champs de réponse

| Champ | Type | Description |
| ----- | ---- | ----------- |
| `plate` | string | Plaque normalisée. |
| `totalRecords` | string | Nombre d'enregistrements de contrôle technique renvoyés. |
| `inspections` | object[] | Enregistrements de contrôle technique. |
| `inspections[].verificationId` | string \| null | Identifiant de vérification. |
| `inspections[].verificationDate` | string \| null | Date de vérification du contrôle. |
| `inspections[].resultTypeId` | string \| null | Identifiant du type de résultat renvoyé par la source. |
| `inspections[].stickerNumber` | string \| null | Numéro de vignette de contrôle. |
| `inspections[].expirationDate` | string \| null | Date d'expiration du contrôle. |
| `inspections[].brand` | string \| null | Marque du véhicule. |
| `inspections[].model` | string \| null | Modèle du véhicule. |
| `inspections[].plantName` | string \| null | Nom du centre de contrôle. |

### Remarques

- Ce point d'accès est limité aux contrôles techniques de Buenos Aires.
- Les champs de date sont normalisés lorsque la source renvoie une valeur jour/mois/année reconnaissable.
