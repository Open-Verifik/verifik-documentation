# Chili — Informations véhicule

Le service de validation véhicule au Chili permet d'obtenir des informations détaillées sur les véhicules immatriculés dans le pays. Utile pour **flottes**, **assurance**, **garanties** et contrôles opérationnels.

## Consultation par plaque

### Endpoint

```
https://api.verifik.co/v2/cl/vehicle
```

Interrogez par **plaque** (`plate`, sans espaces ni points). Les champs renvoyés incluent en général **marque**, **modèle**, **année**, **châssis/VIN**, **moteur**, **couleur**, **type**, **résumé d'amendes** et **RUT du propriétaire** lorsque le registre les expose.

### Headers

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parameters

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `plate` | string | Yes | Plaque sans espaces ni points. |

### Request

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/cl/vehicle", {
  params: { plate: "BB985" },
  headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});
console.log(data);
```

## Fonctionnalités

### Informations véhicule
- **Immatriculation** : vérification au Chili.
- **Spécifications** : châssis, moteur, constructeur, modèle.
- **Détails** : année, couleur, type.
- **Propriétaire** : RUT lorsque disponible.
- **Amendes** : état des sanctions liées au véhicule.
- **Vol** : lorsque la source l'indique.
- **Transport public** : classification le cas échéant.

### Général
- Exemples **Node.js** et **Python**.
- Données **temps réel** selon la source.
- **Erreurs** structurées (404, 409, etc.).
- Réponses **JSON** avec `signature` lorsque fourni.
