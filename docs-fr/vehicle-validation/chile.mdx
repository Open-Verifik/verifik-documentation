# 🇨🇱 Chili — Informations véhicule

Le service de validation de véhicules chiliens fournit des capacités de vérification complètes pour les informations de véhicules. Ce service vous permet d'obtenir des informations détaillées sur les véhicules immatriculés au Chili.

Endpoints véhicule Chili associés :

- **Informations véhicule (`v2/cl/vehicle`)** : données générales du registre véhicule.
- **Contrôle technique véhicule (`v3/cl/vehicle`)** : statut et historique d’inspection. Voir [Chili - Contrôle technique véhicule](/verifik-fr/vehicle-validation/chile/vehicle-technical-review).
- **SOAP véhicule (`v2/cl/vehicle-soap`)** : certificat d’assurance obligatoire. Voir [Chili - SOAP véhicule](/verifik-fr/vehicle-validation/chile/vehicle-soap).

## Recherche d'informations véhicule

### Endpoint

```
https://api.verifik.co/v2/cl/vehicle
```

Interrogation par **numéro de plaque** (`plate`, sans espaces ni points). Les champs typiques incluent **marque**, **modèle**, **année**, **VIN/châssis**, **moteur**, **couleur**, **type de véhicule**, **récapitulatif des amendes** et **RUT du propriétaire** lorsque le registre les expose — utile pour l'**onboarding de flotte**, les **vérifications de garanties** et les flux de **sinistres**.

### En-têtes (Headers)

| Nom            | Valeur              |
| -------------- | ------------------- |
| Accept         | `application/json`  |
| Authorization  | `Bearer <token>`    |

### Paramètres

| Nom     | Type   | Requis  | Description                                    | Exemple   |
| ------- | ------ | ------- | ---------------------------------------------- | --------- |
| `plate` | string | **Oui** | Format actuel `BB-CC-12` (envoyez `BBCC12`, quatre lettres + deux chiffres, sans espaces ni points). | `BBCC12` |

## Fonctionnalités

### Recherche d'informations véhicule
- **Immatriculation du véhicule** : confirmez l'immatriculation au Chili.
- **Caractéristiques techniques** : numéro de châssis, numéro de moteur, fabricant, modèle.
- **Détails du véhicule** : accédez à l'année, la couleur et le type.
- **Informations propriétaire** : récupérez le RUT et les coordonnées du propriétaire.
- **Statut des amendes** : vérifiez les amendes en cours associées au véhicule.
- **Statut de vol** : vérifiez si le véhicule fait l'objet d'un avis de vol.
- **Transport public** : consultez la classification de transport public.

### Fonctionnalités générales
- **Plusieurs langages de programmation** : exemples en JavaScript, Python, PHP et Swift.
- **Données en temps réel** : informations à jour.
- **Gestion d'erreurs complète** : réponses d'erreur détaillées pour chaque scénario.
- **Réponses structurées** : JSON bien formaté avec signature de vérification.
- **Intégration au registre chilien** : accès direct à la base officielle des véhicules au Chili.
