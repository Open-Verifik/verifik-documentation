# Mexique

### Endpoint

```
https://api.verifik.co/v2/mx/vehiculo/placa
```

Le service Informations véhicule fournit des informations détaillées sur un véhicule au Mexique à partir de son numéro de plaque d'immatriculation. Ce service retourne notamment la marque, le modèle, l'année, le VIN et d'autres détails associés. Transmettez la plaque via le paramètre `plate` sur une requête GET ; les réponses contiennent des champs orientés enregistrement lorsqu'une correspondance est trouvée.

### En-têtes (Headers)

| Nom            | Valeur              |
| -------------- | ------------------- |
| Content-Type   | `application/json`  |
| Authorization  | `Bearer <token>`    |

### Paramètres

| Nom    | Type   | Requis  | Description                           | Exemple   |
| ------ | ------ | ------- | ------------------------------------- | --------- |
| plate  | string | **Oui** | Plaque d'immatriculation à interroger.| `ABC1234` |

### Requête (Request)

```javascript
import axios from 'axios';

const options = {
  method: 'GET',
  url: 'https://api.verifik.co/v2/mx/vehiculo/placa',
  params: {plate: 'ABC1234'},
  headers: {
    Accept: 'application/json',
    Authorization: 'Bearer <votre_token>'
  }
};

try {
  const { data } = await axios.request(options);
  console.log(data);
} catch (error) {
  console.error(error);
}
```

### Réponse (Response)

```json
{
  "success": true,
  "data": {
    "plate": "ABC1234",
    "make": "Nissan",
    "model": "Sentra",
    "year": "2021",
    "engine": "2.0L",
    "transmission": "CVT",
    "fuelType": "Gasoline",
    "vin": "1N4AL3AP8JC123456",
    "registrationDate": "10/08/2021",
    "status": "active",
    "owner": "Carlos Rodríguez",
    "state": "Jalisco",
    "city": "Guadalajara"
  }
}
```

### Réponses d'erreur

```json
{
  "success": false,
  "error": "Vehicle not found",
  "code": "VEHICLE_NOT_FOUND"
}
```

## Cas d'usage

- **Vérification d'assurance** : confirmez les détails d'un véhicule à des fins d'assurance.
- **Concessionnaires** : validez les informations d'un véhicule avant achat.
- **Gestion de flotte** : suivez les informations des véhicules de l'entreprise.
- **Forces de l'ordre** : vérifiez l'immatriculation et la propriété d'un véhicule.
- **Diligence raisonnable** : confirmez les informations d'un véhicule pour des transactions.
