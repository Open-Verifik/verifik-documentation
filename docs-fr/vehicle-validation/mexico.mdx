# Mexique

### Endpoint

```
https://api.verifik.co/v2/mx/vehiculo/placa
```

Le service d'informations véhicule fournit des détails sur un véhicule au Mexique à partir de sa plaque d'immatriculation : marque, modèle, année, VIN et autres données associées. Indiquez la plaque dans le paramètre de requête `plate` en GET ; la réponse contient les champs liés à l'immatriculation lorsqu'une correspondance est trouvée.

### En-têtes

| Nom          | Valeur              |
| ------------ | ------------------- |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Paramètres

| Nom   | Type   | Obligatoire | Description                    | Exemple   |
| ----- | ------ | ----------- | ------------------------------ | --------- |
| plate | String | Oui         | Plaque du véhicule à consulter. | `ABC1234` |

### Requête

```javascript
import axios from 'axios';

const options = {
  method: 'GET',
  url: 'https://api.verifik.co/v2/mx/vehiculo/placa',
  params: {plate: 'ABC1234'},
  headers: {
    Accept: 'application/json',
    Authorization: 'Bearer <your_token>'
  }
};

try {
  const { data } = await axios.request(options);
  console.log(data);
} catch (error) {
  console.error(error);
}
```

### Réponse

```json
{
  "success": true,
  "data": {
    "plate": "ABC1234",
    "make": "Nissan",
    "model": "Sentra",
    "year": "2021",
    "engine": "2.0L",
    "transmission": "CVT",
    "fuelType": "Gasoline",
    "vin": "1N4AL3AP8JC123456",
    "registrationDate": "10/08/2021",
    "status": "active",
    "owner": "Carlos Rodríguez",
    "state": "Jalisco",
    "city": "Guadalajara"
  }
}
```

### Réponses d'erreur

```json
{
  "success": false,
  "error": "Vehicle not found",
  "code": "VEHICLE_NOT_FOUND"
}
```

## Cas d'usage

- **Assurance** : vérifier les détails véhicule pour une police
- **Concessionnaires** : valider les informations avant achat
- **Flottes** : suivre les véhicules d'entreprise
- **Forces de l'ordre** : vérifier immatriculation et titulaire
- **Due diligence** : contrôler les informations pour une transaction
