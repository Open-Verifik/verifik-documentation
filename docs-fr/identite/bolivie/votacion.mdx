# Bolivie - Lieu de vote

Verifik interroge **Yo Participo (OEP)** avec la **CI** et la **date de naissance**. Utilisez ce point de terminaison pour la **vérification électorale**, l'**intégration des électeurs** et la **conformité** lorsque les données OEP sont autorisées.

Retourne les détails du lieu de vote lorsque la source les fournit :

- **`documentType`** — valeur autorisée : **CI**
- **`documentNumber`** — numéro de carte d'identité bolivienne (**5–13** caractères)
- **`dateOfBirth`** — date de naissance au format **JJ/MM/AAAA** (doit correspondre aux registres OEP)
- **`fullName`** — nom complet du citoyen dans le registre électoral
- **`votingStatus`** — indique si le citoyen est habilité à voter
- **`department`**, **`locality`**, **`pollingPlace`**, **`pollingTable`** — champs de localisation électorale
- **`election`** — libellé d'élection ou note de mise à jour du registre
- **Champs optionnels** — `isJury`, `jurySortDate`, `politicalMembership`, `latitude`, `longitude` lorsque la source les inclut
- **`force`** — optionnel ; `true` ignore le cache
- Réponse **signée** par Verifik

### Endpoint

```
GET v2/bo/votacion
```

### Headers

| Header | Valeur | Description |
| --- | --- | --- |
| Accept | application/json | Format de réponse |
| Authorization | Bearer \<token\> | Votre JWT Verifik |

### Paramètres

| Champ | Type | Requis | Description |
| --- | --- | --- | --- |
| documentType | String | Oui | Type de document. Valeur autorisée : `CI`. |
| documentNumber | String | Oui | Numéro de CI sans code d'expédition (**5–13** caractères). |
| dateOfBirth | String | Oui | Date de naissance au format `JJ/MM/AAAA`. |
| force | String | Non | Définir sur `true` pour ignorer le cache et interroger la source. |

### Notes

- **`documentType`** doit être **`CI`** ; **`dateOfBirth`** est obligatoire et doit correspondre aux registres OEP.
- Les champs optionnels de jury, militance et coordonnées n'apparaissent que si la source les inclut.
- Utilisez **`force=true`** pour ignorer les enregistrements en cache MongoDB.
- Traitez les réponses comme des **données personnelles sensibles** ; respectez les lois applicables en matière de confidentialité.
