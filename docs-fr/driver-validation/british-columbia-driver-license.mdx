# Colombie-Britannique — Permis de conduire

## Endpoint

```
https://api.verifik.co/v2/ca/british-columbia/driver-license 
```

Le service de vérification du permis de conduire de la Colombie-Britannique offre une méthode simple pour confirmer l'authenticité et la validité d'un permis de conduire dans la province de Colombie-Britannique, au Canada. Grâce à ce service, vous pouvez valider rapidement un permis de conduire à partir du numéro de document et du nom de famille fournis.

### En-têtes (Headers)

| Nom            | Valeur              |
| -------------- | ------------------- |
| Content-Type   | `application/json`  |
| Authorization  | `Bearer <token>`    |

### Paramètres

| Nom              | Type   | Requis  | Description                                                                                            | Exemple   |
| ---------------- | ------ | ------- | ------------------------------------------------------------------------------------------------------ | --------- |
| `documentNumber` | string | **Oui** | Permis de conduire à interroger ; les données doivent être saisies exactement telles qu'elles figurent sur le document. | `1123456` |
| `lastName`       | string | **Oui** | Nom de famille tel qu'il apparaît sur le permis.                                                       | `HELLO`   |
