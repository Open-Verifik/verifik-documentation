# 🇵🇪 Pérou — Permis de conduire

Ce service permet d'obtenir des informations détaillées sur le permis de conduire d'une personne au Pérou à partir de son **DNI** (*Documento Nacional de Identidad*). La réponse inclut les permis actifs, l'historique des procédures, les cours réglementaires, les restrictions, les sanctions et d'autres indicateurs—utile pour la **flotte**, les **ressources humaines** et le **KYC** lorsque les habilitations de conduite comptent.

### Endpoint

```
GET https://api.verifik.co/v2/pe/driver-license
```

Interrogez avec `documentType=DNI` et un `documentNumber` à 8 chiffres. Les champs retournés incluent le **nom complet**, les **catégories** (A, AIV, B), les **permis**, les **procédures**, les **cours réglementaires**, les **sanctions**, les **contraventions**, etc., selon le registre.

### En-têtes

| Name          | Value              |
| ------------- | ------------------ |
| Accept        | `application/json` |
| Authorization | `Bearer <token>`   |

### Paramètres

| Nom | Type | Requis | Description | Exemple |
| ---- | ---- | ------ | ----------- | ------- |
| `documentType` | string | **Oui** | Type de document. Valeur autorisée : `DNI`. | `DNI` |
| `documentNumber` | string | **Oui** | DNI de la personne, sans espaces ni points. | `44141274` |

### Notes

- `documentNumber` doit correspondre à un DNI avec enregistrements de permis dans la source officielle.

## Fonctionnalités

- **Recherche par DNI** : `documentType=DNI` et numéro de document.
- **Catégories de permis** : Indicateurs pour A, AIV, B et permis provisoires.
- **Permis actifs** : Numéro, catégorie, statut, restrictions, centre d'émission et expiration.
- **Historique des procédures** : Revalidations et dates avec statut (ex. Vigente, VENCIDA).
- **Cours réglementaires** : Détails d'inscription et de fin lorsque disponibles.
- **Sanctions et contraventions** : Tableaux optionnels selon le registre.
- **Réponses structurées** : JSON avec signature et certification Verifik.
