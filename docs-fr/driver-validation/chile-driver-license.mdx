# Chili — Permis de conduire

Ce service renvoie des informations détaillées sur un **permis de conduire** chilien à partir du **numéro de licence**. Vous obtenez notamment **RUT**, adresse, **classe**, **dates de contrôle**, commune, **restrictions** et autres champs structurés selon le registre des véhicules à moteur.

Pensé pour **flottes**, **RH**, **mobilité** et tout flux où l'habilitation doit être vérifiée.

### Endpoint

```
https://api.verifik.co/v2/cl/driver-license
```

Recherche par **numéro de permis** (`documentNumber`) : en général **RUT**, **classe**, **restrictions**, **dates de contrôle**, **commune** et **adresse**, selon disponibilité.

### Headers

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parameters

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `documentNumber` | string | Yes | Numéro de permis sans espaces. |

### Request

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/cl/driver-license", {
  params: { documentNumber: "12345678" },
  headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});
console.log(data);
```

## Fonctionnalités

- **RUT et conducteur** : noms, adresse, commune.
- **Classe et restrictions** : catégories A, B, C, etc.
- **Dates de contrôle** : contrôle périodique et dernier contrôle.
- **Procédures** : duplicata, renouvellement, etc., si renvoyé par la source.
- **API REST** avec JSON structuré et gestion d'erreurs.
