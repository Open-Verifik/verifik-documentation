# Vote

## Informations de vote pour les citoyens colombiens

### Endpoint

```
https://api.verifik.co/v2/co/registraduria/votacion
```

Le service d'informations de vote permet de récupérer les détails de vote auprès de la « Registraduría Nacional del Estado Civil » pour les citoyens colombiens. En fournissant le numéro de document du citoyen, vous pouvez accéder à des informations telles que l'adresse, le département, la municipalité, la table de vote et le bureau de vote.

Ce service est utile pour vérifier l'inscription électorale et faciliter le processus de vote.

### En-têtes (Headers)

| Nom           | Valeur             |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Paramètres de requête

| Nom            | Type   | Requis ? | Description                                                         | Exemple      |
| -------------- | ------ | -------- | ------------------------------------------------------------------- | ------------ |
| documentNumber | String | Oui      | Numéro de document de la personne, sans espaces ni points.        | `123456789`  |

### Requête

```javascript
import axios from 'axios';

const options = {
  method: 'GET',
  url: 'https://api.verifik.co/v2/co/registraduria/votacion',
  params: {documentNumber: '123456789'},
  headers: {
    Accept: 'application/json',
    Authorization: 'Bearer <your_token>'
  }
};

try {
  const { data } = await axios.request(options);
  console.log(data);
} catch (error) {
  console.error(error);
}
```

### Réponse

```json
{
  "success": true,
  "data": {
    "documentNumber": "123456789",
    "firstName": "María",
    "lastName": "González",
    "fullName": "María González",
    "address": "Calle 123 #45-67",
    "department": "Cundinamarca",
    "municipality": "Bogotá D.C.",
    "pollingTable": "123",
    "votingStation": "Colegio San José",
    "votingStationAddress": "Carrera 7 #32-16",
    "status": "active"
  }
}
```

### Réponses d'erreur

```json
{
  "success": false,
  "error": "Voter not found",
  "code": "VOTER_NOT_FOUND"
}
```

## Cas d'usage

- **Vérification d'inscription électorale** : confirmer le statut d'inscription
- **Support le jour du scrutin** : aider les électeurs à trouver leur bureau
- **Analyse démographique** : analyser la répartition des électeurs
- **Suivi électoral** : suivre la participation et l'inscription
- **Engagement civique** : soutenir les processus démocratiques et l'éducation électorale
