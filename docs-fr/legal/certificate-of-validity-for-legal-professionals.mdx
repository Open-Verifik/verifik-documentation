### Point d'accès

```
GET https://api.verifik.co/v2/co/rama/certificado/vigencia
```

Vérifie le **certificat et la validité** des professionnels du droit en Colombie. Envoyez **`documentType`** (**`CC`** ou **`CE`**), **`documentNumber`** et **`quality`** (par ex. `ABG`, `JUEZPAZ`, `LT`) en **query**. Le champ **`encalidad`** reprend la graphie de l’API.

### En-têtes

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Paramètres

| name | type | required | description |
| --- | --- | --- | --- |
| `documentType` | string | yes | Uniquement `CC` ou `CE`. |
| `documentNumber` | string | yes | Numéro de document à interroger. |
| `quality` | string | yes | Code de catégorie (ex. `ABG`, `JUEZPAZ`, `LT`). |

### Requête

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/rama/certificado/vigencia", {
	params: { documentType: "CC", documentNumber: "123456789", quality: "ABG" },
	headers: {
		Accept: "application/json",
		Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
	},
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os, requests

url = "https://api.verifik.co/v2/co/rama/certificado/vigencia"
headers = {"Accept": "application/json", "Authorization": f"Bearer {os.getenv('VERIFIK_TOKEN')}"}
params = {"documentType": "CC", "documentNumber": "123456789", "quality": "ABG"}
r = requests.get(url, headers=headers, params=params)
print(r.json())
```
