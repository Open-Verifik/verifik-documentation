# Vérification d'entreprise au Costa Rica (NITE)

Consultez les entités enregistrées avec le **NITE** pour l'onboarding fournisseurs et la due diligence.

## Référence API

### Endpoint

```
https://api.verifik.co/v2/cr/company
```

Passez **`documentType`**=`NITE` et le **`documentNumber`** de l'entité ; la réponse contient les identifiants disponibles dans la source.

### En-têtes

| Nom | Valeur |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Paramètres

| Nom | Type | Obligatoire | Description |
| --- | --- | --- | --- |
| `documentType` | string | Oui | Valeur autorisée : `NITE`. |
| `documentNumber` | string | Oui | Identifiant d'entreprise (NITE). |

### Requête

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/cr/company", {
  params: { documentType: "NITE", documentNumber: "3101122876" },
  headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});
console.log(data);
```
