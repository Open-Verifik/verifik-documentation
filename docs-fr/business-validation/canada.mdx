# Vérification d'entreprise au Canada

L'API de vérification d'entreprise de Verifik vous permet d'authentifier les sociétés canadiennes à partir des données officielles du gouvernement. Elle est conçue pour fluidifier vos processus KYB (Know Your Business), prévenir la fraude et vous assurer de respecter sans effort toutes les exigences réglementaires.

Nous avons construit cette intégration pour les entreprises qui ont besoin d'un moyen rapide, sécurisé et automatisé de confirmer la légitimité de sociétés, partenaires ou fournisseurs.

## Que valide cette API ?

Notre API se connecte directement aux registres officiels pour valider :

-   **Nom et numéro d'entreprise** : prend en charge la recherche au registre canadien des sociétés par nom et par province.
-   **Informations sur l'entreprise** : retourne le nom, le numéro d'entreprise, le type, le statut, les données d'enregistrement et l'adresse.
-   **Correspondance d'entreprise** : confirme que les informations correspondent au registre des sociétés.

En vérifiant ces éléments, vous pouvez avoir la certitude que la société avec laquelle vous traitez est légitime et dûment enregistrée, ce qui réduit considérablement les risques de fraude et garantit votre conformité.

## Référence de l'API

### Endpoint

```
https://api.verifik.co/v2/ca/company
```

### En-têtes (Headers)

| Nom            | Valeur              |
| -------------- | ------------------- |
| Content-Type   | `application/json`  |
| Authorization  | `Bearer <token>`    |

### Paramètres

| Nom        | Type   | Requis  | Description                       |
| ---------- | ------ | ------- | --------------------------------- |
| `business` | string | **Oui** | Nom de la société.                |
| `province` | string | **Oui** | Code de province canadienne.      |

-   **Services financiers** : authentifiez les entreprises avant l'ouverture d'un compte ou le traitement de transactions.
-   **Conformité et diligence raisonnable** : validez les informations d'entreprise dans le cadre de vos flux KYC/KYB.
-   **E-commerce et places de marché** : confirmez la légitimité d'une société avant de l'autoriser à vendre sur votre plateforme.
