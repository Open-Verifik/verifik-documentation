# 🇨🇴 Colombie — RUES dossier complet (v3)

Cet endpoint renvoie un **dossier complet** depuis le registre **RUES** (*Registro Único Empresarial y Social*) pour un **NIT** et une **ligne de registre** (`category`). Pour un **résumé** plus léger, utilisez **`GET /v3/co/rues`** (documentation en anglais : [Colombia RUES (v3)](https://docs.verifik.co/business-validation/colombia-rues-v3)).

## Référence API

### Endpoint

```
GET https://api.verifik.co/v3/co/rues-complete
```

Même chemin sur l’application : `GET https://verifik.app/v3/co/rues-complete`

### En-têtes

| Nom | Valeur |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Paramètres

| Nom | Type | Requis | Description |
| --- | --- | --- | --- |
| `documentType` | string | Oui | Doit être **`NIT`**. |
| `documentNumber` | string | Oui | NIF ; les caractères non alphanumériques sont supprimés côté serveur. |
| `category` | string | Non | **`RM`**, **`PROP`**, **`RUNEOL`**, **`RNT`**, **`ESAL`**, **`ESOL`**, **`JUEGOS`**, **`EXTRANJERAS`**. Par défaut **`RM`**. |

### Notes

- Seuls les paramètres ci-dessus sont acceptés. **`municipality`**, **`chamber`** et **`page`** ne font **pas** partie de **`GET /v3/co/rues-complete`**.
- Exemples de réponses détaillées : [RUES Complete Business Validation (v3)](https://docs.verifik.co/business-validation/rues-complete-v3) (EN).
