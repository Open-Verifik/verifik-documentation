# Vérification d'entreprise au Paraguay

L'API Verifik permet d'authentifier les **entreprises paraguayennes** à partir de données officielles. Elle simplifie le **KYB**, réduit la fraude et aide à la conformité.

Cette intégration s'adresse aux entreprises qui doivent confirmer rapidement la **légitimité** de sociétés, partenaires ou fournisseurs.

## Que valide cette API ?

Notre API interroge les registres officiels pour vérifier :

-   **Raison sociale et RUC** : prise en charge du *Registro Único de Contribuyente* (RUC).
-   **Informations d'entreprise** : nom et détails d'immatriculation.
-   **Cohérence** : confirme que les informations correspondent au RUC.

## Référence API

### Endpoint

```
https://api.verifik.co/v2/py/company
```

### Headers

| Name          | Value              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parameters

| Name           | Type    | Required | Description                                    |
| -------------- | ------- | -------- | ---------------------------------------------- |
| `documentType` | string  | Yes      | Type de document. Valeur autorisée : `RUC`. |
| `documentNumber` | string | Yes      | Identifiant de l'entreprise. |
