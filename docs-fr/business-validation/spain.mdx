# Vérification d'entreprise en Espagne

L'API Verifik permet d'authentifier les entreprises espagnoles à partir de sources officielles. Elle simplifie le KYB et la conformité.

Destinée aux équipes qui doivent valider partenaires, fournisseurs ou contreparties.

## Que valide cette API ?

Notre API interroge les registres officiels pour vérifier :

-   **Identifiant d'entreprise** : CIF/NIF et informations associées.
-   **Données d'entreprise** : raison sociale, statut lorsque disponible.
-   **Cohérence** : les informations correspondent-elles à l'identifiant fourni ?

## Référence API

### Endpoint

```
https://api.verifik.co/v2/es/company
```

Ce point d'accès interroge le registre des entreprises espagnoles avec le paramètre `documentNumber` (identifiant société).

### En-têtes

| Nom | Valeur |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Paramètres

| Nom | Type | Obligatoire | Description |
| --- | --- | --- | --- |
| `documentNumber` | string | Oui | Identifiant d'entreprise (CIF/NIF). |
