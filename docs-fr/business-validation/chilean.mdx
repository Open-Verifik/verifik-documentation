# Chili — Validation d'entreprise (RUT)

L'API Verifik permet d'authentifier les entreprises chiliennes à partir de données gouvernementales officielles. Elle facilite le **KYB**, réduit les risques de fraude et aide à respecter les obligations réglementaires.

L'intégration s'adresse aux équipes qui doivent valider rapidement sociétés, partenaires ou fournisseurs.

Avec le **type et le numéro de document**, vous obtenez la raison sociale, les catégories et activités économiques, ainsi que des informations sur les lignes de service autorisées.

### Endpoint

```
https://api.verifik.co/v2/cl/company
```

Consultation des entreprises chiliennes par **RUT** : raison sociale, activités et données de registre pour **KYB** et diligence fournisseur.

### Headers

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parameters

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `documentType` | string | Yes | `RUT`. |
| `documentNumber` | string | Yes | RUT de la société (format selon la documentation d'intégration). |

## Que valide cette API ?

- **Raison sociale et RUT** : cohérence avec le registre officiel.
- **Activités économiques** : codes, catégories, TVA et dates lorsque la source les renvoie.

## Fiabilité

Données issues de sources gouvernementales chiliennes, avec sécurité et confidentialité.
