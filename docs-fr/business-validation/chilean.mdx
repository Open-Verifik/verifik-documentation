# 🇨🇱 Chili — Vérification d'entreprise

L'API de vérification d'entreprise de Verifik vous permet d'authentifier les sociétés chiliennes à partir des données officielles du gouvernement. Elle est conçue pour fluidifier vos processus KYB (Know Your Business), prévenir la fraude et vous assurer de respecter sans effort toutes les exigences réglementaires.

Nous avons construit cette intégration pour les entreprises qui ont besoin d'un moyen rapide, sécurisé et automatisé de confirmer la légitimité de sociétés, partenaires ou fournisseurs.

Ce service vous permet d'obtenir les informations d'une société au Chili en fournissant son type et son numéro de document. La réponse retourne le nom de la société, sa catégorie d'activité, sa sous-catégorie et son activité, ainsi que la liste des services pour lesquels elle est habilitée à opérer.

### Endpoint

```
https://api.verifik.co/v2/cl/company
```

Recherchez les sociétés chiliennes par RUT : raison sociale, catégorie d'activité, secteur et lignes de service autorisées à partir des données officielles, pour le KYB et la diligence fournisseurs.

### En-têtes (Headers)

| Nom            | Valeur              |
| -------------- | ------------------- |
| Accept         | `application/json`  |
| Authorization  | `Bearer <token>`    |

### Paramètres

| Nom              | Type   | Requis  | Description                                                                                                                                | Exemple        |
| ---------------- | ------ | ------- | ------------------------------------------------------------------------------------------------------------------------------------------ | -------------- |
| `documentType`   | string | **Oui** | Type de document. Valeur autorisée : RUT.                                                                                                  | `RUT`          |
| `documentNumber` | string | **Oui** | RUT de la société à interroger ; saisissez ce paramètre avec les points et tirets tels qu'établis par la documentation officielle.         | `33516727409`  |

## Que valide cette API ?

Notre API se connecte directement aux registres officiels pour valider :

-   **Nom de la société et numéro de RUT** : prend en charge le *Rol Único Tributario* (RUT).
-   **Informations sur l'entreprise** : retourne la raison sociale, les activités économiques, les catégories d'activité et les détails d'habilitation.
-   **Activités économiques** : fournit le détail des activités économiques avec codes, catégories, statut IVA et dates.
-   **Correspondance d'entreprise** : confirme que les informations correspondent au numéro de RUT.

En vérifiant ces éléments, vous pouvez avoir la certitude que la société avec laquelle vous traitez est légitime et dûment enregistrée, ce qui réduit considérablement les risques de fraude et garantit votre conformité.

## Cas d'usage courants

-   **Onboarding B2B** : vérifiez instantanément les sociétés lors de l'inscription d'un partenaire ou d'un fournisseur.
-   **Services financiers** : authentifiez les entreprises avant l'ouverture d'un compte ou le traitement de transactions.
-   **Conformité et diligence raisonnable** : validez les informations d'entreprise dans le cadre de vos flux KYC/KYB.
-   **E-commerce et places de marché** : confirmez la légitimité d'une société avant de l'autoriser à vendre sur votre plateforme.
