# Vérification d'entreprise au Mexique

L'API Verifik authentifie les entreprises mexicaines à partir de données gouvernementales officielles. Elle simplifie le KYB, réduit la fraude et aide à la conformité.

## Que valide cette API ?

-   **Raison sociale et numéro FME** (*Folio Mercantil Electrónico*).
-   **Informations légales** : raison sociale, ville, immatriculation.
-   **Correspondance** : les données correspondent-elles au FME ?

## Référence API

### Endpoint

```
https://api.verifik.co/v2/mx/company
```

Ce point d’accès recherche une entité juridique mexicaine par son identifiant **FME** (*Folio Mercantil Electrónico*). Utilisez GET avec `documentType=FME` et `documentNumber` pour obtenir des données d’enregistrement officielles, telles que la dénomination sociale et la ville, issues du registre du commerce gouvernemental.

### En-têtes

| Nom          | Valeur              |
| ------------ | ------------------- |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Paramètres

| Nom | Type | Obligatoire | Description |
| --- | --- | --- | --- |
| `documentType` | string | Oui | Valeur autorisée : `FME`. |
| `documentNumber` | string | Oui | Identifiant FME (ex. `N-2021007300`). |

### Requête

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/mx/company", {
```
