# Vérification d'entreprise en Argentine

L'API de vérification d'entreprise de Verifik vous permet d'authentifier les sociétés argentines à partir de données gouvernementales officielles. Elle est conçue pour fluidifier vos processus KYB (Know Your Business), prévenir la fraude et respecter sans effort toutes les exigences réglementaires.

Nous avons construit cette intégration pour les entreprises qui ont besoin de confirmer rapidement, de manière sécurisée et automatisée, la légitimité de sociétés, partenaires ou fournisseurs.

## Que valide cette API ?

Notre API se connecte directement aux registres officiels pour valider :

-   **Nom de société et numéro CUIT** : prend en charge le *Código Único de Identificación Tributaria* (CUIT).
-   **Informations sur la société** : retourne le nom, la forme juridique, la date de contrat et les activités économiques.
-   **Activités économiques** : fournit les codes et descriptions des activités principales et secondaires.
-   **Correspondance de société** : confirme que les informations correspondent au numéro CUIT.

En vérifiant ces éléments, vous pouvez confirmer que la société est légitime et correctement enregistrée, réduisant considérablement les risques de fraude.

## Référence de l'API

### Point d'accès

```
GET https://api.verifik.co/v2/ar/company
```

Consultez une société argentine avec **`documentType`** `CUIT` et **`documentNumber`**. Retourne des champs d'enregistrement pour la **KYB**, le **screening fournisseurs** et la **conformité**.

### En-têtes

| Nom           | Valeur             |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Paramètres

| Nom              | Type   | Requis | Description                                |
| ---------------- | ------ | ---- | ------------------------------------------ |
| `documentType`   | string | Oui | Type de document. Valeur autorisée : `CUIT`. |
| `documentNumber` | string | Oui | CUIT de 11 chiffres sans tirets.           |
