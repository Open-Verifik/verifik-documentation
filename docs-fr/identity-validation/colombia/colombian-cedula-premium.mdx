### Point de terminaison

```
GET https://api.verifik.co/v2/co/cedula/premium
```

Également : `GET https://verifik.app/v2/co/cedula/premium`. **POST** accepté avec les mêmes champs.

**CC uniquement** — paramètres : **`documentNumber`** (5–10 caractères).

### En-têtes

| Nom | Valeur |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Exemple de requête

<Tabs>
  <TabItem value="get" label="GET">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/cedula/premium", {
	params: { documentNumber: "1234567890" },
	headers: {
		Accept: "application/json",
		Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
	},
});
console.log(data);
```

  </TabItem>
