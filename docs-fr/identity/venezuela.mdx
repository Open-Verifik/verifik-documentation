# Vérification d'identité au Venezuela (CCVE)

L'API Verifik authentifie les **citoyens vénézuéliens** à partir de sources officielles, via la cédula nationale **CCVE** (*Cédula de Identidad*). Elle simplifie le **KYC**, réduit la fraude et aide à la conformité.

## Que valide cette API ?

Registres officiels pour vérifier :

-   **Nom et numéro de document** : cédula nationale **CCVE**.
-   **Cohérence** : les noms retournés correspondent au document.
-   **Champs optionnels** : date de naissance, genre, etc., selon disponibilité.

## Référence API

### Endpoint

```
https://api.verifik.co/v2/ve/cedula
```

Cette route renvoie les données d'identité des **nationaux vénézuéliens** pour la classe documentaire **CCVE**. Envoyez le numéro de cédula en **`documentNumber`** ; l'API applique **CCVE** et renvoie des champs structurés pour le **KYC** et la prévention des fraudes. Pour les **résidents étrangers**, utilisez [`GET /v2/ve/foreigner-id`](/verifik-fr/identity/venezuela-foreigner-id) (**CEVE**).

### En-têtes

| Nom | Valeur |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Paramètres

| Nom | Type | Obligatoire | Description |
| --- | --- | --- | --- |
| `documentNumber` | string | Oui | Numéro de cédula nationale (CCVE). Aucun paramètre `documentType` requis : le service applique **CCVE** sur cet endpoint. |

### Remarques

-   `documentNumber` est obligatoire.
-   Les champs optionnels (`birthDate`, `gender`, etc.) dépendent des sources.
