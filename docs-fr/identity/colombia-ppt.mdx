### Point d'accès

```
GET https://api.verifik.co/v2/co/foreigner-id/ppt
```

**`documentNumber`** et **`expeditionDate`** (`DD/MM/YYYY`). **`POST`** JSON équivalent.

### En-têtes

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Exigences documentaires

**Pour qui ?** **Ressortissants vénézuéliens** en Colombie titulaires d'un *Permiso de Protección Temporal* (**PPT**) lorsque vous avez besoin du **statut migratoire** auprès de **Migración Colombia** (ex. VIGENTE, dates d'expiration).

| Champ | Consignes |
| --- | --- |
| **`documentNumber`** | Numéro PPT, **chiffres uniquement**. En Colombie, le permis fait généralement **jusqu'à 7 chiffres**. Certains systèmes (SENA, paie) l'affichent complété à **15 caractères** — envoyez la valeur numérique de vos dossiers. Exemple Migración : `2081381`. |
| **`expeditionDate`** | Date d'émission au format **`JJ/MM/AAAA`** (ex. `07/03/2022`). Obligatoire. |

**Également disponible :** recherche de type nom via [`/v2/co/cedula`](/verifik-fr/identity-validation/colombia/colombia-identity-documents-guide) avec `documentType=PPT` (validation 5–10 chiffres, sans date d'émission).

Comparaison complète : [Guide des documents d'identité](/verifik-fr/identity-validation/colombia/colombia-identity-documents-guide).

### Paramètres

| name | type | required | description |
| --- | --- | --- | --- |
| `documentNumber` | string | yes | Numéro PPT, **chiffres uniquement**. Généralement **jusqu'à 7 chiffres** (certains systèmes paddent à 15). Exemple : `2081381`. |
| `expeditionDate` | string | yes | Date d'émission au format **`JJ/MM/AAAA`** (ex. `07/03/2022`). Doit correspondre au document. |

### Requête

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/foreigner-id/ppt", {
	params: { documentNumber: "123456789", expeditionDate: "07/03/2022" },
	headers: {
		Accept: "application/json",
		Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
	},
});
console.log(data);
```

### Remarques

- Format de date **`DD/MM/YYYY`**.
- **`GET`** et **`POST`** identiques.
- Indisponibilité amont possible (**404** / **500**).
