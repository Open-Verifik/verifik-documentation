# Vérification d'identité en Bolivie

L'API de vérification d'identité de Verifik vous permet d'authentifier les citoyens boliviens à partir de données gouvernementales officielles. Elle est conçue pour fluidifier vos processus KYC (Know Your Customer), prévenir la fraude et vous assurer de respecter sans effort toutes les exigences réglementaires.

Nous avons construit cette intégration pour les entreprises qui ont besoin d'un moyen rapide, sécurisé et automatisé de confirmer la véritable identité de leurs utilisateurs, employés ou clients.

## Que valide cette API ?

Notre API se connecte directement aux registres officiels pour valider :

-   **Nom complet et numéro CI** : prend en charge la *Cédula de Identidad* (CI).
-   **Correspondance d'identité** : confirme que le nom fourni correspond au numéro CI.
-   **Vérification de la date de naissance** : valide que la date de naissance correspond aux registres officiels.

En vérifiant ces éléments, vous pouvez avoir la certitude que la personne avec qui vous traitez existe et possède un document valide, ce qui réduit considérablement les risques d'usurpation d'identité et de fraude.

## Référence API

### Point d'accès

```
GET https://api.verifik.co/v2/bo/cedula
```

Vérifiez un citoyen bolivien avec **`documentType`** `CI`, **`documentNumber`** et **`dateOfBirth`** (`JJ/MM/AAAA`). Renvoie les champs d'identité pour le **KYC** et la **conformité**.

### En-têtes

| Nom | Valeur |
| --- | --- |
| Content-Type | `application/json` |
| Authorization | `Bearer <token>` |

### Paramètres

| Nom | Type | Requis | Description |
| --- | --- | --- | --- |
| `documentType` | string | Oui | Type de document. Valeur autorisée : `CI`. |
| `documentNumber` | string | Oui | Numéro de document CI. |
| `dateOfBirth` | string | Oui | Date de naissance au format `JJ/MM/AAAA`. |
