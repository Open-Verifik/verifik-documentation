### Point d'accès

```
GET https://api.verifik.co/v2/co/registraduria/certificado
```

**`documentNumber`**, date d’émission **`date`** (`DD/MM/YYYY`). Même champs en **`POST`** JSON. Service destiné à la **CC** (cédula de citoyenneté).

### En-têtes

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Paramètres

| name | type | required | description |
| --- | --- | --- | --- |
| `documentNumber` | string | oui | Numéro CC sans espaces ni ponctuation. |
| `date` | string | oui | **Date d’émission** du document **`DD/MM/YYYY`**. |

### Requête

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/registraduria/certificado", {
	params: { documentNumber: "123456789", date: "10/10/2020" },
	headers: {
		Accept: "application/json",
		Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
	},
});
console.log(data);
```

### Remarques

- **`date`** est la **date d’émission**, pas la date de naissance.
- **`GET`** et **`POST`** partagent le même handler.
- **pdfBase64** peut contenir des données personnelles.
