### Point d'accès

```
GET https://api.verifik.co/v2/co/cedula/extra
```

**`documentType`**=`CC`, **`documentNumber`**, date d’émission **`date`** (`DD/MM/YYYY`). Même schéma en **`POST`**.

### En-têtes

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Paramètres

| name | type | required | description |
| --- | --- | --- | --- |
| `documentType` | string | yes | **`CC`** uniquement. |
| `documentNumber` | string | yes | Numéro sans espaces. |
| `date` | string | yes | **Date d’émission** **`JJ/MM/AAAA`**. |

### Requête

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/cedula/extra", {
	params: { documentType: "CC", documentNumber: "123456789", date: "10/10/2020" },
	headers: {
		Accept: "application/json",
		Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
	},
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os, requests

url = "https://api.verifik.co/v2/co/cedula/extra"
headers = {"Accept": "application/json", "Authorization": f"Bearer {os.getenv('VERIFIK_TOKEN')}"}
params = {"documentType": "CC", "documentNumber": "123456789", "date": "10/10/2020"}
r = requests.get(url, headers=headers, params=params)
print(r.json())
```

### Remarques

- **`date`** = date d’**émission**, pas de naissance.
- Données sensibles possibles.
