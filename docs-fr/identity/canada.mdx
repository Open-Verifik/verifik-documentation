# Vérification du permis de conduire au Canada

L'API de vérification du permis de conduire de Verifik vous permet d'authentifier les permis canadiens à partir des données officielles du gouvernement. Elle est conçue pour fluidifier vos processus KYC (Know Your Customer), prévenir la fraude et vous assurer de respecter sans effort toutes les exigences réglementaires.

Nous avons construit cette intégration pour les entreprises qui ont besoin d'un moyen rapide, sécurisé et automatisé de confirmer la validité des permis de conduire délivrés en Colombie-Britannique et en Ontario.

## Que valide cette API ?

Notre API se connecte directement aux registres officiels pour valider :

-   **Numéro du permis de conduire** : confirme l'authenticité du numéro de permis.
-   **Statut du permis** : confirme si le permis est actuellement valide.
-   **Correspondance d'identité** : pour la Colombie-Britannique, vérifie que le nom de famille correspond bien à l'enregistrement du permis.

En vérifiant ces éléments, vous pouvez avoir la certitude que le permis est authentique et valide, ce qui réduit considérablement les risques de fraude.

Les services de validation d'identité au Canada incluent la vérification du permis de conduire pour les provinces de Colombie-Britannique et de l'Ontario. Ces services vous permettent de confirmer l'authenticité et la validité des permis délivrés dans ces provinces canadiennes.

## Provinces prises en charge

| Province | Endpoint | Description |
|----------|----------|-------------|
| **Colombie-Britannique** | `/v2/ca/british-columbia/driver-license` | Vérifiez les permis de la C.-B. avec le numéro de document et le nom de famille. |
| **Ontario** | `/v2/ca/ontario/driver-license` | Vérifiez les permis de l'Ontario avec le numéro de document. |

## Permis de conduire — Colombie-Britannique

### Endpoint

```
GET https://api.verifik.co/v2/ca/british-columbia/driver-license
```

Le service de vérification du permis de conduire de la Colombie-Britannique offre une méthode simple pour confirmer l'authenticité et la validité d'un permis dans la province de Colombie-Britannique, au Canada. Vous pouvez ainsi valider rapidement un permis à partir du numéro de document et du nom de famille fournis.

### En-têtes (Headers)

| Nom            | Valeur              |
| -------------- | ------------------- |
| Content-Type   | `application/json`  |
| Authorization  | `Bearer <token>`    |

### Paramètres

| Nom              | Type   | Requis  | Description                                                                                            | Exemple   |
| ---------------- | ------ | ------- | ------------------------------------------------------------------------------------------------------ | --------- |
| `documentNumber` | string | **Oui** | Permis de conduire à interroger ; les données doivent être saisies exactement telles qu'elles figurent sur le document. | `1123456` |
| `lastName`       | string | **Oui** | Nom de famille tel qu'il apparaît sur le permis.                                                       | `HELLO`   |

## Permis de conduire — Ontario
