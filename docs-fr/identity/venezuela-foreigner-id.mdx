# 🇻🇪 Venezuela — Cédula d'étranger (CEVE)

Cette API permet d'authentifier les **personnes étrangères** résidant au Venezuela au moyen de la **CEVE** (*Cédula de Identidad de Extranjero*).

## Que valide cette API ?

-   **Identité CEVE** : numéro de document et noms cohérents avec le registre.
-   **Type de document** : la réponse inclut `documentType: "CEVE"`.

## Référence API

### Endpoint

```
https://api.verifik.co/v2/ve/foreigner-id
```

Endpoint dédié aux **étrangers** titulaires d'une **CEVE**. Passez le numéro en **`documentNumber`** ; le service applique **CEVE**. Pour les **nationaux** (**CCVE**), utilisez [`GET /v2/ve/cedula`](/verifik-fr/fr-docs/identity/venezuela).

### En-têtes

| Nom | Valeur |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Paramètres

| Nom | Type | Obligatoire | Description |
| --- | --- | --- | --- |
| `documentNumber` | string | Oui | Numéro CEVE. Aucun `documentType` requis : **CEVE** est appliqué sur cet endpoint. |

<Tabs>
