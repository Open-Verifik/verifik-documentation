### Point d'accès

```
GET https://api.verifik.co/v2/co/situacion-militar
```

Envoyez **`documentType=CC`** et **`documentNumber`** en **query**. Seule la **CC** (cédula de ciudadanía) est acceptée.

### En-têtes

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Paramètres

| name | type | required | description |
| --- | --- | --- | --- |
| `documentType` | string | yes | Doit être `CC`. CE, PPT, etc. sont refusés sur cet endpoint. |
| `documentNumber` | string | yes | Numéro de la cédula à interroger. |
