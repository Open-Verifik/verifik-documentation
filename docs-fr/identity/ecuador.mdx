# Vérification d'identité en Équateur

L'API de vérification d'identité de Verifik vous permet d'authentifier les citoyens équatoriens à partir de données gouvernementales officielles. Elle est conçue pour fluidifier vos processus KYC (Know Your Customer), prévenir la fraude et vous assurer de respecter sans effort toutes les exigences réglementaires.

Nous avons construit cette intégration pour les entreprises qui ont besoin d'un moyen rapide, sécurisé et automatisé de confirmer la véritable identité de leurs utilisateurs, employés ou clients.

## Que valide cette API ?

Notre API se connecte directement aux registres officiels pour valider :

-   **Nom complet et numéro CCEC** : prend en charge la *Cédula de Ciudadanía Ecuatoriana* (CCEC).
-   **Correspondance d'identité** : confirme que le nom fourni correspond au numéro CCEC.

En vérifiant ces éléments, vous pouvez avoir la certitude que la personne avec qui vous traitez existe et possède un document valide, ce qui réduit considérablement les risques d'usurpation d'identité et de fraude.

## Référence API

### Point d'accès

```
GET https://api.verifik.co/v2/ec/cedula
```

Vérifiez un citoyen équatorien avec **`documentType`** `CCEC` et **`documentNumber`**. Renvoie les champs d'identité des registres officiels pour le **KYC** et la **conformité**.

### En-têtes

| Nom | Valeur |
| --- | --- |
| Content-Type | `application/json` |
| Authorization | `Bearer <token>` |

### Paramètres

| Nom | Type | Requis | Description |
| --- | --- | --- | --- |
| `documentType` | string | Oui | Type de document. Valeur autorisée : `CCEC`. |
| `documentNumber` | string | Oui | Numéro de document sans espaces ni points. |
