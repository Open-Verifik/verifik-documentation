# Vérification d'identité au Mexique

L'API de vérification d'identité de Verifik vous permet d'authentifier les citoyens mexicains à partir du Code unique de registre de population (CURP). Elle est conçue pour fluidifier vos processus KYC (Know Your Customer), prévenir la fraude et vous assurer de respecter sans effort toutes les exigences réglementaires.

Nous avons construit cette intégration pour les entreprises qui ont besoin d'un moyen rapide, sécurisé et automatisé de confirmer la véritable identité de leurs utilisateurs, employés ou clients.

## Que valide cette API ?

Notre API se connecte directement aux registres officiels pour valider :

-   **Nom complet et CURP** : prend en charge la *Clave Única de Registro de Población* (CURP).
-   **Données personnelles** : retourne la date de naissance, le sexe et la nationalité.
-   **Correspondance d'identité** : confirme que le nom fourni correspond au numéro CURP.

En vérifiant ces éléments, vous pouvez avoir la certitude que la personne avec qui vous traitez existe et possède un document valide, ce qui réduit considérablement les risques d'usurpation d'identité et de fraude.

## Référence de l'API

### Endpoint

```
https://api.verifik.co/v2/mx/curp
```

Cet endpoint vérifie le **CURP** (*Clave Única de Registro de Población*) d'un citoyen mexicain auprès des registres officiels. Transmettez `documentType` et `documentNumber` sur une requête GET ; une réponse réussie confirme l'enregistrement et retourne les champs de nom et de biographie validés lorsque le CURP est reconnu.

### En-têtes (Headers)

| Nom | Valeur |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Paramètres

| Nom | Type | Requis | Description |
| --- | --- | --- | --- |
| `documentType` | string | **Oui** | Valeur autorisée : `CURP`. |
| `documentNumber` | string | **Oui** | Valeur du CURP à valider. |
