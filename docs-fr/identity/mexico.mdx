# Vérification d'identité au Mexique

L'API Verifik permet d'authentifier les citoyens mexicains grâce au **CURP** (*Clave Única de Registro de Población*). Elle simplifie le KYC, réduit la fraude et aide au respect réglementaire.

## Que valide cette API ?

Connexion aux registres officiels pour vérifier :

-   **Nom complet et CURP** : format CURP pris en charge.
-   **Données personnelles** : date de naissance, sexe, nationalité.
-   **Correspondance** : le nom fourni correspond-il au CURP ?

## Référence API

### Endpoint

```
https://api.verifik.co/v2/mx/curp
```

Ce point d’accès vérifie le **CURP** (*Clave Única de Registro de Población*) d’un ressortissant mexicain auprès des sources officielles. Envoyez `documentType` et `documentNumber` en requête GET ; une réponse réussie confirme l’enregistrement d’identité et renvoie le nom et les informations biographiques validés lorsque le CURP est reconnu.

### En-têtes

| Nom | Valeur |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Paramètres

| Nom | Type | Obligatoire | Description |
| --- | --- | --- | --- |
| `documentType` | string | Oui | Valeur autorisée : `CURP`. |
| `documentNumber` | string | Oui | CURP à valider. |
