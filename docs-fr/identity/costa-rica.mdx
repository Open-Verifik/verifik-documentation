# Vérification d'identité au Costa Rica (CCCR)

L'API Verifik permet d'authentifier les titulaires de **CCCR** (*Cédula de Identidad Costarricense*) à partir de sources officielles pour le **KYC** et la conformité.

## Que valide cette API ?

- **Identité CCCR** : type de document et numéro de cédula.
- **Cohérence** : les noms retournés correspondent au registre.

## Référence API

### Endpoint

```
https://api.verifik.co/v2/cr/cedula
```

Envoyez **`documentType`**=`CCCR` et **`documentNumber`** (cédula sans espaces) en GET ; la réponse fournit des champs nominatifs structurés.

### En-têtes

| Nom | Valeur |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Paramètres

| Nom | Type | Obligatoire | Description |
| --- | --- | --- | --- |
| `documentType` | string | Oui | Type de document. Valeur autorisée : `CCCR`. |
| `documentNumber` | string | Oui | Numéro de cédula sans espaces. |

### Requête

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/cr/cedula", {

  headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});

```
