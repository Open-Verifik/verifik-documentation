# Chili — Contribuable (RUT)

Utilisez cet endpoint lorsque vous avez besoin de données d'identité **orientées contribuable** pour un **RUT** chilien (pas le flux civil de la cédula). Il complète la **[Vérification d'identité au Chili](/verifik-fr/identity/chile)** (`v2/cl/cedula`), adaptée au **RUN / registre civil**.

## API Reference

### Endpoint

```
https://api.verifik.co/v2/cl/taxpayer
```

Retourne des champs de nom alignés sur le **registre fiscal**—idéal pour **KYB**, onboarding fournisseurs et contrôle de facturation lorsque le **RUT** doit correspondre au **contribuable** enregistré.

### Headers

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parameters

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `documentType` | string | Yes | Doit être `RUT`. |
| `documentNumber` | string | Yes | Identifiant fiscal (normalisation côté service). |

### Request

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/cl/taxpayer", {
  params: { documentType: "RUT", documentNumber: "263007905" },
  headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});
console.log(data);
```

### Notes

- Préférez **`v2/cl/cedula`** pour l'identité **RUN** au registre civil ; utilisez **`v2/cl/taxpayer`** lorsque la référence est le **contribuable RUT**.
