# Vérification d'identité en Argentine

L'API de vérification d'identité de Verifik vous permet d'authentifier les citoyens argentins à partir de données gouvernementales officielles. Elle est conçue pour fluidifier vos processus KYC (Know Your Customer), prévenir la fraude et respecter sans effort toutes les exigences réglementaires.

Nous avons construit cette intégration pour les entreprises qui ont besoin d'un moyen rapide, sécurisé et automatisé de confirmer la véritable identité de leurs utilisateurs, employés ou clients.

## Que valide cette API ?

Notre API se connecte directement aux registres officiels pour valider :

-   **Nom complet et numéro DNI** : prend en charge le *Documento Nacional de Identidad* (DNI).
-   **Correspondance d'identité** : confirme que le nom fourni correspond au numéro DNI.

En vérifiant ces éléments, vous pouvez avoir la certitude que la personne existe et possède un document valide, ce qui réduit considérablement les risques d'usurpation d'identité et de fraude.

## Référence de l'API

### Point d'accès

```
GET https://api.verifik.co/v2/ar/cedula
```

Vérifiez une identité argentine avec **`documentType`** `DNIAR` et **`documentNumber`**. Retourne des champs d'identité pour la **KYC** et la **conformité**.

### En-têtes

| Nom           | Valeur             |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Paramètres

| Nom              | Type   | Requis | Description                                |
| ---------------- | ------ | ---- | ------------------------------------------ |
| `documentType`   | string | Oui | Type de document. Valeur autorisée : `DNIAR`. |
| `documentNumber` | string | Oui | Numéro de document sans espaces.           |

-   **RH et recrutement** : validez les documents des candidats dans votre processus d'embauche.
-   **Assurance et santé** : confirmez l'identité avant d'émettre des polices ou des prestations.
