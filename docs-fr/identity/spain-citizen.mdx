# Vérification d'identité en Espagne

L'API Verifik permet d'authentifier les citoyens et résidents espagnols à partir de sources officielles. Elle simplifie le KYC, réduit la fraude et aide à la conformité.

Cette intégration s'adresse aux entreprises qui ont besoin de confirmer rapidement et en toute sécurité l'identité d'utilisateurs, salariés ou clients.

## Que valide cette API ?

Notre API interroge les registres officiels pour vérifier :

-   **Nom complet et numéro de document** : *Documento Nacional de Identidad* (DNI) et *Número de Identidad de Extranjero* (NIE).
-   **Cohérence identité** : le nom fourni correspond-il au numéro de document ?
-   **Date d'expiration** : validation de la date d'expiration du document.

Ces contrôles réduisent le risque d'usurpation et renforcent la confiance dans vos parcours d'inscription.

## Référence API

### Endpoint

```
https://api.verifik.co/v2/es/cedula
```

Ce point d'accès vérifie les documents d'identité espagnols (DNI/NIE) via une requête GET avec `documentType`, `documentNumber` et `date` (expiration).

### En-têtes

| Nom | Valeur |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Paramètres

| Nom | Type | Obligatoire | Description |
| --- | --- | --- | --- |
| `documentType` | string | Oui | Type de document. Valeurs : `DNIES`, `NIE`. |
| `documentNumber` | string | Oui | Numéro de document sans espaces. |
| `date` | string | Oui | Date d'expiration au format `DD/MM/YYYY`. |

## Cas d'usage courants

-   **Fintech & banque** : vérifier les identités lors de l'ouverture de compte ou d'un crédit.
-   **E-commerce & livraison** : authentifier utilisateurs et coursiers.
-   **RH & recrutement** : valider les documents des candidats.
-   **Assurance & santé** : confirmer l'identité avant une police ou des prestations.
