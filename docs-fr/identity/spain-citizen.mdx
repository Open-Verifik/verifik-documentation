# Vérification d'identité en Espagne

L'API de vérification d'identité de Verifik vous permet d'authentifier les citoyens et résidents espagnols à partir des données officielles du gouvernement. Elle est conçue pour fluidifier vos processus KYC (Know Your Customer), prévenir la fraude et vous assurer de respecter sans effort toutes les exigences réglementaires.

Nous avons construit cette intégration pour les entreprises qui ont besoin d'un moyen rapide, sécurisé et automatisé de confirmer la véritable identité de leurs utilisateurs, employés ou clients.

## Que valide cette API ?

Notre API se connecte directement aux registres officiels pour valider :

-   **Nom complet et numéro de document** : prend en charge le *Documento Nacional de Identidad* (DNI) et le *Número de Identidad de Extranjero* (NIE).
-   **Correspondance d'identité** : confirme que le nom fourni correspond au numéro de document.
-   **Vérification de la date d'expiration** : valide la date d'expiration du document.

En vérifiant ces éléments, vous pouvez avoir la certitude que la personne avec qui vous traitez existe et possède un document valide, ce qui réduit considérablement les risques d'usurpation d'identité et de fraude.

## Référence de l'API

### Endpoint

```
https://api.verifik.co/v2/es/cedula
```

### En-têtes (Headers)

| Nom            | Valeur              |
| -------------- | ------------------- |
| Content-Type   | `application/json`  |
| Authorization  | `Bearer <token>`    |

### Paramètres

| Nom              | Type   | Requis  | Description                                                  |
| ---------------- | ------ | ------- | ------------------------------------------------------------ |
| `documentType`   | string | **Oui** | Type de document. Valeurs autorisées : `DNIES`, `NIE`.       |
| `documentNumber` | string | **Oui** | Numéro de document sans espaces.                             |
| `date`           | string | **Oui** | Date d'expiration au format `JJ/MM/AAAA`.                    |
