# Vérification d'identité au Panama

L'API Verifik permet d'authentifier les titulaires de **cédula panaméenne** (*Cédula de Identidad Personal*, CCPA) à partir de sources officielles. Elle simplifie le **KYC**, réduit la fraude et aide à la conformité.

## Que valide cette API ?

Notre API interroge les registres officiels pour vérifier :

-   **Identité et CCPA** : numéro de document normalisé, cohérence avec les données d'état civil.
-   **Date de naissance** : doit correspondre au registre (`dateOfBirth` au format `DD/MM/YYYY`).

## Référence API

### Endpoint

```
https://api.verifik.co/v2/pa/cedula
```

Vérification de la **CCPA** panaméenne contre les registres civils officiels. Envoyez **`documentType`** (`CCPA`), un **`documentNumber`** normalisé et **`dateOfBirth`** au format **`DD/MM/YYYY`** ; la réponse confirme la cohérence des données pour le **KYC**, l'onboarding et la lutte contre la fraude.

### En-têtes

| Nom | Valeur |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Paramètres

| Nom | Type | Obligatoire | Description |
| --- | --- | --- | --- |
| `documentType` | string | Oui | Type de document. Valeur autorisée : `CCPA`. |
| `documentNumber` | string | Oui | Cédula complète après normalisation (voir la doc principale pour les formats). |
| `dateOfBirth` | string | Oui | Date de naissance au format `DD/MM/YYYY`. |

### Notes

- La date de naissance est obligatoire et doit correspondre au titulaire.
- Les formats de `documentNumber` suivent le validateur [cedula-panama](https://github.com/merlos/cedula-panama).
