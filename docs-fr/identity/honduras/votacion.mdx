# Honduras - Lieu de vote

Verifik interroge le registre national de consultation électorale **Donde Me Toca Votar** avec le **DNIHN** (numéro d'identité à **13 chiffres**). Utilisez ce point de terminaison pour la **vérification électorale**, l'**intégration des électeurs** et la **conformité** lorsque les données électorales honduriennes sont autorisées.

Retourne les détails du lieu de vote lorsque la source les fournit :

- **`documentType`** — valeur autorisée : **DNIHN**
- **`documentNumber`** — numéro d'identité hondurien (**13** chiffres)
- **`fullName`** — nom complet du citoyen dans le registre électoral
- **`gender`** — sexe lorsque la source le fournit
- **`department`**, **`municipality`**, **`electoralSector`** — résidence et secteur électoral
- **`pollingPlace`**, **`pollingTable`** (JRV), **`lineNumber`** — détails du bureau de vote
- **`enabled`** — indique si le citoyen est habilité à voter
- **`fullAddress`** — adresse formatée du bureau de vote
- **Champs optionnels** — `latitude`, `longitude` lorsque la source les inclut
- Réponse **signée** par Verifik

### Endpoint

```
GET v2/hn/votacion
```

Interrogez le registre national avec **`documentType`** `DNIHN` et un **`documentNumber`** de **13 chiffres**. Les réponses réussies incluent le bureau de vote, la JRV (table), le département, la municipalité et l'éligibilité au vote pour les flux **KYC**, **vérification électorale** et **conformité**.

### Headers

| Header | Valeur | Description |
| --- | --- | --- |
| Accept | application/json | Format de réponse |
| Authorization | Bearer \<token\> | Votre JWT Verifik |

### Paramètres

| Champ | Type | Requis | Description | Exemple |
| --- | --- | --- | --- | --- |
| documentType | String | Oui | Type de document. Valeur autorisée : `DNIHN`. | `DNIHN` |
| documentNumber | String | Oui | Numéro d'identité hondurien (**13** chiffres). | `0501199705651` |

### Notes

- **`documentType`** doit être **`DNIHN`** ; **`documentNumber`** doit comporter exactement **13** chiffres numériques.
- **Exemples en production :** `0501199705651`, `0828200300064`, `0801198000420`, `0801198000427`, `0801198000428`, `0801197302222`, `0801198000421`.
- **Mode sandbox :** utilisez les identifiants de test **`0000010000001`** à **`0000010000010`** pour des profils stables ; **`0000090040401`** renvoie **404**.
- **Limites de la source :** si le registre électoral limite les requêtes, Verifik peut renvoyer **409** avec `Endpoint_out_of_service` (temporaire ; réessayez plus tard).
- Traitez les réponses comme des **données personnelles sensibles** ; respectez les lois sur la vie privée applicables.
- Voir aussi : [Citoyen hondurien](/identity/honduras).
