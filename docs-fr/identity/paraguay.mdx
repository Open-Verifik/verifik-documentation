# Vérification d'identité au Paraguay

L'API de vérification d'identité de Verifik vous permet d'authentifier les citoyens paraguayens à partir des données officielles du gouvernement. Elle est conçue pour fluidifier vos processus KYC (Know Your Customer), prévenir la fraude et vous assurer de respecter sans effort toutes les exigences réglementaires.

Nous avons construit cette intégration pour les entreprises qui ont besoin d'un moyen rapide, sécurisé et automatisé de confirmer la véritable identité de leurs utilisateurs, employés ou clients.

## Que valide cette API ?

Notre API se connecte directement aux registres officiels pour valider :

-   **Nom complet et numéro CIC** : prend en charge la *Cédula de Identidad Civil* (CIC).
-   **Correspondance d'identité** : confirme que le nom fourni correspond au numéro CIC.

En vérifiant ces éléments, vous pouvez avoir la certitude que la personne avec qui vous traitez existe et possède un document valide, ce qui réduit considérablement les risques d'usurpation d'identité et de fraude.

## Référence de l'API

### Endpoint

```
GET https://api.verifik.co/v2/py/cic
```

Vérifiez un citoyen paraguayen avec **`documentNumber`** (CIC). Retourne les champs issus des registres officiels pour le **KYC** et la **conformité**.

### En-têtes (Headers)

| Nom            | Valeur              |
| -------------- | ------------------- |
| Content-Type   | `application/json`  |
| Authorization  | `Bearer <token>`    |

### Paramètres

| Nom              | Type   | Requis  | Description                       |
| ---------------- | ------ | ------- | --------------------------------- |
| `documentNumber` | string | **Oui** | Numéro de document CIC.           |
