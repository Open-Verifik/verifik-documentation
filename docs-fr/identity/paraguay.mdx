# Vérification d'identité au Paraguay

L'API Verifik permet d'authentifier les citoyens paraguayens à partir de **données gouvernementales officielles**. Elle simplifie le **KYC**, réduit la fraude et aide à la conformité réglementaire.

Cette intégration s'adresse aux entreprises qui ont besoin d'un moyen **rapide, sécurisé et automatisé** de confirmer l'identité d'utilisateurs, d'employés ou de clients.

## Que valide cette API ?

Notre API interroge les registres officiels pour vérifier :

-   **Nom complet et numéro CIC** : prise en charge de la *Cédula de Identidad Civil* (CIC).
-   **Cohérence identité** : confirme que le nom fourni correspond au numéro CIC.

Ces contrôles réduisent le risque d'usurpation et renforcent la confiance dans vos parcours d'inscription.

## Référence API

### Endpoint

```
https://api.verifik.co/v2/py/cic
```

### Headers

| Name          | Value              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parameters

| Name           | Type    | Required | Description                                    |
| -------------- | ------- | -------- | ---------------------------------------------- |
| `documentNumber` | string | Yes      | Numéro de document CIC. |

### Request

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/py/cic", {
  params: { documentNumber: "1234567" },
  headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

url = "https://api.verifik.co/v2/py/cic"
headers = {"Accept": "application/json", "Authorization": f"Bearer {os.getenv('VERIFIK_TOKEN')}"}
