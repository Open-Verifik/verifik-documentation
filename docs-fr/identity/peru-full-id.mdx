# 🇵🇪 Pérou — DNI (données étendues)

L'API de vérification d'identité de Verifik vous permet d'effectuer une vérification d'identité étendue pour les citoyens péruviens à l'aide de leur numéro de DNI (*Documento Nacional de Identidad*). Elle est conçue pour automatiser vos processus KYC (Know Your Customer), prévenir la fraude et vous assurer de respecter sans effort toutes les exigences réglementaires.

Nous avons construit cette intégration pour les entreprises qui ont besoin d'un moyen rapide, sécurisé et automatisé d'accéder à des informations personnelles complètes : nom complet, état civil, date de naissance, détails du document, adresse et photo.

**L'en-tête Authorization est requis pour authentifier l'accès à ces informations en toute sécurité.**

## Que valide cette API ?

Notre API se connecte directement aux registres officiels péruviens pour valider :

-   **Nom complet et numéro de DNI** : prend en charge les documents *Documento Nacional de Identidad* (DNI).
-   **Correspondance d'identité** : confirme que le nom fourni correspond au numéro de DNI.
-   **Données personnelles étendues** : retourne l'état civil, la date de naissance, le sexe et les informations d'adresse.
-   **Informations sur le document** : valide le type de document, le numéro, la date d'émission, la date d'expiration et le chiffre de vérification.
-   **Informations géographiques** : fournit le code ubigeoReniec pour l'identification du lieu.
-   **Données photo** : accès à la photo du citoyen au format base64.

En vérifiant ces éléments, vous pouvez avoir la certitude que la personne avec qui vous traitez existe et possède un document valide, ce qui réduit considérablement les risques d'usurpation d'identité et de fraude.

### Endpoint

```
GET https://api.verifik.co/v3/pe/cedula/extra
```

### En-têtes (Headers)

| Nom            | Valeur              |
| -------------- | ------------------- |
| Content-Type   | `application/json`  |
| Authorization  | `Bearer <token>`    |

### Paramètres

| Nom | Type | Requis | Description | Exemple |
| --- | ---- | ------ | ----------- | ------- |
| `documentType` | string | **Oui** | Type de document. Valeur autorisée : DNI. | `DNI` |
| `documentNumber` | string | **Oui** | Numéro de document de la personne à interroger. | `123456789` |
