# Vérification d'identité au Chili

L'API Verifik permet d'authentifier les personnes grâce au **Rol Único Nacional (RUN)**. Elle simplifie le **KYC**, réduit la fraude et aide à respecter les obligations réglementaires.

Nous avons conçu cette intégration pour les entreprises qui ont besoin de confirmer rapidement et en toute sécurité l'identité d'utilisateurs, de salariés ou de clients.

## Que valide cette API ?

L'API interroge des registres officiels pour valider :

- **Nom complet et RUN** : prise en charge du *Rol Único Nacional*.
- **Cohérence** : le nom fourni correspond au numéro RUN.

Vous limitez ainsi l'usurpation d'identité et renforcez la confiance dans l'inscription.

## API Reference

### Endpoint

```
https://api.verifik.co/v2/cl/cedula
```

Vérifie l'identité civile au Chili : envoyez **`documentType`** = `RUN` et **`documentNumber`** (RUN sans espaces) en GET ; la réponse inclut des champs de nom structurés pour **KYC**, onboarding et contrôles antifraude.

### Headers

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parameters

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `documentType` | string | Yes | Valeur autorisée : `RUN`. |
| `documentNumber` | string | Yes | RUN sans espaces ni séparateurs. |

## Cas d'usage courants

- **Fintech et banques** : vérifier l'identité à l'ouverture de compte ou au crédit.
- **E-commerce et livraison** : authentifier utilisateurs et livreurs.
- **RH et recrutement** : valider les pièces dans le processus d'embauche.
- **Assurance et santé** : confirmer l'identité avant contrat ou prestation.

## Sources officielles

Connexion aux sources gouvernementales chiliennes (Registro Civil) pour des données à jour et sécurisées.
