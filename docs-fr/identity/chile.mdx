# Vérification d'identité au Chili

L'API de vérification d'identité de Verifik vous permet d'authentifier les citoyens chiliens à partir du Rôle Unique National (RUN). Elle est conçue pour fluidifier vos processus KYC (Know Your Customer), prévenir la fraude et vous assurer de respecter sans effort toutes les exigences réglementaires.

Nous avons construit cette intégration pour les entreprises qui ont besoin d'un moyen rapide, sécurisé et automatisé de confirmer la véritable identité de leurs utilisateurs, employés ou clients.

## Que valide cette API ?

Notre API se connecte directement aux registres officiels pour valider :

-   **Nom complet et RUN** : prend en charge le *Rol Único Nacional* (RUN).
-   **Correspondance d'identité** : confirme que le nom fourni correspond au numéro RUN.

En vérifiant ces éléments, vous pouvez avoir la certitude que la personne avec qui vous traitez existe et possède un document valide, ce qui réduit considérablement les risques d'usurpation d'identité et de fraude.

## Référence de l'API

### Endpoint

```
https://api.verifik.co/v2/cl/cedula
```

Vérifiez la pièce d'identité nationale chilienne (RUN/RUT) : confirmez le nom et les champs d'identité contre les sources officielles du registre civil pour le KYC, l'onboarding et la prévention de la fraude.

### En-têtes (Headers)

| Nom | Valeur |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Paramètres

| Nom | Type | Requis | Description |
| --- | --- | --- | --- |
| `documentType` | string | **Oui** | Valeur autorisée : `RUN`. |
| `documentNumber` | string | **Oui** | Valeur du RUN sans espaces ni séparateurs. |
