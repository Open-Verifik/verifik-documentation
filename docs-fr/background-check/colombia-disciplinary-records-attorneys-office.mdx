# Antécédents disciplinaires (Procuraduría)

## Référence API

### Endpoint

```
https://api.verifik.co/v2/co/procuraduria/antecedentes
```

Consultez les antécédents disciplinaires en Colombie (Procuraduría). Avec le type et le numéro de document, la réponse indique l'identité et la présence d'éventuelles sanctions, instances, infractions ou inéligibilités.

### En-têtes

| Nom           | Valeur             |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Paramètres

:::info
Pour la liste complète des **types de document** et leurs paramètres obligatoires, consultez le guide [Types de document](/background-check/document-types).
:::

| Nom              | Type   | Obligatoire | Description |
| ---------------- | ------ | ----------- | ----------- |
| `documentType`   | string | Oui         | CC, PEP ou CE. |
| `documentNumber` | string | Oui         | Numéro sans espaces ni points. |
