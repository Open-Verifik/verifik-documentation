### Point d'accès

```
GET https://api.verifik.co/v2/co/sisben
```

Consultation du portail public SISBEN avec **`documentType`** et **`documentNumber`**. Navigation automatisée — délais possibles.

### En-têtes

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Paramètres

| name | type | required | description |
| --- | --- | --- | --- |
| `documentType` | string | yes | **`CC`**, **`CE`**, **`PEP`**, **`PPT`**, **`PA`**. |
| `documentNumber` | string | yes | Sans espaces ni ponctuation. |

### Requête

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/sisben", {
	params: { documentType: "CC", documentNumber: "1121329661" },
	headers: {
		Accept: "application/json",
		Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
	},
});
console.log(data);
```

  <TabItem value="python" label="Python">
