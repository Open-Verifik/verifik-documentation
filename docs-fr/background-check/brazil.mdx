# Brésil

## Référence API

### Endpoint

```
https://api.verifik.co/v2/br/background-check
```

Le service d'antécédents Brésil fournit des informations détaillées sur les personnes physiques brésiliennes. La réponse peut inclure noms associés, capacité à émettre des rapports, numéro de certification, type et numéro de document, prénom, nom, nom complet et un fichier PDF encodé en base64.

Ces données servent au recrutement, à la diligence raisonnable, à la conformité et à la prévention de la fraude.

### En-têtes

| Nom           | Valeur             |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Paramètres

:::info
Pour la liste complète des **types de document** et leurs paramètres obligatoires, consultez le guide [Types de document](/background-check/document-types).
:::

| Nom            | Type   | Obligatoire | Description |
| -------------- | ------ | ----------- | ----------- |
| `documentType` | string | Oui         | Type de document. Valeur autorisée : CPF. |
| `documentNumber` | string | Oui       | Numéro de document de la personne consultée. |
| `dateOfBirth`  | string | Oui         | Date de naissance (format jj/mm/aaaa). |
