# Vérifier la conformité aux mesures correctives (Police - RNMC)

## Référence de l'API

### Endpoint

```
https://api.verifik.co/v2/co/policia/rnmc
```

Le service RNMC de la police permet de vérifier si une personne fait l'objet de mesures correctives en attente. En fournissant le nom, le type et le numéro de document ainsi que la date de vérification, vous obtenez le détail des mesures correctives encore actives.

Ce service vous aide à confirmer la conformité d'une personne aux mesures correctives et à prendre des décisions éclairées en fonction de son statut.

### En-têtes (Headers)

| Nom            | Valeur              |
| -------------- | ------------------- |
| Content-Type   | `application/json`  |
| Authorization  | `Bearer <token>`    |

### Paramètres

:::info
Pour la liste complète des **types de documents** pris en charge ainsi que de leurs paramètres obligatoires spécifiques, consultez le guide [Types de documents](/background-check/document-types).
:::

| Nom              | Type   | Requis  | Description                                                                  |
| ---------------- | ------ | ------- | ---------------------------------------------------------------------------- |
| `documentType`   | string | **Oui** | Type de document. Valeurs autorisées : CC, CE.                               |
| `documentNumber` | string | **Oui** | Numéro de document de la personne à interroger, sans espaces ni points.      |
| `date`           | string | **Oui** | Date de délivrance du document. Format valide : JJ/MM/AAAA.                  |
