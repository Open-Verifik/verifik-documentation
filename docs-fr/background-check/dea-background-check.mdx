# Contrôle d'antécédents DEA

## Référence de l'API

### Endpoint

```
https://api.verifik.co/v2/dea
```

Vérifiez si une personne ou une entité figure sur la liste de surveillance de la Drug Enforcement Administration (DEA) des États-Unis à partir des paramètres `documentType` et `documentNumber`, ou via `fullName`. L'API indique si une correspondance a été trouvée et, le cas échéant, l'URL de l'enregistrement officiel de la DEA.

**Note :** les dates (date de naissance ou date de délivrance) doivent être au format `jj/mm/aaaa`.

### En-têtes (Headers)

| Nom            | Valeur              |
| -------------- | ------------------- |
| Content-Type   | `application/json`  |
| Authorization  | `Bearer <token>`    |

### Paramètres

:::info
Pour la liste complète des **types de documents** pris en charge ainsi que de leurs paramètres obligatoires spécifiques, consultez le guide [Types de documents](/background-check/document-types).
:::

| Nom              | Type   | Requis | Description                                                                                          |
| ---------------- | ------ | ------ | ---------------------------------------------------------------------------------------------------- |
| `documentType`   | string | Non    | Type de document à interroger.                                                                       |
| `documentNumber` | string | Non    | Numéro de document à interroger, sans espaces ni points.                                             |
| `fullName`       | string | Non    | À la place de `documentType` et `documentNumber`, vous pouvez transmettre directement le nom complet. |

### Cas d'usage

- **Conformité réglementaire :** identifiez les signalements liés à la DEA dans le cadre des règles fédérales américaines.
- **Prévention de la fraude :** détectez les risques associés aux dossiers de répression sur les substances contrôlées.
- **KYC et LCB-FT :** soutenez les programmes santé, pharma, logistique et finance.
