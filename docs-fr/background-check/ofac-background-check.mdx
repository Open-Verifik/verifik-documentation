# Contrôle de sanctions OFAC

## Référence API

### Endpoint

```
https://api.verifik.co/v2/ofac
```

Vérifiez les listes **OFAC SDN** avec `documentType` et `documentNumber`, ou `fullName`. Pour conformité aux sanctions et dépistage commercial.

**Remarque :** dates au format `jj/mm/aaaa`.

### En-têtes

| Nom | Valeur |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Paramètres

:::info
Pour la liste complète des **types de document** pris en charge et leurs paramètres obligatoires, consultez le guide [Types de document](/background-check/document-types).
:::

| Nom | Type | Obligatoire | Description |
| --- | --- | --- | --- |
| `documentType`   | string | No       | The document type that you want to request.                                                       |
| `documentNumber` | string | No       | Document number to consult, without spaces or points.                                             |
| `fullName`       | string | No       | Instead of documentType and documentNumber, you can pass the name directly of the person/business. |

### Cas d'usage

- **Conformité sanctions.**
- **Réduction des risques** de conformité au commerce.
- **KYC/AML** bancaire et assurance.
