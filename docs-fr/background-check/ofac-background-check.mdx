# Contrôle de sanctions OFAC

## Référence de l'API

### Endpoint

```
https://api.verifik.co/v2/ofac
```

Vérifiez si une personne ou une entité figure sur la liste SDN (Specially Designated Nationals) et les personnes bloquées de l'Office of Foreign Assets Control (OFAC) du Trésor américain à partir du couple `documentType` + `documentNumber`, ou via `fullName`. Utilisez les résultats pour la conformité aux sanctions, le filtrage commercial et les flux KYC/AML.

**Note :** les dates (date de naissance ou date de délivrance) doivent être au format `jj/mm/aaaa`.

### En-têtes (Headers)

| Nom            | Valeur              |
| -------------- | ------------------- |
| Content-Type   | `application/json`  |
| Authorization  | `Bearer <token>`    |

### Paramètres

:::info
Pour la liste complète des **types de documents** pris en charge ainsi que de leurs paramètres obligatoires spécifiques, consultez le guide [Types de documents](/background-check/document-types).
:::

| Nom              | Type   | Requis | Description                                                                                          |
| ---------------- | ------ | ------ | ---------------------------------------------------------------------------------------------------- |
| `documentType`   | string | Non    | Type de document à interroger.                                                                       |
| `documentNumber` | string | Non    | Numéro de document à interroger, sans espaces ni points.                                             |
| `fullName`       | string | Non    | À la place de `documentType` et `documentNumber`, vous pouvez transmettre directement le nom complet. |

### Cas d'usage

- **Conformité aux sanctions :** évitez de traiter avec des parties bloquées soumises à l'OFAC.
- **Atténuation des risques :** réduisez les pénalités liées aux règles commerciales américaines et internationales.
- **KYC et LCB-FT :** renforcez le filtrage clients dans la banque, l'assurance et l'immobilier.
