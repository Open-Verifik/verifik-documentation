# Marchés publics

## Référence API

### Endpoint

```
https://api.verifik.co/v2/co/contracts
```

Ce service renvoie la liste des contrats publics liés à une personne ou une société à partir du type et du numéro de document (contractant, montants, entités, durées, etc.).

Il favorise la transparence et le suivi des marchés publics avec l'État en Colombie.

### En-têtes

| Nom           | Valeur             |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Paramètres

:::info
Pour la liste complète des **types de document** et leurs paramètres obligatoires, consultez le guide [Types de document](/background-check/document-types).
:::

| Nom              | Type   | Obligatoire | Description |
| ---------------- | ------ | ----------- | ----------- |
| `documentType`   | string | Oui         | CC ou NIT. |
| `documentNumber` | string | Oui         | Numéro de document demandé. |
