# Marchés publics

## Référence de l'API

### Endpoint

```
https://api.verifik.co/v2/co/contracts
```

Le service Marchés publics permet de récupérer la liste des contrats publics associés à une personne ou à une entreprise à partir de son numéro et de son type de document. La réponse contient des informations détaillées sur chaque contrat : contractant, montant, entités impliquées, durée du contrat et bien plus encore.

Ce service favorise la transparence et facilite le suivi des contrats publics conclus avec l'État en Colombie.

### En-têtes (Headers)

| Nom            | Valeur              |
| -------------- | ------------------- |
| Content-Type   | `application/json`  |
| Authorization  | `Bearer <token>`    |

### Paramètres

:::info
Pour la liste complète des **types de documents** pris en charge ainsi que de leurs paramètres obligatoires spécifiques, consultez le guide [Types de documents](/background-check/document-types).
:::

| Nom              | Type   | Requis  | Description                                                                  |
| ---------------- | ------ | ------- | ---------------------------------------------------------------------------- |
| `documentType`   | string | **Oui** | Type de document. Valeurs autorisées : CC, NIT.                              |
| `documentNumber` | string | **Oui** | Numéro de document à interroger.                                             |
