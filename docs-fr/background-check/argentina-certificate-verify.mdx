# Certificat de casier judiciaire (Argentine)

## Référence de l'API

### Endpoint

```
GET https://api.verifik.co/v2/ar/dnrec/antecedentes
```

Vérifie un certificat **DNREC** de casier judiciaire argentin à l'aide du **code de demande** (`Solicitud`) et du **code de sécurité** (`Cod. Segur.`) imprimés sur le PDF officiel.

### En-têtes (Headers)

| Nom           | Valeur             |
| ------------- | ------------------ |
| Accept        | `application/json` |
| Authorization | `Bearer <token>`   |

### Paramètres

| Nom            | Type   | Requis  | Description                                              |
| -------------- | ------ | ------- | -------------------------------------------------------- |
| `requestCode`  | string | **Oui** | Code de demande (`Solicitud`). Ex. : `02118499487`.      |
| `securityCode` | string | **Oui** | Code de sécurité (`Cod. Segur.`). Ex. : `371488F861`.    |

### Requête (Request)

<Tabs>
<TabItem value="javascript" label="JavaScript">

```javascript
import axios from 'axios';

const options = {
  method: 'GET',
  url: 'https://api.verifik.co/v2/ar/dnrec/antecedentes',
  params: { requestCode: '02118499487', securityCode: '371488F861' },
  headers: { Accept: 'application/json', Authorization: 'Bearer <votre_token>' },
};

const { data } = await axios.request(options);
console.log(data);
```

</TabItem>
<TabItem value="python" label="Python">

```python
import http.client

conn = http.client.HTTPSConnection("api.verifik.co")
headers = {"Accept": "application/json", "Authorization": "Bearer <votre_token>"}
conn.request("GET", "/v2/ar/dnrec/antecedentes?requestCode=02118499487&securityCode=371488F861", headers=headers)
print(conn.getresponse().read().decode("utf-8"))
```

### Notes

- Les deux paramètres sont obligatoires ; sinon **409** `MissingParameter`.
- En **mode sandbox** : `requestCode=02118499487`, `securityCode=371488F861` ; `requestCode=90040401` → **404**.
