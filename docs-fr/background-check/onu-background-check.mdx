# Contrôle de liste de sanctions ONU

## Référence API

### Endpoint

```
https://api.verifik.co/v2/onu
```

Interrogez la liste de sanctions de l'ONU avec `documentType` et `documentNumber`, ou `fullName`. Utile pour la conformité internationale, le KYC et la LCB.

**Remarque :** dates au format `jj/mm/aaaa`.

### En-têtes

| Nom | Valeur |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Paramètres

:::info
Pour la liste complète des **types de document** pris en charge et leurs paramètres obligatoires, consultez le guide [Types de document](/background-check/document-types).
:::

| Nom | Type | Obligatoire | Description |
| --- | --- | --- | --- |
| `documentType`   | string | No       | Document type that you want to request                                                            |
| `documentNumber` | string | No       | Document number to consult, without spaces or points                                              |
| `fullName`       | string | No       | Instead of documentType and documentNumber, you can pass the name directly of the person/business |

### Cas d'usage

- **Conformité internationale:** filtrage avant relation d'affaires.
- **Évaluation des risques:** réduire l'exposition aux parties sanctionnées.
- **Obligations réglementaires:** banque, assurance, finance.

import http.client

conn = http.client.HTTPSConnection("api.verifik.co")

headers = {
    'Accept': "application/json",
    'Authorization': "JWT token"
}

conn.request("GET", "/v2/onu?fullName=", headers=headers)

res = conn.getresponse()
data = res.read()

print(data.decode("utf-8"))
