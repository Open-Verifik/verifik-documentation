# Casier judiciaire (police nationale)

## Référence API

### Endpoint

```
https://api.verifik.co/v2/co/policia/consultar
```

Ce service vérifie l'absence ou la présence de poursuites en cours auprès des autorités judiciaires colombiennes (art. 248 de la Constitution). La réponse inclut l'identité et des détails sur le statut juridique. Utile pour le recrutement et la diligence raisonnable.

### En-têtes

| Nom           | Valeur             |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Paramètres

:::info
Pour la liste complète des **types de document** et leurs paramètres obligatoires, consultez le guide [Types de document](/background-check/document-types).
:::

| Nom              | Type   | Obligatoire | Description |
| ---------------- | ------ | ----------- | ----------- |
| `documentType`   | string | Oui         | CC ou CE. |
| `documentNumber` | string | Oui         | Numéro sans espaces ni points. |
