# Casier judiciaire colombien (police nationale)

## Référence de l'API

### Endpoint

```
https://api.verifik.co/v2/co/policia/consultar
```

Le service de casier judiciaire de la police colombienne permet de vérifier les antécédents pénaux d'une personne en Colombie à partir de son numéro de document. Il indique l'existence d'éventuelles affaires en cours auprès des autorités judiciaires colombiennes, conformément à l'article 248 de la Constitution colombienne. La réponse contient le nom complet, le prénom, le nom de famille et des informations supplémentaires sur le statut juridique. Idéal pour les entreprises et organisations qui réalisent des vérifications d'antécédents pour le recrutement ou la diligence raisonnable.

### En-têtes (Headers)

| Nom            | Valeur              |
| -------------- | ------------------- |
| Content-Type   | `application/json`  |
| Authorization  | `Bearer <token>`    |

### Paramètres

:::info
Pour la liste complète des **types de documents** pris en charge ainsi que de leurs paramètres obligatoires spécifiques, consultez le guide [Types de documents](/background-check/document-types).
:::

| Nom              | Type   | Requis  | Description                                                                  |
| ---------------- | ------ | ------- | ---------------------------------------------------------------------------- |
| `documentType`   | string | **Oui** | Type de document. Valeurs autorisées : CC, CE.                               |
| `documentNumber` | string | **Oui** | Numéro de document de la personne à interroger, sans espaces ni points.      |
