# Certificat de la Contraloría

## Référence API

### Endpoint

```
https://api.verifik.co/v2/co/contraloria/certificado
```

Ce service permet de vérifier le certificat de bonne conduite délivré par la Contraloría General de Colombia en saisissant le type et le numéro de document, et renvoie la date de consultation ainsi que le certificat au format PDF encodé en base64. Il est idéal pour confirmer rapidement la situation d'une personne ou d'une entreprise colombienne.

### En-têtes

| Nom           | Valeur             |
| ------------- | ------------------ |
| Accept        | `application/json` |
| Authorization | `Bearer <token>`   |

### Paramètres

:::info
Pour la liste complète des **types de document** et leurs paramètres obligatoires, consultez le guide [Types de document](/background-check/document-types).
:::

| Nom              | Type   | Obligatoire | Description |
| ---------------- | ------ | ----------- | ----------- |
| `documentType`   | string | Oui         | CC, CE, PA ou PEP. |
| `documentNumber` | string | Oui         | Numéro sans espaces ni points. |
