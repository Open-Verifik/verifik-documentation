# Vérification de certificat — Registro Civil (Chili)

## Référence de l'API

### Endpoint

```
GET https://api.verifik.co/v2/cl/certificate/verify
```

Vérifie un certificat du **Registro Civil (SRCEI)** avec le **folio** et le **code de vérification** (`Código Verificador`).

### En-têtes (Headers)

| Nom           | Valeur             |
| ------------- | ------------------ |
| Accept        | `application/json` |
| Authorization | `Bearer <token>`   |

### Paramètres

| Nom                | Type   | Requis  | Description                                           |
| ------------------ | ------ | ------- | ----------------------------------------------------- |
| `folio`            | string | **Oui** | Folio du certificat (10–15 car.). Ex. : `50070080804`. |
| `verificationCode` | string | **Oui** | Code de vérification. Ex. : `2aacbb9a636a`.           |

### Requête (Request)

<Tabs>
<TabItem value="javascript" label="JavaScript">

```javascript
import axios from 'axios';

const { data } = await axios.get('https://api.verifik.co/v2/cl/certificate/verify', {
  params: { folio: '50070080804', verificationCode: '2aacbb9a636a' },
  headers: { Accept: 'application/json', Authorization: 'Bearer <votre_token>' },
});
console.log(data);
```

</TabItem>
<TabItem value="python" label="Python">

```python
import http.client

conn = http.client.HTTPSConnection("api.verifik.co")
headers = {"Accept": "application/json", "Authorization": "Bearer <votre_token>"}
conn.request("GET", "/v2/cl/certificate/verify?folio=50070080804&verificationCode=2aacbb9a636a", headers=headers)
print(conn.getresponse().read().decode("utf-8"))
```

### Notes

- `folio` : **10 à 15** caractères.
- En **mode sandbox** : `folio=50070080804`, `verificationCode=2aacbb9a636a` ; `folio=9004040100001` → **404**.
