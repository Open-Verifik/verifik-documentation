# Contrôle d'antécédents Europol

## Référence API

### Endpoint

```
https://api.verifik.co/v2/europol
```

Interrogez les données Europol en temps réel avec `documentType` et `documentNumber`, ou `fullName` seul. La réponse indique si une correspondance existe et peut inclure une URL Europol.

**Remarque :** les dates (naissance ou délivrance) doivent être au format `jj/mm/aaaa`.

### En-têtes

| Nom | Valeur |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Paramètres

:::info
Pour la liste complète des **types de document** pris en charge et leurs paramètres obligatoires, consultez le guide [Types de document](/background-check/document-types).
:::

| Nom | Type | Obligatoire | Description |
| --- | --- | --- | --- |
| `documentType`   | string | No       | The document type that you want to request.                                                       |
| `documentNumber` | string | No       | Document number to consult, without spaces or points.                                             |
| `fullName`       | string | No       | Instead of documentType and documentNumber, you can pass the name directly of the person/business. |

### Cas d'usage

- **Lutte contre la fraude :** repérer les personnes signalées avant les opérations sensibles.
- **Vérification d'identité :** valider les informations fournies.
- **Conformité :** KYC et LCB dans la banque, la finance et l'immobilier.
