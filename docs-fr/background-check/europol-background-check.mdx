# Contrôle d'antécédents Europol

## Référence de l'API

### Endpoint

```
https://api.verifik.co/v2/europol
```

Vérifiez en temps réel les données du fichier criminel d'Europol à partir du couple `documentType` + `documentNumber`, ou via `fullName` seul. Les réponses indiquent l'existence d'une correspondance et peuvent inclure une URL Europol directe lorsqu'un résultat est trouvé.

**Note :** les dates (date de naissance ou date de délivrance) doivent être au format `jj/mm/aaaa`.

### En-têtes (Headers)

| Nom            | Valeur              |
| -------------- | ------------------- |
| Content-Type   | `application/json`  |
| Authorization  | `Bearer <token>`    |

### Paramètres

:::info
Pour la liste complète des **types de documents** pris en charge ainsi que de leurs paramètres obligatoires spécifiques, consultez le guide [Types de documents](/background-check/document-types).
:::

| Nom              | Type   | Requis | Description                                                                                          |
| ---------------- | ------ | ------ | ---------------------------------------------------------------------------------------------------- |
| `documentType`   | string | Non    | Type de document à interroger.                                                                       |
| `documentNumber` | string | Non    | Numéro de document à interroger, sans espaces ni points.                                             |
| `fullName`       | string | Non    | À la place de `documentType` et `documentNumber`, vous pouvez transmettre directement le nom complet. |

### Cas d'usage

- **Prévention de la fraude :** identifiez les personnes signalées dans la base d'Europol avant les opérations sensibles.
- **Vérification d'identité :** confirmez la légitimité des données personnelles transmises.
- **Conformité :** alimentez les contrôles KYC et LCB-FT dans la banque, la finance et l'immobilier.

### Notes

- **Vérification dans la base Europol** par document ou par nom complet.
- **Détection de correspondance** pouvant inclure une URL Europol directe le cas échéant.
- **Réponses structurées** pour faciliter l'intégration ; exemples disponibles en JavaScript, Python, PHP et Swift ci-dessus.
