# 哥斯达黎加企业核验

Verifik 的企业核验 API 可基于政府官方数据对哥斯达黎加企业进行身份核实。该接口旨在简化您的 KYB（了解您的企业）流程、防范欺诈，并轻松满足合规要求。

我们专为需要快速、安全且自动化确认企业、合作伙伴或供应商合法性的组织打造此集成。

## 此 API 验证哪些内容？

我们的 API 直接对接官方记录，验证以下内容：

-   **公司名称与 NITE 编号**：支持 *Número de Identificación Tributaria de Empresas*（NITE）。
-   **企业信息**：返回公司名称及登记详情。
-   **企业匹配**：确认企业信息与 NITE 编号相符。

通过验证这些信息，您可以确信对接的企业真实合法且依法登记，从而显著降低欺诈风险并保障合规。

## API 参考

### Endpoint

```
https://api.verifik.co/v2/cr/company
```

使用该接口可核验哥斯达黎加法人实体的 **NITE**（*Número de Identificación Tributaria de Empresas*）记录。请传入 **`documentType`** 为 `NITE` 以及企业 **`documentNumber`**；响应将返回已登记的企业标识与相关字段，适用于 **KYB** 与开通流程。

### 请求头（Headers）

| 名称           | 取值                |
| -------------- | ------------------- |
| Accept         | `application/json`  |
| Authorization  | `Bearer <token>`    |

### 参数（Parameters）

| 名称              | 类型    | 必填 | 描述                              |
| ----------------- | ------- | ---- | --------------------------------- |
| `documentType`    | string  | 是   | 证件类型。允许取值：`NITE`。       |
| `documentNumber`  | string  | 是   | 企业标识。                        |

## 常见使用场景

-   **B2B 客户开通**：在合作伙伴或供应商注册时即时核验企业身份。
-   **金融服务**：在开户或交易前对企业进行身份核验。
-   **合规与尽职调查**：将企业信息核验纳入 KYC/KYB 流程。
-   **电商与平台**：在企业入驻销售前核实其合法身份。
