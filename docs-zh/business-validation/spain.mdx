# 西班牙企业核验

通过官方数据认证西班牙企业，支持 KYB 与合规流程。

适用于验证合作伙伴、供应商与交易对手。

## 本 API 核验内容

查询官方记录以验证：

-   **企业标识**：CIF/NIF 及相关信息。
-   **企业信息**：名称、状态（若可返回）。
-   **一致性**：信息与标识是否匹配。

## API 参考

### Endpoint

```
https://api.verifik.co/v2/es/company
```

使用 `documentNumber`（企业标识符）查询西班牙企业登记信息。

### 标头

| 名称 | 值 |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### 参数

| 名称 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `documentNumber` | string | 是 | 企业标识符（CIF/NIF）。 |
