# 西班牙企业核验

Verifik 的企业核验 API 可基于政府官方数据对西班牙企业进行身份核实。该接口旨在简化您的 KYB（了解您的企业）流程、防范欺诈，并轻松满足合规要求。

我们专为需要快速、安全且自动化确认企业、合作伙伴或供应商合法性的组织打造此集成。

## 此 API 验证哪些内容？

我们的 API 直接对接官方记录，验证以下内容：

-   **公司名称与企业标识**：支持西班牙企业标识（CIF/NIF）。
-   **企业信息**：返回公司名称及状态。
-   **企业匹配**：确认企业信息与企业标识相符。

通过验证这些信息，您可以确信对接的企业真实合法且依法登记，从而显著降低欺诈风险并保障合规。

## API 参考

### Endpoint

```
https://api.verifik.co/v2/es/company
```

### 请求头（Headers）

| 名称           | 取值                |
| -------------- | ------------------- |
| Content-Type   | `application/json`  |
| Authorization  | `Bearer <token>`    |

### 参数（Parameters）

| 名称              | 类型    | 必填 | 描述           |
| ----------------- | ------- | ---- | -------------- |
| `documentNumber`  | string  | 是   | 企业标识。     |

## 常见使用场景

-   **B2B 客户开通**：在合作伙伴或供应商注册时即时核验企业身份。
-   **金融服务**：在开户或交易前对企业进行身份核验。
-   **合规与尽职调查**：将企业信息核验纳入 KYC/KYB 流程。
-   **电商与平台**：在企业入驻销售前核实其合法身份。

## 官方数据源与可靠性

我们直接对接西班牙政府官方数据源，确保您获得经过验证且最新的信息。
每次查询均严格遵循安全与监管标准。

## 主要优势

-   **自动化合规**：在不增加合作方摩擦的前提下简化 KYB 检查并防范欺诈。
-   **即时结果**：在数秒内完成核验，适用于实时企业开通。
-   **可信数据**：采用直接来源于官方记录的数据。
-   **轻松集成**：通过 REST API 或兼容 SDK 即可便捷接入。

## 合规与安全

我们高度重视您的数据安全。Verifik 采用高级加密（HTTPS / TLS 1.3）和严格的隐私管理标准以确保机密性。
