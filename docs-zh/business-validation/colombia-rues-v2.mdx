# 哥伦比亚 — RUES 企业查询 (v2，旧版)

通过旧版 **v2** API 使用 **NIT** 在 **RUES** 中核验哥伦比亚企业，返回基础登记数据。v2 完整档案请用 `GET /v2/co/rues/complete`。新集成请优先 [RUES 企业查询 (v3)](/business-validation/colombia-rues-v3)。

## 应选用哪个接口？(v2)

| | **`GET /v2/co/rues`**（本接口） | **`GET /v2/co/rues/complete`** |
| --- | --- | --- |
| **用途** | 按 NIT 快速核验 | **完整** 企业信息 |
| **参数** | `nit` | `nit` |
| **目录定价** | 较低（如 **0.2** / **0.3** SmartCheck） | 较高（如 **0.3** / **0.4** SmartCheck） |

### 端点

```
GET https://api.verifik.co/v2/co/rues
```

### 请求头

| 名称 | 值 |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 参数

| 名称 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `nit` | string | 是 | 企业税号 (NIT)，无空格或标点。 |

### 说明

- **旧版：** 新项目请优先 **v3**（[基础](/business-validation/colombia-rues-v3) / [完整](/business-validation/rues-complete-v3)）。
- v2 **完整** RUES 数据请用 **`GET /v2/co/rues/complete`**。
