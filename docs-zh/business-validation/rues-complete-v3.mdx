# 🇨🇴 哥伦比亚 — RUES 企业完整核验 (v3)

本接口从哥伦比亚 **RUES**（*Registro Único Empresarial y Social*）按 **NIT** 与 **登记类别（category）** 返回 **完整** 企业档案。若只需 **摘要**，请使用 **`GET /v3/co/rues`**（英文说明：[Colombia RUES (v3)](https://docs.verifik.co/business-validation/colombia-rues-v3)）。

## API 参考

### 端点

```
GET https://api.verifik.co/v3/co/rues-complete
```

应用域名：`GET https://verifik.app/v3/co/rues-complete`

### 请求头

| 名称 | 值 |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 参数

| 名称 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `documentType` | string | 是 | 必须为 **`NIT`**。 |
| `documentNumber` | string | 是 | 税号；服务端会去除非字母数字字符。 |
| `category` | string | 否 | **`RM`**, **`PROP`**, **`RUNEOL`**, **`RNT`**, **`ESAL`**, **`ESOL`**, **`JUEGOS`**, **`EXTRANJERAS`**。省略时为 **`RM`**。 |

### 说明

- 本路由 **仅** 支持上表参数。**`municipality`**、**`chamber`**、**`page`** **不是** **`GET /v3/co/rues-complete`** 的有效查询参数。
- 完整响应示例见英文文档 [RUES Complete Business Validation (v3)](https://docs.verifik.co/business-validation/rues-complete-v3)。
