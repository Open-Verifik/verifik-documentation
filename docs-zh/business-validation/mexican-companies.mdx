# 墨西哥企业核验

Verifik 的企业核验 API 可基于政府官方数据对墨西哥企业进行身份核实。该接口旨在简化您的 KYB（了解您的企业）流程、防范欺诈，并轻松满足合规要求。

我们专为需要快速、安全且自动化确认企业、合作伙伴或供应商合法性的组织打造此集成。

## 此 API 验证哪些内容？

我们的 API 直接对接官方记录，验证以下内容：

-   **公司名称与 FME 编号**：支持 *Folio Mercantil Electrónico*（FME）。
-   **企业信息**：返回公司名称、所在城市及登记详情。
-   **企业匹配**：确认企业信息与 FME 编号相符。

通过验证这些信息，您可以确信对接的企业真实合法且依法登记，从而显著降低欺诈风险并保障合规。

## API 参考

### Endpoint

```
https://api.verifik.co/v2/mx/company
```

该接口用于通过 **FME**（*Folio Mercantil Electrónico*）查询墨西哥法人实体。请使用 GET 方式，传入 `documentType=FME` 与 `documentNumber`，即可从政府工商登记数据源获取法定企业名称与所在城市等官方登记详情。

### 请求头（Headers）

| 名称           | 取值                |
| -------------- | ------------------- |
| Content-Type   | `application/json`  |
| Authorization  | `Bearer <token>`    |

### 参数（Parameters）

| 名称              | 类型    | 必填 | 描述                                |
| ----------------- | ------- | ---- | ----------------------------------- |
| `documentType`    | string  | 是   | 证件类型。允许取值：`FME`。           |
| `documentNumber`  | string  | 是   | FME 编号（例如 `N-2021007002`）。    |
