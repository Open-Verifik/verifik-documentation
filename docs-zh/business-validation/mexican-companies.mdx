# 墨西哥企业核验

Verifik API 使用墨西哥政府官方数据认证企业，简化 KYB、降低欺诈风险并助力合规。

## 核验内容

-   **商号与 FME**（*Folio Mercantil Electrónico*）
-   **登记信息**：商号、城市、注册信息
-   **一致性**：信息是否与 FME 匹配

## API 参考

### 端点

```
https://api.verifik.co/v2/mx/company
```

此接口通过 **FME**（*Folio Mercantil Electrónico*）标识查询墨西哥法人实体。使用 GET，参数为 `documentType=FME` 与 `documentNumber`，可从政府商业登记来源返回法定企业名称与城市等官方注册信息。

### 请求头

| 名称          | 值              |
| ------------- | ---------------- |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### 参数

| 名称 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `documentType` | string | 是 | 允许值：`FME`。 |
| `documentNumber` | string | 是 | FME 标识（如 `N-2021007300`）。 |

### 请求

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/mx/company", {
  params: { documentType: "FME", documentNumber: "N-2021007300" },
  headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os, requests

url = "https://api.verifik.co/v2/mx/company"
headers = {"Accept": "application/json", "Authorization": f"Bearer {os.getenv('VERIFIK_TOKEN')}"}
params = {"documentType": "FME", "documentNumber": "N-2021007300"}
r = requests.get(url, headers=headers, params=params)
print(r.json())
```
