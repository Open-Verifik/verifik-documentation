# 哥伦比亚 — 法定开票人核验

Verifik 法定开票人核验 API 确认哥伦比亚企业是否在 DIAN（*Dirección de Impuestos y Aduanas Nacionales*）登记 **法定开票人**，用于 KYB、防欺诈与合规。

## 本接口核验什么？

对接 DIAN 官方记录：

- **企业 NIT 与电子发票（facturación electrónica）登记**
- **开票人联系方式**：DIAN 公开时返回登记 **email** 等字段
- **登记核查**：确认实体出现在 DIAN **法定开票人** 数据集中（与一般纳税人状态不同 — 见 [DIAN 税务核验](../colombia-dian-verification.mdx)）

## API 参考

### 端点

```
GET https://api.verifik.co/v2/co/company/dian/invoicer
```

应用域名：`GET https://verifik.app/v2/co/company/dian/invoicer`

### 请求头

| 名称 | 值 |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 参数

| 名称 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `documentType` | string | 是 | 必须为 **`NIT`**。 |
| `documentNumber` | string | 是 | 税号 (NIT)。服务端去除空格；可含或省略校验位。API 校验最少 **5** 个字符。 |

### 说明

- 仅 **`GET` 查询**，无请求体。JSON 响应请用 **`Accept`**（非 `Content-Type`）。
- 本路由返回 **法定开票人 / 电子发票登记** 数据（如可用时的 **email**）。一般 **纳税人** 状态与义务请用 **[DIAN 税务核验](../colombia-dian-verification.mdx)**（`GET /v2/co/company/dian`）。
- 当 DIAN 未返回该 NIT 的开票人 **email** 时可能出现 **404**（见集成逻辑）。
