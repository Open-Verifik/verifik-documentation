# 巴拉圭企业核验

Verifik 企业核验 API 使用**巴拉圭政府官方数据**认证企业。**KYB** 更高效、降低欺诈并助力合规。

适用于需要快速确认公司、合作伙伴或供应商**合法性**的企业。

## 本 API 核验内容

直连官方记录，核验：

-   **公司名称与 RUC 号码**：支持 *Registro Único de Contribuyente*（RUC）。
-   **工商信息**：名称与注册详情。
-   **一致性**：确认信息与 RUC 匹配。

## API 参考

### Endpoint

```
https://api.verifik.co/v2/py/company
```

### Headers

| Name          | Value              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parameters

| Name           | Type    | Required | Description                                    |
| -------------- | ------- | -------- | ---------------------------------------------- |
| `documentType` | string  | Yes      | 证件类型。允许值：`RUC`。 |
| `documentNumber` | string | Yes      | 企业标识符。 |
