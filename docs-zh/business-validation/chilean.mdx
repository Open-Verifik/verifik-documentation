# 智利企业查询（RUT）

Verifik 企业验证 API 使用官方政府数据认证智利公司。简化 **KYB**、降低欺诈风险并帮助满足合规要求。

适合需要快速确认企业、合作伙伴或供应商合法性的团队。

通过**证件类型与号码**可获取商号、行业与经济活动，以及获准经营的服务范围等信息。

### Endpoint

```
https://api.verifik.co/v2/cl/company
```

按 **RUT** 查询企业：商号、经济活动与官方登记信息，用于 **KYB** 与供应商尽调。

### Headers

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parameters

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `documentType` | string | Yes | `RUT` |
| `documentNumber` | string | Yes | 企业 RUT（按集成文档格式） |

### Request

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/cl/company", {
  params: { documentType: "RUT", documentNumber: "212957739" },
  headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});
console.log(data);
```

## 验证范围

- **商号与 RUT**：与登记信息一致。
- **经济活动**：代码、类别、增值税与日期等（以来源返回为准）。

## 数据来源

对接智利政府官方来源，注重安全与隐私。
