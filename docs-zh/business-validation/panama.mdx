# 巴拿马企业核验

依据 **RUC**（*Registro Único de Contribuyente*）与 **`dv`**（验证位）查询法人登记信息，适用于 **KYB**、供应商准入与合规审查。

## 本 API 核验内容

- **RUC 与 dv**：税号与验证数字。
- **登记信息**：商号、组织类型、状态、代表人等（以实际返回为准）。

## API 参考

### Endpoint

```
https://api.verifik.co/v2/pa/company
```

按巴拿马 **RUC** 查询企业。参数包括 **`documentType`**（`RUC`）、**`documentNumber`**、**`dv`**。一次 HTTPS 调用即可支撑 **KYB** 与合规流程。

### 请求头

| 名称 | 值 |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 参数

| 名称 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `documentType` | string | 是 | 固定为 `RUC`。 |
| `documentNumber` | string | 是 | RUC 编号（含连字符格式）。 |
| `dv` | string | 是 | 验证位。 |

### 请求

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/pa/company", {
  params: { documentType: "RUC", documentNumber: "155703400-2-2021", dv: "39" },
  headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});
console.log(data);
```
