# 阿根廷企业核验

Verifik 的企业核验 API 可基于政府官方数据对阿根廷企业进行身份核实。该接口旨在简化您的 KYB（了解您的企业）流程、防范欺诈，并轻松满足合规要求。

我们专为需要快速、安全且自动化确认企业、合作伙伴或供应商合法性的组织打造此集成。

## 此 API 验证哪些内容？

我们的 API 直接对接官方记录，验证以下内容：

-   **公司名称与 CUIT 编号**：支持 *Código Único de Identificación Tributaria*（CUIT）。
-   **企业信息**：返回公司名称、法形態、签约日期及经济活动。
-   **经济活动**：提供主要与次要活动代码及描述。
-   **企业匹配**：确认企业信息与 CUIT 编号相符。

通过验证这些信息，您可以确信对接的企业真实合法且依法登记，从而显著降低欺诈风险并保障合规。

## API 参考

### 端点

```
GET https://api.verifik.co/v2/ar/company
```

使用 **`documentType`** `CUIT` 与 **`documentNumber`** 查询阿根廷法人。返回登记字段以支持 **KYB**、**供应商筛查**与 **合规**。

### 请求头

| 名称          | 取值                 |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### 参数

| 名称             | 类型   | 必填 | 描述                                       |
| ---------------- | ------ | ---- | ------------------------------------------ |
| `documentType`   | string | 是 | 证件类型。允许值：`CUIT`。            |
| `documentNumber` | string | 是 | 11 位 CUIT（不含连字符）。               |

## 常见用例

-   **B2B onboarding**：在合作伙伴或供应商注册时即时核验企业。
-   **金融服务**：在开户或处理交易前认证企业。
-   **合规与尽职调查**：作为 KYC/KYB 工作流的一部分验证企业信息。
-   **电商与市场平台**：在允许销售前确认企业合法性。
