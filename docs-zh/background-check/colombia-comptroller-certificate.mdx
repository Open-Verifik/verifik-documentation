# 审计长（Contraloría）证明

## API 参考

### Endpoint

```
https://api.verifik.co/v2/co/contraloria/certificado
```

通过提交证件类型与号码，本服务可核验哥伦比亚总审计长（Contraloría General de Colombia）签发的良好行为证明，并返回查询日期及以 base64 编码的 PDF 证明文件。该服务非常适合用于快速确认某哥伦比亚企业或个人的合规状态。

### 请求头（Headers）

| 名称           | 取值                |
| -------------- | ------------------- |
| Accept         | `application/json`  |
| Authorization  | `Bearer <token>`    |

### 参数（Parameters）

:::info
有关支持的**证件类型**及其专属必填参数的完整清单，请参阅 [证件类型](/background-check/document-types) 指南。
:::

| 名称              | 类型    | 必填 | 描述                                                                  |
| ----------------- | ------- | ---- | --------------------------------------------------------------------- |
| `documentType`    | string  | 是   | 证件类型。允许取值：CC、CE、PA、PEP。                                 |
| `documentNumber`  | string  | 是   | 待查询人员的证件号码，不含空格或圆点。                                 |
