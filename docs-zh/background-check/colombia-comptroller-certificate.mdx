# 总审计署证明

## API 参考

### Endpoint

```
https://api.verifik.co/v2/co/contraloria/certificado
```

该服务可通过输入证件类型和号码，验证由哥伦比亚总审计署（Contraloría General de Colombia）出具的良好品行证明，并返回查询日期以及以 Base64 编码的 PDF 格式证明。适用于快速确认哥伦比亚个人或企业的相关情况。

### 标头

| 名称          | 值                 |
| ------------- | ------------------ |
| Accept        | `application/json` |
| Authorization | `Bearer <token>`   |

### 参数

:::info
支持的**证件类型**及必填参数完整列表请见 [证件类型](/background-check/document-types)。
:::

| 名称             | 类型   | 必填 | 说明 |
| ---------------- | ------ | ---- | ---- |
| `documentType`   | string | 是   | CC、CE、PA 或 PEP。 |
| `documentNumber` | string | 是   | 不含空格或句点。 |
