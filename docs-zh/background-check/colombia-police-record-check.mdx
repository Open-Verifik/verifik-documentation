# 警方犯罪记录查询

## API 参考

### Endpoint

```
https://api.verifik.co/v2/co/policia/consultar
```

凭身份证件号码核查刑事记录及是否在司法当局有未结事项（宪法第248条）。返回姓名与法律状态等信息。

### 标头

| 名称          | 值                 |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### 参数

:::info
支持的**证件类型**及必填参数完整列表请见 [证件类型](/background-check/document-types)。
:::

| 名称             | 类型   | 必填 | 说明 |
| ---------------- | ------ | ---- | ---- |
| `documentType`   | string | 是   | CC 或 CE。 |
| `documentNumber` | string | 是   | 不含空格或句点。 |

  },

  console.error(error);
}
