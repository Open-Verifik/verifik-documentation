# FBI 背景核查

## API 参考

### Endpoint

```
https://api.verifik.co/v2/fbi
```

通过 `documentType` 与 `documentNumber`，或直接使用 `fullName`，对 FBI 公开发布的记录进行检索。当存在匹配时，响应可包含身份字段以及指向该对象的 FBI 公开 URL。

**注意：** 日期字段（出生日期或签发日期）必须采用 `dd/mm/yyyy` 格式。

### 请求头（Headers）

| 名称           | 取值                |
| -------------- | ------------------- |
| Content-Type   | `application/json`  |
| Authorization  | `Bearer <token>`    |

### 参数（Parameters）

:::info
有关支持的**证件类型**及其专属必填参数的完整清单，请参阅 [证件类型](/background-check/document-types) 指南。
:::

| 名称              | 类型    | 必填 | 描述                                                                                              |
| ----------------- | ------- | ---- | ------------------------------------------------------------------------------------------------- |
| `documentType`    | string  | 否   | 待请求的证件类型。                                                                                 |
| `documentNumber`  | string  | 否   | 待查询的证件号码，不含空格或圆点。                                                                 |
| `fullName`        | string  | 否   | 可不传 documentType 与 documentNumber，而直接传入个人或企业的姓名/名称。                            |

### 使用场景

- **身份核验：** 对照 FBI 公开信息核验证件。
- **合规审查：** 用于敏感岗位或交易的筛查。
- **欺诈预防：** 在建立合作关系之前确认身份。
