# FBI 背景核查

## API 参考

### Endpoint

```
https://api.verifik.co/v2/fbi
```

使用 `documentType` 与 `documentNumber`，或 `fullName` 检索 **FBI** 公开记录；可能返回公开页面链接。

**说明：** 日期 `dd/mm/yyyy`。

### 请求头

| 名称 | 值 |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### 参数

:::info
有关支持的**文档类型**及必填参数，请参阅[文档类型](/background-check/document-types)指南。
:::

| 名称 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `documentType`   | string | No       | The document type that you want to request.                                                       |
| `documentNumber` | string | No       | Document number to consult, without spaces or points.                                             |
| `fullName`       | string | No       | Instead of documentType and documentNumber, you can pass the name directly of the person/business. |

### 使用场景

- **身份与证件核验。**
- **合规审查。**
- **欺诈防范。**

```javascript
import axios from "axios";

const options = {
  method: "GET",
  url: "https://api.verifik.co/v2/fbi",
  params: { documentType: "CC", documentNumber: "80251972" },
  headers: {
    Accept: "application/json",
    Authorization: "jwt <tu_token>",
  },
};

try {
  const { data } = await axios.request(options);
  console.log(data);
} catch (error) {
  console.error(error);
}
```

```python
import http.client
```
