# 犯罪记录证明（阿根廷）

## API 参考

### Endpoint

```
GET https://api.verifik.co/v2/ar/certificate/verify
```

使用官方 PDF 上的 **申请代码**（`Solicitud`）和 **安全代码**（`Cod. Segur.`）验证阿根廷 **DNREC** 犯罪记录证明。

### 请求头（Headers）

| 名称           | 取值               |
| -------------- | ------------------ |
| Accept         | `application/json` |
| Authorization  | `Bearer <token>`   |

### 参数（Parameters）

| 名称            | 类型   | 必填   | 描述                                              |
| --------------- | ------ | ------ | ------------------------------------------------- |
| `requestCode`   | string | **是** | 申请代码（`Solicitud`）。示例：`02118499487`。     |
| `securityCode`  | string | **是** | 安全代码（`Cod. Segur.`）。示例：`371488F861`。   |

### 请求（Request）

<Tabs>
<TabItem value="javascript" label="JavaScript">

```javascript
import axios from 'axios';

const options = {
  method: 'GET',
  url: 'https://api.verifik.co/v2/ar/certificate/verify',
  params: { requestCode: '02118499487', securityCode: '371488F861' },
  headers: { Accept: 'application/json', Authorization: 'Bearer <您的令牌>' },
};

const { data } = await axios.request(options);
console.log(data);
```

</TabItem>
<TabItem value="python" label="Python">

```python
import http.client

conn = http.client.HTTPSConnection("api.verifik.co")
headers = {"Accept": "application/json", "Authorization": "Bearer <您的令牌>"}
conn.request("GET", "/v2/ar/certificate/verify?requestCode=02118499487&securityCode=371488F861", headers=headers)
print(conn.getresponse().read().decode("utf-8"))
```

### 说明

- 两个参数均为必填；否则返回 **409** `MissingParameter`。
- **沙盒模式**：`requestCode=02118499487`，`securityCode=371488F861`；`requestCode=90040401` → **404**。
