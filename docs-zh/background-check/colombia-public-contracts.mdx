# 公共合同

## API 参考

### Endpoint

```
https://api.verifik.co/v2/co/contracts
```

公共合同服务允许用户通过提交证件号码与证件类型，检索与某人或某企业相关的公共合同清单。响应中包含每份合同的关键信息，例如承包方信息、合同金额、相关机构、合同期限等。

该服务有助于提升透明度，便于追踪并监督在哥伦比亚与政府签订的公共合同。

### 请求头（Headers）

| 名称           | 取值                |
| -------------- | ------------------- |
| Content-Type   | `application/json`  |
| Authorization  | `Bearer <token>`    |

### 参数（Parameters）

:::info
有关支持的**证件类型**及其专属必填参数的完整清单，请参阅 [证件类型](/background-check/document-types) 指南。
:::

| 名称              | 类型    | 必填 | 描述                                                       |
| ----------------- | ------- | ---- | ---------------------------------------------------------- |
| `documentType`    | string  | 是   | 证件类型。允许取值：CC、NIT。                               |
| `documentNumber`  | string  | 是   | 待查询的证件号码。                                          |
