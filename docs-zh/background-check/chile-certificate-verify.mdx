# 民事登记处证明核验（智利）

## API 参考

### Endpoint

```
GET https://api.verifik.co/v2/cl/registro-civil/certificado
```

使用 **folio** 和 **验证码**（`Código Verificador`）核验 **Registro Civil（SRCEI）** 证明。

### 请求头（Headers）

| 名称           | 取值               |
| -------------- | ------------------ |
| Accept         | `application/json` |
| Authorization  | `Bearer <token>`   |

### 参数（Parameters）

| 名称                | 类型   | 必填   | 描述                                           |
| ------------------- | ------ | ------ | ---------------------------------------------- |
| `folio`             | string | **是** | 证明 folio（10–15 位）。示例：`50070080804`。   |
| `verificationCode`  | string | **是** | 验证码。示例：`2aacbb9a636a`。                 |

### 请求（Request）

<Tabs>
<TabItem value="javascript" label="JavaScript">

```javascript
import axios from 'axios';

const { data } = await axios.get('https://api.verifik.co/v2/cl/registro-civil/certificado', {
  params: { folio: '50070080804', verificationCode: '2aacbb9a636a' },
  headers: { Accept: 'application/json', Authorization: 'Bearer <您的令牌>' },
});
console.log(data);
```

</TabItem>
<TabItem value="python" label="Python">

```python
import http.client

conn = http.client.HTTPSConnection("api.verifik.co")
headers = {"Accept": "application/json", "Authorization": "Bearer <您的令牌>"}
conn.request("GET", "/v2/cl/registro-civil/certificado?folio=50070080804&verificationCode=2aacbb9a636a", headers=headers)
print(conn.getresponse().read().decode("utf-8"))
```

</TabItem>

### 说明

- `folio`：**10 至 15** 位字符。
- **沙盒模式**：`folio=50070080804`，`verificationCode=2aacbb9a636a`；`folio=9004040100001` → **404**。
