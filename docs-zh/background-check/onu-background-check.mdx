# 联合国（ONU）背景核查

## API 参考

### Endpoint

```
https://api.verifik.co/v2/onu
```

通过 `documentType` 与 `documentNumber`，或直接使用 `fullName`，查询联合国制裁名单中是否存在某人或某机构。返回结果可用于全球合规、KYC、AML 等流程，对交易对手进行筛查。

**注意：** 日期字段（出生日期或签发日期）必须采用 `dd/mm/yyyy` 格式。

### 请求头（Headers）

| 名称           | 取值                |
| -------------- | ------------------- |
| Content-Type   | `application/json`  |
| Authorization  | `Bearer <token>`    |

### 参数（Parameters）

:::info
有关支持的**证件类型**及其专属必填参数的完整清单，请参阅 [证件类型](/background-check/document-types) 指南。
:::

| 名称              | 类型    | 必填 | 描述                                                                                              |
| ----------------- | ------- | ---- | ------------------------------------------------------------------------------------------------- |
| `documentType`    | string  | 否   | 待请求的证件类型。                                                                                 |
| `documentNumber`  | string  | 否   | 待查询的证件号码，不含空格或圆点。                                                                 |
| `fullName`        | string  | 否   | 可不传 documentType 与 documentNumber，而直接传入个人或企业的姓名/名称。                            |

### 使用场景

- **国际合规：** 在客户开通或交易前对照联合国制裁名单核验个人与实体。
- **风险评估：** 降低与受制裁对象相关的财务与声誉风险。
- **监管项目：** 满足银行、金融、保险等行业的 KYC/AML 要求。
