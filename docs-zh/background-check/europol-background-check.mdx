# 欧洲刑警组织（Europol）背景核查

## API 参考

### Endpoint

```
https://api.verifik.co/v2/europol
```

通过 `documentType` 与 `documentNumber`，或单独使用 `fullName`，对 Europol 犯罪记录数据进行实时核查。响应将指明是否命中匹配，并在适用时附带指向 Europol 的官方参考 URL。

**注意：** 日期字段（出生日期或签发日期）必须采用 `dd/mm/yyyy` 格式。

### 请求头（Headers）

| 名称           | 取值                |
| -------------- | ------------------- |
| Content-Type   | `application/json`  |
| Authorization  | `Bearer <token>`    |

### 参数（Parameters）

:::info
有关支持的**证件类型**及其专属必填参数的完整清单，请参阅 [证件类型](/background-check/document-types) 指南。
:::

| 名称              | 类型    | 必填 | 描述                                                                                              |
| ----------------- | ------- | ---- | ------------------------------------------------------------------------------------------------- |
| `documentType`    | string  | 否   | 待请求的证件类型。                                                                                 |
| `documentNumber`  | string  | 否   | 待查询的证件号码，不含空格或圆点。                                                                 |
| `fullName`        | string  | 否   | 可不传 documentType 与 documentNumber，而直接传入个人或企业的姓名/名称。                            |

### 使用场景

- **欺诈预防：** 在敏感交易前识别在 Europol 犯罪数据库中被标记的人员。
- **身份核验：** 校验所提交的个人信息是否真实合法。
- **合规审查：** 支持银行、金融与房地产行业的 KYC 与 AML 检查。
