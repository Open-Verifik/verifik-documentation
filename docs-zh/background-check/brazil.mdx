# 巴西

## API 参考

### Endpoint

```
https://api.verifik.co/v2/br/background-check
```

返回巴西个人的详细信息，可能包括关联姓名、是否可出具报告、认证编号、证件信息、姓名与 base64 PDF 等。

可用于招聘、尽调、合规与反欺诈。

### 标头

| 名称          | 值                 |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### 参数

:::info
支持的**证件类型**及必填参数完整列表请见 [证件类型](/background-check/document-types)。
:::

| 名称           | 类型   | 必填 | 说明 |
| -------------- | ------ | ---- | ---- |
| `documentType` | string | 是   | 仅支持 CPF。 |
| `documentNumber` | string | 是 | 要查询的证件号。 |
| `dateOfBirth` | string | 是   | 出生日期（dd/mm/yyyy）。 |

### 功能

-   **CPF Verification**: Verify the authenticity of Brazilian CPF documents
-   **Complete Background Data**: Returns full name, document details, certification number, and PDF report
-   **Structured Response**: Organized data format for easy integration
-   **Multiple Programming Languages**: Support for JavaScript, Python, PHP, and Swift
-   **Error Handling**: Comprehensive error responses for various scenarios
