# 🇨🇷 哥斯达黎加 — 车辆信息

本服务可使用**车牌**返回在哥斯达黎加登记车辆的详细信息。返回数据通常包括当前车主、车辆参数（排量、估价、毛重、净重、载客数、功率、驱动方式）、**违章**记录，以及在数据源公开时的**历史车主**。

适用于车队核查、保险与合规工作流。

## API 参考

### Endpoint

```
https://api.verifik.co/v2/cr/vehicle
```

发送 GET 请求并附带 **`plate`** 查询参数（不含空格或标点）。成功响应将包含结构化的车辆数据、车主字段，以及在数据可用时的历史数组。

### 请求头（Headers）

| 名称           | 取值                |
| -------------- | ------------------- |
| Accept         | `application/json`  |
| Authorization  | `Bearer <token>`    |

### 参数（Parameters）

| 名称 | 类型 | 必填 | 描述 | 示例 |
| ---- | ---- | ---- | ---- | ---- |
| `plate` | string | **是** | 待查询的车牌，不含空格或圆点。 | `ABC123` |

## 功能特性

- **车辆信息检索**：从哥斯达黎加车辆登记系统获取完整车辆详情。
- **车牌核验**：核验在哥斯达黎加登记的车辆牌号。
- **当前车主信息**：获取当前车主详情。
- **车辆参数**：获取排量、估价、重量、载客数、功率与驱动方式。
- **违章历史**：获取完整的交通违章历史记录。
- **车主历史**：按时间顺序查看所有历史车主。
- **司法机关**：获取处理违章的司法机关信息。
- **罚单信息**：获取违章的罚单编号与摘要编号。
- **多语言支持**：兼容 JavaScript、Python、PHP 与 Swift。
- **实时数据**：获取最新的车辆信息。
- **完善错误处理**：针对多种场景返回详细错误响应。
- **哥斯达黎加登记接入**：直接对接哥斯达黎加官方车辆数据库。
