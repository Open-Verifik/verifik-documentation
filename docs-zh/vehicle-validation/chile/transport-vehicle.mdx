# 🇨🇱 智利 — 公共交通车辆

智利公共交通车辆核验服务可查询 **RNT** 登记，获取公交车及其他公共交通号牌的运营商、服务与车辆详情。

## 公共交通车辆查询

### 端点

```
GET https://api.verifik.co/v2/cl/transport-vehicle
```

### 请求头（Headers）

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 参数（Parameters）

| name | type | required | description |
| --- | --- | --- | --- |
| `plate` | string | yes | Chilean license plate. |

### 请求（Request）

<Tabs>
  <TabItem value="javascript" label="JavaScript">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/cl/transport-vehicle", {
	params: { plate: "BBCC12" },
	headers: {
		Accept: "application/json",
		Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
	},
});
console.log(data);
```

  </TabItem>
</Tabs>

### 说明

- Sandbox: plates `BBCC12`–`BBCC21`; `ERR404` → 404.
