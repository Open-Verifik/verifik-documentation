# 🇨🇱 智利 — 车辆 SOAP

通过 **号牌** 和 **保单号**（AACH 登记）查询智利车辆的 **SOAP**（强制人身意外保险）证书。

## 车辆 SOAP 查询

### 端点

```
GET https://api.verifik.co/v2/cl/vehicle-soap
```

### 标头

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 参数

| name | type | required | description |
| --- | --- | --- | --- |
| `plate` | string | yes | 智利号牌（4–6 位）。 |
| `policyNumber` | string | yes | 保单号（使用后四位）。 |

### 请求

<Tabs>
  <TabItem value="javascript" label="JavaScript">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/cl/vehicle-soap", {
	params: { plate: "HTRT86", policyNumber: "94596506" },
	headers: {
		Accept: "application/json",
		Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
	},
});
console.log(data);
```

  </TabItem>
</Tabs>

- 沙盒：`AB1001`–`AB1010`（AB1004 = VENCIDO）；`HTRT86`/`94596506` 演示；`ERR404` → 404。
