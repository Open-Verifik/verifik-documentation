# 🇨🇱 智利 — 车辆 SOAP

智利车辆 **SOAP**（*Seguro Obligatorio de Accidentes Personales*，强制人身意外保险）查询用于验证车辆的强制性人身意外保险。通过**号牌**和**保单号**从 AACH 登记获取保险公司、车主、有效期及车辆属性。

## 车辆 SOAP 查询

### 端点

```
GET https://api.verifik.co/v2/cl/vehicle-soap
```

通过**号牌**（`plate`）和**保单号**（`policyNumber`；服务在 AACH 登记中使用后四位）查询。常见字段包括 **status**、**insuranceCompany**、**ownerName**、**ownerRut**、**validFrom**、**validTo**、**brand**、**model**、**manufactureYear**、**engineNumber**、**premium** — 适用于**保险合规**、**车队接入**和**理赔**流程。

### 标头

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 参数

| Name | Type | Required | Description | Example |
| --- | --- | --- | --- | --- |
| `plate` | string | **Yes** | 智利号牌（Patente），4–6 位字母数字。 | `HTRT86` |
| `policyNumber` | string | **Yes** | 保单号；AACH SOAP 查询使用后四位。 | `94596506` |

### 请求

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/cl/vehicle-soap", {
  params: { plate: "HTRT86", policyNumber: "94596506" },
  headers: {
    Accept: "application/json",
    Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
  },
});
console.log(data);
```

### 说明

- **沙盒模式：** 号牌 **`BBCC12`**–**`BBCC21`** 返回固定 SOAP 配置（`BBCC15` = **VENCIDO**）；**`HTRT86`** + **`policyNumber=94596506`** 返回演示配置；**`ERR404`** 返回 **404**；未映射号牌返回随机 **200**。
- **沙盒示例：** `BBCC12`/`94590001`（轿车，MAPFRE）、`BBCC18`/`94590007`（皮卡，CHUBB）、`HTRT86`/`94596506`（TOYOTA COROLLA 演示）。
- 智利**普通车辆**查询见 [🇨🇱 智利 — 车辆](/verifik-zh/vehicle-validation/chile)。
- 登记**可用性**因环境而异；故障可能返回 **409** 超时。
