# 🇨🇱 智利 - 车辆技术检验

此 endpoint 用于按车牌查询智利车辆的 **技术检验 / 检验历史**。一般车辆登记信息请使用 `v2/cl/vehicle`，SOAP 强制保险证书请使用 `v2/cl/vehicle-soap`。

### Endpoint

```text
GET https://api.verifik.co/v3/cl/vehicle
```

### 参数

| 名称 | 类型 | 必填 | 描述 | 示例 |
| --- | --- | --- | --- | --- |
| `plate` | string | **是** | v3 支持的智利车牌格式，例如四个字母加两个数字，或两个字母加四个数字。 | `XH6640` |

### 请求示例

```javascript
const { data } = await axios.get("https://api.verifik.co/v3/cl/vehicle", {
	params: { plate: "XH6640" },
	headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});
```

{
	"data": {
		"plate": "XH6640",
		"vehicle": {

			"type": "CAMIONETA (CAP. DE CARGA MENOR A 1.750 KGS.) [B] (A2)",
			"mark": "PEUGEOT",

### 错误

- `404 NotFound`：未找到记录。
- `409 Invalid_plate_format`：车牌格式无效。
- `409 timeout_data_source`：外部数据源不可用或超时。

### Notes

`XH6640`、`FHDJ31` 和 `DCCH18` 返回固定资料。`BB985` 返回 `409 Invalid_plate_format`。
