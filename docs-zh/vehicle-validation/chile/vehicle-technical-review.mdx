# 🇨🇱 智利 - 车辆技术检验

当您需要按车牌查询智利车辆的 **技术检验 / 检查** 数据时使用此 endpoint：车辆详情、检验站、证书编号、到期日、检验状态，以及在可用时返回的技术规格字段。

相关智利车辆 endpoint：

- **车辆信息 (`v2/cl/vehicle`)**：一般登记数据，例如品牌、型号、车主 RUT、发动机、底盘、罚单与失窃标记。
- **车辆技术检验 (`v3/cl/vehicle`)**：技术检验状态、历史，以及在可用时返回的技术规格数据。
- **车辆 SOAP (`v2/cl/vehicle-soap`)**：按车牌与保单号查询强制人身意外保险证书。

### Endpoint

```text
GET https://api.verifik.co/v3/cl/vehicle
```

返回某车牌的智利技术检验数据，包括车辆属性、检验历史，以及在可用时返回的可选 `technicalSheet`（`engine`、`performance`、`dimensions`）。适用于车队合规、保险、理赔或车辆入驻流程。

### Headers

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parameters

| Name | Type | Required | Description | Example |
| --- | --- | --- | --- | --- |
| `plate` | string | **Yes** | v3 支持的智利车牌格式，例如四个字母加两个数字，或两个字母加四个数字。 | `XH6640` |

### Request

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v3/cl/vehicle", {
	params: { plate: "XH6640" },
	headers: {
		Accept: "application/json",
		Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
	},
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os, requests

url = "https://api.verifik.co/v3/cl/vehicle"
headers = {"Accept": "application/json", "Authorization": f"Bearer {os.getenv('VERIFIK_TOKEN')}"}
params = {"plate": "XH6640"}
r = requests.get(url, headers=headers, params=params)
print(r.json())
```

### Notes

- 当无匹配技术数据时，`technicalSheet` 可能为 `null` 或部分填充。
- `XH6640`、`FHDJ31` 和 `DCCH18` 返回稳定的技术检验资料。
- `BB985` 因不符合 v3 车牌格式而返回 `409 Invalid_plate_format`。
