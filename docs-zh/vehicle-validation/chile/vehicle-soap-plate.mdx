# 🇨🇱 智利 — 车辆 SOAP（仅车牌）

**仅用车牌**查询智利 **SOAP**（*Seguro Obligatorio de Accidentes Personales*）。服务通过 Boostr 查询 SOAP，有保单号时经 AACH  enrichment，无 SOAP 时从 Carvuk 返回车辆属性。

源响应时的**嵌套**载荷：

- **`plate`** — 智利号牌（Patente），**4–6** 位字母数字
- **`message`** — 找到 SOAP 时为空；车辆存在但无有效 SOAP 时为 `No_active_SOAP_certificate_found_for_this_plate`
- **`soap`** — 找到时：**`status`**, **`policyNumber`**, **`insuranceCompany`**, **`validFrom`**, **`validTo`**, **`folioNumber`**, **`ownerName`**, **`ownerRut`**, **`premium`**
- **`vehicle`** — **`brand`**, **`model`**, **`year`**, **`type`**, **`engineNumber`**
- Verifik **签名**响应

适用于仅知车牌的**保险合规**、**车队接入**和**理赔**流程。

### 端点

```
GET v2/cl/vehicle-soap-plate
```

仅通过**车牌**（`plate`）查询。**`soap`** 含 **status**、**insuranceCompany**、**ownerName**、**validFrom**、**validTo**；**`vehicle`** 含 **brand**、**model**、**year** 等。

### 标头

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 参数

| Name | Type | Required | Description | Example |
| --- | --- | --- | --- | --- |
| `plate` | string | **Yes** | 智利号牌（Patente），4–6 位字母数字。 | `HTRT86` |

### 说明

- **无有效 SOAP** 返回 **HTTP 200**、空 **`soap`** 及 **`message: No_active_SOAP_certificate_found_for_this_plate`** — 不是 404。
- **Live 车牌：** `CLTK27`、`HTRT86`、`RRWX51`、`THHV64`、`CSXZ52`、`CSXZ57` 有有效 SOAP；`CSXZ27` 仅有车辆无 SOAP。
- **沙盒：** **`AB1001`**–**`AB1010`**、**`HTRT86`** 返回稳定嵌套配置；**`ERR404`** → **404**。
- 车牌+保单查询见 [车辆 SOAP（车牌+保单）](/verifik-zh/vehicle-validation/chile/vehicle-soap)。
- 登记**可用性**因环境而异；故障可能返回 **409** 超时。
