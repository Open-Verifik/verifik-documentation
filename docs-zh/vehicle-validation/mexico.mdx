# 墨西哥

### Endpoint

```
https://api.verifik.co/v2/mx/vehiculo/placa
```

车辆信息服务可基于墨西哥车辆的车牌返回其详细信息。该服务专门返回车辆的品牌、车型、年份、VIN 及其他相关信息。请通过 GET 请求将车牌作为 `plate` 查询参数传入；命中匹配时，响应将返回与登记相关的字段。

### 请求头（Headers）

| 名称           | 取值                |
| -------------- | ------------------- |
| Content-Type   | `application/json`  |
| Authorization  | `Bearer <token>`    |

### 参数（Parameters）

| 名称   | 类型    | 必填 | 描述                          | 示例       |
| ------ | ------- | ---- | ----------------------------- | ---------- |
| plate  | String  | 是   | 待查询的车辆牌号。             | `ABC1234`  |

### 请求（Request）

```javascript
import axios from 'axios';

const options = {
  method: 'GET',
  url: 'https://api.verifik.co/v2/mx/vehiculo/placa',
  params: {plate: 'ABC1234'},
  headers: {
    Accept: 'application/json',
    Authorization: 'Bearer <your_token>'
  }
};

try {
  const { data } = await axios.request(options);
  console.log(data);
} catch (error) {
  console.error(error);
}
```

### 响应（Response）

```json
{
  "success": true,
  "data": {
    "plate": "ABC1234",
    "make": "Nissan",
    "model": "Sentra",
    "year": "2021",
    "engine": "2.0L",
    "transmission": "CVT",
    "fuelType": "Gasoline",
    "vin": "1N4AL3AP8JC123456",
    "registrationDate": "10/08/2021",
    "status": "active",
    "owner": "Carlos Rodríguez",
    "state": "Jalisco",
    "city": "Guadalajara"
  }
}
```

### 错误响应

```json
{
  "success": false,
  "error": "Vehicle not found",
  "code": "VEHICLE_NOT_FOUND"
}
```

## 使用场景

- **保险核验**：在保险流程中核验车辆详情。
- **车辆经销商**：在交易前核验车辆信息。
- **车队管理**：追踪企业车辆信息。
- **执法用途**：核验车辆登记与车主信息。
- **尽职调查**：在交易中核验车辆信息。
