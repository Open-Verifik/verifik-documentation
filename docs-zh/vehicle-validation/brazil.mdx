# 🇧🇷 巴西 — 车辆信息

## 车辆信息查询

### Endpoint

```
GET https://api.verifik.co/v2/br/vehicle
```

给定巴西**车牌**，**Verifik** 在一次请求中返回结构化车辆数据，便于将实物资产与已验证、已规范化的属性关联。适用于**车队与出行入驻**、**保险报价与理赔**、**担保融资与抵押审核**，以及以车牌为主要标识的**市场或经销商核验**。

Verifik API 返回该车牌对应的**规范化车辆记录**，包括**车牌**、**品牌**、**型号**、**车型年款**与**制造年份**、**底盘（VIN）**、**发动机**、**颜色**、**车身类型**、**燃料**、**变速箱**、**车门数**、**制造商或工厂**、适用于该车辆的 **FIPE 代码**，以及该登记存在时的**违规或限制指标**。用于在继续流程前确认车辆与官方信息一致。

### 请求头（Headers）

| 名称           | 取值                |
| -------------- | ------------------- |
| Accept         | `application/json`  |
| Authorization  | `Bearer <token>`    |

### 参数（Parameters）

| 名称 | 类型 | 必填 | 描述 |
| ---- | ---- | ---- | ---- |
| `plate` | string | **是** | 待查询车牌（字母与数字；大小写会规范化）。示例：`ABC1D23`。 |

### 备注

- 部分账户还会启用**按车牌扩展技术表**产品；启用时请使用 AppFeature 目录中给出的 URL。
