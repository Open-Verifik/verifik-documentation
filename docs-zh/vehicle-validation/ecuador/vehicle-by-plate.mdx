# 🇪🇨 厄瓜多尔 — 车辆核验

此服务根据车牌（**`plate`**）返回在厄瓜多尔登记的车辆信息：品牌、型号、年份、类型、VIN、发动机号、检验元数据，以及在可用时返回的技术规格字段。

### Endpoint

```
GET https://api.verifik.co/v3/ec/vehiculo/placa
```

按 **`plate`**（无空格或句点）查询厄瓜多尔车辆。响应在可用时可包含 `brand`、`model`、`year`、`vin`、`engineNumber`、`vehicleClass`、`vehicleType`、`color`、`serviceType`、`revision` 和 `technicalSheet`（`engine`、`performance`、`dimensions`）。适用于**核验**、**保险**、**车队**与**合规**流程。

### Headers

| Name          | Value              |
| ------------- | ------------------ |
| Accept | `application/json` |
| Authorization | `Bearer <token>`   |

### Parameters

| Name | Type | Required | Description | Example |
| ---- | ---- | -------- | ----------- | ------- |
| `plate` | string | **Yes** | 要查询的车牌，不含空格或句点。 | `PBB3239` |

### Request

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from 'axios';

const { data } = await axios.get('https://api.verifik.co/v3/ec/vehiculo/placa', {
  params: { plate: 'PBB3239' },
  headers: {
    Accept: 'application/json',
    Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
  },
});
console.log(data);
```

  <TabItem value="python" label="Python">

```python
import os
import requests

url = "https://api.verifik.co/v3/ec/vehiculo/placa"
headers = {"Accept": "application/json", "Authorization": f"Bearer {os.getenv('VERIFIK_TOKEN')}"}
params = {"plate": "PBB3239"}
response = requests.get(url, headers=headers, params=params)
print(response.json())
```
