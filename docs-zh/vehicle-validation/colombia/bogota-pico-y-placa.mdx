### 端点

```
GET https://api.verifik.co/v2/co/vehiculo/pico-y-placa
```

针对在**波哥大**登记的车辆，获取 **Pico y Placa**（按号牌**末位**限行）的**实时**信息。若 **`plate`** 存在**例外**（残障等获批情形），响应中可能包含相关信息。

### 请求头

| 名称 | 值 |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 参数

| 名称 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `plate` | string | 是 | 待查车牌（无空格，波哥大号牌格式）。 |

### 请求

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/vehiculo/pico-y-placa", {
  params: { plate: "ABC123" },
  headers: { 
    Accept: "application/json", 
    Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` 
  },
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os, requests

url = "https://api.verifik.co/v2/co/vehiculo/pico-y-placa"
headers = {"Accept": "application/json", "Authorization": f"Bearer {os.getenv('VERIFIK_TOKEN')}"}
params = {"plate": "ABC123"}
r = requests.get(url, headers=headers, params=params)
print(r.json())
```

### 说明

- 仅适用于在**波哥大**登记的车辆。
- 限行依据号牌**末位数字**。
- 存在**特别许可**（残障、应急等）时可能作为例外返回。
- **`ACTIVO`** 表示例外当前有效（示例）。
