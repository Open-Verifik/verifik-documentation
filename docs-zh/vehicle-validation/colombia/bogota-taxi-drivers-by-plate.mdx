### Endpoint

```
GET https://api.verifik.co/v2/co/bogota/taxi-drivers/plate
```

本接口返回波哥大出租车驾驶员与营运卡相关信息。将车辆 **车牌（plate）** 作为 URL 最后一段 **路径** 传递（例如 `…/plate/WMN482`），不要使用查询参数。车牌为 5–6 位字符，不含空格或英文句点。

### 请求头（Headers）

| 名称 | 取值 |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 参数（Parameters）

| 名称 | 类型 | 必填 | 描述 |
| --- | --- | --- | --- |
| `plate` | string | 是 | `/plate/` 后的路径段。5–6 个字符；不含空格或圆点。 |

### 请求（Request）

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/bogota/taxi-drivers/plate/WMN482", {
  headers: {
    Accept: "application/json",
    Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
  },
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os, requests

url = "https://api.verifik.co/v2/co/bogota/taxi-drivers/plate/WMN482"
headers = {"Accept": "application/json", "Authorization": f"Bearer {os.getenv('VERIFIK_TOKEN')}"}
r = requests.get(url, headers=headers)
print(r.json())
```
