### Endpoint

```
https://api.verifik.co/v2/co/simit/resoluciones
```

SIMIT 决议服务可获取哥伦比亚某特定个人或企业的交通违章决议详情。响应中将返回违章人姓名、决议状态、决议日期及应付金额等信息。

### 请求头（Headers）

| 名称 | 取值 |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 参数（Parameters）

| 名称 | 类型 | 必填 | 描述 |
| --- | --- | --- | --- |
| `documentType` | string | 是 | 证件类型。允许取值：`CC`、`PA`、`CE`、`TI`、`RC`。 |
| `documentNumber` | string | 是 | 待查询的证件号码，不含空格或圆点。 |

### 请求（Request）

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/simit/resoluciones", {
  params: { 
    documentType: "CC", 
    documentNumber: "123456789" 
  },
  headers: { 
    Accept: "application/json", 
    Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` 
  },
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os, requests

url = "https://api.verifik.co/v2/co/simit/resoluciones"
headers = {"Accept": "application/json", "Authorization": f"Bearer {os.getenv('VERIFIK_TOKEN')}"}
```

### 备注

- SIMIT 决议是与交通违章相关的官方裁决。
- "Pendiente de pago" 状态表示该决议待支付。
- 决议编号用于追踪具体的行政裁决。
- 金额以哥伦比亚比索（COP）计。
