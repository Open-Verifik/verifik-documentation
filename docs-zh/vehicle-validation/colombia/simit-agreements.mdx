### Endpoint

```
https://api.verifik.co/v2/co/simit/acuerdos
```

SIMIT 分期付款协议服务可高效检索哥伦比亚 SIMIT 系统中与交通违章相关的分期付款协议。通过提交相关参数，用户可获取分期付款协议的完整信息，包括违章人详情、决议编号与应付金额等。

### 请求头（Headers）

| 名称 | 取值 |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 参数（Parameters）

| 名称 | 类型 | 必填 | 描述 |
| --- | --- | --- | --- |
| `documentType` | string | 是 | 证件类型。允许取值：`CC`、`PA`、`CE`、`TI`、`RC`。 |
| `documentNumber` | string | 是 | 待查询的证件号码，不含空格或圆点。 |

### 请求（Request）

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/simit/acuerdos", {
  params: { 
    documentType: "CC", 
    documentNumber: "123456789" 
  },
  headers: { 
    Accept: "application/json", 
    Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` 
  },
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os, requests

url = "https://api.verifik.co/v2/co/simit/acuerdos"
headers = {"Accept": "application/json", "Authorization": f"Bearer {os.getenv('VERIFIK_TOKEN')}"}
params = {"documentType": "CC", "documentNumber": "123456789"}
r = requests.get(url, headers=headers, params=params)
print(r.json())
```

### 备注

- SIMIT 分期付款协议允许违章人分期支付交通罚款。
- "Acuerdo de pago" 状态表示存在有效的分期付款协议。
- `permitePago: "N"` 表示当前不允许支付。
- 金额以哥伦比亚比索（COP）计。
- 决议编号用于追踪具体协议。
