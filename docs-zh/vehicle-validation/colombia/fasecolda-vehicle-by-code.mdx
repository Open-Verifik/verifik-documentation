### 端点

```
GET https://api.verifik.co/v2/co/fasecolda/values-by-code
```

通过 **查询参数** **`codeFasecolda`**（目录代码）获取车辆属性与 Fasecolda 商业价格信息。

### 请求头

| 名称 | 值 |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 参数

| 名称 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `codeFasecolda` | string | 是 | Fasecolda 车辆配置代码。 |

### 请求

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/fasecolda/values-by-code", {
  params: { codeFasecolda: "08032036" },
  headers: {
    Accept: "application/json",
    Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
  },
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os, requests

url = "https://api.verifik.co/v2/co/fasecolda/values-by-code"
headers = {"Accept": "application/json", "Authorization": f"Bearer {os.getenv('VERIFIK_TOKEN')}"}
params = {"codeFasecolda": "08032036"}
r = requests.get(url, headers=headers, params=params)
print(r.json())
```

### 说明

- Fasecolda 代码标识车型与配置组合。
- 返回详细程度通常与按车牌查询（`values-by-plate`）相当。
- 金额为哥伦比亚比索（COP）。
- **`bcpp`** 为基准商业基础价。
- **`estado: "USADO"`** 为二手车价值分类示例。
