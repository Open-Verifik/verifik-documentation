### 端点

```
https://api.verifik.co/v2/co/simit/comparendos
```

交通违章服务提供哥伦比亚境内与特定车辆相关的违章详细信息。输入车牌号即可查看每条违章的状态、日期、地点及违章类型。

### 请求头

| 名称 | 值 |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 参数

| 名称 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `documentType` | string | 是 | 证件类型。有效值：`CC`、`PA`、`CE`、`TI`、`RC`。 |
| `documentNumber` | string | 是 | 要查询的证件号码，不含空格或句号。 |

### 请求

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/simit/comparendos", {
  params: { 
    documentType: "CC", 
    documentNumber: "123456789" 
  },
  headers: { 
    Accept: "application/json", 
    Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` 
  },
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os, requests

url = "https://api.verifik.co/v2/co/simit/comparendos"
headers = {"Accept": "application/json", "Authorization": f"Bearer {os.getenv('VERIFIK_TOKEN')}"}
params = {"documentType": "CC", "documentNumber": "123456789"}
r = requests.get(url, headers=headers, params=params)
print(r.json())
```

### 说明

- SIMIT 是哥伦比亚综合交通违章管理系统。
- 「Comparendo」为西班牙语，指交通违章记录。
- 金额单位为哥伦比亚比索（COP）。
- 状态「Pendiente」表示该违章款项尚未缴纳。
- 照片检测类违章以 `fotodeteccion: true` 表示。
