# 🇧🇴 玻利维亚 — 车辆信息

按车牌查询玻利维亚登记车辆，可返回保单号、品牌、类别、登记地、服务类型（如私人）及型号等信息。

### 端点

```
GET https://api.verifik.co/v2/bo/vehicle
```

使用 **`plate`**（车牌）查询车辆登记与基本信息。

### 标头

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 参数

| name | type | required | description |
| --- | --- | --- | --- |
| `plate` | string | yes | 待查询车牌。 |

### 请求

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/bo/vehicle", {
	params: { plate: "ABC1234" },
	headers: {
		Accept: "application/json",
		Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
	},
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os, requests

url = "https://api.verifik.co/v2/bo/vehicle"
headers = {"Accept": "application/json", "Authorization": f"Bearer {os.getenv('VERIFIK_TOKEN')}"}
params = {"plate": "ABC1234"}
r = requests.get(url, headers=headers, params=params)
print(r.json())
```
