# 🇨🇦 加拿大 — 安大略车牌到期

使用**车牌号**和**车辆许可号**（安大略登记文件 / 绿卡）查询安大略车牌贴纸到期日。

典型响应字段：`expired`、`plateNumber`、`permitNumber`、`expiryDate`、`success`。当 `expiryDate` 早于当前时间时，`expired` 为 `true`。

用于快速确认安大略政府登记源中的贴纸有效性，适用于车队入驻、合规检查与车辆核验。

## API 参考

### Endpoint

```
https://api.verifik.co/v2/ca/ontario/plate
```

发送带有查询参数 **`plate`** 和 **`permitNumber`** 的 GET 请求。

### 请求头

| 名称 | 值 |
| ---- | -- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 参数

| 名称 | 类型 | 必填 | 说明 | 示例 |
| ---- | ---- | ---- | ---- | ---- |
| `plate` | string | **是** | 安大略乘用车牌 **ABCD123** 格式（4 位字母 + 3 位数字，无空格）。 | `BKTR456` |
| `permitNumber` | string | **是** | 安大略登记文件上的许可号（8 位）。 | `N9166001` |

### 请求

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/ca/ontario/plate", {
  params: { plate: "BKTR456", permitNumber: "N9166001" },
  headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import requests

url = "https://api.verifik.co/v2/ca/ontario/plate"
params = {"plate": "BKTR456", "permitNumber": "N9166001"}
headers = {"Accept": "application/json", "Authorization": "Bearer <token>"}
response = requests.get(url, params=params, headers=headers)
print(response.text)
```
