# 🇵🇾 巴拉圭 — 车辆信息

通过**车牌号码**查询巴拉圭登记车辆的详细信息。响应可包含品牌、车架、类型、年份、车轴数、车主、服务类型、车辆当前状态及车牌等。

可用于车辆核验、所有权核查与合规监控。

## Endpoint

```
GET https://api.verifik.co/v2/py/vehicle
```

### Headers

| Name          | Value              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parameters

| Name | Type | Required | Description | Example |
| ---- | ---- | -------- | ----------- | ------- |
| `plate` | string | **Yes** | 待查询车牌。 | `ABC123` |

## 功能

- **登记查询**：巴拉圭车辆登记详细信息
- **车牌校验**：国内注册车牌
- **规格参数**：品牌、车架、类型、年份、车轴
- **车主信息**：登记车主
- **服务与状态**：服务类型（如 CARGA NACIONAL）与车辆状况
- **多语言示例**：JavaScript、Python、PHP、Swift
- **实时数据**：基于官方登记的最新信息
