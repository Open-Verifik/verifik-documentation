# 🇵🇾 巴拉圭 — 车辆信息

本服务可使用车牌返回巴拉圭车辆的详细信息。响应中包含车辆的品牌、底盘号、类型、年份、轴数、车主姓名、服务类型、当前状态及车牌号。

该信息可用于车辆核验、车主核查及合规监控等多种场景。

## Endpoint

```
GET https://api.verifik.co/v2/py/vehicle
```

### 请求头（Headers）

| 名称           | 取值                |
| -------------- | ------------------- |
| Content-Type   | `application/json`  |
| Authorization  | `Bearer <token>`    |

### 参数（Parameters）

| 名称 | 类型 | 必填 | 描述 | 示例 |
| ---- | ---- | ---- | ---- | ---- |
| `plate` | string | **是** | 待查询的车牌。 | `ABC123` |

## 功能特性

- **车辆信息检索**：从巴拉圭车辆登记系统获取完整车辆详情。
- **车牌核验**：核验在巴拉圭登记的车辆牌号。
- **车辆参数**：获取详细车辆参数，包括：
  - 品牌与制造商信息
  - 用于车辆识别的底盘号
  - 车辆类型与分类
  - 制造年份与车型详情
  - 车辆配置中的轴数
- **车主信息**：获取当前车主详情。
- **服务类型分类**：获取车辆的服务类型（例如 CARGA NACIONAL）。
- **车辆状态监控**：获取车辆当前状况与状态。
- **合规核验**：核验车辆合规与登记状态。
- **多语言支持**：兼容 JavaScript、Python、PHP 与 Swift。
- **实时数据**：获取最新的车辆信息。
- **完善错误处理**：针对多种场景返回详细错误响应。
- **巴拉圭登记接入**：直接对接巴拉圭官方车辆数据库。
- **多种车型**：支持卡车、油罐车与其他商用车等多种车辆类型。
- **车主核验**：用于法律与商业目的的车主核验。
