# 秘鲁 — 车辆信息

秘鲁车辆信息服务返回某车牌（**`plate`**）的登记车辆详情：品牌、型号、年份、发动机与底盘标识、状态、在可用时返回的车主，以及可选的技术规格字段。

### Endpoint

```
GET https://api.verifik.co/v3/pe/vehiculo/placa
```

按 **`plate`**（无空格或句点）查询秘鲁车辆。响应在可用时可包含 `brand`、`model`、`year`、`engineSerial`、`chasisSerial`、`vin`、`serial`、`color`、`status`、`owners` 和 `technicalSheet`（`engine`、`performance`、`dimensions`）。适用于**核验**、**保险**、**车队**与**合规**流程。

### Headers

| Name          | Value              |
| ------------- | ------------------ |
| Accept | `application/json` |
| Authorization | `Bearer <token>`   |

### Parameters

| Name    | Type   | Required | Description | Example |
| ------- | ------ | -------- | ----------- | ------- |
| `plate` | string | **Yes**  | 要查询的车牌号，不含空格或句点。 | `ABC123` |

### Request

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from 'axios';

const { data } = await axios.get('https://api.verifik.co/v3/pe/vehiculo/placa', {
  params: { plate: 'ABC123' },
  headers: {
    Accept: 'application/json',
    Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
  },
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os
import requests

url = "https://api.verifik.co/v3/pe/vehiculo/placa"
headers = {"Accept": "application/json", "Authorization": f"Bearer {os.getenv('VERIFIK_TOKEN')}"}
params = {"plate": "ABC123"}
response = requests.get(url, headers=headers, params=params)
print(response.json())
```

### Notes

- 当无匹配技术数据时，`technicalSheet` 可能为 `null` 或部分填充。
- 沙箱车牌如 `ABC10001`–`ABC10010` 返回稳定的演示资料。
