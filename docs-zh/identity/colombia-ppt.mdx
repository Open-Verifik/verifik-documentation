### 端点

```
GET https://api.verifik.co/v2/co/foreigner-id/ppt
```

使用 **`documentNumber`** 与签发日期 **`expeditionDate`**（`DD/MM/YYYY`）。也可用 **`POST`** 提交相同 JSON 字段。

### 标头

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 参数

| name | type | required | description |
| --- | --- | --- | --- |
| `documentNumber` | string | yes | PPT 号码，不含空格。 |
| `expeditionDate` | string | yes | 签发日期 **`DD/MM/YYYY`**。 |

### 请求

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/foreigner-id/ppt", {
	params: { documentNumber: "123456789", expeditionDate: "07/03/2022" },
	headers: {
		Accept: "application/json",
		Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
	},
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os, requests

url = "https://api.verifik.co/v2/co/foreigner-id/ppt"
headers = {"Accept": "application/json", "Authorization": f"Bearer {os.getenv('VERIFIK_TOKEN')}"}
params = {"documentNumber": "123456789", "expeditionDate": "07/03/2022"}
r = requests.get(url, headers=headers, params=params)
print(r.json())
```

### 说明

- 日期格式为 **`DD/MM/YYYY`**。
- **`GET`** 与 **`POST`** 行为一致。
- 上游不可用可能出现 **404** / **500**。
