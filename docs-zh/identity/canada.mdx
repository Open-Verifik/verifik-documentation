# 加拿大驾驶证核验

Verifik 的驾驶证核验 API 可基于政府官方数据对加拿大驾驶证进行身份核实。该接口旨在简化您的 KYC（了解您的客户）流程、防范欺诈，并轻松满足合规要求。

我们专为需要快速、安全且自动化确认不列颠哥伦比亚省与安大略省签发的驾驶证有效性的企业打造此集成。

## 此 API 验证哪些内容？

我们的 API 直接对接官方记录，验证以下内容：

-   **驾驶证号码**：核验驾驶证号的真实性。
-   **证件状态**：确认驾驶证当前是否有效。
-   **身份匹配**：对于不列颠哥伦比亚省，校验姓氏与登记记录是否一致。

通过验证这些信息，您可以确信驾驶证真实有效，从而显著降低欺诈风险。

加拿大身份核验服务包含不列颠哥伦比亚省与安大略省的驾驶证核验。这些服务可帮助您核实在上述省份签发的驾驶证的真实性与有效性。

## 支持的省份

| 省份 | 端点 | 描述 |
|------|------|------|
| **不列颠哥伦比亚（British Columbia）** | `/v2/ca/british-columbia/driver-license` | 通过证件号与姓氏核验 BC 省驾驶证 |
| **安大略（Ontario）** | `/v2/ca/ontario/driver-license` | 通过证件号核验安大略省驾驶证 |

## 不列颠哥伦比亚（British Columbia）驾驶证

### Endpoint

```
GET https://api.verifik.co/v2/ca/british-columbia/driver-license
```

不列颠哥伦比亚驾驶证核验服务提供了一种简便方式，用于核验加拿大不列颠哥伦比亚省驾驶证的真实性与有效性。借助该服务，您可以根据所提交的证件号与姓氏快速完成核验。

### 请求头（Headers）

| 名称           | 取值                |
| -------------- | ------------------- |
| Content-Type   | `application/json`  |
| Authorization  | `Bearer <token>`    |

### 参数（Parameters）

| 名称              | 类型    | 必填   | 描述                                                            | 示例         |
| ----------------- | ------- | ------ | --------------------------------------------------------------- | ------------ |
| `documentNumber`  | string  | **是** | 待查询的驾驶证号；所有数据须严格按照该证件原文录入。              | `1123456`    |
| `lastName`        | string  | **是** | 驾驶证上显示的姓氏。                                             | `HELLO`      |
