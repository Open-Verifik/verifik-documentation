# 阿根廷身份核验

Verifik 的身份核验 API 可使用官方政府数据对阿根廷公民进行身份认证。该接口旨在简化您的 KYC（了解您的客户）流程、防范欺诈，并轻松满足合规要求。

我们专为需要快速、安全且自动化确认用户、员工或客户真实身份的企业打造此集成。

## 此 API 验证哪些内容？

我们的 API 直接对接官方记录，验证以下内容：

-   **姓名全称与 DNI 号码**：支持 *Documento Nacional de Identidad*（DNI）。
-   **身份匹配**：确认所提供姓名与 DNI 号码相符。

通过验证这些信息，您可以确信对接的人员真实存在并持有有效证件，从而显著降低身份冒用与欺诈风险。

## API 参考

### 端点

```
GET https://api.verifik.co/v2/ar/cedula
```

使用 **`documentType`** `DNIAR` 与 **`documentNumber`** 核验阿根廷身份。返回身份字段以支持 **KYC** 与 **合规**。

### 请求头

| 名称          | 取值                 |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### 参数

| 名称             | 类型   | 必填 | 描述                                       |
| ---------------- | ------ | ---- | ------------------------------------------ |
| `documentType`   | string | 是 | 证件类型。允许值：`DNIAR`。          |
| `documentNumber` | string | 是 | 待查询人员的证件号码（不含空格）。     |
