# 哥伦比亚 — 通过 Registraduría（SCCRC）查询公民

使用哥伦比亚 **cédula（CC）** 号码，从 **Registraduría Nacional del Estado Civil**（**SCCRC**）查询**出生民事登记**（*registro civil de nacimiento*）。

与[按证件查询公民](/identity-validation/colombia/colombian-citizen)（`/v2/co/cedula`）不同，此接口**仅使用 Registraduría**，不会尝试 Procuraduría、DIAN 或 Olimpia。

## 返回内容

- **`documentNumber`**、**`documentType`**
- **`firstName`**、**`lastName`**、**`fullName`**、**`arrayName`**
- **`sexo`**、**`serial`**
- **`oficina`** / **`expeditionPlace`**、**`fecha`** / **`dateOfBirth`**（如有）
- **`recordType`** — `REGISTRO CIVIL DE NACIMIENTO`
- **已签名**的 Verifik 响应

## API 参考

### Endpoint

```
GET https://api.verifik.co/v2/co/cedula/registraduria
```

也支持 **`POST`**。**`GET`** 使用下方 **query parameters**。

### Headers

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parameters

| name | type | required | description | Examples |
| --- | --- | --- | --- | --- |
| `documentType` | string | yes | 必须为 **`CC`** | `CC` |
| `documentNumber` | string | yes | 哥伦比亚身份证号 | `1032386359` |

## 功能

- SCCRC **证件模式**查询（仅 CC）
- 不经过 Procuraduría、DIAN、Olimpia
- 单条记录响应
- **GET** 与 **POST** 共用同一处理逻辑

### 说明

- **`documentType`** 必须为 **`CC`**。
- SCCRC 证件搜索可能返回 **404**，即使[按姓名查询](/identity-validation/colombia/colombian-citizen-by-name)能找到同一人。
- 临时不可用可能表现为 **409**（`Endpoint_out_of_service`）。
- 相关：[按证件查询](/identity-validation/colombia/colombian-citizen)、[按姓名查询](/identity-validation/colombia/colombian-citizen-by-name)、[按序列号查询](/identity/colombia-registraduria-serial)。
