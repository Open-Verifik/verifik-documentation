# 巴西身份核验

Verifik 身份核验 API 使用 **Cadastro de Pessoas Físicas（CPF）** 认证巴西国民，旨在简化 KYC、防范欺诈并满足合规要求。

## 本 API 核验什么？

连接官方登记数据以核验：

- **全名与 CPF 号码**：支持 *Cadastro de Pessoas Físicas*（CPF）。
- **身份一致性**：提供的姓名是否与 CPF 号码一致。
- **出生日期**：出生日期是否与官方记录一致。

## API 参考

### Endpoint

```
https://api.verifik.co/v2/br/cedula
```

将巴西自然人的 **CPF** 与官方来源比对。请提供 **CPF**、**出生日期**（`DD/MM/YYYY`）以及带或不带分隔符的证件号码；数据一致时，响应返回规范化姓名组成部分与标识符。适用于 **KYC**、**反欺诈**与**受监管入驻**。

### 请求头（Headers）

| 名称           | 取值                |
| -------------- | ------------------- |
| Accept         | `application/json`  |
| Authorization  | `Bearer <token>`    |

### 参数（Parameters）

| 名称 | 类型 | 必填 | 描述 |
| ---- | ---- | ---- | ---- |
| `documentType` | string | **是** | 证件类型。允许值：`CPF`。 |
| `documentNumber` | string | **是** | CPF 号码（可含或不含分隔符）。 |
| `dateOfBirth` | string | **是** | 出生日期，格式：`DD/MM/YYYY`。 |

### 请求（Request）

  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/br/cedula", {

  headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});

```
