# 智利国民身份验证（RUN）

Verifik 身份 API 使用 **Rol Único Nacional（RUN）** 认证智利公民身份。帮助简化 **KYC**、降低欺诈风险并满足合规要求。

适用于需要快速、安全、自动确认用户、员工或客户真实身份的企业。

## 本 API 验证什么

对接官方记录以验证：

- **全名与 RUN**：支持 *Rol Único Nacional*。
- **一致性**：确认所填姓名与 RUN 号码匹配。

从而降低冒用身份风险，提高注册可信度。

## API 参考

### Endpoint

```
https://api.verifik.co/v2/cl/cedula
```

核验智利民事登记意义上的国民身份：以 GET 方式发送 **`documentType`**=`RUN` 与无空格的 **`documentNumber`**；响应包含用于 **KYC**、入职与风控的结构化姓名字段。

### Headers

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parameters

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `documentType` | string | Yes | 允许值：`RUN`。 |
| `documentNumber` | string | Yes | 无空格或分隔符的 RUN。 |

## 常见用例

- **金融科技与银行**：开户、信贷时核验身份。
- **电商与配送**：激活前认证用户与配送员。
- **招聘**：核验候选人证件。
- **保险与医疗**：承保或给付前确认身份。
