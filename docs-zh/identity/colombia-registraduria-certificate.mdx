### 端点

```
GET https://api.verifik.co/v2/co/registraduria/certificado
```

**`documentNumber`**、签发日期 **`date`**（`DD/MM/YYYY`）。也可用相同字段 **`POST`** JSON。适用于 **CC（公民身份证）** 有效性证明。

### 请求头

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 参数

| name | type | required | description |
| --- | --- | --- | --- |
| `documentNumber` | string | 是 | 无空格、无标点的 CC 号码。 |
| `date` | string | 是 | 证件 **签发日期** **`DD/MM/YYYY`**。 |

### 请求

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/registraduria/certificado", {
	params: { documentNumber: "123456789", date: "10/10/2020" },
	headers: {
		Accept: "application/json",
		Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
	},
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os, requests

url = "https://api.verifik.co/v2/co/registraduria/certificado"
headers = {"Accept": "application/json", "Authorization": f"Bearer {os.getenv('VERIFIK_TOKEN')}"}
params = {"documentNumber": "123456789", "date": "10/10/2020"}
r = requests.get(url, headers=headers, params=params)
print(r.json())
```

### 备注

- **`date`** 为 **签发日期**，不是出生日期。
- **`GET`** 与 **`POST`** 共用同一处理逻辑。
- **pdfBase64** 可能含敏感个人信息。
