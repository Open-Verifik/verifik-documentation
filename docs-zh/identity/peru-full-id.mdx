# 🇵🇪 秘鲁 — DNI（扩展信息）

Verifik 的身份核验 API 可使用秘鲁公民的 DNI（*Documento Nacional de Identidad*）号码进行扩展身份验证。该 API 旨在简化您的 KYC（了解您的客户）流程、防范欺诈，并轻松满足合规要求。

我们专为需要快速、安全且自动化获取完整个人信息（包括姓名全称、婚姻状态、出生日期、证件信息、地址和照片）的企业打造此集成。

**为安全访问该信息，必须传入 Authorization 头。**

## 此 API 验证哪些内容？

我们的 API 直接对接秘鲁官方记录，验证以下内容：

-   **姓名全称与 DNI 号码**：支持 *Documento Nacional de Identidad*（DNI）证件。
-   **身份匹配**：确认所提供姓名与 DNI 号码相符。
-   **扩展个人信息**：返回婚姻状态、出生日期、性别和地址信息。
-   **证件信息**：验证证件类型、号码、签发日期、到期日期及校验位。
-   **地理信息**：提供 ubigeoReniec 编码用于地址识别。
-   **照片数据**：以 base64 格式获取公民照片。

通过验证这些信息，您可以确信对接的人员真实存在并持有有效证件，从而显著降低身份冒用与欺诈风险。

### 端点

```
GET https://api.verifik.co/v3/pe/cedula/extra
```

### 请求头（Headers）

| 名称           | 取值                |
| -------------- | ------------------- |
| Content-Type   | `application/json`  |
| Authorization  | `Bearer <token>`    |

### 参数（Parameters）

| 名称 | 类型 | 必填 | 描述 | 示例 |
| ---- | ---- | ---- | ---- | ---- |
| `documentType` | string | **是** | 证件类型。允许值：DNI。 | `DNI` |
| `documentNumber` | string | **是** | 待查询人员的证件号码。 | `123456789` |

-   **欺诈检测**：使用包含照片与人口学信息在内的扩展数据进行高级反欺诈。
