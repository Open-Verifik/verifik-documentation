# 委内瑞拉国籍身份核验（CCVE）

Verifik API 使用 **CCVE**（*Cédula de Identidad*）认证 **委内瑞拉国民**身份，简化 KYC、降低欺诈风险并助力合规。

## 核验内容

-   **姓名与证件号码**：国籍身份证 **CCVE**。
-   **一致性**：返回姓名与登记信息一致。
-   **可选字段**：视数据可用性可能包含出生日期、性别等。

## API 参考

### 端点

```
https://api.verifik.co/v2/ve/cedula
```

本接口面向 **委内瑞拉国民**的 **CCVE** 证件。将身份证号传入 **`documentNumber`**；服务按 **CCVE** 处理并返回结构化姓名字段。**外国居民**请使用 [`GET /v2/ve/foreigner-id`](/verifik-zh/identity/venezuela-foreigner-id)（**CEVE**）。

### 请求头

| 名称 | 值 |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 参数

| 名称 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `documentNumber` | string | 是 | 国籍身份证（CCVE）号码。查询参数无需 `documentType`—本端点固定为 **CCVE**。 |
