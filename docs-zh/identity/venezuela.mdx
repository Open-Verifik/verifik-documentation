# 委内瑞拉身份核验

Verifik 的身份核验 API 可基于政府官方数据对委内瑞拉公民进行身份核实。该接口旨在简化您的 KYC（了解您的客户）流程、防范欺诈，并轻松满足合规要求。

我们专为需要快速、安全且自动化确认用户、员工或客户真实身份的企业打造此集成。

## 此 API 验证哪些内容？

我们的 API 直接对接官方记录，验证以下内容：

-   **姓名全称与证件号码**：支持委内瑞拉身份证件（CCVE）。
-   **身份匹配**：确认所提供姓名与证件号码相符。
-   **可选个人详情**：在数据可用时可包含出生日期、性别与婚姻状态。

通过验证这些信息，您可以确信对接的人员真实存在并持有有效证件，从而显著降低身份冒用与欺诈风险。

## API 参考

### Endpoint

```
https://api.verifik.co/v2/ve/cedula
```

该接口面向**委内瑞拉本国公民**，使用 **CCVE**（*Cédula de Identidad*）证件类别返回身份数据。请将 cédula 号码作为 **`documentNumber`** 传入；API 会查询记录并返回结构化的姓名字段，适用于 **KYC**、客户开通与反欺诈核查。如需核验**外国居民**，请改用 [`GET /v2/ve/foreigner-id`](/identity/venezuela-foreigner-id)（CEVE）。

### 请求头（Headers）

| 名称           | 取值                |
| -------------- | ------------------- |
| Accept         | `application/json`  |
| Authorization  | `Bearer <token>`    |

### 参数（Parameters）

| 名称              | 类型    | 必填 | 描述                                                                                                |
| ----------------- | ------- | ---- | --------------------------------------------------------------------------------------------------- |
| `documentNumber`  | string  | 是   | 国民 cédula 号码（CCVE）。无需传 `documentType`，本接口会自动按 **CCVE** 处理。                       |

### 备注

- `documentNumber` 为必填项。
- 视数据可用性，响应中可能包含 `birthDate`、`gender` 与 `maritalStatus` 字段；并非所有响应都会返回这些字段。
- 当返回 `birthDate` 时，格式为 `YYYY-MM-DD`（例如 "1998-03-01"）。
