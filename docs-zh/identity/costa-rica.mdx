# 哥斯达黎加国民身份验证（CCCR）

## API 参考

### 端点

```
https://api.verifik.co/v2/cr/cedula
```

将 **`documentType`** 设为 `CCCR`，**`documentNumber`** 为无空格的证件号码。

### 请求头

| 名称 | 值 |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 参数

| 名称 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `documentType` | string | 是 | 允许值：`CCCR`。 |
| `documentNumber` | string | 是 | 证件号码。 |

### 请求

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/cr/cedula", {
  params: { documentType: "CCCR", documentNumber: "123456789" },
  headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});
console.log(data);
```
