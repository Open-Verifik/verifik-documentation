# 哥斯达黎加身份核验

Verifik 的身份核验 API 可基于政府官方数据对哥斯达黎加公民进行身份核实。该接口旨在简化您的 KYC（了解您的客户）流程、防范欺诈，并轻松满足合规要求。

我们专为需要快速、安全且自动化确认用户、员工或客户真实身份的企业打造此集成。

## 此 API 验证哪些内容？

我们的 API 直接对接官方记录，验证以下内容：

-   **姓名全称与 CCCR 号码**：支持 *Cédula de Identidad Costarricense*（CCCR）。
-   **身份匹配**：确认所提供姓名与 CCCR 号码相符。

通过验证这些信息，您可以确信对接的人员真实存在并持有有效证件，从而显著降低身份冒用与欺诈风险。

## API 参考

### Endpoint

```
https://api.verifik.co/v2/cr/cedula
```

该接口基于官方数据源核验 **CCCR**（*Cédula de Identidad Costarricense*）。请通过 GET 请求传入 **`documentType`** 为 `CCCR` 与 **`documentNumber`**（不含空格的 cédula 号码）；响应包含结构化的姓名字段，适用于 **KYC**、客户开通与反欺诈核查。

### 请求头（Headers）

| 名称           | 取值                |
| -------------- | ------------------- |
| Accept         | `application/json`  |
| Authorization  | `Bearer <token>`    |

### 参数（Parameters）

| 名称              | 类型    | 必填 | 描述                            |
| ----------------- | ------- | ---- | ------------------------------- |
| `documentType`    | string  | 是   | 证件类型。允许取值：`CCCR`。     |
| `documentNumber`  | string  | 是   | 不含空格的 cédula 号码。         |
