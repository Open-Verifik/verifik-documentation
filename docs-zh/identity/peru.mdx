# 秘鲁身份核验

Verifik 的身份核验 API 可使用国民身份证（DNI）对秘鲁公民进行身份核实。该接口旨在简化您的 KYC（了解您的客户）流程、防范欺诈，并轻松满足合规要求。

我们专为需要快速、安全且自动化确认用户、员工或客户真实身份的企业打造此集成。

## 此 API 验证哪些内容？

我们的 API 直接对接官方记录，验证以下内容：

-   **姓名全称与 DNI**：支持 *Documento Nacional de Identidad*（DNI）。
-   **个人详情**：返回出生日期、性别、婚姻状态与地址。
-   **身份匹配**：确认所提供姓名与 DNI 号码相符。

通过验证这些信息，您可以确信对接的人员真实存在并持有有效证件，从而显著降低身份冒用与欺诈风险。

## API 参考

### Endpoint

```
https://api.verifik.co/v3/pe/cedula
```

秘鲁 **DNI（v3）** 标准核验：对接 RENIEC，校验 **证件号码** 并返回核心身份信息。使用 `documentType=DNI` 与 8 位 `documentNumber`。

### 请求头（Headers）

| 名称           | 取值                |
| -------------- | ------------------- |
| Content-Type   | `application/json`  |
| Authorization  | `Bearer <token>`    |

### 参数（Parameters）

| 名称 | 类型 | 必填 | 描述 |
| ---- | ---- | ---- | ---- |
| `documentType` | string | 是 | 证件类型。允许取值：`DNI`。 |
| `documentNumber` | string | 是 | 待查询人员的证件号码。 |

## 常见使用场景

-   **金融科技与银行**：在开户或贷款申请过程中即时核验身份。
-   **电商与配送**：在用户与配送员入驻您的平台前完成身份核验。
-   **人力资源与招聘**：将候选人证件核验纳入招聘流程。
-   **保险与医疗**：在出单或提供医疗福利前确认身份。
