# 墨西哥身份核验

Verifik API 使用 **CURP**（*Clave Única de Registro de Población*）认证墨西哥公民身份，简化 KYC、降低欺诈风险并助力合规。

## 核验内容

对接官方登记信息，验证：

-   **全名与 CURP**
-   **个人信息**：出生日期、性别、国籍
-   **一致性**：提供的姓名是否与 CURP 匹配

## API 参考

### 端点

```
https://api.verifik.co/v2/mx/curp
```

此接口根据官方记录验证墨西哥国民的 **CURP**（*Clave Única de Registro de Población*）。在 GET 请求中传入 `documentType` 与 `documentNumber`；若 CURP 有效，成功响应将确认身份记录并返回经校验的姓名与人口统计字段。

### 请求头

| 名称 | 值 |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 参数

| 名称 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `documentType` | string | 是 | 允许值：`CURP`。 |
| `documentNumber` | string | 是 | 要验证的 CURP。 |

### 请求

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/mx/curp", {
  params: { documentNumber: "ABCD890513ABCDEF09", documentType: "CURP" },
  headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os, requests

url = "https://api.verifik.co/v2/mx/curp"
headers = {"Accept": "application/json", "Authorization": f"Bearer {os.getenv('VERIFIK_TOKEN')}"}
params = {"documentNumber": "ABCD890513ABCDEF09", "documentType": "CURP"}
r = requests.get(url, headers=headers, params=params)
print(r.json())
```
