# 巴拿马身份核验

通过官方数据源核验 **CCPA**（*Cédula de Identidad Personal*），用于 **KYC**、反欺诈与合规流程。

## 本 API 核验内容

- **证件与身份**：规范化后的证件号码与户籍信息是否一致。
- **出生日期**：`dateOfBirth` 使用 `DD/MM/YYYY` 格式。

## API 参考

### Endpoint

```
https://api.verifik.co/v2/pa/cedula
```

核验巴拿马 **CCPA**。传入 **`documentType`**（`CCPA`）、规范化后的 **`documentNumber`** 以及 **`dateOfBirth`**（**`DD/MM/YYYY`**），返回可用于 **KYC**、开户与风控的一致性结果。

### 请求头

| 名称 | 值 |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 参数

| 名称 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `documentType` | string | 是 | 证件类型，允许 `CCPA`。 |
| `documentNumber` | string | 是 | 完整身份证编号（规范化后）。 |
| `dateOfBirth` | string | 是 | 出生日期 `DD/MM/YYYY`。 |

### 请求

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/pa/cedula", {
  params: { documentType: "CCPA", documentNumber: "8-1251-1829", dateOfBirth: "02/03/1992" },
  headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});
console.log(data);
```
