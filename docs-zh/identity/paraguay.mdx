# 巴拉圭身份核验

Verifik 身份核验 API 使用**巴拉圭政府官方数据**认证公民身份。简化 **KYC**、降低欺诈风险并助力合规。

适用于需要**快速、安全、自动化**确认用户、员工或客户真实身份的企业。

## 本 API 核验内容

直连官方记录，核验：

-   **姓名与 CIC 号码**：支持 *Cédula de Identidad Civil*（CIC）。
-   **一致性**：确认所填姓名与 CIC 号码匹配。

有助于降低冒用风险并提升数字化开户信任度。

## API 参考

### Endpoint

```
https://api.verifik.co/v2/py/cic
```

### Headers

| Name          | Value              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parameters

| Name           | Type    | Required | Description                                    |
| -------------- | ------- | -------- | ---------------------------------------------- |
| `documentNumber` | string | Yes      | CIC 证件号码。 |

### Request

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/py/cic", {
  params: { documentNumber: "1234567" },
  headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os, requests

url = "https://api.verifik.co/v2/py/cic"
headers = {"Accept": "application/json", "Authorization": f"Bearer {os.getenv('VERIFIK_TOKEN')}"}
params = {"documentNumber": "1234567"}
r = requests.get(url, headers=headers, params=params)
print(r.json())
```

## 常见使用场景

-   **金融科技与银行**：开户或放贷时即时核验身份。
-   **电商与配送**：激活前认证用户与配送员。
-   **招聘**：核验候选人证件。
-   **保险与医疗**：承保或给付前确认身份。

## 官方来源与可靠性

直连**巴拉圭政府官方来源**，提供经核验的最新信息。

## 主要优势

-   **自动化合规**：在减少用户摩擦的同时强化 KYC。
-   **即时结果**：适合实时数字化开户。
-   **可信数据**：来自官方登记。
-   **易于集成**：REST API 与 SDK。

## 合规与安全

采用 HTTPS/TLS 1.3 加密与严格隐私标准。**7×24** 可用性监控及基于角色的访问控制。
