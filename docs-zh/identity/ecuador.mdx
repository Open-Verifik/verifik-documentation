# 厄瓜多尔身份核验

Verifik 身份核验 API 可对接厄瓜多尔官方记录，核验公民身份，简化 KYC、防范欺诈并满足合规要求。

## 此 API 验证哪些内容？

- **姓名与 CCEC 号码**：支持 *Cédula de Ciudadanía Ecuatoriana*（CCEC）。
- **身份匹配**：确认姓名与证件号码一致。

## API 参考

### 端点

```
GET https://api.verifik.co/v2/ec/cedula
```

使用 **`documentType`**=`CCEC` 与 **`documentNumber`** 查询厄瓜多尔公民身份。

### 标头

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 参数

| name | type | required | description |
| --- | --- | --- | --- |
| `documentType` | string | yes | 仅 **`CCEC`**。 |
| `documentNumber` | string | yes | 证件号码，不含空格或句点。 |

### 请求

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/ec/cedula", {
	params: { documentType: "CCEC", documentNumber: "0102260098" },
	headers: {
		Accept: "application/json",
		Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
	},
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os, requests

url = "https://api.verifik.co/v2/ec/cedula"
headers = {"Accept": "application/json", "Authorization": f"Bearer {os.getenv('VERIFIK_TOKEN')}"}
params = {"documentType": "CCEC", "documentNumber": "0102260098"}
r = requests.get(url, headers=headers, params=params)
print(r.json())
```
