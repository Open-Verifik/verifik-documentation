### 端点

```
https://api.verifik.co/v2/ip-lookup
```

本服务对 **IPv4/IPv6 地址**进行地理定位，返回国家、地区、城市、ISP/ASN、坐标和时区。适用于**欺诈检测**、**访问控制**和**风险评分**。

### 请求头

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 参数

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `ip` | string | Yes | 要定位的 IPv4 或 IPv6 地址。 |

### 请求

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/ip-lookup", {
  params: { ip: "8.8.8.8" },
  headers: {
    Accept: "application/json",
    Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
  },
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os, requests

url = "https://api.verifik.co/v2/ip-lookup"
headers = {"Accept": "application/json", "Authorization": f"Bearer {os.getenv('VERIFIK_TOKEN')}"}
params = {"ip": "8.8.8.8"}
r = requests.get(url, headers=headers, params=params)
print(r.json())
```

</Tabs>
