# 🇻🇪 委内瑞拉 — 外国人身份证（CEVE）

核验持 **CEVE**（*Cédula de Identidad de Extranjero*）的 **外国人**身份。

## 核验内容

-   **CEVE 身份**：证件号码与登记姓名一致。
-   **证件类型**：响应包含 `documentType: "CEVE"`。

## API 参考

### 端点

```
https://api.verifik.co/v2/ve/foreigner-id
```

面向 **外国人 CEVE**。传入 **`documentNumber`** 即可，服务按 **CEVE** 处理。**国民（CCVE）**请使用 [`GET /v2/ve/cedula`](/verifik-zh/identity/venezuela)。

### 请求头

| 名称 | 值 |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### 参数

| 名称 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `documentNumber` | string | 是 | CEVE 号码。无需 `documentType` 查询参数—本端点为 **CEVE**。 |

### 请求

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/ve/foreigner-id", {
  params: { documentNumber: "123456789" },
  headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});
console.log(data);
```
