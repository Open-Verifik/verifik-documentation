# 🇻🇪 委内瑞拉 — 外国人身份证

Verifik 的身份核验 API 可基于政府官方数据对在委内瑞拉的外国人进行身份核实。该接口旨在简化您的 KYC（了解您的客户）流程、防范欺诈，并轻松满足合规要求。

我们专为需要快速、安全且自动化核验委内瑞拉外籍客户、员工或合作方身份的组织打造此集成。

## 此 API 验证哪些内容？

我们的 API 直接对接官方记录，验证以下内容：

-   **姓名全称与证件号码**：外国人识别证（**CEVE** — *Cédula de Identidad de Extranjero*）。
-   **身份匹配**：返回的姓名与官方登记一致。
-   **证件类型**：本路由响应中包含 `documentType: "CEVE"`。

通过验证这些信息，您可以确信对接的外籍人士真实存在并持有有效证件，从而显著降低身份冒用与欺诈风险。

## API 参考

### Endpoint

```
https://api.verifik.co/v2/ve/foreigner-id
```

请使用此接口核验持 **CEVE** cédula 的**外国国民**。请将证件号作为 **`documentNumber`** 传入；API 默认按 **CEVE** 处理并返回结构化身份字段。如需核验**委内瑞拉本国公民**（**CCVE**），请使用 [`GET /v2/ve/cedula`](/identity/venezuela)。

### 请求头（Headers）

| 名称           | 取值                |
| -------------- | ------------------- |
| Accept         | `application/json`  |
| Authorization  | `Bearer <token>`    |

### 参数（Parameters）

| 名称              | 类型    | 必填 | 描述                                                                                                |
| ----------------- | ------- | ---- | --------------------------------------------------------------------------------------------------- |
| `documentNumber`  | string  | 是   | 外国居民的 CEVE cédula 号码。无需传 `documentType`，本接口会自动按 **CEVE** 处理。                   |
