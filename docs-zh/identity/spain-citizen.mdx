# 西班牙身份核验

通过官方数据源认证西班牙公民与居民，简化 KYC、降低欺诈并支持合规。

适合需要快速、安全核实用户、员工或客户身份的企业。

## 本 API 核验内容

直连官方记录以验证：

-   **姓名与证件号**：支持 DNI、NIE。
-   **身份匹配**：姓名是否与证件号一致。
-   **有效期**：核验证件到期日。

有助于降低冒用与欺诈风险。

## API 参考

### Endpoint

```
https://api.verifik.co/v2/es/cedula
```

通过 GET 传入 `documentType`、`documentNumber` 与到期日 `date`，核验西班牙身份证件（DNI/NIE）。

### 标头

| 名称 | 值 |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### 参数

| 名称 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `documentType` | string | 是 | 证件类型。允许 `DNIES`、`NIE`。 |
| `documentNumber` | string | 是 | 不含空格的证件号码。 |
| `date` | string | 是 | 到期日，格式 `DD/MM/YYYY`。 |

### 请求

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/es/cedula", {
```
