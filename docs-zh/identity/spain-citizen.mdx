# 西班牙身份核验

Verifik 的身份核验 API 可基于政府官方数据对西班牙公民与居民进行身份核实。该接口旨在简化您的 KYC（了解您的客户）流程、防范欺诈，并轻松满足合规要求。

我们专为需要快速、安全且自动化确认用户、员工或客户真实身份的企业打造此集成。

## 此 API 验证哪些内容？

我们的 API 直接对接官方记录，验证以下内容：

-   **姓名全称与证件号码**：支持 *Documento Nacional de Identidad*（DNI）与 *Número de Identidad de Extranjero*（NIE）。
-   **身份匹配**：确认所提供姓名与证件号码相符。
-   **到期日期核验**：核验证件到期日期。

通过验证这些信息，您可以确信对接的人员真实存在并持有有效证件，从而显著降低身份冒用与欺诈风险。

## API 参考

### Endpoint

```
https://api.verifik.co/v2/es/cedula
```

### 请求头（Headers）

| 名称           | 取值                |
| -------------- | ------------------- |
| Content-Type   | `application/json`  |
| Authorization  | `Bearer <token>`    |

### 参数（Parameters）

| 名称              | 类型    | 必填 | 描述                                          |
| ----------------- | ------- | ---- | --------------------------------------------- |
| `documentType`    | string  | 是   | 证件类型。允许取值：`DNIES`、`NIE`。           |
| `documentNumber`  | string  | 是   | 证件号码，不含空格。                            |
| `date`            | string  | 是   | 到期日期，格式为 `DD/MM/YYYY`。                 |

## 常见使用场景

-   **金融科技与银行**：在开户或贷款申请过程中即时核验身份。
-   **电商与配送**：在用户与配送员入驻您的平台前完成身份核验。
-   **人力资源与招聘**：将候选人证件核验纳入招聘流程。
-   **保险与医疗**：在出单或提供医疗福利前确认身份。
