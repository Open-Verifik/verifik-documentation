# 阿根廷身份核验（扩展）

Verifik 身份核验 API 可使用官方政府数据认证阿根廷公民。本端点返回标准 cédula 响应之外的扩展身份字段，包括 **出生日期**、**性别**、结构化姓名片段及 **生存状态**（`isAlive`）。

## 此 API 验证哪些内容？

我们的 API 直接对接官方记录，验证以下内容：

-   **姓名全称与 DNI 号码**：支持 *Documento Nacional de Identidad*（DNI），证件类型 `DNIAR`。
-   **身份匹配**：确认证件号码对应已登记人员。
-   **扩展数据**：返回 **出生日期**、**性别**、**firstName**、**lastName**、**fullName**、**arrayName**（姓名片段）及 **isAlive**。

通过验证这些信息，您可以确信对接的人员真实存在并持有有效证件，从而显著降低身份冒用与欺诈风险。

## API 参考

### 端点

```
https://api.verifik.co/v2/ar/cedula/extra
```

### 请求头

| 名称          | 取值              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### 参数

| 名称             | 类型   | 必填 | 描述                                                                 |
| ---------------- | ------ | -------- | --------------------------------------------------------------------------- |
| `documentType`   | string | 是      | 证件类型。允许值：`DNIAR`。                                      |
| `documentNumber` | string | 是      | 待查询 DNI 号码（不含空格）。                                        |

### 说明

- `documentType` 必须为 `DNIAR`。
- 可用时 `gender` 以单字母代码返回（例如 `F` 或 `M`）。
- `isAlive` 表示查询时登记处是否报告该人员存活。
