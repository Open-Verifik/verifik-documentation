# 投票

## 检索哥伦比亚公民的投票信息

### Endpoint

```
https://api.verifik.co/v2/co/registraduria/votacion
```

投票信息服务可让您从「Registraduría Nacional del Estado Civil」检索哥伦比亚公民的投票详情。提供公民的证件号码后，可获取地址、省、市、投票桌与投票站等信息。

该服务有助于核验选民登记并便利投票流程。

### Headers

| 名称          | 值                 |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Query Parameters

| 名称           | 类型   | 必填? | 说明                                         | 示例         |
| -------------- | ------ | ----- | -------------------------------------------- | ------------ |
| documentNumber | String | 是    | 待查询人员的证件号码，不含空格或点。       | `123456789`  |

### Request

```javascript
import axios from 'axios';

const options = {
  method: 'GET',
  url: 'https://api.verifik.co/v2/co/registraduria/votacion',
  params: {documentNumber: '123456789'},
  headers: {
    Accept: 'application/json',
    Authorization: 'Bearer <your_token>'
  }
};

try {
  const { data } = await axios.request(options);
  console.log(data);
} catch (error) {
  console.error(error);
}
```

### Response

```json
{
  "success": true,
  "data": {
    "documentNumber": "123456789",
    "firstName": "María",
    "lastName": "González",
    "fullName": "María González",
    "address": "Calle 123 #45-67",
    "department": "Cundinamarca",
    "municipality": "Bogotá D.C.",
    "pollingTable": "123",
    "votingStation": "Colegio San José",
    "votingStationAddress": "Carrera 7 #32-16",
    "status": "active"
  }
}
```

### Error Responses

```json
{
  "success": false,
  "error": "Voter not found",
  "code": "VOTER_NOT_FOUND"
}
```
