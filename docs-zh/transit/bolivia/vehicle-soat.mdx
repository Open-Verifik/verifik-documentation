# 玻利维亚 - 车辆 SOAT

Verifik 通过**车牌号**查询玻利维亚官方 **SOAT**（*Seguro Obligatorio de Accidentes de Tránsito*）登记。可用于**车队合规**、**保险核验**和**交通检查**。

在数据源提供时，返回 SOAT 保险详情：

- **`plate`** — 车辆车牌号（**5–8** 位），规范为大写
- **`managementYear`** — SOAT 管理或保险年度
- **`validFrom`** / **`validTo`** — 保险有效期（DD/MM/YYYY）
- **`vehicleType`** — 车辆类别（如 AUTOMOVIL、VAGONETA、MOTOCICLETA）
- **`useType`** — 使用分类（PARTICULAR、COMERCIAL）
- **`department`** — 保单关联的省（department）
- **`status`** — 计算状态：VIGENTE、VENCIDO 或 PENDIENTE
- **`force`** — 可选；设为 `true` 可绕过缓存并从数据源刷新
- Verifik **签名** 响应

### Endpoint

```
GET v2/bo/vehicle-soat
```

### Headers

| Header | Value | Description |
| --- | --- | --- |
| Accept | application/json | 响应格式 |
| Authorization | Bearer \<token\> | 您的 Verifik JWT |

### Parameters

| Field | Type | Required | Description |
| --- | --- | --- | --- |
| plate | String | Yes | 无空格的车牌号（**5–8** 位）。 |
| force | String | No | 设为 `true` 可绕过缓存并获取最新数据。 |

### Notes

- **`plate`** 为必填；中间件会将其规范为大写。
- **`status`** 根据当前日期相对 **`validFrom`** 和 **`validTo`** 计算。
- 需要绕过 MongoDB 缓存记录时使用 **`force=true`**。
- 响应包含**敏感车辆数据**；请遵守适用的隐私法规。
