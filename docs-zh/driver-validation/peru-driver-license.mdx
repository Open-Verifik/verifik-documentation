# 🇵🇪 秘鲁 — 驾驶证

通过持有人 **DNI**（*Documento Nacional de Identidad*）获取秘鲁驾驶证详细信息。响应包含有效驾照、手续历史、规范课程、限制、处罚等，适用于 **车队**、**人力** 与 **KYC** 等需核验驾驶资格的场景。

### Endpoint

```
GET https://api.verifik.co/v2/pe/driver-license
```

使用 `documentType=DNI` 与 8 位 `documentNumber` 查询。返回 **姓名**、**驾照类别**（A、AIV、B）、**licenses**、**procedures**、**regulatoryCourses**、**sanctions**、**tickets** 等（以登记信息为准）。

### 请求头

| Name          | Value              |
| ------------- | ------------------ |
| Accept        | `application/json` |
| Authorization | `Bearer <token>`   |

### 参数

| 名称 | 类型 | 必填 | 说明 | 示例 |
| ---- | ---- | ---- | ---- | ---- |
| `documentType` | string | **是** | 证件类型。允许值：`DNI`。 | `DNI` |
| `documentNumber` | string | **是** | 待查询 DNI，不含空格或点。 | `44141274` |

### 请求

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";
```

### 说明

- `documentNumber` 须与官方登记中存在驾驶证记录的 DNI 一致。

## 功能

- **DNI 查询**：`documentType=DNI` 与证件号码。
- **驾照类别**：A、AIV、B 及临时许可标识。
- **有效驾照**：号码、类别、状态、限制、发证中心与有效期。
- **手续历史**：换证（Revalidación）及状态（如 Vigente、VENCIDA）。
- **规范课程**：报名与完成信息（如有）。
- **处罚与罚单**：按登记情况返回的可选数组。
- **结构化响应**：含 Verifik 签名与认证元数据的 JSON。
