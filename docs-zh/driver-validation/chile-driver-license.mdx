# 🇨🇱 智利 — 驾驶证

本服务可使用驾驶证号作为查询参数，获取智利驾驶证的详细信息。可获取的信息包括 RUT（唯一税务识别号）、地址、驾驶证等级、控制日期等，并以结构化格式返回与驾驶证相关的关键数据。

### Endpoint

```
GET https://api.verifik.co/v2/cl/driver-license
```

通过 **驾驶证号**（查询参数 `documentNumber`）查询智利驾驶证：返回字段通常包含 **RUT**、**驾驶证等级**、**限制条件**、**控制日期**、**所在市镇** 与 **地址**（视登记数据可用性而定），适用于 **车队**、**员工合规** 与对驾驶资质有要求的 **KYC** 场景。

### 请求头（Headers）

| 名称           | 取值                |
| -------------- | ------------------- |
| Accept         | `application/json`  |
| Authorization  | `Bearer <token>`    |

### 参数（Parameters）

| 名称              | 类型    | 必填   | 描述                                  | 示例         |
| ----------------- | ------- | ------ | ------------------------------------- | ------------ |
| `documentNumber`  | string  | **是** | 待查询的驾驶证号，不含空格或圆点。     | `12345678`   |

## 功能特性

- **RUT 校验**：核验智利税务识别号。
- **驾驶证等级信息**：获取详细的驾驶证等级信息（A、B、C 等）。
- **控制日期**：获取驾驶证控制日期与上次控制日期。
- **驾驶人信息**：返回驾驶人姓名、地址与所在市镇。
- **驾驶证限制**：获取与驾驶相关的限制信息。
- **手续信息**：获取驾驶证办理手续详情（补办、续期等）。
- **多语言支持**：兼容 JavaScript、Python、PHP 与 Swift。
- **实时数据**：访问最新的驾驶证信息。
- **完善错误处理**：针对多种场景返回详细错误响应。
- **结构化响应**：JSON 响应格式规整且带有签名校验。
- **智利登记接入**：直接对接智利官方驾驶证数据库。
