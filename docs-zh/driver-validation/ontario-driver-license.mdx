# 🇨🇦 安大略 — 驾驶证

Verifik 的驾驶证核验 API 可基于政府官方数据对加拿大安大略省（Ontario）签发的驾驶证进行核实。该接口旨在简化您的 KYC（了解您的客户）流程、防范欺诈，并轻松满足合规要求。

我们专为需要快速、安全且自动化确认安大略省驾驶证有效性的企业打造此集成。

**为安全访问该信息，必须传入 Authorization 头。**

## 此 API 验证哪些内容？

我们的 API 直接对接安大略省官方驾照管理记录，验证以下内容：

-   **驾驶证号码**：核验在安大略省签发的驾驶证号的真实性。
-   **证件状态**：返回当前驾驶证状态（如 `Valid` 等）。
-   **核验编号**：返回与该次查询绑定的核验编号，便于审计与追溯。

通过验证这些信息，您可以确信驾驶证真实有效，从而显著降低身份冒用与欺诈风险。

## API 参考

### Endpoint

```
GET https://api.verifik.co/v2/ca/ontario/driver-license
```

安大略驾驶证服务可帮助您快速获取加拿大安大略省签发的驾驶证关键信息。提交证件号后，可获取驾驶证状态及关联的核验编号等信息。

借助该服务，您可以高效核验安大略省驾驶证的真实性。无论是用于身份识别或合规目的核实驾驶证有效性，本服务均可提供准确且最新的证件信息。

### 请求头（Headers）

| 名称           | 取值                |
| -------------- | ------------------- |
| Content-Type   | `application/json`  |
| Authorization  | `Bearer <token>`    |

### 参数（Parameters）

| 名称              | 类型    | 必填   | 描述                                                            | 示例                  |
| ----------------- | ------- | ------ | --------------------------------------------------------------- | --------------------- |
| `documentNumber`  | string  | **是** | 待查询的驾驶证号；所有数据须严格按照该证件原文录入。              | `S123456789123456`    |
