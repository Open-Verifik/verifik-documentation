# 🇨🇦 不列颠哥伦比亚 — 驾驶证

Verifik 的驾驶证核验 API 可基于政府官方数据对加拿大不列颠哥伦比亚省（British Columbia, BC）签发的驾驶证进行核实。该接口旨在简化您的 KYC（了解您的客户）流程、防范欺诈，并轻松满足合规要求。

我们专为需要快速、安全且自动化确认 BC 省驾驶证有效性的企业打造此集成。

**为安全访问该信息，必须传入 Authorization 头。**

## 此 API 验证哪些内容？

我们的 API 直接对接不列颠哥伦比亚省官方驾照管理记录，验证以下内容：

-   **驾驶证号码**：核验在不列颠哥伦比亚省签发的驾驶证号的真实性。
-   **身份匹配**：校验所提交的姓氏与驾驶证记录是否一致。
-   **证件状态**：确认驾驶证当前是否有效。

通过验证这些信息，您可以确信驾驶证真实有效且属于其声称的持有人，从而显著降低身份冒用与欺诈风险。

## API 参考

### Endpoint

```
GET https://api.verifik.co/v2/ca/british-columbia/driver-license
```

不列颠哥伦比亚驾驶证核验服务提供了一种简便方式，用于核验加拿大 BC 省驾驶证的真实性与有效性。借助该服务，您可以根据所提交的证件号与姓氏快速完成核验。

### 请求头（Headers）

| 名称           | 取值                |
| -------------- | ------------------- |
| Content-Type   | `application/json`  |
| Authorization  | `Bearer <token>`    |

### 参数（Parameters）

| 名称              | 类型    | 必填   | 描述                                                            | 示例         |
| ----------------- | ------- | ------ | --------------------------------------------------------------- | ------------ |
| `documentNumber`  | string  | **是** | 待查询的驾驶证号；所有数据须严格按照该证件原文录入。              | `1123456`    |
| `lastName`        | string  | **是** | 驾驶证上显示的姓氏。                                             | `HELLO`      |
