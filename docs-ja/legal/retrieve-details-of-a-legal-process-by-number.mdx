### エンドポイント

```
GET https://api.verifik.co/v2/co/rama/proceso
```

**`processNumber`** を **パスの最後のセグメント**として付けます。通常は [司法手続き](/verifik-ja/legal/colombian-legal-processes) 一覧の **`llaveProceso`** です。

### ヘッダー(Headers)

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### パラメータ(Parameters)

| name | type | required | description |
| --- | --- | --- | --- |
| `processNumber` | string | yes | パスパラメータ。7 文字以上。 |

### リクエスト(Request)

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const processNumber = "18738473";

const { data } = await axios.get(
	`https://api.verifik.co/v2/co/rama/proceso/${processNumber}`,
	{
		headers: {
			Accept: "application/json",
			Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
		},
	},
);
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os, requests

process_number = "18738473"
url = f"https://api.verifik.co/v2/co/rama/proceso/{process_number}"
headers = {
    "Accept": "application/json",
    "Authorization": f"Bearer {os.getenv('VERIFIK_TOKEN')}",
}
r = requests.get(url, headers=headers)
print(r.json())
```

### 備考(Notes)

- **`processNumber`** は **パス**に付けます（Explorer ではフィールド名のみの場合があります）。
- **409** は未指定・短すぎる番号などで返りやすいです。
