# 投票

## コロンビア市民の投票情報の取得

### Endpoint

```
https://api.verifik.co/v2/co/registraduria/votacion
```

投票情報サービスでは、「Registraduría Nacional del Estado Civil」からコロンビア市民の投票詳細を取得できます。市民の文書番号を指定すると、住所、県、市区町村、投票テーブル、投票所などの情報にアクセスできます。

このサービスは、有権者登録の確認と投票プロセスの円滑化に役立ちます。

### Headers

| 名前          | 値                 |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Query Parameters

| 名前           | 型     | 必須? | 説明                                           | 例           |
| -------------- | ------ | ----- | ---------------------------------------------- | ------------ |
| documentNumber | String | はい  | 照会する人物の文書番号（スペースや点なし）。 | `123456789`  |

### Request

```javascript
import axios from 'axios';

const options = {
  method: 'GET',
  url: 'https://api.verifik.co/v2/co/registraduria/votacion',
  params: {documentNumber: '123456789'},
  headers: {
    Accept: 'application/json',
    Authorization: 'Bearer <your_token>'
  }
};

try {
  const { data } = await axios.request(options);
  console.log(data);
} catch (error) {
  console.error(error);
}
```

### Response

```json
{
  "success": true,
  "data": {
    "documentNumber": "123456789",
    "firstName": "María",
    "lastName": "González",
    "fullName": "María González",
    "address": "Calle 123 #45-67",
    "department": "Cundinamarca",
    "municipality": "Bogotá D.C.",
    "pollingTable": "123",
    "votingStation": "Colegio San José",
    "votingStationAddress": "Carrera 7 #32-16",
    "status": "active"
  }
}
```

### Error Responses

```json
{
  "success": false,
  "error": "Voter not found",
  "code": "VOTER_NOT_FOUND"
}
```

## ユースケース

- **有権者登録の確認**: 登録ステータスを確認
- **選挙日サポート**: 有権者が投票所を見つけるのを支援
- **人口統計分析**: 有権者の分布を分析
- **選挙モニタリング**: 参加と登録を追跡
- **市民参加**: 民主的プロセスと有権者教育を支援
