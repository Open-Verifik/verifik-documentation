# コントラロリア証明

## APIリファレンス

### Endpoint

```
https://api.verifik.co/v2/co/contraloria/certificado
```

コロンビア総監査院（Contraloría General de Colombia）が発行した品行証明書を、身分証の種類と番号を入力して照合できます。照会日付と、base64でエンコードされたPDF形式の証明書を返します。コロンビアの個人や法人の状況を迅速に確認するのに適しています。

### ヘッダー

| 名前          | 値                 |
| ------------- | ------------------ |
| Accept        | `application/json` |
| Authorization | `Bearer <token>`   |

### パラメータ

:::info
対応する**文書タイプ**と必須パラメータの一覧は [文書タイプ](/background-check/document-types) を参照してください。
:::

| 名前             | 種別   | 必須 | 説明 |
| ---------------- | ------ | ---- | ---- |
| `documentType`   | string | はい | CC、CE、PA または PEP。 |
| `documentNumber` | string | はい | スペース・句点なし。 |
