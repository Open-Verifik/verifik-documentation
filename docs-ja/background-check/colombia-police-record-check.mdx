# 警察の犯罪経歴照会

## APIリファレンス

### Endpoint

```
https://api.verifik.co/v2/co/policia/consultar
```

身分証番号により刑事記録および司法当局の未処理事項の有無を確認します（憲法248条）。氏名と法的状況の情報が含まれます。

### ヘッダー

| 名前          | 値                 |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### パラメータ

:::info
対応する**文書タイプ**と必須パラメータの一覧は [文書タイプ](/background-check/document-types) を参照してください。
:::

| 名前             | 種別   | 必須 | 説明 |
| ---------------- | ------ | ---- | ---- |
| `documentType`   | string | はい | CC または CE。 |
| `documentNumber` | string | はい | スペース・句点なし。 |
