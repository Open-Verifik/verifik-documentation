# Registro Civil 証明書照合（チリ）

## API リファレンス

### エンドポイント

```
GET https://api.verifik.co/v2/cl/registro-civil/certificado
```

**folio** と **検証コード**（`Código Verificador`）を使用して **Registro Civil（SRCEI）** 証明書を照合します。

### ヘッダー (Headers)

| 名前           | 値                 |
| -------------- | ------------------ |
| Accept         | `application/json` |
| Authorization  | `Bearer <token>`   |

### パラメータ

| 名前                | 型     | 必須     | 説明                                           |
| ------------------- | ------ | -------- | ---------------------------------------------- |
| `folio`             | string | **はい** | 証明書 folio（10〜15 文字）。例: `50070080804`.  |
| `verificationCode`  | string | **はい** | 検証コード。例: `2aacbb9a636a`.                 |

### リクエスト (Request)

<Tabs>
<TabItem value="javascript" label="JavaScript">

```javascript
import axios from 'axios';

const { data } = await axios.get('https://api.verifik.co/v2/cl/registro-civil/certificado', {
  params: { folio: '50070080804', verificationCode: '2aacbb9a636a' },
  headers: { Accept: 'application/json', Authorization: 'Bearer <トークン>' },
});
console.log(data);
```

</TabItem>
<TabItem value="python" label="Python">

```python
import http.client

conn = http.client.HTTPSConnection("api.verifik.co")
headers = {"Accept": "application/json", "Authorization": "Bearer <トークン>"}
conn.request("GET", "/v2/cl/registro-civil/certificado?folio=50070080804&verificationCode=2aacbb9a636a", headers=headers)
print(conn.getresponse().read().decode("utf-8"))
```

### 備考

- `folio`: **10〜15** 文字。
- **サンドボックスモード**: `folio=50070080804`、`verificationCode=2aacbb9a636a`; `folio=9004040100001` → **404**。
