# 犯罪歴証明書（アルゼンチン）

## API リファレンス

### エンドポイント

```
GET https://api.verifik.co/v2/ar/certificate/verify
```

公式 PDF に記載された **申請コード**（`Solicitud`）と **セキュリティコード**（`Cod. Segur.`）を使用して、アルゼンチン **DNREC** 犯罪歴証明書を検証します。

### ヘッダー (Headers)

| 名前           | 値                 |
| -------------- | ------------------ |
| Accept         | `application/json` |
| Authorization  | `Bearer <token>`   |

### パラメータ

| 名前            | 型     | 必須     | 説明                                              |
| --------------- | ------ | -------- | ------------------------------------------------- |
| `requestCode`   | string | **はい** | 申請コード（`Solicitud`）。例: `02118499487`.      |
| `securityCode`  | string | **はい** | セキュリティコード（`Cod. Segur.`）。例: `371488F861`. |

### リクエスト (Request)

<Tabs>
<TabItem value="javascript" label="JavaScript">

```javascript
import axios from 'axios';

const options = {
  method: 'GET',
  url: 'https://api.verifik.co/v2/ar/certificate/verify',
  params: { requestCode: '02118499487', securityCode: '371488F861' },
  headers: { Accept: 'application/json', Authorization: 'Bearer <トークン>' },
};

const { data } = await axios.request(options);
console.log(data);
```

</TabItem>
<TabItem value="python" label="Python">

```python
import http.client
```

### 備考

- 両方のパラメータが必須です。欠落時は **409** `MissingParameter` が返されます。
- **サンドボックスモード**: `requestCode=02118499487`、`securityCode=371488F861`; `requestCode=90040401` → **404**。
