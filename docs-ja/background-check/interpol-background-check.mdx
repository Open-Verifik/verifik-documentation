# インターポール（Interpol）身元照会

## API リファレンス

### Endpoint

```
https://api.verifik.co/v2/interpol
```

このエンドポイントは、自然人または法人について `documentType` と `documentNumber`、または `fullName` のみでインターポール関連の刑事経歴情報を検索します。該当記録がある場合は本人確認フィールドと照合詳細が返り、許可された捜査・セキュリティ用途向けです。

**注:** 生年月日または発行日などの日付は `dd/mm/yyyy` 形式にしてください。

### ヘッダー

| 名前 | 値 |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### パラメーター

:::info
サポートされている**文書タイプ**と必須パラメータの一覧は、[文書タイプ](/background-check/document-types)ガイドを参照してください。
:::

| 名前 | 型 | 必須 | 説明 |
| --- | --- | --- | --- |
| `documentType`   | string | No       | Document type that you want to request                                                            |
| `documentNumber` | string | No       | Document number to consult, without spaces or points                                              |
| `fullName`       | string | No       | Instead of documentType and documentNumber, you can pass the name directly of the person/business |

### ユースケース

- **刑事経歴の確認:** 採用、セキュリティ審査、国際記録を含む法令遵守。
- **捜査機関:** インターポールデータに対する承認されたスクリーニング。

    'Authorization': "JWT token"
