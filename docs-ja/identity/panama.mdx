# パナマ本人確認

**CCPA**（*Cédula de Identidad Personal*）を公的ソースで検証します。**KYC**と不正対策に利用できます。

## このAPIで確認できること

- **CCPAと一致確認**：正規化済み番号と戸籍データ。
- **生年月日**：`dateOfBirth` は `DD/MM/YYYY` 形式。

## APIリファレンス

### Endpoint

```
https://api.verifik.co/v2/pa/cedula
```

パナマの**CCPA**を公的戸籍と照合します。**`documentType`**（`CCPA`）、正規化した **`documentNumber`**、**`dateOfBirth`**（`DD/MM/YYYY`）を送信します。**KYC**やオンボーディング向けです。

### ヘッダー

| 名前 | 値 |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### パラメータ

| 名前 | 型 | 必須 | 説明 |
| --- | --- | --- | --- |
| `documentType` | string | はい | 許可値: `CCPA`。 |
| `documentNumber` | string | はい | 正規化後の番号。 |
| `dateOfBirth` | string | はい | `DD/MM/YYYY`。 |

### リクエスト

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/pa/cedula", {
  params: { documentType: "CCPA", documentNumber: "8-1251-1829", dateOfBirth: "02/03/1992" },
  headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});
console.log(data);
```
