### エンドポイント

```
https://api.verifik.co/v2/ip-lookup
```

**IPv4/IPv6 アドレス**をジオロケーションし、国、地域、都市、ISP/ASN、座標、タイムゾーンを返します。**不正検知**、**アクセス制御**、**リスクスコアリング**に利用できます。

### ヘッダー

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### パラメーター

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `ip` | string | Yes | IPv4 または IPv6 アドレス。 |

### リクエスト

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/ip-lookup", {
  params: { ip: "8.8.8.8" },
  headers: {
    Accept: "application/json",
    Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
  },
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os, requests

url = "https://api.verifik.co/v2/ip-lookup"
headers = {"Accept": "application/json", "Authorization": f"Bearer {os.getenv('VERIFIK_TOKEN')}"}
params = {"ip": "8.8.8.8"}
r = requests.get(url, headers=headers, params=params)
print(r.json())
```
