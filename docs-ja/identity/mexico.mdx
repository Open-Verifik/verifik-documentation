# メキシコにおける本人確認

Verifik の API は **CURP**（*Clave Única de Registro de Población*）でメキシコ国民を認証します。KYC を効率化し、不正を抑止し、コンプライアンスを支援します。

## 検証内容

公的記録に接続し、次を確認します。

-   **氏名と CURP**
-   **個人情報**：生年月日、性別、国籍
-   **照合**：提供された氏名が CURP と一致するか

## API リファレンス

### エンドポイント

```
https://api.verifik.co/v2/mx/curp
```

このエンドポイントは、公的記録に照らしてメキシコ国民の**CURP**（Clave Única de Registro de Población）を確認します。GET で `documentType` と `documentNumber` を送ります。CURP が認識されれば、本人確認済みの氏名・属性フィールドが返ります。

### ヘッダー

| 名前 | 値 |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### パラメーター

| 名前 | 型 | 必須 | 説明 |
| --- | --- | --- | --- |
| `documentType` | string | はい | 許可値: `CURP`. |
| `documentNumber` | string | はい | 検証する CURP。 |

### リクエスト

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/mx/curp", {

  headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});
console.log(data);
```

## 主な用途

-   **フィンテック・銀行**：口座開設・融資
-   **EC・配送**：ユーザー・配達員の認証
-   **採用**：候補者の確認
-   **保険・医療**：契約・給付前の本人確認

## ソースと信頼性

メキシコ政府の公的ソース（例：RENAPO）に接続し、セキュアに提供。

## メリット

-   **コンプライアンス自動化**：低摩擦の KYC
-   **即時結果**：リアルタイムオンボーディング
-   **信頼できるデータ**：公的記録
-   **容易な統合**：REST API と SDK

## コンプライアンスとセキュリティ

HTTPS/TLS 1.3、プライバシー、可用性の監視。

## Verifik について

ラテンアメリカ向け本人確認・コンプライアンス・不正防止プラットフォーム — KYC、KYB、AML、バイオメトリクスと公的データの接続。
