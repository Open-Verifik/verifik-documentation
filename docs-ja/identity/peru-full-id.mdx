# 🇵🇪 ペルー — DNI（拡張データ）

Verifik の本人確認 API を使用すると、ペルー市民の DNI（*Documento Nacional de Identidad*）番号で拡張本人確認を実行できます。KYC（Know Your Customer）プロセスの効率化、不正防止、各種規制要件への対応を簡単に実現できるよう設計されています。

氏名、配偶者の有無、生年月日、文書情報、住所、写真などの包括的な個人情報に、迅速・安全・自動的にアクセスする必要があるビジネス向けに構築しました。

**この情報に安全にアクセスするには Authorization ヘッダーが必要です。**

## この API は何を検証しますか？

本 API はペルーの公式記録と直接接続し、以下を検証します。

-   **氏名と DNI 番号**: *Documento Nacional de Identidad*（DNI）文書をサポートします。
-   **本人一致**: 入力された氏名が DNI 番号と一致するかを確認します。
-   **拡張個人情報**: 配偶者の有無、生年月日、性別、住所情報を返します。
-   **文書情報**: 文書種別、番号、発行日、有効期限、検証桁を検証します。
-   **地理情報**: 場所の特定に用いる ubigeoReniec コードを提供します。
-   **写真データ**: 市民の写真を base64 形式で取得できます。

これらを検証することで、相手が実在し有効な文書を保有していることを確認でき、なりすましや不正のリスクを大幅に低減できます。

### エンドポイント

```
GET https://api.verifik.co/v3/pe/cedula/extra
```

ペルー国民の **DNI 拡張照会**です。氏名・配偶者の有無・生年月日・**写真（base64）**・住所・ubigeo・検証桁などを返します。`documentType=DNI` と 8 桁の `documentNumber` を指定します。任意で `force=1` とすると再照会を試みます（リクエスト例参照）。

### ヘッダー（Headers）

| 名前           | 値                  |
| -------------- | ------------------- |
| Content-Type   | `application/json`  |
| Authorization  | `Bearer <token>`    |

### パラメーター（Parameters）

| 名前 | 型 | 必須 | 説明 | 例 |
| ---- | ---- | ---- | ---- | ---- |
| `documentType` | string | **はい** | 文書種別。許可値は `DNI` のみ。 | `DNI` |
| `documentNumber` | string | **はい** | 8 桁の DNI 番号（スペースなし）。 | `12345678` |
| `force` | string | いいえ | 任意。`1` で再照会を試みる場合に使用（例は `force=1`）。 | `1` |

## 一般的なユースケース

-   **高度な KYC プロセス**: 包括的な本人確認によりコンプライアンスと不正検知を強化します。
-   **金融サービス**: 銀行・金融サービスの申請における詳細な本人情報を検証します。
-   **雇用確認**: 採用プロセスにおける包括的な個人情報を確認します。
-   **コンプライアンスとデューデリジェンス**: 拡張本人情報を要求するペルーの規制に準拠します。
-   **不正検知**: 写真や人口統計情報を含む拡張データを活用し、高度な不正防止を行います。

## 公式ソースと信頼性

ペルー政府の公式ソース（RENIEC）と直接接続し、検証済みかつ最新の情報を提供します。
すべての照会は厳格なセキュリティおよび規制基準に基づいて処理されます。

## 主な利点

-   **自動化されたコンプライアンス**: ユーザー体験を損なうことなく KYC チェックを自動化し、不正を防止します。
-   **拡張データへのアクセス**: 写真、住所、配偶者の有無、人口統計情報を含む包括的な本人情報を取得できます。
-   **即時の結果**: 数秒で照会を処理できるため、リアルタイム本人確認に最適です。
-   **信頼できるデータ**: 公式 RENIEC 記録から直接取得したデータを利用できます。
-   **容易な統合**: REST API または互換 SDK を通じて簡単に接続できます。

## コンプライアンスとセキュリティ

データの安全性を最優先します。Verifik は高度な暗号化（HTTPS / TLS 1.3）と厳格なプライバシー管理基準を採用し、機密性を確保します。
当サービスは可用性を 24 時間 365 日監視しており、チームのアクセスを安全に保つロールベースのアクセス制御を提供します。

## Verifik について

Verifik は、ラテンアメリカ全域における本人確認、コンプライアンス、不正防止のためのリーディングプラットフォームです。
当社の API は KYC、KYB、AML、生体検証プロセスを自動化し、コロンビア、メキシコ、ペルー、チリ、ウルグアイ、アルゼンチン、ブラジルなどの公式データソースとビジネスを接続します。
