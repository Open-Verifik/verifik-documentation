# カナダの運転免許検証

Verifik の運転免許検証 API は、公的な政府データを利用してカナダの運転免許を認証します。KYC（Know Your Customer）プロセスの効率化、不正防止、各種規制要件への対応を簡単に実現できるよう設計されています。

ブリティッシュコロンビア州およびオンタリオ州で発行された運転免許の有効性を、迅速・安全・自動的に確認したいビジネス向けに構築しました。

## この API は何を検証しますか？

本 API は公式記録と直接接続し、以下を検証します。

-   **運転免許番号**: 運転免許番号の真正性を検証します。
-   **免許ステータス**: 免許が現在有効であるかを確認します。
-   **本人一致**: ブリティッシュコロンビアでは、姓が免許記録と一致するかを検証します。

これらを検証することで、運転免許が真正かつ有効であることを確認でき、不正リスクの大幅な低減を実現できます。

カナダの本人確認サービスには、ブリティッシュコロンビア州およびオンタリオ州の運転免許検証が含まれます。これらのサービスにより、両州で発行された運転免許の真正性と有効性を検証できます。

## 対応する州

| 州                       | エンドポイント                                  | 説明                                                                       |
|--------------------------|-------------------------------------------------|----------------------------------------------------------------------------|
| **ブリティッシュコロンビア** | `/v2/ca/british-columbia/driver-license`        | BC の運転免許を文書番号と姓で検証します。                                    |
| **オンタリオ**            | `/v2/ca/ontario/driver-license`                 | オンタリオの運転免許を文書番号で検証します。                                |

## ブリティッシュコロンビア — 運転免許

### エンドポイント

```
GET https://api.verifik.co/v2/ca/british-columbia/driver-license
```

ブリティッシュコロンビア州の運転免許検証サービスは、カナダ・ブリティッシュコロンビア州で発行された運転免許の真正性および有効性をシンプルに検証する手段を提供します。文書番号と姓を指定するだけで、運転免許を迅速に検証できます。

### ヘッダー

| 名前          | 値                 |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### パラメーター

| 名前 | 型 | 必須 | 説明 | 例 |
| ---- | ---- | ---- | ---- | ---- |
| `documentNumber` | string | **はい** | 照会対象の運転免許番号。ドキュメント上の表記どおりに入力してください。 | `1123456` |
| `lastName` | string | **はい** | 運転免許に記載されている姓。 | `HELLO` |

## 一般的なユースケース

-   **フィンテックと銀行**: 口座開設や融資申請の際に運転免許を即時検証します。
-   **ライドシェアと配送**: プラットフォーム稼働前にドライバーを認証します。
-   **HR と採用**: 採用ワークフローの一部としてドライバー資格を検証します。
-   **保険と自動車**: 保険契約や請求処理の前に運転免許の有効性を確認します。

## 公式ソースと信頼性

カナダの公式州ライセンスシステム（ブリティッシュコロンビア州およびオンタリオ州）と直接接続し、検証済みかつ最新の情報を提供します。
すべての照会は厳格なセキュリティおよび規制基準に基づいて処理されます。

## 主な利点

-   **自動化されたコンプライアンス**: ユーザー体験を損なうことなく検証チェックを効率化し、不正を防止します。
-   **即時の結果**: 数秒で照会を処理できるため、リアルタイムでのデジタルオンボーディングに最適です。
-   **信頼できるデータ**: 公式の州レコードから直接取得したデータを利用できます。
-   **容易な統合**: REST API または互換 SDK を通じて簡単に接続できます。

## コンプライアンスとセキュリティ

データの安全性を最優先します。Verifik は高度な暗号化（HTTPS / TLS 1.3）と厳格なプライバシー管理基準を採用し、機密性を確保します。
当サービスは可用性を 24 時間 365 日監視しており、チームのアクセスを安全に保つロールベースのアクセス制御を提供します。

## Verifik について

Verifik は、ラテンアメリカおよび北米における本人確認、コンプライアンス、不正防止のためのリーディングプラットフォームです。
当社の API は KYC、KYB、AML、生体検証プロセスを自動化し、コロンビア、メキシコ、ペルー、チリ、ウルグアイ、アルゼンチン、ブラジル、カナダなどの公式データソースとビジネスを接続します。
