# 🇻🇪 ベネズエラ — 外国人身分証（CEVE）

**CEVE**（*Cédula de Identidad de Extranjero*）を持つ **外国人**の本人を確認します。

## 検証内容

-   **CEVE**：文書番号と登録氏名の一致。
-   **文書種別**：レスポンスに `documentType: "CEVE"` が含まれます。

## API リファレンス

### エンドポイント

```
https://api.verifik.co/v2/ve/foreigner-id
```

**外国人 CEVE** 専用です。**`documentNumber`** を渡すとサービスが **CEVE** を適用します。**国民（CCVE）**は [`GET /v2/ve/cedula`](/verifik-ja/identity/venezuela) を使用してください。

### ヘッダー

| 名前 | 値 |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### パラメータ

| 名前 | 型 | 必須 | 説明 |
| --- | --- | --- | --- |
| `documentNumber` | string | はい | CEVE 番号。`documentType` クエリは不要—このエンドポイントは **CEVE** です。 |

### リクエスト

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/ve/foreigner-id", {

  headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});
console.log(data);
```
