# 🇻🇪 ベネズエラ — 外国人 ID

Verifik の本人確認 API は、公的な政府データを利用してベネズエラに居住する外国人を認証します。KYC（Know Your Customer）プロセスの効率化、不正防止、各種規制要件への対応を簡単に実現できるよう設計されています。

ベネズエラにおける外国人顧客、従業員、取引先の本人性を、迅速・安全・自動的に確認したいビジネス向けに構築しました。

## この API は何を検証しますか？

本 API は公式記録と直接接続し、以下を検証します。

-   **氏名と文書番号**: 外国人身分証（**CEVE** — *Cédula de Identidad de Extranjero*）。
-   **本人一致**: 返却される氏名が登録上の文書と一致します。
-   **文書タイプ**: このルートでは、レスポンスに `documentType: "CEVE"` が含まれます。

これらを検証することで、相手の外国人が実在し有効な文書を保有していることを確認でき、なりすましや不正のリスクを大幅に低減できます。

## API リファレンス

### エンドポイント

```
https://api.verifik.co/v2/ve/foreigner-id
```

本エンドポイントは、**CEVE** cédula で登録された **外国人** に使用します。文書番号を **`documentNumber`** に指定すると、API は **CEVE** を適用し、構造化された本人確認フィールドを返します。**ベネズエラ国民**（国民 **CCVE**）には [`GET /v2/ve/cedula`](/identity/venezuela) を使用してください。

### ヘッダー

| 名前          | 値                 |
| ------------- | ------------------ |
| Accept        | `application/json` |
| Authorization | `Bearer <token>`   |

### パラメーター

| 名前             | 型     | 必須 | 説明                                       |
| ---------------- | ------ | ---- | ------------------------------------------ |
| `documentNumber` | string | はい | 外国人居住者の CEVE cédula 番号。`documentType` のクエリは不要で、このエンドポイントでは **CEVE** が自動的に適用されます。 |

## 一般的なユースケース

-   **外国人居住者の検証**: 雇用目的でベネズエラの外国人個人の本人確認を行います。
-   **金融サービス**: 銀行・金融サービス申請における外国人本人確認を行います。
-   **法令遵守**: 外国人居住者に関するベネズエラの規制を遵守します。
-   **文書認証**: 公式目的で外国人文書を認証します。
-   **本人確認**: 外国人の本人確認プロセスを効率化します。

## 公式ソースと信頼性

ベネズエラの公的な政府ソースと直接接続し、検証済みかつ最新の情報を提供します。
すべての照会は厳格なセキュリティおよび規制基準に基づいて処理されます。

## 主な利点

-   **自動化されたコンプライアンス**: ユーザー体験を損なうことなく KYC チェックを効率化し、不正を防止します。
-   **即時の結果**: 数秒で照会を処理できるため、リアルタイム本人確認に最適です。
-   **信頼できるデータ**: 公式記録から直接取得したデータを利用できます。
-   **容易な統合**: REST API または互換 SDK を通じて簡単に接続できます。

## コンプライアンスとセキュリティ

データの安全性を最優先します。Verifik は高度な暗号化（HTTPS / TLS 1.3）と厳格なプライバシー管理基準を採用し、機密性を確保します。
当サービスは可用性を 24 時間 365 日監視しており、チームのアクセスを安全に保つロールベースのアクセス制御を提供します。

## Verifik について

Verifik は、ラテンアメリカ全域における本人確認、コンプライアンス、不正防止のためのリーディングプラットフォームです。
当社の API は KYC、KYB、AML、生体検証プロセスを自動化し、コロンビア、メキシコ、ペルー、チリ、ウルグアイ、アルゼンチン、ブラジル、ベネズエラなどの公式データソースとビジネスを接続します。
