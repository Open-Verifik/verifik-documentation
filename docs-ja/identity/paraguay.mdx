# パラグアイの本人確認

Verifikの本人確認APIは、**パラグアイ政府の公式データ**を用いて国民を認証します。**KYC**を効率化し、不正を抑止し、コンプライアンスを支援します。

ユーザー・従業員・顧客の本人を**迅速かつ安全に自動確認**する必要がある企業向けです。

## このAPIで検証できること

公式記録に直接接続し、次を確認します。

-   **氏名とCIC番号**: *Cédula de Identidad Civil*（CIC）に対応。
-   **一致確認**: 提供された氏名がCIC番号と一致するか。

なりすましリスクを下げ、オンボーディングの信頼性を高めます。

## APIリファレンス

### Endpoint

```
https://api.verifik.co/v2/py/cic
```

### Headers

| Name          | Value              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parameters

| Name           | Type    | Required | Description                                    |
| -------------- | ------- | -------- | ---------------------------------------------- |
| `documentNumber` | string | Yes      | CIC文書番号。 |

### Request

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/py/cic", {
  params: { documentNumber: "1234567" },
  headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});
console.log(data);
```

  </TabItem>

## 主なユースケース

-   **フィンテック・銀行**: 口座開設・融資時の本人確認。
-   **EC・配送**: 有効化前にユーザー・配送員を認証。
-   **採用**: 応募者の身分証明を検証。
-   **保険・医療**: 契約・給付前の本人確認。

## 公式ソースと信頼性

**パラグアイ政府の公式ソース**に直接接続し、検証済みの最新情報を提供します。

## 主なメリット

-   **コンプライアンス自動化**: ユーザー負担を抑えつつKYCを強化。
-   **即時結果**: リアルタイムのデジタルオンボーディングに最適。
-   **信頼できるデータ**: 公式記録に基づく。
-   **簡単な連携**: REST APIとSDK。

## コンプライアンスとセキュリティ

HTTPS/TLS 1.3の暗号化、厳格なプライバシー基準。**24/7**の可用性監視とロールベースのアクセス制御。

## Verifikについて

Verifikはラテンアメリカで本人確認・コンプライアンス・不正防止を提供する主要プラットフォームです。
KYC・KYB・AMLおよび生体認証を自動化し、**パラグアイ**および地域の公式データソースと接続します。
