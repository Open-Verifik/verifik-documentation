# ベネズエラにおける国籍本人確認（CCVE）

Verifik API は **CCVE**（*Cédula de Identidad*）により **ベネズエラ国民**を認証します。KYC の効率化、不正抑止、コンプライアンス支援に役立ちます。

## 検証内容

-   **氏名と文書番号**：国籍身分証 **CCVE**。
-   **照合**：返却される氏名が登録と一致します。
-   **任意項目**：取得可能な場合、生年月日・性別などが含まれることがあります。

## API リファレンス

### エンドポイント

```
https://api.verifik.co/v2/ve/cedula
```

**ベネズエラ国民**の **CCVE** 向けです。**`documentNumber`** に身分証番号を渡すと、サービスが **CCVE** を適用し、構造化された氏名フィールドを返します。**外国人居住者**は [`GET /v2/ve/foreigner-id`](/verifik-ja/identity/venezuela-foreigner-id)（**CEVE**）を使用してください。

### ヘッダー

| 名前 | 値 |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### パラメータ

| 名前 | 型 | 必須 | 説明 |
| --- | --- | --- | --- |
| `documentNumber` | string | はい | 国籍身分証（CCVE）番号。クエリに `documentType` は不要で、このエンドポイントでは **CCVE** が適用されます。 |
