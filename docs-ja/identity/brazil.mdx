# ブラジルの本人確認

Verifik の本人確認 API は、**Cadastro de Pessoas Físicas（CPF）** を用いてブラジル国民を認証します。KYC（Know Your Customer）の効率化、不正防止、規制要件への対応を支援します。

## この API は何を検証しますか？

公式記録と接続し、次を検証します。

- **氏名と CPF 番号**: *Cadastro de Pessoas Físicas*（CPF）に対応します。
- **本人一致**: 入力された氏名が CPF 番号と一致するかを確認します。
- **生年月日**: 生年月日が公式記録と一致するかを検証します。

## API リファレンス

### エンドポイント

```
https://api.verifik.co/v2/br/cedula
```

ブラジル自然人の **CPF** を公式ソースと照合します。**CPF**、**生年月日**（`DD/MM/YYYY`）、および区切りの有無にかかわらない文書番号を渡します。データが一致すると、正規化された氏名の要素と識別子を返します。**KYC**、**不正防止**、**規制対象のオンボーディング**向けです。

### ヘッダー

| 名前          | 値                 |
| ------------- | ------------------ |
| Accept        | `application/json` |
| Authorization | `Bearer <token>`   |

### パラメーター

| 名前 | 型 | 必須 | 説明 |
| ---- | ---- | ---- | ---- |
| `documentType` | string | **はい** | 文書タイプ。許可される値: `CPF`。 |
| `documentNumber` | string | **はい** | CPF 番号（区切りあり・なし）。 |
| `dateOfBirth` | string | **はい** | 生年月日。形式: `DD/MM/YYYY`。 |

### リクエスト

  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/br/cedula", {
  params: { documentType: "CPF", documentNumber: "012.345.678-01", dateOfBirth: "17/02/2002" },
  headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});
console.log(data);
```
