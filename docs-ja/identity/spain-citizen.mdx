# スペインの本人確認

公的データソースでスペイン国民・居住者を認証します。KYCの効率化、不正防止、コンプライアンスに役立ちます。

ユーザー・従業員・顧客の本人確認を迅速に行う企業向けです。

## このAPIで検証できること

公的記録に接続して確認します:

-   **氏名と文書番号**: DNI / NIE に対応。
-   **一致確認**: 提示した氏名が文書番号と一致するか。
-   **有効期限**: 満了日の検証。

なりすましや不正のリスクを下げます。

## APIリファレンス

### Endpoint

```
https://api.verifik.co/v2/es/cedula
```

GET で `documentType`、`documentNumber`、有効期限の `date` を渡し、スペインの身分証（DNI/NIE）を検証します。

### ヘッダー

| 名前 | 値 |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### パラメータ

| 名前 | 種別 | 必須 | 説明 |
| --- | --- | --- | --- |
| `documentType` | string | はい | 文書種別。`DNIES` または `NIE`。 |
| `documentNumber` | string | はい | スペースなしの文書番号。 |
| `date` | string | はい | 有効期限 `DD/MM/YYYY`。 |

## 主なユースケース

-   **フィンテック・銀行**: 口座開設・融資時の本人確認
-   **EC・配送**: ユーザー・配達員の認証
-   **採用**: 応募者書類の検証
-   **保険・医療**: 契約・給付前の本人確認

## 公式ソースと信頼性

スペイン政府系ソースに接続し、検証済み情報を提供します。

## 主なメリット

-   **自動コンプライアンス**: KYC を効率化
-   **即時結果**: リアルタイムオンボーディング
-   **信頼できるデータ**: 公的記録
-   **容易な連携**: REST API・SDK

## コンプライアンスとセキュリティ

HTTPS/TLS と厳格なプライバシー管理を採用しています。

## Verifikについて

ラテンアメリカとヨーロッパで本人確認・コンプライアンスAPIを提供します。
