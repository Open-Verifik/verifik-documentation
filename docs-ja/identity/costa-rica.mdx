# コスタリカにおける国民本人確認（CCCR）

## API リファレンス

### エンドポイント

```
https://api.verifik.co/v2/cr/cedula
```

**`documentType`** に `CCCR`、**`documentNumber`** に番号（スペースなし）を指定します。

### ヘッダー

| 名前 | 値 |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### パラメータ

| 名前 | 型 | 必須 | 説明 |
| --- | --- | --- | --- |
| `documentType` | string | はい | 許可値: `CCCR`。 |
| `documentNumber` | string | はい | 身分証番号。 |

### リクエスト

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/cr/cedula", {
  params: { documentType: "CCCR", documentNumber: "123456789" },
  headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});
console.log(data);
```
