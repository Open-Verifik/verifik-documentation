# チリ身分証の有効性確認

このエンドポイントは、識別子に加えて物理 ID に**印刷されたシリアル番号**を用いて、チリの **RUN** または **RUT** の**有効性**を確認します。番号が存在するだけでなく**カードが有効か**確認する必要がある場合—単純な cédula 照会を超える**高信頼 KYC**や**コンプライアンス**に使います。

シリアル**なし**で民籍の名義のみ確認する場合は **[チリにおける本人確認](/identity/chile)**（`v2/cl/cedula`）を参照してください。

### Endpoint

```
https://api.verifik.co/v2/cl/validate/documents
```

**RUN** または **RUT**、**文書番号**、**印刷シリアル**を送ると、民籍または税務の検証ソースから**有効性に相当するステータス**が返ります。物理文書を確認する必要があるときに使います。

### Headers

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parameters

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `documentType` | string | Yes | `RUN`（民籍）または `RUT`（税務）。 |
| `documentNumber` | string | Yes | 統合ガイドに従いスペースなし。 |
| `serialNumber` | string | Yes | カードに印字されたシリアル（ソース必須）。 |

### Request

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/cl/validate/documents", {
  params: {
    documentType: "RUN",
    documentNumber: "18562530-3",
    serialNumber: "A123456789",
  },
  headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});
console.log(data);
```

### Notes

- `status` の文言は上流レジストリに依存します。**有効性の目安**として扱い、法的助言の代わりにしないでください。
- **`serialNumber`** は物理カードから取得してください。ないと民籍 **Registro Civil** フローは完了しません。
