# チリにおける本人確認（RUN）

Verifik の本人確認 API は、**Rol Único Nacional（RUN）**を用いてチリ国民を認証します。**KYC** を効率化し、不正を抑え、規制要件への対応を支援します。

ユーザー・従業員・顧客の本人を迅速かつ安全に確認したい企業向けに設計しています。

## この API が検証するもの

公的記録と照合して次を確認します。

- **氏名と RUN**：*Rol Único Nacional* に対応。
- **一致**：提示した氏名が RUN 番号と一致するか。

なりすましリスクを下げ、オンボーディングの信頼性を高めます。

## API リファレンス

### Endpoint

```
https://api.verifik.co/v2/cl/cedula
```

チリの民籍に基づく本人確認：GET で **`documentType`** を `RUN`、**`documentNumber`** をスペースなしで送ります。応答には **KYC**、オンボーディング、不正チェック用の構造化された氏名フィールドが含まれます。

### Headers

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parameters

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `documentType` | string | Yes | 許可値: `RUN`。 |
| `documentNumber` | string | Yes | スペース・区切りなしの RUN。 |

## 主な用途

- **フィンテック・銀行**：口座開設や融資時の本人確認。
- **EC・配送**：アクティブ化前のユーザー・ドライバー認証。
- **採用**：書類検証。
- **保険・医療**：契約・給付前の本人確認。

## 公式ソース

チリ政府（民籍など）の公式ソースに接続し、検証済みの情報を提供します。

## メリット

- **コンプライアンス自動化**：ユーザー負担を抑えた KYC。
- **高速応答**：リアルタイムオンボーディング向け。
- **信頼できるデータ**：公的登録簿に基づく。

## セキュリティ

HTTPS/TLS、プライバシーと可用性のベストプラクティス。

## Verifik について

Verifik はラテンアメリカ向けに本人確認・KYB・不正防止 API を提供しています。
