# チリ — 納税者（RUT）照会

本エンドポイントは、チリの **RUT** に対する **税務向け** の本人確認データを取得する場合に使用します（戸籍 cedula のフローではありません）。**RUN / 戸籍** 照会向けの **[チリの本人確認](/identity/chile)**（`v2/cl/cedula`）を補完します。

## API リファレンス

### エンドポイント

```
https://api.verifik.co/v2/cl/taxpayer
```

**税務登録** に整合した氏名フィールドおよび識別子を返します。**KYB**、**仕入先オンボーディング**、**RUT** が **登録納税者** データと一致する必要のある **請求書** 検証に最適です。

### ヘッダー

| 名前 | 値 |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### パラメーター

| 名前 | 型 | 必須 | 説明 |
| --- | --- | --- | --- |
| `documentType` | string | はい | `RUT` を指定する必要があります。                                       |
| `documentNumber` | string | はい | 納税者番号（数字のみ。フォーマットはサービス側で正規化されます）。       |

### リクエスト

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/cl/taxpayer", {
  params: { documentType: "RUT", documentNumber: "263007905" },
  headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os, requests

url = "https://api.verifik.co/v2/cl/taxpayer"

```

### 注意事項

- 戸籍登録の **RUN** による本人一致照会には **`v2/cl/cedula`** を優先して使用してください。**RUT 納税者** 記録を真実の情報源とする場合は **`v2/cl/taxpayer`** を使用します。
