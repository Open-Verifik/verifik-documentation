# エクアドルの本人確認

Verifik の本人確認 API は、公的な政府データを利用してエクアドル市民を認証します。KYC（Know Your Customer）プロセスの効率化、不正防止、規制要件への対応を支援します。

ユーザー、従業員、顧客の本人性を迅速・安全に確認したいビジネス向けに構築しました。

## この API は何を検証しますか？

本 API は公式記録と直接接続し、以下を検証します。

-   **氏名と CCEC 番号**: *Cédula de Ciudadanía Ecuatoriana*（CCEC）に対応します。
-   **本人一致**: 入力された氏名が CCEC 番号と一致するかを確認します。

これらを検証することで、なりすましや不正のリスクを低減できます。

## API リファレンス

### エンドポイント

```
GET https://api.verifik.co/v2/ec/cedula
```

**`documentType`**（`CCEC`）と **`documentNumber`** でエクアドル国民の本人確認を行います。公式記録に基づく主要フィールドを返し、**KYC** や **コンプライアンス**に利用できます。

### ヘッダー

| 名前          | 値                 |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### パラメーター

| 名前             | 型     | 必須 | 説明                                       |
| ---------------- | ------ | ---- | ------------------------------------------ |
| `documentType`   | string | はい | 文書タイプ。許可される値: `CCEC`。            |
| `documentNumber` | string | はい | スペースや句点のない文書番号。               |

## 一般的なユースケース

-   **フィンテック・銀行**: 口座開設や融資申請時の本人確認
-   **EC・配送**: 利用者・配送員の認証
-   **人事・採用**: 候補者文書の検証
-   **保険・ヘルスケア**: 保険発行前の本人確認

## 公式ソースと信頼性

エクアドルの公的ソースと直接接続し、検証済みの情報を提供します。

## 主な利点

-   **コンプライアンス**: KYC を効率化し不正を抑制
-   **即時性**: 数秒で照会可能
-   **信頼できるデータ**: 公式記録ベース

## コンプライアンスとセキュリティ

HTTPS/TLS による通信の保護と厳格なプライバシー管理を重視しています。

## Verifik について

Verifik はラテンアメリカ全域で本人確認・コンプライアンス・不正防止のためのプラットフォームです。
