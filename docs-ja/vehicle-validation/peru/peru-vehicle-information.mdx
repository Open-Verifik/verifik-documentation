# ペルー — 車両情報

ペルー車両情報サービスは、ナンバープレート（**`plate`**）に対する登録車両詳細を返します：メーカー、モデル、年式、エンジン・シャシー識別子、ステータス、利用可能な場合の所有者、任意の技術仕様フィールド。

### Endpoint

```
GET https://api.verifik.co/v3/pe/vehiculo/placa
```

**`plate`**（スペースや句点なし）でペルー車両を照会します。応答には、利用可能な場合に `brand`、`model`、`year`、`engineSerial`、`chasisSerial`、`vin`、`serial`、`color`、`status`、`owners`、`technicalSheet`（`engine`、`performance`、`dimensions`）が含まれることがあります。**検証**、**保険**、**フリート**、**コンプライアンス**に利用できます。

### Headers

| Name          | Value              |
| ------------- | ------------------ |
| Accept | `application/json` |
| Authorization | `Bearer <token>`   |

### Parameters

| Name    | Type   | Required | Description | Example |
| ------- | ------ | -------- | ----------- | ------- |
| `plate` | string | **Yes**  | 照会するナンバープレート（スペースや句点なし）。 | `ABC123` |

### Request

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from 'axios';

const { data } = await axios.get('https://api.verifik.co/v3/pe/vehiculo/placa', {
  params: { plate: 'ABC123' },
  headers: {
    Accept: 'application/json',
    Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
  },
});
console.log(data);
```

### Notes

- 一致する技術データがない場合、`technicalSheet` は `null` または部分的にしか埋まらないことがあります。
- `ABC10001`–`ABC10010` などのサンドボックスプレートは安定したデモプロファイルを返します。
