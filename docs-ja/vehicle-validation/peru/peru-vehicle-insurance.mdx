# ペルー — 車両保険（SOAT）

## エンドポイント

```
https://api.verifik.co/v2/pe/vehiculo/soat
```

本サービスは、ペルーにおける車両の保険ステータスに関する情報を提供します。保険会社名、契約開始日・終了日、車両のナンバープレート、契約番号、用途種別、車両クラス、契約ステータスなどの詳細を返します。

## ヘッダー

| 名前          | 値                 |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

## パラメーター

| 名前      | 型     | 必須 | 説明                                                                       |
| --------- | ------ | ---- | -------------------------------------------------------------------------- |
| `plate`   | string | はい | 照会するナンバープレート。スペースやドットを含めずに入力してください。     |

## ユースケース

- **SOAT 検証**: 強制交通事故保険（SOAT）の現在の有効性を検証します。
- **保険サービス**: 有効な契約および有効期限を検証します。
- **交通管理**: 強制保険の遵守を検証します。
- **運送会社**: フリート車両の保険を検証します。
- **コンプライアンス検証**: 契約の有効性を確認します。

## 機能

- ✅ ナンバープレートによる照会
- ✅ SOAT の完全な契約情報
- ✅ 保険会社名
- ✅ 契約開始日および終了日
- ✅ 契約ステータス（有効 / 無効）
- ✅ 一意の契約コード
- ✅ SBS 保険会社コード
- ✅ 警察管理日
- ✅ Verifik.co による認証済みレスポンス
