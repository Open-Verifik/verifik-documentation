# 🇦🇷 アルゼンチン — 車両 v3

### エンドポイント

```
GET https://api.verifik.co/v3/ar/vehicle
```

このエンドポイントでナンバープレートからアルゼンチン車両の詳細を取得します。レスポンスにはエンジン番号、シャシー、ブランド、モデル、年式、色、車両タイプ、アクティブ保険の状態、技術シートフィールド（利用可能な場合）が含まれます。

### ヘッダー

| Name          | Value              |
| ------------- | ------------------ |
| Accept        | `application/json` |
| Authorization | `Bearer <token>`   |

### パラメーター

| 名前 | 型 | 必須 | 説明 |
| ---- | ---- | -------- | ----------- |
| `plate` | string | **Yes** | メルコスール形式 `AA-123-BB`（`AA123BB` として送信。スペース・句読点なし）。 |

const { data } = await axios.get('https://api.verifik.co/v3/ar/vehicle', {

  headers: {

    Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,

### レスポンスフィールド

| フィールド | 型 | 説明 |
| ----- | ---- | ----------- |
| `plate` | string | 正規化されたナンバープレート。 |
| `engineNumber` | string \| null | ソースから返されたエンジン番号。 |
| `chassis` | string \| null | ソースから返されたシャシー識別子。 |
| `year` | string \| null | 車両年式。 |
| `brand` | string \| null | 車両ブランド。 |
| `modelGroup` | string \| null | モデルファミリー。 |
| `model` | string \| null | モデル/バージョンラベル。 |
| `color` | string \| null | 登録色。 |
| `type` | string \| null | 車両ユニットタイプ。 |
| `isActivePolicyFound` | boolean | アクティブ保険が見つかったかどうか。 |
| `technicalSheet` | object \| null | 利用可能な場合の技術シートデータ。 |

### メモ

- 一致する技術データがない場合、`technicalSheet` は `null` または部分的にのみ入力されることがあります。
- サービスはソース照会前にプレートの大文字小文字を正規化します。
