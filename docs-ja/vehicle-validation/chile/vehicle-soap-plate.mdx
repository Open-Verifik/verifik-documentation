# 🇨🇱 Chile - Vehicle SOAP (by plate)

Query Chile vehicular **SOAP** using **license plate only**. Nested **`soap`** / **`vehicle`** response.

### Endpoint

```
GET v2/cl/vehicle-soap-plate
```

### Headers

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parameters

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `plate` | string | Yes | Chilean license plate (4–6 chars). |
| `force` | string | No | `true` to bypass cache. |

### Request

<Tabs>
  <TabItem value="javascript" label="JavaScript">

```javascript
const { data } = await axios.get("https://api.verifik.co/v2/cl/vehicle-soap-plate", {
  params: { plate: "HTRT86" },
  headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});
```

### Notes

- **No active SOAP** returns **HTTP 200** with empty **`soap`**, not 404.
- Live plates: `CLTK27`, `HTRT86`, `RRWX51`, `THHV64`, `CSXZ52`, `CSXZ57`; `CSXZ27` has no active SOAP.
- See [Vehicle SOAP (plate + policy)](/verifik-ja/vehicle-validation/chile/vehicle-soap).
