# 🇨🇱 チリ — 車両 SOAP（ナンバーのみ）

**ナンバープレートのみ**でチリの **SOAP**（*Seguro Obligatorio de Accidentes Personales*）を照会します。Boostr で SOAP を取得し、証券番号があれば AACH で enrich、SOAP がない場合は Carvuk から車両属性を返します。

ソースが応答する場合の**ネスト**ペイロード：

- **`plate`** — チリナンバー（Patente）、**4–6** 文字の英数字
- **`message`** — SOAP あり時は空；車両のみでアクティブ SOAP なしは `No_active_SOAP_certificate_found_for_this_plate`
- **`soap`** — 見つかった場合：**`status`**, **`policyNumber`**, **`insuranceCompany`**, **`validFrom`**, **`validTo`**, **`folioNumber`**, **`ownerName`**, **`ownerRut`**, **`premium`**
- **`vehicle`** — **`brand`**, **`model`**, **`year`**, **`type`**, **`engineNumber`**
- Verifik **署名**付きレスポンス

ナンバーのみ判明時の**保険コンプライアンス**、**フリートオンボーディング**、**クレーム**に使用します。

### エンドポイント

```
GET v2/cl/vehicle-soap-plate
```

**ナンバー**（`plate`）のみで照会。**`soap`** の **status**, **insuranceCompany**, **ownerName**, **validFrom**, **validTo** および **`vehicle`** の **brand**, **model**, **year** などが含まれます。

### ヘッダー

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### パラメータ

| Name | Type | Required | Description | Example |
| --- | --- | --- | --- | --- |
| `plate` | string | **Yes** | チリナンバー（Patente）、4–6 文字の英数字。 | `HTRT86` |

### 注意

- **アクティブ SOAP なし**は **HTTP 200**、空の **`soap`**、**`message: No_active_SOAP_certificate_found_for_this_plate`** — 404 ではありません。
- **Live プレート:** `CLTK27`, `HTRT86`, `RRWX51`, `THHV64`, `CSXZ52`, `CSXZ57` はアクティブ SOAP；`CSXZ27` は SOAP なしで車両のみ。
- **サンドボックス:** **`BBCC12`**–**`BBCC21`**, **`HTRT86`** は安定ネストプロフィール；**`ERR404`** → **404**。
- ナンバー+証券照会は [車両 SOAP（ナンバー+証券）](/verifik-ja/vehicle-validation/chile/vehicle-soap) を参照。
- レジストリ**可用性**は環境依存；**409** タイムアウトの可能性あり。
