# 🇨🇱 チリ - 車両技術検査

この endpoint は、チリ車両の **技術検査 / 検査履歴** をナンバープレートで取得します。一般的な登録情報は `v2/cl/vehicle`、SOAP 強制保険証明書は `v2/cl/vehicle-soap` を使用します。

### Endpoint

```text
GET https://api.verifik.co/v3/cl/vehicle
```

### パラメータ

| 名前 | 型 | 必須 | 説明 | 例 |
| --- | --- | --- | --- | --- |
| `plate` | string | **はい** | v3 対応のチリ車両ナンバー形式。4文字+2数字、または2文字+4数字。 | `XH6640` |

### リクエスト例

```javascript
const { data } = await axios.get("https://api.verifik.co/v3/cl/vehicle", {
	params: { plate: "XH6640" },
	headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});
```

### 200 レスポンス

```json
{
	"data": {
		"plate": "XH6640",
		"vehicle": {
			"plate": "XH6640",
			"type": "CAMIONETA (CAP. DE CARGA MENOR A 1.750 KGS.) [B] (A2)",
			"mark": "PEUGEOT",
			"model": "PARTNER TOLE 1.9",
			"year": "2004",
			"engineNumber": "10DXDM0003203",
			"chasisNumber": "60548210",
			"vin": null,
			"sealType": "SELLO VERDE"
		},
		"history": [
			{
				"date": "2026-01-12",
				"plantCode": "PRT-014",
				"plantName": "REVISIONES METROPOLITANA",
				"certificateNumber": "RT-XH6640-2026",
				"expirationDate": "2027-01-12",
				"status": "APROBADA"
			}
		]
	}
}
```

### エラー

- `404 NotFound`: レコードが見つかりません。
- `409 Invalid_plate_format`: ナンバープレート形式が無効です。
- `409 timeout_data_source`: 外部データソースが利用できないかタイムアウトしました。

### Notes

`XH6640`、`FHDJ31`、`DCCH18` は固定プロファイルを返します。`BB985` は `409 Invalid_plate_format` を返します。
