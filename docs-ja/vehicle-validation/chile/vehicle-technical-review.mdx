# 🇨🇱 チリ - 車両技術検査

チリ車両の **技術検査 / 点検** データが必要なときにこの endpoint を使用します。ナンバープレートで車両詳細、検査場、証明書番号、有効期限、検査ステータス、および利用可能な場合の技術仕様フィールドを取得します。

関連するチリ車両 endpoint:

- **車両情報 (`v2/cl/vehicle`)**：メーカー、モデル、所有者 RUT、エンジン、シャシー、罰金、盗難フラグなどの一般登録データ。
- **車両技術検査 (`v3/cl/vehicle`)**：技術検査ステータス、履歴、および利用可能な場合の技術仕様データ。
- **車両 SOAP (`v2/cl/vehicle-soap`)**：ナンバーと証券番号による強制人身事故保険証明書。

### Endpoint

```text
GET https://api.verifik.co/v3/cl/vehicle
```

ナンバープレートに対するチリ技術検査データを返します。車両属性、検査履歴、および利用可能な場合の任意の `technicalSheet`（`engine`、`performance`、`dimensions`）を含みます。フリートコンプライアンス、保険、クレーム、車両オンボーディングに利用できます。

### Headers

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parameters

| Name | Type | Required | Description | Example |
| --- | --- | --- | --- | --- |
| `plate` | string | **Yes** | v3 対応のチリ車両ナンバー形式。例: 4文字+2数字、または2文字+4数字。 | `XH6640` |

### Request

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v3/cl/vehicle", {
	params: { plate: "XH6640" },
	headers: {
		Accept: "application/json",
		Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
	},
});
console.log(data);
```

import os, requests

url = "https://api.verifik.co/v3/cl/vehicle"

r = requests.get(url, headers=headers, params=params)

### Notes

- 一致する技術データがない場合、`technicalSheet` は `null` または部分的にしか埋まらないことがあります。
- `XH6640`、`FHDJ31`、`DCCH18` は安定した技術検査プロファイルを返します。
- `BB985` は v3 ナンバー形式に一致しないため `409 Invalid_plate_format` を返します。
