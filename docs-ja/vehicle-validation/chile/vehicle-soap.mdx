# 🇨🇱 チリ — 車両 SOAP

チリ登録車両の **SOAP**（人身事故強制保険）証明書を **ナンバープレート** と **保険証券番号**（AACH レジストリ）で照会します。

## 車両 SOAP 照会

### エンドポイント

```
GET https://api.verifik.co/v2/cl/vehicle-soap
```

### ヘッダー

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### パラメータ

| name | type | required | description |
| --- | --- | --- | --- |
| `plate` | string | yes | チリナンバー（4–6 文字）。 |
| `policyNumber` | string | yes | 保険証券番号（下 4 桁を使用）。 |

### リクエスト

<Tabs>
  <TabItem value="javascript" label="JavaScript">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/cl/vehicle-soap", {
	params: { plate: "HTRT86", policyNumber: "94596506" },
	headers: {
		Accept: "application/json",
		Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
	},
});
console.log(data);
```

  </TabItem>
</Tabs>

### 注意

- サンドボックス: `AB1001`–`AB1010`（AB1004 = VENCIDO）; `HTRT86`/`94596506` デモ; `ERR404` → 404。
