# 🇨🇱 チリ — 車両 SOAP

チリ登録車両の **SOAP**（*Seguro Obligatorio de Accidentes Personales*、人身事故強制保険）照会は、車両の義務的個人事故保険を検証します。**ナンバープレート**と**保険証券番号**で AACH レジストリから保険会社、所有者、有効期間、車両属性を取得します。

## 車両 SOAP 照会

### エンドポイント

```
GET https://api.verifik.co/v2/cl/vehicle-soap
```

**ナンバープレート**（`plate`）と**保険証券番号**（`policyNumber`；サービスは AACH レジストリで下 4 桁を使用）で照会します。一般的なフィールド：**status**、**insuranceCompany**、**ownerName**、**ownerRut**、**validFrom**、**validTo**、**brand**、**model**、**manufactureYear**、**engineNumber**、**premium** — **保険コンプライアンス**、**フリートオンボーディング**、**クレーム**ワークフローに使用します。

### ヘッダー

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### パラメータ

| Name | Type | Required | Description | Example |
| --- | --- | --- | --- | --- |
| `plate` | string | **Yes** | チリナンバー（Patente）、4–6 文字の英数字。 | `HTRT86` |
| `policyNumber` | string | **Yes** | 保険証券番号；AACH SOAP 照会では下 4 桁を使用。 | `94596506` |

### リクエスト

### 注意

- **サンドボックスモード:** プレート **`BBCC12`**–**`BBCC21`** は固定 SOAP プロフィール（`BBCC15` = **VENCIDO**）；**`HTRT86`** + **`policyNumber=94596506`** はデモプロフィール；**`ERR404`** → **404**；未マップのプレートはランダム **200**。
- **サンドボックス例:** `BBCC12`/`94590001`（セダン、MAPFRE）、`BBCC18`/`94590007`（ピックアップ、CHUBB）、`HTRT86`/`94596506`（TOYOTA COROLLA デモ）。
- チリの一般**車両**照会は [🇨🇱 チリ — 車両](/verifik-ja/vehicle-validation/chile) を参照。
- レジストリの**可用性**は環境依存；障害時は **409** タイムアウトの可能性があります。
