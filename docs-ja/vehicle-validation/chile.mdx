# 🇨🇱 チリ — 車両情報

チリの車両検証サービスは、車両情報の包括的な検証機能を提供します。本サービスでは、チリで登録された車両に関する詳細情報を取得できます。

関連するチリ車両 endpoint:

- **車両情報 (`v2/cl/vehicle`)**: 一般的な車両登録情報。
- **車両技術検査 (`v3/cl/vehicle`)**: 技術検査ステータスと履歴。[チリ - 車両技術検査](/verifik-ja/vehicle-validation/chile/vehicle-technical-review) を参照してください。
- **車両 SOAP (`v2/cl/vehicle-soap`)**: 強制保険証明書。[チリ - 車両 SOAP](/verifik-ja/vehicle-validation/chile/vehicle-soap) を参照してください。

## 車両情報照会

### エンドポイント

```
https://api.verifik.co/v2/cl/vehicle
```

**ナンバープレート**（`plate`、スペースやドットなし）で照会します。一般的なフィールドには、**メーカー**、**車種**、**年式**、**VIN/シャーシ**、**エンジン**、**色**、**車両種別**、**罰金サマリー**、登記が公開している場合は **所有者の RUT** などが含まれます。**フリートオンボーディング**、**担保確認**、**請求** ワークフローに使用できます。

### ヘッダー

| 名前          | 値                 |
| ------------- | ------------------ |
| Accept        | `application/json` |
| Authorization | `Bearer <token>`   |

### パラメーター

| 名前 | 型 | 必須 | 説明 | 例 |
| ---- | ---- | ---- | ---- | ---- |
| `plate` | string | **はい** | 照会対象のナンバープレート。スペースやドットを含めずに入力してください。 | `ABC123` |

## 機能

### 車両情報照会
- **車両登録**: チリにおける車両登録を検証します。
- **技術仕様**: シャーシ番号、エンジン番号、製造元、モデルを取得します。
- **車両詳細**: 年式、色、種別の情報にアクセスできます。
- **所有者情報**: 車両所有者の RUT と詳細情報を取得します。
- **罰金状況**: 当該車両に紐づく未払い罰金の有無を確認できます。
- **盗難状況**: 車両に盗難届が出されているかを検証します。
- **公共交通**: 公共交通分類を確認できます。

### 一般機能
- **複数のプログラミング言語に対応**: JavaScript、Python、PHP、Swift をサポートします。
- **リアルタイムデータ**: 最新の情報にアクセスできます。
- **包括的なエラーハンドリング**: 各種シナリオに対応した詳細なエラーレスポンスを提供します。
- **構造化されたレスポンス**: 署名検証付きの整形された JSON レスポンスを返します。
- **チリ登録機関との連携**: チリの公式車両データベースに直接アクセスできます。
