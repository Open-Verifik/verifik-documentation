### エンドポイント

```
GET https://api.verifik.co/v2/co/fasecolda/values-by-plate
```

コロンビア登録車両について、クラス・メーカー・モデル・年式・推定価値・積載量・燃料タイプなどの詳細を取得します。**ナンバー** を **クエリ** パラメータ **`plate`** で渡してください（スペース・句点なし）。

### ヘッダー

| 名前 | 値 |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### パラメーター

| 名前 | 型 | 必須 | 説明 |
| --- | --- | --- | --- |
| `plate` | string | はい | 照会するナンバー（スペース・句点なし）。 |

### リクエスト

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/fasecolda/values-by-plate", {
  params: { plate: "ABC123" },
  headers: {
    Accept: "application/json",
    Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
  },
});
console.log(data);
```

```python
import os, requests
```

### メモ

- Fasecolda はコロンビア公式の車両評価システムです。
- **`valueModel`** はモデル年ごとの履歴価格の配列です。
- 金額はコロンビアペソ（COP）です。
- **`bcpp`** は基準的な商業ベース価格の目安です。
- **`estado: "USADO"`** は中古価格区分の例です。
