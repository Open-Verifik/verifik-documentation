### エンドポイント

```
https://api.verifik.co/v2/co/medellin/vehicle/fines
```

**コロンビア・メデジン**のナンバーに対する**地方交通反則と comparendo**を返します。違反行、金額、違反者情報（利用可能な場合）を含みます。**市政コンプライアンス**、**フリート管理**、メデジン mobility 記録に限定した**デューデリジェンス**に利用できます。

主なフィールド: **`plate`**、**`informacionComparendo`**、**`valorTotalAdeudado`** および関連配列。

### ヘッダー

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### パラメーター

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `plate` | string | Yes | ナンバー（5–8 文字）。スペースやピリオドなし。 |

### リクエスト

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/medellin/vehicle/fines", {

  headers: {
    Accept: "application/json",
    Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
  },
});
```

### 備考

- **Sandbox:** ナンバー **`ABC10001`**–**`ABC10010`** は固定フィクスチャ; **`ERR404`** → **404**。
- **メデジンのみ**対象; 全国 SIMIT は [SIMIT - 反則](/vehicle-validation/colombia/simit-fines) を参照。
- 市政ポータルの**可用性**は変動する場合があり、障害時は **5xx** エラーになることがあります。
