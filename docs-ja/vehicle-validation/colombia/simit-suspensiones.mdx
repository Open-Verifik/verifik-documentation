### エンドポイント

```
GET https://api.verifik.co/v2/co/simit/suspensiones
```

**SIMIT** で、運転免許の**停止・取消**や関連制裁（ソースが返す場合は決議番号・日付など）を照会します。名義人は **`documentType`** と **`documentNumber`** で指定します。

### ヘッダー

| 名前 | 値 |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### パラメーター

| 名前 | 型 | 必須 | 説明 |
| --- | --- | --- | --- |
| `documentType` | string | はい | 名義人の証明書種別。例: `CC`, `PA`, `CE`, `TI`, `RC`, `NIT`。 |
| `documentNumber` | string | はい | 照会する番号（スペース・句読点なし）。長さ超過時は `409` になることがあります。 |

### リクエスト

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/simit/suspensiones", {
	params: { documentType: "CC", documentNumber: "123456789" },
	headers: {
		Accept: "application/json",
		Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
	},
});
console.log(data);
```

### 注記

- **SIMIT** はコロンビアの全国的交通違反・制裁データベースです。
- `data` のキー名や配列は**上流の SIMIT** に依存します（例は代表的な形）。
- **`suspensiones`** が空の場合は、その照会で停止がなかった可能性があります。
