### エンドポイント

```
https://api.verifik.co/v2/co/simit/comparendos
```

交通違反サービスは、コロンビアにおいて特定の車両に関連する違反の詳細情報を提供します。ナンバープレートを入力すると、各違反の状態、日付、場所、違反の種類を確認できます。

### ヘッダー

| 名前 | 値 |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### パラメーター

| 名前 | 型 | 必須 | 説明 |
| --- | --- | --- | --- |
| `documentType` | string | はい | 書類の種類。有効値: `CC`, `PA`, `CE`, `TI`, `RC`。 |
| `documentNumber` | string | はい | 照会する書類番号（スペース・ピリオドなし）。 |

### リクエスト

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/simit/comparendos", {
  params: { 
    documentType: "CC", 
    documentNumber: "123456789" 
  },
  headers: { 
    Accept: "application/json", 
    Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` 
  },
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os, requests

url = "https://api.verifik.co/v2/co/simit/comparendos"
```

### 注記

- SIMIT はコロンビアの統合交通違反管理システムです。
- 「Comparendo」はスペイン語で交通違反を指します。
- 金額はコロンビア・ペソ（COP）です。
- 状態「Pendiente」は違反金の支払いが未払いであることを示します。
- 写真検知による違反は `fotodeteccion: true` で示されます。
