### エンドポイント

```
GET https://api.verifik.co/v2/co/bogota/vehicle/accidentality
```

コロンビア・ボゴタに登録された車両の事故履歴を取得します。クエリ **`plate`** にナンバーを指定します。レスポンスの `accident` 配列には日付、様式（`form`）、深刻度（`seriousness`）が含まれます。

### ヘッダー

| 名前 | 値 |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### パラメーター

| 名前 | 型 | 必須 | 説明 |
| --- | --- | --- | --- |
| `plate` | string | はい | 照会するプレート。5～6文字。スペースやピリオドを含めないでください。 |

### リクエスト

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/bogota/vehicle/accidentality", {
  params: { plate: "ABC123" },
  headers: { 
    Accept: "application/json", 
    Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` 
  },
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os, requests

url = "https://api.verifik.co/v2/co/bogota/vehicle/accidentality"
headers = {"Accept": "application/json", "Authorization": f"Bearer {os.getenv('VERIFIK_TOKEN')}"}
params = {"plate": "ABC123"}
r = requests.get(url, headers=headers, params=params)
print(r.json())
```

### メモ

- **SOLO DAÑOS** は物損のみ（負傷なし）を意味します。
- **`form`** は公式の事故届・報告書の参照番号です。
- **`date`** はスペイン語表記（曜日と日付）で返る場合があります。
