### エンドポイント

```
GET https://api.verifik.co/v2/co/fasecolda/values-by-code
```

Fasecolda の **`codeFasecolda`**（車種・グレード等のカタログコード）を **クエリ** で渡し、車両の属性と商用価格情報を取得します。

### ヘッダー

| 名前 | 値 |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### パラメーター

| 名前 | 型 | 必須 | 説明 |
| --- | --- | --- | --- |
| `codeFasecolda` | string | はい | Fasecolda が割り当てる車両構成のコード。 |

### リクエスト

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/fasecolda/values-by-code", {
  params: { codeFasecolda: "08032036" },
  headers: {
    Accept: "application/json",
    Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
  },
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os, requests

url = "https://api.verifik.co/v2/co/fasecolda/values-by-code"
headers = {"Accept": "application/json", "Authorization": f"Bearer {os.getenv('VERIFIK_TOKEN')}"}
params = {"codeFasecolda": "08032036"}
r = requests.get(url, headers=headers, params=params)
print(r.json())
```

### メモ

- Fasecolda のコードは車種・装備の組み合わせを示す識別子です。
- ナンバー照会（values-by-plate）と同等の詳細情報が得られる構成です。
- 金額はコロンビアペソ（COP）です。
- **`bcpp`** は基準的な商業ベース価格の目安です。
- **`estado: "USADO"`** は中古価格区分の例です。
