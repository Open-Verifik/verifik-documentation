### エンドポイント

```
https://api.verifik.co/v2/co/simit/consultar/placa
```

このサービスでは、コロンビアで特定の車両ナンバープレートに関連する反則金および違反の詳細情報を取得できます。応答には各違反の種類、反則金額、現在の状態が含まれます。

### ヘッダー

| 名前 | 値 |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### パラメーター

| 名前 | 型 | 必須 | 説明 |
| --- | --- | --- | --- |
| `plate` | string | はい | 照会するナンバープレート（スペース・ピリオドなし）。 |

### リクエスト

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/simit/consultar/placa", {
  params: { plate: "ABC123" },
  headers: { 
    Accept: "application/json", 
    Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` 
  },
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os, requests

url = "https://api.verifik.co/v2/co/simit/consultar/placa"
headers = {"Accept": "application/json", "Authorization": f"Bearer {os.getenv('VERIFIK_TOKEN')}"}
params = {"plate": "ABC123"}
r = requests.get(url, headers=headers, params=params)
print(r.json())
```

### 注記

- この総合サービスは、反則金、講習、分割払い合意を含む車両に関するすべての SIMIT データを提供します。
- 「Multas」には車両ナンバーに関連する交通違反の詳細があります。
- 「Cursos」には車両の違反に関連する交通安全講習が表示されます。
- 「AcuerdosPago」には未払い反則金に関する分割払い合意が表示されます。
- 金額はコロンビア・ペソ（COP）です。
