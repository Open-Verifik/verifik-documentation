### エンドポイント

```
GET https://api.verifik.co/v2/co/runt/vehicle-by-vin
```

**`vin`** を **クエリ** で渡し、コロンビア **RUNT** に登録された車両の **包括的な履歴と技術レポート** を取得します。ナンバーが分からない場合や、対物照合で VIN を確認する用途に適します。

### ヘッダー

| 名前 | 値 |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### パラメーター

| 名前 | 型 | 必須 | 説明 |
| --- | --- | --- | --- |
| `vin` | string | はい | 照会する **17 桁の VIN**（スペースなし、RUNT の表記どおり）。 |

### リクエスト

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/runt/vehicle-by-vin", {
	params: { vin: "JTEBH9FJ7DK091425" },
	headers: {
		Accept: "application/json",
		Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
	},
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os, requests

url = "https://api.verifik.co/v2/co/runt/vehicle-by-vin"
headers = {"Accept": "application/json", "Authorization": f"Bearer {os.getenv('VERIFIK_TOKEN')}"}
params = {"vin": "JTEBH9FJ7DK091425"}
r = requests.get(url, headers=headers, params=params)
print(r.json())
```

### 注記

- **VIN** は車両固有の **17 文字**の識別子です。
- レスポンスには一般情報、SOAT、テクノメカニカ等、**RUNT** が返すフィールドが含まれる場合があります。
- **`ACTIVO`** は登録が有効な状態の例です。
