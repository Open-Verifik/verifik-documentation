# SIMIT — 照会

SIMIT 照会サービスは、コロンビアの車両および運転者に関する包括的な交通違反情報へのアクセスを提供します。

### エンドポイント

```
https://api.verifik.co/v2/co/simit/consultar
```

### ヘッダー

| 名前 | 値 |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### パラメーター

| 名前 | 型 | 必須 | 説明 |
| --- | --- | --- | --- |
| `documentType` | string | はい | 文書タイプ。許可される値: `CC`, `PA`, `CE`, `TI`, `RC`, `NIT`。                              |
| `documentNumber` | string | はい | 照会する文書番号。スペースやドットを含めずに入力してください。                                |

### リクエスト

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/simit/consultar", {
  params: { 
    documentType: "CC", 
    documentNumber: "123456789" 
  },
  headers: { 
    Accept: "application/json", 
    Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` 
  },
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os, requests

url = "https://api.verifik.co/v2/co/simit/consultar"
headers = {"Accept": "application/json", "Authorization": f"Bearer {os.getenv('VERIFIK_TOKEN')}"}
params = {"documentType": "CC", "documentNumber": "123456789"}

print(r.json())
```

### 注意事項

- 本サービスは、罰金、講習、分割払い合意を含む、対象者の SIMIT データ全体を提供する包括的なサービスです。
- "Multas" には、違反コードと説明を含む交通違反の詳細が含まれます。
- "Cursos" は、対象者が受講した交通教育講習を示します。
- "AcuerdosPago" は、未払いの罰金に対する分割払い合意を表示します。
- 金額はコロンビアペソ（COP）で表示されます。
