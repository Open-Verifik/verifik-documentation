# SIMIT - 照会

SIMIT 照会サービスは、コロンビアにおける車両および運転者の交通違反情報に包括的にアクセスできます。

### エンドポイント

```
https://api.verifik.co/v2/co/simit/consultar
```

### ヘッダー

| 名前 | 値 |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### パラメーター

| 名前 | 型 | 必須 | 説明 |
| --- | --- | --- | --- |
| `documentType` | string | はい | 書類の種類。有効値: `CC`, `PA`, `CE`, `TI`, `RC`, `NIT`。 |
| `documentNumber` | string | はい | 照会する書類番号（スペース・ピリオドなし）。 |

### リクエスト

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/simit/consultar", {
  params: { 
    documentType: "CC", 
    documentNumber: "123456789" 
  },
  headers: { 
    Accept: "application/json", 
    Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` 
  },
});
console.log(data);
```

```python
import os, requests
```

### 注記

- この総合サービスは、反則金、講習、分割払い合意を含む個人に関するすべての SIMIT データを提供します。
- 「Multas」には違反コードと説明を含む交通違反の詳細があります。
- 「Cursos」には本人が受講した交通安全講習が表示されます。
- 「AcuerdosPago」には未払い反則金に関する分割払い合意が表示されます。
- 金額はコロンビア・ペソ（COP）です。
