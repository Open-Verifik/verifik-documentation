# 🇵🇾 パラグアイ — 車両情報

本サービスは、ナンバープレートを用いてパラグアイの車両に関する詳細情報を提供します。レスポンスには、ブランド、シャーシ、種別、年式、車軸数、所有者氏名、サービス種別、車両の現在状況、ナンバープレートなどの詳細が含まれます。

これらの情報は、車両検証、所有権確認、コンプライアンス監視などさまざまな用途に役立ちます。

## エンドポイント

```
GET https://api.verifik.co/v2/py/vehicle
```

### ヘッダー

| 名前          | 値                 |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### パラメーター

| 名前 | 型 | 必須 | 説明 | 例 |
| ---- | ---- | ---- | ---- | ---- |
| `plate` | string | **はい** | 照会するナンバープレート。 | `ABC123` |

## 機能

- **車両情報の取得**: パラグアイの車両登記簿から包括的な車両詳細を取得します。
- **ナンバープレート検証**: パラグアイで登録された車両のナンバープレートを検証します。
- **車両仕様**: 以下を含む詳細な車両仕様にアクセスできます。
  - ブランドおよび製造元情報
  - 車両識別のためのシャーシ番号
  - 車両種別および分類
  - 製造年式およびモデル詳細
  - 車両構成の車軸数
- **所有者情報**: 車両の現在の所有者情報を取得します。
- **サービス種別の分類**: 車両のサービス種別（例: CARGA NACIONAL）に関する情報を取得します。
- **車両ステータスの監視**: 車両の現在状況とステータスにアクセスできます。
- **コンプライアンス検証**: 車両のコンプライアンスおよび登録状況を検証します。
- **複数のプログラミング言語に対応**: JavaScript、Python、PHP、Swift をサポートします。
- **リアルタイムデータ**: 最新の車両情報にアクセスできます。
- **包括的なエラーハンドリング**: 各種シナリオに対応した詳細なエラーレスポンスを提供します。
- **パラグアイ登録機関との連携**: パラグアイの公式車両データベースに直接アクセスできます。
- **車両種別の分類**: トラック、タンカー、商用車など各種車両に対応しています。
- **所有権確認**: 法的・商業的目的のために車両所有権を検証します。
