# 🇵🇾 パラグアイ — 車両情報

**ナンバープレート**を指定して、パラグアイの車両に関する詳細情報を取得します。ブランド、シャーシ、種別、年式、車軸数、所有者、サービス種別、車両の現状、プレート番号などが含まれる場合があります。

車両確認、所有権チェック、コンプライアンス監視に利用できます。

## Endpoint

```
GET https://api.verifik.co/v2/py/vehicle
```

### Headers

| Name          | Value              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parameters

| Name | Type | Required | Description | Example |
| ---- | ---- | -------- | ----------- | ------- |
| `plate` | string | **Yes** | 照会するプレート。 | `ABC123` |

## 機能

- **登録照会**: パラグアイ車両登録からの詳細
- **プレート検証**: 国内登録プレート
- **仕様**: ブランド、シャーシ、種別、年式、車軸
- **所有者**: 名義情報
- **サービス・状態**: サービス種別（例: CARGA NACIONAL）と車両状況
- **言語例**: JavaScript、Python、PHP、Swift
- **リアルタイムデータ**: 公式登録に基づく最新情報
