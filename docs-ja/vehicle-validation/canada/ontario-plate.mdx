# 🇨🇦 カナダ — オンタリオ ナンバー有効期限

**ナンバー**と**車両許可番号**（オンタリオ登録書類 / グリーンカード）を使用して、オンタリオのナンバーステッカー有効期限を確認します。

典型的なレスポンスフィールド: `expired`、`plateNumber`、`permitNumber`、`expiryDate`、`success`。`expiryDate` が現在時刻より前の場合、`expired` は `true` です。

オンタリオ政府の登録情報源に対してステッカーの有効性を迅速に確認する際に利用します。フリートオンボーディング、コンプライアンス、車両検証に適しています。

## API リファレンス

### Endpoint

```
https://api.verifik.co/v2/ca/ontario/plate
```

クエリパラメータ **`plate`** と **`permitNumber`** を指定して GET リクエストを送信します。

### ヘッダー

| 名前 | 値 |
| ---- | -- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### パラメータ

| 名前 | 型 | 必須 | 説明 | 例 |
| ---- | -- | ---- | ---- | -- |
| `plate` | string | **はい** | オンタリオ乗用ナンバー **ABCD123** 形式（英字4 + 数字3、スペースなし）。 | `BKTR456` |
| `permitNumber` | string | **はい** | オンタリオ登録書類の許可番号（8文字）。 | `N9166001` |

### リクエスト

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/ca/ontario/plate", {
  params: { plate: "BKTR456", permitNumber: "N9166001" },
  headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});
console.log(data);
```
