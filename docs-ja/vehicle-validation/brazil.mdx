# 🇧🇷 ブラジル — 車両情報

## 車両情報照会

### エンドポイント

```
GET https://api.verifik.co/v2/br/vehicle
```

ブラジルの**ナンバープレート**を指定すると、**Verifik** は1回のリクエストで構造化された車両データを返し、実物の資産を検証済みの正規化された属性に紐づけられます。**フリート・モビリティのオンボーディング**、**保険の見積もり・事故対応**、**担保付き融資・担保審査**、ナンバーが主な識別子となる**マーケットプレイスやディーラーでの確認**などを想定しています。

Verifik API はそのナンバーに対する**正規化された車両レコード**を返します。**ナンバー**、**メーカー**、**モデル**、**モデル年式**と**製造年**、**シャーシ（VIN）**、**エンジン**、**色**、**ボディタイプ**、**燃料種別**、**トランスミッション**、**ドア数**、**製造元または工場**、該当する場合は **FIPE コード**、登記に**不備や制限の指標**がある場合はその指標などを含みます。処理を続ける前に、目の前の車両が公式のラインと一致するか確認するために使用します。

### ヘッダー

| 名前          | 値                 |
| ------------- | ------------------ |
| Accept        | `application/json` |
| Authorization | `Bearer <token>`   |

### パラメーター

| 名前 | 型 | 必須 | 説明 |
| ---- | ---- | ---- | ---- |
| `plate` | string | **はい** | 照会するナンバープレート（英数字。大文字・小文字は正規化されます）。例: `PZY7H82`。 |

### レスポンスフィールド

| フィールド | 型 | 説明 |
| ---------- | -- | ---- |
| `bodyType` | string | 登録機関から返される車両タイプのラベルです。 |
| `brand` | string | 車両ブランドです。 |
| `chassis` | string | 登録機関から返されるシャーシ番号または VIN です。 |
| `color` | string | 登録されている車両色です。 |
| `country` | string | 車両レコードの国です。 |
| `denatranWarning` | string | `Sem Restrição` などの Denatran または登録ステータス通知です。 |
| `doors` | string | 登録ソースから返される乗車定員です。 |
| `engine` | string | 登録上の排気量またはエンジン参照値です。 |
| `factory` | string | 原産地または製造地域です。 |
| `fipeCodes` | string[] | 利用可能な場合の車両に関連する FIPE 参照コードです。 |
| `fuelType` | string | 登録されている燃料タイプです。 |
| `irregularitiesCount` | number | 検出された有効な制限または異常の数です。 |
| `irregularityCode` | string | 有効な制限ラベルを 1 つの文字列にまとめた値です。制限がない場合は空です。 |
| `manufacturer` | string | 登録機関から返されるメーカー名です。 |
| `model` | string | 登録されているモデル説明です。 |
| `modelYear` | string | モデル年です。 |
| `plate` | string | 照会結果として返される正規化済みナンバープレートです。 |
| `transmission` | string | 利用可能な場合のトランスミッション値です。登録ソース由来のレコードでは空文字になることがあります。 |
| `vehicle` | string | 登録上の車両カテゴリです。 |
| `version` | string | 利用可能な場合の最適な FIPE モデル/バージョン一致です。 |
| `yearOfManufacture` | string | 製造年です。 |

### メモ

- クエリ文字列で **必須** なのは **`plate`** のみです。
- 登録ソース由来のレコードでは **`transmission`** が空文字で返されることがあります。
- **`doors`** は登録ソースの乗車定員を表します。
- FIPE の一致がない場合、**`fipeCodes`** は空配列になることがあります。
- プライマリソースで一致がない場合、fallback プロバイダーでは返されるフィールド範囲が少し異なることがあります。
- 一部のアカウントでは **ナンバープレート別の拡張データシート** プロダクトが有効になります。その機能が有効なときは AppFeature カタログに記載の URL を使用してください。
