# 🇪🇨 エクアドル — 車両罰金

ナンバープレートに紐づくエクアドルの車両について、登録情報（プレート、クラス、モデル等）や現状、サービス種別、登録有効期限、**未払いの罰金**などを取得します。

交通法規へのコンプライアンスやフリート管理に利用できます。

### エンドポイント

```
GET https://api.verifik.co/v2/ec/vehiculo/placa/multas
```

**`plate`**（ナンバープレート）で照会します。登録情報と**未払い罰金**を返し、**コンプライアンス**・**フリート**・**リスク**確認に利用できます。

### ヘッダー

| 名前          | 値                 |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### パラメーター

| 名前 | 型 | 必須 | 説明 | 例 |
| ---- | ---- | ---- | ---- | ---- |
| `plate` | string | **はい** | 照会するプレート（スペースや句点なし）。 | `ABC1234` |

## 機能

- **罰金情報**: プレートに紐づく罰金の一覧と詳細
- **登録情報**: クラス、モデル、ステータス、有効期限など
- **車両仕様**: 排気量、製造国、年式など（ソースが返す範囲）
- **コンプライアンス**: 交通法規に基づくモニタリング
- **多言語サンプル**: JavaScript、Python、PHP、Swift
