# ペルー — 車両検証

ペルーの公式データベースおよびシステムへのアクセスを通じて、車両検証および保険情報を提供する包括的な車両検証サービスです。

## 利用可能なサービス

### 🚗 **車両情報サービス**
- **[車両情報](/vehicle-validation/peru/vehicle-information)** — ナンバープレートを用いて、ペルーで登録された車両に関する詳細情報を取得します。
- **[車両保険](/vehicle-validation/peru/vehicle-insurance)** — ペルーの車両に関する保険ステータスおよび契約情報を確認します。

## サービス詳細

### 車両情報
ペルーの車両情報サービスは、ナンバープレートに基づいて、ペルーで登録された車両に関する詳細データを提供します。レスポンスには、メーカー、モデル、年式、エンジンおよびシャーシの製造番号、定員、用途種別などの主要情報が含まれます。

### 車両保険
本サービスは、ペルーにおける車両の保険ステータスに関する情報を提供します。保険会社名、契約開始日・終了日、車両のナンバープレート、契約番号、用途種別、車両クラス、契約ステータスなどの詳細を返します。

## 機能

- ✅ **公式データソース**: ペルーの公式車両データベースにアクセスできます。
- ✅ **リアルタイム情報**: 最新の車両および保険データを取得できます。
- ✅ **複数のサービス**: 車両情報と保険検証の両方に対応します。
- ✅ **シンプルなパラメーター**: 最小限の必須フィールドで使いやすい API を提供します。
- ✅ **包括的なレスポンス**: 各照会に対して詳細な情報を返します。
- ✅ **認証された結果**: すべてのレスポンスは Verifik.co により認証されます。
