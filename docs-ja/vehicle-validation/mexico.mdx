# メキシコ

### エンドポイント

```
https://api.verifik.co/v2/mx/vehiculo/placa
```

車両情報サービスは、ナンバープレートに基づいてメキシコの車両に関する詳細情報を提供します。本サービスでは、メーカー、モデル、年式、VIN などの情報を取得できます。GET リクエストの `plate` クエリパラメーターでナンバープレートを送信すると、該当する場合は登録関連のフィールドを含むレスポンスが返されます。

### ヘッダー

| 名前          | 値                 |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### パラメーター

| 名前   | 型     | 必須 | 説明                              | 例        |
| ------ | ------ | ---- | --------------------------------- | --------- |
| plate  | String | はい | 照会する車両のナンバープレート。   | `ABC1234` |

### リクエスト

```javascript
import axios from 'axios';

const options = {
  method: 'GET',
  url: 'https://api.verifik.co/v2/mx/vehiculo/placa',
  params: {plate: 'ABC1234'},
  headers: {
    Accept: 'application/json',
    Authorization: 'Bearer <your_token>'
  }
};

try {
  const { data } = await axios.request(options);
  console.log(data);
} catch (error) {
  console.error(error);
}
```

```json

```

## ユースケース

- **保険の検証**: 保険目的で車両詳細を検証します。
- **自動車販売店**: 購入前に車両情報を検証します。
- **フリート管理**: 会社の車両情報を追跡します。
- **法執行機関**: 車両登録および所有権を検証します。
- **デューデリジェンス**: 取引のための車両情報を検証します。
