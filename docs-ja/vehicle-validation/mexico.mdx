# メキシコ

### エンドポイント

```
https://api.verifik.co/v2/mx/vehiculo/placa
```

車両情報サービスは、メキシコのナンバープレートからメーカー、車種、年式、VIN などの詳細を返します。GET のクエリパラメータ `plate` にプレートを指定します。一致があれば登録に関するフィールドが含まれます。

### ヘッダー

| 名前          | 値              |
| ------------- | ---------------- |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### パラメーター

| 名前  | 型     | 必須 | 説明           | 例       |
| ----- | ------ | ---- | -------------- | -------- |
| plate | String | はい | 照会するナンバー。 | `ABC1234` |

### リクエスト

```javascript
import axios from 'axios';

const options = {
  method: 'GET',
  url: 'https://api.verifik.co/v2/mx/vehiculo/placa',
  params: {plate: 'ABC1234'},
  headers: {
    Accept: 'application/json',
    Authorization: 'Bearer <your_token>'
  }
};

try {
  const { data } = await axios.request(options);
  console.log(data);
} catch (error) {
  console.error(error);
}
```

### レスポンス

```json
{
  "success": true,
  "data": {
    "plate": "ABC1234",
    "make": "Nissan",
    "model": "Sentra",
    "year": "2021",
    "engine": "2.0L",
    "transmission": "CVT",
    "fuelType": "Gasoline",
    "vin": "1N4AL3AP8JC123456",
    "registrationDate": "10/08/2021",
    "status": "active",
    "owner": "Carlos Rodríguez",
    "state": "Jalisco",

```

### エラー

```json
{
  "success": false,
  "error": "Vehicle not found",
  "code": "VEHICLE_NOT_FOUND"
}
```

## 用途

- **保険**: 車両詳細の確認
- **ディーラー**: 購入前の検証
- **フリート**: 社用車管理
- **当局**: 登録・所有者の確認
- **デューデリジェンス**: 取引前の確認
