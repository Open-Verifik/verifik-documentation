# 🇧🇴 ボリビア — 車両情報

本サービスは、ナンバープレートを用いてボリビア国内に登録された車両の詳細情報を取得します。保険ポリシー番号、ブランド、クラス（例: 業務用オートバイ）、登録地、サービス種別（例: 自家用）、モデルなどを返します。

### エンドポイント

```
GET https://api.verifik.co/v2/bo/vehicle
```

ナンバープレート（**`plate`**）でボリビアの車両を照会します。車両詳細・登録情報などを返し、**車両検証**や **コンプライアンス**に利用できます。

### ヘッダー

| 名前          | 値                 |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### パラメーター

| 名前 | 型 | 必須 | 説明 | 例 |
| ---- | ---- | ---- | ---- | ---- |
| `plate` | string | **はい** | 照会するナンバープレート。 | `ABC1234` |

## 機能

- **車両情報の取得**: ボリビアの車両登記から詳細情報を取得します。
- **ナンバープレート検証**: 国内登録のナンバープレートを照会します。
- **保険情報**: 車両の保険ポリシー番号を取得します。
- **車両分類**: クラス（例: 業務用オートバイ）などの詳細を取得します。
- **登録地**: 登録管轄などの情報にアクセスします。
- **サービス種別**: 自家用・商用などの区分を確認します。
- **ブランド・モデル**: メーカーおよびモデル情報を取得します。
- **複数言語のサンプル**: JavaScript、Python、PHP、Swift の例を提供します。
- **リアルタイムデータ**: 最新の車両情報にアクセスします。
- **エラーハンドリング**: 各種シナリオ向けのエラーレスポンスを返します。
