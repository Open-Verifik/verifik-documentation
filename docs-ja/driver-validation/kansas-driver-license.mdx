# カンザス州（米国）— 運転免許の照会

**カンザス州**で発行された運転免許の **状態、制限、裏書、指定** などを API で照会します。有効な免許番号と、必須の本人確認パラメータを指定すると、免許の状態・有効期限・制限・裏書・指定などの情報を返します。

州の規制に沿った **運転者資格の確認** に利用できます。

### エンドポイント

```
GET https://api.verifik.co/v2/usa/kansas/driver-license
```

**`documentNumber`**（免許番号）、**`dateOfBirth`**（生年月日、**DD/MM/YYYY**）、**`firstName`**、**`lastName`** をクエリパラメータで指定します。日付形式は API の要件に合わせてください。

### ヘッダー

| 名前          | 値                 |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### パラメーター

| 名前 | 型 | 必須 | 説明 | 例 |
| ---- | ---- | ---- | ---- | ---- |
| `documentNumber` | string | はい | カンザス州の運転免許番号。 | `K12345678` |
| `dateOfBirth` | string | はい | 生年月日（**DD/MM/YYYY**）。 | `29/01/1974` |
| `firstName` | string | はい | 名。 | `MATEO` |
| `lastName` | string | はい | 姓。 | `VERIFIK` |
