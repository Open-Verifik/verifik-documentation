# 🇵🇪 ペルー — 運転免許

**DNI**（*Documento Nacional de Identidad*）でペルーの運転免許の詳細を取得します。有効免許、手続き履歴、規定コース、制限、制裁などが含まれ、**フリート**、**人事**、**KYC** で運転資格の確認に利用できます。

### Endpoint

```
GET https://api.verifik.co/v2/pe/driver-license
```

`documentType=DNI` と8桁の `documentNumber` で照会します。**氏名**、**免許区分**（A、AIV、B）、**licenses**、**procedures**、**regulatoryCourses**、**sanctions**、**tickets** などが返ります（登録の可否による）。

### ヘッダー

| Name          | Value              |
| ------------- | ------------------ |
| Accept        | `application/json` |
| Authorization | `Bearer <token>`   |

### パラメータ

| 名前 | 型 | 必須 | 説明 | 例 |
| ---- | ---- | ---- | ---- | ---- |
| `documentType` | string | **はい** | 書類種別。許可値: `DNI`。 | `DNI` |
| `documentNumber` | string | **はい** | 照会するDNI（空白・ピリオドなし）。 | `44141274` |

### 備考

- `documentNumber` は公式登録に運転免許記録がある DNI と一致する必要があります。

## 機能

- **DNI 照会**: `documentType=DNI` と書類番号。
- **免許区分**: A、AIV、B および仮免許のフラグ。
- **有効免許**: 番号、区分、状態、制限、発行センター、有効期限。
- **手続き履歴**: 更新（Revalidación）と状態（例: Vigente、VENCIDA）。
- **規定コース**: 登録・修了の詳細（該当時）。
- **制裁・違反**: 登録に応じた任意の配列。
- **構造化レスポンス**: Verifik 署名・認証メタデータ付き JSON。
