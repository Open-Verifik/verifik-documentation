# 🇨🇦 ケベック — 運転免許

Verifik の運転免許検証 API は、カナダ・ケベック州で発行された運転免許を**文書番号**で照会します。免許クラス、ステータス、発行関連情報、照会に紐づく確認番号などが返る場合があります。**KYC**、**フリート・モビリティのオンボーディング**、**コンプライアンス** に利用できます。

**安全にアクセスするには Authorization ヘッダーが必要です。**

## API リファレンス

### エンドポイント

```
GET https://api.verifik.co/v2/ca/quebec/driver-license
```

ケベックの運転免許を州の記録と照合します。免許証に記載の**文書番号**を、そのままの表記で渡してください。

### ヘッダー

| 名前          | 値                 |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### パラメーター

| 名前 | 型 | 必須 | 説明 | 例 |
| ---- | ---- | ---- | ---- | ---- |
| `documentNumber` | string | **はい** | 照会する運転免許番号。証明書の表記どおりに入力してください。 | `L12345678901` |
