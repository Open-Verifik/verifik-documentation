# 🇨🇱 チリ — 運転免許

本サービスでは、ライセンス番号をクエリパラメーターとして指定することで、チリの運転免許に関する詳細情報を取得できます。RUT（納税者識別番号）、住所、免許クラス、管理日などの情報を取得でき、運転免許に関する重要なデータを構造化された形式で提供します。

### エンドポイント

```
GET https://api.verifik.co/v2/cl/driver-license
```

チリの運転免許を **ライセンス番号**（クエリ `documentNumber`）で照会します。返却されるフィールドには、登録の可用性に応じて、**RUT**、**免許クラス**、**制限事項**、**管理日**、**自治体**、**住所** などが含まれます。**フリート**、**労働力**、**KYC** チェックで運転資格が重要となる場面で有用です。

### ヘッダー

| 名前          | 値                 |
| ------------- | ------------------ |
| Accept        | `application/json` |
| Authorization | `Bearer <token>`   |

### パラメーター

| 名前 | 型 | 必須 | 説明 | 例 |
| ---- | ---- | ---- | ---- | ---- |
| `documentNumber` | string | **はい** | 照会対象のライセンス番号。スペースやドットを含めずに入力してください。 | `12345678` |

## 機能

- **RUT 検証**: チリの納税者識別番号を検証します。
- **免許クラス情報**: 免許クラス（A、B、C など）の詳細を取得します。
- **管理日**: 免許の管理日および直近の管理日を取得できます。
- **運転者情報**: 運転者の氏名、住所、自治体を取得します。
- **免許制限**: 運転に関する制限事項を取得できます。
- **手続き情報**: 免許の手続き（再発行、更新など）の詳細を取得します。
- **複数のプログラミング言語に対応**: JavaScript、Python、PHP、Swift をサポートします。
- **リアルタイムデータ**: 最新の運転免許情報にアクセスできます。
- **包括的なエラーハンドリング**: 各種シナリオに対応した詳細なエラーレスポンスを提供します。
- **構造化されたレスポンス**: 署名検証付きの整形された JSON レスポンスを返します。
- **チリ登録機関との連携**: チリの公式運転免許データベースに直接アクセスできます。
