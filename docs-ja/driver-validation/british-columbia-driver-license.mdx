# 🇨🇦 ブリティッシュコロンビア — 運転免許

Verifik の運転免許検証 API を使用すると、文書番号と姓により、ブリティッシュコロンビア州（カナダ）で発行された運転免許の真正性および有効性を検証できます。KYC（Know Your Customer）プロセスの効率化、不正防止、各種規制要件への対応を簡単に実現できるよう設計されています。

ドライバーの運転資格を、迅速・安全・自動的に確認したいビジネス向けに構築しました。

**この情報に安全にアクセスするには Authorization ヘッダーが必要です。**

## この API は何を検証しますか？

本 API はカナダ・ブリティッシュコロンビア州の公式記録と直接接続し、以下を検証します。

-   **運転免許番号**: BC 州運転免許番号の真正性を検証します。
-   **本人一致**: 入力された姓が運転免許の登録情報と一致するかを確認します。
-   **免許の有効性**: 運転免許が現在有効かどうかを確認します。

これらを検証することで、運転免許が真正かつ有効であることを確認でき、なりすましや不正のリスクを大幅に低減できます。

### エンドポイント

```
GET https://api.verifik.co/v2/ca/british-columbia/driver-license
```

ブリティッシュコロンビア州運転免許検証サービスは、カナダ・ブリティッシュコロンビア州で発行された運転免許の真正性および有効性をシンプルに検証する手段を提供します。文書番号と姓を指定するだけで、運転免許を迅速に検証できます。

### ヘッダー（Headers）

| 名前          | 値                 |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### パラメーター（Parameters）

| 名前 | 型 | 必須 | 説明 | 例 |
| ---- | ---- | ---- | ---- | ---- |
| `documentNumber` | string | **はい** | 照会対象の運転免許番号。ドキュメント上の表記どおりに入力してください。 | `1123456` |
| `lastName` | string | **はい** | 運転免許に記載されている姓。 | `HELLO` |

## 一般的なユースケース

-   **フィンテックと銀行**: 口座開設や融資申請の際に運転免許を即時検証します。
-   **ライドシェアと配送**: プラットフォーム稼働前にドライバーを認証します。
-   **HR と採用**: 採用ワークフローの一部としてドライバー資格を検証します。
-   **保険と自動車**: 保険契約や請求処理の前に運転免許の有効性を確認します。
-   **フリート管理**: 運用開始前に法人ドライバーの運転資格を検証します。

## 公式ソースと信頼性

カナダ・ブリティッシュコロンビア州の公式運転免許システムと直接接続し、検証済みかつ最新の情報を提供します。
すべての照会は厳格なセキュリティおよび規制基準に基づいて処理されます。

-   **自動化されたコンプライアンス**: ユーザー体験を損なうことなく検証チェックを効率化し、不正を防止します。

-   **容易な統合**: REST API または互換 SDK を通じて簡単に接続できます。

データの安全性を最優先します。Verifik は高度な暗号化（HTTPS / TLS 1.3）と厳格なプライバシー管理基準を採用し、機密性を確保します。
当サービスは可用性を 24 時間 365 日監視しており、チームのアクセスを安全に保つロールベースのアクセス制御を提供します。

## Verifik について

Verifik は、ラテンアメリカおよび北米における本人確認、コンプライアンス、不正防止のためのリーディングプラットフォームです。
