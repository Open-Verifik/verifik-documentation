# ボリビア - 車両 SOAT

Verifikは**ナンバープレート**でボリビア公式**SOAT**（*Seguro Obligatorio de Accidentes de Tránsito*）登録を照会します。**フリートコンプライアンス**、**保険確認**、**交通チェック**にこのエンドポイントを使用してください。

ソースが提供する場合、SOAT補償の詳細を返します：

- **`plate`** — 車両ナンバープレート（**5–8**文字）、大文字に正規化
- **`managementYear`** — SOAT管理または補償年
- **`validFrom`** / **`validTo`** — 補償有効期間（DD/MM/YYYY）
- **`vehicleType`** — 車両カテゴリ（例：AUTOMOVIL、VAGONETA、MOTOCICLETA）
- **`useType`** — 使用分類（PARTICULAR、COMERCIAL）
- **`department`** — 保険に関連する県（department）
- **`status`** — 算出ステータス：VIGENTE、VENCIDO、PENDIENTE
- **`force`** — 任意；`true`でキャッシュをバイパスしソースから更新
- Verifik **署名**付きレスポンス

### Endpoint

```
GET v2/bo/vehicle-soat
```

### Headers

| Header | Value | Description |
| --- | --- | --- |
| Accept | application/json | レスポンス形式 |
| Authorization | Bearer \<token\> | Verifik JWT |

### Parameters

| Field | Type | Required | Description |
| --- | --- | --- | --- |
| plate | String | Yes | スペースなしの車両ナンバープレート（**5–8**文字）。 |
| force | String | No | `true`に設定するとキャッシュをバイパスし最新データを取得します。 |

### Notes

- **`plate`**は必須で、ミドルウェアが大文字に正規化します。
- **`status`**は現在日付に対する**`validFrom`**と**`validTo`**から算出されます。
- MongoDBキャッシュレコードをバイパスする必要がある場合は**`force=true`**を使用してください。
- レスポンスは**機微な車両データ**です。適用されるプライバシー法を遵守してください。
