# ブラジルの法人検証

Verifik の法人検証 API は、公的データを用いてブラジルの企業を認証します。KYB（Know Your Business）の効率化、不正防止、規制対応を支援します。

企業、パートナー、ベンダーの適法性を迅速・安全に確認したいビジネス向けに構築しました。

## この API は何を検証しますか？

公式記録と接続し、次を検証します。

- **企業名と CNPJ 番号**: *Cadastro Nacional da Pessoa Jurídica*（CNPJ）に対応します。
- **企業情報**: 商号、法人格、開設日、ステータス、住所などを返します。
- **経済活動**: 主たる付帯の業種コードと説明を提供します。
- **企業の一致**: 企業情報が CNPJ 番号と一致するかを確認します。

## API リファレンス

### エンドポイント

```
https://api.verifik.co/v2/br/company
```

ブラジル法人を公式の **CNPJ** 登記と照合します。`documentType=CNPJ` と、区切りなしの 14 桁の `documentNumber` を送ります。典型的なフィールドには登記上の状況、住所、法人格、主・従たる経済活動などが含まれます。**KYB**、**取引先デューデリジェンス**、**コンプライアンス** プログラムに利用できます。

### ヘッダー

| 名前          | 値                 |
| ------------- | ------------------ |
| Accept        | `application/json` |
| Authorization | `Bearer <token>`   |

### パラメーター

| 名前 | 型 | 必須 | 説明 |
| ---- | ---- | ---- | ---- |
| `documentType` | string | **はい** | 文書タイプ。許可される値: `CNPJ`。 |
| `documentNumber` | string | **はい** | 区切りなしの 14 桁 CNPJ。 |
