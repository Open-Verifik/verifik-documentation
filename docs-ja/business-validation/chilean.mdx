# チリ法人（RUT）照会

### Endpoint

```
https://api.verifik.co/v2/cl/company
```

**RUT** で法人情報を取得します。

### Headers

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parameters

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `documentType` | string | Yes | `RUT` |
| `documentNumber` | string | Yes | 法人 RUT |

### Request

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/cl/company", {
  params: { documentType: "RUT", documentNumber: "212957739" },
  headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});
console.log(data);
```
