# パナマ法人確認

**RUC**（*Registro Único de Contribuyente*）と検証桁（**`dv`**）で法人情報を取得します。**KYB**や取引先審査に利用できます。

## このAPIで確認できること

- **RUC + dv**：納税者番号と検証数字。
- **登記情報**：商号、組織形態、ステータス、役員など（提供される場合）。

## APIリファレンス

### Endpoint

```
https://api.verifik.co/v2/pa/company
```

パナマの**RUC**による法人照会。**`documentType`**（`RUC`）、**`documentNumber`**、**`dv`** を指定します。1回のHTTPS呼び出しで **KYB** ワークフローを支援します。

### ヘッダー

| 名前 | 値 |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### パラメータ

| 名前 | 型 | 必須 | 説明 |
| --- | --- | --- | --- |
| `documentType` | string | はい | `RUC`。 |
| `documentNumber` | string | はい | RUC番号。 |
| `dv` | string | はい | 検証桁。 |

### リクエスト

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/pa/company", {
  params: { documentType: "RUC", documentNumber: "155703400-2-2021", dv: "39" },
  headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});
console.log(data);
```
