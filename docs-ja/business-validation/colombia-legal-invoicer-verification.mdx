# 🇨🇴 コロンビア — DIAN 法定請求者（電子請求）

Verifik の法定請求者確認 API は、コロンビア企業が **DIAN**（*Dirección de Impuestos y Aduanas Nacionales*）に **法定請求者（legal invoicer）** として登録されているかを検証します。KYB の効率化、不正防止、規制要件への対応を支援します。

## この API は何を検証しますか？

本 API は公式 DIAN 記録と直接接続し、以下を検証します。

-   **NIT と電子請求（facturación electrónica）登録**
-   **請求者連絡先**: DIAN が公開する場合、登録 **メール** 等を返します
-   **登録確認**: 電子請求向け **法定請求者** データセットへの登録（一般納税者ステータスとは別 — [DIAN 税務確認](/verifik-ja/business-validation/colombia-dian-verification) を参照）

## API リファレンス

### エンドポイント

```
GET https://api.verifik.co/v2/co/company/dian/invoicer
```

アプリホスト: `GET https://verifik.app/v2/co/company/dian/invoicer`

### ヘッダー

| 名前 | 値 |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### パラメータ

| 名前 | 型 | 必須 | 説明 |
| --- | --- | --- | --- |
| `documentType` | string | はい | 必ず **`NIT`**。 |
| `documentNumber` | string | はい | 税 ID（NIT）。サーバー側で **空白除去**；検証桁の有無は任意。API 検証上 **最小 5 文字**。 |

### 注意

- **`GET` クエリのみ** — ボディなし。JSON レスポンスには **`Accept`** を使用してください。
- 本ルートは **法定請求者 / 電子請求登録** データ（利用可能な場合 **email** 等）を返します。一般 **納税者** ステータスには [DIAN 税務確認](/verifik-ja/business-validation/colombia-dian-verification)（`GET /v2/co/company/dian`）を使用してください。
- DIAN が当該 NIT の請求者 **email** を返さない場合 **404** となることがあります。
