# スペイン企業の検証

公的ソースでスペイン企業を認証します。KYBプロセスを支援します。

パートナー・ベンダー・取引先の確認に適しています。

## このAPIで検証できること

公的記録を照会します:

-   **企業識別子**: CIF/NIF および関連情報。
-   **事業情報**: 名称、ステータス（取得できる場合）。
-   **一致**: 識別子と情報の整合性。

## APIリファレンス

### Endpoint

```
https://api.verifik.co/v2/es/company
```

`documentNumber`（企業識別子）でスペインの法人情報を取得します。

### ヘッダー

| 名前 | 値 |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### パラメータ

| 名前 | 種別 | 必須 | 説明 |
| --- | --- | --- | --- |
| `documentNumber` | string | はい | 企業識別子（CIF/NIF）。 |

-   **B2Bオンボーディング**: パートナー・ベンダー検証
