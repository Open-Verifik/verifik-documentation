# 🇨🇴 コロンビア — RUES 企業フル照会 (v3)

コロンビアの **RUES**（*Registro Único Empresarial y Social*）から、**NIT** と **登録区分（category）** に基づき **フル** の企業情報を返します。要約のみ必要な場合は **`GET /v3/co/rues`** を利用してください（英語: [Colombia RUES (v3)](https://docs.verifik.co/business-validation/colombia-rues-v3)）。

## API リファレンス

### エンドポイント

```
GET https://api.verifik.co/v3/co/rues-complete
```

アプリホスト: `GET https://verifik.app/v3/co/rues-complete`

### ヘッダー

| 名前 | 値 |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### パラメータ

| 名前 | 型 | 必須 | 説明 |
| --- | --- | --- | --- |
| `documentType` | string | はい | 必ず **`NIT`**。 |
| `documentNumber` | string | はい | NIT（サーバー側で英数字以外を除去）。 |
| `category` | string | いいえ | **`RM`**, **`PROP`**, **`RUNEOL`**, **`RNT`**, **`ESAL`**, **`ESOL`**, **`JUEGOS`**, **`EXTRANJERAS`**。省略時は **`RM`**。 |

### メモ

- 上記以外のクエリ（**`municipality`**, **`chamber`**, **`page`** など）は **サポートされません**。
- レスポンス例の詳細は英語ページ [RUES Complete Business Validation (v3)](https://docs.verifik.co/business-validation/rues-complete-v3) を参照してください。
