# パラグアイの企業検証

Verifikの企業検証APIは、**パラグアイ政府の公式データ**で法人を認証します。**KYB**を効率化し、不正を抑止し、コンプライアンスを支援します。

取引先・パートナー・ベンダーの**適法性**を迅速に確認する必要がある企業向けです。

## このAPIで検証できること

公式記録に直接接続し、次を確認します。

-   **会社名とRUC番号**: *Registro Único de Contribuyente*（RUC）に対応。
-   **事業者情報**: 名称および登録詳細。
-   **一致確認**: 情報がRUCと一致するか。

## APIリファレンス

### Endpoint

```
https://api.verifik.co/v2/py/company
```

### Headers

| Name          | Value              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parameters

| Name           | Type    | Required | Description                                    |
| -------------- | ------- | -------- | ---------------------------------------------- |
| `documentType` | string  | Yes      | 文書タイプ。許可値: `RUC`。 |
| `documentNumber` | string | Yes      | 事業者識別子。 |

## 主なユースケース

-   **B2Bオンボーディング**: パートナー・ベンダー登録時の企業検証。
-   **金融サービス**: 口座・取引前の法人認証。
-   **デューデリ**: KYC/KYBフローでの情報検証。
-   **マーケットプレイス**: 事業者売り手の適法性確認。

## 公式ソースと信頼性

**パラグアイ政府の公式ソース**に直接接続します。

## 主なメリット

-   **コンプライアンス自動化**: パートナーオンボーディングを円滑に。
-   **即時結果**: リアルタイムの法人オンボーディングに最適。
-   **信頼できるデータ**: 公式記録。
-   **簡単な連携**: REST APIとSDK。

## コンプライアンスとセキュリティ

HTTPS/TLS 1.3、プライバシー、**24/7**監視。

## Verifikについて

Verifikはラテンアメリカで本人・企業検証と不正防止を提供する主要プラットフォームであり、**パラグアイ**および地域の公式ソースと接続します。
