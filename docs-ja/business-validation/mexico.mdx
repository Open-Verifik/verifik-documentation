# メキシコにおける法人検証

Verifik API はメキシコ政府の公式データで企業を認証します。KYB を効率化し、不正を抑止します。

## 検証内容

-   **商号と FME**（*Folio Mercantil Electrónico*）
-   **登記情報**：商号、都市、登録
-   **照合**：情報が FME と一致するか

## API リファレンス

### エンドポイント

```
https://api.verifik.co/v2/mx/company
```

このエンドポイントは **FME**（Folio Mercantil Electrónico）でメキシコの法人を照会します。GET で `documentType=FME` と `documentNumber` を指定し、商業登記の正式名称・所在地などの公式登録情報を取得します。

### ヘッダー

| 名前          | 値              |
| ------------- | ---------------- |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### パラメーター

| 名前 | 型 | 必須 | 説明 |
| --- | --- | --- | --- |
| `documentType` | string | はい | 許可値: `FME`. |
| `documentNumber` | string | はい | FME 識別子（例: `N-2021007300`）。 |

### リクエスト

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/mx/company", {
  params: { documentType: "FME", documentNumber: "N-2021007300" },
  headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os, requests

url = "https://api.verifik.co/v2/mx/company"
headers = {"Accept": "application/json", "Authorization": f"Bearer {os.getenv('VERIFIK_TOKEN')}"}
params = {"documentType": "FME", "documentNumber": "N-2021007300"}
r = requests.get(url, headers=headers, params=params)
print(r.json())
```
