# エクアドルの法人検証

Verifik の法人検証 API は、公的な政府データを利用してエクアドルの企業を認証します。KYB（Know Your Business）プロセスの効率化、不正防止、規制要件への対応を支援します。

## この API は何を検証しますか？

本 API は公式記録と直接接続し、以下を検証します。

-   **商号・RUCEC**: *Registro Único de Contribuyentes del Ecuador*（RUCEC）に対応します。
-   **企業情報**: 商号、ステータス、識別子などを返します。
-   **企業の一致**: 入力情報が RUCEC 番号と一致するかを確認します。

これらを検証することで、不正リスクの低減とコンプライアンス運用を支援できます。

## API リファレンス

### エンドポイント

```
GET https://api.verifik.co/v2/ec/company
```

**`documentType`**（`RUCEC`）と **`documentNumber`**（13 桁・区切りなし）でエクアドル法人を照会します。登記ベースのフィールドを返し、**KYB**、**取引先審査**、**コンプライアンス**に利用できます。

### ヘッダー

| 名前          | 値                 |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### パラメーター

| 名前             | 型     | 必須 | 説明                                       |
| ---------------- | ------ | ---- | ------------------------------------------ |
| `documentType`   | string | はい | 文書タイプ。許可される値: `RUCEC`。            |
| `documentNumber` | string | はい | 区切りなしの 13 桁 RUCEC。                   |

## 一般的なユースケース

-   **B2B オンボーディング**: 取引先登録時の企業確認
-   **金融**: 口座開設前の法人認証
-   **コンプライアンス**: KYC/KYB フローでの法人データ検証

## 公式ソースと信頼性

エクアドルの公的ソースと直接接続し、検証済みの情報を提供します。

## Verifik について

Verifik はラテンアメリカ全域で本人確認・KYB・コンプライアンスのため、公式データソースとビジネスを接続します。
