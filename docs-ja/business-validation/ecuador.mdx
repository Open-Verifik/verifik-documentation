# エクアドルの法人検証

Verifik の法人検証 API は、公的な政府データを利用してエクアドルの企業を認証します。KYB（Know Your Business）プロセスの効率化、不正防止、規制要件への対応を支援します。

企業、パートナー、ベンダーの正当性を迅速・安全・自動で確認する必要があるビジネス向けに構築されています。

## この API は何を検証しますか？

本 API は公式記録と直接接続し、以下を検証します。

-   **商号・RUCEC**: *Registro Único de Contribuyentes del Ecuador*（RUCEC）に対応します。
-   **企業情報**: 商号、ステータス、納税者種別、制度、主要経済活動、会計・源泉徴収フラグ、納税者日付、法定代理人を返します。
-   **企業の一致**: 入力情報が RUCEC 番号と一致するかを確認します。

これらを検証することで、不正リスクの低減とコンプライアンス運用を支援できます。

## API リファレンス

### エンドポイント

```
GET https://api.verifik.co/v3/ec/company
```

**`documentType`** `RUCEC` と **`documentNumber`**（13 桁・区切りなし）でエクアドル法人を照会します。経済活動、制度、法定代理人など登記ベースのフィールドを返し、**KYB**、**取引先審査**、**コンプライアンス**に利用できます。

### ヘッダー（Headers）

| 名前           | 値                  |
| -------------- | ------------------- |
| Content-Type   | `application/json`  |
| Authorization  | `Bearer <token>`    |

### パラメータ（Parameters）

| 名前             | 型     | 必須 | 説明                                           | 例              |
| ---------------- | ------ | ---- | ---------------------------------------------- | --------------- |
| `documentType`   | string | はい | 書類種別。許可値: `RUCEC`。                    | `RUCEC`         |
| `documentNumber` | string | はい | スペースやピリオドなしの 13 桁 RUCEC。         | `1793197955001` |

### Notes

- `documentNumber` は区切り文字なしの正確に 13 桁である必要があります。
- サンドボックスデモ RUC: `1790008959001`（ACERO COMERCIAL ECUATORIANO S.A.）。
- ライブ例: `1793197955001`、`1191795453001`。
- レガシー `GET /v2/ec/company` は、より簡潔なレスポンス形式で引き続き利用可能です。

## 一般的なユースケース

-   **B2B オンボーディング**: パートナーまたはベンダー登録時に企業を即座に検証します。
-   **金融サービス**: 口座開設や取引処理の前に企業を認証します。
-   **コンプライアンスとデューデリジェンス**: KYC/KYB ワークフローの一環として企業情報を検証します。
-   **E コマースとマーケットプレイス**: プラットフォームでの販売を許可する前に企業の正当性を確認します。

## 公式ソースと信頼性

エクアドル政府の公式ソースと直接接続し、検証済みで最新の情報を提供します。
すべての照会は、セキュリティおよび規制基準を厳守して処理されます。

## 主なメリット

-   **自動化されたコンプライアンス**: 取引先に摩擦を増やさず不正を防ぐため、KYB チェックを合理化します。
-   **即時結果**: リアルタイムの企業オンボーディングに適した、数秒での検証処理。
-   **信頼できるデータ**: 公式記録から直接取得したデータに依拠できます。
-   **簡単な統合**: REST API で簡単に接続するか、互換 SDK を利用できます。

## コンプライアンスとセキュリティ

データの安全性を最優先します。Verifik は高度な暗号化（HTTPS/TLS 1.3）と厳格なプライバシー管理基準により機密性を確保します。
サービスは可用性のため 24 時間 365 日監視され、ロールベースのアクセス制御でチームのアクセスを安全に保ちます。

## Verifik について

Verifik は、ラテンアメリカ全域における本人確認、コンプライアンス、不正防止のリーディングプラットフォームです。
当社の API は KYC、KYB、AML、生体認証プロセスを自動化し、コロンビア、メキシコ、ペルー、チリ、エクアドル、アルゼンチン、ブラジルなどの公式データソースと企業を接続します。
