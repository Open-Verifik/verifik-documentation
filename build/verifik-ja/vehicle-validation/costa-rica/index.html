🇨🇷 コスタリカ — 車両情報

本サービスは、ナンバープレート を用いてコスタリカで登録された車両に関する詳細情報を返します。一般的なデータには、現在の所有者、車両仕様（排気量、評価額、重量、定員、出力、駆動方式）、違反 履歴、ソースが公開している場合は 過去の所有者 が含まれます。

フリート確認、保険、コンプライアンスのワークフローに使用します。

API リファレンス​

エンドポイント​

https://api.verifik.co/v2/cr/vehicle

クエリパラメーター plate（スペースや句読点なし）を含む GET リクエストを送信します。レスポンスには、構造化された車両データ、所有者フィールド、利用可能な場合は履歴配列が含まれます。

ヘッダー​

名前値
Acceptapplication/json
AuthorizationBearer <token>

パラメーター​

名前型必須説明例
platestringはい照会するナンバープレート。スペースやドットを含めずに入力してください。ABC123

機能​

  • 車両情報の取得: コスタリカの車両登記簿から包括的な車両詳細を取得します。
  • ナンバープレート検証: コスタリカで登録された車両のナンバープレートを検証します。
  • 現在の所有者情報: 車両の現在の所有者情報を取得します。
  • 車両仕様: 排気量、評価額、重量、定員、出力、駆動方式を取得します。
  • 違反履歴: 交通違反の完全な履歴にアクセスできます。
  • 所有者履歴: 過去の車両所有者の時系列リストを参照できます。
  • 司法当局: 違反を扱う司法当局の情報を取得します。
  • チケット情報: 違反のチケット番号やサマリー番号にアクセスできます。
  • 複数のプログラミング言語に対応: JavaScript、Python、PHP、Swift をサポートします。
  • リアルタイムデータ: 最新の車両情報にアクセスできます。
  • 包括的なエラーハンドリング: 各種シナリオに対応した詳細なエラーレスポンスを提供します。
  • コスタリカ登録機関との連携: コスタリカの公式車両データベースに直接アクセスできます。
