Fasecolda — Sinistros (seguro)

Endpoint​

GET https://api.verifik.co/v2/co/fasecolda/sinister

Consulta o histórico de acidentes e sinistros de seguro (sinister) na base Fasecolda. Envie a placa colombiana como parâmetro de consulta plate.

Cabeçalhos​

NomeValor
Acceptapplication/json
AuthorizationBearer <token>

Parâmetros​

NomeTipoObrigatórioDescrição
platestringSimPlaca do veículo, sem espaços nem pontos.

Requisição​

  • Node.js
  • Python
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/fasecolda/sinister", {
  params: { plate: "ABC123" },
  headers: {
    Accept: "application/json",
    Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
  },
});
console.log(data);
import os, requests

url = "https://api.verifik.co/v2/co/fasecolda/sinister"
headers = {"Accept": "application/json", "Authorization": f"Bearer {os.getenv('VERIFIK_TOKEN')}"}
params = {"plate": "ABC123"}
r = requests.get(url, headers=headers, params=params)
print(r.json())

Notas​

  • Sinister no path da API corresponde à terminologia colombiana de sinistros/reclamações.
  • Pérdida Menor Cuantía indica sinistros de menor montante.
  • Cada item pode incluir ID, data do acidente e tipo de cobertura.
