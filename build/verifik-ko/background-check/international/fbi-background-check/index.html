FBI 배경 조회

API 레퍼런스​

엔드포인트​

https://api.verifik.co/v2/fbi

documentType과 documentNumber, 또는 fullName을 사용해 FBI가 공개한 기록을 조회합니다. 일치하는 항목이 있을 경우 응답에는 신원 필드와 대상자에 대한 공개 FBI URL이 포함될 수 있습니다.

참고: 날짜(출생일 또는 발급일)는 dd/mm/yyyy 형식이어야 합니다.

헤더(Headers)​

이름값
Content-Typeapplication/json
AuthorizationBearer <token>

매개변수(Parameters)​

info

지원되는 문서 종류의 전체 목록과 각 종류별 필수 매개변수는 Document Types 가이드를 참고해 주세요.

이름형식필수설명
documentTypestring아니오조회할 문서 종류.
documentNumberstring아니오조회할 문서 번호. 공백이나 점 없이 입력합니다.
fullNamestring아니오documentType과 documentNumber 대신 개인 또는 법인의 이름을 직접 전달할 수 있습니다.

활용 사례​

  • 신원 검증: FBI가 공개한 정보를 기준으로 문서를 검증합니다.
  • 컴플라이언스: 민감한 직위 또는 거래에 대한 검증을 지원합니다.
  • 사기 예방: 관계 형성 전에 신원을 확인합니다.

요청(Request)​

  • JavaScript
  • Python
  • Swift
  • PHP
import axios from "axios";

const options = {
  method: "GET",
  url: "https://api.verifik.co/v2/fbi",
  params: { documentType: "CC", documentNumber: "80251972" },
  headers: {
    Accept: "application/json",
    Authorization: "jwt <tu_token>",
  },
};

try {
  const { data } = await axios.request(options);
  console.log(data);
} catch (error) {
  console.error(error);
}
import http.client

conn = http.client.HTTPSConnection("api.verifik.co")
payload = ''
headers = {}
conn.request("GET", "/v2/fbi?fullName=", payload, headers)
res = conn.getresponse()
data = res.read()
print(data.decode("utf-8"))
var request = URLRequest(url: URL(string: "https://api.verifik.co/v2/fbi?fullName=")!,timeoutInterval: Double.infinity)
request.httpMethod = "GET"

let task = URLSession.shared.dataTask(with: request) { data, response, error in
  guard let data = data else {
    print(String(describing: error))
    return
  }
  print(String(data: data, encoding: .utf8)!)
}

task.resume()

<?php
require_once 'HTTP/Request2.php';
$request = new HTTP_Request2();
$request->setUrl('https://api.verifik.co/v2/fbi?fullName=');
$request->setMethod(HTTP_Request2::METHOD_GET);
$request->setConfig(array(
  'follow_redirects' => TRUE
));
try {
  $response = $request->send();
  if ($response->getStatus() == 200) {
    echo $response->getBody();
  }
  else {
    echo 'Unexpected HTTP status: ' . $response->getStatus() . ' ' .
    $response->getReasonPhrase();
  }
}
catch(HTTP_Request2_Exception $e) {
  echo 'Error: ' . $e->getMessage();
}
