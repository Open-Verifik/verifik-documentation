# Verificar Afiliaciones Colombianas

### Endpoint

```
https://api.verifik.co/v2/co/afiliaciones
```

El servicio de Verificar Afiliaciones Colombianas permite a los usuarios acceder a información detallada sobre las afiliaciones de un individuo con varias entidades en Colombia.

Al proporcionar parámetros como tipo de documento, número de documento y fecha, el servicio devuelve una respuesta que incluye información personal junto con detalles sobre las afiliaciones del individuo con EPS (proveedor de atención médica), ARL (seguro de riesgos laborales), AP (fondo de pensiones), caja de compensación y cesantías (fondo de desempleo).

Este servicio es particularmente valioso para empresas y organizaciones que necesitan verificar las afiliaciones de un individuo en Colombia para propósitos como verificación de empleo o elegibilidad de beneficios.

### **Headers**

| Nombre         | Valor              |
| -------------- | ------------------ |
| Content-Type   | `application/json` |
| Authorization  | `Bearer <token>`   |

### **Parámetros de Consulta**

| Nombre           | Tipo   | Requerido | Descripción                                    | Ejemplo      |
| ---------------- | ------ | --------- | ---------------------------------------------- | ------------ |
| documentType     | String | Sí        | Tipo de documento de la persona a consultar. Parámetros: CC, PA, CE, PEP. | `CC`         |
| documentNumber   | String | Sí        | Número de documento de la persona a consultar, sin espacios ni puntos. | `123456789`  |
| date             | String | Sí        | Fecha de expedición del documento, debe ingresarse como: DD/MM/YYYY. | `05/05/2020` |

### Características

-   **Verificación de Afiliaciones**: Verificar afiliaciones con EPS, ARL, AP, cajas de compensación y cesantías
-   **Información Personal**: Detalles completos de información personal del individuo
-   **Historial de Afiliaciones**: Acceso a historial completo de afiliaciones actuales y anteriores
-   **Información de Seguridad Social**: Detalles sobre régimen, estado de afiliación y beneficios
-   **Respuesta Estructurada**: Formato de datos organizado para fácil integración
-   **Múltiples Lenguajes de Programación**: Soporte para JavaScript, Python, PHP y Swift
-   **Manejo de Errores**: Respuestas de error comprensivas para varios escenarios

## Casos de Uso

- **Verificación de Empleo**: Confirmar afiliaciones de seguridad social para procesos de contratación
- **Elegibilidad de Beneficios**: Verificar afiliaciones para determinar elegibilidad de beneficios
- **Cumplimiento Legal**: Asegurar cumplimiento con requisitos legales de afiliación
- **Auditoría de Nómina**: Verificar que los empleados tengan las afiliaciones requeridas
- **Gestión de Recursos Humanos**: Mantener registros actualizados de afiliaciones de empleados
