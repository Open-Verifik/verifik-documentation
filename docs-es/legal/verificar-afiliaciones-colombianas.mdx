# Verificar Afiliaciones Colombianas

## Endpoint

```
https://api.verifik.co/v2/co/afiliaciones
```

El servicio Verificar Afiliaciones Colombianas permite a los usuarios acceder a información detallada sobre las afiliaciones de un individuo con varias entidades en Colombia.

Al proporcionar parámetros como tipo de documento, número de documento y fecha, el servicio devuelve una respuesta que incluye información personal junto con detalles sobre las afiliaciones del individuo con EPS (proveedor de salud), ARL (seguro de riesgos laborales), AP (fondo de pensiones), caja de compensación (fondo de compensación) y cesantías (fondo de desempleo).

Este servicio es particularmente valioso para empresas y organizaciones que necesitan verificar las afiliaciones de un individuo en Colombia para propósitos como verificación de empleo o elegibilidad de beneficios.

### Headers

| Nombre         | Valor              |
| -------------- | ------------------ |
| Content-Type   | `application/json` |
| Authorization | `Bearer <token>`   |

### Parámetros

| Nombre           | Tipo   | Requerido | Descripción                                    | Ejemplo      |
| ---------------- | ------ | --------- | ---------------------------------------------- | ------------ |
| documentType     | String | Sí        | Tipo de documento de la persona a consultar. Parámetros: CC, PA, CE, PEP. | `CC`         |
| documentNumber   | String | Sí        | Número de documento de la persona a consultar, sin espacios ni puntos. | `123456789`  |
| date             | String | Sí        | Fecha de expedición del documento, debe ingresarse de la siguiente manera: DD/MM/YYYY. | `05/05/2020` |

## Features

-   **Verificación de Afiliaciones**: Verificar afiliaciones con EPS, ARL, AP, cajas de compensación y cesantías
-   **Información Personal**: Acceder a información personal del individuo
-   **Estado de Afiliaciones**: Verificar el estado actual de todas las afiliaciones
-   **Información Laboral**: Acceder a información sobre empleadores y actividades laborales
-   **Información de Salud**: Verificar afiliación a EPS y régimen de salud
-   **Información de Pensiones**: Verificar afiliación a fondos de pensiones
-   **Información de Riesgos**: Verificar afiliación a ARL y actividades laborales
-   **Structured Response**: Formato de datos organizado para fácil integración
-   **Multiple Programming Languages**: Soporte para JavaScript, Python, PHP y Swift
-   **Error Handling**: Respuestas de error comprensivas para varios escenarios

## Casos de Uso

- **Verificación de Empleo**: Verificar afiliaciones laborales para procesos de contratación
- **Elegibilidad de Beneficios**: Verificar afiliaciones para determinar elegibilidad de beneficios
- **Due Diligence**: Verificar afiliaciones para procesos de debida diligencia
- **Cumplimiento Regulatorio**: Cumplir con requisitos regulatorios para verificación de afiliaciones
- **Verificación de Identidad**: Verificar información personal y afiliaciones del individuo
