# 🇨🇱 Chile - Servicios de Validación de Vehículos

Los servicios de validación de vehículos chilenos proporcionan capacidades de verificación integrales tanto para licencias de conducir como para información vehicular. Estos servicios te permiten recuperar información detallada sobre conductores y vehículos chilenos registrados en el país.

## Servicios Disponibles

### 1. Verificación de Licencia de Conducir
Recuperar información detallada sobre una licencia de conducir chilena usando el número de licencia como parámetro de consulta.

### 2. Consulta de Información Vehicular
Verificar información de vehículos registrados en Chile proporcionando el número de placa de matrícula.

---

## Verificación de Licencia de Conducir

### Endpoint

```
GET https://api.verifik.co/v2/cl/driver-license
```

### Headers

| Name          | Value              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parámetros

| Name | Type | Required | Description | Example |
| ---- | ---- | -------- | ----------- | ------- |
| `documentNumber` | string | **Sí** | Licencia a consultar, sin espacios o puntos. | `12345678` |

## Características

### Verificación de Licencia de Conducir
- **Validación de RUT**: Verificar números de identificación tributaria chilenos
- **Información de Clase de Licencia**: Obtener detalles de clase de licencia (A, B, C, etc.)
- **Fechas de Control**: Acceder a fechas de control de licencia y último control
- **Información del Conductor**: Recuperar nombres, direcciones y municipio del conductor
- **Restricciones de Licencia**: Obtener información sobre restricciones de conducción
- **Información de Procedimiento**: Acceder a detalles de procedimiento de licencia (duplicado, renovación, etc.)

### Consulta de Información Vehicular
- **Registro Vehicular**: Verificar registro de vehículos en Chile
- **Especificaciones Técnicas**: Obtener número de chasis, número de motor, fabricante, modelo
- **Detalles del Vehículo**: Acceder a información de año, color y tipo
- **Información del Propietario**: Recuperar RUT del propietario y detalles
- **Estado de Multas**: Verificar multas pendientes asociadas al vehículo
- **Estado de Robo**: Verificar si el vehículo tiene órdenes de robo
- **Transporte Público**: Verificar clasificación de transporte público

### Características Generales
- **Múltiples Lenguajes de Programación**: Soporte para JavaScript, Python, PHP y Swift
- **Datos en Tiempo Real**: Acceder a información actual y actualizada
- **Manejo Completo de Errores**: Respuestas de error detalladas para varios escenarios
- **Respuestas Estructuradas**: Respuestas JSON bien formateadas con verificación de firma
- **Integración con Registro Chileno**: Acceso directo a bases de datos oficiales de vehículos y conductores chilenos
