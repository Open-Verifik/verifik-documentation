# 🇺🇸 Estados Unidos - Información de Vehículos

Este servicio te permite recuperar información detallada sobre un vehículo en Estados Unidos usando su placa de matrícula y estado. La respuesta incluye varios puntos de datos como marca, modelo, año del modelo, configuración del motor, clase del cuerpo, estilo de transmisión y más.

## Endpoint

```
GET https://api.verifik.co/v2/usa/vehicle
```

### Headers

| Name          | Value              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parámetros

| Name | Type | Required | Description | Example |
| ---- | ---- | -------- | ----------- | ------- |
| `plate` | string | **Sí** | Placa de matrícula a consultar, sin espacios ni puntos. | `ABC123` |
| `state` | string | **Sí** | Abreviación del estado donde está registrado el vehículo. | `AL` |

## Características

- **Recuperación de Información de Vehículos**: Obtener detalles completos de vehículos del registro vehicular estadounidense
- **Validación de Placa de Matrícula**: Verificar placas de matrícula de vehículos registradas en estados de EE.UU.
- **Consulta Específica por Estado**: Consultar vehículos por abreviación de estado para resultados precisos
- **Especificaciones Técnicas**: Obtener configuración del motor, transmisión, tipo de combustible y VIN
- **Clasificación de Vehículos**: Acceder a clase del cuerpo e información del tipo de vehículo
- **Detalles de Registro**: Recuperar fecha de registro, estado e información del propietario
- **Información de Ubicación**: Obtener detalles de ciudad y estado de registro
- **Múltiples Lenguajes de Programación**: Soporte para JavaScript, Python, PHP y Swift
- **Datos en Tiempo Real**: Acceder a información vehicular actual y actualizada
- **Manejo Completo de Errores**: Respuestas de error detalladas para varios escenarios
- **Integración con Registro Estadounidense**: Acceso directo a bases de datos oficiales de vehículos de EE.UU.
