### Endpoint

```
https://api.verifik.co/v2/co/fasecolda/sinister
```

El servicio de Verificación de Siniestros Fasecolda proporciona una plataforma digital detallada para verificar el historial de reclamos de seguros de vehículos en Colombia. Al ingresar la placa de matrícula de un vehículo, puedes acceder a información completa sobre cualquier reclamo de seguro registrado asociado con ese vehículo.

### Headers

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parámetros

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `plate` | string | Yes | Placa del vehículo a consultar, sin espacios ni puntos. |

### Solicitud

<Tabs>

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/fasecolda/sinister", {

  headers: { 
    Accept: "application/json", 
    Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`
```

### Notas

- "Siniestro" se refiere a reclamos de seguros o accidentes en la terminología de seguros colombiana.
- "Pérdida Menor Cuantía" indica reclamos de pérdida menor.
- Este servicio es esencial para evaluar el historial de seguros de un vehículo y riesgos potenciales.
- Cada registro de reclamo incluye un ID único, fecha de accidente y tipo de protección.
