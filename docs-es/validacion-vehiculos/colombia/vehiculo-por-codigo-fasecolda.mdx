### Endpoint

```
https://api.verifik.co/v2/co/fasecolda/values-by-code
```

El servicio de Consulta de Valores de Vehículos Fasecolda proporciona información completa sobre vehículos registrados en Colombia usando el código Fasecolda. Puedes acceder a datos detallados incluyendo clase de vehículo, marca, modelo, año, valor estimado, capacidad de carga, tipo de combustible y otros atributos clave.

### Headers

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parámetros

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `codeFasecolda` | string | Yes | Código específico que Fasecolda tiene para identificar vehículos y sus características. |

### Solicitud

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/fasecolda/values-by-code", {
  params: { codeFasecolda: "08032036" },
  headers: { 
    Accept: "application/json", 
    Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` 
  },
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os, requests

url = "https://api.verifik.co/v2/co/fasecolda/values-by-code"
headers = {"Accept": "application/json", "Authorization": f"Bearer {os.getenv('VERIFIK_TOKEN')}"}
```

### Notas

- Los códigos Fasecolda son identificadores únicos para modelos de vehículos y configuraciones.
- Este servicio proporciona la misma información detallada que el servicio basado en placa.
- Los valores se proporcionan en Pesos Colombianos (COP).
- El campo `bcpp` contiene el precio comercial base del vehículo.
- `estado: "USADO"` indica valores de vehículos usados.
