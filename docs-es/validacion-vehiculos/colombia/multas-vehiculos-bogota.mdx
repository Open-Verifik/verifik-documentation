### Endpoint

```
https://api.verifik.co/v2/co/bogota/vehicle/fines
```

Este servicio te permite verificar multas asociadas con un vehículo en Bogotá, Colombia, usando su número de placa de matrícula. La respuesta proporciona información detallada incluyendo tipo de multa, estado, número, fechas y detalles de pago.

### Headers

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parámetros

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `plate` | string | Yes | Placa a consultar, sin espacios ni puntos. |

### Solicitud

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/bogota/vehicle/fines", {
  params: { plate: "ABC123" },
  headers: { 
    Accept: "application/json", 
    Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` 
  },
});
console.log(data);
```

  <TabItem value="python" label="Python">

```python
import os, requests

url = "https://api.verifik.co/v2/co/bogota/vehicle/fines"
```

### Notas

- El estado de multa "vigente" indica que la multa está actualmente activa y requiere pago.
- El campo `total` incluye el monto original de la multa más cualquier interés acumulado.
- Usa el campo `number` para referenciar multas específicas para pago o apelaciones.
