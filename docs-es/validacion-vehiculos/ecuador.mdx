# 🇪🇨 Ecuador - Multas de Vehículos

El servicio de Multas de Placa de Vehículos de Ecuador te permite recuperar información sobre multas asociadas con la placa de matrícula de un vehículo en Ecuador. Puedes acceder a detalles como la información de registro del vehículo, incluyendo el número de placa, clase y modelo. El servicio también proporciona el estado actual, tipo de servicio y fecha de vencimiento del registro, junto con cualquier multa pendiente vinculada a la placa de matrícula.

Este servicio es útil para individuos y empresas que buscan mantenerse informados sobre las multas asociadas con sus vehículos y asegurar el cumplimiento de las leyes de tránsito ecuatorianas.

## Endpoint

```
GET https://api.verifik.co/v2/ec/vehiculo/placa/multas
```

### Headers

| Name          | Value              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parámetros

| Name | Type | Required | Description | Example |
| ---- | ---- | -------- | ----------- | ------- |
| `plate` | string | **Sí** | Número de placa a consultar, sin espacios ni puntos. | `ABC1234` |

## Características

- **Información de Multas de Vehículos**: Recuperar información integral sobre multas asociadas con placas de matrícula de vehículos ecuatorianos
- **Detalles de Registro Vehicular**: Acceder a información de registro vehicular incluyendo número de placa, clase y modelo
- **Información de Estado del Vehículo**: Obtener estado actual del vehículo, tipo de servicio y fecha de vencimiento del registro
- **Multas Pendientes**: Ver todas las multas pendientes vinculadas a la placa de matrícula
- **Detalles de Multas**: Acceder a información detallada de multas incluyendo:
  - Números de infracción e información de entidad
  - Números de citación y documentación
  - Fechas de emisión, notificación y límites de pago
  - Puntos de penalización y estado de pago
  - Montos de sanción, multas y totales a pagar
  - Referencias de artículos legales y descripciones de violaciones
- **Especificaciones del Vehículo**: Obtener capacidad del cilindro, país de fabricación, modelo y año
- **Información de Registro**: Acceder a cantón de registro, última fecha de registro y fechas de vencimiento
- **Historial de Pagos**: Ver último año de pago y fechas de revisión
- **Monitoreo de Cumplimiento**: Mantenerse informado sobre multas y asegurar cumplimiento de leyes de tránsito ecuatorianas
- **Múltiples Lenguajes de Programación**: Soporte para JavaScript, Python, PHP y Swift
- **Datos en Tiempo Real**: Acceder a información de multas actual y actualizada
- **Manejo Completo de Errores**: Respuestas de error detalladas para varios escenarios
- **Integración con Registro Ecuatoriano**: Acceso directo a base de datos oficial de multas vehiculares de Ecuador
