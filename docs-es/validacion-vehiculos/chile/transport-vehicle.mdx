# 🇨🇱 Chile - Vehículo de transporte público

Consulta datos de inscripción de vehículos de transporte público en Chile (tipo de servicio, capacidad, operador, folio y estado del servicio). Usa este endpoint cuando necesites información de taxi o servicio urbano asociada a la placa, además del [registro vehicular general](..).

### Endpoint

```
GET https://api.verifik.co/v2/cl/transport-vehicle
```

### Encabezados

| Name          | Value              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parámetros

| Name    | Type   | Required | Description                               | Example  |
| ------- | ------ | -------- | ----------------------------------------- | -------- |
| `plate` | string | **Sí**   | Placa a consultar, sin espacios o puntos. | `CXTB61` |

### Campos de respuesta (`data`)

| Campo | Descripción |
| ----- | ----------- |
| `plate` | Placa consultada |
| `reportGeneratedAt` | Texto con fecha y hora del informe generado |
| `rntEntryDate` | Fecha de inscripción en el registro de transporte |
| `serviceType` | Tipo de servicio público autorizado |
| `capacity` | Capacidad de pasajeros |
| `vehicleStatus` | Estado del vehículo en el registro de transporte |
| `vehicleCertificateExpiryDate` | Fecha de vencimiento del certificado del vehículo |
| `region` | Código de región administrativa |
| `manufactureYear` | Año de fabricación |
| `seatBeltsMandatory` | Cinturones obligatorios (`SI` / `NO`) |
| `vehicleAgeYears` | Antigüedad del vehículo en años |
| `brand` | Marca |
| `model` | Modelo |
| `serviceFolio` | Número de folio del servicio |
| `serviceFleetNumber` | Número de flota |
| `serviceResponsibleName` | Razón social del operador de transporte responsable |
| `serviceStatus` | Estado de la autorización del servicio |
| `serviceExpiryDate` | Fecha de vencimiento de la autorización del servicio |
| `replacementInfo` | Información de reemplazo o notas (puede ir vacío) |
