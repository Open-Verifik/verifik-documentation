# Perú - Información de Vehículo

El servicio de Información de Vehículos Peruanos devuelve datos registrados para una placa (**`plate`**): marca, modelo, año, identificadores de motor y chasis, estado, propietarios cuando estén disponibles, y campos opcionales de ficha técnica.

### Endpoint

```
GET https://api.verifik.co/v3/pe/vehiculo/placa
```

Consulta un vehículo peruano por **`plate`** (sin espacios ni puntos). La respuesta puede incluir `brand`, `model`, `year`, `engineSerial`, `chasisSerial`, `vin`, `serial`, `color`, `status`, `owners` y `technicalSheet` (`engine`, `performance`, `dimensions`) cuando estén disponibles. Útil para **verificación**, **seguros**, **flotas** y **cumplimiento**.

### Headers

| Nombre          | Valor              |
| --------------- | ------------------ |
| Accept | `application/json` |
| Authorization   | `Bearer <token>`   |

### Parámetros

| Nombre    | Tipo   | Requerido | Descripción | Ejemplo |
| --------- | ------ | --------- | ----------- | ------- |
| `plate`   | string | **Sí**    | Número de placa a consultar, sin espacios ni puntos. | `ABC123` |

### Solicitud

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from 'axios';

const { data } = await axios.get('https://api.verifik.co/v3/pe/vehiculo/placa', {
  params: { plate: 'ABC123' },
  headers: {
    Accept: 'application/json',
    Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
  },
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os
import requests

url = "https://api.verifik.co/v3/pe/vehiculo/placa"
headers = {"Accept": "application/json", "Authorization": f"Bearer {os.getenv('VERIFIK_TOKEN')}"}
params = {"plate": "ABC123"}
response = requests.get(url, headers=headers, params=params)
print(response.json())
```

### Notas

- `technicalSheet` puede ser `null` o estar parcialmente poblado cuando no hay datos técnicos coincidentes.
- Placas de sandbox como `ABC10001`–`ABC10010` devuelven perfiles demo estables.
