### Endpoint

```
GET https://api.verifik.co/v2/br/company
```

Este servicio te permite verificar empresas brasileñas usando CNPJ (Cadastro Nacional da Pessoa Jurídica). La API se conecta directamente con fuentes oficiales para validar información empresarial en Brasil.

### Headers

| Name          | Value              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parámetros

| Name | Type | Required | Description | Example |
| ---- | ---- | -------- | ----------- | ------- |
| `cnpj` | string | **Sí** | CNPJ a consultar, sin espacios ni puntos. | `12345678000190` |

### Solicitud

<Tabs>
  <TabItem value="javascript" label="JavaScript">

```javascript
import axios from 'axios';

const options = {
  method: 'GET',
  url: 'https://api.verifik.co/v2/br/company',
  params: {
    cnpj: '12345678000190'
  },
  headers: {
    'Accept': 'application/json',
    'Authorization': 'Bearer <tu_token>'
  }
};

try {
  const { data } = await axios.request(options);
  console.log(data);
} catch (error) {
  console.error(error);
}
```
