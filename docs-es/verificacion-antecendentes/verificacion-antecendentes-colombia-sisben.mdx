# Colombia — Consulta puntaje SISBEN

### Endpoint

```
GET https://api.verifik.co/v2/co/sisben
```

Devuelve información de **grupo / puntaje SISBEN** desde el portal público de consulta usando **`documentType`** y **`documentNumber`**. La integración utiliza **automatización de navegador**; la respuesta puede tardar más que un REST típico.

### Encabezados

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parámetros

| name | type | required | description |
| --- | --- | --- | --- |
| `documentType` | string | sí | **`CC`**, **`CE`**, **`PEP`**, **`PPT`**, **`PA`**. |
| `documentNumber` | string | sí | Número sin espacios ni puntuación. |

## Características

- **Grupo SISBEN** y textos asociados cuando el portal devuelve resultado.
- **Ubicación**: Municipio y departamento cuando se parsean del informe.

## Casos de uso

- **Programas sociales**: Validación de clasificación SISBEN.
- **Onboarding**: Apoyo a verificación de beneficiarios donde aplique la normativa.

### Notas

- La consulta puede tardar **varios segundos** (navegador automatizado).
- **`PPT`** está permitido por validación; si el formulario oficial cambia, pueden aparecer errores **500** hasta actualizar la integración.
