# Personas Expuestas Políticamente (PEP) en Colombia

La API de Personas Expuestas Políticamente (PEP) de Verifik te ayuda a verificar si un ciudadano o empresa colombiana está listada como Persona Expuesta Políticamente. Este servicio está diseñado para agilizar tus procesos de cumplimiento KYC/AML (Conoce a Tu Cliente/Anti-Lavado de Dinero), prevenir fraudes y asegurar que cumplas con todos los requisitos regulatorios.

Construimos esta integración para organizaciones que necesitan identificar individuos o empresas que ocupan posiciones públicas prominentes o tienen relaciones cercanas con tales personas, según lo requerido por las regulaciones AML.

## ¿Qué verifica esta API?

Nuestra API se conecta directamente con registros oficiales del gobierno colombiano para verificar:

-   **Estado PEP**: Verifica si la persona o empresa está listada como Persona Expuesta Políticamente
-   **Detalles de Declaraciones**: Devuelve información sobre declaraciones, posiciones, entidades y fechas de publicación
-   **Validación de Documentos**: Soporta Cédula de Ciudadanía (CC) para individuos y NIT para empresas
-   **Datos de Cumplimiento**: Proporciona información detallada requerida para verificaciones de cumplimiento AML

Al verificar estos detalles, puedes identificar PEPs y aplicar medidas de diligencia debida mejoradas según lo requerido por las regulaciones financieras.

## Endpoint

```
GET https://api.verifik.co/v2/co/politically-exposed-persons
```

### Headers

| Name          | Value              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parámetros

| Name | Type | Required | Description | Example |
| ---- | ---- | -------- | ----------- | ------- |
| `documentType` | string | **Sí** | Tipo de documento. Valores permitidos: `CC` (para ciudadanos), `NIT` (para empresas). | `CC` |
| `documentNumber` | string | **Sí** | Número de documento sin espacios ni puntos. | `123456789` |

## Notas Importantes

- Este servicio verifica el estado de Personas Expuestas Políticamente (PEP), que es diferente del Permiso Especial de Permanencia (PEP) para ciudadanos venezolanos.
- Usa `documentType=CC` para ciudadanos colombianos; `documentType=NIT` para empresas.
- La API devuelve información detallada sobre declaraciones, posiciones y entidades cuando se encuentra una coincidencia PEP.
- Si no se encuentran registros PEP, la API devuelve un error 404.
