# Deudores Morosos

Este servicio te permite consultar información sobre deudores morosos y registros financieros.

## Endpoint

```
GET https://api.verifik.co/v2/co/deudoresmorosos
```

### Headers

| Name          | Value              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parámetros

| Name | Type | Required | Description | Example |
| ---- | ---- | -------- | ----------- | ------- |
| `documentNumber` | string | **Sí** | Número de documento a consultar. | `123456789` |

### Solicitud

<Tabs>
  <TabItem value="javascript" label="JavaScript">

```javascript
import axios from 'axios';

const options = {
  method: 'GET',
  url: 'https://api.verifik.co/v2/co/deudoresmorosos',
  params: {
    documentNumber: '123456789'
  },
  headers: {
    'Accept': 'application/json',
    'Authorization': 'Bearer <tu_token>'
  }
};

try {
  const { data } = await axios.request(options);
  console.log(data);
} catch (error) {
  console.error(error);
}
```

</Tabs>
