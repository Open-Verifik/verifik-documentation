# Validaciones de Teléfono

Las validaciones de teléfono permiten verificar números de teléfono y enviar códigos de verificación.

## Recursos Disponibles

- [Crear Validación de Teléfono](/verifik-es/recursos/validaciones-de-telefono/crear-validacion-de-telefono)
- [Recuperar una Validación de Teléfono](/verifik-es/recursos/validaciones-de-telefono/recuperar-una-validacion-de-telefono)
- [El Objeto de Validaciones de Teléfono](/verifik-es/resources/validaciones-de-telefono/el-objeto-de-validaciones-de-telefono)

## Características

- **Verificación de Teléfono**: Verifica números de teléfono
- **Códigos de Verificación**: Envía códigos de verificación por SMS o WhatsApp
- **Validación**: Proceso de validación de números de teléfono
