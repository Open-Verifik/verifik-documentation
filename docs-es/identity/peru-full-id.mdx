# 🇵🇪 Perú - ID Completo

Este servicio permite a los usuarios recuperar información detallada sobre las cédulas de identidad peruanas (DNI). El servicio devuelve una respuesta que incluye el tipo de documento, número de documento, nombre completo, primer nombre, apellido, dígito de verificación, fecha de nacimiento, fecha de expedición, fecha de vencimiento, fecha de muerte (si aplica), edad, género, estado civil, provincia, área, distrito de residencia, estado de donante, ubigeo, código postal, área de muerte (si aplica), provincia de muerte (si aplica) y distrito de residencia de muerte (si aplica).

Este servicio integral proporciona información personal y demográfica extensa para ciudadanos peruanos.

## Endpoint

```
GET https://api.verifik.co/v2/pe/cedula/extra
```

### Headers

| Name          | Value              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parámetros

| Name | Type | Required | Description | Example |
| ---- | ---- | -------- | ----------- | ------- |
| `documentType` | string | **Sí** | Tipo de documento. Valor permitido: DNI. | `DNI` |
| `documentNumber` | string | **Sí** | Número de documento de la persona a consultar. | `1234567` |

## Características

- **Información Completa de DNI**: Recuperar información detallada sobre cédulas de identidad peruanas (DNI)
- **Información Personal**: Acceder a nombre completo, primer nombre, apellido y dígito de verificación
- **Datos Demográficos**: Obtener edad, género, estado civil y fecha de nacimiento
- **Detalles del Documento**: Acceder a tipo de documento, número, fecha de expedición y fecha de vencimiento
- **Información Geográfica**: Obtener provincia, área, distrito de residencia, ubigeo y código postal
- **Información de Muerte**: Acceder a fecha de muerte y detalles de ubicación de muerte (si aplica)
- **Estado de Donante**: Verificar estado de donante de órganos
- **Ciclo de Vida del Documento**: Rastrear fechas de expedición y vencimiento
- **Dígito de Verificación**: Acceder al dígito de verificación del DNI para validación
- **Múltiples Lenguajes de Programación**: Soporte para JavaScript, Python, PHP y Swift
- **Datos en Tiempo Real**: Acceder a información de ciudadanos actual y actualizada
- **Manejo Completo de Errores**: Respuestas de error detalladas para varios escenarios
- **Integración con Registro Peruano**: Acceso directo a base de datos oficial de ciudadanos peruanos
- **Demografía Completa**: Información demográfica y geográfica completa
- **Validación de Documento**: Validación completa de DNI con todos los campos disponibles
