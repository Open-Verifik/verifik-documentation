### Endpoint

```
https://api.verifik.co/v2/ve/cedula
```

Verificar identificación venezolana por `documentNumber`.

### Headers

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parámetros

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `documentNumber` | string | Sí | Número de documento. |

### Notas

- `documentNumber` es requerido.
- Algunos campos pueden estar incluidos en la respuesta dependiendo de la disponibilidad de datos: `birthDate`, `gender`, y `maritalStatus`. Estos campos no están garantizados en todas las respuestas.
- Cuando está presente, `birthDate` sigue el formato `YYYY-MM-DD` (ejemplo: "1998-03-01").
