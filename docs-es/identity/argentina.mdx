# Argentina

## Endpoint

```
https://api.verifik.co/v2/ar/cedula
```

El servicio de información de ciudadanos argentinos permite a los desarrolladores verificar la autenticidad de una tarjeta de identificación argentina (Documento Nacional de Identidad, o DNI) proporcionando el número de DNI. El servicio devuelve información como el nombre completo de la persona, el nombre y apellido por separado, y el número de DNI.

Esta información puede ser utilizada para una variedad de propósitos, como verificar la identidad de un cliente o validar la información proporcionada por un usuario.

### Encabezados

| Nombre        | Valor              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parámetros

| Parámetro        | Tipo   | Requerido | Descripción                                    |
| ---------------- | ------ | -------- | ---------------------------------------------- |
| `documentType`   | string | Sí       | Tipo de documento. Parámetro permitido: DNIAR. |
| `documentNumber` | string | Sí       | Número de documento de la persona a consultar sin espacios. |

### Características

-   **Verificación de DNI**: Verificar la autenticidad de los Documentos Nacionales de Identidad argentinos
-   **Datos de Identidad Completos**: Devuelve nombre completo, nombre, apellido y número de documento
-   **Respuesta Estructurada**: Formato de datos organizado para fácil integración
-   **Múltiples Lenguajes de Programación**: Soporte para JavaScript, Python, PHP y Swift
-   **Manejo de Errores**: Respuestas de error completas para varios escenarios
