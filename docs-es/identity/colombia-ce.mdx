# Colombia - CE

Verifica la Cédula de Extranjería (CE), el documento de identificación oficial para extranjeros residentes en Colombia. Este endpoint de API te permite recuperar detalles autenticados, incluyendo el nombre, apellido, número de documento y fecha de vencimiento del titular, simplificando la verificación de identidad para cumplimiento y procesamiento de datos.

## Endpoint

```
GET https://api.verifik.co/v2/co/foreigner-id/ce
```

## Headers

| Nombre        | Valor             |
| ------------- | ----------------- |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`  |

## Parámetros

| Nombre            | Tipo   | Requerido | Descripción                                    |
| ----------------- | ------ | --------- | ---------------------------------------------- |
| `documentNumber` | string | Sí        | Número de documento de la persona a consultar, sin espacios o puntos. |
| `expeditionDate` | string | Sí        | Fecha de expedición del documento. Formato válido: DD/MM/YYYY. |

## Casos de Uso

- **Verificación de Residente Extranjero**: Validar el estado de residencia de extranjeros en Colombia para propósitos de empleo
- **Servicios Financieros**: Verificar la identidad de extranjeros para aplicaciones bancarias y de servicios financieros
- **Cumplimiento Legal**: Asegurar el cumplimiento con las regulaciones colombianas para residentes extranjeros
- **Autenticación de Documentos**: Autenticar documentos de Cédula de Extranjería para propósitos oficiales
- **Verificación de Identidad**: Simplificar los procesos de verificación de identidad para nacionales extranjeros

## Características

- ✅ **Validación de Documento de Extranjero**: Verifica documentos de Cédula de Extranjería (CE) emitidos por Migración Colombia
- ✅ **Datos de Identidad Completos**: Recupera nombre, apellido, número de documento y fecha de vencimiento
- ✅ **Verificación de Estado**: Verifica el estado del documento (activo, vencido, etc.)
- ✅ **Validación en Tiempo Real**: Obtiene información actualizada de fuentes oficiales
- ✅ **Múltiples Lenguajes de Programación**: Soporte para JavaScript, Python, Swift y PHP
- ✅ **Manejo Integral de Errores**: Respuestas de error detalladas para varios escenarios
- ✅ **Cumplimiento de Privacidad**: Sujeto a las leyes de privacidad colombianas y manejo responsable de datos
