# 🇺🇸 Estados Unidos - Verificación de SSN

El Servicio de Verificación de SSN es una herramienta robusta diseñada para verificar Números de Seguro Social (SSN) emitidos en Estados Unidos. Este servicio permite a los usuarios validar rápidamente un SSN proporcionando detalles como el estado emisor y el primer año de emisión. La respuesta incluye el SSN en sí, el año en que fue emitido por primera vez, el estado emisor y el estado de validez del SSN. Esta herramienta es esencial para asegurar la precisión y autenticidad de los Números de Seguro Social.

## Endpoint

```
GET https://api.verifik.co/v2/usa/ssn
```

### Headers

| Name          | Value              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parámetros

| Name | Type | Required | Description | Example |
| ---- | ---- | -------- | ----------- | ------- |
| `documentNumber` | string | **Sí** | SSN a consultar, debe escribirse exactamente como aparece en el documento. | `123-45-678` |

## Características

- **Validación de SSN**: Verificar Números de Seguro Social emitidos en Estados Unidos
- **Información del Estado Emisor**: Obtener detalles sobre qué estado emitió el SSN
- **Primer Año Emitido**: Recuperar el año cuando el SSN fue emitido por primera vez
- **Estado de Validez**: Confirmar si el SSN es válido y activo
- **Múltiples Lenguajes de Programación**: Soporte para JavaScript, Python, PHP y Swift
- **Verificación en Tiempo Real**: Validación instantánea de la autenticidad del SSN
- **Integración Gubernamental**: Integración directa con datos de la Administración del Seguro Social
- **Autenticación Segura**: Autenticación con token Bearer para todas las solicitudes
- **Manejo Completo de Errores**: Respuestas de error detalladas para varios escenarios
- **Validación de Formato**: Asegura que el SSN se proporcione en formato correcto (XXX-XX-XXXX)
