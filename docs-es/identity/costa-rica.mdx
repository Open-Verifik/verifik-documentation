### Endpoint

```
https://api.verifik.co/v2/cr/cedula
```

Verificar Cédula costarricense y retornar detalles de identidad.

### Headers

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parámetros

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `documentType` | string | Sí | Permitido: `CCCR`. |
| `documentNumber` | string | Sí | Número de cédula sin espacios. |

<?php
$ch = curl_init("https://api.verifik.co/v2/cr/cedula");
