# 🇨🇴 Colombia - Cédula de Extranjero

Verificar la Cédula de Extranjería (CE), el documento de identificación oficial para extranjeros residentes en Colombia. Este endpoint de API te permite recuperar detalles autenticados, incluyendo el primer nombre, apellido, número de documento y fecha de vencimiento del titular, simplificando la verificación de identidad para cumplimiento y procesamiento de datos.

Perfecto para empresas y organizaciones que necesitan validar el estado de residencia de extranjeros en Colombia, como para empleo, servicios financieros o cumplimiento legal.

## Endpoint

```
GET https://api.verifik.co/v2/co/foreigner-id/ce
```

### Headers

| Name          | Value              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parámetros

| Name | Type | Required | Description | Example |
| ---- | ---- | -------- | ----------- | ------- |
| `documentNumber` | string | **Sí** | Número de documento de la persona a consultar, sin espacios ni puntos. | `123456789` |
| `expeditionDate` | string | **Sí** | Fecha de expedición del documento. Formato válido: DD/MM/YYYY. | `10/10/2022` |

## Notas Importantes

- La Cédula de Extranjería (CE) es emitida por Migración Colombia para extranjeros residentes en el país. Para otros tipos de documentos, por favor consulta nuestros servicios de validación alternativos.
- El uso de este servicio está restringido a entidades autorizadas o permiso directo del usuario final. Los datos proporcionados están sujetos a las leyes de privacidad colombianas y deben manejarse de manera responsable.
- Siempre verifica con la cédula física CE para confirmar la información recuperada.

## Características

- **Verificación de Cédula de Extranjero**: Verificar documentos de Cédula de Extranjería (CE) para extranjeros residentes en Colombia
- **Validación de Estado de Residencia**: Validar el estado de residencia de extranjeros en Colombia
- **Información del Documento**: Acceder a número de documento, tipo, fecha de expedición y fecha de vencimiento
- **Información Personal**: Recuperar primer nombre, apellido y nombre completo del titular
- **Estado del Documento**: Verificar estado del documento (ej. VENCIDO - vencido)
- **Verificación de Cumplimiento**: Simplificar la verificación de identidad para cumplimiento y procesamiento de datos
- **Aplicaciones Empresariales**: Perfecto para empleo, servicios financieros o cumplimiento legal
- **Múltiples Lenguajes de Programación**: Soporte para JavaScript, Python, PHP y Swift
- **Datos en Tiempo Real**: Acceder a información de extranjeros actual y actualizada
- **Manejo Completo de Errores**: Respuestas de error detalladas para varios escenarios
- **Integración con Inmigración Colombiana**: Acceso directo a base de datos oficial de inmigración colombiana
- **Cumplimiento de Privacidad**: Sujeto a leyes de privacidad colombianas y manejo responsable de datos
