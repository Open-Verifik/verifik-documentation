# 🇨🇴 Colombia - ID PEP

Validar el Permiso Especial de Permanencia (PEP) para ciudadanos venezolanos en Colombia. Este endpoint de API recupera datos autenticados del PEP, incluyendo el nombre completo del titular del permiso, número de documento, fecha de vencimiento y estado, simplificando la verificación de identidad para cumplimiento, verificaciones de elegibilidad o provisión de servicios.

Ideal para organizaciones que necesitan verificar el estatus legal de nacionales venezolanos en Colombia, como para empleo, servicios financieros o programas humanitarios.

## Endpoint

```
GET https://api.verifik.co/v2/co/foreigner-id/pep
```

### Headers

| Name          | Value              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parámetros

| Name | Type | Required | Description | Example |
| ---- | ---- | -------- | ----------- | ------- |
| `documentNumber` | string | **Sí** | Número de documento sin espacios ni puntos. | `123456789` |
| `expeditionDate` | string | **Sí** | Fecha cuando se generó el documento. | `10/10/2024` |

## Notas Importantes

- El PEP es un permiso especial emitido por Migración Colombia exclusivamente para ciudadanos venezolanos. Para otras nacionalidades o tipos de documentos, por favor consulta nuestros servicios de validación alternativos.
- El uso de este servicio está restringido a entidades autorizadas o con el permiso del usuario final. Los datos proporcionados están sujetos a las leyes de privacidad colombianas y deben manejarse de manera responsable.
- Siempre verifica con el documento físico PEP para confirmar la información recuperada.

## Características

- **Validación de Permiso Especial de Permanencia**: Validar PEP (Permiso Especial de Permanencia) para ciudadanos venezolanos en Colombia
- **Verificación de Nacionales Venezolanos**: Verificar el estatus legal de nacionales venezolanos en Colombia
- **Información del Documento**: Acceder a número de documento, tipo PEP, fecha de expedición y fecha de vencimiento
- **Información Personal**: Recuperar primer nombre, apellido y nombre completo del titular del permiso
- **Número de Identificación**: Acceder al número de identificación venezolano
- **Estado del Documento**: Verificar estado del permiso (ej. VENCIDO - vencido)
- **Verificación de Estatus Legal**: Verificar estatus legal para empleo, servicios financieros o programas humanitarios
- **Verificación de Cumplimiento**: Simplificar la verificación de identidad para cumplimiento y verificaciones de elegibilidad
- **Múltiples Lenguajes de Programación**: Soporte para JavaScript, Python, PHP y Swift
- **Datos en Tiempo Real**: Acceder a información PEP actual y actualizada
- **Manejo Completo de Errores**: Respuestas de error detalladas para varios escenarios
- **Integración con Inmigración Colombiana**: Acceso directo a base de datos oficial de inmigración colombiana
- **Cumplimiento de Privacidad**: Sujeto a leyes de privacidad colombianas y manejo responsable de datos
- **Programas Humanitarios**: Ideal para verificación de elegibilidad de programas humanitarios
