# Documentos Permitidos - Scan Studio

Scan Studio soporta los siguientes tipos de documentos:

## Tipos de Documentos Soportados

| Código | Descripción |
| --- | --- |
| `CC` | Cédula de Ciudadanía (Colombia) |
| `RUT` | Registro Único Tributario (Colombia) |
| `PA` | Pasaporte |
| `ENSAPA` | ENSAPA (Panamá) |
| `DNI` | Documento Nacional de Identidad |
| `CCVE` | Cédula de Ciudadanía Venezolana |
| `RIFVE` | RIF Venezolano |
| `TLCC` | Tarjeta de Licencia de Conducir Colombiana |

## Características

- **Extracción de Datos**: Extracción precisa de información de documentos
- **Validación**: Proceso de validación y verificación de datos extraídos
- **Múltiples Formatos**: Soporte para imágenes Base64 o URLs
- **Alta Precisión**: Tecnología OCR avanzada con puntuaciones de confianza
