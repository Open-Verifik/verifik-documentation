# ONU / otros

## 128. ONU

**Ubicación en la colección:** v2 › onu › ONU

**Qué hace (resumen):**

El endpoint de la ONU te permite verificar si una persona está incluida en la lista de sanciones de las Naciones Unidas.

**Tipo de consulta:** `GET`

**Ejemplo de dirección completa** (con valores de ejemplo):

```
https://api.verifik.co/v2/onu?fullName=MUHAMMAD TAHER ANWARI
```

**Parámetros en la dirección (después del `?`):**

| Nombre | Valor de ejemplo | Notas |
| --- | --- | --- |
| documentType | CC | (Required) Tipo de documento. Valores permitidos: CC,CE,PA,RC,,PEP,CCVE,NIT,CURP,DNI, CCECP. |
| documentNumber | 1121329661 | (Required) Número de documento de la persona a consultar, sin espacios ni puntos. |
| fullName | MUHAMMAD TAHER ANWARI |  |
| fullName | ANUAR DE JESUS HERNANDEZ GARCIA |  |

**Cabeceras habituales:**

| Nombre | Valor de ejemplo |
| --- | --- |
| Accept | application/json |

**Ejemplo con la herramienta `curl` (para quien lo pida tu equipo técnico):**

```bash
curl -X GET "https://api.verifik.co/v2/onu?fullName=MUHAMMAD TAHER ANWARI" \
  -H "Accept: application/json"
```
