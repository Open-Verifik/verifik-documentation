# Guatemala

## 95. Cedula Guatemala

**Ubicación en la colección:** v2 › gt › cedula › Cedula Guatemala

**Qué hace (resumen):**

La API de Verificación de Cédula de Identidad Venezolana permite a los usuarios verificar la autenticidad de una cédula de identidad venezolana proporcionando el número de documento. La API devuelve el nombre completo del titular, el nombre y apellido por separado, y el tipo de documento.

Esta información puede ser utilizada por empresas y organizaciones para verificar la identidad de sus clientes, empleados o socios.

**Tipo de consulta:** `GET`

**Ejemplo de dirección completa** (con valores de ejemplo):

```
https://api.verifik.co/v2/gt/cedula?documentType=CUI&documentNumber=2222624190903
```

**Parámetros en la dirección (después del `?`):**

| Nombre | Valor de ejemplo | Notas |
| --- | --- | --- |
| documentType | CUI |  |
| documentNumber | 2222624190903 | (Required) Número de documento a consultar, sin espacios ni puntos. |

**Cabeceras habituales:**

| Nombre | Valor de ejemplo |
| --- | --- |
| Accept | application/json |

**Ejemplo con la herramienta `curl` (para quien lo pida tu equipo técnico):**

```bash
curl -X GET "https://api.verifik.co/v2/gt/cedula?documentType=CUI&documentNumber=2222624190903" \
  -H "Accept: application/json"
```

**Ejemplos de respuesta que guarda Postman:**

- **HTTP 200 (OK)**

```json
{

    "data": {

        "arrayName": [

            "APELLIDO",

            "NOMBRE",

            "SEGUNDO"

        ],

        "documentNumber": "2222624190903",

        "documentType": "CUI",

        "firstName": "NOMBRE SEGUNDO",

        "fullName": "NOMBRE SEGUNDO APELLIDO",

        "lastName": "APELLIDO"

    },

    "signature": {

        "dateTime": "April 8, 2026 11:00 PM",

        "message": "Certified by Verifik.co"

    },

    "id": "IDNT1"

}
```

- **HTTP 404 (Not Found)**

```json
{

    "code": "NotFound",

    "message": "Record not found."

}
```

- **HTTP 409 (Conflict)**

```json
{

    "code": "MissingParameter",

    "message": "documentNumber maximum length exceeded.\n"

}
```
