# Sesiones - Smart Access

Las sesiones en Smart Access se refieren a cada autenticación exitosa realizada por un usuario.

## Definición

Una sesión se cuenta cada vez que un usuario completa exitosamente el proceso de autenticación, independientemente del método utilizado (email, teléfono o biométrica).

## Facturación

- Cada autenticación exitosa cuenta como una sesión
- Los intentos fallidos no cuentan como sesiones
- Las sesiones se facturan según el plan seleccionado

## Límites

Los límites de sesiones varían según el plan seleccionado. Contacta a nuestro equipo de ventas para más información sobre los límites específicos de tu plan.
