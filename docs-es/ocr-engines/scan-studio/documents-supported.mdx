# Documents Supported - Scan Studio

Scan Studio supports the following document types:

## Supported Document Types

| Code | Description |
| --- | --- |
| `CC` | Cédula de Ciudadanía (Colombia) |
| `RUT` | Registro Único Tributario (Colombia) |
| `PA` | Passport |
| `ENSAPA` | ENSAPA (Panama) |
| `DNI` | Documento Nacional de Identidad |
| `CCVE` | Cédula de Ciudadanía Venezolana |
| `RIFVE` | RIF Venezolano |
| `TLCC` | Tarjeta de Licencia de Conducir Colombiana |

## Features

- **Data Extraction**: Precise extraction of document information
- **Validation**: Validation and verification process for extracted data
- **Multiple Formats**: Support for Base64 images or URLs
- **High Accuracy**: Advanced OCR technology with confidence scores
