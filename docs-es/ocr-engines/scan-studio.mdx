### Endpoint

```
https://api.verifik.co/v2/ocr/scan-studio
```

Escaneo Estudio es nuestra API OCR avanzada que utiliza modelos entrenados para lograr una extracción de texto de alta precisión desde varios tipos de documentos. Este servicio proporciona capacidades OCR especializadas con modelos pre-entrenados para tipos de documentos específicos, asegurando alta precisión y extracción de texto confiable.

### Headers

| Name | Value |
| --- | --- |
| Content-Type | `application/json` |
| Authorization | `Bearer <token>` |

### Parámetros

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `image` | string | Yes | Imagen en formato Base64 codificado o una URL donde está alojada la imagen. |
| `model` | string | Yes | Modelo entrenado a usar para la extracción. |
| `cropFace` | boolean | No | Si quieres recortar la cara del documento, pasa esto como `true`. |

### Modelos Disponibles

- **`id_documents`**: Tarjetas de identificación emitidas por el gobierno
- **`drivers_licenses`**: Licencias de conducir de varios países
- **`passports`**: Documentos de pasaporte internacionales
- **`business_documents`**: Facturas, contratos y formularios comerciales
- **`custom`**: Modelos entrenados personalizados disponibles bajo solicitud

### Características

- **Alta Precisión**: Los modelos entrenados proporcionan precisión superior (95%+)
- **Datos Estructurados**: Devuelve datos de campos organizados
- **Múltiples Idiomas**: Soporte para varios idiomas
- **Procesamiento Rápido**: Modelos optimizados para extracción rápida
- **Modelos Personalizados**: Disponibles bajo solicitud para casos de uso específicos

### Notas

- Usa imágenes codificadas en Base64 o URLs directas para el parámetro `image`.
- Especifica el `model` apropiado para mejores resultados.
- El parámetro `cropFace` ayuda a mejorar la precisión enfocándose en el contenido del documento.
