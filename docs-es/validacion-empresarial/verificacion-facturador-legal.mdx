### Endpoint

```
https://api.verifik.co/v2/co/dian/legal-invoicer
```

Valida si una empresa tiene un facturador legal registrado en DIAN.

### Headers

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parámetros

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `nit` | string | Yes | NIT (con o sin dígito de verificación). |

### Solicitud

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/dian/legal-invoicer", {
  params: { nit: "900123456" },
  headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os, requests

url = "https://api.verifik.co/v2/co/dian/legal-invoicer"
headers = {"Accept": "application/json", "Authorization": f"Bearer {os.getenv('VERIFIK_TOKEN')}"}
params = {"nit": "900123456"}
r = requests.get(url, headers=headers, params=params)
print(r.json())
```
