# Colombia - Registro Empresarial RUES

Este servicio te permite validar el registro empresarial colombiano desde RUES (Registro Único Empresarial y Social).

## Endpoint

```
GET https://api.verifik.co/v2/co/rues
```

### Headers

| Name          | Value              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parámetros

| Name | Type | Required | Description | Example |
| ---- | ---- | -------- | ----------- | ------- |
| `nit` | string | **Sí** | NIT de la empresa a consultar, sin espacios ni puntos. | `123456789` |

### Solicitud

<Tabs>
  <TabItem value="javascript" label="JavaScript">

```javascript
import axios from 'axios';

const options = {
  method: 'GET',
  url: 'https://api.verifik.co/v2/co/rues',
  params: {
    nit: '123456789'
  },
  headers: {
    'Accept': 'application/json',
    'Authorization': 'Bearer <tu_token>'
  }
};

try {
  const { data } = await axios.request(options);
  console.log(data);
} catch (error) {
  console.error(error);
}
```

</Tabs>
