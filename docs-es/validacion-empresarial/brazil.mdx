### Endpoint

```
https://api.verifik.co/v2/br/company
```

Obtén información de empresas brasileñas por CNPJ.

### Encabezados

| Nombre | Valor |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parámetros

| Nombre | Tipo | Requerido | Descripción |
| --- | --- | --- | --- |
| `documentType` | string | Sí | Permitido: `CNPJ`. |
| `documentNumber` | string | Sí | CNPJ de 14 dígitos, sin separadores. |
