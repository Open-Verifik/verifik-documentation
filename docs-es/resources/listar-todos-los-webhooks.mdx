# Listar Todos los Webhooks

## Endpoint

```
GET https://api.verifik.co/v2/webhooks
```

Con este servicio, puedes traer todos los Webhooks que has creado.

### Headers

| Name          | Value              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parámetros

| Name       | Type    | Required | Description                                                  |
| ---------- | ------- | -------- | ------------------------------------------------------------ |
| `page`     | number  | No       | Puedes definir el número de página si tienes demasiados registros. |
| `perPage`  | number  | No       | Cuántos registros necesitas por página.                          |
| `like_name` | string | No       | Puedes consultar una condición 'like'                             |
| `where_url` | string | No       | Puedes hacer una comparación exacta dentro de un campo.               |

### Características

- **Soporte de Paginación**: Navegar a través de listas grandes de webhooks
- **Funcionalidad de Búsqueda**: Filtrar webhooks por nombre o URL
- **Datos Completos de Webhook**: Devuelve todos los detalles del webhook incluyendo configuración
- **Múltiples Lenguajes de Programación**: Soporte para JavaScript, Python, PHP y Swift
- **Consultas Flexibles**: Usar condiciones like y coincidencias exactas para filtrar
