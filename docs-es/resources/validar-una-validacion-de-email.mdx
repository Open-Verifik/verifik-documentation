# Validar una Validación de Email

## Endpoint

```
POST https://api.verifik.co/v2/email-validations/{id}/validate
```

Este servicio valida una validación de email proporcionando el código de verificación. El sistema verificará el código contra el código de verificación almacenado y actualizará el estado de validación en consecuencia.

### Headers

| Name          | Value                        |
| ------------- | ---------------------------- |
| Content-Type  | `application/json`           |
| Authorization | `Bearer {YOUR_ACCESS_TOKEN}` |

### Parámetros de Ruta

| Name | Type   | Description                                                      |
| ---- | ------ | ---------------------------------------------------------------- |
| `id` | string | El identificador único del registro de validación de email que quieres validar. |

### Parámetros del Cuerpo

| Name               | Type   | Required | Description                                    |
| ------------------ | ------ | -------- | ---------------------------------------------- |
| `verificationCode` | string | **Sí**   | El código de verificación recibido por email  |

  }

### Características

- **Validación de Códigos**: Valida códigos OTP de 6 dígitos enviados por email
- **Control de Intentos**: Rastrea intentos de validación y límites máximos
- **Expiración**: Maneja códigos expirados con respuestas apropiadas
- **Estado Actualizado**: Actualiza el estado de validación tras verificación exitosa
- **Timestamps**: Registra cuándo se completó la validación
- **Múltiples Lenguajes**: Soporte para JavaScript, Python, PHP y Swift
- **Manejo de Errores**: Respuestas de error detalladas para diferentes escenarios
