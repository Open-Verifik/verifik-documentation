# Listar Todas las Colecciones

Obtener todas las colecciones generadas por un cliente específico. El único parámetro requerido es enviar el token de acceso. Con esto, Verifik identificará las colecciones generadas por este usuario.

## Endpoint

```
GET https://api.verifik.co/v2/face-recognition/collections
```

## Headers

| Nombre        | Valor             |
| ------------- | ----------------- |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`  |

## Casos de Uso

- **Gestión de Colecciones**: Ver todas las colecciones asociadas con tu cuenta de cliente
- **Organización de Proyectos**: Ver colecciones organizadas por diferentes proyectos
- **Resumen de Datos**: Obtener una visión general de todas las colecciones de almacenamiento de datos faciales
- **Integración de API**: Listar colecciones para integración con otros servicios biométricos
- **Monitoreo**: Monitorear el uso y organización de colecciones entre proyectos

## Características

- ✅ **Específico del Cliente**: Retorna solo colecciones pertenecientes a tu cuenta de cliente
- ✅ **Asociación de Proyecto**: Muestra a qué proyecto pertenece cada colección
- ✅ **Información Completa**: Retorna detalles completos de la colección incluyendo códigos y marcas de tiempo
- ✅ **Formato de Array**: Retorna colecciones en un formato de array fácil de procesar
- ✅ **Múltiples Lenguajes de Programación**: Soporte para JavaScript, Python, Swift y PHP
- ✅ **Manejo Integral de Errores**: Respuestas de error detalladas para varios escenarios
- ✅ **Autenticación Requerida**: Acceso seguro usando autenticación Bearer token
