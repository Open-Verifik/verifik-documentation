# Crear una Validación de Nombre

## Endpoint

```
PUT https://api.verifik.co/v2/document-validations/{id}/validate
```

Este endpoint realiza validación de nombre comparando los nombres extraídos del documento de un usuario con registros oficiales del gobierno. Valida que los nombres proporcionados por el usuario coincidan con sus registros de identidad oficiales a través de integraciones de API gubernamentales.

### Headers

| Name          | Value                        |
| ------------- | ---------------------------- |
| Content-Type  | `application/json`           |
| Authorization | `Bearer {YOUR_ACCESS_TOKEN}` |

:::warning
**Prerrequisitos**: Los usuarios deben completar tanto el paso `signUpForm` como el paso `document` antes de que se pueda realizar la validación de nombre.
:::

:::warning
**Característica Adicional**: Las Validaciones de Nombre son una característica adicional y están sujetas a cargos. Cada solicitud de validación incurrirá en una tarifa basada en tu plan.
:::

### Parámetros de Ruta

| Name | Type   | Required | Description                                                          |
| ---- | ------ | -------- | -------------------------------------------------------------------- |
| `id` | string | **Sí**   | El identificador único del registro de validación de documento a validar. |

### Parámetros de Consulta

| Parameter | Type    | Description                                    | Example      |
| --------- | ------- | ---------------------------------------------- | ------------ |
| `force`   | boolean | Forzar re-validación incluso si ya fue validado. | `force=true` |

### Parámetros del Cuerpo

Este endpoint no requiere un cuerpo de solicitud. Toda la información necesaria se recupera automáticamente del registro de validación de documento.

### Regiones y Tipos de Documento Soportados

La validación de nombre está actualmente soportada para las siguientes regiones y tipos de documento:

| Región       | Tipo de Documento | Descripción                  |
| ------------ | ------------- | ---------------------------- |
| **Panamá**   | `CCPA`        | Cédula Nacional de Panamá      |
| **Colombia** | `CC`          | Cédula Nacional de Colombia   |
| **Colombia** | `DRCC`        | Cédula Diplomática de Colombia |
| **Chile**    | `CL`          | Cédula Nacional de Chile     |

:::info
**Nota**: El soporte para regiones y tipos de documento adicionales puede estar disponible. Contacta a tu gerente de cuenta para la lista más actualizada de países y tipos de documento soportados.
:::

### Características

- **Validación Gubernamental**: Compara nombres con registros oficiales del gobierno
- **Múltiples Países**: Soporte para Panamá, Colombia y Chile
- **Tipos de Documento**: Cédulas nacionales y diplomáticas
- **Validación Automática**: Recupera información automáticamente del documento
- **Re-validación**: Opción para forzar re-validación con parámetro `force`
- **Puntuaciones de Confianza**: Puntuaciones detalladas para cada nombre
- **Prerrequisitos**: Requiere completar formulario de registro y documento
- **Característica Premium**: Servicio adicional con cargos por validación
- **Múltiples Lenguajes**: Soporte para JavaScript, Python, PHP y Swift
- **Manejo de Errores**: Respuestas de error detalladas para diferentes escenarios
