### Descripción General

Los créditos sirven como la moneda universal para acceder y consumir nuestro conjunto de servicios, incluyendo SmartCheck, SmartAccess, SmartEnroll, y más. Los créditos simplifican la facturación y el seguimiento de uso al consolidar todas las interacciones—ya sea mediante API o soluciones sin código—en una sola cuenta manejable. Este sistema está diseñado para proporcionar flexibilidad y escalabilidad a los clientes, permitiendo que tanto su personal como los sistemas integrados utilicen sin problemas las ofertas de Verifik.

### Por Qué Usamos Créditos

Los créditos son la columna vertebral del modelo de consumo de servicios de Verifik. He aquí por qué:

* **Acceso Unificado:** Los créditos permiten a los clientes acceder a todos los servicios de Verifik—como SmartCheck (verificación), SmartAccess (entrada segura), y SmartEnroll (automatización de inscripción)—bajo una sola cuenta. Esto elimina la necesidad de mecanismos de facturación o seguimiento separados por servicio.
* **Colaboración en Equipo:** Los clientes pueden asignar personal a su cuenta, y todos los miembros del equipo pueden utilizar créditos para sus respectivas tareas, ya sea a través de llamadas API o soluciones sin código preconstruidas.
* **Flexibilidad Impulsada por API:** La mayoría de las interacciones con Verifik ocurren a través de nuestras APIs, y los créditos se deducen según el uso (por ejemplo, solicitudes API). Esto asegura un modelo de pago por uso que escala con sus necesidades.
* **Simplicidad Sin Código:** Para clientes que prefieren soluciones plug-and-play, nuestras herramientas sin código se integran perfectamente con el mismo sistema de créditos, permitiendo a usuarios no técnicos aprovechar las características de Verifik sin desarrollo personalizado.
* **Transparencia y Control:** Los créditos proporcionan una forma clara de monitorear el uso, gestionar presupuestos y rastrear el consumo de servicios a lo largo del tiempo.

### Tipos de Créditos

- **Créditos SmartCheck**: Para verificación de identidad y documentos
- **Créditos SmartAccess**: Para acceso seguro y autenticación
- **Créditos SmartEnroll**: Para inscripción e incorporación de usuarios
- **Créditos API**: Para uso directo de API
- **Créditos Biométricos**: Para servicios de verificación biométrica

### Gestión de Créditos

#### Compra de Créditos
Los créditos se pueden comprar a través del panel de Verifik o mediante integración API con su sistema de facturación.

#### Seguimiento de Uso
Todo el uso de créditos se rastrea en tiempo real y puede ser monitoreado a través de:
- Analíticas del panel
- Endpoints de API
- Notificaciones por webhook

#### Expiración
Los créditos tienen fechas de expiración para asegurar un uso activo y prevenir la acumulación de créditos no utilizados.

### Casos de Uso

- **Pago por uso**: Escale su uso según las necesidades reales
- **Gestión de Equipo**: Comparta créditos entre los miembros del equipo
- **Control de Presupuesto**: Establezca límites y monitoree el gasto
- **Integración de Servicios**: Use créditos en todos los servicios de Verifik
- **Transparencia**: Visibilidad clara del consumo de servicios
