# Recuperar una Validación de Teléfono

## Endpoint

```
GET https://api.verifik.co/v2/phone-validations/{id}
```

Este servicio recupera un registro de validación de teléfono específico usando su identificador único. La respuesta incluye todos los detalles sobre el proceso de validación de teléfono, incluyendo estado, información OTP y datos del proyecto asociado. Este endpoint es útil para verificar el estado actual de una validación de teléfono o recuperar detalles para propósitos de auditoría.

### Headers

| Name          | Value                        |
| ------------- | ---------------------------- |
| Content-Type  | `application/json`           |
| Authorization | `Bearer {YOUR_ACCESS_TOKEN}` |

### Parámetros de Ruta

| Name | Type   | Description                                                      |
| ---- | ------ | ---------------------------------------------------------------- |
| `id` | string | El identificador único de la Validación de Teléfono que quieres recuperar. |

### Parámetros de Consulta

| Name          | Type   | Description                                                                                    |
| ------------- | ------ | ---------------------------------------------------------------------------------------------- |
| `populates[]` | string | Opcional. Una lista de objetos relacionados para incluir en la respuesta. Opciones disponibles: `client`, `project`, `projectFlow`. |

### Características

- **Recuperación por ID**: Obtén una validación de teléfono específica usando su ID único
- **Información Completa**: Incluye todos los detalles del proceso de validación
- **Datos Relacionados**: Información del cliente, proyecto y flujo de proyecto
- **Estado Detallado**: Estado actual, intentos, límites y timestamps
- **Configuración**: URLs de redirección, webhooks y configuraciones de seguridad
- **Códigos de País**: Información completa del código de país y número
- **Múltiples Lenguajes**: Soporte para JavaScript, Python, PHP y Swift
- **Manejo de Errores**: Respuestas de error detalladas para diferentes escenarios
