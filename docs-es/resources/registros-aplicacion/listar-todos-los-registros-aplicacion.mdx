### Endpoint

**GET** `https://api.verifik.co/v2/app-registrations`

Este endpoint te permite recuperar una lista de todos los Registros de Aplicación dentro de la API de Verifik. Puedes usar esto para obtener una visión detallada de múltiples registros de aplicación, incluyendo su estado, proyecto asociado, estados de validación y más.

### Encabezados

#### Content-Type
Tipo: String  
Requerido: Sí

`application/json`

#### Authorization
Tipo: String  
Requerido: Sí

`Bearer <token>`

### Parámetros de Consulta

#### `page`
Tipo: Number  
Requerido: No

Especifica el número de página para la paginación, comenzando desde 1.

#### `perPage`
Tipo: Number  
Requerido: No

Define el número de elementos por página para la paginación.

#### `populates[]`
Tipo: String  
Requerido: No

Rellena el campo especificado, transformando referencias ObjectId en objetos completos. Opciones disponibles: `emailValidation`, `phoneValidation`, `informationValidation`, `biometricValidation`, `documentValidation`, `documentFace`, `failedBiometricValidations`, `project`, `projectFlow`

#### `where_status`
Tipo: String  
Requerido: No

Condición where para filtrar por estado. Opciones: `STARTED`, `ONGOING`, `COMPLETED_WITHOUT_KYC`, `COMPLETED`, `FAILED`, `NEEDS_MANUAL_VERIFICATION`, `EXPIRED`

<TabItem value="nodejs" label="Node.js">

```javascript
const axios = require("axios");

const config = {
  method: "get",
  url: "https://api.verifik.co/v2/app-registrations",
  headers: {
    "Content-Type": "application/json",
    "Authorization": "Bearer YOUR_ACCESS_TOKEN"
  },
  params: {
    page: 1,
    perPage: 10,
    "populates[]": ["project", "projectFlow", "emailValidation", "phoneValidation"],
    where_status: "COMPLETED"
  }
};

axios.request(config)
.then((response) => {
  console.log(JSON.stringify(response.data));
})
.catch((error) => {
  console.log(error);
});
```

package main

import (
  "fmt"
  "net/http"
  "io/ioutil"
)

func main() {
  url := "https://api.verifik.co/v2/app-registrations?page=1&perPage=10&populates[]=project&populates[]=projectFlow&populates[]=emailValidation&populates[]=phoneValidation&where_status=COMPLETED"

  body, _ := ioutil.ReadAll(res.Body)

  fmt.Println(string(body))
}
