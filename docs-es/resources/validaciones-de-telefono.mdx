# Validaciones de Teléfono

Las Validaciones de Teléfono son el sistema central de verificación dentro de Verifik que procesa y valida números de teléfono de usuarios durante el proceso de autenticación y registro. Piensa en ellas como el punto de control de seguridad que asegura que los usuarios son quienes dicen ser verificando la propiedad de su teléfono a través de códigos de verificación seguros.

## Cómo Funcionan las Validaciones de Teléfono

Cada Validación de Teléfono representa una sesión única de verificación de teléfono que procesa números de teléfono enviados por usuarios. Cuando alguien ingresa su teléfono durante el registro, login o verificación de cuenta, Verifik crea una Validación de Teléfono para enviar un código de verificación seguro por SMS y confirmar la propiedad del teléfono del usuario.

Actualmente, las Validaciones de Teléfono soportan estos tipos:

* **validation** - Proceso de verificación de teléfono estándar
* **login** - Verificación de teléfono para autenticación de usuario
* **onboarding** - Verificación de teléfono para registro de nuevo usuario
* **twoFactor** - Autenticación de dos factores por SMS

## Qué Rastrean las Validaciones de Teléfono

Las Validaciones de Teléfono son sistemas de verificación completos que procesan cada aspecto de la verificación de teléfono:

### Información del Teléfono:
* Verificación de número de teléfono y propiedad
* Código de país e información del operador
* Nombre del usuario y detalles de contacto
* Asociación de proyecto y cliente

### Proceso de Verificación:
* Generación y entrega de contraseña de un solo uso (OTP)
* Estado de envío de SMS y confirmación de entrega
* Validación de código e intentos de verificación
* Expiración y gestión de seguridad

### Características de Seguridad:
* Limitación de velocidad y prevención de abuso
* Generación y validación segura de códigos
* Seguimiento y confirmación de entrega
* Detección y prevención de fraude

## Endpoints de API

### Crear una Validación de Teléfono
```http
POST https://api.verifik.co/v2/phone-validations
```

### Listar Todas las Validaciones de Teléfono
```http
GET https://api.verifik.co/v2/phone-validations
```

### Recuperar una Validación de Teléfono
```http
GET https://api.verifik.co/v2/phone-validations/{validationId}
```

### Validar una Validación de Teléfono
```http
POST https://api.verifik.co/v2/phone-validations/{validationId}/validate
```

## Casos de Uso

- **Autenticación de Usuario**: Verificar propiedad de teléfono para login seguro
- **Registro de Cuenta**: Confirmar números de teléfono durante el registro
- **Autenticación de Dos Factores**: Mejorar la seguridad con verificación por SMS
- **Seguridad**: Fortalecer la seguridad de la cuenta a través de verificación de teléfono
