# Recuperar una Validación Biométrica

## Endpoint

```
GET https://api.verifik.co/v2/biometric-validations/{id}
```

Una Validación Biométrica es una instancia dentro del sistema de Verifik que te permite procesar y validar identidades de usuarios a través de reconocimiento facial y detección de vivacidad. Este endpoint recupera un registro de validación biométrica específico por su identificador único, incluyendo todos los datos asociados y relaciones.

### Headers

| Name          | Value                        |
| ------------- | ---------------------------- |
| Authorization | `Bearer {YOUR_ACCESS_TOKEN}` |

### Parámetros de Ruta

| Name | Type   | Required | Description                                    |
| ---- | ------ | -------- | ---------------------------------------------- |
| `id` | string | **Sí**   | El identificador único de la validación biométrica a recuperar. |

### Parámetros de Consulta

### Parámetros de Consulta Básicos

| Parameter    | Type  | Description                        | Example                                    |
| ------------ | ----- | ---------------------------------- | ------------------------------------------ |
| `populates[]` | array | Campos para poblar con datos relacionados. | `populates[]=client&populates[]=project` |

### Parámetros de Consulta Avanzados

| Parameter | Type   | Description                                                                 | Example           |
| --------- | ------ | --------------------------------------------------------------------------- | ----------------- |
| `sort`    | string | Orden de clasificación para resultados. Prefijo con `-` para descendente.                   | `sort=-createdAt` |
| `limit`   | number | Número máximo de resultados a devolver.                                       | `limit=10`        |

### Parámetros de Consulta de Filtro

| Parameter              | Type   | Description              | Example                                    |
| ---------------------- | ------ | ------------------------ | ------------------------------------------ |
| `where_client`         | string | Filtrar por ID de cliente.     | `where_client=507f1f77bcf86cd799439013`    |
| `where_project`        | string | Filtrar por ID de proyecto.    | `where_project=507f1f77bcf86cd799439011`   |
| `where_projectFlow`    | string | Filtrar por ID de flujo de proyecto. | `where_projectFlow=507f1f77bcf86cd799439015` |
| `where_status`         | string | Filtrar por estado de validación. | `where_status=validated`               |
| `where_type`           | string | Filtrar por tipo de validación. | `where_type=login`                     |
| `where_livenessSession` | string | Filtrar por ID de sesión de vivacidad. | `where_livenessSession=674de8df21c72be3cc42b8a7` |

### Características

- **Recuperación por ID**: Obtén una validación biométrica específica usando su ID único
- **Información Completa**: Incluye todos los detalles del proceso de validación biométrica
- **Datos Relacionados**: Información del cliente, proyecto y flujo de proyecto
- **Datos Biométricos**: Imágenes faciales, plantillas biométricas y puntuaciones de calidad
- **Resultados de Verificación**: Detección de vivacidad, coincidencia de identidad y anti-spoofing
- **Configuración**: URLs de redirección, webhooks y configuraciones de seguridad
- **Sesiones de Vivacidad**: Información de sesiones de detección de vivacidad
- **Múltiples Lenguajes**: Soporte para JavaScript, Python, PHP y Swift
- **Manejo de Errores**: Respuestas de error detalladas para diferentes escenarios
