# Validaciones de Documentos

Las Validaciones de Documentos son el sistema central de verificación dentro de Verifik que procesa y valida documentos de identidad de usuarios durante el proceso de registro de aplicaciones. Piensa en ellas como el sistema de control de calidad que asegura que los documentos sean auténticos, legibles y coincidan con la identidad reclamada por el usuario.

## Cómo Funcionan las Validaciones de Documentos

Cada Validación de Documento representa una sesión única de verificación de documentos que procesa documentos de identidad enviados por usuarios. Cuando alguien sube o escanea su ID, pasaporte o licencia, Verifik crea una Validación de Documento para analizar el documento, extraer información y verificar su autenticidad.

Actualmente, las Validaciones de Documentos soportan estos tipos:

* **validation** - Proceso de verificación de documentos estándar
* **login** - Verificación de documentos para flujos de login
* **signup** - Verificación de documentos para registro de nuevos usuarios
* **ocr** - Procesamiento de reconocimiento óptico de caracteres
* **demo** - Propósitos de demostración y prueba

## Qué Rastrean las Validaciones de Documentos

Las Validaciones de Documentos son sistemas de verificación completos que procesan cada aspecto de los documentos de identidad:

### Información del Documento:
* Tipo de documento y categoría (ID, pasaporte, licencia)
* Número de documento y detalles de identificación
* Verificación de país y nacionalidad
* Extracción de información de edad y género

### Proceso de Verificación:
* Método de entrada (cámara, carga de archivo, o no establecido)
* Evaluación de calidad y legibilidad del documento
* Extracción y validación de texto OCR
* Verificación de características de seguridad

### Resultados de Validación:
* Verificación de autenticidad del documento
* Validación de precisión de la información
* Detección de características de seguridad
* Evaluación de riesgo y puntuación

## Endpoints de API

### Crear una Validación de Documento
```http
POST https://api.verifik.co/v2/document-validations
```

### Listar Todas las Validaciones de Documentos
```http
GET https://api.verifik.co/v2/document-validations
```

### Recuperar una Validación de Documento
```http
GET https://api.verifik.co/v2/document-validations/{validationId}
```

### Actualizar una Validación de Documento
```http
PUT https://api.verifik.co/v2/document-validations/{validationId}
```

### Eliminar una Validación de Documento
```http
DELETE https://api.verifik.co/v2/document-validations/{validationId}
```

## Casos de Uso

- **Verificación de Identidad**: Verificar identidad de usuario a través de validación de documentos
- **Cumplimiento**: Cumplir requisitos regulatorios para verificación de documentos
- **Prevención de Fraude**: Detectar documentos fraudulentos o manipulados
- **Extracción de Datos**: Extraer datos estructurados de documentos de identidad
