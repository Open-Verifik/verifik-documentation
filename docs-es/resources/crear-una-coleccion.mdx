# Crear una Colección

Crear una nueva colección para almacenar caras generadas usando los servicios de Detección de Vida de Verifik. Las colecciones son requeridas para la mayoría de servicios biométricos.

## Endpoint

```
POST https://api.verifik.co/v2/face-recognition/collections
```

## Headers

| Nombre        | Valor             |
| ------------- | ----------------- |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`  |

## Parámetros

| Nombre        | Tipo   | Requerido | Descripción                                    |
| ------------- | ------ | --------- | ---------------------------------------------- |
| `name`        | string | Sí        | Nombre de la colección.                       |
| `description` | string | Sí        | Breve descripción de para qué se usará la colección. |

## Casos de Uso

- **Configuración de Reconocimiento Facial**: Crear colecciones para almacenar datos faciales para servicios biométricos
- **Gestión Multi-Proyecto**: Organizar datos faciales por diferentes proyectos o aplicaciones
- **Integración de Servicios**: Preparar colecciones requeridas para servicios de comparación y búsqueda facial
- **Organización de Datos**: Categorizar datos faciales para diferentes casos de uso o grupos de usuarios
- **Desarrollo de API**: Configurar colecciones para pruebas y propósitos de desarrollo

## Características

- ✅ **Código de Colección Único**: Cada colección obtiene un código único para operaciones de API
- ✅ **Asociación de Proyecto**: Las colecciones se vinculan automáticamente a tu proyecto
- ✅ **Gestión de Cliente**: Las colecciones están asociadas con tu cuenta de cliente
- ✅ **Marcas de Tiempo Automáticas**: Las marcas de tiempo de creación y actualización se gestionan automáticamente
- ✅ **Soporte de Eliminación Suave**: Las colecciones pueden marcarse como eliminadas sin eliminación permanente
- ✅ **Múltiples Lenguajes de Programación**: Soporte para JavaScript, Python, Swift y PHP
- ✅ **Manejo Integral de Errores**: Respuestas de error detalladas para varios escenarios
