# Listar Todas las Validaciones Biométricas

## Endpoint

```
GET https://api.verifik.co/v2/biometric-validations
```

Recupera una lista de registros de validación biométrica basados en filtros y parámetros especificados.

### Headers

| Name          | Value                        |
| ------------- | ---------------------------- |
| Authorization | `Bearer {YOUR_ACCESS_TOKEN}` |

### Parámetros de Consulta

### Paginación

| Parameter | Type   | Description                    | Example      |
| --------- | ------ | ------------------------------ | ------------ |
| `page`    | number | Número de página (comienza en 1).     | `page=1`     |
| `perPage` | number | Elementos por página (por defecto: 20).  | `perPage=10` |
| `offset`  | number | Alternativa a página para omitir registros. | `offset=20` |

### Selección de Datos

| Parameter    | Type  | Description                        | Example                                    |
| ------------ | ----- | ---------------------------------- | ------------------------------------------ |
| `populates[]` | array | Campos para poblar con datos relacionados. | `populates[]=client&populates[]=project` |

### Filtrado

| Parameter           | Type   | Description                        | Example                                    |
| ------------------- | ------ | ---------------------------------- | ------------------------------------------ |
| `where_client`      | string | Filtrar por ID de cliente.               | `where_client=507f1f77bcf86cd799439013`    |
| `where_status`      | string | Filtrar por estado de validación.       | `where_status=validated`                   |
| `where_type`        | string | Filtrar por tipo de validación.         | `where_type=login`                        |
| `where_livenessSession` | string | Filtrar por ID de sesión de vivacidad. | `where_livenessSession=674de8df21c72be3cc42b8a7` |
| `in_status`         | array  | Filtrar por múltiples estados.      | `in_status[]=validated&in_status[]=failed` |
| `where>_createdAt`  | string | Filtrar registros creados después de fecha. | `where>_createdAt=2024-12-01`              |

### Ordenamiento

| Parameter | Type   | Description                                                                 | Example           |
| --------- | ------ | --------------------------------------------------------------------------- | ----------------- |
| `sort`    | string | Orden de clasificación (prefijo con `-` para descendente).                               | `sort=-createdAt` |

### Características

- **Listado Completo**: Recupera todas las validaciones biométricas de tu cuenta
- **Paginación Avanzada**: Soporte para paginación con control de página, elementos por página y offset
- **Filtrado Completo**: Filtra por cliente, estado, tipo, sesión de vivacidad y fechas
- **Población de Datos**: Incluye información relacionada como datos del cliente y proyecto
- **Múltiples Estados**: Filtra por estado (nuevo, pendiente, completado, fallido, expirado)
- **Ordenamiento**: Clasificación por fecha de creación y otros campos
- **Datos Biométricos**: Incluye resultados de verificación y puntuaciones de calidad
- **Múltiples Lenguajes**: Soporte para JavaScript, Python, PHP y Swift
- **Información Detallada**: Incluye timestamps, puntuaciones de riesgo y resultados de verificación
