# Listar Todas las Validaciones de Documentos

## Endpoint

```
GET https://api.verifik.co/v2/document-validations
```

Este endpoint te permite recuperar una lista de todas las Validaciones de Documentos dentro de la API de Verifik. Puedes usar esto para obtener una vista detallada de múltiples validaciones de documentos, incluyendo su estado, proyecto asociado, métodos de validación y más.

### Headers

| Name          | Value              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parámetros de Consulta

| Parameter                | Type   | Description                                                                                                                                                                 |
| ------------------------ | ------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `page`                   | number | Especifica el número de página para paginación, comenzando desde 1.                                                                                                                  |
| `perPage`                | number | Define el número de elementos por página para paginación.                                                                                                                        |
| `populates[]`            | string | Pobla el campo especificado, transformando referencias ObjectId en objetos completos. Opciones disponibles: `appRegistration`, `projectFlow` |
| `where_status`           | string | Condición where para filtrar por estado. Opciones: ASSESSING, ACTIVE, FAILED, NEEDS_MANUAL_VERIFICATION, NOT_FOUND, EXPIRED, ACTIVE_BUT_UNVERIFIED |
| `where_documentType`     | string | Filtrar por tipo de documento (ej., "driver_license", "passport", "national_id")                                                                                               |
| `where_validationMethod` | string | Filtrar por método de validación. Opciones: MANUAL, OCR, SCAN_PROMPT, SCAN_STUDIO                                                                                               |
| `where_type`             | string | Filtrar por tipo de validación. Opciones: validation, login, signup, ocr, demo                                                                                                    |
| `where_inputMethod`      | string | Filtrar por método de entrada. Opciones: CAMERA, FILE_UPLOAD, NOT_SET                                                                                                             |

### Características

- **Listado Completo**: Recupera todas las validaciones de documentos de tu cuenta
- **Paginación Avanzada**: Control de página y elementos por página
- **Filtrado Completo**: Filtra por estado, tipo de documento, método de validación y más
- **Población de Datos**: Incluye información relacionada como registros de aplicación y flujos de proyecto
- **Múltiples Estados**: Filtra por diferentes estados de validación
- **Tipos de Documento**: Soporte para licencias de conducir, pasaportes, IDs nacionales
- **Métodos de Validación**: OCR, manual, escaneo y más
- **Múltiples Lenguajes**: Soporte para JavaScript, Python, PHP y Swift
- **Información Detallada**: Incluye datos del documento, resultados de validación y timestamps
