# Validaciones Biométricas

Las Validaciones Biométricas son el sistema central de verificación dentro de Verifik que procesa y valida datos biométricos de usuarios durante el proceso de autenticación y registro. Piensa en ellas como el punto de control de seguridad que asegura que los usuarios son quienes dicen ser verificando su identidad biométrica a través de reconocimiento facial y detección de vivacidad.

## Cómo Funcionan las Validaciones Biométricas

Cada Validación Biométrica representa una sesión única de verificación biométrica que procesa datos biométricos enviados por usuarios. Cuando alguien toma una selfie o proporciona datos biométricos durante el registro, login o verificación de cuenta, Verifik crea una Validación Biométrica para analizar los datos biométricos y confirmar la identidad del usuario.

Actualmente, las Validaciones Biométricas soportan estos tipos:

* **validation** - Proceso de verificación biométrica estándar
* **login** - Verificación biométrica para autenticación de usuario
* **onboarding** - Verificación biométrica para registro de nuevo usuario
* **liveness** - Detección de vivacidad y anti-spoofing

## Qué Rastrean las Validaciones Biométricas

Las Validaciones Biométricas son sistemas de verificación completos que procesan cada aspecto de la verificación biométrica:

### Información Biométrica:
* Reconocimiento facial y coincidencia de identidad
* Detección de vivacidad y anti-spoofing
* Evaluación de calidad y análisis de imagen
* Verificación de identidad del usuario

### Proceso de Verificación:
* Captura y procesamiento de imagen
* Generación de plantilla biométrica
* Coincidencia de identidad y comparación
* Detección de vivacidad y validación

### Características de Seguridad:
* Anti-spoofing y detección de fraude
* Evaluación de calidad y validación
* Coincidencia de plantillas y comparación
* Evaluación de riesgo y puntuación

## Endpoints de API

### Crear una Validación Biométrica
```http
POST https://api.verifik.co/v2/biometric-validations
```

### Listar Todas las Validaciones Biométricas
```http
GET https://api.verifik.co/v2/biometric-validations
```

### Recuperar una Validación Biométrica
```http
GET https://api.verifik.co/v2/biometric-validations/{validationId}
```

## Casos de Uso

- **Autenticación de Usuario**: Verificar identidad a través de verificación biométrica
- **Registro de Cuenta**: Confirmar identidad del usuario durante el registro
- **Prevención de Fraude**: Detectar y prevenir ataques de spoofing biométrico
- **Seguridad**: Mejorar la seguridad de la cuenta a través de verificación biométrica
