# Crear una Validación de Vivacidad de Documento

## Endpoint

```
POST https://api.verifik.co/v2/document-liveness
```

Una Validación de Vivacidad de Documento es el resultado de todas las validaciones realizadas en el documento pasado a través de una imagen base64. Tenemos cuatro tipos de validaciones que realizamos al documento: **Reproducción de Pantalla, Copia Impresa, Sustitución de Retrato, Manipulación Digital.** También puedes decidir almacenar el base64 en nuestro CDN para que puedas recuperar la imagen que probaste.

### Headers

| Name          | Value              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parámetros del Cuerpo

| Name                               | Type    | Description                                                                                    |
| ---------------------------------- | ------- | ---------------------------------------------------------------------------------------------- |
| `image`                            | blob    | El documento a probar en formato base64.                                                        |
| `saveImage`                        | Boolean | Este booleano definirá si guardamos la imagen o no para inspección posterior.                 |
| `validateScreenReplay`             | Boolean | Realizar una validación para verificar si el ataque vino de una reproducción de pantalla                         |
| `validatePrintedCopy`              | Boolean | Realizar una validación para verificar si el ataque vino de una copia impresa                          |
| `validatePortraitSubstitution`    | Boolean | Realizar una validación para verificar si el ataque vino de una sustitución de retrato                 |
| `validateDigitalManipulation`     | Boolean | Realizar una validación para verificar si el ataque vino de una manipulación digital                  |
| `screenReplayCalibration`          | String  | Puedes ajustar la calibración de `SOFT`, `REGULAR` o `HARD`. El valor por defecto es `REGULAR`. |
| `printedCopyCalibration`          | String  | Puedes ajustar la calibración de `SOFT`, `REGULAR` o `HARD`. El valor por defecto es `REGULAR`. |
| `portraitSubstitutionCalibration` | String  | Puedes ajustar la calibración de `SOFT`, `REGULAR` o `HARD`. El valor por defecto es `REGULAR`. |
| `ignoreDocumentCroppedValidation` | Boolean | Ignorará si el documento fue recortado.                                                   |
| `ignoreColourLessValidation`      | Boolean | Ignorará la falta de color en el documento.                                            |

import Foundation

let headers = [
  "Content-Type": "application/json",

  "image": "base64_encoded_document_image",

  "validatePortraitSubstitution": true,
  "validateDigitalManipulation": true,

  "printedCopyCalibration": "REGULAR",
  "portraitSubstitutionCalibration": "REGULAR",
  "ignoreDocumentCroppedValidation": false,
  "ignoreColourLessValidation": false
] as [String : Any]

let postData = try JSONSerialization.data(withJSONObject: parameters, options: [])

let request = NSMutableURLRequest(url: NSURL(string: "https://api.verifik.co/v2/document-liveness")! as URL,
                                        cachePolicy: .useProtocolCachePolicy,
                                    timeoutInterval: 10.0)
request.httpMethod = "POST"
request.allHTTPHeaderFields = headers
request.httpBody = postData as Data

let session = URLSession.shared
let dataTask = session.dataTask(with: request as URLRequest, completionHandler: { (data, response, error) -> Void in
  if (error != nil) {
    print(error as Any)
  } else {
    let httpResponse = response as? HTTPURLResponse
    print(httpResponse)
  }
})

dataTask.resume()

### Características

- **Detección Anti-Spoofing**: Cuatro tipos de validaciones para detectar ataques
- **Reproducción de Pantalla**: Detecta si el documento viene de una pantalla
- **Copia Impresa**: Detecta si el documento es una copia impresa
- **Sustitución de Retrato**: Detecta si la foto fue sustituida
- **Manipulación Digital**: Detecta manipulación digital del documento
- **Calibración Ajustable**: Niveles SOFT, REGULAR y HARD para cada validación
- **Almacenamiento de Imagen**: Opción para guardar imágenes en CDN
- **Puntuaciones de Confianza**: Puntuaciones detalladas para cada validación
- **Múltiples Lenguajes**: Soporte para JavaScript, Python, PHP y Swift
- **Manejo de Errores**: Respuestas de error detalladas para diferentes escenarios
