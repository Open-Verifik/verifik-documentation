# Eliminar un Webhook

## Endpoint

```
DELETE https://api.verifik.co/v2/webhooks/{id}
```

Para eliminar un webhook específico, realiza una solicitud DELETE al endpoint donde {id} debe ser reemplazado con el identificador único del webhook que quieres eliminar.

### Headers

| Name          | Value              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parámetros

| Name | Type   | Required | Description                                                      |
| ---- | ------ | -------- | ---------------------------------------------------------------- |
| `id` | string | Sí       | ID del webhook que quieres eliminar.                       |

### Características

- **Eliminación Permanente**: Eliminar completamente webhooks de tu cuenta
- **Respuesta de Confirmación**: Devuelve los datos del webhook eliminado para confirmación
- **Múltiples Lenguajes de Programación**: Soporte para JavaScript, Python, PHP y Swift
- **Manejo de Errores**: Respuestas 404 apropiadas para webhooks inexistentes
- **Operación Segura**: Verificar ID del webhook antes de la eliminación
