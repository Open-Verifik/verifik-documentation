# Recuperar una Colección

Obtener una colección generada usando su _id específico, no confundir con el parámetro code.

## Endpoint

```
GET https://api.verifik.co/v2/face-recognition/collections/{id}
```

## Headers

| Nombre        | Valor             |
| ------------- | ----------------- |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`  |

## Parámetros

| Nombre | Tipo   | Requerido | Descripción                                    |
| ------ | ------ | --------- | ---------------------------------------------- |
| `id`   | string | Sí        | ID de la colección de la que quieres obtener la información. |

## Casos de Uso

- **Detalles de Colección**: Obtener información detallada sobre una colección específica
- **Integración de Servicios**: Recuperar información de colección para usar con servicios biométricos
- **Validación de Datos**: Verificar que la colección existe y obtener sus propiedades
- **Desarrollo de API**: Probar la recuperación de colección para propósitos de desarrollo
- **Monitoreo**: Verificar el estado específico de la colección y metadatos

## Características

- ✅ **Colección Específica**: Recupera detalles para una sola colección por ID
- ✅ **Información Completa**: Retorna detalles completos de la colección incluyendo código y marcas de tiempo
- ✅ **Asociación de Proyecto**: Muestra a qué proyecto pertenece la colección
- ✅ **Verificación de Cliente**: Asegura que la colección pertenece a tu cuenta de cliente
- ✅ **Múltiples Lenguajes de Programación**: Soporte para JavaScript, Python, Swift y PHP
- ✅ **Manejo Integral de Errores**: Respuestas de error detalladas para varios escenarios
- ✅ **Autenticación Requerida**: Acceso seguro usando autenticación Bearer token
