# Listar Todas las Validaciones de Email

## Endpoint

```
GET https://api.verifik.co/v2/email-validations
```

Este servicio recupera una lista de todas las validaciones de email con filtrado opcional y paginación. Puedes filtrar por proyecto, estado, tipo y otros parámetros para encontrar validaciones de email específicas.

### Headers

| Name          | Value                        |
| ------------- | ---------------------------- |
| Content-Type  | `application/json`           |
| Authorization | `Bearer {YOUR_ACCESS_TOKEN}` |

### Parámetros de Consulta

| Name          | Type    | Description                                                                                    |
| ------------- | ------- | ---------------------------------------------------------------------------------------------- |
| `page`        | number  | Número de página para paginación (por defecto: 1)                                             |
| `limit`       | number  | Número de registros por página (por defecto: 10, máximo: 100)                                |
| `project`     | string  | Filtrar por ID de proyecto                                                                    |
| `status`      | string  | Filtrar por estado: `new`, `sent`, `validated`, `expired`, `failed`                          |
| `type`        | string  | Filtrar por tipo: `validation`, `login`, `onboarding`, `oneTimeLink`                         |
| `email`       | string  | Filtrar por dirección de email                                                                |
| `populates[]` | string  | Array opcional de datos relacionados para incluir. Opciones disponibles: `client`, `project`, `projectFlow`. |

### Características

- **Listado Completo**: Recupera todas las validaciones de email de tu cuenta
- **Paginación**: Soporte para paginación con control de página y límite
- **Filtrado Avanzado**: Filtra por proyecto, estado, tipo y dirección de email
- **Población de Datos**: Incluye información relacionada como datos del cliente y proyecto
- **Múltiples Estados**: Filtra por estado (nuevo, enviado, validado, expirado, fallido)
- **Múltiples Lenguajes**: Soporte para JavaScript, Python, PHP y Swift
- **Información Detallada**: Incluye intentos, límites y timestamps de validación
