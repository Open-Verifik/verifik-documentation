# El Objeto Colección

Una Colección es el lugar donde se almacenan todas las caras generadas usando los servicios de Detección de Vida de Verifik. La mayoría de servicios requieren una Colección para su operación, por lo que es crucial entender cómo generar una Colección y manipularla a nivel de API.

## Atributos

**`name`** - String - Requerido

Nombre que quieres darle a la colección.

**`description`** - String - Requerido

Breve descripción de para qué se va a usar la colección o cualquier cosa que quieras agregar.

**`_id`** - ObjectId - Requerido

Identificador único para la colección.

**`deleted`** - Boolean - Requerido

Si la colección ha sido marcada como eliminada (eliminación suave).

**`project`** - ObjectId - Requerido

Referencia al modelo Project, vinculando la colección a un proyecto específico.

**`client`** - ObjectId - Requerido

Referencia al modelo Client, vinculando la colección a una cuenta de cliente específica.

**`code`** - String - Requerido

Código único para la colección usado en operaciones de API (referido como collection_id cuando se solicita).

**`createdAt`** - Date - Requerido

Marca de tiempo cuando se creó la colección.

**`updatedAt`** - Date - Requerido

Marca de tiempo cuando se actualizó por última vez la colección.

**`__v`** - Number - Requerido

Clave de versión para el objeto colección.

## Características

- **Almacenamiento de Caras**: Almacena caras generadas usando los servicios de Detección de Vida de Verifik
- **Integración de Servicios**: Requerido para la mayoría de servicios biométricos
- **Identificación Única**: Cada colección tiene un código único para operaciones de API
- **Asociación de Proyecto**: Las colecciones están vinculadas a proyectos específicos
- **Gestión de Cliente**: Las colecciones están asociadas con clientes específicos
- **Soporte de Eliminación Suave**: Las colecciones pueden marcarse como eliminadas sin eliminación permanente
- **Marcas de Tiempo Automáticas**: Las marcas de tiempo de creación y actualización se gestionan automáticamente
