# 🇪🇨 Ecuador - Verificación de Vehículo

Este servicio devuelve información de un vehículo registrado en Ecuador a partir de la placa (**`plate`**): marca, modelo, año, tipo, VIN, número de motor, metadatos de revisión y campos de ficha técnica cuando estén disponibles.

### Endpoint

```
GET https://api.verifik.co/v3/ec/vehiculo/placa
```

Consulta un vehículo ecuatoriano por **`plate`** (sin espacios ni puntos). La respuesta puede incluir `brand`, `model`, `year`, `vin`, `engineNumber`, `vehicleClass`, `vehicleType`, `color`, `serviceType`, `revision` y `technicalSheet` (`engine`, `performance`, `dimensions`) cuando estén disponibles. Útil para **verificación**, **seguros**, **flotas** y **cumplimiento**.

### Headers

| Nombre          | Valor              |
| ------------- | ------------------ |
| Accept | `application/json` |
| Authorization | `Bearer <token>`   |

### Parámetros

| Nombre | Tipo | Requerido | Descripción | Ejemplo |
| ---- | ---- | -------- | ----------- | ------- |
| `plate` | string | **Sí** | Placa a consultar, sin espacios ni puntos. | `PBB3239` |

### Solicitud

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from 'axios';

const { data } = await axios.get('https://api.verifik.co/v3/ec/vehiculo/placa', {
  params: { plate: 'PBB3239' },
  headers: {
    Accept: 'application/json',
    Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
  },
});
console.log(data);
```

  <TabItem value="python" label="Python">

```python
import os

url = "https://api.verifik.co/v3/ec/vehiculo/placa"

response = requests.get(url, headers=headers, params=params)
print(response.json())
```

### Notas

- `technicalSheet` puede ser `null` o estar parcialmente poblado cuando no hay datos técnicos coincidentes.
- `revision` puede ser `null` o estar parcialmente poblado según disponibilidad.
