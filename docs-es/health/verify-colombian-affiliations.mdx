# Datos de Verificación RETHUS Colombiano

## Endpoint

```
https://api.verifik.co/v2/co/salud/rethus
```

El servicio de Datos de Verificación RETHUS Colombiano permite verificar información de profesionales de la salud colombianos y sus afiliaciones en el Registro Especial de Prestadores de Servicios de Salud (RETHUS).

### Headers

| Nombre         | Valor              |
| -------------- | ------------------ |
| Content-Type   | `application/json` |
| Authorization | `Bearer <token>`   |

### Parámetros

| Nombre           | Tipo   | Requerido | Descripción                                    |
| ---------------- | ------ | --------- | ---------------------------------------------- |
| `documentType`   | string | Sí        | Tipo de documento. Parámetros válidos: CC, CE. |
| `documentNumber` | string | Sí        | Número de documento del profesional de la salud. |

### Solicitud

<Tabs>
  <TabItem value="javascript" label="JavaScript">

```javascript
import axios from 'axios';

const options = {
  method: 'GET',
  url: 'https://api.verifik.co/v2/co/salud/rethus',
  params: {
    documentType: 'CC',
    documentNumber: '123456789'
  },
  headers: {
    Accept: 'application/json',
    Authorization: 'jwt <tu_token>'
  }
};

try {
  const { data } = await axios.request(options);
  console.log(data);
} catch (error) {
  console.error(error);
}
```
