# Health Verification

Health verification services for Colombian healthcare professionals and affiliations.

## Available Services

- **[Verify Colombian Affiliations](/health/verify-colombian-affiliations)** — EPS, ARL, pension, and social security affiliations (`GET /v2/co/afiliaciones`; canonical doc under Legal)

## Features

- **Healthcare Professional Verification**: Verify data of Colombian healthcare professionals
- **Affiliation Verification**: Verify professional affiliations and registrations
- **Real-time Data**: Access current and up-to-date information from official RETHUS sources

For detailed API documentation and integration guides, please refer to the specific service documentation.
