# Canadá - Licencia de Conducir British Columbia

## Endpoint

```
https://api.verifik.co/v2/ca/british-columbia/driver-license
```

El servicio de Verificación de Licencia de Conducir de British Columbia ofrece un método directo para verificar la autenticidad y validez de una licencia de conducir en la provincia de British Columbia, Canadá. Utilizando este servicio, puedes validar rápidamente una licencia de conducir basándose en el número de documento y apellido proporcionados.

## Headers

| Nombre          | Valor              |
| --------------- | ------------------ |
| Content-Type    | `application/json` |
| Authorization   | `Bearer <token>`   |

## Parameters

| Nombre           | Tipo    | Requerido | Descripción                                    |
| ---------------- | ------- | --------- | ---------------------------------------------- |
| `documentNumber` | string  | Sí        | Licencia de conducir a consultar, todos los datos deben ingresarse exactamente como aparecen en este documento. |
| `lastName`       | string  | Sí        | Apellido que aparece en la Licencia de Conducir. |

## Use Cases

- **Verificación de Licencias**: Verificar la autenticidad y validez de licencias de conducir de British Columbia
- **Servicios de Seguros**: Validar licencias para procesos de aseguramiento
- **Agencias de Tránsito**: Consultar información oficial de licencias

## Features

- ✅ Verificación de licencia por número de documento y apellido
- ✅ Parámetros simples (solo número de documento y apellido)
- ✅ Soporte para múltiples lenguajes de programación
- ✅ Manejo completo de errores con respuestas detalladas
- ✅ Validación en tiempo real basada en registros oficiales
- ✅ Respuesta certificada por Verifik.co
- ✅ Validación de longitud máxima del número de documento
