# 🇵🇪 Perú - Licencia de Conducir

Este servicio te permite obtener información detallada sobre la licencia de conducir de una persona en Perú usando su **DNI** (*Documento Nacional de Identidad*). La respuesta incluye licencias vigentes, historial de trámites, cursos normativos, restricciones, sanciones y otros datos—útil para **flotas**, **RR.HH.** y **KYC** cuando importa la habilitación para conducir.

### Endpoint

```
GET https://api.verifik.co/v2/pe/driver-license
```

Consulta la licencia con `documentType=DNI` y el `documentNumber` de 8 dígitos. Los campos devueltos incluyen **nombre completo**, **categorías de licencia** (A, AIV, B), **licencias**, **trámites** (`procedures`), **cursos normativos**, **sanciones**, **papeletas** y más, según disponibilidad en el registro.

### Headers

| Name          | Value              |
| ------------- | ------------------ |
| Accept        | `application/json` |
| Authorization | `Bearer <token>`   |

### Parámetros

| Name | Type | Required | Description | Example |
| ---- | ---- | -------- | ----------- | ------- |
| `documentType` | string | **Sí** | Tipo de documento. Valor permitido: `DNI`. | `DNI` |
| `documentNumber` | string | **Sí** | DNI de la persona a consultar, sin espacios ni puntos. | `44141274` |

### Notas

- `documentNumber` debe corresponder a un DNI con registros de licencia de conducir en la fuente oficial.

## Características

- **Consulta por DNI**: Parámetros `documentType=DNI` y número de documento.
- **Categorías de licencia**: Indicadores y listas para categorías A, AIV, B y permisos provisionales.
- **Licencias vigentes**: Número, categoría, estado, restricciones, centro de emisión y vencimiento.
- **Historial de trámites**: Revalidaciones y fechas con estado (p. ej. Vigente, VENCIDA).
- **Cursos normativos**: Detalle de inscripción y finalización cuando aplique.
- **Sanciones y papeletas**: Arreglos opcionales según el registro.
- **Respuestas estructuradas**: JSON con firma y certificación de Verifik.
