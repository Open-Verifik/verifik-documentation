# Alquilar Etiqueta

Alquilar una etiqueta para un nombre Zelf.

```
POST /api/tags/lease
```

## Descripción

Este endpoint te permite alquilar una etiqueta para tu nombre Zelf, creando una vinculación entre tu identidad y la etiqueta elegida.
