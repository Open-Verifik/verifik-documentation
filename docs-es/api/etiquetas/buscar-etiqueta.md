# Buscar Etiqueta

Buscar etiquetas disponibles en el Zelf Name Service.

## Endpoint

```
GET /api/tags/search
```

## Descripción

Este endpoint te permite buscar etiquetas disponibles que pueden ser utilizadas para nombres Zelf.
