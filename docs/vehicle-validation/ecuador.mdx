# ðŸ‡ªðŸ‡¨ Ecuador - Vehicle Fines

The Ecuador Vehicle License Plate Fines service allows you to retrieve information about fines associated with a vehicle's license plate in Ecuador. You can access details such as the vehicle's registration information, including the plate number, class, and model. The service also provides the current status, service type, and registration expiration date, along with any outstanding fines linked to the license plate.

This service is useful for individuals and businesses aiming to stay informed about fines associated with their vehicles and ensure compliance with Ecuadorian traffic laws.

## Endpoint

```
GET https://api.verifik.co/v2/ec/vehiculo/placa/multas
```

### Headers

| Name          | Value              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parameters

| Name | Type | Required | Description | Example |
| ---- | ---- | -------- | ----------- | ------- |
| `plate` | string | **Yes** | Plate number to consult, without spaces or points. | `ABC1234` |

## Features

- **Vehicle Fines Information**: Retrieve comprehensive information about fines associated with Ecuadorian vehicle license plates
- **Vehicle Registration Details**: Access vehicle registration information including plate number, class, and model
- **Vehicle Status Information**: Get current vehicle status, service type, and registration expiration date
- **Outstanding Fines**: View all outstanding fines linked to the license plate
- **Fine Details**: Access detailed fine information including:
  - Infraction numbers and entity information
  - Citation numbers and documentation
  - Issue dates, notification dates, and payment deadlines
  - Penalty points and payment status
  - Sanction amounts, fines, and total amounts to pay
  - Legal article references and violation descriptions
- **Vehicle Specifications**: Get cylinder capacity, manufacture country, model, and year
- **Registration Information**: Access registration canton, last registration date, and expiration dates
- **Payment History**: View last payment year and check-up dates
- **Compliance Monitoring**: Stay informed about fines and ensure compliance with Ecuadorian traffic laws
- **Multiple Programming Languages**: Support for JavaScript, Python, PHP, and Swift
- **Real-time Data**: Access current and up-to-date fine information
- **Comprehensive Error Handling**: Detailed error responses for various scenarios
- **Ecuadorian Registry Integration**: Direct access to official Ecuadorian vehicle fines database
