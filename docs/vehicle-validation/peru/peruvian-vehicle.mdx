# Peru - Vehicle Information

The Peruvian Vehicle Information service returns registered vehicle details for a license plate (**`plate`**): make, model, year, engine and chassis identifiers, status, owners when available, and optional technical-sheet fields.

### Endpoint

```
GET https://api.verifik.co/v3/pe/vehiculo/placa
```

Query a Peruvian vehicle by **`plate`** (no spaces or periods). The response may include `brand`, `model`, `year`, `engineSerial`, `chasisSerial`, `vin`, `serial`, `color`, `status`, `owners`, and `technicalSheet` (`engine`, `performance`, `dimensions`) when available. Use for **verification**, **insurance**, **fleet**, and **compliance** workflows.

### Headers

| Name          | Value              |
| ------------- | ------------------ |
| Accept | `application/json` |
| Authorization | `Bearer <token>`   |

### Parameters

| Name    | Type   | Required | Description | Example |
| ------- | ------ | -------- | ----------- | ------- |
| `plate` | string | **Yes**  | Plate number to consult, without spaces or points. | `ABC123` |

### Request

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from 'axios';

const { data } = await axios.get('https://api.verifik.co/v3/pe/vehiculo/placa', {
  params: { plate: 'ABC123' },
  headers: {
    Accept: 'application/json',
    Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
  },
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os
import requests

url = "https://api.verifik.co/v3/pe/vehiculo/placa"
headers = {"Accept": "application/json", "Authorization": f"Bearer {os.getenv('VERIFIK_TOKEN')}"}
params = {"plate": "ABC123"}
response = requests.get(url, headers=headers, params=params)
print(response.json())
```

### Notes

- `technicalSheet` can be `null` or partially populated when matching technical data is unavailable.
- Sandbox plates such as `ABC10001`–`ABC10010` return stable demo profiles.
