# 🇨🇱 Chile - Transport vehicle

Consults public transport vehicle registration data in Chile (service type, capacity, operator, folio, and service status). Use this endpoint when you need taxi or urban public service details linked to a license plate, in addition to the [general vehicle registry](..).

### Endpoint

```
GET https://api.verifik.co/v2/cl/transport-vehicle
```

### Headers

| Name          | Value              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parameters

| Name | Type | Required | Description | Example |
| ---- | ---- | -------- | ----------- | ------- |
| `plate` | string | **Yes** | Plate to consult, without spaces or points. | `CXTB61` |

### Response fields (`data`)

| Field | Description |
| ----- | ----------- |
| `plate` | License plate consulted |
| `reportGeneratedAt` | Timestamp text of the generated report |
| `rntEntryDate` | Transport registry entry date |
| `serviceType` | Authorized public service type |
| `capacity` | Passenger capacity |
| `vehicleStatus` | Vehicle registration status in the transport registry |
| `vehicleCertificateExpiryDate` | Vehicle certificate validity end date |
| `region` | Administrative region code |
| `manufactureYear` | Vehicle manufacture year |
| `seatBeltsMandatory` | Whether seat belts are mandatory (`SI` / `NO`) |
| `vehicleAgeYears` | Vehicle age in years |
| `brand` | Brand |
| `model` | Model |
| `serviceFolio` | Service folio number |
| `serviceFleetNumber` | Fleet number |
| `serviceResponsibleName` | Legal name of the responsible transport operator |
| `serviceStatus` | Service authorization status |
| `serviceExpiryDate` | Service authorization expiry date |
| `replacementInfo` | Replacement or additional notes (may be empty) |
