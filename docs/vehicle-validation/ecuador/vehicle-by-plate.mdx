# 🇪🇨 Ecuador - Vehicle Verification

This service returns vehicle information registered in Ecuador using the license plate (**`plate`**): make, model, year, type, VIN, engine number, revision metadata, and technical-sheet fields when available.

### Endpoint

```
GET https://api.verifik.co/v3/ec/vehiculo/placa
```

Query an Ecuadorian vehicle by **`plate`** (no spaces or periods). The response may include `brand`, `model`, `year`, `vin`, `engineNumber`, `vehicleClass`, `vehicleType`, `color`, `serviceType`, `revision`, and `technicalSheet` (`engine`, `performance`, `dimensions`) when available. Use for **verification**, **insurance**, **fleet**, and **compliance** workflows.

### Headers

| Name          | Value              |
| ------------- | ------------------ |
| Accept | `application/json` |
| Authorization | `Bearer <token>`   |

### Parameters

| Name | Type | Required | Description | Example |
| ---- | ---- | -------- | ----------- | ------- |
| `plate` | string | **Yes** | License plate to query, without spaces or periods. | `PBB3239` |

### Request

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from 'axios';

const { data } = await axios.get('https://api.verifik.co/v3/ec/vehiculo/placa', {
  params: { plate: 'PBB3239' },
  headers: {
    Accept: 'application/json',
    Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
  },
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os
import requests

url = "https://api.verifik.co/v3/ec/vehiculo/placa"
headers = {"Accept": "application/json", "Authorization": f"Bearer {os.getenv('VERIFIK_TOKEN')}"}
params = {"plate": "PBB3239"}
response = requests.get(url, headers=headers, params=params)
print(response.json())
```
