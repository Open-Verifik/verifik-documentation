# The Project Flow Object / El Objeto de Flujo de Proyecto

### Overview / Resumen

The Project Flow object represents flow configurations for projects in Verifik. Project Flows define various data points that Verifik will use to perform validations using passwordless and liveness detection technologies.

El objeto Project Flow representa configuraciones de flujo para proyectos en Verifik. Los Project Flows definen varios puntos de datos que Verifik utilizará para realizar validaciones usando tecnologías sin contraseña y detección de vivacidad.

### Attributes / Atributos

#### `project`
**Type / Tipo:** ObjectId  
**Required / Requerido:** Yes / Sí

The project is the `_id` generated when creating a project correctly.

El proyecto es el `_id` generado al crear un proyecto correctamente.

#### `type`
**Type / Tipo:** String  
**Required / Requerido:** Yes / Sí

The type determines what type of service you want to generate in this project flow. Can be:
* `"login"` - For existing users to access your system
* `"onboarding"` - For new users to enroll and create accounts

El tipo determina qué tipo de servicio desea generar en este flujo de proyecto. Puede ser:
* `"login"` - Para usuarios existentes para acceder a su sistema
* `"onboarding"` - Para nuevos usuarios para inscribirse y crear cuentas

#### `status`
**Type / Tipo:** String  
**Required / Requerido:** Yes / Sí

The current status of the project flow. Can be:
* `"draft"` - Project flow is in draft mode
* `"active"` - Project flow is active and can be used
* `"paused"` - Project flow is paused

El estado actual del flujo de proyecto. Puede ser:
* `"draft"` - El flujo de proyecto está en modo borrador
* `"active"` - El flujo de proyecto está activo y puede ser usado
* `"paused"` - El flujo de proyecto está pausado

Only "active" projects can be used.

Solo los proyectos "activos" pueden ser usados.

#### `version`
**Type / Tipo:** Number  
**Required / Requerido:** No

Version number of the project flow. Defaults to 1.

Número de versión del flujo de proyecto. Por defecto es 1.

#### `name`
**Type / Tipo:** String  
**Required / Requerido:** Yes / Sí

Name of the project flow for identification purposes.

Nombre del flujo de proyecto para fines de identificación.

#### `description`
**Type / Tipo:** String  
**Required / Requerido:** No

Description of the project flow and its purpose.

Descripción del flujo de proyecto y su propósito.
