# List All Credit Records

## Endpoint

```
https://api.verifik.co/v2/credits
```

Retrieve a list of Credit records in Verifik's system. This endpoint returns an array of credit objects, each containing detailed information about credit transactions, balances, and associated client accounts.

### Headers

| Name          | Value              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parameters

| Parameter     | Type   | Required | Description                                                                                    |
| ------------- | ------ | -------- | ---------------------------------------------------------------------------------------------- |
| `page`        | number | No       | Specifies the page number for pagination, starting from 1.                                    |
| `perPage`     | number | No       | Defines the number of items per page for pagination.                                          |
| `populates[]` | string | No       | Populates the specified field, transforming ObjectId references into full objects. Available options: `client`, `superAdmin` |
| `where_status` | string | No       | Where condition to filter by status. Options: `approved`, `pending`, `failed`, `postPaid` |
| `where_category` | string | No       | Where condition to filter by category. Options: `purchase`, `usage` |
| `where_client` | string | No       | Filter by specific client ID |
