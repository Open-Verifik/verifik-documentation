# List All Collections

Fetch all collections generated by a specific client. The only parameter required is to send the access token. With this, Verifik will identify the collections generated by this user.

## Endpoint

```
GET https://api.verifik.co/v2/face-recognition/collections
```

## Headers

| Name          | Value              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

## Use Cases

- **Collection Management**: View all collections associated with your client account
- **Project Organization**: See collections organized by different projects
- **Data Overview**: Get an overview of all face data storage collections
- **API Integration**: List collections for integration with other biometric services
- **Monitoring**: Monitor collection usage and organization across projects

## Features

- ✅ **Client-Specific**: Returns only collections belonging to your client account
- ✅ **Project Association**: Shows which project each collection belongs to
- ✅ **Complete Information**: Returns full collection details including codes and timestamps
- ✅ **Array Format**: Returns collections in an easy-to-process array format
- ✅ **Multiple Programming Languages**: Support for JavaScript, Python, Swift, and PHP
- ✅ **Comprehensive Error Handling**: Detailed error responses for various scenarios
- ✅ **Authentication Required**: Secure access using Bearer token authentication
