### Endpoint

```
https://api.verifik.co/v2/br/company
```

Retrieve Brazilian company information by CNPJ.

### Headers

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Params

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `documentType` | string | Yes | Allowed: `CNPJ`. |
| `documentNumber` | string | Yes | 14-digit CNPJ, no separators. |

import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/br/company", {

<?php
$ch = curl_init("https://api.verifik.co/v2/br/company");

    "Authorization: Bearer " . getenv("VERIFIK_TOKEN")
