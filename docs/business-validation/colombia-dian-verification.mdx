### Endpoint

```
https://api.verifik.co/v2/co/dian/verification
```

Validates a company's DIAN (Colombia tax authority) status.

### Headers

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Params

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `nit` | string | Yes | NIT (with or without verification digit). |

### Request

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/dian/verification", {
  params: { nit: "900123456" },
  headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os, requests

url = "https://api.verifik.co/v2/co/dian/verification"
headers = {"Accept": "application/json", "Authorization": f"Bearer {os.getenv('VERIFIK_TOKEN')}"}
params = {"nit": "900123456"}
r = requests.get(url, headers=headers, params=params)
print(r.json())
```
