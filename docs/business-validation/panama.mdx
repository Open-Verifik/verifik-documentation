### Endpoint

```
https://api.verifik.co/v2/pa/company
```

Retrieve Panamanian company data by identifier.

### Headers

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Params

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `documentType` | string | Yes | Document type. Allowed parameter: RUC. |
| `documentNumber` | string | Yes | Company identifier. |
| `dv` | string | Yes | Verification digit. |
