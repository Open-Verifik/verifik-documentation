# 🇵🇪 Peru - Driver License

This service lets you retrieve detailed information about a Peruvian driver's license using the holder's **DNI** (*Documento Nacional de Identidad*). The response includes active licenses, procedure history, regulatory courses, restrictions, sanctions, and related flags—useful for **fleet**, **workforce**, and **KYC** checks where driving credentials matter.

### Endpoint

```
GET https://api.verifik.co/v2/pe/driver-license
```

Look up a Peruvian driver license with `documentType=DNI` and the 8-digit `documentNumber`. Returned fields include **full name**, **license categories** (A, AIV, B), **licenses**, **procedures**, **regulatory courses**, **sanctions**, **tickets**, and more, subject to registry availability.

### Headers

| Name          | Value              |
| ------------- | ------------------ |
| Accept        | `application/json` |
| Authorization | `Bearer <token>`   |

### Parameters

| Name | Type | Required | Description | Example |
| ---- | ---- | -------- | ----------- | ------- |
| `documentType` | string | **Yes** | Document type. Allowed value: `DNI`. | `DNI` |
| `documentNumber` | string | **Yes** | DNI of the person to query, without spaces or dots. | `44141274` |

### Notes

- `documentNumber` must match an existing DNI with driver-license records in the official registry.

## Features

- **DNI lookup**: Query by Peruvian national ID (`documentType=DNI`).
- **License categories**: Flags and lists for categories A, AIV, B, and provisional permits.
- **Active licenses**: Number, category, status, restrictions, issuance center, and expiration.
- **Procedure history**: Revalidations and related dates with status (e.g. Vigente, VENCIDA).
- **Regulatory courses**: Enrollment and completion details when available.
- **Sanctions & tickets**: Optional arrays when the registry reports them.
- **Structured responses**: JSON with Verifik signature and certification metadata.
