# Health

## Colombia — API coverage

- **[Colombia — RETHUS (health professionals registry)](/legal/colombia-rethus)** — `GET /v2/co/cedula/rethus` using **document type** (CC, CE, PPT) and **document number**.

- **[Colombia — Health & social security affiliations](/legal/verify-colombian-affiliations)** — `GET /v2/co/afiliaciones` (EPS, ARL, pension, compensation funds, severance) using **document type**, **document number**, and **issue date** (`DD/MM/YYYY`).

## Use cases

- **Healthcare verification** and **credentialing** (RETHUS)
- **Employment** and **benefits** checks (affiliations / RUAF)
