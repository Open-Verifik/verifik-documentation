### Endpoint

```
https://api.verifik.co/v2/ocr/scan-studio
```

Scan Studio is our advanced OCR API that uses trained models to achieve high accuracy text extraction from various document types. This service provides specialized OCR capabilities with pre-trained models for specific document types, ensuring high accuracy and reliable text extraction.

### Headers

| Name | Value |
| --- | --- |
| Content-Type | `application/json` |
| Authorization | `Bearer <token>` |

### Parameters

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `image` | string | Yes | Image in Base64 encoded format or a URL where the image is hosted. |
| `model` | string | Yes | Trained model to use for extraction. |
| `cropFace` | boolean | No | If you want to crop the face out of the document, pass this as `true`. |

### Request

import axios from "axios";

const { data } = await axios.post("https://api.verifik.co/v2/ocr/scan-studio", {

  model: "id_documents",
  cropFace: true
}, {
  headers: { 
    "Content-Type": "application/json", 
    Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` 
  },
});

import os, requests

### Available Models

- **`id_documents`**: Government-issued identification cards
- **`drivers_licenses`**: Various country driver's licenses
- **`passports`**: International passport documents
- **`business_documents`**: Invoices, contracts, and business forms
- **`custom`**: Custom trained models available upon request

### Features

- **High Accuracy**: Trained models provide superior accuracy (95%+)
- **Structured Data**: Returns organized field data
- **Multiple Languages**: Support for various languages
- **Fast Processing**: Optimized models for quick extraction
- **Custom Models**: Available upon request for specific use cases

### Notes

- Use Base64 encoded images or direct URLs for the `image` parameter.
- Specify the appropriate `model` for best results.
- The `cropFace` parameter helps improve accuracy by focusing on document content.
