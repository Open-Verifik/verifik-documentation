# Verifik Biometrics APIs

Comprehensive biometric verification APIs for face recognition, liveness detection, and identity comparison.

## Available Services

### Face Recognition

- **[Face Search (1:N)](/verifik-biometrics-apis/search/search-1-n-or-crop)** - Search for a face in a collection (1:N identification) or search using cropped images
- **[Face Comparison](/verifik-biometrics-apis/compare)** - 1:1 face comparison to verify identity
- **[Compare Biometrics](/verifik-biometrics-apis/compare-biometrics)** - Compare biometric data for identity verification

### Liveness Detection

- **[Liveness Detection](/verifik-biometrics-apis/liveness-detection)** - Detect if a face is live or from a static image/video
- **[Face Detection](/verifik-biometrics-apis/liveness/face-detection)** - Detect and extract faces from images

## Features

- **Real-time Processing**: Fast and accurate biometric verification
- **Multiple Modes**: Support for both FAST and ACCURATE search modes
- **Spoof Detection**: Advanced liveness detection to prevent fraud
- **Scalable**: Handle large collections and high-volume requests
- **Secure**: Industry-standard authentication and data protection

## Integration

All APIs require authentication via Bearer token. See the [Authentication](/authentication/authentication) documentation for details.

For detailed API documentation and integration guides, please refer to the specific endpoint documentation.
