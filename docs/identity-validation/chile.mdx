# Chile Identity Validation

Identity validation services for Chile, including citizen verification using RUN (Rol Único Nacional).

## Available Services

- **[Chilean Citizen Information](/identity-validation/chile/chilean-citizen-information)** - Verify Chilean citizens using RUN
- **[Validate Chilean Document](/identity-validation/chile/validate-chilean-document)** - Validate Chilean identity documents

## Features

- **Citizen Verification**: Verify Chilean citizens using RUN (Rol Único Nacional)
- **Document Validation**: Validate identity documents issued in Chile
- **Real-time Validation**: Access current and up-to-date information from official Registro Civil sources

For detailed API documentation and integration guides, please refer to the specific service documentation.
