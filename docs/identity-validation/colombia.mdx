# Colombia Identity Validation

Comprehensive identity validation services for Colombia, including citizen verification, Politically Exposed Persons (PEP) checks, and special permits for foreign residents.

## Available Services

- **[Colombian Citizen](/identity-validation/colombia/colombian-citizen)** - Verify Colombian citizens using Cédula de Ciudadanía
- **[Colombian Citizen with Extra Data](/identity-validation/colombia/colombian-citizen-with-extra-data)** - Enhanced Colombian citizen verification with extended data
- **[Politically Exposed Persons (PEP)](/identity-validation/colombia/colombian-politically-exposed-persons)** - Check if a person or business is listed as a PEP
- **[Special Stay Permit (PEP)](/identity-validation/colombia/special-stay-permit-pep)** - Verify Special Stay Permit for Venezuelan citizens in Colombia

## Features

- **Citizen Verification**: Verify Colombian citizens using CC (Cédula de Ciudadanía) and PPT (Permiso por Protección Temporal)
- **PEP Checks**: Identify Politically Exposed Persons for AML compliance
- **Foreign Resident Verification**: Verify special permits for Venezuelan citizens
- **Real-time Validation**: Access current and up-to-date information from official government sources

For detailed API documentation and integration guides, please refer to the specific service documentation.
