# Canada Identity Validation

Identity validation services for Canada, including driver license verification for Ontario.

## Currently Supported Provinces

Driver license validation is currently available for:

- **[Ontario Driver License](/driver-validation/ontario-driver-license)** - Verify Ontario driver licenses

## Features

- **Driver License Verification**: Validate driver licenses issued in supported Canadian provinces
- **Real-time Validation**: Access current and up-to-date information from official provincial licensing systems

For detailed API documentation and integration guides, please refer to the specific service documentation or visit the [Canada Identity Verification](/identity/canada) section.
