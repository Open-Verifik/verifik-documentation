# Peru Identity Validation

Comprehensive identity validation services for Peru, including citizen verification, foreigner resident verification, and extended data verification.

## Available Services

- **[Peru Citizen](/identity-validation/peru/peru-citizen)** - Verify identities of Peruvian citizens using DNI v3 API
- **[Peruvian Foreigner Resident](/identity-validation/peru/peruvian-foreigner-resident)** - Verify foreigner resident identity documents
- **[V3 Peruvian Citizen](/identity-validation/peru/v3-peruvian-citizen)** - Extended identity verification with comprehensive DNI details

## Features

- **Citizen Verification**: Verify Peruvian citizens using DNI (Documento Nacional de Identidad)
- **Foreigner Resident Verification**: Verify foreigner identity documents and immigration status
- **Extended Data**: Access comprehensive identity information including photo, address, and demographic data
- **Real-time Validation**: Access current and up-to-date information from official RENIEC sources

For detailed API documentation and integration guides, please refer to the specific service documentation.
