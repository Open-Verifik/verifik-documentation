# Quebec Driver License Validation

:::info Service Availability
Quebec driver license validation is currently not available through Verifik's API.

We currently support driver license validation for the following Canadian provinces:

- **[British Columbia Driver License](/driver-validation/british-columbia-driver-license)** - Verify BC driver licenses
- **[Ontario Driver License](/driver-validation/ontario-driver-license)** - Verify Ontario driver licenses
:::

## Available Canadian Driver License Services

For Canadian driver license validation, please use:

- **[British Columbia Driver License](/driver-validation/british-columbia-driver-license)** - Verify driver licenses issued in British Columbia
- **[Ontario Driver License](/driver-validation/ontario-driver-license)** - Verify driver licenses issued in Ontario
