# Colombia — National ID premium lookup (CC)

Premium endpoint for **Colombian *Cédula de Ciudadanía* (CC)**. The service **resolves the expedition date** automatically, then runs the **same enriched citizen lookup** as [**cédula extra**](/identity/colombia-full-id). You **do not** send `documentType` or `date`—only **`documentNumber`**.

Compared to [**/v2/co/cedula**](/identity-validation/colombia/colombian-citizen), this returns **full extra fields** (birth date, expedition, name parts, etc.) at a **higher credit cost**.

## API reference

### Endpoint

```
GET https://api.verifik.co/v2/co/cedula/premium
```

Also: `GET https://verifik.app/v2/co/cedula/premium`.

**POST** is supported with the **same parameters** in the JSON or form body.

### Headers

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parameters

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `documentNumber` | string | Yes | CC number, **digits only** after normalization (non-digits stripped). **5–10** characters (API validation). |

### Notes

- **CC only** — there is no `documentType` on this route; the flow always treats the number as **CC** (see controller).
- Prefer **digits only** in `documentNumber` (spaces and separators are stripped server-side).
- For **PPT**, **CE**, or other types, use the appropriate dedicated endpoints—not this one.

### Request

<Tabs>
  <TabItem value="get" label="GET">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/cedula/premium", {
	params: { documentNumber: "1234567890" },
	headers: {
		Accept: "application/json",
		Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
	},
});
console.log(data);
```

  </TabItem>
  <TabItem value="post" label="POST">

```javascript
import axios from "axios";

const { data } = await axios.post(
	"https://api.verifik.co/v2/co/cedula/premium",
	{ documentNumber: "1234567890" },
	{
		headers: {
			Accept: "application/json",
			Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
		},
	}
);
console.log(data);
```

  </TabItem>
