### Endpoint

```
https://api.verifik.co/v2/pa/cedula
```

Validate identity for Panamanian citizens and permanent residents via Cédula de Identidad Personal.

### Headers

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Params

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `documentType` | string | Yes | Allowed: `CCPA`. |
| `documentNumber` | string | Yes | Cédula number. |
| `dateOfBirth` | string | Yes | `DD/MM/YYYY`. |

package main

import (
