# Bolívia - SOAT de Veículo

Verifik consulta o registro oficial de **SOAT** (*Seguro Obligatorio de Accidentes de Tránsito*) na Bolívia por **placa de matrícula**. Use este endpoint para **conformidade de frota**, **verificação de seguros** e **controles de trânsito**.

Devolve detalhes de cobertura SOAT quando a fonte os fornece:

- **`plate`** — placa de matrícula (**5–8** caracteres), normalizada em maiúsculas
- **`managementYear`** — ano de gestão ou cobertura SOAT
- **`validFrom`** / **`validTo`** — datas de validade da cobertura (DD/MM/AAAA)
- **`vehicleType`** — categoria do veículo (p. ex. AUTOMOVIL, VAGONETA, MOTOCICLETA)
- **`useType`** — classificação de uso (PARTICULAR, COMERCIAL)
- **`department`** — departamento associado à apólice
- **`status`** — status calculado: VIGENTE, VENCIDO ou PENDIENTE
- **`force`** — opcional; `true` ignora o cache e atualiza da fonte
- Resposta **assinada** pela Verifik

### Endpoint

```
GET v2/bo/vehicle-soat
```

### Headers

| Header | Valor | Descrição |
| --- | --- | --- |
| Accept | application/json | Formato de resposta |
| Authorization | Bearer \<token\> | Seu JWT da Verifik |

### Parâmetros

| Campo | Tipo | Obrigatório | Descrição |
| --- | --- | --- | --- |
| plate | String | Sim | Placa de matrícula sem espaços (**5–8** caracteres). |
| force | String | Não | Defina como `true` para ignorar o cache e buscar dados atualizados. |

### Notas

- **`plate`** é obrigatória; o middleware a normaliza em maiúsculas.
- **`status`** é calculado a partir de **`validFrom`** e **`validTo`** em relação à data atual.
- Use **`force=true`** quando precisar ignorar registros em cache no MongoDB.
- Trate as respostas como **dados sensíveis do veículo**; cumpra a legislação de privacidade aplicável.
