# Verificação de sanções da ONU

## Referência da API

### Endpoint

```
https://api.verifik.co/v2/onu
```

Consulte a lista de sanções das Nações Unidas para verificar se uma pessoa ou entidade está listada usando `documentType` e `documentNumber`, ou `fullName`. Use os resultados em fluxos globais de conformidade, KYC e AML para triagem de contrapartes.

**Observação:** as datas (data de nascimento ou data de expedição) devem estar no formato `dd/mm/aaaa`.

### Cabeçalhos

| Nome          | Valor              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parâmetros

:::info
Para a lista completa de **tipos de documento** suportados e seus parâmetros obrigatórios específicos, consulte o guia [Tipos de documento](/background-check/document-types).
:::

| Nome             | Tipo   | Obrigatório | Descrição                                                                                                  |
| ---------------- | ------ | ----------- | ---------------------------------------------------------------------------------------------------------- |
| `documentType`   | string | Não         | Tipo de documento que se deseja consultar.                                                                 |
| `documentNumber` | string | Não         | Número do documento a ser consultado, sem espaços ou pontos.                                               |
| `fullName`       | string | Não         | Em vez de documentType e documentNumber, é possível enviar diretamente o nome da pessoa/empresa.           |

### Casos de uso

- **Conformidade internacional:** verificar pessoas e entidades contra sanções da ONU antes de onboarding ou transações.
- **Avaliação de risco:** reduzir exposição financeira e reputacional a partes sancionadas.
- **Programas regulatórios:** apoiar requisitos de KYC/AML em bancos, finanças e seguros.
