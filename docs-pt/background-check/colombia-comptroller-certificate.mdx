# Certificado da Contraloría

## Referência da API

### Endpoint

```
https://api.verifik.co/v2/co/contraloria/certificado
```

O serviço permite verificar o certificado de boa conduta emitido pela Contraloría General da Colômbia informando o tipo e o número do documento, e retorna a data da consulta juntamente com o certificado em PDF codificado em base64. É ideal para confirmar rapidamente a situação de pessoas ou empresas colombianas.

### Cabeçalhos

| Nome          | Valor              |
| ------------- | ------------------ |
| Accept        | `application/json` |
| Authorization | `Bearer <token>`   |

### Parâmetros

:::info
Para a lista completa de **tipos de documento** suportados e parâmetros obrigatórios, consulte o guia [Tipos de documento](/background-check/document-types).
:::

| Nome             | Tipo   | Obrigatório | Descrição |
| ---------------- | ------ | ----------- | --------- |
| `documentType`   | string | Sim         | CC, CE, PA ou PEP. |
| `documentNumber` | string | Sim         | Sem espaços ou pontos. |
