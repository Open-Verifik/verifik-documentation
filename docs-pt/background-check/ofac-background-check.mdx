# Verificação de sanções OFAC

## Referência da API

### Endpoint

```
https://api.verifik.co/v2/ofac
```

Verifique se uma pessoa ou entidade aparece na lista de Specially Designated Nationals (SDN) e Blocked Persons da OFAC (Office of Foreign Assets Control) do Tesouro dos EUA usando `documentType` com `documentNumber`, ou `fullName`. Use os resultados em conformidade com sanções, triagem de comércio e fluxos de KYC/AML.

**Observação:** as datas (data de nascimento ou data de expedição) devem estar no formato `dd/mm/aaaa`.

### Cabeçalhos

| Nome          | Valor              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parâmetros

:::info
Para a lista completa de **tipos de documento** suportados e seus parâmetros obrigatórios específicos, consulte o guia [Tipos de documento](/background-check/document-types).
:::

| Nome             | Tipo   | Obrigatório | Descrição                                                                                                  |
| ---------------- | ------ | ----------- | ---------------------------------------------------------------------------------------------------------- |
| `documentType`   | string | Não         | Tipo de documento que se deseja consultar.                                                                 |
| `documentNumber` | string | Não         | Número do documento a ser consultado, sem espaços ou pontos.                                               |
| `fullName`       | string | Não         | Em vez de documentType e documentNumber, é possível enviar diretamente o nome da pessoa/empresa.           |

### Casos de uso

- **Conformidade com sanções:** evitar negociações com partes bloqueadas sujeitas à execução pela OFAC.
- **Mitigação de risco:** reduzir penalidades das normas de comércio dos EUA e internacionais.
- **KYC e AML:** fortalecer a triagem de clientes em bancos, seguros e mercado imobiliário.
