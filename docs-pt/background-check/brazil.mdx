# Brasil

## Referência da API

### Endpoint

```
https://api.verifik.co/v2/br/background-check
```

O serviço retorna dados detalhados sobre pessoas no Brasil, incluindo nomes associados, capacidade de emitir relatórios, certificação, documento, nome completo e PDF em base64.

Útil para recrutamento, diligência, conformidade e prevenção a fraudes.

### Cabeçalhos

| Nome          | Valor              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parâmetros

:::info
Para a lista completa de **tipos de documento** suportados e parâmetros obrigatórios, consulte o guia [Tipos de documento](/background-check/document-types).
:::

| Nome           | Tipo   | Obrigatório | Descrição |
| -------------- | ------ | ----------- | --------- |
| `documentType` | string | Sim         | Apenas CPF. |
| `documentNumber` | string | Sim       | Número do documento. |
| `dateOfBirth` | string | Sim         | Data de nascimento (dd/mm/aaaa). |

### Recursos

-   **CPF Verification**: Verify the authenticity of Brazilian CPF documents
-   **Complete Background Data**: Returns full name, document details, certification number, and PDF report
-   **Structured Response**: Organized data format for easy integration
-   **Multiple Programming Languages**: Support for JavaScript, Python, PHP, and Swift
-   **Error Handling**: Comprehensive error responses for various scenarios
