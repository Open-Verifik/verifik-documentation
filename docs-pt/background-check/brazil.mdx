# Verificação de antecedentes (CPF)

## Referência da API

### Endpoint

```
https://api.verifik.co/v2/br/background-check
```

**Triagem de antecedentes individual no Brasil** usando **CPF** e **data de nascimento**. Uma resposta bem-sucedida inclui campos relacionados à identidade e, quando o provedor de dados o disponibiliza, um **relatório PDF codificado em base64**. Indicado para **triagem de candidatos**, **due diligence** e **conformidade**; trate o PDF e os metadados de acordo com suas próprias regras legais e de retenção.

### Cabeçalhos

| Nome          | Valor              |
| ------------- | ------------------ |
| Accept        | `application/json` |
| Authorization | `Bearer <token>`   |

### Parâmetros

:::info
Para a lista completa de **tipos de documento** suportados e seus parâmetros obrigatórios específicos, consulte o guia [Tipos de documento](/background-check/document-types).
:::

| Nome           | Tipo    | Obrigatório | Descrição                                    |
| -------------- | ------- | ----------- | -------------------------------------------- |
| `documentType` | string  | Sim         | Tipo de documento. Parâmetro permitido: CPF. |
| `documentNumber` | string | Sim        | Número do documento da pessoa a ser consultada. |
| `dateOfBirth` | string | Sim          | Data de nascimento da pessoa a ser consultada, formato válido: dd/mm/aaaa. |

### Recursos

-   **Verificação de CPF**: valida a autenticidade de documentos CPF brasileiros.
-   **Dados completos de antecedentes**: retorna nome completo, detalhes do documento, número de certificação e relatório em PDF.
-   **Resposta estruturada**: formato de dados organizado para fácil integração.
-   **Múltiplas linguagens de programação**: suporte para JavaScript, Python, PHP e Swift.
-   **Tratamento de erros**: respostas de erro abrangentes para diversos cenários.
