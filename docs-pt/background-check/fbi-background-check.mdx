# Verificação de antecedentes FBI

## Referência da API

### Endpoint

```
https://api.verifik.co/v2/fbi
```

Pesquise registros publicados pelo FBI usando `documentType` e `documentNumber`, ou `fullName`. Quando há correspondência, a resposta pode incluir campos de identidade e uma URL pública do FBI para o sujeito.

**Observação:** as datas (data de nascimento ou data de expedição) devem estar no formato `dd/mm/aaaa`.

### Cabeçalhos

| Nome          | Valor              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parâmetros

:::info
Para a lista completa de **tipos de documento** suportados e seus parâmetros obrigatórios específicos, consulte o guia [Tipos de documento](/background-check/document-types).
:::

| Nome             | Tipo   | Obrigatório | Descrição                                                                                                  |
| ---------------- | ------ | ----------- | ---------------------------------------------------------------------------------------------------------- |
| `documentType`   | string | Não         | Tipo de documento que se deseja consultar.                                                                 |
| `documentNumber` | string | Não         | Número do documento a ser consultado, sem espaços ou pontos.                                               |
| `fullName`       | string | Não         | Em vez de documentType e documentNumber, é possível enviar diretamente o nome da pessoa/empresa.           |

### Casos de uso

- **Verificação de identidade:** validar documentos com base em informações publicadas pelo FBI.
- **Conformidade:** apoiar triagens para cargos ou transações sensíveis.
- **Prevenção a fraudes:** confirmar identidades antes de estabelecer relacionamentos.

import axios from "axios";

const options = {
  method: "GET",

    Authorization: "jwt <seu_token>",
  },

}
