### Endpoint

```
GET https://api.verifik.co/v2/co/sisben
```

Consulta ao portal público SISBEN com **`documentType`** e **`documentNumber`**. Usa automação de navegador; a resposta pode demorar.

### Cabeçalhos

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parâmetros

| name | type | required | description |
| --- | --- | --- | --- |
| `documentType` | string | yes | **`CC`**, **`CE`**, **`PEP`**, **`PPT`**, **`PA`**. |
| `documentNumber` | string | yes | Sem espaços ou pontuação. |

### Requisição

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/sisben", {
	params: { documentType: "CC", documentNumber: "1121329661" },
	headers: {
		Accept: "application/json",
		Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
	},
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os, requests

url = "https://api.verifik.co/v2/co/sisben"
headers = {"Accept": "application/json", "Authorization": f"Bearer {os.getenv('VERIFIK_TOKEN')}"}
params = {"documentType": "CC", "documentNumber": "1121329661"}
r = requests.get(url, headers=headers, params=params)
print(r.json())
```

### Notas

- Consulta pode levar **vários segundos**.
- **`PPT`** é aceito pela API; o formulário oficial pode gerar **500** se mudar.
