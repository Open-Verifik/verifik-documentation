# Verificação de antecedentes Interpol

## Referência da API

### Endpoint

```
https://api.verifik.co/v2/interpol
```

Use este endpoint para pesquisar informações de antecedentes criminais ligados à Interpol para pessoa ou empresa com `documentType` e `documentNumber`, ou apenas `fullName`. Respostas bem-sucedidas retornam campos de identidade e detalhes de correspondência quando há registro; destinado a uso autorizado de aplicação da lei e segurança.

**Nota:** as datas (nascimento ou expedição) devem estar no formato `dd/mm/aaaa`.

### Cabeçalhos

| Nome | Valor |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parâmetros

:::info
Para a lista completa de **tipos de documento** suportados e parâmetros obrigatórios, consulte o guia [Tipos de documento](/background-check/document-types).
:::

| Nome | Tipo | Obrigatório | Descrição |
| --- | --- | --- | --- |
| `documentType`   | string | No       | Document type that you want to request                                                            |
| `documentNumber` | string | No       | Document number to consult, without spaces or points                                              |
| `fullName`       | string | No       | Instead of documentType and documentNumber, you can pass the name directly of the person/business |

### Casos de uso

- **Verificação de antecedentes criminais:** contratação, credenciais de segurança ou conformidade legal, inclusive registros internacionais.
- **Forças de segurança:** triagem autorizada com dados Interpol.

### Notas

- Acesso à base Interpol apenas para órgãos autorizados.
- Dados sensíveis: cumpra leis de privacidade.
- Confirme com canais oficiais da Interpol ou autoridades locais em decisões críticas.
- **Busca estruturada** em registros ligados à Interpol.
- **Respostas estruturadas** para integração; exemplos acima.
