# Verificação de antecedentes Interpol

## Referência da API

### Endpoint

```
https://api.verifik.co/v2/interpol
```

Use este endpoint para consultar informações de antecedentes criminais ligadas à Interpol para uma pessoa física ou jurídica utilizando `documentType` e `documentNumber`, ou apenas `fullName`. As respostas bem-sucedidas retornam campos de identidade e detalhes de correspondência quando há registro; o serviço destina-se a casos de uso seguros e autorizados de aplicação da lei e segurança.

**Observação:** as datas (data de nascimento ou data de expedição) devem estar no formato `dd/mm/aaaa`.

### Cabeçalhos

| Nome          | Valor              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parâmetros

:::info
Para a lista completa de **tipos de documento** suportados e seus parâmetros obrigatórios específicos, consulte o guia [Tipos de documento](/background-check/document-types).
:::

| Nome             | Tipo   | Obrigatório | Descrição                                                                                                  |
| ---------------- | ------ | ----------- | ---------------------------------------------------------------------------------------------------------- |
| `documentType`   | string | Não         | Tipo de documento que se deseja consultar.                                                                 |
| `documentNumber` | string | Não         | Número do documento a ser consultado, sem espaços ou pontos.                                               |
| `fullName`       | string | Não         | Em vez de documentType e documentNumber, é possível enviar diretamente o nome da pessoa/empresa.           |

### Casos de uso

- **Verificação de antecedentes criminais:** contratação, autorizações de segurança ou conformidade legal, incluindo registros internacionais.
- **Aplicação da lei:** agências autorizadas que realizam triagens com dados da Interpol.

### Observações

- Este serviço acessa a base de dados da Interpol e destina-se exclusivamente a agências autorizadas de aplicação da lei e segurança. O uso não autorizado é estritamente proibido.
- Os dados obtidos são altamente sensíveis e estão sujeitos a leis internacionais de privacidade e proteção de dados. Manuseie com cuidado e assegure conformidade com todas as regulamentações aplicáveis.
- Sempre verifique as informações com canais oficiais da Interpol ou autoridades locais para decisões críticas.
- **Pesquisa de antecedentes criminais** em registros vinculados à Interpol.
