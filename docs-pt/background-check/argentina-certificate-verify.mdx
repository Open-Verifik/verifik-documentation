# Certificado de antecedentes criminais (Argentina)

## Referência da API

### Endpoint

```
GET https://api.verifik.co/v2/ar/certificate/verify
```

Verifica um certificado **DNREC** de antecedentes criminais argentino usando o **código de solicitação** (`Solicitud`) e o **código de segurança** (`Cod. Segur.`) impressos no PDF oficial.

### Cabeçalhos (Headers)

| Nome           | Valor              |
| -------------- | ------------------ |
| Accept         | `application/json` |
| Authorization  | `Bearer <token>`   |

### Parâmetros

| Nome            | Tipo   | Obrigatório | Descrição                                              |
| --------------- | ------ | ----------- | ------------------------------------------------------ |
| `requestCode`   | string | **Sim**     | Código de solicitação (`Solicitud`). Ex.: `02118499487`. |
| `securityCode`  | string | **Sim**     | Código de segurança (`Cod. Segur.`). Ex.: `371488F861`.   |

### Requisição (Request)

<Tabs>
<TabItem value="javascript" label="JavaScript">

```javascript
import axios from 'axios';

const options = {
  method: 'GET',
  url: 'https://api.verifik.co/v2/ar/certificate/verify',
  params: { requestCode: '02118499487', securityCode: '371488F861' },
  headers: { Accept: 'application/json', Authorization: 'Bearer <seu_token>' },
};

const { data } = await axios.request(options);
console.log(data);
```

</TabItem>
<TabItem value="python" label="Python">

```python
import http.client

conn = http.client.HTTPSConnection("api.verifik.co")
headers = {"Accept": "application/json", "Authorization": "Bearer <seu_token>"}
conn.request("GET", "/v2/ar/certificate/verify?requestCode=02118499487&securityCode=371488F861", headers=headers)
print(conn.getresponse().read().decode("utf-8"))
```

### Notas

- Ambos os parâmetros são obrigatórios; caso contrário, **409** `MissingParameter`.
- Em **modo sandbox**: `requestCode=02118499487`, `securityCode=371488F861`; `requestCode=90040401` → **404**.
