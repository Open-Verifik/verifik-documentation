# Verificação de antecedentes Europol

## Referência da API

### Endpoint

```
https://api.verifik.co/v2/europol
```

Consulte registos da Europol em tempo real com `documentType` e `documentNumber`, ou apenas `fullName`. A resposta indica se há correspondência e pode incluir URL da Europol.

**Nota:** as datas devem estar no formato `dd/mm/aaaa`.

### Cabeçalhos

| Nome | Valor |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parâmetros

:::info
Para a lista completa de **tipos de documento** suportados e parâmetros obrigatórios, consulte o guia [Tipos de documento](/background-check/document-types).
:::

| Nome | Tipo | Obrigatório | Descrição |
| --- | --- | --- | --- |
| `documentType`   | string | No       | The document type that you want to request.                                                       |
| `documentNumber` | string | No       | Document number to consult, without spaces or points.                                             |
| `fullName`       | string | No       | Instead of documentType and documentNumber, you can pass the name directly of the person/business. |

### Casos de uso

- **Prevenção de fraude:** identificar pessoas sinalizadas antes de transações sensíveis.
- **Verificação de identidade:** validar dados fornecidos.
- **Conformidade:** KYC e AML em bancos, finanças e imobiliário.
