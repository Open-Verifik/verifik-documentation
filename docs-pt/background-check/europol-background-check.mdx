# Verificação de antecedentes Europol

## Referência da API

### Endpoint

```
https://api.verifik.co/v2/europol
```

Consulte em tempo real a base de dados de registros criminais da Europol usando `documentType` com `documentNumber`, ou `fullName` isoladamente. As respostas indicam se há correspondência e podem incluir uma URL de referência direta na Europol quando encontrada.

**Observação:** as datas (data de nascimento ou data de expedição) devem estar no formato `dd/mm/aaaa`.

### Cabeçalhos

| Nome          | Valor              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parâmetros

:::info
Para a lista completa de **tipos de documento** suportados e seus parâmetros obrigatórios específicos, consulte o guia [Tipos de documento](/background-check/document-types).
:::

| Nome             | Tipo   | Obrigatório | Descrição                                                                                                  |
| ---------------- | ------ | ----------- | ---------------------------------------------------------------------------------------------------------- |
| `documentType`   | string | Não         | Tipo de documento que se deseja consultar.                                                                 |
| `documentNumber` | string | Não         | Número do documento a ser consultado, sem espaços ou pontos.                                               |
| `fullName`       | string | Não         | Em vez de documentType e documentNumber, é possível enviar diretamente o nome da pessoa/empresa.           |

### Casos de uso

- **Prevenção a fraudes:** identificar pessoas sinalizadas na base criminal da Europol antes de transações sensíveis.
- **Verificação de identidade:** validar a legitimidade dos dados pessoais informados.
- **Conformidade:** apoiar verificações KYC e AML em bancos, finanças e mercado imobiliário.
