# Contratos públicos

## Referência da API

### Endpoint

```
https://api.verifik.co/v2/co/contracts
```

O serviço de Contratos Públicos permite obter a lista de contratos públicos associados a uma pessoa ou empresa, fornecendo o tipo e o número do documento. A resposta inclui detalhes valiosos sobre cada contrato, como informações do contratante, valores, entidades envolvidas, duração do contrato e mais.

Este serviço promove a transparência e facilita o monitoramento de contratos públicos com o Estado na Colômbia.

### Cabeçalhos

| Nome          | Valor              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parâmetros

:::info
Para a lista completa de **tipos de documento** suportados e seus parâmetros obrigatórios específicos, consulte o guia [Tipos de documento](/background-check/document-types).
:::

| Nome             | Tipo   | Obrigatório | Descrição                                                                 |
| ---------------- | ------ | ----------- | ------------------------------------------------------------------------- |
| `documentType`   | string | Sim         | Tipo de documento. Parâmetros permitidos: CC, NIT.                        |
| `documentNumber` | string | Sim         | Número do documento a ser consultado.                                     |
