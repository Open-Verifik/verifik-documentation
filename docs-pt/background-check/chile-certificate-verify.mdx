# Verificação de certificado — Registro Civil (Chile)

## Referência da API

### Endpoint

```
GET https://api.verifik.co/v2/cl/registro-civil/certificado
```

Verifica um certificado do **Registro Civil (SRCEI)** usando o **folio** e o **código verificador** (`Código Verificador`).

### Cabeçalhos (Headers)

| Nome           | Valor              |
| -------------- | ------------------ |
| Accept         | `application/json` |
| Authorization  | `Bearer <token>`   |

### Parâmetros

| Nome                | Tipo   | Obrigatório | Descrição                                           |
| ------------------- | ------ | ----------- | --------------------------------------------------- |
| `folio`             | string | **Sim**     | Folio do certificado (10–15 car.). Ex.: `50070080804`. |
| `verificationCode`  | string | **Sim**     | Código verificador. Ex.: `2aacbb9a636a`.            |

### Requisição (Request)

<Tabs>
<TabItem value="javascript" label="JavaScript">

```javascript
import axios from 'axios';

const { data } = await axios.get('https://api.verifik.co/v2/cl/registro-civil/certificado', {
  params: { folio: '50070080804', verificationCode: '2aacbb9a636a' },
  headers: { Accept: 'application/json', Authorization: 'Bearer <seu_token>' },
});
console.log(data);
```

</TabItem>
<TabItem value="python" label="Python">

```python
import http.client

conn = http.client.HTTPSConnection("api.verifik.co")
headers = {"Accept": "application/json", "Authorization": "Bearer <seu_token>"}
conn.request("GET", "/v2/cl/registro-civil/certificado?folio=50070080804&verificationCode=2aacbb9a636a", headers=headers)
print(conn.getresponse().read().decode("utf-8"))
```

</TabItem>
