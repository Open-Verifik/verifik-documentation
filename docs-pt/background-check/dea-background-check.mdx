# Verificação de antecedentes DEA

## Referência da API

### Endpoint

```
https://api.verifik.co/v2/dea
```

Rastreie pessoas físicas ou jurídicas na lista de vigilância da Drug Enforcement Administration (DEA) dos EUA usando `documentType` e `documentNumber`, ou `fullName`. A API retorna se houve correspondência e, quando aplicável, uma URL para o registro oficial da DEA.

**Observação:** as datas (data de nascimento ou data de expedição) devem estar no formato `dd/mm/aaaa`.

### Cabeçalhos

| Nome          | Valor              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parâmetros

:::info
Para a lista completa de **tipos de documento** suportados e seus parâmetros obrigatórios específicos, consulte o guia [Tipos de documento](/background-check/document-types).
:::

| Nome             | Tipo   | Obrigatório | Descrição                                                                                                  |
| ---------------- | ------ | ----------- | ---------------------------------------------------------------------------------------------------------- |
| `documentType`   | string | Não         | Tipo de documento que se deseja consultar.                                                                 |
| `documentNumber` | string | Não         | Número do documento a ser consultado, sem espaços ou pontos.                                               |
| `fullName`       | string | Não         | Em vez de documentType e documentNumber, é possível enviar diretamente o nome da pessoa/empresa.           |

### Casos de uso

- **Conformidade regulatória:** triagem de sinalizações relacionadas à DEA conforme normas federais dos EUA.
- **Prevenção a fraudes:** identificação de risco vinculado a registros de combate a substâncias controladas.
- **KYC e AML:** apoio a programas de saúde, farmacêutico, logística e finanças.
