# Conformidade com medidas corretivas (RNMC)

## Referência da API

### Endpoint

```
https://api.verifik.co/v2/co/policia/rnmc
```

Permite verificar se há medidas corretivas pendentes. Informe tipo e número de documento e data de verificação.

Ajuda a garantir cumprimento de medidas e decisões informadas.

### Cabeçalhos

| Nome          | Valor              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parâmetros

:::info
Para a lista completa de **tipos de documento** suportados e parâmetros obrigatórios, consulte o guia [Tipos de documento](/background-check/document-types).
:::

| Nome             | Tipo   | Obrigatório | Descrição |
| ---------------- | ------ | ----------- | --------- |
| `documentType`   | string | Sim         | CC ou CE. |
| `documentNumber` | string | Sim         | Sem espaços ou pontos. |
| `date`           | string | Sim         | Data do documento (DD/MM/AAAA). |
