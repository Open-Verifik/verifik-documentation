### Endpoint

```
GET https://api.verifik.co/v2/co/sigep/number
```

Use **`documentType`** e **`documentNumber`**. Para busca só por **nome completo**, use [SIGEP por nome](/verifik-pt/legal/sigep-public-servant-by-name).

### Cabeçalhos

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parâmetros

| name | type | required | description |
| --- | --- | --- | --- |
| `documentType` | string | yes | **`CC`** ou **`NIT`** (validação da API). |
| `documentNumber` | string | yes | Sem espaços ou pontuação. |

### Requisição

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/sigep/number", {
	params: { documentType: "CC", documentNumber: "34503110" },
	headers: {
		Accept: "application/json",
		Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
	},
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os, requests

url = "https://api.verifik.co/v2/co/sigep/number"
headers = {"Accept": "application/json", "Authorization": f"Bearer {os.getenv('VERIFIK_TOKEN')}"}
params = {"documentType": "CC", "documentNumber": "34503110"}
r = requests.get(url, headers=headers, params=params)
print(r.json())
```

### Notas

- **`documentType`** deve ser **`CC`** ou **`NIT`**.
- **200** com **`records`** vazio é possível.
