### Endpoint

```
GET https://api.verifik.co/v2/co/rama/certificado/vigencia
```

Verifica o **certificado e a vigência** de profissionais jurídicos na Colômbia. Envie **`documentType`** (**`CC`** ou **`CE`**), **`documentNumber`** e **`quality`** (ex.: **`ABG`**, **`JUEZPAZ`**, **`LT`**) na **query**. O campo **`encalidad`** segue a grafia da API.

### Cabeçalhos

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parâmetros

| name | type | required | description |
| --- | --- | --- | --- |
| `documentType` | string | yes | Apenas `CC` ou `CE`. |
| `documentNumber` | string | yes | Número do documento a consultar. |
| `quality` | string | yes | Código da categoria (ex.: `ABG`, `JUEZPAZ`, `LT`). |

### Requisição

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/rama/certificado/vigencia", {
	params: { documentType: "CC", documentNumber: "123456789", quality: "ABG" },
	headers: {
		Accept: "application/json",
		Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
	},
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os, requests

url = "https://api.verifik.co/v2/co/rama/certificado/vigencia"
headers = {"Accept": "application/json", "Authorization": f"Bearer {os.getenv('VERIFIK_TOKEN')}"}
params = {"documentType": "CC", "documentNumber": "123456789", "quality": "ABG"}
r = requests.get(url, headers=headers, params=params)
print(r.json())
```
