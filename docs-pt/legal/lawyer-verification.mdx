### Endpoint

```
GET https://api.verifik.co/v2/co/rama/abogados
```

Envie **`documentType`** (`CC`, `CE` ou `NIT`) e **`documentNumber`** na **query**. Os registros vêm no array **`results`**.

### Cabeçalhos

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parâmetros

| name | type | required | description |
| --- | --- | --- | --- |
| `documentType` | string | yes | `CC`, `CE` ou `NIT`. |
| `documentNumber` | string | yes | Número a consultar. |

### Requisição

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/rama/abogados", {
	params: { documentType: "CC", documentNumber: "123456789" },
	headers: {
		Accept: "application/json",
		Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
	},
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os, requests
```
