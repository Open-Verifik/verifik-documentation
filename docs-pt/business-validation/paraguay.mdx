# Verificação de empresas no Paraguai

A API Verifik ajuda a autenticar **empresas paraguaias** com **dados oficiais do governo**. Ela simplifica o **KYB**, reduz fraudes e apoia a conformidade.

Foi criada para empresas que precisam confirmar a **legitimidade** de companhias, parceiros ou fornecedores de forma rápida e segura.

## O que esta API valida?

Nossa API consulta registros oficiais para validar:

-   **Nome da empresa e RUC**: suporta o *Registro Único de Contribuyente* (RUC).
-   **Informações cadastrais**: nome e dados de registro.
-   **Correspondência**: confirma que as informações batem com o RUC.

## Referência da API

### Endpoint

```
https://api.verifik.co/v2/py/company
```

### Headers

| Name          | Value              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parameters

| Name           | Type    | Required | Description                                    |
| -------------- | ------- | -------- | ---------------------------------------------- |
| `documentType` | string  | Yes      | Tipo de documento. Valor permitido: `RUC`. |
| `documentNumber` | string | Yes      | Identificador da empresa. |

## Casos de uso comuns

-   **Onboarding B2B**: verificar empresas ao registrar parceiros ou fornecedores.
-   **Serviços financeiros**: autenticar empresas antes de contas ou transações.
-   **Compliance e due diligence**: validar dados em fluxos KYC/KYB.
-   **E-commerce e marketplaces**: confirmar legitimidade de vendedores PJ.

## Fontes oficiais e confiabilidade

Conexão direta com **fontes governamentais paraguaias**, com segurança em cada consulta.

## Principais benefícios

-   **Conformidade automatizada**: KYB mais fluido para parceiros.
-   **Resultados instantâneos**: onboarding em tempo real.
-   **Dados confiáveis**: registros oficiais.
-   **Integração simples**: API REST e SDKs.

## Conformidade e segurança

A Verifik é uma plataforma líder em verificação de identidade, conformidade e prevenção a fraudes na América Latina, com APIs KYC/KYB ligadas a fontes oficiais no **Paraguai** e na região.
