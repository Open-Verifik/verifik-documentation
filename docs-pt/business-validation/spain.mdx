# Verificação de empresas na Espanha

A API Verifik autentica empresas espanholas com base em fontes oficiais e apoia processos de KYB.

Indicada para validar parceiros, fornecedores e contrapartes.

## O que esta API valida?

A API consulta registos oficiais para validar:

-   **Identificador da empresa**: CIF/NIF e dados associados.
-   **Informações comerciais**: nome e status quando disponíveis.
-   **Correspondência**: os dados batem com o identificador?

## Referência da API

### Endpoint

```
https://api.verifik.co/v2/es/company
```

Este endpoint consulta o cadastro de empresas com o parâmetro `documentNumber`.

### Cabeçalhos

| Nome | Valor |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parâmetros

| Nome | Tipo | Obrigatório | Descrição |
| --- | --- | --- | --- |
| `documentNumber` | string | Sim | Identificador da empresa (CIF/NIF). |
