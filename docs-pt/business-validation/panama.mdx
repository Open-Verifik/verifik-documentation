# Verificação de empresas no Panamá

A API de verificação de empresas da Verifik permite autenticar empresas panamenhas usando dados oficiais do governo. Foi projetada para automatizar seus processos de KYB (*Know Your Business*), prevenir fraudes e garantir o cumprimento de todos os requisitos regulatórios sem esforço.

Construímos esta integração para empresas que precisam de uma forma rápida, segura e automatizada de confirmar a legitimidade de companhias, parceiros ou fornecedores.

## O que esta API valida?

Nossa API conecta-se diretamente aos registros oficiais para validar:

-   **Nome da empresa e número do RUC**: suporta o *Registro Único de Contribuyente* (RUC).
-   **Informações empresariais**: retorna nome da empresa, endereço, capital, tipo de organização, data de registro e representantes.
-   **Status da empresa**: fornece status atual, validade e detalhes de registro.
-   **Correspondência da empresa**: confirma que as informações da empresa correspondem ao número do RUC.

Ao validar esses elementos, você pode confiar que a empresa com a qual está negociando é legítima e está devidamente registrada, reduzindo significativamente os riscos de fraude e garantindo conformidade.

## Referência da API

### Endpoint

```
https://api.verifik.co/v2/pa/company
```

Consulte o registro de **RUC** (*Registro Único de Contribuyente*) do Panamá. Envie **`documentType`** (`RUC`), o **`documentNumber`** da empresa e o **`dv`** (dígito verificador). Usos comuns incluem **KYB**, onboarding de fornecedores e revisões de conformidade quando você precisa de razão social, status, representantes e campos comerciais essenciais em uma única resposta HTTPS.

### Cabeçalhos

| Nome          | Valor              |
| ------------- | ------------------ |
| Accept        | `application/json` |
| Authorization | `Bearer <token>`   |

### Parâmetros

| Nome           | Tipo    | Obrigatório | Descrição                                    |
| -------------- | ------- | ----------- | -------------------------------------------- |
| `documentType` | string  | Sim         | Tipo de documento. Valor permitido: `RUC`.   |
| `documentNumber` | string | Sim        | Identificador da empresa.                    |
| `dv`           | string  | Sim         | Dígito verificador.                          |

## Casos de uso comuns

-   **Onboarding B2B**: verifique empresas instantaneamente durante o cadastro de parceiros ou fornecedores.
-   **Serviços financeiros**: autentique empresas antes de abrir contas ou processar transações.
-   **Conformidade e diligência**: valide informações empresariais como parte de seus fluxos KYC/KYB.
-   **E-commerce e marketplaces**: confirme a legitimidade de empresas antes de permitir vendas em sua plataforma.

## Fontes oficiais e confiabilidade

Conectamo-nos diretamente às fontes oficiais do governo panamenho para garantir que você receba informações verificadas e atualizadas.
Cada consulta é tratada com estrita aderência a padrões de segurança e regulação.

## Principais benefícios

-   **Conformidade automatizada**: simplifique seus controles de KYB para prevenir fraudes sem adicionar atrito para seus parceiros de negócio.

-   **Dados confiáveis**: apoie-se em dados obtidos diretamente dos registros oficiais.
-   **Integração simples**: conecte-se facilmente via nossa API REST ou utilize nossos SDKs compatíveis.
