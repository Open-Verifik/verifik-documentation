# Verificação empresarial no Panamá

A API Verifik consulta o **RUC** (*Registro Único de Contribuyente*) com o **dígito verificador** (`dv`). Ideal para **KYB**, onboarding de fornecedores e due diligence.

## O que esta API valida?

- **RUC e DV** : identificador fiscal e dígito de controlo.
- **Dados de registo** : razão social, tipo de organização, estado, representantes quando disponíveis.

## Referência da API

### Endpoint

```
https://api.verifik.co/v2/pa/company
```

Consulta de empresa no **Panamá** por **RUC**. Parâmetros: **`documentType`** (`RUC`), **`documentNumber`** e **`dv`**. Uso típico: **KYB**, validação de parceiros e conformidade num único pedido HTTPS.

### Cabeçalhos

| Nome | Valor |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parâmetros

| Nome | Tipo | Obrigatório | Descrição |
| --- | --- | --- | --- |
| `documentType` | string | Sim | Tipo de documento. Valor: `RUC`. |
| `documentNumber` | string | Sim | Número RUC (formato com hífens). |
| `dv` | string | Sim | Dígito verificador. |

### Requisição

const { data } = await axios.get("https://api.verifik.co/v2/pa/company", {
