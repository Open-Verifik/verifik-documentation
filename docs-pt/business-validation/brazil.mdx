# Verificação empresarial no Brasil

A API de verificação empresarial da Verifik autentica empresas brasileiras com base em dados governamentais oficiais. Foi projetada para agilizar KYB, prevenir fraudes e apoiar conformidade regulatória.

## O que esta API valida?

Ela consulta registros oficiais para validar:

- **Razão social e número CNPJ**: compatível com *Cadastro Nacional da Pessoa Jurídica* (CNPJ).
- **Informações da empresa**: nome empresarial, natureza jurídica, data de abertura, status e endereço.
- **Atividades econômicas**: códigos e descrições das atividades principal e secundárias.
- **Correspondência**: as informações conferem com o número CNPJ?

## Referência da API

### Endpoint

```
https://api.verifik.co/v2/br/company
```

Verifica uma pessoa jurídica brasileira nos registros oficiais de **CNPJ**. Envie `documentType=CNPJ` e um `documentNumber` com 14 dígitos sem separadores. Os campos típicos incluem situação cadastral, endereço, natureza jurídica e atividades econômicas principal e secundária. Use para **KYB**, **due diligence de fornecedores** e **compliance**.

### Cabeçalhos

| Nome          | Valor              |
| ------------- | ------------------ |
| Accept        | `application/json` |
| Authorization | `Bearer <token>`   |

### Parâmetros

| Nome | Tipo | Obrigatório | Descrição |
| ---- | ---- | ----------- | --------- |
| `documentType` | string | **Sim** | Tipo de documento. Valor permitido: `CNPJ`. |
| `documentNumber` | string | **Sim** | CNPJ de 14 dígitos sem separadores. |
