# Verificação de empresas na Argentina

A API de verificação de empresas da Verifik permite autenticar sociedades argentinas usando dados governamentais oficiais. Foi projetada para automatizar processos KYB (*Know Your Business*), prevenir fraudes e garantir conformidade regulatória.

Construímos esta integração para empresas que precisam confirmar rapidamente, de forma segura e automatizada, a legitimidade de sociedades, parceiros ou fornecedores.

## O que esta API valida?

Nossa API conecta-se diretamente aos registros oficiais para validar:

-   **Nome da empresa e número CUIT**: suporta o *Código Único de Identificación Tributaria* (CUIT).
-   **Informações da empresa**: retorna nome, forma jurídica, data de contrato e atividades econômicas.
-   **Atividades econômicas**: fornece códigos e descrições das atividades principais e secundárias.
-   **Correspondência da empresa**: confirma que as informações correspondem ao número CUIT.

Ao validar esses elementos, você confirma que a empresa é legítima e devidamente registrada, reduzindo significativamente riscos de fraude.

## Referência da API

### Ponto de acesso

```
GET https://api.verifik.co/v2/ar/company
```

Consulte uma sociedade argentina com **`documentType`** `CUIT` e **`documentNumber`**. Retorna campos de registro para **KYB**, **triagem de fornecedores** e **conformidade**.

### Cabeçalhos

| Nome          | Valor              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parâmetros

| Nome             | Tipo   | Obrigatório | Descrição                                  |
| ---------------- | ------ | ---- | ------------------------------------------ |
| `documentType`   | string | Sim | Tipo de documento. Valor permitido: `CUIT`. |
| `documentNumber` | string | Sim | CUIT de 11 dígitos sem hífens.           |

## Casos de uso comuns

-   **Onboarding B2B**: verifique empresas instantaneamente no registro de parceiros ou fornecedores.
-   **Serviços financeiros**: autentique empresas antes de abrir contas ou processar transações.
-   **Conformidade e due diligence**: valide informações de empresas em seus fluxos KYC/KYB.
-   **E-commerce e marketplaces**: confirme a legitimidade das empresas antes de permitir vendas.

## Fontes oficiais e confiabilidade

Conectamo-nos diretamente às fontes governamentais oficiais da Argentina para garantir informações verificadas e atualizadas.
Cada consulta é tratada com rigorosos padrões de segurança e conformidade regulatória.

## Principais benefícios

-   **Conformidade automatizada**: simplifique verificações KYB sem adicionar atrito para parceiros.
-   **Resultados instantâneos**: processe verificações em segundos, ideal para onboarding de empresas em tempo real.
-   **Dados confiáveis**: use dados obtidos diretamente de registros oficiais.
-   **Integração fácil**: conecte-se via nossa API REST ou SDKs compatíveis.

## Conformidade e segurança

Priorizamos a segurança dos seus dados. A Verifik usa criptografia avançada (HTTPS/TLS 1.3) e padrões rigorosos de privacidade.
Nosso serviço é monitorado 24/7 e oferece controles de acesso baseados em funções.
