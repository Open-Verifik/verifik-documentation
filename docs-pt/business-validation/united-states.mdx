# 🇺🇸 Estados Unidos — Verificação de empresas (*KYB*)

A API de verificação de empresas da Verifik permite **consultar empresas dos Estados Unidos** usando dados públicos do **SEC EDGAR**. É adequada para **KYB** (*Know Your Business*), diligência de fornecedores e conformidade regulatória.

## O que esta API valida?

A API consulta registros oficiais para retornar informações como:

-   **Nome empresarial e identificadores**: busca por nome completo ou parcial da empresa.
-   **Dados cadastrais**: nome, CIK, EIN, endereços, tipo de entidade, bolsas de valores, tickers, códigos SIC, entre outros campos disponíveis na fonte.
-   **Correspondência**: ajuda a confirmar se os dados conferem com o registro público da empresa.

## Referência da API

### Endpoint

```
POST https://api.verifik.co/v2/usa/company
```

Consulte uma empresa dos EUA pelo **nome comercial**. Envie o campo **`business`** no **corpo JSON** de uma requisição **POST** (`Content-Type: application/json`).

### Cabeçalhos

| Nome          | Valor              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parâmetros

| Nome | Tipo | Obrigatório | Descrição |
| ---- | ---- | ----------- | --------- |
| `business` | string | Sim | Nome da empresa (completo ou parcial). |

### Requisição

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.post(
  "https://api.verifik.co/v2/usa/company",
  { business: "APPLE INC" },
  { headers: { "Content-Type": "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` } }
);
console.log(data);
```

  <TabItem value="python" label="Python">

```python
import os, requests

url = "https://api.verifik.co/v2/usa/company"
headers = {"Content-Type": "application/json", "Authorization": f"Bearer {os.getenv('VERIFIK_TOKEN')}"}
payload = {"business": "APPLE INC"}
r = requests.post(url, json=payload, headers=headers)
print(r.json())
```
