# Verificação de empresas no Canadá

A API de verificação de empresas da Verifik permite autenticar empresas canadenses usando dados oficiais do governo. Foi projetada para automatizar seus processos de KYB (*Know Your Business*), prevenir fraudes e garantir o cumprimento de todos os requisitos regulatórios sem esforço.

Construímos esta integração para empresas que precisam de uma forma rápida, segura e automatizada de confirmar a legitimidade de companhias, parceiros ou fornecedores.

## O que esta API valida?

Nossa API conecta-se diretamente aos registros oficiais para validar:

-   **Nome da empresa e número empresarial**: oferece consultas no registro empresarial canadense por nome da empresa e província.
-   **Informações empresariais**: retorna nome da empresa, número empresarial, tipo de empresa, status, detalhes de registro e endereço.
-   **Correspondência da empresa**: confirma que as informações da empresa correspondem ao registro empresarial.

Ao validar esses elementos, você pode confiar que a empresa com a qual está negociando é legítima e está devidamente registrada, reduzindo significativamente os riscos de fraude e garantindo conformidade.

## Referência da API

### Endpoint

```
https://api.verifik.co/v2/ca/company
```

### Cabeçalhos

| Nome          | Valor              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parâmetros

| Nome           | Tipo    | Obrigatório | Descrição                                    |
| -------------- | ------- | ----------- | -------------------------------------------- |
| `business`     | string  | Sim         | Nome da empresa.                             |
| `province`     | string  | Sim         | Código da província canadense.               |

## Casos de uso comuns

-   **Onboarding B2B**: verifique empresas instantaneamente durante o cadastro de parceiros ou fornecedores.
-   **Serviços financeiros**: autentique empresas antes de abrir contas ou processar transações.
-   **Conformidade e diligência**: valide informações empresariais como parte de seus fluxos KYC/KYB.
-   **E-commerce e marketplaces**: confirme a legitimidade de empresas antes de permitir vendas em sua plataforma.

## Fontes oficiais e confiabilidade

Conectamo-nos diretamente às fontes oficiais do governo canadense para garantir que você receba informações verificadas e atualizadas.
Cada consulta é tratada com estrita aderência a padrões de segurança e regulação.

## Principais benefícios

-   **Conformidade automatizada**: simplifique seus controles de KYB para prevenir fraudes sem adicionar atrito para seus parceiros de negócio.
-   **Resultados instantâneos**: processe verificações em segundos, ideal para onboarding empresarial em tempo real.
-   **Dados confiáveis**: apoie-se em dados obtidos diretamente dos registros oficiais.
-   **Integração simples**: conecte-se facilmente via nossa API REST ou utilize nossos SDKs compatíveis.

## Conformidade e segurança

Priorizamos a segurança dos seus dados. A Verifik utiliza criptografia avançada (HTTPS / TLS 1.3) e padrões rigorosos de gerenciamento de privacidade para garantir a confidencialidade.
Nosso serviço é monitorado 24/7 quanto à disponibilidade e oferece controles de acesso baseados em funções para manter o acesso da sua equipe seguro.
