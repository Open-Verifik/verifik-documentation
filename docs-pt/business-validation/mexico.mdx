# Verificação de empresas no México

A API de verificação de empresas da Verifik permite autenticar empresas mexicanas usando dados oficiais do governo. Foi projetada para automatizar seus processos de KYB (*Know Your Business*), prevenir fraudes e garantir o cumprimento de todos os requisitos regulatórios sem esforço.

Construímos esta integração para empresas que precisam de uma forma rápida, segura e automatizada de confirmar a legitimidade de companhias, parceiros ou fornecedores.

## O que esta API valida?

Nossa API conecta-se diretamente aos registros oficiais para validar:

-   **Nome da empresa e número de FME**: suporta o *Folio Mercantil Electrónico* (FME).
-   **Informações empresariais**: retorna nome da empresa, cidade e detalhes de registro.
-   **Correspondência da empresa**: confirma que as informações da empresa correspondem ao número do FME.

Ao validar esses elementos, você pode confiar que a empresa com a qual está negociando é legítima e está devidamente registrada, reduzindo significativamente os riscos de fraude e garantindo conformidade.

## Referência da API

### Endpoint

```
https://api.verifik.co/v2/mx/company
```

Este endpoint consulta uma pessoa jurídica mexicana pelo identificador **FME** (*Folio Mercantil Electrónico*). Use GET com `documentType=FME` e `documentNumber` para obter detalhes oficiais de registro, como razão social e cidade, a partir de fontes do registro mercantil governamental.

### Cabeçalhos

| Nome          | Valor              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parâmetros

| Nome           | Tipo    | Obrigatório | Descrição                                    |
| -------------- | ------- | ----------- | -------------------------------------------- |
| `documentType` | string  | Sim         | Tipo de documento. Valor permitido: `FME`.   |
| `documentNumber` | string | Sim        | Identificador FME (ex.: `N-2021007002`).     |

-   **Conformidade e diligência**: valide informações empresariais como parte de seus fluxos KYC/KYB.
-   **E-commerce e marketplaces**: confirme a legitimidade de empresas antes de permitir vendas em sua plataforma.
