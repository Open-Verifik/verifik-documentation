# Verificação de empresas no México

A API Verifik autentica empresas mexicanas com dados governamentais oficiais. Agiliza KYB, reduz fraude e apoia conformidade.

## O que esta API valida?

-   **Razão social e FME** (*Folio Mercantil Electrónico*).
-   **Dados cadastrais**: razão, cidade, registro.
-   **Correspondência**: as informações batem com o FME?

## Referência da API

### Endpoint

```
https://api.verifik.co/v2/mx/company
```

Este endpoint consulta uma pessoa jurídica mexicana pelo identificador **FME** (*Folio Mercantil Electrónico*). Use GET com `documentType=FME` e `documentNumber` para obter dados oficiais de registro, como razão social e cidade, a partir de fontes do registro comercial governamental.

### Cabeçalhos

| Nome          | Valor              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parâmetros

| Nome | Tipo | Obrigatório | Descrição |
| --- | --- | --- | --- |
| `documentType` | string | Sim | Valor permitido: `FME`. |
| `documentNumber` | string | Sim | Identificador FME (ex.: `N-2021007300`). |
