# 🇨🇱 Chile — Validação empresarial

A API de verificação de empresas da Verifik permite autenticar empresas chilenas usando dados oficiais do governo. Foi projetada para automatizar seus processos de KYB (*Know Your Business*), prevenir fraudes e garantir o cumprimento de todos os requisitos regulatórios sem esforço.

Construímos esta integração para empresas que precisam de uma forma rápida, segura e automatizada de confirmar a legitimidade de companhias, parceiros ou fornecedores.

Este serviço permite obter informações sobre uma empresa no Chile fornecendo o tipo e número do documento. A resposta inclui o nome da empresa, sua categoria empresarial, subcategoria e atividade. Adicionalmente, o serviço fornece uma lista de serviços para os quais a empresa está autorizada a operar.

### Endpoint

```
https://api.verifik.co/v2/cl/company
```

Consulte empresas chilenas por RUT — razão social, categoria, atividade e linhas de serviço autorizadas a partir de dados de registros oficiais para KYB e diligência prévia de fornecedores.

### Cabeçalhos

| Nome          | Valor              |
| ------------- | ------------------ |
| Accept        | `application/json` |
| Authorization | `Bearer <token>`   |

### Parâmetros

| Nome | Tipo | Obrigatório | Descrição | Exemplo |
| ---- | ---- | ----------- | --------- | ------- |
| `documentType` | string | **Sim** | Tipo de documento. Parâmetro permitido: RUT. | `RUT` |
| `documentNumber` | string | **Sim** | RUT da empresa que se deseja consultar; informe este parâmetro com pontos e hífens conforme a documentação nativa estabelece. | `33516727409` |

## O que esta API valida?

Nossa API conecta-se diretamente aos registros oficiais para validar:

-   **Nome da empresa e número do RUT**: suporta o *Rol Único Tributario* (RUT).
-   **Informações empresariais**: retorna nome da empresa, atividades econômicas, categorias empresariais e detalhes de autorização.
-   **Atividades econômicas**: fornece atividades econômicas detalhadas com códigos, categorias, status de IVA e datas.
-   **Correspondência da empresa**: confirma que as informações da empresa correspondem ao número do RUT.

Ao validar esses elementos, você pode confiar que a empresa com a qual está negociando é legítima e está devidamente registrada, reduzindo significativamente os riscos de fraude e garantindo conformidade.

## Casos de uso comuns

-   **Onboarding B2B**: verifique empresas instantaneamente durante o cadastro de parceiros ou fornecedores.
-   **Serviços financeiros**: autentique empresas antes de abrir contas ou processar transações.
-   **Conformidade e diligência**: valide informações empresariais como parte de seus fluxos KYC/KYB.
-   **E-commerce e marketplaces**: confirme a legitimidade de empresas antes de permitir vendas em sua plataforma.
