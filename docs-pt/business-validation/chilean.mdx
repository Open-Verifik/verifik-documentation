# Chile — Validação empresarial (RUT)

A API de verificação empresarial da Verifik ajuda a autenticar empresas chilenas com dados governamentais oficiais. Ela simplifica **KYB** (Know Your Business), apoia a prevenção a fraudes e reforça conformidade regulatória.

Foi feita para quem precisa confirmar de forma rápida e segura a legitimidade de empresas, parceiros ou fornecedores.

Com **tipo e número de documento**, você obtém nome da empresa, categoria e atividades econômicas, além de informações sobre serviços autorizados.

### Endpoint

```
https://api.verifik.co/v2/cl/company
```

Consulta empresas chilenas por **RUT**: razão social, atividades e dados cadastrais oficiais para **KYB** e diligência de fornecedores.

### Headers

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parameters

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `documentType` | string | Yes | `RUT`. |
| `documentNumber` | string | Yes | RUT da empresa (pontos e hífens conforme a documentação da integração). |

## O que esta API valida?

- **Nome e RUT da empresa**: confirma que os dados batem com o registro.
- **Atividades econômicas**: códigos, categorias, IVA e datas, quando retornados pela fonte.

## Fontes oficiais

Dados alinhados a registros governamentais chilenos, com segurança e boas práticas de privacidade.
