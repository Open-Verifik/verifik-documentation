# Verificação empresarial na Costa Rica (NITE)

## Referência da API

### Endpoint

```
https://api.verifik.co/v2/cr/company
```

Informe **`documentType`**=`NITE` e **`documentNumber`** do contribuinte.

### Cabeçalhos

| Nome | Valor |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parâmetros

| Nome | Tipo | Obrigatório | Descrição |
| --- | --- | --- | --- |
| `documentType` | string | Sim | Valor: `NITE`. |
| `documentNumber` | string | Sim | Identificador NITE. |

### Requisição

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/cr/company", {
  params: { documentType: "NITE", documentNumber: "3101122876" },
  headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});
console.log(data);
```
