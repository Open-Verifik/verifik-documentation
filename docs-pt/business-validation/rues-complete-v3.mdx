# 🇨🇴 Colômbia — RUES empresa completa (v3)

Este endpoint retorna o **dossiê completo** do registro **RUES** (*Registro Único Empresarial y Social*) para um **NIT** e uma **linha de registro** (`category`). Para um **resumo** mais leve, use **`GET /v3/co/rues`** (documentação em inglês: [Colombia RUES (v3)](https://docs.verifik.co/business-validation/colombia-rues-v3)).

## Referência da API

### Endpoint

```
GET https://api.verifik.co/v3/co/rues-complete
```

Mesmo caminho no app: `GET https://verifik.app/v3/co/rues-complete`

### Cabeçalhos

| Nome | Valor |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parâmetros

| Nome | Tipo | Obrigatório | Descrição |
| --- | --- | --- | --- |
| `documentType` | string | Sim | Deve ser **`NIT`**. |
| `documentNumber` | string | Sim | NIT; caracteres não alfanuméricos são removidos no servidor. |
| `category` | string | Não | **`RM`**, **`PROP`**, **`RUNEOL`**, **`RNT`**, **`ESAL`**, **`ESOL`**, **`JUEGOS`**, **`EXTRANJERAS`**. Padrão **`RM`**. |

### Notas

- Somente os parâmetros acima são aceitos. **`municipality`**, **`chamber`** e **`page`** **não** fazem parte de **`GET /v3/co/rues-complete`**.
- Exemplos de resposta detalhados: [RUES Complete Business Validation (v3)](https://docs.verifik.co/business-validation/rues-complete-v3) (EN).
