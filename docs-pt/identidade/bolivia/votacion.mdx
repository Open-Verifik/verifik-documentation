# Bolívia - Local de Votação

Verifik consulta **Yo Participo (OEP)** usando **CI** e **data de nascimento**. Use este endpoint para **verificação eleitoral**, **integração de eleitores** e fluxos de **conformidade** quando os dados OEP forem permitidos.

Devolve detalhes do local de votação quando a fonte os fornece:

- **`documentType`** — valor permitido: **CI**
- **`documentNumber`** — número de cédula de identidade boliviana (**5–13** caracteres)
- **`dateOfBirth`** — data de nascimento no formato **DD/MM/AAAA** (deve coincidir com os registros OEP)
- **`fullName`** — nome completo do cidadão no registro eleitoral
- **`votingStatus`** — se o cidadão está habilitado para votar
- **`department`**, **`locality`**, **`pollingPlace`**, **`pollingTable`** — campos de localização eleitoral
- **`election`** — rótulo de eleição ou nota de atualização do registro quando fornecido
- **Campos opcionais** — `isJury`, `jurySortDate`, `politicalMembership`, `latitude`, `longitude` quando incluídos pela fonte
- Resposta **assinada** pela Verifik

### Endpoint

```
GET v2/bo/votacion
```

### Headers

| Header | Valor | Descrição |
| --- | --- | --- |
| Accept | application/json | Formato de resposta |
| Authorization | Bearer \<token\> | Seu JWT da Verifik |

### Parâmetros

| Campo | Tipo | Obrigatório | Descrição |
| --- | --- | --- | --- |
| documentType | String | Sim | Tipo de documento. Valor permitido: `CI`. |
| documentNumber | String | Sim | Número de CI boliviana sem código de expedição (**5–13** caracteres). |
| dateOfBirth | String | Sim | Data de nascimento no formato `DD/MM/AAAA`. |
| force | String | Não | Defina como `true` para ignorar o cache e buscar dados atualizados. |

### Notas

- **`documentType`** deve ser **`CI`**; **`dateOfBirth`** é obrigatória e deve coincidir com os registros OEP.
- Campos opcionais de jurado, militância e coordenadas aparecem apenas quando a fonte HTML/JSON os inclui.
- Use **`force=true`** quando precisar ignorar registros em cache no MongoDB.
- Trate as respostas como **dados pessoais sensíveis**; cumpra a legislação de privacidade aplicável.
