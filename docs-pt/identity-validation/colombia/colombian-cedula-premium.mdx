### Endpoint

```
GET https://api.verifik.co/v2/co/cedula/premium
```

Mesmo caminho: `GET https://verifik.app/v2/co/cedula/premium`. **POST** com os mesmos parâmetros.

**Somente CC** — envie apenas **`documentNumber`** (5–10 caracteres).

### Cabeçalhos

| Nome | Valor |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Exemplo

<Tabs>
  <TabItem value="get" label="GET">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/cedula/premium", {
	params: { documentNumber: "1234567890" },
	headers: {
		Accept: "application/json",
		Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
	},
});
console.log(data);
```
