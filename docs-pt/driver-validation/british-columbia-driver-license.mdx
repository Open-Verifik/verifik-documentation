# Colúmbia Britânica — Carteira de motorista

A API de verificação de habilitação da Verifik permite autenticar carteiras de motorista canadenses emitidas pela província da **Colúmbia Britânica** usando dados oficiais do governo. Foi projetada para automatizar seus processos de KYC (*Know Your Customer*), prevenir fraudes e garantir o cumprimento de todos os requisitos regulatórios sem esforço.

Construímos esta integração para empresas que precisam de uma forma rápida, segura e automatizada de confirmar a validade de habilitações emitidas na província da Colúmbia Britânica.

## O que esta API valida?

Nossa API conecta-se diretamente aos registros oficiais para validar:

-   **Número da habilitação**: verifica a autenticidade do número de habilitação emitido na Colúmbia Britânica.
-   **Status da habilitação**: confirma se a habilitação está atualmente válida.
-   **Correspondência de identidade**: valida que o sobrenome informado corresponde ao registro da habilitação.

Ao validar esses elementos, você pode confiar que a habilitação é autêntica e válida, reduzindo significativamente os riscos de personificação e fraude.

## Referência da API

### Endpoint

```
GET https://api.verifik.co/v2/ca/british-columbia/driver-license
```

O serviço de verificação de habilitação da Colúmbia Britânica oferece um método simples para verificar a autenticidade e validade de uma habilitação na província da Colúmbia Britânica, Canadá. Utilizando este serviço, é possível validar rapidamente uma habilitação com base no número do documento e sobrenome informados.

### Cabeçalhos

| Nome          | Valor              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parâmetros

| Nome | Tipo | Obrigatório | Descrição | Exemplo |
| ---- | ---- | ----------- | --------- | ------- |
| `documentNumber` | string | **Sim** | Habilitação a ser consultada; todos os dados devem ser informados exatamente como aparecem neste documento. | `1123456` |
| `lastName` | string | **Sim** | Sobrenome que aparece na habilitação. | `HELLO` |
