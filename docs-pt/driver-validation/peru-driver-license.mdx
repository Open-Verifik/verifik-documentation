# 🇵🇪 Peru — Carteira de motorista

Este serviço permite obter informações detalhadas sobre a carteira de motorista de uma pessoa no Peru usando o **DNI** (*Documento Nacional de Identidad*). A resposta inclui licenças ativas, histórico de trâmites, cursos normativos, restrições, sanções e outros indicadores—útil para **frota**, **força de trabalho** e **KYC** quando credenciais de condução são relevantes.

### Endpoint

```
GET https://api.verifik.co/v2/pe/driver-license
```

Consulte com `documentType=DNI` e `documentNumber` de 8 dígitos. Os campos retornados incluem **nome completo**, **categorias** (A, AIV, B), **licenças**, **trâmites**, **cursos normativos**, **sanções**, **multas** e mais, conforme o registro.

### Cabeçalhos

| Name          | Value              |
| ------------- | ------------------ |
| Accept        | `application/json` |
| Authorization | `Bearer <token>`   |

### Parâmetros

| Nome | Tipo | Obrigatório | Descrição | Exemplo |
| ---- | ---- | ----------- | --------- | ------- |
| `documentType` | string | **Sim** | Tipo de documento. Valor permitido: `DNI`. | `DNI` |
| `documentNumber` | string | **Sim** | DNI da pessoa, sem espaços ou pontos. | `44141274` |

### Notas

- `documentNumber` deve corresponder a um DNI com registros de carteira de motorista na fonte oficial.

## Recursos

- **Consulta por DNI**: `documentType=DNI` e número do documento.
- **Categorias de licença**: Indicadores para A, AIV, B e permissões provisórias.
- **Licenças ativas**: Número, categoria, status, restrições, centro de emissão e validade.
- **Histórico de trâmites**: Revalidações e datas com status (ex.: Vigente, VENCIDA).
- **Cursos normativos**: Detalhes de inscrição e conclusão quando disponíveis.
- **Sanções e multas**: Arrays opcionais conforme o registro.
- **Respostas estruturadas**: JSON com assinatura e certificação Verifik.
