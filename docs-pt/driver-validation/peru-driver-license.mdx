# Carteira de motorista (Peru)

### Endpoint

```
GET https://api.verifik.co/v2/pe/driver-license
```

Consulta uma **carteira de motorista** peruana por **DNI** (categoria, status, validade, restrições, etc.).

### Cabeçalhos (Headers)

| Nome           | Valor              |
| -------------- | ------------------ |
| Accept         | `application/json` |
| Authorization  | `Bearer <token>`   |

### Parâmetros

| Nome              | Tipo   | Obrigatório | Descrição                         | Exemplo    |
| ----------------- | ------ | ----------- | --------------------------------- | ---------- |
| `documentType`    | string | **Sim**     | Tipo de documento. Valor: `DNI`.  | `DNI`      |
| `documentNumber`  | string | **Sim**     | Número DNI (6–8 dígitos).         | `10000001` |

### Requisição (Request)

<Tabs>
<TabItem value="javascript" label="JavaScript">

```javascript
import axios from 'axios';

const { data } = await axios.get('https://api.verifik.co/v2/pe/driver-license', {
  params: { documentType: 'DNI', documentNumber: '10000001' },
  headers: { Accept: 'application/json', Authorization: 'Bearer <seu_token>' },
});
console.log(data);
```

</TabItem>
<TabItem value="python" label="Python">

```python
import http.client

conn = http.client.HTTPSConnection("api.verifik.co")
headers = {"Accept": "application/json", "Authorization": "Bearer <seu_token>"}
conn.request("GET", "/v2/pe/driver-license?documentType=DNI&documentNumber=10000001", headers=headers)
print(conn.getresponse().read().decode("utf-8"))
```

### Notas

- Apenas `documentType=DNI` é suportado.
- Em **modo sandbox**: `documentNumber` **10000001**–**10000010**; `90040401` → **404**.
