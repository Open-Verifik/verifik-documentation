# 🇺🇸 Flórida (EUA) — Carteira de motorista

Este serviço permite **validar o status, restrições, endossos e designações** de carteiras de motorista emitidas no estado da **Flórida** (EUA). Informando um **número válido** da licença, a resposta pode incluir status, data de validade, restrições, endossos e designações retornados pela fonte.

É adequado para conferência de credenciais de motoristas e fluxos de **conformidade** e **segurança**.

### Endpoint

```
GET https://api.verifik.co/v2/usa/florida/driver-license
```

Consulta por **`documentNumber`** (número da carteira de motorista da Flórida), enviado como parâmetro de consulta na requisição **GET**.

### Cabeçalhos

| Nome          | Valor              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parâmetros

| Nome | Tipo | Obrigatório | Descrição | Exemplo |
| ---- | ---- | ----------- | --------- | ------- |
| `documentNumber` | string | Sim | Número da carteira de motorista da Flórida. | `B123-456-78-910-0` |
