# Ontário — Carteira de motorista

A API de verificação de habilitação da Verifik permite autenticar carteiras de motorista canadenses emitidas pela província de **Ontário** usando dados oficiais do governo. Foi projetada para automatizar seus processos de KYC (*Know Your Customer*), prevenir fraudes e garantir o cumprimento de todos os requisitos regulatórios sem esforço.

Construímos esta integração para empresas que precisam de uma forma rápida, segura e automatizada de confirmar a validade de habilitações emitidas na província de Ontário.

## O que esta API valida?

Nossa API conecta-se diretamente aos registros oficiais para validar:

-   **Número da habilitação**: verifica a autenticidade do número de habilitação emitido em Ontário.
-   **Status da habilitação**: confirma se a habilitação está atualmente válida.
-   **Número de verificação**: retorna um número de verificação associado para rastreabilidade da consulta.

Ao validar esses elementos, você pode confiar que a habilitação é autêntica e válida, reduzindo significativamente os riscos de personificação e fraude.

## Referência da API

### Endpoint

```
GET https://api.verifik.co/v2/ca/ontario/driver-license
```

O serviço de habilitação de Ontário permite obter rapidamente informações essenciais sobre uma habilitação emitida na província de Ontário, Canadá. Fornecendo o número do documento, é possível acessar detalhes como o status da licença e o número de verificação associado.

Use este serviço para validar e verificar a autenticidade de habilitações de Ontário de forma eficiente. Seja para confirmar a validade de uma habilitação para fins de identificação ou regulatórios, este serviço fornece informações precisas e atualizadas da licença.

### Parâmetros

| Nome | Tipo | Obrigatório | Descrição | Exemplo |
| ---- | ---- | ----------- | --------- | ------- |
| `documentNumber` | string | **Sim** | Habilitação a ser consultada; todos os dados devem ser informados exatamente como aparecem neste documento. | `S123456789123456` |

## Casos de uso comuns

-   **Fintech e bancos**: verifique a validade da habilitação durante a abertura de contas, solicitações de empréstimo ou onboarding digital de novos clientes em Ontário.
-   **Aplicativos de mobilidade e entregas**: autentique motoristas e entregadores antes que se tornem ativos em sua plataforma na província.
-   **RH e recrutamento**: valide credenciais de condução como parte do seu fluxo de contratação para cargos que exigem direção.
-   **Seguros e automotivo**: confirme a validade da habilitação antes de emitir apólices, processar sinistros ou aprovar locações.
-   **Gestão de frota**: assegure que motoristas contratados ou prestadores de serviço possuem habilitação válida emitida em Ontário.
