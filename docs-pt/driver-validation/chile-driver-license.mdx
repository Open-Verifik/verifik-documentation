# Chile — Carteira de motorista

Este serviço retorna informações detalhadas sobre a **carteira de motorista** chilena usando o **número da licença** como parâmetro. Você pode obter **RUT**, endereço, **classe** da carteira, **datas de controle**, município, **restrições** e outros campos estruturados conforme o registro oficial.

Útil para **frotas**, **RH**, **mobilidade** e processos em que a habilitação precisa ser verificada.

### Endpoint

```
https://api.verifik.co/v2/cl/driver-license
```

Consulta por **número da licença** (`documentNumber`): em geral inclui **RUT**, **classe**, **restrições**, **datas de controle**, **município** e **endereço**, conforme disponibilidade do registro.

### Headers

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parameters

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `documentNumber` | string | Yes | Número da licença sem espaços. |

### Request

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/cl/driver-license", {
  params: { documentNumber: "12345678" },
  headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});
console.log(data);
```

## Recursos

- **RUT e dados do condutor**: nome, endereço, município.
- **Classe e restrições**: categorias A, B, C, etc., e restrições de condução.
- **Datas de controle**: controle periódico e último controle.
- **Procedimentos**: duplicidade, renovação, etc., quando retornado pela fonte.
- **Integração REST** com respostas JSON estruturadas e tratamento de erros.
