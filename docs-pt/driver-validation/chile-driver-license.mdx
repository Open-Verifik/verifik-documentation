# 🇨🇱 Chile — Carteira de motorista

Este serviço permite obter informações detalhadas sobre uma carteira de motorista chilena utilizando o número da licença como parâmetro de consulta. É possível obter informações como o RUT (número único de identificação tributária), endereço, classe da licença, datas de controle e mais. Fornece dados essenciais relacionados à carteira de motorista em formato estruturado.

### Endpoint

```
GET https://api.verifik.co/v2/cl/driver-license
```

Consulte uma carteira de motorista chilena pelo seu **número da licença** (parâmetro `documentNumber`): os campos retornados normalmente incluem **RUT**, **classe da licença**, **restrições**, **datas de controle**, **município** e **endereço**, sujeitos à disponibilidade no registro — útil para verificações de **frota**, **força de trabalho** e **KYC** quando credenciais de condução são relevantes.

### Cabeçalhos

| Nome          | Valor              |
| ------------- | ------------------ |
| Accept        | `application/json` |
| Authorization | `Bearer <token>`   |

### Parâmetros

| Nome | Tipo | Obrigatório | Descrição | Exemplo |
| ---- | ---- | ----------- | --------- | ------- |
| `documentNumber` | string | **Sim** | Licença a ser consultada, sem espaços ou pontos. | `12345678` |

## Recursos

- **Validação de RUT**: verifica o número de identificação tributária chileno.
- **Informações de classe da licença**: obtém detalhes da classe da licença (A, B, C etc.).
- **Datas de controle**: acessa as datas de controle e o último controle da licença.
- **Informações do motorista**: recupera nomes, endereços e município do motorista.
- **Restrições da licença**: obtém informações sobre restrições de condução.
- **Informações do procedimento**: acessa detalhes do procedimento da licença (duplicata, renovação etc.).
- **Múltiplas linguagens de programação**: suporte para JavaScript, Python, PHP e Swift.
- **Dados em tempo real**: acesso a informações atualizadas da carteira de motorista.
- **Tratamento de erros abrangente**: respostas de erro detalhadas para diversos cenários.
- **Respostas estruturadas**: respostas JSON bem formatadas com verificação de assinatura.
- **Integração com o registro chileno**: acesso direto ao banco oficial de carteiras de motorista do Chile.
