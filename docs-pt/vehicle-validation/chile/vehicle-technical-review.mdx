# 🇨🇱 Chile - Revisão técnica veicular

Use este endpoint quando precisar de dados de **revisão técnica / inspeção** de um veículo chileno por placa: detalhes do veículo, planta de inspeção, número do certificado, data de validade, status da revisão e campos da ficha técnica quando disponíveis.

Endpoints veiculares relacionados no Chile:

- **Informações do veículo (`v2/cl/vehicle`)**: dados gerais do registro, como marca, modelo, RUT do titular, motor, chassi, multas e indicadores de furto.
- **Revisão técnica veicular (`v3/cl/vehicle`)**: status e histórico de inspeções técnicas, e dados da ficha técnica quando disponíveis.
- **SOAP veicular (`v2/cl/vehicle-soap`)**: certificado de seguro obrigatório por placa e número da apólice.

### Endpoint

```text
GET https://api.verifik.co/v3/cl/vehicle
```

Retorna dados de revisão técnica chilena para uma placa, incluindo atributos do veículo, histórico de inspeções e `technicalSheet` opcional (`engine`, `performance`, `dimensions`) quando disponível. Use para conformidade de frotas, seguros, sinistros ou onboarding veicular.

### Headers

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parameters

| Name | Type | Required | Description | Example |
| --- | --- | --- | --- | --- |
| `plate` | string | **Yes** | Placa chilena em formato compatível com v3, como quatro letras e dois dígitos ou duas letras e quatro dígitos. | `XH6640` |

### Request

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v3/cl/vehicle", {
	params: { plate: "XH6640" },
	headers: {
		Accept: "application/json",
		Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
	},
});
console.log(data);
```

### Notes

- `technicalSheet` pode ser `null` ou parcialmente preenchido quando não há dados técnicos correspondentes.
- `XH6640`, `FHDJ31` e `DCCH18` retornam perfis estáveis de revisão técnica.
- `BB985` retorna `409 Invalid_plate_format` porque não corresponde ao formato de placa v3.
