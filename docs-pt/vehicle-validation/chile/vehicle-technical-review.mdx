# 🇨🇱 Chile - Revisão técnica veicular

Use este endpoint para obter **revisão técnica / histórico de inspeções** de um veículo chileno por placa. Ele é diferente de `v2/cl/vehicle` (dados gerais do registro) e de `v2/cl/vehicle-soap` (seguro obrigatório SOAP).

### Endpoint

```text
GET https://api.verifik.co/v3/cl/vehicle
```

### Parâmetros

| Nome | Tipo | Obrigatório | Descrição | Exemplo |
| --- | --- | --- | --- | --- |
| `plate` | string | **Sim** | Placa chilena compatível com v3: quatro letras e dois dígitos, ou duas letras e quatro dígitos. | `XH6640` |

### Requisição

```javascript
const { data } = await axios.get("https://api.verifik.co/v3/cl/vehicle", {
	params: { plate: "XH6640" },
	headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});
```

### Resposta 200

```json
{
	"data": {
		"plate": "XH6640",
		"vehicle": {
			"plate": "XH6640",
			"type": "CAMIONETA (CAP. DE CARGA MENOR A 1.750 KGS.) [B] (A2)",
			"mark": "PEUGEOT",
			"model": "PARTNER TOLE 1.9",
			"year": "2004",
			"engineNumber": "10DXDM0003203",
			"chasisNumber": "60548210",
			"vin": null,
			"sealType": "SELLO VERDE"
		},
		"history": [

```

### Erros

- `404 NotFound`: registro não encontrado.
- `409 Invalid_plate_format`: formato de placa inválido.
- `409 timeout_data_source`: fonte externa indisponível ou tempo limite.

### Notes

`XH6640`, `FHDJ31` e `DCCH18` retornam perfis estáveis. `BB985` retorna `409 Invalid_plate_format`.
