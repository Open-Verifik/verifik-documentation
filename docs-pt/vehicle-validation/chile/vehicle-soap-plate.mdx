# 🇨🇱 Chile — SOAP veicular (por placa)

Consulte o certificado **SOAP** (*Seguro Obligatorio de Accidentes Personales*) usando **somente a placa**. Resposta aninhada com **`soap`** e **`vehicle`**.

### Endpoint

```
GET v2/cl/vehicle-soap-plate
```

### Cabeçalhos

| Nome | Valor |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parâmetros

| Nome | Tipo | Obrigatório | Descrição |
| --- | --- | --- | --- |
| `plate` | string | Sim | Placa chilena (4–6 caracteres). |
| `force` | string | Não | `true` para ignorar cache. |

### Solicitação

<Tabs>
  <TabItem value="javascript" label="JavaScript">

```javascript
const { data } = await axios.get("https://api.verifik.co/v2/cl/vehicle-soap-plate", {
  params: { plate: "HTRT86" },
  headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});
```

### Notas

- **Sem SOAP ativo** retorna **HTTP 200** com **`soap` vazio**, não 404.
- Placas live: `CLTK27`, `HTRT86`, `RRWX51`, `THHV64`, `CSXZ52`, `CSXZ57` (SOAP ativo); `CSXZ27` (sem SOAP).
