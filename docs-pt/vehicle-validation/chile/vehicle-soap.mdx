# 🇨🇱 Chile — SOAP veicular

A consulta **SOAP** veicular (*Seguro Obligatorio de Accidentes Personales*) valida o seguro obrigatório de acidentes pessoais de um veículo registrado no Chile. Consulte por **placa** e **número da apólice** para obter seguradora, titular, datas de vigência e atributos do veículo no registro AACH.

## Consulta SOAP veicular

### Endpoint

```
GET https://api.verifik.co/v2/cl/vehicle-soap
```

Consulte por **placa** (`plate`) e **número da apólice** (`policyNumber`; o serviço usa os últimos quatro dígitos no registro AACH). Campos típicos incluem **status**, **insuranceCompany**, **ownerName**, **ownerRut**, **validFrom**, **validTo**, **brand**, **model**, **manufactureYear**, **engineNumber** e **premium** — útil para **conformidade de seguros**, **onboarding de frotas** e **sinistros**.

### Cabeçalhos

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parâmetros

| Name | Type | Required | Description | Example |
| --- | --- | --- | --- | --- |
| `plate` | string | **Yes** | Placa chilena (Patente), 4–6 caracteres alfanuméricos. | `HTRT86` |
| `policyNumber` | string | **Yes** | Número da apólice; os últimos quatro dígitos são usados na consulta SOAP AACH. | `94596506` |

### Notas

- **Modo sandbox:** placas **`BBCC12`**–**`BBCC21`** retornam perfis SOAP fixos (`BBCC15` = **VENCIDO**); **`HTRT86`** + **`policyNumber=94596506`** retorna o perfil demo; **`ERR404`** retorna **404**; placas não mapeadas retornam **200** aleatório.
- **Exemplos sandbox:** `BBCC12`/`94590001` (sedan, MAPFRE), `BBCC18`/`94590007` (pick-up, CHUBB), `HTRT86`/`94596506` (demo TOYOTA COROLLA).
- Para consulta geral de **veículo particular** no Chile, veja [🇨🇱 Chile — Veículo](/verifik-pt/vehicle-validation/chile).
- A **disponibilidade** do registro depende do ambiente; indisponibilidade pode retornar **409** (timeout).
