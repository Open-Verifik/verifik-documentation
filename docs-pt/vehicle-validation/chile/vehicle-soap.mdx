# 🇨🇱 Chile — SOAP veicular

Consulta o certificado **SOAP** (*Seguro Obrigatório de Acidentes Pessoais*) de um veículo no Chile por **placa** e **número da apólice** (registro AACH).

## Consulta SOAP veicular

### Endpoint

```
GET https://api.verifik.co/v2/cl/vehicle-soap
```

### Cabeçalhos

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parâmetros

| name | type | required | description |
| --- | --- | --- | --- |
| `plate` | string | yes | Placa chilena (4–6 caracteres). |
| `policyNumber` | string | yes | Número da apólice (últimos 4 dígitos). |

### Requisição

<Tabs>
  <TabItem value="javascript" label="JavaScript">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/cl/vehicle-soap", {
	params: { plate: "HTRT86", policyNumber: "94596506" },
	headers: {
		Accept: "application/json",
		Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
	},
});
console.log(data);
```

  </TabItem>
</Tabs>
