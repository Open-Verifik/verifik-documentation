# 🇨🇦 Canadá — Vencimento placa Ontário

Consulte o vencimento da etiqueta de placa de Ontário usando o **número da placa** e o **número do permiso do veículo** (cartão verde / documento de registro emitido por Ontário).

Campos típicos de resposta: `expired`, `plateNumber`, `permitNumber`, `expiryDate` e `success`. O indicador `expired` é `true` quando `expiryDate` é anterior à hora atual.

Use este serviço para confirmar rapidamente a validade da etiqueta de placa junto ao registo governamental de Ontário. Ideal para onboarding de frotas, conformidade e verificação de veículos.

## Referência da API

### Endpoint

```
https://api.verifik.co/v2/ca/ontario/plate
```

Envie uma solicitação GET com os parâmetros de consulta **`plate`** e **`permitNumber`**.

### Cabeçalhos

| Nome          | Valor              |
| ------------- | ------------------ |
| Accept        | `application/json` |
| Authorization | `Bearer <token>`   |

### Parâmetros

| Nome | Tipo | Obrigatório | Descrição | Exemplo |
| ---- | ---- | ----------- | --------- | ------- |
| `plate` | string | **Sim** | Placa de passageiro de Ontário no formato **ABCD123** (4 letras + 3 dígitos, sem espaços). | `BKTR456` |
| `permitNumber` | string | **Sim** | Número do permiso no documento de registro de Ontário (8 caracteres). | `N9166001` |

### Solicitação

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/ca/ontario/plate", {
  params: { plate: "BKTR456", permitNumber: "N9166001" },
  headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});
console.log(data);
```
