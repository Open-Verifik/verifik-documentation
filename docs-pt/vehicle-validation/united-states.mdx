# 🇺🇸 Estados Unidos — Informações do veículo (placa)

Este serviço permite obter **informações detalhadas sobre um veículo nos Estados Unidos** usando a **placa** e o **estado** de registro. A resposta pode incluir marca, modelo, ano do modelo, motor, classe da carroceria, tipo de câmbio, VIN e outros campos retornados pela fonte (dados alinhados ao decode **NHTSA**).

### Endpoint

```
GET https://api.verifik.co/v2/usa/vehicle
```

Consulta por **`plate`** (placa, sem espaços ou pontos) e **`state`** (sigla do estado de registro, 2 letras). Ambos são **parâmetros de consulta** na requisição GET.

### Cabeçalhos

| Nome          | Valor              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parâmetros

| Nome | Tipo | Obrigatório | Descrição | Exemplo |
| ---- | ---- | ----------- | --------- | ------- |
| `plate` | string | **Sim** | Placa a consultar, sem espaços ou pontos. | `ABC123` |
| `state` | string | **Sim** | Sigla do estado onde o veículo está registrado. | `AL` |
