# 🇺🇸 Estados Unidos — Veículo por VIN

Este serviço decodifica um **VIN de 17 caracteres** para veículos do mercado norte-americano e devolve atributos estruturados (marca, modelo, ano, motor, classe da carroceria e campos relacionados) a partir de dados estilo **NHTSA VPIC**.

### Endpoint

```
GET https://api.verifik.co/v2/usa/vehicle-by-vin
```

Informe o **`vin`** na consulta (exatamente **17 caracteres**). O parâmetro opcional **`force`** pode forçar uma nova decodificação em vez de usar registro em cache, quando a integração permitir.

### Cabeçalhos

| Nome          | Valor              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parâmetros

| Nome | Tipo | Obrigatório | Descrição | Exemplo |
| ---- | ---- | ----------- | --------- | ------- |
| `vin` | string | **Sim** | VIN de 17 caracteres (letras e números, sem espaços). | `1G3AK47A7CM535209` |
| `force` | string | Não | Se `true`, força nova decodificação em vez de correspondência armazenada (se aplicável). | `true` |
