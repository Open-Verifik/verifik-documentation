# 🇪🇨 Equador — Verificação de veículo

Este serviço retorna informações de um veículo registrado no Equador a partir da placa (**`plate`**): marca, modelo, ano, tipo, VIN, número do motor, metadados de revisão e campos da ficha técnica quando disponíveis.

### Endpoint

```
GET https://api.verifik.co/v3/ec/vehiculo/placa
```

Consulte um veículo equatoriano por **`plate`** (sem espaços nem pontos). A resposta pode incluir `brand`, `model`, `year`, `vin`, `engineNumber`, `vehicleClass`, `vehicleType`, `color`, `serviceType`, `revision` e `technicalSheet` (`engine`, `performance`, `dimensions`) quando disponíveis. Use para **verificação**, **seguros**, **frotas** e **conformidade**.

### Headers

| Name          | Value              |
| ------------- | ------------------ |
| Accept | `application/json` |
| Authorization | `Bearer <token>`   |

### Parameters

| Name | Type | Required | Description | Example |
| ---- | ---- | -------- | ----------- | ------- |
| `plate` | string | **Yes** | Placa a consultar, sem espaços ou pontos. | `PBB3239` |

### Request

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from 'axios';

const { data } = await axios.get('https://api.verifik.co/v3/ec/vehiculo/placa', {
  params: { plate: 'PBB3239' },
  headers: {
    Accept: 'application/json',
    Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
  },
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os
import requests

url = "https://api.verifik.co/v3/ec/vehiculo/placa"
headers = {"Accept": "application/json", "Authorization": f"Bearer {os.getenv('VERIFIK_TOKEN')}"}
params = {"plate": "PBB3239"}
response = requests.get(url, headers=headers, params=params)
print(response.json())
```

### Notes

- `technicalSheet` pode ser `null` ou parcialmente preenchido quando não há dados técnicos correspondentes.
- `revision` pode ser `null` ou parcialmente preenchido conforme a disponibilidade.
