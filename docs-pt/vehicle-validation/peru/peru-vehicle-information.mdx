# Peru — Informações de veículo

O serviço peruano de informações de veículo retorna detalhes registrados para uma placa (**`plate`**): marca, modelo, ano, identificadores de motor e chassi, status, proprietários quando disponíveis e campos opcionais da ficha técnica.

### Endpoint

```
GET https://api.verifik.co/v3/pe/vehiculo/placa
```

Consulte um veículo peruano por **`plate`** (sem espaços nem pontos). A resposta pode incluir `brand`, `model`, `year`, `engineSerial`, `chasisSerial`, `vin`, `serial`, `color`, `status`, `owners` e `technicalSheet` (`engine`, `performance`, `dimensions`) quando disponíveis. Use para **verificação**, **seguros**, **frotas** e **conformidade**.

### Headers

| Name          | Value              |
| ------------- | ------------------ |
| Accept | `application/json` |
| Authorization | `Bearer <token>`   |

### Parameters

| Name    | Type   | Required | Description | Example |
| ------- | ------ | -------- | ----------- | ------- |
| `plate` | string | **Yes**  | Número da placa a consultar, sem espaços ou pontos. | `ABC123` |

### Request

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from 'axios';

const { data } = await axios.get('https://api.verifik.co/v3/pe/vehiculo/placa', {
  params: { plate: 'ABC123' },
  headers: {
    Accept: 'application/json',
    Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
  },
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os
import requests

url = "https://api.verifik.co/v3/pe/vehiculo/placa"
headers = {"Accept": "application/json", "Authorization": f"Bearer {os.getenv('VERIFIK_TOKEN')}"}
params = {"plate": "ABC123"}
response = requests.get(url, headers=headers, params=params)
print(response.json())
```

### Notes

- `technicalSheet` pode ser `null` ou parcialmente preenchido quando não há dados técnicos correspondentes.
- Placas sandbox como `ABC10001`–`ABC10010` retornam perfis de demonstração estáveis.
