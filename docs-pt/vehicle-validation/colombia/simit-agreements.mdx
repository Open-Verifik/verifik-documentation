### Endpoint

```
https://api.verifik.co/v2/co/simit/acuerdos
```

O serviço de Acordos do SIMIT fornece um método eficiente para obter acordos de pagamento relacionados a infrações de trânsito no sistema SIMIT da Colômbia. Consultando o serviço com os detalhes pertinentes, é possível obter informações abrangentes sobre acordos de pagamento, incluindo dados do infrator, números de resolução e valores devidos.

### Cabeçalhos

| Nome | Valor |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parâmetros

| Nome | Tipo | Obrigatório | Descrição |
| --- | --- | --- | --- |
| `documentType` | string | Sim | Tipo de documento. Parâmetros válidos: `CC`, `PA`, `CE`, `TI`, `RC`. |
| `documentNumber` | string | Sim | Número do documento a ser consultado, sem espaços ou pontos. |

### Requisição

const { data } = await axios.get("https://api.verifik.co/v2/co/simit/acuerdos", {

### Observações

- Os acordos do SIMIT permitem que infratores paguem multas de trânsito em parcelas.
- O status "Acuerdo de pago" indica um acordo de pagamento ativo.
- `permitePago: "N"` significa que o pagamento não está atualmente permitido.
- Os valores são informados em pesos colombianos (COP).
- Os números de resolução são usados para rastrear acordos específicos.
