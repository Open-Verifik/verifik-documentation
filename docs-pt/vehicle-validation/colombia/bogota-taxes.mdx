### Endpoint

```
GET https://api.verifik.co/v2/co/bogota/vehicle/tax
```

Retorna informações sobre obrigações tributárias de um veículo registrado em Bogotá. Envie o tipo e o número do documento do proprietário (`documentType`, `documentNumber`) e a placa (`plate`) como **parâmetros de consulta**. A resposta inclui dados do veículo e o estado das obrigações por ano fiscal.

### Cabeçalhos

| Nome | Valor |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parâmetros

| Nome | Tipo | Obrigatório | Descrição |
| --- | --- | --- | --- |
| `plate` | string | Sim | Placa a consultar, sem espaços nem pontos. |
| `documentNumber` | string | Sim | Número do documento, sem espaços nem pontos. |
| `documentType` | string | Sim | Tipo de documento. Permitidos: `CC`, `NIT`. |

const { data } = await axios.get("https://api.verifik.co/v2/co/bogota/vehicle/tax", {

### Notas

- Use `CC` para cédula de cidadania colombiana ou `NIT` para identificação tributária.
- Se `obligationStatus` for **Moroso**, há tributos em atraso a pagar.
