### Endpoint

```
GET https://api.verifik.co/v2/co/fasecolda/sinister
```

Consulta o histórico de acidentes e sinistros de seguro (**sinister**) na base Fasecolda. Envie a **placa** colombiana como parâmetro de consulta **`plate`**.

### Cabeçalhos

| Nome | Valor |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parâmetros

| Nome | Tipo | Obrigatório | Descrição |
| --- | --- | --- | --- |
| `plate` | string | Sim | Placa do veículo, sem espaços nem pontos. |

### Requisição

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/fasecolda/sinister", {
  params: { plate: "ABC123" },
  headers: {
    Accept: "application/json",
    Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
  },
});
console.log(data);
```

  </TabItem>

### Notas

- **Sinister** no path da API corresponde à terminologia colombiana de sinistros/reclamações.
- **Pérdida Menor Cuantía** indica sinistros de menor montante.
- Cada item pode incluir ID, data do acidente e tipo de cobertura.
