### Endpoint

```
GET https://api.verifik.co/v2/co/sisconmp/trainings
```

Verifica **capacitação e registro** no **SISCONMP** (Sistema de Control y Monitoreo de Personal). Envie **`documentType`** e **`documentNumber`** na **query** (instituição, curso, emissão/validade, situação da licença).

### Cabeçalhos

| Nome | Valor |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parâmetros

| Nome | Tipo | Obrigatório | Descrição |
| --- | --- | --- | --- |
| `documentType` | string | Sim | Tipo de documento: `CC`, `CE` ou `PA`. |
| `documentNumber` | string | Sim | Número do participante sem espaços nem pontos. |

### Requisição

const { data } = await axios.get("https://api.verifik.co/v2/co/sisconmp/trainings", {

  headers: {

### Notas

- O SISCONMP regula o acompanhamento de pessoal em transporte de mercadorias perigosas.
- **CURSO BASICO** é um exemplo de curso básico.
- Os certificados têm validade e podem exigir renovação.
- **`inactive: "No"`** indica que o registro está ativo (exemplo).
