### Endpoint

```
https://api.verifik.co/v2/co/simit/resoluciones
```

O serviço de Resoluções do SIMIT fornece acesso a informações detalhadas sobre resoluções de infrações de trânsito de uma pessoa ou empresa específica na Colômbia. Este serviço retorna dados como o nome do infrator, o status da resolução, a data da resolução e o valor devido.

### Cabeçalhos

| Nome | Valor |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parâmetros

| Nome | Tipo | Obrigatório | Descrição |
| --- | --- | --- | --- |
| `documentType` | string | Sim | Tipo de documento. Parâmetros válidos: `CC`, `PA`, `CE`, `TI`, `RC`. |
| `documentNumber` | string | Sim | Número do documento a ser consultado, sem espaços ou pontos. |

### Observações

- As Resoluções do SIMIT são decisões oficiais sobre infrações de trânsito.
- O status "Pendiente de pago" indica que a resolução está pendente de pagamento.
- Os números de resolução são usados para rastrear decisões administrativas específicas.
- Os valores são informados em pesos colombianos (COP).
