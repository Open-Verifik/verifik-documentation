### Endpoint

```
https://api.verifik.co/v2/co/simit/resoluciones
```

O serviço de resoluções SIMIT oferece acesso a informações detalhadas sobre resoluções de infrações de trânsito para uma pessoa ou empresa na Colômbia: nome do infrator, status da resolução, data e valor devido.

### Cabeçalhos

| Nome | Valor |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parâmetros

| Nome | Tipo | Obrigatório | Descrição |
| --- | --- | --- | --- |
| `documentType` | string | Sim | Tipo de documento. Valores válidos: `CC`, `PA`, `CE`, `TI`, `RC`. |
| `documentNumber` | string | Sim | Número do documento a consultar, sem espaços nem pontos. |

### Notas

- As resoluções SIMIT são decisões administrativas oficiais sobre infrações de trânsito.
- O status « Pendiente de pago » indica que o pagamento da resolução está pendente.
- Os números de resolução permitem rastrear cada decisão administrativa.
- Os valores estão em pesos colombianos (COP).
