### Endpoint

```
GET https://api.verifik.co/v2/co/bogota/taxi-drivers/plate
```

Retorna informações do motorista e da carta de operação de um táxi de Bogotá. Envie a **placa** do veículo como último **segmento de caminho** (por exemplo `…/plate/WMN482`), não como parâmetro de consulta. A placa deve ter 5–6 caracteres, sem espaços nem pontos.

### Cabeçalhos

| Nome | Valor |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parâmetros

| Nome | Tipo | Obrigatório | Descrição |
| --- | --- | --- | --- |
| `plate` | string | Sim | Segmento após `/plate/`. 5–6 caracteres; sem espaços ou pontos. |

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/bogota/taxi-drivers/plate/WMN482", {
  headers: {
    Accept: "application/json",
    Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
  },
});
```
