### Endpoint

```
https://api.verifik.co/v2/co/simit/comparendos
```

O serviço de autuações de trânsito fornece informações detalhadas sobre autuações associadas a um veículo na Colômbia. Ao informar a placa do veículo, é possível consultar o status, a data, o local e o tipo de infração de cada autuação.

### Cabeçalhos

| Nome | Valor |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parâmetros

| Nome | Tipo | Obrigatório | Descrição |
| --- | --- | --- | --- |
| `documentType` | string | Sim | Tipo de documento. Valores válidos: `CC`, `PA`, `CE`, `TI`, `RC`. |
| `documentNumber` | string | Sim | Número do documento a consultar, sem espaços nem pontos. |

### Requisição

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/simit/comparendos", {
  params: { 
    documentType: "CC", 
    documentNumber: "123456789" 
  },
  headers: { 
    Accept: "application/json", 
    Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` 
  },
});
console.log(data);
```

### Notas

- SIMIT é o sistema integrado colombiano de infrações de trânsito.
- « Comparendo » é o termo em espanhol para uma autuação de trânsito.
- Os valores estão em pesos colombianos (COP).
- O status « Pendiente » indica que o pagamento da autuação está pendente.
- Infrações detectadas por foto são indicadas por `fotodeteccion: true`.
