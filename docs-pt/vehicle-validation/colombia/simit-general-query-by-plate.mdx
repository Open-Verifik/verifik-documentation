### Endpoint

```
https://api.verifik.co/v2/co/simit/consultar/placa
```

Este serviço permite obter informações detalhadas sobre multas e autuações de trânsito associadas a um número específico de placa de veículo na Colômbia. A resposta inclui detalhes abrangentes sobre cada autuação, como o tipo de infração, o valor da multa e o status atual da autuação.

### Cabeçalhos

| Nome | Valor |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parâmetros

| Nome | Tipo | Obrigatório | Descrição |
| --- | --- | --- | --- |
| `plate` | string | Sim | Placa a ser consultada, sem espaços ou pontos. |

### Requisição

<Tabs>

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/simit/consultar/placa", {

  headers: {

    Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`
```

### Observações

- Este serviço abrangente fornece todos os dados do SIMIT para um veículo, incluindo multas, cursos e acordos de pagamento.
- "Multas" contém detalhes de infrações de trânsito associadas à placa do veículo.
- "Cursos" exibe cursos de educação para o trânsito relacionados às infrações do veículo.
- "AcuerdosPago" exibe acordos de pagamento para multas pendentes.
- Os valores são informados em pesos colombianos (COP).
