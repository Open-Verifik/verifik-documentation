# México

### Endpoint

```
https://api.verifik.co/v2/mx/vehiculo/placa
```

O serviço de informações de veículo fornece dados detalhados sobre um veículo no México com base no número de sua placa. Este serviço retorna especificamente informações sobre marca, modelo, ano, VIN e outros detalhes relacionados. Envie a placa no parâmetro de query `plate` em uma requisição GET; as respostas incluem campos voltados ao registro quando há correspondência.

### Cabeçalhos

| Nome          | Valor              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parâmetros

| Nome   | Tipo   | Obrigatório | Descrição                              | Exemplo   |
| ------ | ------ | ----------- | -------------------------------------- | --------- |
| plate  | String | Sim         | Placa do veículo a ser consultada.     | `ABC1234` |

### Requisição

```javascript
import axios from 'axios';

const options = {
  method: 'GET',
  url: 'https://api.verifik.co/v2/mx/vehiculo/placa',
  params: {plate: 'ABC1234'},
  headers: {
    Accept: 'application/json',
    Authorization: 'Bearer <seu_token>'
  }
};

try {
  const { data } = await axios.request(options);
  console.log(data);
} catch (error) {
  console.error(error);
}
```

### Resposta

```json
{
  "success": true,
  "data": {
    "plate": "ABC1234",
    "make": "Nissan",
    "model": "Sentra",
    "year": "2021",
    "engine": "2.0L",
    "transmission": "CVT",
    "fuelType": "Gasoline",
    "vin": "1N4AL3AP8JC123456",
    "registrationDate": "10/08/2021",
    "status": "active",
    "owner": "Carlos Rodríguez",
    "state": "Jalisco",
    "city": "Guadalajara"
  }
}
```

### Respostas de erro

```json
{
  "success": false,
  "error": "Vehicle not found",
  "code": "VEHICLE_NOT_FOUND"
}
```

## Casos de uso

- **Verificação de seguros**: verificar dados do veículo para fins de seguro.
- **Concessionárias**: validar informações do veículo antes da compra.
- **Gestão de frotas**: rastrear informações de veículos da empresa.
- **Aplicação da lei**: verificar registro e propriedade do veículo.
- **Diligência prévia**: verificar informações do veículo para transações.
