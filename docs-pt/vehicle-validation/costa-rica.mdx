# Veículo na Costa Rica (placa)

## Referência da API

### Endpoint

```
https://api.verifik.co/v2/cr/vehicle
```

Parâmetro obrigatório **`plate`** (sem espaços).

### Cabeçalhos

| Nome | Valor |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parâmetros

| Nome | Tipo | Obrigatório | Descrição | Exemplo |
| --- | --- | --- | --- | --- |
| `plate` | string | Sim | Placa a consultar. | `ABC123` |

### Requisição

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/cr/vehicle", {
  params: { plate: "ABC123" },
  headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});
console.log(data);
```
