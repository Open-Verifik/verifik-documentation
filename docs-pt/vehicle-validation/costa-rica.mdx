# 🇨🇷 Costa Rica — Informações de veículo

Este serviço retorna informações detalhadas sobre um veículo registrado na Costa Rica usando sua **placa**. Os dados típicos incluem o proprietário atual, especificações do veículo (cilindrada, valor fiscal, pesos, capacidade, potência, tração), histórico de **infrações** e **proprietários anteriores** quando disponibilizados pela fonte.

Use-o para verificações de frota, seguros e fluxos de conformidade.

## Referência da API

### Endpoint

```
https://api.verifik.co/v2/cr/vehicle
```

Envie uma requisição GET com o parâmetro de query **`plate`** (sem espaços ou pontuação). Uma resposta bem-sucedida inclui dados estruturados do veículo, campos do proprietário e arrays de histórico quando disponíveis.

### Cabeçalhos

| Nome          | Valor              |
| ------------- | ------------------ |
| Accept        | `application/json` |
| Authorization | `Bearer <token>`   |

### Parâmetros

| Nome | Tipo | Obrigatório | Descrição | Exemplo |
| ---- | ---- | ----------- | --------- | ------- |
| `plate` | string | **Sim** | Placa a ser consultada, sem espaços ou pontos. | `ABC123` |

## Recursos

- **Recuperação de informações do veículo**: obtém dados completos do veículo a partir do registro veicular costarriquense.
- **Validação de placa**: verifica placas de veículos registrados na Costa Rica.
- **Informações do proprietário atual**: recupera detalhes do proprietário atual do veículo.
- **Especificações do veículo**: obtém cilindrada, valor fiscal, peso, capacidade, potência e tração.
- **Histórico de infrações**: acessa o histórico completo de infrações de trânsito.
- **Histórico de proprietários**: visualiza a lista cronológica de todos os proprietários anteriores do veículo.
- **Autoridade judicial**: obtém informações sobre as autoridades judiciais que tratam das infrações.
- **Informações da autuação**: acessa números de autuação e de sumários para infrações.
- **Múltiplas linguagens de programação**: suporte para JavaScript, Python, PHP e Swift.
- **Dados em tempo real**: acesso a informações atualizadas do veículo.
- **Tratamento de erros abrangente**: respostas de erro detalhadas para diversos cenários.
- **Integração com o registro costarriquense**: acesso direto ao banco oficial de veículos da Costa Rica.
