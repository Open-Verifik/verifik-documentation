# 🇵🇾 Paraguai — Informações do veículo

Este serviço retorna informações detalhadas sobre um veículo no **Paraguai** usando o **número da placa**. A resposta pode incluir marca, chassi, tipo, ano, eixos, proprietário, tipo de serviço, situação atual e placa.

Útil para verificação de veículos, checagem de propriedade e monitoramento de conformidade.

## Endpoint

```
GET https://api.verifik.co/v2/py/vehicle
```

### Headers

| Name          | Value              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parameters

| Name | Type | Required | Description | Example |
| ---- | ---- | -------- | ----------- | ------- |
| `plate` | string | **Yes** | Placa a consultar. | `ABC123` |

## Recursos

- **Consulta ao registro**: detalhes do veículo no registro paraguaio
- **Validação de placa**: placas cadastradas no Paraguai
- **Especificações**: marca, chassi, tipo, ano, eixos
- **Proprietário**: dados do titular
- **Serviço e situação**: tipo de serviço (ex.: CARGA NACIONAL) e status
- **Linguagens**: exemplos em JavaScript, Python, PHP e Swift
- **Dados atualizados**: informações recentes do registro oficial
