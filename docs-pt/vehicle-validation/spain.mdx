# Espanha

## Referência da API

### Endpoint

```
https://api.verifik.co/v2/es/vehicle
```

Retorna informações de veículo para uma placa de registro espanhola, incluindo campos como marca, modelo, ano e tipo de veículo quando há correspondência.

### Cabeçalhos

| Nome          | Valor              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parâmetros

| Nome    | Tipo   | Obrigatório | Descrição                                  |
| ------- | ------ | ----------- | ------------------------------------------ |
| `plate` | string | Sim         | Placa a ser consultada (sem espaços).      |

### Requisição

<Tabs>
  <TabItem value="javascript" label="JavaScript">

```javascript
import axios from "axios";

const options = {
  method: "GET",
  url: "https://api.verifik.co/v2/es/vehicle",
  params: { plate: "5684CCD" },
  headers: {
    Accept: "application/json",
    Authorization: "Bearer <seu_token>",
  },
};

try {
  const { data } = await axios.request(options);
  console.log(data);
} catch (error) {
  console.error(error);
}
```

## Casos de uso

- **Seguros e subscrição**: confirmar atributos do veículo a partir da placa.
- **Marketplaces e logística**: validar veículos de frota ou listados.
- **Conformidade**: apoiar verificações vinculadas a veículos registrados na Espanha.
