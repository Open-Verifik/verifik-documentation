# Espanha

## Referência da API

### Endpoint

```
https://api.verifik.co/v2/es/vehicle
```

Retorna informações do veículo pela placa (marca, modelo, ano, tipo, etc.) quando houver correspondência.

### Cabeçalhos

| Nom          | Valor              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parâmetros

| Nome | Tipo | Obrigatório | Descrição |
| --- | --- | --- | --- |
| `plate` | string | Sim | Matrícula a consultar (sem espaços). |

### Solicitação

<Tabs>
  <TabItem value="javascript" label="JavaScript">

```javascript
import axios from "axios";

const options = {
  method: "GET",
  url: "https://api.verifik.co/v2/es/vehicle",
  params: { plate: "5684CCD" },
  headers: {
    Accept: "application/json",
    Authorization: "Bearer <your_token>",
  },
};

try {
  const { data } = await axios.request(options);
  console.log(data);
} catch (error) {
  console.error(error);
}
```

## Casos de uso

- **Seguros**: confirmar dados do veículo pela matrícula
- **Marketplaces e logística**: validar veículos listados
- **Conformidade**: apoiar verificações de veículos na Espanha
