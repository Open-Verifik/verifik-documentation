# Buenos Aires — Inspeção técnica

### Endpoint

```
GET https://api.verifik.co/v2/ar/buenos-aires/technical-inspection
```

Use este endpoint para consultar o histórico de inspeção técnica em Buenos Aires pela placa. A resposta retorna registros com identificadores de verificação, datas de verificação e vencimento, tipo de resultado, número do selo, comentários, marca e modelo do veículo, e nome da planta de inspeção.

### Cabeçalhos

| Name          | Value              |
| ------------- | ------------------ |
| Accept        | `application/json` |
| Authorization | `Bearer <token>`   |

### Parâmetros

| Nome | Tipo | Obrigatório | Descrição |
| ---- | ---- | -------- | ----------- |
| `plate` | string | **Yes** | Placa a consultar, sem espaços ou pontuação. Exemplo: `ABC10001`. |

### Campos de resposta

| Campo | Tipo | Descrição |
| ----- | ---- | ----------- |
| `plate` | string | Placa normalizada. |
| `totalRecords` | string | Número de registros de inspeção técnica retornados. |
| `inspections` | object[] | Registros de inspeção técnica. |
| `inspections[].verificationId` | string \| null | Identificador de verificação. |
| `inspections[].verificationDate` | string \| null | Data de verificação da inspeção. |
| `inspections[].resultTypeId` | string \| null | Identificador do tipo de resultado retornado pela fonte. |
| `inspections[].stickerNumber` | string \| null | Número do selo de inspeção. |
| `inspections[].expirationDate` | string \| null | Data de vencimento da inspeção. |
| `inspections[].brand` | string \| null | Marca do veículo. |
| `inspections[].model` | string \| null | Modelo do veículo. |
| `inspections[].plantName` | string \| null | Nome da planta de inspeção. |

### Notas

- Este endpoint está limitado a registros de inspeção técnica de Buenos Aires.
- Campos de data são normalizados quando a fonte retorna um valor dia/mês/ano reconhecível.
