# 🇦🇷 Argentina — Veículo v3

### Endpoint

```
GET https://api.verifik.co/v3/ar/vehicle
```

Use este endpoint para recuperar detalhes de um veículo argentino a partir da placa. A resposta inclui identificadores normalizados como número do motor, chassi, marca, modelo, ano, cor, tipo de unidade, status da apólice ativa e campos da ficha técnica quando disponíveis.

### Cabeçalhos

| Name          | Value              |
| ------------- | ------------------ |
| Accept        | `application/json` |
| Authorization | `Bearer <token>`   |

### Parâmetros

| Nome | Tipo | Obrigatório | Descrição |
| ---- | ---- | -------- | ----------- |
| `plate` | string | **Yes** | Formato Mercosur `AA-123-BB` (envie `AA123BB`, sem espaços ou pontuação). |

import axios from 'axios';

### Campos de resposta

| Campo | Tipo | Descrição |
| ----- | ---- | ----------- |
| `plate` | string | Placa normalizada. |
| `engineNumber` | string \| null | Número do motor retornado pela fonte. |
| `chassis` | string \| null | Identificador do chassi retornado pela fonte. |
| `year` | string \| null | Ano do veículo. |
| `brand` | string \| null | Marca do veículo. |
| `modelGroup` | string \| null | Família ou grupo do modelo. |
| `model` | string \| null | Rótulo de modelo/versão. |
| `color` | string \| null | Cor registrada. |
| `type` | string \| null | Tipo de unidade do veículo. |
| `isActivePolicyFound` | boolean | Indica se uma apólice ativa foi encontrada. |
| `technicalSheet` | object \| null | Dados da ficha técnica quando disponíveis. |

### Notas

- `technicalSheet` pode ser `null` ou parcialmente preenchido quando não há dados técnicos correspondentes.
- O serviço normaliza o formato da placa antes de consultar a fonte.
