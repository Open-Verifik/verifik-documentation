# Buenos Aires — Infrações de trânsito

### Endpoint

```
GET https://api.verifik.co/v2/ar/buenos-aires/traffic-infractions
```

Use este endpoint para verificar se uma placa possui infrações de trânsito em Buenos Aires. A resposta inclui a placa, total de registros, números de caso e auto, status, valores, autoridade emissora, datas e detalhes da infração quando registros forem encontrados.

### Cabeçalhos

| Name          | Value              |
| ------------- | ------------------ |
| Accept        | `application/json` |
| Authorization | `Bearer <token>`   |

### Parâmetros

| Nome | Tipo | Obrigatório | Descrição |
| ---- | ---- | -------- | ----------- |
| `plate` | string | **Yes** | Placa a consultar, sem espaços ou pontuação. Exemplo: `ABC10001`. |

### Campos de resposta

| Campo | Tipo | Descrição |
| ----- | ---- | ----------- |
| `plate` | string | Placa normalizada. |
| `totalRecords` | string | Número de registros retornados pela fonte. |
| `infractions` | object[] | Registros de infrações associados à placa. |
| `infractions[].caseNumber` | string \| null | Número do caso. |
| `infractions[].actNumber` | string \| null | Número do auto. |
| `infractions[].status` | string \| null | Status público do caso. |
| `infractions[].isExpired` | boolean \| null | Indica se o registro está vencido. |
| `infractions[].totalAmount` | string \| null | Valor total retornado pela fonte. |
| `infractions[].details` | object[] | Artigos e descrições da infração. |

### Notas

- Este endpoint está limitado a registros de infrações de trânsito de Buenos Aires.
- `infractions` pode ser um array vazio quando a placa não possui infrações retornadas.
