# Verificação de identidade na Espanha

A API Verifik autentica cidadãos e residentes espanhóis com base em fontes oficiais. Simplifica KYC, reduz fraude e apoia a conformidade.

Pensada para empresas que precisam confirmar com rapidez a identidade de usuários, colaboradores ou clientes.

## O que esta API valida?

A API consulta registos oficiais para validar:

-   **Nome completo e número do documento**: DNI e NIE.
-   **Correspondência**: o nome corresponde ao número do documento?
-   **Data de validade**: confirmação da data de expiração.

Essas verificações reduzem risco de fraude e impersonação.

## Referência da API

### Endpoint

```
https://api.verifik.co/v2/es/cedula
```

Este endpoint valida documentos de identidade espanhóis (DNI/NIE) via GET com `documentType`, `documentNumber` e `date` (validade).

### Cabeçalhos

| Nome | Valor |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parâmetros

| Nome | Tipo | Obrigatório | Descrição |
| --- | --- | --- | --- |
| `documentType` | string | Sim | Tipo de documento. Valores: `DNIES`, `NIE`. |
| `documentNumber` | string | Sim | Número do documento sem espaços. |
| `date` | string | Sim | Data de validade no formato `DD/MM/YYYY`. |
