# Verificação de identidade na Venezuela (CCVE)

A API Verifik autentica **cidadãos venezuelanos** com a cédula nacional **CCVE** (*Cédula de Identidad*), com base em fontes oficiais. Agiliza **KYC**, reduz fraude e apoia conformidade.

## O que esta API valida?

-   **Nome e número do documento**: cédula nacional **CCVE**.
-   **Correspondência**: nomes retornados batem com o registro.
-   **Campos opcionais**: data de nascimento, gênero, etc., conforme disponibilidade.

## Referência da API

### Endpoint

```
https://api.verifik.co/v2/ve/cedula
```

Esta rota retorna dados de identidade para **nacionais venezuelanos** com documento **CCVE**. Envie o número da cédula em **`documentNumber`** ; a API aplica **CCVE** e devolve campos estruturados para **KYC** e prevenção a fraudes. Para **estrangeiros**, use [`GET /v2/ve/foreigner-id`](/verifik-pt/identity/venezuela-foreigner-id) (**CEVE**).

### Cabeçalhos

| Nome | Valor |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parâmetros

| Nome | Tipo | Obrigatório | Descrição |
| --- | --- | --- | --- |
| `documentNumber` | string | Sim | Número da cédula nacional (CCVE). Não é necessário `documentType` na query: o serviço aplica **CCVE** neste endpoint. |
