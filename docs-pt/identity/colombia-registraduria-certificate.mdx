### Endpoint

```
GET https://api.verifik.co/v2/co/registraduria/certificado
```

**`documentNumber`**, data de expedição **`date`** (`DD/MM/YYYY`). Os mesmos campos em corpo **`POST`** JSON. Serviço para **CC** (cédula de cidadania).

### Cabeçalhos

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parâmetros

| name | type | required | description |
| --- | --- | --- | --- |
| `documentNumber` | string | sim | Número da CC sem espaços ou pontuação. |
| `date` | string | sim | **Data de expedição** do documento **`DD/MM/YYYY`**. |

### Requisição

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/registraduria/certificado", {
	params: { documentNumber: "123456789", date: "10/10/2020" },
	headers: {
		Accept: "application/json",
		Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
	},
});
console.log(data);
```

  </TabItem>
  <TabItem value="python" label="Python">

```python
import os, requests

url = "https://api.verifik.co/v2/co/registraduria/certificado"

params = {"documentNumber": "123456789", "date": "10/10/2020"}
```

### Notas

- **`date`** é a **data de expedição**, não a data de nascimento.
- **`GET`** e **`POST`** usam o mesmo handler.
- **pdfBase64** pode conter dados pessoais.
