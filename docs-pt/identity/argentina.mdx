# Verificação de identidade na Argentina

A API de verificação de identidade da Verifik permite autenticar cidadãos argentinos usando dados governamentais oficiais. Foi projetada para automatizar seus processos de KYC (*Know Your Customer*), prevenir fraudes e garantir o cumprimento de todos os requisitos regulatórios.

Construímos esta integração para empresas que precisam confirmar de forma rápida, segura e automatizada a verdadeira identidade de usuários, funcionários ou clientes.

## O que esta API valida?

Nossa API conecta-se diretamente aos registros oficiais para validar:

-   **Nome completo e número DNI**: suporta o *Documento Nacional de Identidad* (DNI).
-   **Correspondência de identidade**: confirma que o nome informado corresponde ao número DNI.

Ao validar esses elementos, você pode confiar que a pessoa é real e possui um documento válido, reduzindo significativamente os riscos de personificação e fraude.

## Referência da API

### Ponto de acesso

```
GET https://api.verifik.co/v2/ar/cedula
```

Verifique uma identidade argentina usando **`documentType`** `DNIAR` e **`documentNumber`**. Retorna campos de identidade para **KYC** e **conformidade**.

### Cabeçalhos

| Nome          | Valor              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parâmetros

| Nome             | Tipo   | Obrigatório | Descrição                                  |
| ---------------- | ------ | ---- | ------------------------------------------ |
| `documentType`   | string | Sim | Tipo de documento. Valor permitido: `DNIAR`. |
| `documentNumber` | string | Sim | Número do documento sem espaços.           |

## Casos de uso comuns

-   **Fintech e bancos**: verifique identidades instantaneamente na abertura de contas ou solicitações de empréstimo.
-   **E-commerce e entregas**: autentique usuários e entregadores antes de ativá-los.
-   **RH e recrutamento**: valide documentos de candidatos no fluxo de contratação.
-   **Seguros e saúde**: confirme identidades antes de emitir apólices ou benefícios.

## Fontes oficiais e confiabilidade

Conectamo-nos diretamente às fontes governamentais oficiais da Argentina para garantir informações verificadas e atualizadas.
Cada consulta é tratada com rigorosos padrões de segurança e conformidade regulatória.

## Principais benefícios

-   **Conformidade automatizada**: simplifique verificações KYC para prevenir fraudes sem adicionar atrito.
-   **Resultados instantâneos**: processe verificações em segundos, ideal para onboarding digital em tempo real.
-   **Dados confiáveis**: use dados obtidos diretamente de registros oficiais.
-   **Integração fácil**: conecte-se via nossa API REST ou SDKs compatíveis.

## Conformidade e segurança

Priorizamos a segurança dos seus dados. A Verifik usa criptografia avançada (HTTPS/TLS 1.3) e padrões rigorosos de privacidade.
Nosso serviço é monitorado 24/7 e oferece controles de acesso baseados em funções.
