# Verificação de identidade na Costa Rica (CCCR)

Autentique titulares de **CCCR** com fontes oficiais para **KYC** e conformidade.

## O que esta API valida?

- **CCCR** com `documentType` e `documentNumber`.
- **Correspondência** entre número e dados retornados.

## Referência da API

### Endpoint

```
https://api.verifik.co/v2/cr/cedula
```

Envie **`documentType`**=`CCCR` e **`documentNumber`** (cédula sem espaços) em GET.

### Cabeçalhos

| Nome | Valor |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parâmetros

| Nome | Tipo | Obrigatório | Descrição |
| --- | --- | --- | --- |
| `documentType` | string | Sim | Valor permitido: `CCCR`. |
| `documentNumber` | string | Sim | Número da cédula sem espaços. |

### Requisição

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/cr/cedula", {
  params: { documentType: "CCCR", documentNumber: "123456789" },
  headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});
console.log(data);
```
