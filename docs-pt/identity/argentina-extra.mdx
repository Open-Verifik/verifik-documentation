# Verificação de identidade na Argentina (ampliada)

A API Verifik autentica cidadãos argentinos com dados governamentais oficiais. Este endpoint retorna campos estendidos além da cédula padrão: **data de nascimento**, **gênero**, tokens de nome estruturados e **status vital** (`isAlive`).

## O que esta API valida?

Nossa API conecta-se diretamente aos registros oficiais para validar:

-   **Nome completo e número DNI**: suporta *Documento Nacional de Identidad* (DNI), tipo `DNIAR`.
-   **Correspondência de identidade**: confirma que o número corresponde a uma pessoa registrada.
-   **Dados extras**: retorna **dateOfBirth**, **gender**, **firstName**, **lastName**, **fullName**, **arrayName** e **isAlive**.

Ao validar esses elementos, você reduz significativamente riscos de personificação e fraude.

## Referência da API

### Ponto de acesso

```
https://api.verifik.co/v2/ar/cedula/extra
```

### Cabeçalhos

| Nome          | Valor              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parâmetros

| Nome             | Tipo   | Obrigatório | Descrição                                                                   |
| ---------------- | ------ | -------- | --------------------------------------------------------------------------- |
| `documentType`   | string | Sim         | Tipo de documento. Valor permitido: `DNIAR`.                                |
| `documentNumber` | string | Sim         | Número DNI a consultar, sem espaços.                                        |

### Notas

- `documentType` deve ser `DNIAR`.
- `gender` é retornado como código de uma letra (ex.: `F` ou `M`) quando disponível.
- `isAlive` indica se o registro reporta a pessoa como viva no momento da consulta.
