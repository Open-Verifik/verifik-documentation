### Endpoint

```
GET https://api.verifik.co/v2/co/foreigner-id/pep
```

Envie **`documentNumber`** e **`expeditionDate`** (`DD/MM/YYYY`) na **query**. O mesmo payload em **`POST`** (JSON).

### Cabeçalhos

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Requisitos do documento

**Para quem?** **Nacionais venezuelanos** na Colômbia com *Permiso Especial de Permanencia* (**PEP**) — **permissão de imigração**, não triagem PLD de pessoa politicamente exposta (veja [PEP colombiano (PLD)](/verifik-pt/identity-validation/colombia/colombian-politically-exposed-persons)).

| Campo | Orientação |
| --- | --- |
| **`documentNumber`** | Número PEP, **somente dígitos**. Na Colômbia este número tem **sempre 15 dígitos**. Envie os 15 dígitos sem espaços ou pontuação. Exemplo: `123456789012345`. |
| **`expeditionDate`** | Data de expedição **`DD/MM/AAAA`** (ex. `10/10/2024`). Obrigatória. |

Comparação completa: [Guia de documentos de identidade](/verifik-pt/identity-validation/colombia/colombia-identity-documents-guide).

### Parâmetros

| name | type | required | description |
| --- | --- | --- | --- |
| `documentNumber` | string | yes | Número PEP, **somente dígitos**. **15 dígitos** (comprimento fixo na Colômbia). Exemplo: `123456789012345`. |
| `expeditionDate` | string | yes | Data de expedição **`DD/MM/AAAA`** (ex. `10/10/2024`). Deve coincidir com o documento. |

### Requisição

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/foreigner-id/pep", {
	params: { documentNumber: "123456789", expeditionDate: "10/10/2024" },
	headers: {
		Accept: "application/json",
		Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
	},
});
console.log(data);
```
