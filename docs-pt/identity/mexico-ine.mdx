# México — Validação INE

Valide uma **INE** mexicana (*Credencial para Votar*) no cadastro eleitoral. Envie `documentType=INE` e um identificador principal em `documentNumber`. A API detecta automaticamente se o valor é um **CIC** (9 dígitos), **OCR** (13 dígitos) ou **clave de elector** (18 caracteres).

Campos opcionais cobrem combinações por modelo exigidas pela fonte oficial:

- **`citizenIdentifier`** — necessário para modelos **E, F, G, H** com CIC
- **`emissionNumber`** — necessário para o modelo **C** com OCR e clave de elector
- **`model`** — dica opcional (`C`–`H`) para selecionar a combinação correta

Para consulta por **CURP**, use **[Cidadão mexicano (CURP)](/verifik-pt/identity/mexico)** (`v2/mx/curp`). Para extrair campos de imagens da INE, use **[México — OCR INE](/verifik-pt/identity/mexico-ine-ocr)** (`v2/mx/ine/ocr`).


## API reference

### Endpoint

```
https://api.verifik.co/v2/mx/ine
```

Submit an INE identifier to obtain electoral **status**, **validity**, emission metadata, and related registry fields when the credential is recognized.

### Headers

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parameters

| Name | Type | Required | Description | Example |
| --- | --- | --- | --- | --- |
| `documentType` | string | Yes | Allowed value: `INE`. | `INE` |
| `documentNumber` | string | Yes | CIC (9 digits), OCR (13 digits), or clave de elector (18 alphanumeric). | `187639699` |
| `citizenIdentifier` | string | No | Citizen identifier from the back of the card. Needed for models E–H with CIC. | `031206745` |
| `emissionNumber` | string | No | Emission number. Needed for model C. | `1` |
| `model` | string | No | Optional card model hint: `C`, `D`, `E`, `F`, `G`, or `H`. | `E` |

### Notes

- `documentType` must be `INE`.
- `documentNumber` auto-detects CIC (9 digits), OCR (13 digits), or clave de elector (18 alphanumeric).
- Models E–H typically need `citizenIdentifier` with CIC; model C needs `emissionNumber` with OCR and clave de elector.
- Negative electoral verdicts (for example not vigente) still return HTTP **200** with `status` and `message` reflecting the registry result.
- Sandbox fixtures: `187639699`, `0276031206745`, `CSFTGV76061131H400`. Document `900404019` returns **404**.
- Related: [Mexican Citizen (CURP)](/verifik-pt/identity/mexico), [Mexico — INE OCR](/verifik-pt/identity/mexico-ine-ocr).
