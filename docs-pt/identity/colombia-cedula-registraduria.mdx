# Colômbia — Consulta de cidadão via Registraduría (SCCRC)

Consulte um registro civil de **nascimento** (*registro civil de nacimiento*) da **Registraduría Nacional del Estado Civil** (**SCCRC**) usando o número de **cédula (CC)**.

Diferente de [cidadão por documento](/identity-validation/colombia/colombian-citizen) (`/v2/co/cedula`), este endpoint usa **somente a Registraduría** — não tenta Procuraduría, DIAN nem Olimpia.

## O que esta API retorna

- **`documentNumber`**, **`documentType`**
- **`firstName`**, **`lastName`**, **`fullName`**, **`arrayName`**
- **`sexo`**, **`serial`**
- **`oficina`** / **`expeditionPlace`**, **`fecha`** / **`dateOfBirth`** quando disponíveis
- **`recordType`** — `REGISTRO CIVIL DE NACIMIENTO`
- Resposta Verifik **assinada**

## Referência da API

### Endpoint

```
GET https://api.verifik.co/v2/co/cedula/registraduria
```

Também disponível como **`POST`**. **`GET`** usa **query parameters** abaixo.

### Headers

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parameters

| name | type | required | description | Examples |
| --- | --- | --- | --- | --- |
| `documentType` | string | yes | Deve ser **`CC`** | `CC` |
| `documentNumber` | string | yes | Número da cédula colombiana | `1032386359` |

## Recursos

- Consulta SCCRC em **modo documento** (somente CC)
- Sem Procuraduría, DIAN ou Olimpia
- Resposta de registro único
- **GET** e **POST** compartilham o mesmo handler

### Notas

- **`documentType`** deve ser **`CC`**.
- A busca SCCRC por documento pode retornar **404** mesmo quando [por nome](/identity-validation/colombia/colombian-citizen-by-name) encontra a pessoa.
- Indisponibilidade temporária pode aparecer como **409** (`Endpoint_out_of_service`).
- Relacionado: [Cidadão por documento](/identity-validation/colombia/colombian-citizen), [Por nome](/identity-validation/colombia/colombian-citizen-by-name), [Por serial](/identity/colombia-registraduria-serial).
