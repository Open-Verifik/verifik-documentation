# Verificação de identidade no Paraguai

A API Verifik ajuda a autenticar **cidadãos paraguaios** usando **dados oficiais do governo**. Ela simplifica o **KYC**, reduz fraudes e apoia a conformidade.

Foi criada para empresas que precisam confirmar a identidade de usuários, funcionários ou clientes de forma **rápida, segura e automatizada**.

## O que esta API valida?

Nossa API consulta registros oficiais para validar:

-   **Nome completo e número CIC**: suporta a *Cédula de Identidad Civil* (CIC).
-   **Correspondência**: confirma que o nome informado corresponde ao número CIC.

Com isso, você reduz risco de impersonação e aumenta a confiança no onboarding.

## Referência da API

### Endpoint

```
https://api.verifik.co/v2/py/cic
```

### Headers

| Name          | Value              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parameters

| Name           | Type    | Required | Description                                    |
| -------------- | ------- | -------- | ---------------------------------------------- |
| `documentNumber` | string | Yes      | Número do documento CIC. |

### Request

<Tabs>
