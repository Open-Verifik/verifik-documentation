# Verificação de habilitação no Canadá

A API de verificação de habilitação da Verifik permite autenticar habilitações canadenses usando dados oficiais do governo. Foi projetada para automatizar seus processos de KYC (*Know Your Customer*), prevenir fraudes e garantir o cumprimento de todos os requisitos regulatórios sem esforço.

Construímos esta integração para empresas que precisam de uma forma rápida, segura e automatizada de confirmar a validade de habilitações emitidas nas províncias da Colúmbia Britânica e Ontário.

## O que esta API valida?

Nossa API conecta-se diretamente aos registros oficiais para validar:

-   **Número da habilitação**: verifica a autenticidade dos números de habilitação.
-   **Status da habilitação**: confirma se a habilitação está atualmente válida.
-   **Correspondência de identidade**: para a Colúmbia Britânica, valida se o sobrenome corresponde ao registro da habilitação.

Ao validar esses elementos, você pode confiar que a habilitação é autêntica e válida, reduzindo significativamente os riscos de fraude.

Os serviços canadenses de validação de identidade incluem verificação de habilitações para as províncias da Colúmbia Britânica e Ontário. Esses serviços permitem verificar a autenticidade e validade das habilitações emitidas nessas províncias canadenses.

## Províncias suportadas

| Província | Endpoint | Descrição |
|-----------|----------|-----------|
| **Colúmbia Britânica** | `/v2/ca/british-columbia/driver-license` | Verifica habilitações da BC com número do documento e sobrenome |
| **Ontário** | `/v2/ca/ontario/driver-license` | Verifica habilitações de Ontário com número do documento |

## Habilitação - Colúmbia Britânica

### Endpoint

```
GET https://api.verifik.co/v2/ca/british-columbia/driver-license
```

O serviço de verificação de habilitação da Colúmbia Britânica oferece um método simples para verificar a autenticidade e validade de uma habilitação na província da Colúmbia Britânica, Canadá. Utilizando este serviço, é possível validar rapidamente uma habilitação com base no número do documento e sobrenome informados.

### Cabeçalhos

| Nome          | Valor              |
| ------------- | ------------------ |
| Content-Type  | `application/json` |
| Authorization | `Bearer <token>`   |

### Parâmetros

| Nome | Tipo | Obrigatório | Descrição | Exemplo |
| ---- | ---- | ----------- | --------- | ------- |
| `documentNumber` | string | **Sim** | Habilitação a ser consultada; todos os dados devem ser informados exatamente como aparecem neste documento. | `1123456` |
| `lastName` | string | **Sim** | Sobrenome que aparece na habilitação. | `HELLO` |

## Habilitação - Ontário

### Endpoint

```
GET https://api.verifik.co/v2/ca/ontario/driver-license
```

O serviço de habilitação de Ontário permite obter rapidamente informações essenciais sobre uma habilitação emitida na província de Ontário, Canadá. Fornecendo o número do documento, é possível acessar detalhes como o status da licença e o número de verificação associado.

Use este serviço para validar e verificar a autenticidade de habilitações de Ontário de forma eficiente. Seja para confirmar a validade de uma habilitação para fins de identificação ou regulatórios, este serviço fornece informações precisas e atualizadas da licença.

## Casos de uso comuns

-   **Fintech e bancos**: verifique habilitações instantaneamente durante a abertura de contas ou solicitações de empréstimo.
-   **Mobilidade e entregas**: autentique motoristas antes de tornarem-se ativos em sua plataforma.
-   **RH e recrutamento**: valide credenciais de motoristas como parte do seu fluxo de contratação.
-   **Seguros e automotivo**: confirme a validade de habilitações antes de emitir apólices ou processar sinistros.

## Fontes oficiais e confiabilidade

Conectamo-nos diretamente aos sistemas oficiais de licenciamento provinciais canadenses (Colúmbia Britânica e Ontário) para garantir que você receba informações verificadas e atualizadas.
Cada consulta é tratada com estrita aderência a padrões de segurança e regulação.

## Principais benefícios

-   **Conformidade automatizada**: simplifique suas verificações para prevenir fraudes sem adicionar atrito para seus usuários.

-   **Dados confiáveis**: apoie-se em dados obtidos diretamente dos registros provinciais oficiais.
