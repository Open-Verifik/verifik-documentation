# Verificação de identidade na Bolívia

A API de verificação de identidade da Verifik permite autenticar cidadãos bolivianos usando dados governamentais oficiais. Foi projetada para automatizar seus processos KYC (*Know Your Customer*), prevenir fraudes e garantir o cumprimento de todos os requisitos regulatórios sem esforço.

Construímos esta integração para empresas que precisam de uma forma rápida, segura e automatizada de confirmar a verdadeira identidade de usuários, funcionários ou clientes.

## O que esta API valida?

Nossa API conecta-se diretamente aos registros oficiais para validar:

-   **Nome completo e número CI**: suporta a *Cédula de Identidad* (CI).
-   **Correspondência de identidade**: confirma que o nome informado corresponde ao número CI.
-   **Verificação de data de nascimento**: valida que a data de nascimento corresponde aos registros oficiais.

Ao validar esses elementos, você pode confiar que a pessoa com quem está negociando é real e possui um documento válido, reduzindo significativamente os riscos de personificação e fraude.

## Referência da API

### Ponto de acesso

```
GET https://api.verifik.co/v2/bo/cedula
```

Verifique um cidadão boliviano com **`documentType`** `CI`, **`documentNumber`** e **`dateOfBirth`** (`DD/MM/AAAA`). Retorna campos de identidade para **KYC** e **conformidade**.

### Cabeçalhos

| Nome | Valor |
| --- | --- |
| Content-Type | `application/json` |
| Authorization | `Bearer <token>` |

### Parâmetros

| Nome | Tipo | Obrigatório | Descrição |
| --- | --- | --- | --- |
| `documentType` | string | Sim | Tipo de documento. Valor permitido: `CI`. |
| `documentNumber` | string | Sim | Número do documento CI. |
| `dateOfBirth` | string | Sim | Data de nascimento no formato `DD/MM/AAAA`. |

## Casos de uso comuns

-   **Fintech e bancos**: verifique identidades instantaneamente na abertura de contas ou solicitações de empréstimo.
-   **E-commerce e entrega**: autentique usuários e entregadores antes que fiquem ativos na plataforma.
-   **RH e recrutamento**: valide documentos de candidatos no processo de contratação.
-   **Seguros e saúde**: confirme identidades antes de emitir apólices ou benefícios médicos.
