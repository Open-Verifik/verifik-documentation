# Verificação de identidade no Chile

A API de verificação de identidade da Verifik ajuda a autenticar cidadãos chilenos usando o **Rol Único Nacional (RUN)**. Ela foi pensada para simplificar seus processos de **KYC** (Know Your Customer), reduzir fraudes e apoiar requisitos regulatórios.

Construímos esta integração para empresas que precisam de um jeito rápido, seguro e automatizado de confirmar a identidade real de usuários, colaboradores ou clientes.

## O que esta API valida?

Nossa API consulta registros oficiais para validar:

- **Nome completo e RUN**: suporta o *Rol Único Nacional* (RUN).
- **Correspondência de identidade**: confirma se o nome informado corresponde ao número RUN.

Com isso, você reduz o risco de impersonação e aumenta a confiança no cadastro.

## API Reference

### Endpoint

```
https://api.verifik.co/v2/cl/cedula
```

Verifica identidade civil no Chile: envie **`documentType`** = `RUN` e **`documentNumber`** (RUN sem espaços) em uma requisição GET; a resposta inclui campos de nome estruturados para **KYC**, onboarding e checagens antifraude.

### Headers

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parameters

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `documentType` | string | Yes | Valor permitido: `RUN`. |
| `documentNumber` | string | Yes | RUN sem espaços ou separadores. |

## Casos de uso comuns

- **Fintech e bancos**: validar identidades na abertura de conta ou em crédito.

- **Seguros e saúde**: confirmar identidade antes de apólices ou benefícios.
