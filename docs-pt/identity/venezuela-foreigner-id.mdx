# 🇻🇪 Venezuela — Cédula de estrangeiro (CEVE)

Autenticação de **estrangeiros** na Venezuela com a **CEVE** (*Cédula de Identidad de Extranjero*).

## O que esta API valida?

-   **Identidade CEVE**: número e nomes coerentes com o registro.
-   **Tipo**: a resposta inclui `documentType: "CEVE"`.

## Referência da API

### Endpoint

```
https://api.verifik.co/v2/ve/foreigner-id
```

Use para **estrangeiros** com **CEVE**. Passe **`documentNumber`** ; o serviço aplica **CEVE**. Para **nacionais** (**CCVE**), use [`GET /v2/ve/cedula`](/verifik-pt/identity/venezuela).

### Cabeçalhos

| Nome | Valor |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parâmetros

| Nome | Tipo | Obrigatório | Descrição |
| --- | --- | --- | --- |
| `documentNumber` | string | Sim | Número CEVE. Não é necessário `documentType` na query: **CEVE** neste endpoint. |

### Requisição

<Tabs>
