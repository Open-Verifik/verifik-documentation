# Costa Rica - Local de votação

A Verifik consulta o registro eleitoral **Donde Votar do TSE** usando **CCCR** (cédula costarriquenha, **9 a 13 dígitos**). Use este endpoint para **verificação eleitoral**, **integração de eleitores** e **conformidade** quando os dados eleitorais costarriquenhos forem permitidos.

Retorna detalhes do local de votação quando a fonte os fornece:

- **`documentType`** — valor permitido: **CCCR**
- **`documentNumber`** — cédula costarriquenha (**9 a 13** dígitos)
- **`fullName`**, **`firstName`**, **`firstLastName`**, **`secondLastName`** — nome do cidadão no registro eleitoral
- **`gender`** — código de sexo quando fornecido pela fonte
- **`province`**, **`canton`**, **`district`** — residência e geografia eleitoral
- **`pollingPlace`**, **`pollingTable`** — detalhes do local de votação
- **`electoralCode`**, **`electorNumber`**, **`schoolAddress`** — metadados de junta do TSE
- **Campos opcionais** — `latitude`, `longitude` quando incluídos pela fonte
- Resposta **assinada** pela Verifik

### Endpoint

```
GET v2/cr/votacion
```

Consulte o registro Donde Votar do TSE com **`documentType`** `CCCR` e um **`documentNumber`** de **9 a 13 dígitos**. Respostas bem-sucedidas incluem local de votação, junta (mesa), província, cantão e distrito para fluxos de **KYC**, **verificação eleitoral** e **conformidade**.

### Headers

| Header | Valor | Descrição |
| --- | --- | --- |
| Accept | application/json | Formato de resposta |
| Authorization | Bearer \<token\> | Seu JWT Verifik |

### Parâmetros

| Campo | Tipo | Obrigatório | Descrição | Exemplo |
| --- | --- | --- | --- | --- |
| documentType | String | Sim | Tipo de documento. Valor permitido: `CCCR`. | `CCCR` |
| documentNumber | String | Sim | Cédula costarriquenha (**9 a 13** dígitos). | `401310036` |

### Notas

- **`documentType`** deve ser **`CCCR`**; **`documentNumber`** deve ter **9 a 13** dígitos numéricos.
- **IDs de exemplo em produção:** `401310036`, `401310030`, `401310039`, `402330708`, `114070037`.
- **Modo sandbox:** use IDs de teste **`010000001`** a **`010000010`** para perfis estáveis; **`090040401`** retorna **404**.
- Trate as respostas como **dados pessoais sensíveis**; cumpra as leis de privacidade aplicáveis.
- Veja também: [Verificação de identidade na Costa Rica (CCCR)](/identity/costa-rica).
