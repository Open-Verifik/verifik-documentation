# Verificação de identidade no Equador

A API de verificação de identidade da Verifik permite autenticar cidadãos equatorianos usando dados governamentais oficiais. Foi projetada para automatizar seus processos KYC (*Know Your Customer*), prevenir fraudes e garantir o cumprimento de todos os requisitos regulatórios sem esforço.

Construímos esta integração para empresas que precisam de uma forma rápida, segura e automatizada de confirmar a verdadeira identidade de usuários, funcionários ou clientes.

## O que esta API valida?

Nossa API conecta-se diretamente aos registros oficiais para validar:

-   **Nome completo e número CCEC**: suporta a *Cédula de Ciudadanía Ecuatoriana* (CCEC).
-   **Correspondência de identidade**: confirma que o nome informado corresponde ao número CCEC.

Ao validar esses elementos, você pode confiar que a pessoa com quem está negociando é real e possui um documento válido, reduzindo significativamente os riscos de personificação e fraude.

## Referência da API

### Ponto de acesso

```
GET https://api.verifik.co/v2/ec/cedula
```

Verifique um cidadão equatoriano com **`documentType`** `CCEC` e **`documentNumber`**. Retorna campos de identidade de registros oficiais para **KYC** e **conformidade**.

### Cabeçalhos

| Nome | Valor |
| --- | --- |
| Content-Type | `application/json` |
| Authorization | `Bearer <token>` |

### Parâmetros

| Nome | Tipo | Obrigatório | Descrição |
| --- | --- | --- | --- |
| `documentType` | string | Sim | Tipo de documento. Valor permitido: `CCEC`. |
| `documentNumber` | string | Sim | Número do documento sem espaços ou pontos. |

## Casos de uso comuns

-   **Fintech e bancos**: verifique identidades instantaneamente na abertura de contas ou solicitações de empréstimo.
-   **E-commerce e entrega**: autentique usuários e entregadores antes que fiquem ativos na plataforma.
-   **RH e recrutamento**: valide documentos de candidatos no processo de contratação.
-   **Seguros e saúde**: confirme identidades antes de emitir apólices ou benefícios médicos.

## Fontes oficiais e confiabilidade

Conectamo-nos diretamente a fontes governamentais oficiais equatorianas para garantir informações verificadas e atualizadas.
Cada consulta é tratada com estrita aderência a padrões de segurança e regulamentação.

## Principais benefícios

-   **Conformidade automatizada**: simplifique verificações KYC para prevenir fraudes sem atrito para usuários.
-   **Resultados instantâneos**: processe verificações em segundos, ideal para onboarding digital em tempo real.
-   **Dados confiáveis**: confie em dados provenientes diretamente de registros oficiais.
-   **Integração fácil**: conecte-se via nossa API REST ou SDKs compatíveis.
