### Endpoint

```
GET https://api.verifik.co/v2/co/foreigner-id/ce
```

Envie **`documentNumber`** e **`expeditionDate`** (`DD/MM/YYYY`) na **query**. O mesmo payload em **`POST`** (JSON).

### Cabeçalhos

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Requisitos do documento

**Para quem?** **Estrangeiros** legalmente residentes na Colômbia com *Cédula de Extranjería* (**CE**). **Não** serve para cidadãos colombianos (**`CC`** via `/v2/co/cedula`) nem para **PPT** ou **PEP** de imigração venezuelanos.

| Campo | Orientação |
| --- | --- |
| **`documentNumber`** | Número CE, **somente dígitos** (sem espaços ou pontuação). Na Colômbia, o CE costuma ter **6 ou 7 dígitos** (comprimento variável). Exemplos: `712046`, `1156043`. |
| **`expeditionDate`** | Data de expedição no formato **`DD/MM/AAAA`** (ex. `15/06/2020`). Obrigatória — formato incorreto costuma retornar **409**. |

Comparação completa: [Guia de documentos de identidade](/verifik-pt/identity-validation/colombia/colombia-identity-documents-guide).

### Parâmetros

| name | type | required | description |
| --- | --- | --- | --- |
| `documentNumber` | string | yes | Número CE, **somente dígitos**. Geralmente **6–7 dígitos** na Colômbia. Exemplos: `712046`, `1156043`. |
| `expeditionDate` | string | yes | Data de expedição **`DD/MM/AAAA`** (ex. `15/06/2020`). Deve coincidir com o documento. |

### Requisição

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/co/foreigner-id/ce", {
	params: { documentNumber: "987654321", expeditionDate: "15/06/2020" },
	headers: {
		Accept: "application/json",
		Authorization: `Bearer ${process.env.VERIFIK_TOKEN}`,
	},
});
console.log(data);
```
