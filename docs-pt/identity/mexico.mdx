# Verificação de identidade no México

A API Verifik autentica cidadãos mexicanos com o **CURP** (*Clave Única de Registro de Población*). Agiliza KYC, reduz fraude e apoia conformidade.

## O que esta API valida?

Registros oficiais para verificar:

-   **Nome completo e CURP**.
-   **Dados pessoais**: data de nascimento, sexo, nacionalidade.
-   **Correspondência**: o nome informado bate com o CURP?

## Referência da API

### Endpoint

```
https://api.verifik.co/v2/mx/curp
```

Este endpoint verifica o **CURP** (*Clave Única de Registro de Población*) de um cidadão mexicano em fontes oficiais. Envie `documentType` e `documentNumber` em uma requisição GET; em caso de sucesso, o registro de identidade é confirmado e nome e dados biográficos validados são retornados quando o CURP é reconhecido.

### Cabeçalhos

| Nome | Valor |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parâmetros

| Nome | Tipo | Obrigatório | Descrição |
| --- | --- | --- | --- |
| `documentType` | string | Sim | Valor permitido: `CURP`. |
| `documentNumber` | string | Sim | CURP a validar. |
