# Chile — Contribuinte (RUT)

Use este endpoint quando precisar de dados de identidade **orientados ao contribuinte** para um **RUT** chileno (não o fluxo civil da cédula). Ele complementa a **[Verificação de identidade no Chile](/identity/chile)** (`v2/cl/cedula`), voltada ao **RUN/registro civil**.

## API Reference

### Endpoint

```
https://api.verifik.co/v2/cl/taxpayer
```

Retorna campos de nome e identificadores alinhados ao **registro tributário**—indicado para **KYB**, onboarding de fornecedores e validação de notas fiscais em que o **RUT** deve bater com o **contribuinte** registrado.

### Headers

| Name | Value |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parameters

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `documentType` | string | Yes | Deve ser `RUT`. |
| `documentNumber` | string | Yes | Identificador fiscal (dígitos; formatação normalizada pelo serviço). |

### Request

<Tabs>
  <TabItem value="node" label="Node.js">

```javascript
import axios from "axios";

const { data } = await axios.get("https://api.verifik.co/v2/cl/taxpayer", {
  params: { documentType: "RUT", documentNumber: "263007905" },
  headers: { Accept: "application/json", Authorization: `Bearer ${process.env.VERIFIK_TOKEN}` },
});
console.log(data);
```

### Notes

- Prefira **`v2/cl/cedula`** para correspondência de identidade **RUN** no registro civil; use **`v2/cl/taxpayer`** quando a fonte de verdade for o **contribuinte RUT**.
