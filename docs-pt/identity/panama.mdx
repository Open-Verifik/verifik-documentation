# Verificação de identidade no Panamá

A API Verifik autentica titulares da **cédula panamenha** (*Cédula de Identidad Personal*, CCPA) com base em fontes oficiais. Simplifica **KYC**, reduz fraude e apoia a conformidade.

## O que esta API valida?

A API consulta registos oficiais para validar:

-   **CCPA e identidade** : número de documento normalizado e correspondência com o registo civil.
-   **Data de nascimento** : deve coincidir com o registo (`dateOfBirth` em `DD/MM/YYYY`).

## Referência da API

### Endpoint

```
https://api.verifik.co/v2/pa/cedula
```

Validação da **CCPA** panamenha contra registos civis oficiais. Envie **`documentType`** (`CCPA`), **`documentNumber`** normalizado e **`dateOfBirth`** em **`DD/MM/YYYY`** ; a resposta confirma a coerência dos dados para **KYC**, onboarding e controlos antifraude.

### Cabeçalhos

| Nome | Valor |
| --- | --- |
| Accept | `application/json` |
| Authorization | `Bearer <token>` |

### Parâmetros

| Nome | Tipo | Obrigatório | Descrição |
| --- | --- | --- | --- |
| `documentType` | string | Sim | Tipo de documento. Valor permitido: `CCPA`. |
| `documentNumber` | string | Sim | Cédula completa após normalização. |
| `dateOfBirth` | string | Sim | Data de nascimento no formato `DD/MM/YYYY`. |
